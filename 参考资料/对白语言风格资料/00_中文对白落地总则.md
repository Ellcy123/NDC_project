# 00 · 中文对白落地总则（最高优先级 — 先读本份，再读其余 6 份）

> **本份是 governing 文档。** 同目录其余 6 份（俚语 / 场合语域 / 性别 / 族裔 / 职业 / 阶级）是 1920s 美国英语的**史料底座**，里面全是英文语料。
>
> **但本游戏的对白是中文。** 6 份里的英文整句，是用来让你**理解某一类人说话时的紧张感、节奏、底气、绕弯方式**——不是可以逐句直译成台词的模板。先用本总则建立"怎么把英文语域转成克制的中文"的判断，再去 6 份里查具体的族裔 / 阶级 / 职业 / 俚语知识。
>
> 适用对象：narrative-designer / puzzle-designer / dialogue-writer / dialogue-reviewer。产出对白后，**逐条对照第二节铁律 + 第五节自检清单**。
>
> ⚠️ **修订（2026-05）**：原「铁律 1 · 经济性优先」**已删除**。台词不再以"能短就短 / 最少字传达"为取向——语言节奏与人设质感以 team-dialogue skill **规则 1（像人话）+ 规则 1.5（人设碎嘴写足 / 补连接戏与世界质感）** 为准，二者优先级最高。碎嘴型人设（Morrison 形态一等）的废话是人设，要写足；整场不得砍成信息点清单。Zack 的克制单列在铁律「Zack 声音」，不受此影响。

## 一、语言载体现实

1. **台词与动作括注一律中文。**
2. 英文只在三种情况出现，且每次都要被语境立即消化：
   - 专有名词（人名 / 地名 / 报名 / 机构名）：`Webb`、`蓝月亮`、`《艾斯弗德先驱报》`
   - in-fiction 的暗号 / 术语，作为剧情道具出现：`King Oliver sent us.`（说完插销依次拉开，语境自洽）
   - 角色**刻意的一处**夹词 / 口误，作为人物标记：Rosa `我是……我是……Witches!(女巫)` → Zack `那是Witness。` —— 当场有中文回应，口误本身是戏剧节拍
3. **禁止**：把英文俚语整句塞进中文台词；写"翻译腔"中文（从英文模板逐字译过来、读起来不像中国人写的句子，如"我恐怕必须恭敬地表示不同意"）。

## 二、Unit9 风格铁律（产出后逐条自检）

> 原「铁律 1 · 经济性优先」已删除（2026-05）。台词节奏与人设密度以 skill 规则 1 / 1.5 为准——单句不强求最短，碎嘴型人设要写足，整场不得砍成信息点清单。Zack 的钝、收场不靠台词撑场（`好。`/`哼。`）等克制写法，**移入并见下方铁律「Zack 声音」**，仅约束 Zack，不外推到 NPC。下列铁律编号保留原序（2/3/4/5）以兼容既有引用。

### 铁律 2 · 角色塑造靠动作括注，不靠台词堆特征

人物是谁，主要由 `[ ]` 里的身体动作、节奏、视线承载，不是让台词自报家门或满口标签。

✅ 一句动作括注立住人物：
> **扎克·布伦南** [把棒棒糖从左边移到右边，语气直白]
> 彼此彼此。

> **罗莎·马丁内斯** [把拖把把儿攥得更紧，手指关节发白]
> 我...月底了，Webb 先生要求彻底清洁。

谎言 / 情绪信号也走动作和节奏，**不写"她紧张地撒谎说"**——写她怎么动、语速怎么变（903001206：`[语速稳下来，像是在背一段话——句子有点太整齐]`）。

### 铁律 3 · 族裔 / 阶级标记 —— 少而准，一处足矣

❌ 写手直接套 `族裔与移民语言特征.md` 的英文 broken-English 模板、逐句译成中文：
> **罗莎·马丁内斯**
> 先生，我不知道什么，我只是清洁工，我什么都没看见，求您了先生，我是老实人，我每天都老实工作，señor，我对圣母发誓……

刻板、啰嗦、像漫画，破坏克制感。

✅ Unit9 实际（Loop1 903001005）—— 一个点同时立住族裔 + 人物 + 戏剧张力：
> **罗莎·马丁内斯**
> 所以我在场，先生。我有资格说——我是……我是……Witches!(女巫)
>
> **扎克·布伦南**
> 那是Witness。(目击者)

原则：移民 / 阶级身份用**一处**精准语言点（一个用词、一个口误、一个夹词、一种过度客气的句式），其余靠动作括注。**每个 NPC 最多立 1-2 个识别维度，不是族裔+性别+职业+阶级四个全堆。**

### 铁律 4 · Zack 声音

钝、克制、零感叹号。**只陈述他观察到的事实，然后停**——不替玩家把逻辑补完、不说"所以这说明……"、不戳穿（Talk 阶段尤其）。单次发言最长 2 句。出示证据 + 指出矛盾 + 停，是他的固定句法。

✅ 指出可疑但不下结论（Loop1 903001207）：
> **扎克·布伦南** [看着她]
> 你说话听起来，像是事先想好了要说这几句。

—— 没有接"所以你在撒谎"。判断留给玩家（呼应游戏设计三大原则·原则一：绝不提前破梗）。

Zack 卡住 / 收场不靠台词撑场：`好。` / `哼。`（903901405：`[把枪收回。停了三秒，没有说话]` + `哼。`）。**注意**：此"钝/短/沉默"仅约束 Zack，不外推到 NPC——NPC（尤其碎嘴型人设）按 skill 规则 1.5 写足。

### 铁律 5 · 1920s 味靠道具 / 氛围 / in-fiction 行为，不靠俚语密度

观察孔盖、三道插销依次拉开、棒棒糖、速记本、没散尽的油墨味、暗号——这些落在动作括注和场景里。**不要靠往台词里堆英文俚语来"做年代感"**，译成中文就垮，且违反第一节。6 份资料里的俚语 / 黑话用来**理解**那个圈子的人怎么想、在意什么，不是台词配料。

## 三、英文语域 → 中文落地的转换原则

不要译 idiom，要译**语域信号**。同一个"意思"在英文里靠词汇阶层区分，在中文里主要靠下面这些维度落点：

| 英文资料给的信号 | 中文里怎么落 |
|---|---|
| Old Money 长从句 / 间接 / 被动 | 句子完整、绕一层、用"恐怕""未必""不妨"，不抬高音量 |
| 工人阶级 ain't / 双重否定 / 短促 | 短句、断句、省略主语、动作补足，不修辞 |
| 移民 broken English / 母语夹词 | **一处**用词卡顿或口误 + 过度客气的称呼（"先生""求您了"），不堆语法错误 |
| 黑帮委婉语（"had an accident"） | 中文里也用回避式说法（"出了点事""被处理掉了"），靠语境而非直译 |
| 硬汉侦探金句（"Cut the crap"） | Zack 不说金句。换成钝陈述 + 动作（见铁律 4） |
| 感叹号 / 夸张情绪 | 改为动作括注里的身体反应（"杯子轻微抖了一下"），台词本身收着 |

核心：英文资料告诉你"这种人**在意什么、怕什么、绕不绕弯**"；你用中文 + 动作括注把那个**底气和紧张感**落地，而不是把那句英文搬过来。

## 四、6 份资料的正确用法（流程）

1. **先定人物**：从 `{npc}.md` 选定这个 NPC 要立的 **1-2 个**识别维度（不是四个全要）。
2. **再查资料**：去对应那 1-2 份资料，读英文示例是为了**理解那类人说话的紧张感 / 节奏 / 底气 / 在意点**。
3. **中文落地**：用中文台词 + 动作括注表达那个语域，按第三节转换原则，不直译。
4. **对照自检**：逐条过第二节五条铁律 + 第五节清单。

资料是字典，按需查；不是模板，不照抄。
