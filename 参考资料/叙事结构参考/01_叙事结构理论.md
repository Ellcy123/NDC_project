# 叙事结构理论

## 1.1 起承转合（Kishotenketsu）

起承转合是东亚传统的四段式叙事结构，源自中国古典文论，在日本被系统化为 kishotenketsu，广泛应用于漫画、游戏和电影。

**与西方三幕剧的核心区别：**

| 维度 | 三幕剧 | 起承转合 |
|------|--------|---------|
| 驱动力 | 冲突升级 | 认知重构 |
| 第三段的功能 | 危机加深（Climax） | 视角翻转（Ten/转） |
| 观众体验 | "事情越来越糟了！" | "原来如此！之前理解的都是错的！" |
| 适合的类型 | 动作、惊悚 | 悬疑、推理、解谜 |

**四段详解：**

- **起（Ki / 引入）**：建立世界观、角色关系和初始问题。观众/玩家接受一个"理解框架"。
  - 例：《逆转裁判》每个案件的第一天审判——看似铁证如山的控方论述建立了"错误的框架"。
  - NDC 例：L1-L2，玩家接受"Webb 被情杀"的初始框架，调查 Rosa 和 Tommy 获取关于 Vivian 的信息。

- **承（Sho / 展开）**：在引入的框架内深化。不是制造新冲突，而是让观众越来越深地"相信"这个框架。
  - 例：《弹丸论破》学级裁判中段——越来越多的证据似乎指向某个嫌疑人，玩家逐渐被引导向错误方向。
  - NDC 例：L2 Tommy 的保护行为和 Vivian 的仇恨日记让"Vivian 是凶手"这个框架越来越可信。

- **转（Ten / 翻转）**：这是起承转合最关键的一段。不是冲突升级，而是**引入一个全新视角，让之前积累的所有信息产生新的含义**。观众不是"被吓到"而是"被翻转"。
  - 例：《超级马里奥》关卡设计（宫本茂/林田宏一在 GDC 演讲中阐述）——前两段教会玩家一个机制，第三段用意想不到的方式组合这个机制，让玩家重新理解它的可能性。
  - 例：《逆转裁判3》第五案——当玩家发现自己一直在扮演的"师匠"其实是案件的关键参与者时，之前所有的"教学场景"被重新解读。
  - NDC 例：L3 Vivian "认罪"但凶器不匹配——"Webb 被情杀"的框架崩塌，玩家被迫重新审视所有证据。

- **合（Ketsu / 收束）**：在新理解的基础上收束所有线索。观众带着"翻转后的眼光"重新审视之前的一切，获得更深层的满足。
  - NDC 例：L5 完整时间线重建后，玩家回顾 L1-L4 的所有细节，发现每一条看似无关的证据都指向 Jimmy。

**为什么起承转合特别适合悬疑游戏：**

悬疑游戏的核心体验不是"越来越紧张"（那是惊悚），而是"恍然大悟"——"原来如此！所有线索早就在那里了！"这正是"转"的功能：不是给新信息，而是让旧信息产生新含义。

玩家在"承"阶段积累的所有证据、证词、疑点，在"转"的瞬间被重新解读。这种体验比单纯的"情节反转"更有深度，因为玩家不是被动接受反转，而是主动发现"自己之前的推理框架是错的"。

## 1.2 逆转裁判的"案中案"循环结构

巧舟（Shu Takumi）在 GCC '18 的演讲中分享了逆转裁判系列的核心叙事引擎：

```
大谜题
 └── 小谜题 A → 解决 → 新发展 →
     小谜题 B → 解决 → 新发展 →
     小谜题 C → 解决 → 大逆转 → 结案
```

**核心洞见（来自巧舟原话大意）：**

> 做第一代逆转裁判时，我先列出自己喜欢的"矛盾点"，然后围绕矛盾设计案件。到了第二代，我反过来——先设计情感故事，再反向工程出支撑故事的矛盾。

这意味着成熟的悬疑设计应该是：
1. 先确定你想讲的**情感故事**（角色的选择、困境、代价）
2. 再反向设计**支撑这个故事的矛盾点**（谎言、证据冲突、时间线矛盾）
3. 最后安排**这些矛盾被发现的顺序**（节奏控制）

**逆转裁判的两种悬疑类型：**

巧舟区分了两种悬疑模式，逆转裁判巧妙地混合使用：

| 类型 | 定义 | 玩家体验 | 例 |
|------|------|---------|---|
| 纯悬疑（Mystery） | 观众和角色都不知道真相 | 和角色一起探索 | L1-L3：谁杀了 Webb？ |
| 戏剧反讽（Suspense） | 观众知道（或隐约知道），角色不知道 | 看角色逼近真相的紧张感 | L4-L5：玩家已怀疑 Jimmy，看 Zack 如何证明 |

好的悬疑游戏在前半段使用"纯悬疑"（玩家和角色一起探索），在后半段逐渐过渡到"戏剧反讽"（玩家隐约猜到真相，享受看证据拼图完成的过程）。

## 1.3 Carolyn Wheat 的"鬼屋 vs 过山车"

Carolyn Wheat 在《How to Write Killer Fiction》中将悬疑叙事分为两种根本模式：

**鬼屋模式（Mystery / Whodunit）：**
- 核心体验：探索隐藏的房间，每打开一扇门就发现新秘密
- 节奏：缓慢积累，逐层深入
- 玩家心态："这扇门后面藏着什么？"
- 适合：推理解谜类游戏（NDC 主要属于这一类）

**过山车模式（Suspense / Thriller）：**
- 核心体验：速度和失控感，事情不断恶化
- 节奏：快速升级，一个危机接一个
- 玩家心态："接下来会发生什么可怕的事？"
- 适合：动作惊悚类游戏

**Wheat 的关键原则——掩盖比犯罪更重要：**

> 在好的悬疑中，凶手的掩盖行为（cover-up）比原始犯罪更能推动故事。因为掩盖行为会产生新的悬念、新的谎言、新的危险。

这解释了为什么好的悬疑故事中凶手必须"继续活动"——不是因为他们需要杀更多人，而是因为他们的掩盖行为本身成为新的谜题层。

**NDC 的应用：**
- Morrison 急着以"情杀"结案 = 掩盖行为
- Jimmy 伪造枪声、模仿 Webb 声音 = 精心设计的掩盖
- 这些掩盖行为本身就是最好的线索——它们既推动故事，又为玩家提供破案方向

## 1.4 四幕悬疑结构（Zara Altair 框架）

Zara Altair 提出了专门针对悬疑小说的四幕结构：

| 幕 | 功能 | 对应 NDC 设计 |
|---|------|--------------|
| Act 1：建立 | 案件发生 + 初步调查 + 建立调查方向 | L1：Zack 醒来，开始调查谁迷晕了他 |
| Act 2a：误导 | 调查深入 + 假线索引导 + 主嫌疑人浮现 | L2-L3 前半：证据指向 Vivian |
| Act 2b：中点翻转 | 主嫌疑人被排除或案件性质改变 | L3 后半：凶器不匹配，Vivian 不是凶手 |
| Act 3：真相逼近 | 新方向调查 + 真凶浮现 + 最终对决 | L4-L5：Morrison 阴谋 → Jimmy 真凶 |
| Act 4：解决 | 收束线索 + 情感落地 + 新悬念（如有续集） | L6：录音带 + Whale 威胁 + Unit 2 钩子 |

**中点翻转（Midpoint Reversal）是关键：**

Altair 强调中点翻转是悬疑叙事最重要的结构节点。好的中点翻转必须同时做到三件事：
1. **排除**之前最可能的嫌疑人（否则没有翻转）
2. **揭示**案件比预想的更复杂（否则没有深度）
3. **指出**新的调查方向（否则故事会停滞）
