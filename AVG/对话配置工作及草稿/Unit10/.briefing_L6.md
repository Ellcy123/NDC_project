## 一、破梗红线（CLAUDE.md 原则一·全局最高·违反=FAIL）

终局逐条铁律：

1. **第三重反转（Vinnie 单方面深爱 Leonard）台词层完全零可见**。① Vinnie 字条仅四句中文（"Leo——/这件事，让我替你做完。/我从来没指望你知道。/但能再帮你一次也好。/——V."），**没有一个字直接说"爱"**——玩家能读懂、Leonard 能读懂，靠的是"替你做完/没指望你知道/再帮你一次"的份量，不是明说。② **Leonard R4 崩溃后最后一句完整台词"是...是我做的..."里绝无 Vinnie 名字**——这是角色设计核心，**严禁**写手在崩溃后加"对不起 Vinnie"/"我害了他"/任何提及 Vinnie 的句子；这个缺席比任何台词都说明问题。③ Leonard 全程不知 Vinnie 为他能做到如此（双重信息不对称）——他对 Vinnie 顶罪的最大反应是"这不合理"不是"他怎么会爱我到这地步"；R4 崩溃是**信息差**，不是悲伤/感激/震惊。④ Act2 Leonard 知晓字条："我不知道他会……"然后没有后续——无泪、长时间不动，不展开。
2. **Leonard 不是咆哮的银行家**。全部威胁感来自合规话术密度——越平静越危险。R3 招供"是！我用烛台打了他！"后立刻甩锅"我离开时他还活着，是 Vinnie 后来杀的"必须**机械语气**、不加任何情绪强度（这是他全本最冷的句子，对 Vinnie 零愧疚）。不咆哮、不哭、不拍桌、不求饶。说谎告示词"我要直说"/"说实话"反讽标记**全程一致**——他真想认罪（R4 崩溃）反而**不说**这两个词；若崩溃后写了"我要直说，是我做的"则设计崩坏=FAIL。
3. **1900 南区双家纵火 / Russo 火里救 Vinnie / Vinnie 手疤来源 Unit10 一字不点破**（Unit11+ 才展开）。10609 旧报纸剪报是 envir 环境叙事——Leonard 对改名/旧事**回避式**：不主动谈、不主动否认、被直接出示证据才简短带过然后转移话题。意大利舌音仅 3 情况漏（被当面叫"Russo"瞬间即纠正 / R3 招供失控几秒 / 童年记忆被触发如出示剪报），其余 99% 标准白人发音。
4. **Mickey=Whale（Unit11+）绝不暴雷**。ending 无 Mickey 在场；opening 若提及 L5 Mickey 拿举报材料只中性陈述不可疑。**Whale 系统 / Moore 真凶逍遥**是 Unit11 开门钩子——Act4 Zack 苦笑"Moore 银行呢？真正的罪犯还在逍遥法外"+Emma"我们还没完"点到即止，**绝不展开 Whale 真身/不暗示 Mickey**。
5. **Foster-Miller 资助关系 / Morrison-Whale 往来 Unit10 保留不主动提**（foster.md / state withheld）。Foster 专业冷静只在证据范围内说，Miller 线一个字不碰。Morrison 签搜查令是**疲惫腐败警察兑现唯一一句承诺的体面，不是悔改**——不争辩、不拖延、签字本身就是全部态度。
6. **诱导招供系统喂话豁免仅限该阶段**（state expose.induce_system.mechanic_exception_note）。3 个 induce 节点的"温和追问"正确选项确实是在喂 Leonard 招供——这是 state/大纲背书的设计机制（道德灰度：利用 Leonard 对 Vinnie 的工具性信任逼供），**仅在 induce_system 这 3 节点内豁免"不替玩家说结论"**；① 喂的是 Leonard 招供，**不是向玩家揭真相** ② L6 其他所有对话（R1-R3 出证 / R4 出证 / ending）仍严格遵守 CLAUDE.md 原则一不替玩家下结论 ③ 道德灰度本身不点破（不写"Zack 在欺骗他""这样做对吗"）。
7. **Margaret 证词三维度（时间锚/「独自」/踉跄）玩家自己意识到**。Margaret 只说一次（按时间顺序、无多余形容词、不说"我觉得他可疑"只说看到的），Zack/Emma **不点破**"这句话有三个用处"。Margaret 是**母亲立场不是证人立场**——不表演母爱、不眼泪汪汪、苏醒第一反应先问 Frank 案再问自己（角色弧灵魂）。
8. **R4 真相（Vinnie 到场时 Frank 已死、Vinnie 把尸体搬上床伪装"睡觉失火"为 Leonard 顶罪）由证词矛盾推出**。R4 出证用 5080002b（Vinnie"透过窗户看到他在床上睡觉然后放火"）对撞 Leonard"打斗中倒地"——Zack 出证指出矛盾即可，**不替玩家说"所以 Vinnie 在护着 Leonard 编故事"**（让玩家自己拼；induce 阶段豁免不延伸到 R4 出证）。
9. **Zack 不替玩家下结论**（induce 3 节点除外）：可出证、指矛盾、追问、回呛、带情绪、用私人重量（"我妈"——Margaret 是他母亲，这层私人重量可用但不煽情），但不像旁白说"所以这说明…"；玩家自己补逻辑链。
10. **N3 大纲"待确认"已由 state/讨论结论5.3 定稿**：Vinnie 字条方案落地（四句中文版本），按 state ending_sequence 原样写，大纲该处为 stale，不据大纲"待确认"动摇。

## 三、文本规则 R1–R10（原样，违反=FAIL）

### R1 · 一来一回的对白，不是演讲
对白是 dialogue 不是 monologue。任何一方不能写成"一方播报/演讲+另一方点头/转述"。该追问就追问、该回呛就回呛，可打断、可半句、可语气词、可带情绪。沉默用动作括注承载，不写空台词。

### R2 · 先懂大纲，再写本场
动笔前弄清本场在整章叙事功能：揭哪层真相、为哪个 Loop 埋什么伏笔、玩家此刻情绪、角色弧光哪一步、与上下场衔接。L6 特别：这是终局合拢，玩家带着 6 个 Loop 的积累而来；第三重反转必须"台词零可见、玩家自己拼"。

### R3 · 人设是角色声音的唯一依据
只看 `{npc}.md` 的声音风格/禁忌/设计备注/关键关系。照人设写，不堆刻板标签。每个 NPC 首句让人不看档案也认得出。Leonard 四形态（合规话术+主动表白 / 被轻触动慢半拍 / 被精确证据卡住退守 / R3 招供甩锅+R4 崩溃机械冷）切换靠词汇密度句长不靠音量。**严禁引用 `参考资料/对白语言风格资料/`（已弃用）**。

### R4 · 人设质感与连接戏（正向强制）
state 信息点是骨架不是全部。每单元要有服务①人物声音②人物关系③世界/时代质感的台词。删掉所有 state 信息点后剩下台词仍能让人认出"这是谁/什么关系/什么世界"。Margaret"地下室反正空着"式的克制、Edith"不解释的拒绝"、Leonard 永远歪的领带式社畜疲惫——这些质感要写足。

### R5 ·（已删除，由 CLAUDE.md 原则一全局管辖，见上"破梗红线"+诱导招供豁免）
Zack 声音说明：Zack 可激烈一来一回、追问、回呛、带情绪、用私人重量（Margaret 是他母亲）顶上去——他逼线索不替玩家说答案（induce 3 节点除外）。

### R6 · 信息严密 · 游戏设计原则二
证据物理属性/数值/时间前后一致、跨 Loop 引用不矛盾；证词摘要与原文信息密度对等（不丢限定词不膨胀结论）；每个 NPC 在知识边界内自洽——Leonard 即使说谎，谎言本身也有内在逻辑（合规话术的"就算…也不能证明"框架）。

### R7 · 旁白克制不空 + 无大段独白
环境/动作旁白少写；世界质感由人物台词承载。任何角色不写大段内心独白/演讲（opening Zack 的"察觉 Vinnie 认罪不对劲"靠简短动作/与 Emma 短对，**不写成 Zack 内心独白长篇**）。纯画面/转场/NPC 静默反应用 `<!-- -->` 注释或并进相邻台词动作括号 `[...]`，**不写成空 `**角色名**` 发言块**（承载 @get/@branch/@lie/@goto/@induce 机制处除外）。ending_sequence forced_cutscene 4 act 用旁白文本块+少量台词（过场体裁本身，非空发言块）。

### R8 · 格式与命名
- 节点边界=一个 `**角色名**` 发言块；不写 9 位对话 ID。
- @指令：`@branch/@opt "文本" -> 标签 #维度/@path/@goto/@label`；`@get 证词|证据 {ID} "{摘要}" #{keyInfoType}`；Expose `@round N/@lie anchor={id|null}/@evidence "名" -> 标签 #correct|#trap "理由"`。诱导招供按 state induce_system 设计落地（见统一方案 §8，用 @branch/@opt 结构表达 3 节点温和/强硬选择 + 槽值注释 + QTE 注释；无ID规范无专用 @induce 语法则用 @branch 等价表达并注释机制）。
- 证词 7 位 `{loop}{npc_code:2}{seq:4}`（**注：state 定义 6110001/6120001-3/6050001 等为 7位，state 行22 误写"6位"——按 state 实际 ID 原样使用，属上游 ID 体系待定，非本 skill 处理**）；Unit10 证据 5 位（10xxx）。
- 角色发言名统一中文译名：扎克·布伦南 / 玛格丽特 / 福斯特 / 莫里森 / 伦纳德 / 伊迪丝 / 露拉 / 艾玛。台词内英文名/刻字/字条署名"V."/"Leo"属对白内容不算违规。
- 文件名取自 state 字段。

### R9 · 承上启下：开场锚定 + 结尾留勾（都不机械）
- 开场锚定（≠重新进场）：每段非硬切对话开头一两句自然台词/动作让玩家落地"我此刻在哪、上一步刚发生什么、现在紧张点"。靠人物之口/动作，禁旁白 info-dump、禁重复进场。
- 结尾留勾（≠任务提示）：结尾由人物自然带出"下一步往哪使劲"。禁 Zack 总结式、禁任务播报、禁破梗。
- cutscene/transition/forced_cutscene 不强制开场锚定但结尾仍要自然前推/悬念。**终局 ending Act4 是 Unit10 收束+Unit11 钩子**——Zack 苦笑+Emma"我们还没完"画面对视淡出，是整个 Unit10 的句点也是开门钩，分量最重，最克制。
- 段与段衔接：上一段的"勾"≈下一段开场要落地的"处境"。

### R10 · 地位跷跷板：场景必须转弯（不机械、不破梗）
每场对手戏"谁占上风"必须场内至少易手一次、被一句台词/一个动作挣得（一个转弯），非设定好/非旁白宣布。低位方必须有至少一拍真把强势方挡回去。**Loop6_leonard Expose 是重头戏**：Leonard 每轮用合规话术"就算…也不能证明"框架退守一层、挣回一拍（低位赢拍），Zack 逐轮精确证据压下；R3 后 Leonard 盯桌面沉默（全本最沉一拍）；诱导招供阶段天平由**诱导槽 + Edith 裙角第三方支点**掀（Edith 不主动配合，她出现的事实本身是 Leonard 招供触发点）；R4 Vinnie 证词矛盾 full_victory。第三方支点（Edith）鼓励用——她倒向"离开"的事实让 Leonard 失去比坐牢更重。豁免：opening/morrison 过场/ending forced_cutscene（纯过场无对手关系）、废墟搜证。
