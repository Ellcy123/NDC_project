# Unit 2 跨循环场景继承补充说明

## 核心规则

**解锁状态跨循环继承**：一旦解锁的场景，在后续循环中保持可用。

玩家可以自由进入已解锁的场景，但在非关键循环访问时可能无法获取新的关键信息。

---

## 场景类型与标记说明

### 场景阶段类型

| 阶段类型 | 标记 | 说明 | 是否可自由访问 |
|:---|:---|:---|:---:|
| 开篇场景 | 🎬 开篇 | 循环开始的剧情演出 | ❌ 硬切 |
| 自由探索 | 🔓 自由探索 | 玩家可自由移动、搜证、对话 | ✅ 可自由访问 |
| 突发事件 | ⚡ 突发事件 | 剧情中触发的特殊事件 | ❌ 硬切 |
| 指证场景 | ⚔️ 指证 | 进行指证对决的场景 | ❌ 硬切 |
| 结尾场景 | 🎭 结尾 | 循环结束的剧情演出 | ❌ 硬切 |

### 继承场景状态标记

| 标记 | 含义 |
|:---|:---|
| 🆕 | 本循环新增场景 |
| 📦 | 继承场景（来自之前循环） |
| 🔄 有新线索 | 搜证场景，本循环有新证据可获取 |
| ⏸️ 无新线索 | 搜证场景，本循环无新内容 |
| 🔒 封锁 | 场景因特定原因无法进入（如搜查令失效） |
| 💬 NPC在场 | 对话场景，NPC仍在原位，可进行闲聊 |
| 🚫 NPC不在 | 对话场景，NPC因剧情原因不在场 |

### NPC闲聊对话设计建议

当玩家在非主线循环访问继承场景时，NPC可提供以下类型对话：

**类型1：简短问候**
> NPC: "嗨，Brennan探员。有什么事吗？"
> Zack: "只是路过看看。"
> NPC: "好的，有需要随时找我。"

**类型2：状态描述**
> NPC: "最近没什么特别的事。"

**类型3：引导提示**
> NPC: "你看起来很忙。是不是有更重要的事要处理？"

**类型4：情感回应（特定NPC）**
> Mrs. O'Hara: "还在调查Frank的事吗？希望能找到真相..."
> Rose: "谢谢你还记得来看我..."
> Danny: "...你还来干什么？"（指证后态度变差）
