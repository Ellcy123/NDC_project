- **场景阶段**: 🎬 开篇 + 🔓 限时搜证

- **场景状态**: 火灾现场，一具烧焦的尸体被白布盖着。尸体露出一只手，手被烧得焦黑但没有戒指

**开场对话**:

# 🔓 场景解锁机制

## 解锁类型说明

| 解锁类型 | 说明 |
|:---|:---|
| 硬切 | 开篇/结尾/指证/突发事件，剧情强制切换，无需条件 |
| 自动解锁 | 同一建筑或相邻区域，进入自由探索时自动可用 |
| 地图解锁 | 通过地图道具批量解锁一组地点 |
| 证据解锁 | 获取特定证据后解锁新地点 |
| 道具解锁 | 拥有特定道具（如钥匙）后解锁 |
| 剧情解锁 | 完成特定剧情事件后解锁 |
| 已解锁 | 前循环已访问，本循环继续可用 |
