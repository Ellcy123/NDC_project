│                                     │
│  贷款专员：Leonard Ross             │

## 💡 设计亮点

### 1. 连号钞票谜题的拼图设计

```
O'Hara钞票：900-904, [缺失905-909], 910-914
  +
Vinnie钞票：905, 906, 908
  =
完整序列：900-914（15张连号新钞）

结论：
• 这批钞票原本是同一批银行新钞
• O'Hara缺失的部分 = Vinnie手里的钞票
• 只有银行职员才能提取连号新钞
• 证明Leonard给了Vinnie钱
```

### 2. 三轮递进的逻辑闭环

| 轮次 | 谎言升级 | 证据击破 |

| 第二轮 | "没来往" → "给了钱" | 连号钞票拼图 |
| 第三轮 | "是借钱" → "是雇佣" | 工作记录+语言习惯 |
