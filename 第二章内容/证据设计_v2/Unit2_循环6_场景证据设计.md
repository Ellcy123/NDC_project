## 📝 设计说明

### 核心玩法

1. **烛台找茬**：照片对比发现异常 → 移开烛台 → 发现眼镜碎片
2. **跨循环合并**：空相框（循环5-506）+ 旧照片（循环3-318）= 揭示真相
3. **跨循环证据链**：循环5拜访Leonard时看到眼镜维修单据（505）→ 循环6发现眼镜碎片（608）→ 证据链完成

> **重要说明**：照片对比（烛台找茬）是一个**玩法机制**，不是独立的证据。
> - 证据606【房屋抵押证明（含照片）】是玩法的**触发道具**
> - 通过玩法发现烛台位置异常，从而获得601【烛台】和608【眼镜碎片】
> - 玩法本身的"发现"不作为单独证据呈堂，而是引导玩家获取实物证据

### 证据逻辑链

```
【循环5发现链】
拜访Leonard住所 → 观察到眼镜维修单据（505）
                → 观察到空相框（506）
                → 当时无法确定意义，但留下疑点

【循环6发现链】
照片对比 → 烛台位置变化 → 有人动过现场 → 移开烛台 → 发现眼镜碎片（608）
                                                    ↓
回忆循环5的眼镜维修单据（505）→ 证据链完成！

【推理链】
眼镜碎片在烛台下方
    ↓
碎片先掉落 → 然后烛台被移动盖住
    ↓
时间线：打斗时眼镜碎裂 → 凶手用烛台行凶 → Vinnie移动烛台布置现场
    ↓
烛台位置变化 = 有人伪造/整理过现场

【指向凶手】
碎片有弧度 → 眼镜镜片
碎片发黄 → 使用多年的旧镜片
Frank不戴眼镜 → 碎片来自凶手
    ↓
眼镜维修单据（循环5-505）：11月7日（火灾后第二天）
    ↓
Leonard换了右眼镜片 + 要求丢弃旧镜片（销毁证据）
    ↓
Leonard戴眼镜 + Margaret证词"看到戴眼镜的男人"
    ↓
Leonard = 真凶
```

### 设计亮点

| 要点 | 说明 |
|:---|:---|
| **照片对比玩法** | 利用循环1的照片作为"时间快照"，与当前现场对比发现异常 |
| **玩家主动发现** | 不是直接告诉玩家"烛台有问题"，而是让玩家通过对比自己发现 |
| **双重证据** | 一个发现揭示两件事：①有人伪造现场 ②凶手戴眼镜 |
| **跨循环设计** | 循环3照片 + 循环5相框 = 揭示真相；循环5维修单据 + 循环6碎片 = 锁定凶手 |
| **先伏笔后揭示** | 循环5拜访时看到的线索，在循环6发现新证据后才完成证据链 |
