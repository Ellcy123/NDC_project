## 🎯 Moore三层欺诈诡计解析

### 诡计结构

```
【第一层：代填代签】
Leonard帮Frank"填表签字"
  → Frank不知道自己签了什么
  → 用笔迹证据戳穿

【第二层：违法条款】
月利率12%复利 = 年化289%
  → 远超州法7%上限（超标40倍）
  → 用违法条款证据组合攻击

【第三层：投资公司马甲】
协议抬头是"投资公司"不是"银行"
  → 投资公司不受银行法限制
  → 但诱骗老人签署 = 刑事犯罪
```

## 💡 设计亮点

### 1. 利率计算器的数据分布

| 数据 | 来源证据 | 场景 | 备注 |
|:---|:---|:---|:---|
| 欠款$3,200 | 206催款单 | L2 Frank家信箱 | 玩家先知道结果 |
| 本金$600 | 303银行档案 | L3 Leonard办公室 | 发现只借600→震撼 |
| 月利率12% | 305利率条款 | L3 银行档案室 | 核心计算参数 |
| 期限12个月 | 306账户服务条款 | L3 银行档案室 | 核心计算参数 |
| 月管理费$50 | 308管理费条款 | L3 银行档案室 | 隐藏费用 |

**误导内容分布**（增加谜题难度）：
| 误导数据 | 来源证据 | 说明 |
|:---|:---|:---|
| 开户费$25 | 309客户对账单 | 与利率计算无关 |
| 宽限期30天 | 309客户对账单 | 与利率计算无关 |
| 续期36个月 | 306账户服务条款 | 是续期上限，非实际期限 |

玩家需要从多个证据中拼凑数据，同时排除误导信息，增加探索感。

### 2. 条款的真实性设计

银行文件使用1920s高利贷者的伪装语言：
- "月利率百分之十二，按复利方式计算"（305利率条款）
- "未结清之服务费用，将自动并入下月账户余额"（306账户服务条款）
  - "服务费用" = 利息的伪装
  - "账户余额" = 本金的伪装
- "每月须支付账户管理费五十美元"（308管理费条款）

玩家需要自己理解这些术语的真实含义。

### 3. 三层诡计的逻辑递进

```
第一层（表面）：代填代签
  ↓ 被戳穿后
第二层（深入）：违法条款
  ↓ 被戳穿后
第三层（核心）：投资公司马甲
  ↓ 系统性欺诈=刑事犯罪
```

### 4. 误导延续

- 玩家在循环3仍不知道Frank借款的真正原因
- Danny证词中的"支支吾吾"暗示他与原因有关
- 真相在循环4揭示
