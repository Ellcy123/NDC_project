## 循环5特殊设计

### 误导性结局的叙事意图

```
【表面】
Vinnie被打火机证据击破谎言
  → 认罪："是我放的火！Frank是我杀的！"
  → 玩家以为案件结束

【实际】
Vinnie认罪太快，像在掩护什么
  → 打火机"太完美"是事后放置
  → Vinnie对Mickey大喊"你背叛了Leo！"
  → 玩家开始怀疑：Leo是谁？

【循环6揭示】
Vinnie在掩护童年挚友Leonard (Leo)
  → 真凶是Leonard，Vinnie只是协助放火焚尸
```

### 悬念设置

| 疑点 | 当前状态 | 循环6揭示 |
|:---|:---|:---|
| 打火机太完好 | 玩家可能注意到 | Vinnie事后放置栽赃自己 |
| 认罪太快 | Emma说"有问题" | 掩护Leonard |
| "出卖了Leo" | Leo是谁？ | Leo = Leonard |
| Mickey知道Margaret位置 | 太巧了 | Mickey在黑白两道都有线人 |

## 设计亮点

### 1. 误导性结局的叙事技巧

| 表面叙事 | 真实叙事 |
|:---|:---|
| Vinnie是凶手 | Vinnie是帮凶，在掩护Leonard |
| 打火机是Vinnie遗落的 | 打火机是Vinnie事后放置的 |
| 案件结束 | 案件才刚开始 |
| Mickey是好人 | Mickey为Unit 3埋下伏笔 |

### 2. 疑点设置

```
疑点1: 打火机太完好
  → 玩家可能注意到
  → Emma提出疑问
  → 循环6揭示是事后放置

疑点2: 认罪太快
  → "别问了！"
  → 不等追问动机
  → 循环6揭示在掩护Leonard

疑点3: "出卖了Leo"
  → Leo是谁？
  → 循环6揭示Leo = Leonard
  → Vinnie和Leonard是童年好友

疑点4: Mickey神秘情报
  → 怎么知道Margaret位置？
  → 为Unit 3 Mickey背叛埋伏笔
```

### 3. 玩家情感设计

| 阶段 | 玩家情感 |

| 开篇 | "Frank的遗书...太悲伤了" |
| 指证时 | "终于抓到Vinnie了！" |
