### 方案1：Rose的定情戒指刻字 【循环4】

**对应证据**: 新增证据（Rose家搜证获得）

**设计背景**:
- Frank和Rose秘密相爱5年，因社会偏见（波兰裔+墨西哥裔）不敢公开
- Frank一定送过Rose某种定情信物
- 这个信物藏着两人的秘密

**新增证据设计**:

```

│  ┌─────────────────────────────────────────────────────┐   │

### 方案5：贷款协议的打字机对比 【循环3】

**对应证据**: 307（贷款申请表）、303（银行内部档案）

**现状**: 直接阅读发现违法条款

**改进设计**:

```
┌─────────────────────────────────────────────────────────────┐
│                 【分析玩法：打字机对比】                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  背景知识：                                                  │
│  1920年代，每台打字机都有独特的"指纹"：                     │
│  • 字母间距微小差异                                         │
│  • 某些字母的磨损痕迹                                       │
│  • 墨带浓淡不均                                             │
│                                                             │
│  触发条件：                                                  │
│  玩家将【贷款申请表】与【银行标准文件】放在一起对比         │
│                                                             │
│  玩法流程：                                                  │
│  1. 选择两份文件进行"打字特征对比"                          │
│  2. 放大镜模式：找出3处字母特征差异                         │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              【打字机特征对比】                       │   │
│  │                                                      │   │
│  │  银行标准文件：          贷款申请表：                │   │
│  │  ─────────────          ─────────────               │   │
│  │  "e"字母完整            "e"字母右下角缺损           │   │
│  │  "r"字母正常            "r"字母偏高0.5mm            │   │
│  │  行距均匀                行距第3-5行略窄             │   │
│  │                                                      │   │
│  │  结论：两份文件来自【不同的打字机】！                │   │

## 🎯 物理拆分设计原则

### 容器类型
1. **相框** - 可放入/取出照片
2. **首饰盒** - 有特定槽位放置首饰
3. **怀表** - 内盖可放小照片
4. **打火机** - 底盖可拆卸
5. **信封** - 放入信件
6. **吊坠** - 可打开的心形吊坠

### 交互方式
- **放入**：将内容物拖拽到容器上
- **打开**：点击容器查看内部
- **取出**：从容器中移除内容物
- **匹配**：系统检测尺寸/形状是否吻合

### 信息分层
- **容器单独**：提供部分信息（谁的东西、在哪发现）
- **内容物单独**：提供部分信息（内容是什么）
- **合并后**：完整信息 + 推理结论
