# NDC 设置系统策划案

## 一、系统概述

### 1.1 系统定位
设置系统（Settings & Help）是NDC游戏的全局配置界面，通过键盘 **F1** 键在游戏任意阶段唤出。该系统负责游戏进度管理、难度调整、音视频配置等核心功能。

### 1.2 入口方式
| 入口 | 触发方式 | 说明 |
|------|----------|------|
| F1键 | 键盘快捷键 | 在游戏任意界面均可触发 |
| 暂停菜单 | Esc键 | 暂停游戏后显示设置入口 |
| 主菜单 | 点击"设置"按钮 | 从游戏主界面进入 |

### 1.3 界面布局
采用左右分栏布局：
- **左侧面板**：设置选项区域（复古纸张风格）
- **右侧区域**：游戏场景背景（模糊处理）

## 二、主菜单选项

### 2.1 继续游戏（继续游戏 / 保存并继续游戏）
| 属性 | 说明 |
|------|------|
| 功能 | 关闭设置界面，返回当前游戏进度 |
| 触发条件 | 游戏进行中时可用 |
| 快捷键 | Esc |
| 状态显示 | 默认高亮选中状态 |

#### 设置修改状态提示
| 状态 | 按钮文本 | 样式 |
|------|----------|------|
| 无修改 | 继续游戏 | 红色背景（默认） |

#### 修改检测范围
检测以下设置项的变更：难度、语言、字幕开关、字幕大小、音乐音效、语音、亮度、屏幕比例、显示模式

### 2.2 案情回顾（案情回顾）
| 属性 | 说明 |
|------|------|
| 功能 | 跳转到案情回顾系统页面 |

| 跳转目标 | `案情回顾系统/case_review_system.html` |
| 功能说明 | 玩家可以查看当前章节的案情时间线、关键事件、人物关系和已收集证据 |

### 2.3 返回主菜单（返回主菜单）

|------|------|

| 选项 | [确定] / [取消] |

## 三、难度设置

### 3.1 难度选项
| 难度名称 | 中文显示 | 英文显示 |
|----------|----------|----------|
| 普通模式 | 普通模式 | NORMAL MODE |
| 专家模式 | 专家模式 | EXPERT MODE |

### 3.2 难度差异对照表

#### 3.2.1 场景探索
| 功能项 | 普通模式 | 专家模式 |
|--------|----------|----------|
| 热区提示 | 按F1键可高亮显示可交互区域 | 无任何提示 |
| 热区大小 | 较大（半径+20%） | 标准大小 |
| 已收集标记 | 已收集的证据位置显示勾选标记 | 不显示标记 |
| 环境叙事提示 | 靠近时自动显示提示图标 | 需要精确点击 |

#### 3.2.2 NPC对话
| 功能项 | 普通模式 | 专家模式 |
|--------|----------|----------|
| 关键词高亮 | 可提取内容显示下划线 | 无任何高亮 |
| 提取提示 | 提示"此内容可提取" | 无提示 |
| 对话回顾 | 可随时查看完整对话记录 | 同左 |

#### 3.2.3 案情分析
| 功能项 | 普通模式 | 专家模式 |
|--------|----------|----------|
| 组合提示 | 显示"需要X件物品组合" | 不显示数量 |
| 可组合标记 | 可组合的物品边框发光 | 无标记 |
| 分析建议 | 显示"建议分析此物品" | 无建议 |

#### 3.2.4 调查指证
| 功能项 | 普通模式 | 专家模式 |
|--------|----------|----------|
| 矛盾高亮 | 自动高亮时间线/事件矛盾点 | 需手动对比发现 |
| 指证提示 | 选错证据时提示"证据不足" | 仅显示"指证失败" |
| 重试限制 | 无限制 | 每轮最多3次错误（超过后强制使用提示） |

### 3.3 难度切换规则
| 规则 | 说明 |
|------|------|
| 切换时机 | 游戏任意阶段均可切换（含循环中） |
| 生效时机 | 确认后立即生效 |
| 成就影响 | 切换到普通模式后，当前章节无法获得"专家通关"成就 |

#### 难度切换确认弹窗
| 切换方向 | 英文标题 | 中文提示 | 警告信息 |
|----------|----------|----------|----------|
| 专家→普通 | Switch to Normal Mode? | 是否切换到普通模式？ | ⚠ 切换后当前章节无法获得"专家通关"成就 |
| 普通→专家 | Switch to Expert Mode? | 是否切换到专家模式？ | （无） |

### 3.4 循环中切换难度的技术处理
| 处理项 | 说明 |
|--------|------|
| UI提示立即更新 | 热区高亮、关键词下划线等立即按新难度显示/隐藏 |
| 已收集证据保留 | 已收集的证据不受影响 |
| 提示使用记录保留 | 本循环已使用的提示次数继续累计 |
| 评分规则切换 | 切换后的提示使用按新难度规则计算评分影响 |
| 重试次数重置 | 专家模式的3次重试限制在切换时不重置 |

## 七、画面设置

### 7.1 亮度调节（亮度）
| 属性 | 说明 |
|------|------|
| 控制方式 | 滑块（0-100） |
| 默认值 | 50 |
| 影响内容 | 整体画面明暗 |
| 实时预览 | 调整时实时生效 |

#### 亮度参考
| 数值 | 效果描述 |
|------|----------|
| 0-30 | 较暗，适合全黑环境 |
| 30-70 | 标准，适合一般环境 |
| 70-100 | 较亮，适合明亮环境 |

### 7.2 屏幕比例
| 选项 | 中文显示 | 说明 | 典型分辨率 |
|------|----------|------|------------|
| 16:9 | 16:9 | 标准宽屏（默认） | 1920×1080, 2560×1440, 3840×2160 |
| 21:9 | 21:9 | 超宽屏/带鱼屏 | 2560×1080, 3440×1440 |
| 16:10 | 16:10 | 宽屏变种 | 1920×1200, 2560×1600 |
| 4:3 | 4:3 | 传统屏幕 | 1024×768, 1280×960, 1600×1200 |

#### 屏幕比例说明
| 比例 | 适用场景 | 注意事项 |
|------|----------|----------|
| 16:9 | 最通用，兼容性最佳 | 推荐大多数玩家使用 |
| 21:9 | 超宽显示器，沉浸感更强 | 左右视野扩展，UI需适配 |
| 16:10 | 部分笔记本/显示器 | 垂直空间稍多 |
| 4:3 | 老式显示器 | 两侧会有黑边或裁剪 |

### 7.3 显示模式
| 选项 | 中文显示 | 说明 |
|------|----------|------|
| 全屏 | 全屏 | 独占全屏模式，性能最佳 |
| 窗口化 | 窗口化 | 可自由调整窗口大小，便于多任务 |
| 无边框窗口 | 无边框窗口 | 全屏显示但不独占，切换程序更流畅 |

#### 显示模式切换规则
| 规则 | 说明 |
|------|------|
| 生效时机 | 切换后立即生效 |
| 推荐模式 | 全屏（独占GPU资源，帧率最稳定） |
| 多任务推荐 | 无边框窗口（Alt+Tab切换无延迟） |
