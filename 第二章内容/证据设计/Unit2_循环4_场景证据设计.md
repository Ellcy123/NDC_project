## 🎯 Danny三重诬陷结构解析

### 诬陷结构

```
【第一重诬陷：时间】
"Rose最近才勾引我叔叔"
  → 用家具磨损、窗帘褪色、地板磨痕证明至少5年
  → Danny被迫承认"认识久了"

【第二重诬陷：金钱】
"Rose骗叔叔贷款"
  → 用贷款申请表真实用途（维修约会房间）
  → 用Margaret租赁协议证明Frank主动付出
  → Danny转而攻击"图遗产"

【第三重诬陷：财产】
"Rose图遗产，遗嘱就是证据"
  → 用遗嘱附加条款证明Frank尊重Rose意愿
  → Rose当场签署放弃继承，用行动击碎诬陷
  → Danny真实动机暴露：他才是图财产的人
```
