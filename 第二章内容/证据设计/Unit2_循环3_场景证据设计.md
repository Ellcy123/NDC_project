## 🎯 Moore三层欺诈诡计解析

### 诡计结构

```
【第一层：代填代签】
Leonard帮Frank"填表签字"
  → Frank不知道自己签了什么
  → 用笔迹证据戳穿

【第二层：违法条款】
月利率12%复利 = 年化289%
  → 远超州法7%上限（超标40倍）
  → 用违法条款证据组合攻击

【第三层：投资公司马甲】
协议抬头是"投资公司"不是"银行"
  → 投资公司不受银行法限制
  → 但诱骗老人签署 = 刑事犯罪
```

### 为什么这个诡计在1925年成立？

| 法律背景 | 说明 |
|:---|:---|
| 银行法利率上限 | 伊利诺伊州规定消费者贷款年化利率上限7% |
| 投资公司监管空白 | 1925年投资公司无联邦监管（直到1933年证券法才有） |
| Moore的漏洞利用 | 用"投资公司"马甲规避银行法，同时让客户以为是银行贷款 |
