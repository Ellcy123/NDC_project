## 🎯 循环5特殊设计

### 误导性结局的叙事意图

```
【表面】
Vinnie被打火机+Rose证词击破谎言
  → 认罪："是我放的火！Frank是我杀的！"
  → 玩家以为案件结束

【实际】
Vinnie认罪太快，像在掩护什么
  → 打火机"太完美"是事后放置
  → Vinnie对Mickey大喊"你背叛了Leo！"
  → 玩家开始怀疑：Leo是谁？

【循环6揭示】
Vinnie在掩护童年挚友Leonard (Leo)
  → 真凶是Leonard，Vinnie只是协助放火焚尸
```

### 悬念设置

| 疑点 | 当前状态 | 循环6揭示 |
|:---|:---|:---|
| 打火机太完好 | 玩家可能注意到 | Vinnie事后放置栽赃自己 |
| 认罪太快 | Emma说"有问题" | 掩护Leonard |
| "出卖了Leo" | Leo是谁？ | Leo = Leonard |
| Mickey知道Margaret位置 | 太巧了 | Mickey在黑白两道都有线人 |
