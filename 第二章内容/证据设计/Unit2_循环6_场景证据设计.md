## 🎯 循环6特殊设计

### 证据困境与解决方案

```
【证据困境】
大火烧毁了大部分直接证据
  → 血迹、指纹、现场痕迹全部毁灭
  → 只剩物理证据（烛台凹痕+眼镜碎片）
  → 和间接证词（Margaret目击+发现Frank倒地）

【解决方案：诱导招供】
Zack不知道Vinnie认罪细节
Leonard也不知道Vinnie认罪细节
  → Zack告知Leonard"Vinnie已认罪"
  → Leonard以为"承认打斗没事"
  → Leonard主动承认用烛台打Frank
  → 但Leonard甩锅给Vinnie"是他烧死的"
  → Zack用Vinnie证词矛盾一击致命
```

### 双重信息不对称

| 不对称 | Leonard不知道 | 导致结果 |
|:---|:---|:---|
| 情感层 | Vinnie深爱他 | Leonard利用Vinnie的忠诚 |
| 信息层 | Vinnie认罪细节 | Leonard编造的"Vinnie看到受伤倒地"与Vinnie说的"床上睡觉"矛盾 |

### 🏷️ 证据601：烧焦的烛台（带眼镜碎片） ⭐

**证据类型**: 实体证据（指证核心）
**发现位置**: 物品栏（通过组合获得）
**获取方式**: 使用【证据611：医用镊子】对【证据506：烧焦的烛台】进行互动分析

```
┌─────────────────────────────────────┐
│    【物品互动分析】                  │
├─────────────────────────────────────┤
│                                     │
│  [Zack拿出背包里的烧焦烛台]          │
│                                     │
│  Zack："既然死因是钝器..."          │
│                                     │
│  [使用医用镊子检查底座缝隙]          │
│                                     │
│  "这里好像卡着什么东西..."           │
│                                     │
│  [小心翼翼地夹出一块微小的碎片]      │
│                                     │
│  "这是...玻璃碎片？"                │
│  "而且有弧度...像是眼镜镜片。"       │
│                                     │
│  ─────────────────────────────      │
│                                     │
│  证据更新：【烧焦的烛台】            │
│  状态变更：已提取关键微物线索        │
│                                     │
│  新发现：                            │
│  • 烛台底座缝隙卡着眼镜碎片          │
│  • 碎片边缘锐利，是在撞击时碎裂的    │
│  • 说明持有烛台的人戴眼镜            │
│    或者被攻击的人戴眼镜              │
│    (注：Frank不戴眼镜)               │
│                                     │
└─────────────────────────────────────┘
```

**证据作用**:
- 证明烛台就是造成Frank头部伤口的凶器
- 证明凶手当时戴着眼镜，且眼镜在冲突中破碎
- 结合【证据612/604】（Leonard的异色眼镜），直接指向Leonard

**与循环5证据506的关系**:
- 循环5中发现"烧焦的烛台"（506），只能观察到凹痕和"缝隙里好像有东西"
- 循环6获得镊子后，返回鞋坊深度检查
- 用镊子夹出眼镜碎片，证据升级为601

**证据作用**:
- 第一轮指证：证明Leonard到过现场
- 第三轮指证：证明Leonard用烛台击打Frank
- 眼镜碎片颜色将与第二轮证据604对应

**指证时机**: 循环6第一、三轮指证

---

**场景互动**:
- 可仔细检查烛台（发现凹痕）
- 可检查底部缝隙（发现眼镜碎片）
- 可对比Foster报告（确认凶器）

## 💡 设计亮点

### 1. 诱导招供系统的创新

```
传统指证：
  玩家选择证据 → 反驳谎言 → 下一轮

诱导招供系统：
  玩家选择话术 → 影响诱导槽 → 心理压迫
  → QTE演出 → 目标崩溃承认

设计理由：
  大火烧毁直接证据，只能靠心理战术
  体现Zack的智慧，不是暴力威胁
  让玩家体验"证据不足时的侦讯困境"
```
