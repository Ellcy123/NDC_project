## 修改方案1：进入Frank家的方式

### 基本信息

| 项目 | 内容 |
|:---|:---|
| **所属循环** | 循环2（L2） |
| **涉及场景** | 银行贵宾室 → Frank家 |
| **涉及NPC** | Leonard、Danny |
| **采用方案** | 方案A：Danny回家开门 |

---

### 原设计

```
┌─────────────────────────────────────────────────────────────┐
│  流程：                                                      │
│  1. 指证Leonard成功                                          │
│  2. Leonard被迫交出Frank家的钥匙                            │
│  3. Zack用钥匙进入Frank家                                   │
│                                                             │
│  问题：                                                      │
│  • NPC直接给钥匙，玩家被动接收                              │
│  • Leonard为什么会有Frank家的钥匙？逻辑略牵强               │
└─────────────────────────────────────────────────────────────┘
```

---

### 新设计

```
┌─────────────────────────────────────────────────────────────┐
│  流程：                                                      │
│  1. 指证Leonard成功                                          │
│  2. Leonard被迫提供Frank家的地址                            │
│  3. Zack前往Frank家（解锁新场景）                           │
│  4. 到达时Danny正好在家/刚回来                              │
│  5. Danny开门，允许Zack进入搜证                             │
└─────────────────────────────────────────────────────────────┘
```

---

### 证据变化

| 变化类型 | 原设计 | 新设计 |
|:---:|:---|:---|
| 获得物品 | 【Frank家钥匙】 | 无（不需要钥匙） |
| 获得信息 | - | 【Frank家地址】 |

---

### 场景解锁逻辑

```
【原逻辑】
指证Leonard → 获得钥匙 → 解锁Frank家

【新逻辑】
指证Leonard → 获得地址 → 前往Frank家 → Danny开门 → 进入搜证
```

---

### NPC对话设计

**Leonard（指证成功后）**:
```
Leonard（无奈）："好吧...Frank住在港口区的老工人公寓，
                 第三街42号，二楼左边那间..."
```

**Danny（Frank家门口）**:
```
场景：Zack敲门，Danny开门

Danny（疲惫）："你是...来调查叔叔的事的？"

Danny（让开门）："进来吧...我刚回来拿些东西。
                  这里太安静了，我一个人待不住..."

Danny（看着房间）："叔叔走了以后，这里什么都没变...
                    但又好像什么都变了。"
```

---

### 设计优点

1. **逻辑自然**：Danny住在Frank家，他有钥匙开门理所当然
2. **消除NPC给道具**：不再是"NPC交出钥匙"的被动设计
3. **增强Danny存在感**：Danny可以在场提供更多背景信息
4. **情感铺垫**：Danny的台词可以为后续他的债务问题做铺垫

### 分析玩法设计：碘蒸气指纹显影

**原理**：碘蒸气可使潜在指纹显现（1863年发明的技术）
        通过对比指纹密度，判断钞票经手人数

#### 阶段1：道具收集

| 道具 | 获取方式 |
|:---|:---|
| 【碘酒】 | 药店购买（花费$0.25） |
| 【酒精灯】 | Zack办公室已有 / 或向药店借用 |
| 【对照钞票】 | 向Tony买东西获得找零（流通过的旧钞） |

#### 阶段2：实验操作（互动小游戏）

```
场景：Zack办公室桌面
┌─────────────────────────────────────────────────────┐
│                                                      │
│   ┌────────┐        ┌────────┐                      │
│   │对照钞票│        │可疑钞票│   ← 拖拽到蒸气上方   │
│   └────────┘        └────────┘                      │
│                                                      │
│           ～～～蒸气～～～                           │
│               ∧∧∧                                  │
│            ┌──────┐                                 │
│            │碘酒皿│   ← 点击加热                    │
│            └──────┘                                 │
│          [酒精灯火焰]                               │
│                                                      │
└─────────────────────────────────────────────────────┘

操作步骤：
① 点击酒精灯 → 开始加热（出现进度条）
② 等待碘酒蒸发（进度条填满，约3秒）
③ 拖拽【对照钞票】到蒸气上方 → 指纹显现
④ 拖拽【可疑钞票】到蒸气上方 → 指纹显现
```

#### 阶段3：观察对比

```
实验结果呈现：
┌─────────────────────────────────────────────────────┐
│                                                      │
│  【对照钞票】              【可疑钞票】              │
│  ┌───────────┐            ┌───────────┐            │
│  │ ∿∿∿∿∿∿∿ │            │     ⊙     │            │
│  │ ∿∿∿∿∿∿∿ │            │     ⊙     │            │
│  │ 指纹密集  │            │ 指纹稀少  │            │
│  │ 层层叠加  │            │ 仅2-3枚   │            │
│  └───────────┘            └───────────┘            │
│                                                      │
│  Zack："这差别也太大了..."                         │
│                                                      │
└─────────────────────────────────────────────────────┘

Zack得出结论：
"正常流通的钞票经过无数人的手，指纹早就叠成一团...
 但这几张钱上只有少量清晰指纹。
 说明这些钱几乎没经过其他人的手——是刚从银行出来的！"
```

#### 完成奖励

- 证据升级：【可疑钞票】→【Vinnie的连号新钞】
- 解锁线索：确认钞票是全新的银行直出钞票
- 解锁对话：Tony对话选项 [询问钞票来源]

---

### 证据变化

| 阶段 | 证据状态 | 说明 |
|:---|:---|:---|
| 发现时 | 【可疑钞票】 | 未分析，只知道是钞票 |
| 初步观察后 | 【可疑钞票】 | 发现编号连续、看起来很新 |
| 碘蒸气分析后 | 【Vinnie的连号新钞】 | 确认是刚从银行出来的钱 |

---

### 玩家体验流程

```
Tony暗示 → 主动搜索座位 → 发现钞票 → 初步观察发现异常
→ 收集道具 → 碘蒸气实验 → 确认是新钞 → 解锁Tony对话选项
```

**核心改进**：
1. 玩家主动发现（不是NPC给）
2. 需要分析才能得出结论（不是直接告知）
3. 分析后才解锁新对话选项（有因果关系）

```
┌─────────────────────────────────────────────────────────────┐
```

### 玩家体验流程

```
搜索档案室 → 发现Moore私人档案柜 → 透过玻璃门看到Frank文件夹 → 柜子上锁无法取出
→ 指证Moore时质问私人档案柜 → 劝说Moore → Moore开柜取出遗嘱
```

**核心改进**：
1. 玩家先发现遗嘱存在（不是NPC告知）
2. 有"发现→障碍→突破"的节奏
3. 指证中获得新证据，增加戏剧性
4. Moore被迫配合有合理动机（避免法律麻烦）
