所有方案均遵循以下设计原则：

| **角色动机合理** | NPC行为符合其立场和性格 |

### 分析玩法设计：碘蒸气指纹显影

**原理**：碘蒸气可使潜在指纹显现（1863年发明的技术）
        通过对比指纹密度，判断钞票经手人数

#### 阶段1：道具收集

| 道具 | 获取方式 |
|:---|:---|
| 【碘酒】 | 药店购买（花费$0.25） |
| 【酒精灯】 | Zack办公室已有 / 或向药店借用 |
| 【对照钞票】 | 向Tony买东西获得找零（流通过的旧钞） |

#### 阶段2：实验操作（互动小游戏）

```
场景：Zack办公室桌面
┌─────────────────────────────────────────────────────┐
│                                                      │
│   ┌────────┐        ┌────────┐                      │
│   │对照钞票│        │可疑钞票│   ← 拖拽到蒸气上方   │
│   └────────┘        └────────┘                      │
│                                                      │
│           ～～～蒸气～～～                           │
│               ∧∧∧                                  │
│            ┌──────┐                                 │
│            │碘酒皿│   ← 点击加热                    │
│            └──────┘                                 │
│          [酒精灯火焰]                               │
│                                                      │
└─────────────────────────────────────────────────────┘

操作步骤：
① 点击酒精灯 → 开始加热（出现进度条）
② 等待碘酒蒸发（进度条填满，约3秒）
③ 拖拽【对照钞票】到蒸气上方 → 指纹显现
④ 拖拽【可疑钞票】到蒸气上方 → 指纹显现
```

#### 阶段3：观察对比

```
实验结果呈现：
┌─────────────────────────────────────────────────────┐
│                                                      │
│  【对照钞票】              【可疑钞票】              │
│  ┌───────────┐            ┌───────────┐            │
│  │ ∿∿∿∿∿∿∿ │            │     ⊙     │            │
│  │ ∿∿∿∿∿∿∿ │            │     ⊙     │            │
│  │ 指纹密集  │            │ 指纹稀少  │            │
│  │ 层层叠加  │            │ 仅2-3枚   │            │
│  └───────────┘            └───────────┘            │
│                                                      │
│  Zack："这差别也太大了..."                         │
│                                                      │
└─────────────────────────────────────────────────────┘

Zack得出结论：
"正常流通的钞票经过无数人的手，指纹早就叠成一团...
 但这几张钱上只有少量清晰指纹。
 说明这些钱几乎没经过其他人的手——是刚从银行出来的！"
```

#### 完成奖励

- 证据升级：【可疑钞票】→【Vinnie的连号新钞】
- 解锁线索：确认钞票是全新的银行直出钞票
- 解锁对话：Tony对话选项 [询问钞票来源]

## 修改方案9：烛台照片对比分析

### 基本信息

| 项目 | 内容 |
|:---|:---|
| **所属循环** | 循环1（拍照）→ 循环5-6（对比分析） |
| **涉及场景** | Margaret鞋坊一楼（火灾后） |
| **证据类型** | 照片对比 + 分析玩法 |
| **核心发现** | 烛台位置变化 → 烛台下方眼镜碎片 |

### 证据逻辑链

```
【发现链】
照片对比 → 烛台位置变化 → 有人动过现场 → 移开烛台 → 发现眼镜碎片

【推理链】
眼镜碎片在烛台下方
    ↓
碎片先掉落 → 然后烛台被移动盖住
    ↓
时间线：打斗时眼镜碎裂 → 凶手离开 → Vinnie移动烛台布置现场
    ↓
烛台位置变化 = 有人伪造/整理过现场

【指向凶手】
碎片有弧度 → 眼镜镜片
碎片发黄 → 使用多年的旧镜片
Frank不戴眼镜 → 碎片来自凶手
    ↓
配合证据612（Leonard眼镜一新一旧）
    ↓
现场碎片发黄 = Leonard右眼旧镜片颜色
    ↓
Leonard就是用烛台打死Frank的凶手
```

---

### 眼镜碎片证据描述

| 状态 | 描述 |
|:---|:---|
| **分析前** | 一块小玻璃碎片，从烛台下方的地板缝隙中取出。碎片很小，边缘锐利。 |
| **分析后** | 仔细观察后发现：碎片有明显的弧度，不是平板玻璃，而是有曲率的光学玻璃——这是眼镜镜片的特征。更重要的是，碎片的颜色有些发黄，说明这是一片使用多年的旧镜片（玻璃老化后会泛黄）。Frank不戴眼镜，这片碎片只能来自凶手。 |

---

### 与其他证据的配合

| 证据 | 关联 |
|:---|:---|
| **烛台凹痕** | 烛台是凶器，凹痕与Frank头骨伤口吻合 |
| **眼镜碎片** | 证明凶手戴眼镜，碎片在打斗时掉落 |
| **Leonard眼镜异常（612）** | 左眼透明（新）、右眼发黄（旧），证明最近换过一片 |
| **Margaret证词** | 看到戴眼镜的男人从鞋坊方向匆匆离开，走路踉跄像看不清路 |

---

### 设计亮点

| 要点 | 说明 |
|:---|:---|
| **照片对比玩法** | 利用循环1的照片作为"时间快照"，与当前现场对比发现异常 |
| **玩家主动发现** | 不是直接告诉玩家"烛台有问题"，而是让玩家通过对比自己发现 |
| **双重证据** | 一个发现揭示两件事：①有人伪造现场 ②凶手戴眼镜 |
| **颜色对比闭环** | 碎片发黄 ↔ Leonard右眼旧镜片发黄，形成完美对应 |
| **跨循环设计** | 循环1拍照 → 循环5-6对比，体现"过去的线索在未来派上用场" |
