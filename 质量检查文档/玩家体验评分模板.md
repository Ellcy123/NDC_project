# 玩家体验评分模板

## 📋 文档说明

- **目的**: 模拟玩家视角完整体验谜题，从三个维度评分
- **使用方式**: AI角色扮演玩家，走完整个谜题流程，最后打分
- **评分标准**: 每个维度1-5分，**每个维度不能低于3.5分**
- **版本**: v1.0
- **日期**: 2025-11-13

---

## 🎭 评分流程

### 第1步：AI角色扮演

**指令**:
```
你现在是一个第一次玩NDC游戏的玩家。
你刚进入这个谜题，完全不知道答案。
请从玩家视角完整体验这个谜题的全流程。
```

### 第2步：完整体验四个阶段

按照真实玩家的游戏流程，逐阶段体验：

---

## 📝 阶段1：初见谜题（第一印象）

### 体验内容

**谜题背景呈现**:
- 你看到了什么场景设定？
- 你知道要做什么吗？（指证谁？揭露什么？）

**第一反应**:
- 这个设定有意思吗？
- 是否有好奇心继续玩下去？
- 第一印象评分：？/5

### 记录要点

- [ ] 谜题设定是否清晰？
- [ ] 是否有吸引力？
- [ ] 是否期待后续？

---

## 📝 阶段2：收集证据（搜证过程）

### 体验内容

**搜证流程**:
- 你需要去哪些地方收集证据？
- 是否知道下一步该做什么？
- 是否遇到卡点？

**证据获取**:
- 证据获取是否顺畅？
- 是否需要提示？提示是否有效？
- 是否有"发现关键线索"的兴奋感？

**推理过程**:
- 是否能自己推理出关联？
- 还是需要游戏明示？
- 推理过程是否有成就感？

### 记录要点

- [ ] 搜证流程是否流畅？
- [ ] 是否有卡点？卡在哪里？
- [ ] 提示系统是否有效？
- [ ] 是否有"啊哈时刻"？

---

## 📝 阶段3：指证环节（高潮体验）

### 体验内容

**证据展示**:
- 展示证据的UI是否震撼？
- 是否有"戳穿谎言"的爽感？
- 逻辑链是否清晰？

**对话推进**:
- 嫌疑人的反应是否合理？
- 对话是否有张力？
- 是否有情感共鸣？

**认罪时刻**:
- 嫌疑人认罪是否有满足感？
- 真相揭露是否震撼？
- 是否有惊喜/反转？

### 记录要点

- [ ] 指证过程是否爽快？
- [ ] 逻辑链是否通顺？
- [ ] 是否有情感冲击？
- [ ] 嫌疑人认罪是否有满足感？

---

## 📝 阶段4：谜题结束（回味体验）

### 体验内容

**整体感受**:
- 这个谜题好玩吗？
- 会记住这个谜题吗？
- 会想分享给朋友吗？

**情感余韵**:
- 是否有情感共鸣？
- 是否被打动/震撼？
- 是否想继续玩下去？

**可能的质疑**:
- 有没有觉得"不合理"的地方？
- 有没有"为什么会这样"的疑问？
- 有没有逻辑漏洞让你困惑？

### 记录要点

- [ ] 整体是否满意？
- [ ] 是否有情感余韵？
- [ ] 是否有未解的疑问？

---

|------|------|--------|---------|
| **趣味性** | ？/5 | ≥ 3.5分 | ✅/❌ |

- ✅ **全部通过**: 三个维度都 ≥ 3.5分 → 谜题通过第5层检验

- ❌ **多项不通过**: 两个或以上维度 < 3.5分 → 建议重新设计

## 🔍 玩家疑问预判清单

在体验过程中，玩家可能会提出以下疑问，请逐一检查能否回答：

### 动机类疑问

- [ ] "为什么嫌疑人要这样做？" → 动机是否清晰
- [ ] "为什么会留下这个破绽？" → 是否有合理解释

### 逻辑类疑问

- [ ] "这合理吗？" → 逻辑是否自洽
- [ ] "怎么会这么巧？" → 是否过于巧合
- [ ] "证据哪来的？" → 获取途径是否合理

### 体验类疑问

- [ ] "接下来该做什么？" → 指引是否清晰
- [ ] "这有意思吗？" → 是否有趣味性
- [ ] "我为什么要在乎？" → 是否有情感驱动

**结果**:
- 所有问题都能回答 → ✅ 通过
- 有无法回答的问题 → ⚠️ 需要优化

## 🎯 评分参考案例

### 案例1：连号钞票谜题（高分参考）

**趣味性**: 5/5
- 有惊喜感：连号钞票的发现很精巧
- 有"啊哈时刻"：看到编号对比表时恍然大悟
- 有成就感：需要玩家自己发现规律
- 想分享：这个设计很巧妙，想告诉朋友

**合理性**: 5/5
- 逻辑链完整：受害者→Leonard→Vinnie，环环相扣
- 人物行为合理：Leonard不会想到钞票有编号
- 证据获取合理：酒吧老板提供钞票，途径清晰
- 无质疑点：身份契合（银行职员才能取连号钞票）

**情感共鸣**: 4/5
- 有正义感：为受害者揭露真相
- 有愤怒：对Leonard和Vinnie的勾结感到愤怒
- 有挫败感：Leonard威胁"拿不到内部证据"，有无力感
- 略有遗憾：情感冲击不如Rose和Frank的爱情故事强

**综合评价**: ✅ 全部通过（平均4.67分）

---

### 案例2：雨伞水渍谜题（低分参考）

**趣味性**: 2/5
- 无惊喜感：水渍证据很常规
- 无"啊哈时刻"：一眼就能看出来
- 无成就感：太简单了，没有推理乐趣

**合理性**: 1/5
- 人物智商下线：为什么擦了雨伞不擦水渍？
- 行为不合理：如果想掩盖，应该两个都擦
- 逻辑漏洞：角色显得很蠢

**情感共鸣**: 2/5
- 无情感冲击：只是个技术性证据
- 无共情：对角色无感

**综合评价**: ❌ 不通过（平均1.67分，远低于3.5及格线）

## 🔧 优化建议（如果不通过）

### 趣味性 < 3.5分 如何优化

**问题诊断**:
- 缺乏惊喜感
- 太老套/常规
- 无"啊哈时刻"

**优化方向**:
1. 增加视觉陷阱：表面证据vs隐藏真相
2. 增加反转设计：先以为A，其实是B
3. 增加玩家参与：需要玩家自己发现规律
4. 参考高分案例：镜子反射、连号钞票、窗帘褪色

---

### 合理性 < 3.5分 如何优化

**问题诊断**:
- 人物行为不合理
- 逻辑链有漏洞
- 证据获取过于巧合

**优化方向**:
1. 修正人物行为：给出合理的失误理由
2. 补充逻辑链：增加中间证据
3. 明确证据来源：合理的获取途径
4. 堵死辩解空间：增加证据独特性

---

### 情感共鸣 < 3.5分 如何优化

**问题诊断**:
- 情感平淡
- 无法共情角色
- 缺乏情感驱动

**优化方向**:
1. 增加受害者证词：让玩家看到伤害
2. 增加角色深度：背景故事、动机
3. 增加道德选择：正义vs无奈
4. 增加情感余韵：结尾留下思考

## 📌 使用说明

### 对于AI设计工具

在生成谜题后，自动运行本模板：
1. 角色扮演玩家
2. 完整体验四个阶段
3. 对三个维度打分（1-5分）
4. 检查是否每个维度 ≥ 3.5分
5. 输出体验报告

### 对于策划/编剧

1. 将谜题设计交给测试玩家
2. 让测试玩家按照本模板体验
3. 收集三个维度的评分
4. 如果有维度 < 3.5分，根据建议优化

## 🎯 及格线说明

### 为什么每个维度都是3.5分？

**3.5分代表**:
- 不追求完美（5分），但确保基本质量
- 允许有小瑕疵（4分），但不能平庸（3分）
- 每个维度都要及格，不能有明显短板

**对比**:
- **5分**: 非常优秀，必须用
- **4分**: 优秀，值得用
- **3.5分**: 及格，可以用
- **3分**: 平庸，需要优化
- **< 3分**: 不合格，不推荐

### 三个维度同等重要

不同于逻辑自查清单（有权重），本层评分中：
- 三个维度**同等重要**
- **每个维度都必须 ≥ 3.5分**
- 不能用高分维度弥补低分维度
- 原因：缺乏任何一个维度，玩家体验都会大打折扣
