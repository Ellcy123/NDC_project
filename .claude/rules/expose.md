# 指证规则

## 递进击穿结构
指证遵循逐层深入的击穿结构：
- 事实矛盾 → 物证逻辑 → 物理排除
- 每轮 Lie 用不同维度的证据反驳
- 同一维度的证据不跨轮拆分

## 谎言来源
- 第一个 Lie 必须对应 Talk 中已收集的证词（嫌疑人重申）
- 后续 Lie 是嫌疑人在 Expose 中被逼出的新谎言（止损式）
- 谎言由嫌疑人**主动说出**，不是 Zack 喂话后的被动否认

## 证据匹配
- 谎言措辞必须精确到只有对应维度的证据能反驳
- 陷阱证据不是随机干扰项，而是有明确的"看似相关但不构成反驳"的逻辑原因
- 玩家事后能理解为什么选错了

## 完整结构
```
Talk: 收集证词（get → TestimonyItem）
  → Zack 发现证词可疑 → 进入 Expose

Expose Round 1: Lie = Talk 中的证词（嫌疑人重申）
  → 玩家出示证据揭穿
  → 嫌疑人被迫承认部分事实，主动缩小范围（新谎言）

Expose Round 2: Lie = 嫌疑人的新狡辩
  → 玩家出示新证据揭穿 → 依此递进
```
