# 证据规则

## ID 编码
- EPI01 = 1xxx（loop1=11xx, loop2=12xx, ..., 派生=17xx）
- EPI02 = 2xxx（loop1=21xx, loop2=22xx, ..., 派生=27xx）
- EPI03 = 3xxx（loop1=31xx, loop2=32xx, ..., 派生=37xx）

## 物理属性
- 所有证据的物理属性（尺寸、时间、距离、品牌、价格）必须前后一致
- 跨循环引用时不能出现数值矛盾

## 获取链
- 直接获取：对话 `script:"get"` 或场景物品点击
- 分析派生：对直接证据执行"分析"
- 合成派生：两个证据组合

## note 字段
- `关键——{用途}`：Expose 关键证据
- `场景道具`：环境性证据
- 空：普通证据

## 核心原则
- 单一证据不定案：每个关键结论至少需要两种不同来源的信息交叉验证
- 证据获取时间不能早于其存在时间
- 证词原文与提取摘要信息密度对等，不丢失关键限定词
- 疑点解锁需要至少两种不同来源的信息交叉验证
