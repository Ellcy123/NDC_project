# 对白演出节奏规则

适用于 NDC 项目所有 Unit 的对白草稿（`AVG/对话配置工作及草稿/Unit{N}/Loop{N}_生成草稿.md`）及落地 JSON（`AVG/EPI{NN}/Talk/`、`AVG/EPI{NN}/Expose/`）。

写对白和审对白时按以下规则检查。

---

## 规则一：群体对话中 NPC 入场/退场的表演

### 入场规则

当对话已经开始、玩家与某些 NPC 已在场，**新 NPC 物理走进场景**时：

- **首句必须极短、抓注意力**——起"开场击响"作用
- 合格范式："等等！" / "我来了" / "让一让。" / "我下来了。"
- 不合格范式：长句陈述、温吞的招呼、复杂的情绪铺垫

**首句之后**才能进入正常对白节奏。如果原本的入场台词较长，应拆成两个节点：首句承担"入场击响"，第二句承担"情境承接"。

### 退场规则

NPC 在对话**中途离开场景**时：

- **最后一句必须明确告别**——让玩家清楚 NPC 离场了
- 合格范式："我先走了。" / "我们待会见。" / "我去厨房。" / "走吧。"
- 不合格范式：嘟囔无关的话、用动作描述代替告别语、说"您慢走"等送客语（这是请别人走，不是 NPC 自己退场）

### 不算入场/退场的情况（无需套用规则）

- 玩家与 NPC 一起从前一场景走过来（同步移动，不算入场）
- 对话开始时 NPC 就在场（即使一开始没说话）
- NPC 是场景核心驻场（Talk 对话的主体 NPC）
- 玩家方（Zack/Emma）离开场景——以玩家方动作处理，不套此规则

---

## 规则二：Expose 击穿首句必须是 Zack + 力道感

### 触发时机

Expose 中，玩家出示**正确证据击穿某轮谎言**之后的下一个对话节点（即"反驳段第一句"）。

### 硬性约束

1. **说话人必须是 Zack**——不是 NPC、不是 Emma、不是嫌疑人
2. **质感必须有"力道感"**——直击谎言要害的具体性

### "力道感"的合格风格（任选一种或组合）

| 风格 | 示例 |
|---|---|
| 直接陈述证据关键属性 | "枪管里没有硝烟残留。无黑化痕迹。" |
| 命令式 + 证据具体展开 | "那你来解释解释这个。你的账本上写着四千，Webb 的私账上却是一万五千。" |
| 反讽 NPC 谎言论点 | "绝对的不在场证明？你是不是把这个借口在心里默念了一百遍……" |
| 反问戳穿心理/动机 | "你真的以为，今晚只有你一个人想让 Webb 死吗？" |
| 冷讽 + 暗示证据将打脸 | "古董？那可真不凑巧。" |
| 反讽 + 命令戳穿身份 | "你不是目击者。这是写给你的台词。" |
| 命令式 + 证据指向反驳 | "看看这封 James 给 Whale 的回信——他接的，就是杀 Webb 的委托。" |

### 不合格的风格（必须改）

| 反例 | 问题 |
|---|---|
| "看清楚。" | 抽象命令，没说证据 |
| "撒谎。" | 抽象否定，没说证据 |
| "看看这个。" | 命令但没解释证据是什么 |
| "这是弹壳的拓印。" / "这是 James 写给 Whale 的回信。" | 只点名证据，没展开反驳作用，没戳谎言 |

### 核心判据

> 第一句要么戳到谎言要害（反讽/反问 NPC 的具体论点），要么直接展开证据的反驳属性。
> "只是点名一件东西放在桌上" 是不够的。

### 应用范围

Expose 文件（`AVG/EPI{NN}/Expose/Loop{N}_{npc}.json`）中所有 `Lie` 节点之后的反驳段第一句。

## 配套审查清单（写完一段对白后自查）

- [ ] 群体对话中是否有 NPC 中途入场？入场首句够不够简短？
- [ ] 群体对话中是否有 NPC 中途退场？末句有没有明确告别？
- [ ] Expose 每轮击穿后第一句是不是 Zack？
- [ ] Expose 每轮击穿首句是不是"直击谎言要害的具体陈述/反讽"，而不是模糊命令或孤立点名？
