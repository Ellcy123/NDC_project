## 💬 NPC对话系统设计

### Tommy对话树结构

**主对话选项**：

- "昨晚你在做什么？"
- "有听到什么异常声响吗？" → 获得枪声证词
- "你对Rosa了解多少？"
- "关于Webb先生的事..."

**对话状态管理**：

- 初次对话：Tommy较为谨慎，只给基本信息
- 获得部分证据后：可以追问更多细节
- 循环1结束后：为循环2做铺垫

### 对话与证据的关联

**证词作为证据**：Tommy的证词会自动加入证据清单
**对话解锁**：某些深入问题需要先获得相关物证才能询问

## 🎯 指证总览

**总时长**：54秒
**核心逻辑**：三次证据否定，层层击破谎言
**设计理念**：专业侦探 vs 恐惧母亲的心理博弈

---

## ⚔️ 第一阶段：否定地点谎言

### Rosa的初始谎言

**Rosa信息来源**：DAT003
*"于是我一直在地下室酒窖清理，整理酒瓶和架子"*

### 指证发起

**Zack反驳**（取出工作记录卡，平静但严厉）：

```
Zack："Rosa，这是你的工作安排表。上面明确写着你昨晚23:00到01:00
在后台走廊清洁，不是地下室酒窖。你为什么要撒谎？"
```

### Rosa的慌乱反应

**Rosa的防御**（声音颤抖，开始搓手）：

```
Rosa："哦...哦天哪，我...我可能记错了...最近太累了，总是搞混工作安排...
对，我是在后台清洁..."
```

### 阶段结果

✅ **地点谎言被戳穿**

- Rosa被迫修正说法，承认在后台走廊工作
- Rosa开始出现紧张情绪，为下一轮指证做铺垫

---

## ⚔️ 第二阶段：否定目击谎言 (⏱️ 16秒)

### Rosa的修正说法

**Rosa新版本**：

```
Rosa："我在后台走廊工作，但我很专心清洁地板和墙壁，
那里很安静，什么异常都没发生..."
```

### 指证发起

**Zack追击**（严肃地展示毛巾+氯仿瓶）：

```
Zack："专心清洁？Rosa，这条毛巾在你的工作区域被发现，上面有氯仿残留。
而这个氯仿瓶是在后台走廊的垃圾桶内发现的，你说当时你在后台清洁，
你工作区域的毛巾沾染了案发现场附近的氯仿，氯仿是医用麻醉剂，不是清洁用品。
如果你什么都没看到，怎么解释这个？"
```

### Rosa心理防线崩溃

**Rosa的情绪爆发**（长时间沉默，看着毛巾，眼中涌出泪水）：

```
Rosa："我...我..."
```

**（抱头哭泣）**

```
Rosa："好吧！是我！是我用这条毛巾迷昏了您！我儿子Miguel生病了，
需要手术费，我实在没办法了...求求您别让我坐牢，Miguel还需要我照顾..."
```

### 阶段结果

✅ **目击谎言被彻底击破**

- Rosa承认使用氯仿迷晕Zack
- 揭露母爱动机：为生病儿子Miguel筹集手术费
- Rosa进入自我保护模式，编造自己是主犯的故事

---

## ⚔️ 第三阶段：否定自认，揭露真相 (⏱️ 16秒)

### Rosa的自认说法

**Rosa的"坦白"**：

```
Rosa："我等您路过走廊时，从背后用毛巾捂住您的嘴和鼻子...
您昏倒后我把您拖到Webb先生的办公室..."
```

### 指证发起

**Zack专业分析**（温和但坚定，展示照片）：

```
Zack："Rosa，我相信你爱你的儿子，但你不是凶手。
看这些拖拽痕迹——需要150磅的力量才能造成2.5厘米的压痕。
你体重不到120磅，根本做不到。谁威胁你这样说的？"
```

### Rosa最终崩溃供认

**真相的揭露**（声音几乎是耳语，充满恐惧）：

```
Rosa："是...是Morrison警官..."
```

**（抱头痛哭）**

```
Rosa："他...他说如果我不配合，就让我失去工作...
他知道我儿子生病的事，知道我需要这份工作..."

Rosa："我不知道他要做什么！我只是按他说的，
如果有人问起，就说什么都没看到...如果实在瞒不住，就说是我做的..."
```

**（看向Zack，眼中充满恐惧）**

```
Rosa："求求您别让Morrison知道我说了...我真的不知道他具体做了什么...我发誓！"
```

### 阶段结果

✅ **自认谎言被彻底否定**

- **真正幕后黑手揭露**：Morrison警官
- Rosa承认被威胁配合栽赃
- 为循环2的Morrison调查做完美铺垫
