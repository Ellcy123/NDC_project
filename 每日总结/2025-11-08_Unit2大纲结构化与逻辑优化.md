### 3. 大纲结构化重构

**创建文件**：[unit2大纲-结构化版本.md](../第二章内容/unit2大纲-结构化版本.md)

**新结构设计**：每个循环分为四个标准部分
```
开篇 → 搜证（场景搜证 + 人物对话）→ 指证 → 结尾
```

#### 关键术语调整
- ❌ "任务对话" → ✅ "人物对话"
- ❌ "玩家内心OS" → ✅ "玩家的体验心路历程"

#### 突发事件控制
- **原则**：每循环最多1个突发事件
- **实际使用**：
  - 循环3：Mickey提供银行内部记录
  - 循环5：Mickey突然出现救出Margaret

**Markdown格式优化**：

**补充完整的"玩家的体验心路历程"**（所有6个循环）：

#### 循环1：情绪过山车
- 从以为母亲死亡的绝望到发现是误导的愤怒
- 对Morrison的不信任和无奈
- 悬念驱动：母亲在哪？
- 对Frank案件的初步困惑

#### 循环2：负面印象建立
- 对Frank形成负面刻板印象：赌博欠债、被女人骗
- 对银行-黑手党勾结感到不安
- 侦探成就感：通过时间线规律表推理
- 挫败感：Leonard威胁"拿不到令状"
- 对母亲安危的持续焦虑

#### 循环3：第一重叙事反转
- **第一重叙事反转的震撼**：Frank从"赌博老赖"翻转为"对抗不义的英雄"
- 对自己之前偏见的羞愧和反思
- 对Mickey的感激和好奇：为什么帮我？为什么有内部资料？
- Danny的反应引发新怀疑

#### 循环4：第二重情感反转
- **第二重叙事反转的情感冲击**：Frank不仅是英雄，更是为爱勇敢的老人
- 被Rose和Frank的黄昏之爱深深打动
- 对社会偏见的愤怒：为什么真挚的爱要藏在暗处？
- 对Danny的种族主义言论感到厌恶
- Frank完整形象确立：为爱为义而战的悲剧英雄

#### 循环5：情感高潮与悬念升级
- 保险箱内容的震撼：Frank的完整举报材料
- 尸检反转的惊讶：Frank是被钝器杀害，不是火灾！
- Vinnie认罪的"不对劲感"：太快了，太顺利了
- **情感高潮**：Mickey救出母亲！感激、激动、终于松了口气
- Vinnie的愤怒咆哮引发新疑问："Mickey背叛了约定？出卖了Leo？"
- 对Mickey的完全信任（为Unit 3的背叛埋下伏笔）
- 悬念升级：Leo是谁？真凶是谁？

#### 循环6：第三重反转与主题升华
- Margaret证词带来的真相揭示
- 4轮递进指证的侦探高潮：每一轮都在剥开Leonard的谎言外壳
- **第三重情感反转**：Leonard和Vinnie的禁忌之爱关系揭露，震撼而悲凉
- 对Leonard失手杀人的复杂情感：他不是冷血杀手，只是被逼到绝境
- 对Vinnie为爱承担一切的动容：从"黑手党恶棍"到"悲剧情人"的形象逆转
- **两种爱的镜像对比**带来的深刻共鸣：
  - Frank+Rose：真挚的爱被社会偏见压迫，最终爱人惨死
  - Vinnie+Leonard：扭曲的单恋在黑暗中挣扎，最终酿成悲剧
- **系统性批判的觉醒**：真正的罪犯（Moore银行、Miller集团）安然无恙，底层人互相伤害
- 黑色电影美学的苦涩余味：正义得到伸张，但没有赢家，只有悲剧
- 对Mickey角色的模糊感：他帮了很多，但Vinnie的指控"背叛约定"是什么意思？

## 🎨 设计亮点

### 1. 三重叙事反转结构
```
循环1-2: 负面印象建立（Frank = 赌博老赖）
    ↓
循环3: 第一重反转（Frank = 正义斗士）
    ↓
循环4: 第二重反转（Frank = 为爱勇敢的老人）
    ↓
循环6: 第三重反转（Leonard-Vinnie禁忌之爱悲剧）
```

### 2. 两种爱的悲剧镜像
- **Frank + Rose**：真挚的黄昏之爱，被社会偏见压迫
- **Vinnie + Leonard**：扭曲的单恋，在黑暗中挣扎

### 3. 玩家情感曲线设计
```
循环1: 绝望 → 愤怒 → 悬念
循环2: 偏见建立 → 挫败感
循环3: 羞愧 → 反转震撼 → 好奇
循环4: 感动 → 愤怒（社会偏见）→ 正义感
循环5: 震撼 → 怀疑 → 情感高潮（母亲获救）
循环6: 真相揭示 → 复杂情感 → 主题共鸣 → 苦涩余味
```

### 4. 结构化设计规范
- **标准循环结构**：开篇 → 搜证 → 指证 → 结尾
- **搜证细分**：场景搜证 + 人物对话
- **突发事件**：每循环最多1个，用于关键转折
- **指证递进**：循环2（3轮）、循环6（4轮）
- **情感递进曲线**：自信 → 动摇 → 慌张 → 崩溃 → 认罪
