# Episode 1 逻辑问题修改方案汇总

## 问题1：Emma的"10分钟"质疑

**问题描述**：
Emma质疑Morrison"从案发到现场不到10分钟"，但根据时间线，Morrison实际00:30离家，01:00到达，与"10分钟"矛盾。

**修改方案**：
删除时间质疑，改为质疑Morrison直接认定凶手的行为，不先调查现场的异常做法。

**具体修改内容**：

### 修改位置1：第25行对话
**原文**：
```
Emma："Morrison警探，您来得太及时了。从案发到现场不到10分钟，除非您早知道会发生什么？"
```

**改为**：
```
Emma："Morrison警探，您来得真及时。不过我很好奇，为什么不先调查现场，而是直接认定Brennan是凶手？"
```

### 修改位置2：第108行对话
**原文**：
```
Emma（不退缩）：
"我看到您破门的时间。从警局到这里至少要十二分钟。您到得太快了。"
```

**改为**：
```
Emma（不退缩）：
"我有证据，警探。不过更重要的是——一个称职的警探不会在看到现场后的第一反应就是'逮捕他'，而不是'封锁现场、寻找线索'。"
```

## 问题3：Jimmy声音模仿改为简单回应

**问题描述**：

**修改方案（采用方案B）**：
1. 将Jimmy的话改为简单的"走开"或"滚开"
2. 补充Vivian证词：音乐太大，她听不清楚具体是"滚"还是"走开"
3. 所有相关描述从"模仿Webb声音"改为"简单回应"

**具体修改内容**：

### 修改位置1：第4803行 - Vivian第一次描述

```
Vivian：
"没有...门锁着...他说'今晚不要再来了'...我就走了..."
```

**改为**：
```
Vivian：
"没有...门锁着...他说'走开'...我就走了..."
```

### 修改位置2：第4913行 - Vivian详细回忆
**原文**：
```
Vivian：
"门锁着...我敲门...然后听到Webb的声音...'今晚不要再来了'...就这句话..."

```

"门锁着...我敲门...然后听到里面有声音...好像是'滚开'...或者'走开'...我也不确定...当时舞台的音乐太大了，我听得不太清楚..."
