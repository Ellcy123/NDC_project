Zack独自坐在办公桌前，面前摊开所有证据。

#### 重审证据1：楼顶酒瓶

**🏷️ 证据309：楼顶空酒瓶（重新分析）**

```
┌─────────────────────────────────────┐
│  证据重审：楼顶酒瓶                 │
├─────────────────────────────────────┤
│                                     │
│  原发现：楼顶有一个空酒瓶           │
│                                     │
│  新发现（Foster尸检补充报告）：     │
│  • 酒瓶标签：高档威士忌             │
│  • Thomas家只有廉价酒               │
│  • 酒瓶是Helen的                    │
│                                     │
│  Thomas血液酒精浓度：0.25%          │
│  → 严重醉酒状态                     │
│                                     │
│  ⚠️ 疑点：                          │
│  Thomas喝的是Helen的酒              │
│  Helen在案发前灌醉了Thomas？        │
```

**Zack内心独白**：
> "Thomas的血液酒精浓度是0.25%...这是严重醉酒。一个严重醉酒的人，怎么能'要推Mary下楼'？他当时可能连站都站不稳。"
>
> "而且这酒瓶...是Helen的高档威士忌，不是Thomas家的廉价酒。是Helen给Thomas喝的？"

## 💡 设计要点

### 为什么没有NPC指证？

**设计理念**：
- 循环5是Zack的内心推理，不需要外部对抗
- 玩家和Zack一起重新审视证据，发现真相
- 制造"胜利后的反转"——原来事情没那么简单

### 推理回顾的作用

1. **信息整合**：将前4个循环的证据串联
2. **真相揭示**：让玩家明白这不是正当防卫
3. **道德铺垫**：为循环6的选择做铺垫

### 与其他循环的关系

| 循环 | 玩家认知 |
|:---:|:---|
| 循环1-4 | Mary是正当防卫，正义得到伸张 |
| **循环5** | 原来是预谋杀人...但Mary确实没有其他选择 |
| 循环6 | Zack会如何选择？ |
