# AI美术面试测试题 — NPC角色立绘制作

## 测试说明

本测试用于评估候选人使用AI美术工具（Midjourney / Stable Diffusion / ComfyUI 等）进行游戏角色立绘制作的能力。请在 **3小时** 内完成以下全部交付物。

**评估维度：**
- 角色理解力：能否从文字设定中提炼视觉特征
- 风格还原力：能否匹配已有角色的美术风格
- 工具熟练度：AI出图效率、prompt工程、后期修图能力
- 一致性控制：同一角色多张图的面部/服装/气质一致性
- 审美判断力：构图、光影、色彩的综合审美水平

---

## 项目背景

**游戏类型：** 2D侦探推理AVG（视觉小说）
**时代背景：** 1925年，美国芝加哥，禁酒令时期
**美术风格：** 半写实绘画风，偏欧美graphic novel质感，色调偏暗沉/去饱和，带有noir（黑色电影）氛围
**引擎：** Unity 2D，角色以立绘形式呈现于对话界面

### 现有角色风格参考

以下为已完成的同项目角色立绘，**你的产出必须在风格上与这些角色保持统一**：

| 参考角色 | 文件 | 特征说明 |
|---------|------|---------|
| Morrison（腐败警探） | `参考图/morrison_big.png` | 中年男性，棕色警服外套，严肃凶狠，室内暖光 |
| Tommy（俱乐部经理） | `参考图/tommy_big.png` | 瘦削男性，白衬衫+酒红领结，精明世故，奢华内景 |
| Jimmy（酒保） | `参考图/jimmy_big.png` | 年轻男性，白衬衫+围裙，疲惫忧郁，朴素背景 |

**风格关键词提取（供参考）：**
- 笔触：油画质感，可见笔触肌理，非照片级写实
- 光影：强对比，单侧主光源，noir风格明暗
- 色调：低饱和度，以棕/灰/橄榄绿为主色系
- 构图：胸部以上半身像，3/4侧面角度为主
- 背景：简化的场景暗示（模糊处理），不喧宾夺主
- 面部：五官写实但略带风格化，线条感强

## 角色设定：Danny Kowalski

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | Danny Kowalski |
| 年龄 | 35岁（1890年生） |
| 种族 | 波兰裔白人 |
| 职业 | 工厂工人（消极怠工，混日子） |
| 居住 | 寄住在叔叔Frank家中（芝加哥南区工人社区） |
| 身份 | Frank的侄子，从乡下投奔叔叔，是Frank唯一的亲人 |
| 叙事角色 | 次要反派——因遗产纠纷诬陷他人，被玩家指证揭穿 |

### 外貌特征（必须体现）

| 维度 | 描述 | 设计意图 |
|------|------|---------|
| 体型 | 中等身材，略显臃肿，肚子微凸 | 好吃懒做、缺乏锻炼的生活方式 |
| 面容 | 35岁但看起来更老，面色不佳，眼袋明显 | 不良嗜好+懒散生活的痕迹 |
| 眼神 | 飘忽不定，带有急躁和贪婪感 | 核心性格外化——寄生虫式的不安与觊觎 |
| 发型 | 棕色短发，油腻凌乱，未经打理 | 邋遢、不修边幅 |
| 面部特征 | 下颌略宽，法令纹深，嘴角习惯性下撇 | 常年不满和怨气的积累 |
| 肤色 | 偏白但不健康，略显蜡黄 | 工厂工人但不爱运动的气色 |

### 服装设计（必须体现）

| 部件 | 描述 | 设计意图 |
|------|------|---------|
| 上衣 | 灰蓝色或土黄色工人衬衫，袖口卷起但歪斜不齐 | 工厂工人身份，但邋遢随意 |
| 外搭 | 深色旧背心或旧马甲，扣子缺一个 | 不体面但勉强"穿了件外套"的感觉 |
| 裤子 | 深色工装裤，膝盖处有磨损（如可见） | 底层劳动者 |
| 整体感 | 衣服皱巴巴，领口松垮，像是随手套上的 | 与Tommy的精致、Morrison的整肃形成对比 |

### 气质关键词

**必须传达的核心气质（按优先级）：**

1. **懒散寄生** — 整个人散发出"得过且过"的气息，姿态松垮
2. **贪婪不甘** — 眼神中有藏不住的觊觎和不满
3. **底层粗鄙** — 不是恶人的精明，是小人物的粗糙和短视
4. **急躁冲动** — 随时可能爆发的焦躁感

**不应有的气质：**
- 不要帅气/英俊（他不是魅力型反派）
- 不要精明/城府深（他不聪明，只是贪婪）
- 不要可怜兮兮（他可悲但不值得同情的外表）
- 不要过于肮脏/流浪汉化（他有工作，只是邋遢）

## 交付物要求

### 交付物 A：角色立绘正图（必做）

**数量：** 1张定稿
**规格：**
- 尺寸：1024×1024px 或以上
- 构图：胸部以上半身像，3/4侧面（与参考角色一致）
- 背景：简化的工人社区室内暗示（砖墙/旧家具/昏暗灯光）
- 光影：单侧暖光（油灯/白炽灯感），noir风格强对比

**质量标准：**
- 与三张参考角色放在一起，风格不违和
- 面部细节清晰，五官可辨认
- 服装材质可信（棉麻布料质感）
- 无明显AI瑕疵（多余手指、融合伪影、文字乱码等）

### 交付物 B：表情差分（必做）

基于交付物A的面部，制作 **3个表情变体**，要求面部以外的部分（服装、姿态、背景）保持一致：

| 编号 | 表情 | 情境描述 | 关键表现 |
|------|------|---------|---------|
| B1 | 默认/日常 | 平时懒散的状态 | 眼神飘忽，嘴角微撇，漫不经心 |
| B2 | 愤怒 | 听到遗嘱内容，爆发 | 瞪眼，龇牙，面部肌肉紧绷，青筋 |
| B3 | 崩溃/不甘 | 被指证后情绪崩溃 | 眼眶泛红，表情扭曲，嘴角颤抖，不甘心 |

**规格：** 每张512×512px 或以上，仅面部特写即可

### 交付物 C：设计过程说明（必做）

提交一份简短的文字说明（300-500字），包含：

1. **工具链**：使用了哪些AI工具和后期软件
2. **关键Prompt**：生成正图时使用的核心prompt（可脱敏/简化）
3. **迭代过程**：简述从初稿到定稿的调整思路（附2-3张过程图）
4. **一致性方案**：如何保证表情差分与正图的角色一致性
5. **风格匹配方法**：如何分析参考图并还原风格

## 评分标准

| 维度 | 权重 | 评分要点 |
|------|------|---------|
| 风格一致性 | 30% | 与参考角色放在一起是否像"同一个游戏的角色" |
| 角色表现力 | 25% | 是否一眼能感受到"懒散、贪婪、粗鄙"的气质 |
| 技术质量 | 20% | 无AI瑕疵、细节清晰、光影合理、材质可信 |
| 表情差分 | 15% | 三个表情是否到位、面部一致性是否良好 |
| 过程说明 | 10% | 工作流是否清晰专业、是否展现了可复制的方法论 |

### 加分项（非必需）
- 提供额外的全身像或动态姿势变体
- 表情差分超过3个（如增加"诬陷时指着别人"的姿态）
- 提供可编辑的PSD/分层文件
- 展示ControlNet/IP-Adapter等高级一致性技术的使用

### 扣分项
- 与参考角色风格明显不匹配（如日系二次元风、照片级写实等）
- 角色气质与设定严重偏离（如画成了帅气青年或凶狠恶棍）
- 表情差分中角色面部特征不一致（换了一张脸）
- AI伪影未清理（多余手指、文字噪点、面部融合等）
- 未提供过程说明
