文档性质：全游戏开篇引子，置于 Unit 1 之前。无玩家操作。约 55 秒。
基于：Unit5 大纲 0515 版。
版本：v2 无枪版（修复 v1 与 Unit5 杀人手法的逻辑冲突——枪声会让 Pierce 全程在走廊、Foster 验尸找不到弹道、所有宾客必到现场等连锁矛盾）。
格式：导演镜头表。结构：闪前 → 城市大景 + 画外音桥接 → 车道吵架 → 书房 → 黑屏。

# 序幕

## 第一段：闪前（5 秒）

### 镜头 1 — EXT. 庄园前廊 — 雪夜 / 5s
特写。一只穿浅色丝袜的脚，踩出庄园门外。
镜头慢慢摇起。Emma 一个人走出庄园大门，下台阶，进雪里。
大衣前襟、袖口、手——**大片暗色污渍**（远景像血，色调偏深褐）。脸侧也有一道，看不出是从哪里来的。
她不回头。**步子比正常人慢半拍**。雪还在下。
（最后一秒，老 Charles 的声音从画外起。）
**老CHARLES（画外）** 每一块砖底下——

> 〔反转伏笔〕"污渍"远景像血，实际是威士忌+鼻血+父亲轻伤血的混合。"步子慢半拍"是镇静剂副作用初现。L4 反向指证后玩家回看此镜恍然。

---

## 第二段：城市 + 基调（15 秒）

### 镜头 2 — EXT. 芝加哥街道 — 雪夜 / 2s
俯拍。一条无人的街，雪刚开始下。一盏路灯单独亮着。

### 镜头 3 — EXT. 工厂区 / 2s
仰拍。烟囱黑的，不出烟。一列火车从画面深处缓慢驶过。

### 镜头 4 — EXT. 联排公寓 / 2s
平视。一整面窗，灯一盏一盏熄掉。

### 镜头 5 — EXT. 山坡远景 / 3s
全城在雪里。山坡上一栋白色庄园——整座城市里唯一全亮的窗。

### 镜头 6 — INT. 书房 — 特写 / 3s
老的、有斑的手，把威士忌倒进玻璃杯。壁炉的火光在液体里晃了一下。
**老CHARLES（画外）** ——都有一个不愿意死的人。

### 镜头 7 — INT. 书房 — 中景 / 3s
他坐进高背椅。慢慢把杯子放下。桌角的小录音机，红灯亮着。
**老CHARLES（画外）** 正直的人，在这个城市活不长。

---

## 第三段：车道（13 秒）

### 镜头 8 — EXT. 庄园车道 / 3s
Emma 在砾石路上走。Zack 从画面外冲进来，从后面抓住她胳膊。

### 镜头 9 — EXT. 车道 — 中景 / 5s
Emma 转身。**猛地推开他**。退一步。**手在抖**。眼睛红的，喘。
**EMMA**（沙哑）你别管。
**ZACK**（气息）Emma——

> 〔反转伏笔〕"手在抖"是镇静剂副作用之二。玩家以为是情绪激动，L4 揭穿后才知道。

### 镜头 10 — EXT. 车道 — 门廊 / 4s
Zack 没动。Emma 退上台阶，眼睛不离开他。雪落两人的肩。

---

## 第四段：书房（15 秒）

### 镜头 11 — INT. 书房 / 3s
门被推开。Emma 进来——**手里举着的，是一台录音机。红灯亮着**。Charles 在椅子里没回头。桌上两杯酒、桌角的另一台录音机也在转。
**老CHARLES** 坐。

> 〔关键改动〕Emma 带的是录音机，不是枪。她是记者，她的武器是真相。两台录音机同框：Emma 的采访录音机 + 老 Charles 自己的炫耀录音机。

### 镜头 12 — INT. 书房 / 3s
Emma 不坐。**她把自己的录音机放在 Charles 对面的桌沿，按下录制键**。Charles 端起酒杯喝一口，看着她。
**老CHARLES** 你的报道——明早印不出来。

> 〔关键改动 + Loop6 物证源头〕Emma 在序幕里**亲手按下的录制键**——这台录音机后来录下了她推父亲、父亲呻吟、Emma 离开、然后小 Charles 进来"父亲，我来了"+ 扭打 + 闷响的完整音轨。**Loop6 调取的录音机 = 序幕里这一台**。玩家在序幕里亲眼看见它开始录。

### 镜头 13 — INT. 书房 — Charles 特写 / 4s
他放下杯子。看着她，像看着一件家具。
**老CHARLES** 你不是第一个站在那儿的。那就动手吧。

### 镜头 14 — INT. 书房 — Emma 特写 / 3s
她在喘。粗、急、不规则。眼睛红的。**瞳孔放大异常**。镜头不在她手上，只在她的脸。
**鼻尖一抹深色——可能是阴影，可能不是**。

> 〔反转伏笔〕"瞳孔放大异常"=药物性瞳孔扩散；"鼻尖一抹深色"=鼻血初渗。镇静剂副作用第三、四征兆。玩家以为是情绪表现，L4 揭穿后回想"她那时已经被下药了"。

### 镜头 15 — Zack 特写 / 2s
极近特写。Zack 的脸。背景虚到看不出地点。
**ZACK**（很轻）Emma——你会杀了他吗？

> 〔关键点〕问句而非陈述。L4 揭穿后回响——"Zack 这一问就是答案：他相信她不会，所以才问"。

---

## 第五段：黑屏 + 标题（10 秒）

### 镜头 16 — 黑屏 / 5s
画面全黑。声音按顺序：
1. **一声响——玻璃碎裂**。
2. 一个**很轻的"咚"**——像东西撞回椅子里，**不是人摔下的重音**。
3. **一声老人呻吟，断在喉咙里**。
4. （短停。）
5. 远处警笛——并不知道为什么响，但响着。

> 〔关键改动〕去掉枪声。
> - 玻璃碎 = Emma 推父亲时撞翻的威士忌瓶
> - 轻"咚" = 老 Charles 撞桌后坐回椅子（不是死亡重摔）
> - **老人呻吟** = **他还活着**！这是 Lula 21:58 听见的同一声呻吟
> - 警笛 = 城市远处任何事，脱钩"凶案警笛"暗示
>
> 玩家第一次听以为是"她杀了他+警察来"；L6 录音播放时这段声音**原模原样在录音里**出现——录音机录下了一切，包括那声"他还活着的呻吟"。

### 镜头 17 — 标题 / 5s
继续黑屏。

**NDC**
**无辜之城**

# 调和后的双重读法

## 玩家第一次看（Unit1 开场前）

- 满身"血"走出庄园
- 在车道对 Zack 失控
- 持物（远景像武器）进书房
- 老 Charles 让她"动手"
- 玻璃碎 + 重物 + 呻吟 + 警笛 = **她杀了他**

误导效果 100% 保留。

## L4 反向指证后 + L6 录音播放后

| 第一次看 | 真相 |
|---|---|
| 大片"血"污渍 | 威士忌+鼻血+父亲轻伤血的混合 |
| 步子慢半拍 / 手抖 / 瞳孔放大 / 鼻尖深色 | 镇静剂副作用全套 |
| 手里"武器" | 录音机 |
| 桌沿按下录制键 | **Loop6 关键物证的源头**——玩家亲眼见到它开始录 |
| Zack "你会杀了他吗" | 问句 = 他相信她不会 |
| 玻璃碎 | Emma 撞翻威士忌瓶 |
| 轻"咚" | 老 Charles 撞桌后坐回椅子，不是死亡 |
| 老人呻吟 | **他活着！** Lula 21:58 听见的就是这一声 |
| 警笛 | 城市远处任何事 |

## Loop 6 录音播放时的极致冲击

玩家 L6 听录音段二："Emma 推父亲 → 父亲倒地呻吟 → Emma 离开"——**这就是序幕镜头 16 的声音原型**。瞬间意识到：
- 序幕里听到的"一声响"不是杀人，是 Emma 推父亲
- 那声呻吟是父亲还活着的证明
- 然后录音段三响起——脚步声、"父亲，我来了"、扭打声、闷响
- **真正的杀人是 Emma 走后才发生的**

**序幕的录音机 = Loop6 的录音 = 这是同一台机器**。Emma 在序幕里亲手按下的录制键，最终录到了她自己的清白。
