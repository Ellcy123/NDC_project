# Unit 5 大纲 v0.5 — 人物案件交织融合版

> 目标：保留 `0515` 版的人设质感与 6 Loop 推进手感，接入 `0526` 版的舞台暗门谜题。
>
> 这一版的核心不是“把机关讲清楚”，而是让每个机关证据都从一个人的选择、隐瞒、退守、反水里长出来。

## 标题：系统没有主人 — 笑面虎与他建造的牢笼

**时间**：1928 年 12 月 8-13 日  
**结构**：暴风雪山庄密闭 / 6 天议事 / 6 Loop 调查 / 每日投票  
**核心命题**：系统让坏人也以为自己是好人

---

## 一、融合原则

### 1. 保留 0515 的人物骨架

0515 最强的地方是：每个 Loop 击穿的不是单纯证物，而是一个“系统岗位”。

| Loop | 被击穿的人 | 他/她代表的系统位置 |
|---|---|---|
| L1 | Pierce | 警务/安保工具：把“不问”当生存技能 |
| L2 | Foster | 专业工具：被慈善基金培养出来的顺从者 |
| L3 | 小 Charles | 叙事管理者：把“资助”包装成控制 |
| L4 | Mary Jones / Lula | 潜伏者：用假身份进入系统内部的人 |
| L5 | Moore（Vivian 提供线索） | 怯懦融资者：在压力下说出隐藏空间 |
| L6 | 小 Charles | 系统继承人：终于承认自己也是系统的一部分 |

### 2. 替换 0515 的案件物理层

保留 0515 的 Loop 顺序，但把“书房楼梯 + 录音定案”替换成 0526 的：

- 礼堂舞台命案
- 老 Charles 私人夹层
- 舞台旋转楼梯位置错位
- Miller 家族宝石钥匙制度
- 暗门宝石脱落 + 血迹椭圆空白
- L6 多方证词包围 + 小 Charles 自白定案

### 3. 保留 0515 的情感手感

Emma 不再是“完全无关的摆件”。她没有杀人，但她的自责必须是真的。

本版处理：

- Emma 在书房采访中听到老 Charles 炫耀 1912 旧案，确实失控推开过他，导致一个轻微碰撞/跌坐片段。
- 她随后被人下镇静剂，记忆断裂；真相中是小 Charles，但玩家要到 L5-L6 才把这条线锁回他身上。
- 她大部分时候相信自己没杀人，但“我推过他”和“他后来死了”会被小 Charles 利用，短暂逼成“我也许真的杀了杀父仇人”的自疑。
- Emma 不作为正式指证对象。她的洗清通过 L4 指证 Mary Jones / Lula 的真实身份与药物专业完成：Zack 不是把 Emma 推上指证台，而是证明有人利用她的断片和自责给她套上罪名。

### 4. 保留 0526 的谜题核心

真正的杀人链仍是：

Lula 下药试探 → 老 Charles 警觉走夹层 → Moore 尾随逼谈 → 老 Charles 从舞台二层暗门踩空坠落 → 小 Charles 按原计划到舞台刺杀 → 嫁祸 Emma → 调回楼梯 → 走暗门离开 → 22:05 发现并偷走脱落宝石。

最后锁凶不是“宝石还在现场”，而是：

> 宝石曾经在现场，后来被拿走。  
> 拿走它的人知道它来自夹层暗门。  
> 知道这一点、并且必须隐藏夹层的人，只能是小 Charles。

## 二、前因与场域

### Tidewater / Miller 的双重命名

| 指称 | 含义 |
|---|---|
| **Miller 家族** | 老 Charles / 小 Charles 所属家族，Tidewater 最大股东与创始家族 |
| **Tidewater Trust Company** | 公司正式名，法律文件/议事签约/媒体标题使用 |
| **Lakeshore Investment Co.** | Tidewater 下属的 Whale 系列马甲，金融执行端 |
| **Whale** | Mickey 曾经的内部代号，呼应 Tidewater “潮汐吞没一切”的自我隐喻 |

**主题暗线**：潮汐没有主人，只有方向。老 Charles 以为自己掌控 Tidewater，小 Charles 弑父继承，也只是成为下一只被潮汐推着走的鲸。

### 暴风雪山庄四重锁

| 锁 | 内容 | 戏剧作用 |
|---|---|---|
| 物理 | 暴雪封路 5+ 天 | 无人能离开 |
| 法律 | 小 Charles 以继承人身份主张“家族内部事务”，Pierce 封锁现场 | 调查权落入笑面虎手中 |
| 契约 | 议事合同未签 = 数亿美元利益失效 | 宾客想留下，不是被迫留下 |
| 嫌疑 | Emma 是显性嫌疑人，离开等于自动定罪 | Emma 与 Zack 被锁进庄园 |

### 建筑与机关

Miller 私人庄园包含：

- **大议事厅**：6 天议事与公开指证场。
- **礼堂 / 小晚会厅**：Day1 晚会现场，带舞台与旋转式入场楼梯。
- **老 Charles 书房**：二层私人空间。
- **老 Charles 私人夹层**：秘密连通书房与礼堂舞台二层后台。除老 Charles 与小 Charles 外无人完整知道。

**夹层暗门设定**：

- 书房一侧：壁炉旁挂画后。
- 舞台一侧：旋转楼梯顶端对应的二层暗门。
- 平时楼梯在 A 位置，暗门外刚好接楼梯顶端。
- 当晚为了 Emma 演讲走台，技师把楼梯调到 B 位置，暗门外变成 4 米空中。

**Miller 宝石钥匙制度**：

- 每位 Miller 家族正式成员都有一颗带家族纹章的宝石钥匙。
- 宝石按入金属卡座可以触发机关门。
- 公开登记的宝石都有编号。
- 舞台暗门外侧卡座里有一颗老 Charles 私藏的特殊宝石：带纹章，但无公开编号。

**开篇演示场景**：Day1 入庄园时，小 Charles 当众用自己的宝石钥匙开门，并介绍“每颗宝石都有编号”。这让玩家在 L4 看到血迹椭圆空白时，有机会自然联想到宝石。

## 四、案发夜真实时间线

### 17:00 Emma 演讲彩排

- Emma 在礼堂彩排。
- 技师为她的演讲入场，把旋转楼梯调到 B 位置。
- 后台留下晚会准备清单 / 技师交接笔记。
- Emma 亲眼见过楼梯在 B 位置。

### 19:00-20:30 Day1 议事开幕

- 老 Charles 当众羞辱 Emma：“我们养了你。”
- 他向 Watts 炫耀南区清退完成。
- 他分别与 Watts / Moore / Foster 私下谈话，制造三条压力线。
- 晚会安排：21:30 Emma 在礼堂演讲，公开“答谢 Miller 家族”。

### 20:30-20:50 老 Charles 单独“采访”Emma

- 老 Charles 把 Emma 叫到书房，炫耀 1912 谋杀 Sean O'Malley 的细节。
- Emma 用记者速记本记录下这段供述。
- Emma 情绪失控，推开老 Charles，老 Charles 轻微撞到书桌/跌坐。
- 这不是致命伤，但会成为 Emma 后续自责的真实核心。

### 20:50-21:10 Emma 镇静剂发作 / Lula 下药试探

- Emma 离开书房后，小 Charles 给她演讲前“安神酒”，里面有低剂量镇静剂。
- Emma 开始手抖、鼻血、记忆断裂，把速记本塞进演讲台底。
- 同时 Lula 以 Mary Jones 身份给老 Charles 送日常药，在药里掺少量安眠药。
- 老 Charles 发现药粉颜色/气味不对，假装吃下，实际没吃。
- 他怀疑 Lula 背后有人，要避开正常路线。

### 21:00-21:25 老 Charles 走夹层 / Moore 尾随

- 老 Charles 从书房挂画后的暗门进入私人夹层。
- Moore 在走廊徘徊，想找老 Charles 谈账目；他看到挂画歪斜，误入夹层。
- Moore 在夹层里追上老 Charles，逼谈账目。
- 老 Charles 惊怒，转身想从舞台一侧暗门逃走。

### 21:25-21:30 老 Charles 坠落

- 舞台旋转楼梯仍在 B 位置。
- 老 Charles 从夹层内侧推开舞台暗门，脚下没有楼梯，4 米踩空坠落。
- 他抓扯暗门外侧机关，卡座变形，老 Charles 私藏的无编号宝石脱落到舞台地面。
- 老 Charles 摔伤严重，但没有立即死亡。

### 21:30-21:45 小 Charles 刺杀

- 小 Charles 按原计划把半昏迷 Emma 搬到舞台。
- 他看见父亲已经在舞台地面，误以为父亲“按约来了，只是状态异常”。
- 光线昏暗，舞台灯未全开，他没有意识到父亲已经坠落过一次。
- 他继续原计划，刺杀老 Charles。
- 老 Charles 垂死反抗，抓扯小 Charles 袖口。
- 刀伤才是真正致命伤。
- 血喷溅绕过地上那颗宝石，形成椭圆形空白。

### 21:45-22:00 现场布置与离场

- 小 Charles 把 Emma 摆到尸体旁，使她手上沾血。
- 他把匕首放在 Emma 手边。
- 他把旋转楼梯调回 A 位置，制造“Emma 推下楼梯 + 补刀”的假象。
- 他反锁礼堂门。
- 他用自己的宝石钥匙从舞台二层暗门外侧开锁，进入夹层，从书房一侧离开。
- 他此时没有注意到舞台地面的脱落宝石。

### 22:00-22:30 第一发现与计划外动作

- Pierce 按被调整过的巡查路线来到礼堂，发现门反锁后破门。
- 现场呈现：老 Charles 死、Emma 昏迷、手边有刀、手上有血。
- 小 Charles 作为悲痛儿子赶到。
- 他靠近尸体时第一次看见血迹椭圆空白和那颗无编号宝石。
- 他意识到这颗宝石会暴露夹层暗门。
- 他借着扶尸 / 整理衣领 / 悲痛跪倒，把宝石塞进自己口袋。
- 其他人即使看见，也只以为是某位家族成员遗落的普通宝石。

## 六、每个 Loop 的推进

### Loop 1：守夜人的报告

#### 【情节】

- Day1 议事桌建立所有人的利益关系。
- 老 Charles 当众羞辱 Emma，并向 Watts 炫耀南区清退。
- 晚会前 Emma 缺席，众人误以为她情绪崩溃。
- 22:00 Pierce 破门，礼堂舞台上出现尸体。
- Emma 昏迷在旁，手上有血，手边有刀。
- 小 Charles 第一次以“悲痛继承人 + Emma 守护者”姿态登场，反对立刻拘押 Emma。
- Zack 与 Emma 重逢，Emma 因 Unit4 旧账愤怒，不接受 Zack 的保护。

#### 【案件证据】

**指证对象**：Pierce。

| 层 | Pierce 说法 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 我全程按路线巡查 | 礼堂走廊烟蒂 / 巡查间隔异常 | 侧门把手缺少对应触碰痕迹 |
| L2 | 没看见任何人靠近礼堂 | 侧门男士皮鞋纹路 | Lula 送药途中看见有人影 |
| L3 | 我听到响声后才发现 | 巡查表被人临时改过 | Foster 初始死亡时间与 Pierce 报告不合 |

**击穿后真相**：Pierce 被“幽会掩护”脚本骗了。他以为自己在替某个宾客遮掩私会，所以主动绕开一段巡查。

#### 【自由探索大纲】

**搜索哪里，获得什么物证：**

- **礼堂舞台**：尸体倒在舞台中央偏侧，Emma 昏迷在旁，手边有匕首，手上有血；礼堂门从内侧反锁。得到“密室现场构图”，它强烈指向 Emma。
- **舞台地面**：血迹有拖擦和接触痕迹，但此时玩家只能看出“现场很乱”，还不能解释宝石空白。
- **礼堂走廊 / 侧门**：侧门附近有男士皮鞋印，门把手灰尘断层显示有人靠近过，但 Pierce 报告里没有记录。
- **安保值班桌**：Pierce 的巡查表有一段临时改动，21:30-22:00 的路线被绕开礼堂侧廊。
- **侧廊窗台**：发现烟蒂，位置不在 Pierce 声称的巡查停留点，说明他曾在另一个位置等过。

**和谁聊天，获得什么证词：**

- **Pierce**：他说“我全程按路线巡查，22:00 听到响声才去礼堂”，并坚称没看见任何人靠近。
- **Mary Jones / Lula**：她以护工身份说自己按时送药，经过侧廊时看见一个男士身影，但她不敢确认是谁。
- **Vivian**：她从客房方向听见侧廊有短促脚步声，时间早于 Pierce 所说的“听到响声”。
- **Foster**：给出非常快的初步判断，“像是坠落后补刀”，但此时他还没正式验尸。

#### 【指证大纲】

- **指证对象**：Pierce。
- **怎么指证**：不用证明他杀人，只证明他的守夜报告不真实。用巡查表改动、侧廊鞋印、烟蒂位置、Mary 的男士身影证词和 Vivian 的脚步声证词，击穿“全程巡查、没看见任何人”的说法。
- **剧情推进作用**：案件从“Emma 独自在密室杀人”推进为“有人提前改过安保脚本”。凶手是熟悉庄园和 Tidewater 运作方式的内部人。

#### 【人物推进】

- Pierce 不是主谋，而是一个训练有素的“不问者”。
- Zack 看他像看 Mickey：系统里很多人不是不知道，而是靠不知道活着。
- 小 Charles 的保护姿态极其有效：玩家此时仍会觉得他是 Emma 的盟友。

#### 【本 Loop 结论】

> 现场足以怀疑 Emma，但 Pierce 的谎言证明：案发前有人写好了内部脚本。  
> 凶手熟悉庄园、熟悉安保、熟悉 Tidewater 的“不问文化”。

### Loop 2：验尸的偏倚

#### 【情节】

- Foster 报告倾向：“坠落致死，刀伤可能是 Emma 补刀。”
- Lula 半夜复检尸体，但不告诉 Zack 自己是谁。
- Emma 的 1912 动机被具象化：Sean O'Malley 当年也死于“坠落”。
- Emma 自我怀疑加深：她记得自己推过老 Charles，却记不得之后发生了什么。
- Vivian 私下试探小 Charles 是否延续老 Charles 的庇护协议。
- Moore 私下找 Vivian，希望她不要公开 Webb 档案。

#### 【案件证据】

**指证对象**：Foster。

| 层 | Foster 说法 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 我单独验尸，没有外人接触 | 太平间登记缺口 | Lula 复检擦拭痕迹 |
| L2 | 坠落是主要死因 | 刀伤喷溅形态说明心跳仍强 | 摔伤内出血程度不足以立即致死 |
| L3 | 刀伤可能是死后补刀 | 胸口刀伤角度 / 力道显示生前刺入 | 老 Charles 指甲缝里有他人衣物纤维 |

**击穿后真相**：老 Charles 是先摔伤，后被刺死。Foster 报告被 Miller 慈善基金渠道影响。

#### 【自由探索大纲】

**搜索哪里，获得什么物证：**

- **太平间**：检查尸体，发现后脑/肩肋摔伤与胸口刀伤不是同一类伤；刀伤喷溅更像生前刺入。
- **太平间登记簿**：发现半夜有短暂空白，说明有人绕过 Foster 接触过尸体。

**和谁聊天，获得什么证词：**

- **Foster**：坚持“坠落是主因，刀伤可能是补刀”，并反复强调自己是唯一验尸人。
- **Lula / Mary Jones**：不承认身份，只说她“半夜担心老先生身体，去看过一眼”，语气明显避重就轻。

- **Vivian**：承认她在意老 Charles 生前的庇护协议能否延续，显得像有利益动机。
- **Moore**：承认老 Charles 生前提过要审他的账，但坚称案发时自己在客房。

#### 【指证大纲】

- **指证对象**：Foster。
- **怎么指证**：用尸体伤势、刀伤喷溅、太平间登记缺口、验尸草稿与正式报告差异、基金电话记录，证明他把“先摔后刺”的复杂死因写成了对 Emma 不利的简单结论。
- **剧情推进作用**：案件从“Emma 复仇杀人”推进为“专业系统参与改写真相”。同时 Foster 成为 Emma 的镜像：同样被 Miller 系统养大，但他选择替权力背书。

#### 【人物推进】

- Foster 是“没反水版 Emma”：同样被 Miller 系统养大，但他学会了用专业性替权力背书。
- Emma 意识到“我不是唯一被他们养大的人”，她的身份羞耻感被放大。
- Lula 的复检第一次显示她不是普通护工，但仍不揭身份。

#### 【本 Loop 结论】

> 致命伤是刀，不是摔。  
> 但摔伤发生在刀伤之前。  
> 案件不再是单纯的“Emma 复仇”，而是有人把一个已受伤的人再次杀死，并利用 Foster 把结论写歪。

### Loop 4：Mary Jones 是谁

#### 【情节】

- Emma 记忆部分恢复：她记得自己在书房推开过老 Charles，记得他撞到书桌/跌坐，记得自己后来断片。

- Watts 苏醒，释放三条碎片：
  - 他听见过礼堂方向一声闷响；
  - 他看见 Moore 从书房方向慌忙出现；
  - 他在被袭击前看见礼堂侧有一个被搀扶/拖扶的女性身影，后来与 Emma 的衣物和发饰对上。
- 玩家在演讲台底找到 Emma 速记本：前段清晰记录 1912 供述，后段字迹混乱。
- Mary Jones 在帮 Emma 梳理镇静剂残留时暴露专业能力：她不像临时护工，更像训练过的护士。
- 混乱中 Mary Jones 本能戴回口罩处理伤口，Zack 认出她与 Unit10 的 Lula 有同一套护理习惯；Watts 也能证明救下自己的人就是她。
- 小 Charles 试图用“家族不追究 Emma”收口，但 Zack 转向追问 Mary Jones：Emma 的药物状态是谁最清楚？谁在案发夜接触过药？

#### 【案件证据】

**指证对象**：Mary Jones。

| 层 | Mary Jones 说法 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 我只是临时护工，不懂复杂药理 | 她能准确判断 Emma 的镇静剂反应 | 护理包里有专业护士工具和药物记录 |
| L2 | 我不是 Lula，也不认识 Frank | 行李箱里的 Frank 照片与 Mickey 留信 | Watts 认出救他的人与 Mary 是同一个人 |
| L3 | 我没有对老 Charles 的药动手 | 护理包里安眠药短缺 | 老 Charles 书房药粉残留颜色异常 |
| L4 | Emma 的状态我只是案后看出来的 | 药杯 / 酒杯残留显示 Emma 案发前已被镇静剂影响 | Watts 证词证明 Emma 被搀扶时已经失去行动能力 |

**击穿后真相**：Mary Jones 就是 Lula。她给老 Charles 的药里掺安眠药，是为了让他昏睡后翻找档案，不是杀人；她没有给 Emma 下镇静剂。L4 只确认 Emma 案发前已被另一个人下药，并在失去行动能力后被摆到舞台。Lula 的药物专业洗清 Emma，Lula 对老 Charles 下药则解释他为什么警觉地避开正常路线。至于谁能接触 Emma 的酒杯，留到 L5-L6 继续追。

#### 【新增物理钩子】

- 二次勘察舞台时发现血迹里有一个椭圆形空白。
- 这个空白尺寸接近开篇小 Charles 演示过的 Miller 宝石。
- 摔伤角度显示老 Charles 曾从约 4 米高度坠落。
- 舞台正常楼梯无法解释这个坠落角度。

#### 【自由探索大纲】

**搜索哪里，获得什么物证：**

- **演讲台底部**：找到 Emma 的速记本。前段清晰记录老 Charles 炫耀 1912 谋杀，后段字迹突然崩坏。
- **Emma 房间 / 衣物**：衣袖、手套和发饰上有被人拖扶/搀扶时留下的压痕和摩擦痕。
- **药杯 / 酒杯残留**：确认镇静剂成分，剂量足以造成意识断片和行动能力下降。
- **Mary Jones 护理包**：发现专业护士工具、药物记录、安眠药短缺。
- **Mary Jones 房间 / 行李箱**：找到 Frank 照片、Mickey 留下的接头纸条、旧口罩和化名文件，证明 Mary Jones 不是临时雇来的普通护工。
- **老 Charles 书房药品**：日常药粉颜色异常，残留与护理包里的安眠药能对应。
- **舞台血迹**：Emma 手上的血更像接触沾染，不像主动刺杀喷溅。
- **尸体摔伤角度复核**：摔伤高度约 4 米，正常旋转楼梯位置解释不了。
- **舞台地面血迹椭圆空白**：血迹中有一块椭圆形未沾血区域，像被某个小物体挡住过。

**和谁聊天，获得什么证词：**

- **Emma**：她承认自己在书房推开过老 Charles，但坚持“我想过杀他，不等于我杀了他”；她真正害怕的是记忆空白让自己无法自证。
- **Mary Jones / Lula**：一开始只承认自己看出 Emma 被药物影响，后来被 Frank 照片、Mickey 留信和口罩习惯逼到承认身份。
- **Watts**：苏醒后说听见礼堂方向闷响，看见 Moore 从书房方向慌忙出现，也见过礼堂侧一个被搀扶的女性身影。
- **Watts 追问**：他确认半夜救自己的就是 Mary Jones，并补充她当时避开正式医生，像是在隐藏身份。
- **Foster**：在压力下承认刀伤角度与 Emma 当时状态不合，但仍不愿完全推翻自己报告。
- **小 Charles**：公开宽恕 Emma，“Miller 家族不追究”，但当 Mary 的药物专业开始指向“Emma 被摆放”时，他第一次失去耐心。

#### 【指证大纲】

- **指证对象**：Mary Jones。
- **怎么指证**：先用口罩习惯、Frank 照片、Mickey 留信和 Watts 证词击穿“Mary Jones”假身份；再用她的药理判断、护理包、安眠药短缺、老 Charles 药粉残留、Emma 药杯残留和被搀扶女性证词，击穿“只是案后观察 / 没碰药”的说法。
- **剧情推进作用**：Emma 被洗清，但不是靠指证 Emma，而是靠揭穿 Mary 的真实身份和药物专业。Lula 暴露后转盟，Unit10 的情感债提前回收；同时案件获得新方向：老 Charles 因为药物警觉避开正常路线，他去了哪里？

#### 【人物推进】

- Emma 的弧光不是“我什么都没做”，而是“我做过错事，但我不能替真正的凶手承担罪名”。
- Lula 从潜伏工具人变成主动证人，她用自己的罪错换 Emma 的清白。
- 小 Charles 不是本场正式被指证者，但他的宽恕话术被打断，第一次在公开场失控。

#### 【本 Loop 结论】

> Emma 没有杀人。  
> Mary Jones 就是 Lula，她的药物专业确认 Emma 的断片来自镇静剂，她对老 Charles 下药解释了老 Charles 的警觉。  
> 老 Charles 在被刺杀前，从一个高处摔下。  
> 血迹里曾经压着一颗椭圆形物体。  
> 案件从“谁嫁祸 Emma”升级为“老 Charles 躲去了哪里 / 舞台上方有什么”。

### Loop 5：Moore 与夹层暗门

#### 【情节】

- Day5 历史档案口径讨论，Emma 把 Frank 1912 资料原件砸上议事桌。
- Lula 公开站到 Emma 一边，Frank 资料与 Mickey 档案合流。

- Vivian 释放线索：“我听过老 Charles 有个秘密空间，在书房里。”
- Moore 被连续逼问后退守：“我那晚看见老 Charles 进了书房壁炉旁挂画后的门。”

#### 【案件证据】

**指证对象**：Moore。

| 层 | Moore 说法 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 21:00-22:00 我在客房 | Watts 苏醒证言 | 走廊地毯徘徊鞋印 |
| L2 | 我只是路过书房 | 挂画歪斜 + 指纹 | 衣袖灰尘对应壁炉旁木屑 |
| L3 | 我没进秘密通道 | 夹层入口衣物纤维 | 夹层岔口鞋印 |

**击穿后真相**：Moore 进过夹层，但只到岔口附近。他不知道出口通往舞台，也没有完整逃脱路线。

#### 【夹层发现】

- 玩家进入书房，推开挂画，发现夹层暗门。
- 穿过夹层来到舞台二层。
- 发现舞台暗门外侧机关卡座变形，里面原本应有宝石的位置空了。
- 卡座尺寸、底部 Miller 纹章与 L4 血迹椭圆空白吻合。

#### 【关键推理】

1. 老 Charles 从暗门坠落时，撕扯机关，宝石脱落。
2. 小 Charles 刺杀时，血喷溅绕过地上宝石，留下椭圆空白。
3. Pierce 破门时宝石可能仍在血迹旁，但没有人理解它的意义。
4. 小 Charles 作为悲痛儿子赶到后，在围观与封锁之间把宝石顺走。
5. 之后现场物证清单里没有宝石，只剩血迹椭圆空白。
6. 普通人看到宝石，只会交给 Pierce 或当作家族成员遗落。
7. 只有知道“这颗无编号宝石来自夹层暗门”的人，才会冒险偷偷拿走。

#### 【自由探索大纲】

**搜索哪里，获得什么物证：**

- **议事桌**：Emma 把 Frank 1912 资料原件公开，里面有 Sean O'Malley 旧案、Frank 调查笔记、Tidewater 早期分赃线索。
- **老 Charles 书房**：壁炉旁挂画歪斜，画框边有新鲜指纹和木屑。
- **挂画后暗门**：推开后进入夹层，确认书房隐藏入口真实存在。
- **夹层通道岔口**：有 Moore 鞋印、衣袖纤维和停顿/转身痕迹，证明他到过但没继续深入。
- **舞台二层暗门**：外侧宝石卡座变形且空缺，卡座尺寸与 L4 血迹椭圆空白吻合。
- **现场物证清单**：没有登记任何无编号宝石，反衬“宝石曾在现场但后来消失”。

**和谁聊天，获得什么证词：**

- **Lula**：公开转盟，确认老 Charles 没吃下安眠药，之后行为异常，可能选择避开正常路线。
- **Watts**：确认自己见过 Moore 从书房方向慌忙出现。
- **Vivian**：释放“老 Charles 书房里有秘密空间”的传闻；她不知道入口，但知道老 Charles 多年来用某个私密空间处理交易。
- **Moore**：先说在客房，后退守为“我只是在书房附近”，最终承认看见老 Charles 进挂画后的门，并跟进去一段。
- **Pierce**：回忆小 Charles 赶到现场速度过快，不像从客房来。
- **Foster / 围观宾客**：回忆小 Charles 扑向尸体时动作过深，起身时西装下摆带血。

#### 【指证大纲】

- **指证对象**：Moore。
- **怎么指证**：用 Watts 证词、书房挂画痕迹、指纹、衣袖灰尘、夹层鞋印和岔口停顿痕迹，逼他承认进过夹层。
- **推进作用**：Moore 暴露夹层入口，但也因“不知道出口在舞台”被排除为完整凶手。玩家第一次真正进入案件的隐藏空间。

- **阶段性指向**：小 Charles。
- **怎么指向但不定案**：宝石卡座与血迹空白吻合，现场物证清单没有宝石，说明有人在发现现场后拿走宝石；小 Charles 有机会、有动机，也知道夹层完整路线。但还需要 L6 当庭对质公开封口。

#### 【22:05 现场目击回收】

玩家追问 Pierce / Foster / 围观宾客：

- 小 Charles 扑向尸体时弯腰过深。
- 他整理父亲衣领时手在血迹附近停留过。
- 起身时西装下摆有血。
- Pierce 承认小 Charles 到场速度异常快，不像从客房赶来。

#### 【人物推进】

- Moore 不是最终凶手，但他终于从“金融恐惧”退到“我确实在场”。
- Vivian 的跳船是求生逻辑，不是突然变善。
- Emma 与 Lula 的同盟让“受害者互相作证”成为主题动作。

#### 【本 Loop 结论】

> 夹层真实存在。  
> 老 Charles 从舞台二层暗门坠落。  
> 暗门宝石曾经落在血迹现场，后来被人拿走。  
> Moore 只知道入口，不知道完整出口。  
> 真正的凶手必须同时满足：知道完整夹层路线、能布置 Emma、能调楼梯、能在 22:05 接近尸体偷走宝石。

推理已经指向小 Charles，但还需要当庭公开封口。

### Loop 6：议事桌当庭对质

#### 【情节】

- Day6 最终签约日，小 Charles 已成为实质主持人。
- Zack 在议事桌启动闭环对质。
- 所有 NPC 同时成为听众、证人、被系统塑造过的人。
- 小 Charles 不再只否认事实，而是主动制造开放性怀疑：
  - Moore 进过夹层，他也可能杀人；
  - Pierce 反水可能是在演戏；
  - Foster 的法医推断不稳；
  - Emma 的记忆是镇静剂幻觉；
  - Lula 是复仇者，她的话不可信。

#### 【案件证据】

**指证对象**：小 Charles。

| 层 | 小 Charles 防线 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 我 21:30-22:00 在书房处理文件 | Pierce 反水：到场速度不对 | 22:05 目击：扑尸时弯腰过深 |
| L2 | 夹层不只我可能知道 | Moore 只到岔口，不知舞台出口 | Vivian 只听过秘密空间，不知位置 |
| L3 | 宝石空白不能证明是我拿走 | 暗门卡座与血迹空白吻合 | 拿走宝石的人必须知道无编号宝石会暴露夹层 |
| L4 | Foster / Lula / Moore 都说过谎，不可靠 | 六方证词横向并列互相补足 | 物证链不依赖单一证人 |
| L5 | Moore 或 Pierce 仍有嫌疑 | Foster 反水：刀伤身高/角度/力道匹配小 Charles | Pierce 22:00 破门时已被多人见到 |

#### 【自由探索大纲】

**搜索哪里，获得什么物证：**

- **议事厅会前证据桌**：把速记本、验尸草稿、晚会准备清单、楼梯擦痕拓印、夹层卡座拓印、血迹椭圆空白图、现场物证清单并排摆出。
- **小 Charles 西装**：袖口纤维可与老 Charles 指甲缝纤维对应；下摆残血与 22:05 扑尸动作吻合。
- **暗门宝石卡座复核**：证明卡座缺失物形状、尺寸、底部纹章，与血迹空白完全对应。
- **巡查路线复核**：Pierce 被改动的路线和小 Charles 到场速度共同击穿“不在场”。

**和谁聊天，获得什么证词：**

- **Lula**：确认下药目的、老 Charles 没吃药、之后行为异常，解释他为什么走夹层。
- **Moore**：确认自己只到夹层岔口，没到舞台出口，不知道暗门另一端。
- **Watts**：固定他听到闷响、看见 Moore 慌忙出现的时间点。
- **Foster**：完整说明刀伤角度、身高、力道与凶手体态；排除 Emma、Moore、Pierce 的关键不匹配。
- **Pierce**：承认小 Charles 到场速度异常，且自己被“幽会脚本”骗了。
- **Emma**：确认她半昏迷中的男声碎片，只作为辅助，不把她记忆当单一铁证。
- **Vivian**：确认她只听过秘密空间传闻，不知道入口和出口，排除“她也能利用夹层”的可能。

#### 【指证大纲】

- **指证对象**：小 Charles。
- **怎么指证**：不是靠一件铁证砸死，而是把六方证词和宝石物证横向并列：
  - Lula 解释老 Charles 为什么进夹层；
  - Moore 暴露夹层入口并排除自己完整路线；
  - Watts 拼出坠落前后时序；
  - Foster 证明真正刺杀者更符合小 Charles；
  - Pierce 击穿小 Charles 不在场；
  - Emma 的男声碎片与小 Charles 自白互相印证；
  - 宝石空白证明有人偷走会暴露夹层的证物。
- **剧情推进作用**：L6 不再是“找到凶手”，而是让所有被系统塑造过的人公开说话。小 Charles 被自己管理过的人围住，最终不是被单一录音击败，而是被群体证词击败。

#### 【六方证词包围】

| 证人 | 说出的真相 | 作用 |
|---|---|---|
| Lula | 她下药试探，老 Charles 没吃药并变得警觉 | 解释老 Charles 为什么走夹层 |
| Moore | 他尾随老 Charles 进入夹层，但在岔口退回 | 暴露入口，同时排除自己完整路线 |
| Watts | 他听见礼堂闷响，看见 Moore 慌忙从书房方向出现 | 拼接坠落前后时序 |
| Emma | 她半昏迷中听见男人喊“父亲”，但不确定是不是梦 | 与小 Charles 自白/其他证据互相确认 |
| Foster | 刀伤角度、身高、力道更匹配小 Charles | 专业工具第一次选择说真话 |
| Pierce | 小 Charles 到现场太快，他以为自己掩护的是别的男人 | 击穿小 Charles 不在场 |

#### 【笑面虎崩塌】

第一阶段：逐条否认。

第二阶段：Zack 把证词横向并列，说明每条单独都可疑，但六条同时指向同一方向。

第三阶段：小 Charles 沉默。他意识到自己不是被某一件证物击败，而是被所有被他管理过的人同时说话击败。

他终于说出：

> “你们都对。是我。父亲死了三天，我每天都在等这一刻。  
> 我以为我能再骗过一次。”

随后他讲出当晚细节：

> “我说，父亲，我来了。  
> 我以为他按约到了。  
> 我以为一切都还在我的剧本里。”

最终主题落点：

> “你父亲的事，我父亲做的。  
> 我今天不过是杀了一个会毁掉集团的老人。  
> 我帮过你，是真的；我利用你，也是真的。  
> 我需要你相信 Miller 还有救。  
> 只要你信，别人就会信。”

#### 【人物推进】

- 小 Charles 不是突然从好人变坏人，而是终于承认他的“好”一直服务于控制。
- Emma 的最后痛点不是“他骗了我”，而是她发现自己一直被当成 Miller 可以被原谅的证明：他的照护越真实，控制越彻底。
- Zack 面对 Mickey 之后的选择再次浮现：打倒系统，还是继承系统方法。

#### 【本 Loop 结论】

> 物证链在 L5 已经成立。  
> L6 不是找出凶手，而是让所有被系统塑造的人公开说话。  
> 笑面虎不是被录音打败，而是被他以为可以管理的人打败。

## 九、关键物证链

### 链条 A：楼梯位置

1. 后台准备清单：21:00 楼梯调到 B 位置。
2. Pierce 证词：21:15 巡查时楼梯在 B 位置。
3. Emma 彩排记忆：17:00 楼梯也在 B 位置。
4. 现场发现时楼梯在 A 位置。
5. 地板擦痕证明有人调回过。

阶段结论：有人把楼梯调回 A 位置，伪装正常楼梯事故。

### 链条 B：死亡时序

1. Foster 报告偏向坠落致死。
2. Lula 复检发现刀伤才致命。
3. 摔伤内出血显示摔伤在前。
4. 刀伤喷溅显示刺杀时老 Charles 仍活着。

阶段结论：老 Charles 先摔，后被刺。

### 链条 C：坠落来源

1. 摔伤角度对应约 4 米高度。
2. 舞台正常楼梯无法解释。
3. 夹层暗门出口高度吻合。
4. 当晚楼梯在 B 位置，暗门外无承接。

阶段结论：老 Charles 从舞台二层暗门踩空坠落。

### 链条 D：谁知道夹层

1. 老 Charles 建造夹层。
2. 小 Charles 三个月前发现完整路线。
3. Moore 只撞见入口，到岔口退回。
4. Vivian 只听过秘密空间，不知位置。
5. 其他人不知道。

阶段结论：能用夹层完成密室离场的人，只剩小 Charles。

### 链条 E：宝石空白

1. 开篇演示宝石钥匙制度。
2. L4 血迹中发现椭圆空白。
3. L5 夹层暗门卡座空缺，尺寸吻合。
4. 卡座无编号宝石会暴露夹层来源。
5. 22:05 小 Charles 有接近尸体和血迹的动作窗口。
6. 目击证词显示他扑尸时动作过深。

阶段结论：小 Charles 在发现现场后偷走了宝石。

## 十、每日投票与关系取舍

仍沿用 0515 的投票设计：

| 时间 | 内容 |
|---|---|
| Day1-5 夜末 | 投“最怀疑谁” |
| Day6 终局 | 三选项处置小 Charles |

投票不是单纯猜凶，而是改变第二天谁愿意说话：

- 怀疑某人，会让他/她关闭私聊。
- 但被怀疑的人也可能为了自保，向别的 NPC 泄露信息。
- 这让“错误怀疑”也能推进剧情，只是推进方向不同。

### Day6 终局三选项

| 选项 | 性质 | 基调 |
|---|---|---|
| 当场报警 | 系统外正义 | 小 Charles 入狱，警方介入，但 Tidewater 律师团反扑 |
| Emma 写报道公开 | 舆论正义 | Tidewater 形象崩塌，Emma 继承 Sean 的新闻遗志 |
| 用 Mickey 档案威胁退位 | 系统内正义 | Zack 用 Mickey 的方法，集团仍存，Zack 沾色 |

第三项必须最不舒服：

- Vivian 和 Moore 会活得最舒服。
- Watts 会拒绝继续与 Zack 合作。
- Lula 会沉默归隐，Frank 资料被封存。

这让玩家感到：系统内胜利确实有效，但它会奖励最会适应系统的人。

## 十一、这一版的核心体验

玩家体验应该是：

1. 先被舞台密室推去怀疑 Emma。
2. 再通过 Foster 和 Lula 的专业痕迹发现 Emma 被嫁祸。
3. 接着发现小 Charles 的“好”是叙事管理，但还不能说他是凶手。
4. L4 通过指证 Mary Jones / Lula 的真实身份和药物专业救出 Emma，让小 Charles 的宽恕显得异常，同时打开“老 Charles 去了哪里”的问题。
5. L5 通过 Vivian、Moore 和 Lula 三个人选择说话，找到夹层暗门和宝石空白。
6. L6 不是靠单一铁证，而是让所有被系统塑造的人在议事桌上同时作证，逼笑面虎承认。

一句话总结：

> 0515 提供人物如何一步步说真话，0526 提供真相如何在物理世界留下形状。  
> 融合版要让每个证据都有人的体温：烟蒂来自 Pierce 的不问，验尸来自 Foster 的顺从，楼梯来自小 Charles 的剧本，宝石来自他计划外的恐惧，而最终真相来自所有人终于不再替系统沉默。
