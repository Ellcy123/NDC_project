# Unit 5 大纲 v0.3

## 标题：系统没有主人 — 笑面虎与他建造的牢笼

**时间**：1928 年 12 月 8-13 日（南区清退完成第 3 天 → 第 8 天）
**结构**：暴风雪山庄密闭 / 6 天议事 / 6 Loop 调查 / 每日投票
**核心命题**：系统让坏人也以为自己是好人

---

## 阅读说明

每个 Loop 分三栏：

- **【情节】** 戏剧推进 / 角色弧光 / 主题渗透
- **【案件证据】** 指证对象 / 谎言层级 / 击穿维度（具体物件、台词、动线放 state 文件）
- **【相互推动】** 情节如何变成证据 / 证据如何打开情节

两条线不是平行的——**击穿一个谎言 = 释放一段戏剧**，**一段戏剧 = 暴露下一个谎言的破绽**。

## 一、前因（Unit4 末 → Unit5 开场）

### 命名约定：Miller 家族 vs Tidewater 集团

| 指称 | 含义 |
|---|---|
| **Miller 家族** | 老 Charles / 小 Charles 所属的姓氏家族，集团创始人 + 最大股东 |
| **Tidewater Trust Company**（中：潮汐信托公司 / 简：Tidewater 集团 / 潮汐集团）| 公司品牌正式名——对外签合同、登报纸、走法律的法定名称 |
| **Lakeshore Investment Co.** | Tidewater 集团下属的 Whale 旗下马甲（Unit10 已有，金融执行端）|
| **Mickey 内部代号 "Whale"** | Tidewater 集团内部行动者代号体系之一（鲸——跟着潮水迁徙）|

**各场景用什么**：

| 场景 | 用 |
|---|---|
| 法律文件 / 议事签约 / Emma 报道标题 | Tidewater Trust Company |
| 英文媒体口语 / 正式介绍 | Tidewater Trust |
| 中文 NPC 议事桌 | 潮汐集团 / Tidewater 集团 |
| 中文口语简称 | 潮汐 |
| 家族指称 | Miller 家族 / Miller 家 |

**"Tidewater"和"Miller"互不替换**——一个是公司，一个是家族。

### 公司名内嵌的彩蛋

老 Charles Miller 取名 "Tidewater"——表面上是他早年起家于东海岸航运的纪念（合理化历史），**实际上是"潮汐淹没一切"的暗喻**。

Mickey 的内部代号 "Whale" 正是这个隐喻的延伸——**鲸群跟着潮水迁徙、吞噬猎物**。公司名 Tidewater + 内部代号 Whale 系列之间的关联，是给玩家深挖时的彩蛋：发现两者关系 = 发现这家公司**自我隐喻的本质**——它不是一家公司，是一片"吞噬一切的潮汐"，而鲸只是它的爪牙。

这个彩蛋也呼应 Unit5 主题"系统没有主人"——**潮汐没有主人，只有方向**。老 Charles 以为自己掌控潮汐，实际上他只是被潮汐推到顶端的一只鲸；小 Charles 弑父继承，也只是接过下一只鲸的位置。

### Tidewater 集团状态

- 南区清退完成（11 户 11 人死，含 Mrs. O'Hara），Mickey 死 3 天
- Zack 手握 Mickey 移交的 Tidewater 集团 30 年完整档案
- 老 Charles 召集 6 天议事会，宣布"南区新综合体"规划，邀芝加哥金融/警务/法律/专业/商业五界合伙人

### 议事桌的真实意图

不是商业会议，是 **1912 港口分赃模式的 1928 翻版**。老 Charles 用同一套"清退—合作—洗白"逻辑，把血染地块洗成合法资产。每个宾客是当年某角色的功能等价物。

### 小 Charles 的杀机

- 3 个月前他发现父亲用法律信托剥夺他继承权
- 议事会是他动手的最后窗口期
- 他用父亲的"消除麻烦"逻辑反杀父亲

### Unit4 已植入的伏笔（玩家进 Unit5 时的认知）

**已知**：小 Charles 是 Emma 16 年赞助人 + 新闻行业话事人 + Miller 家族里唯一对 Emma 好的人
**不知**：他的"好"是叙事管理工具

### 暴风雪山庄四重锁

| 锁 | 内容 | 戏剧作用 |
|---|---|---|
| 物理 | Day1 起暴雪封路 5+ 天 | 无人能离开 |
| 法律 | 小 Charles 以家族继承人身份主张"内部事务"，Pierce 封锁现场，市政警察因暴雪无法进入 | 调查权落入笑面虎手中 |
| 契约 | 议事合同未签 = 数亿美元利益失效 | 宾客**想留**，不是被迫 |
| 嫌疑 | Emma 是显性嫌疑人，离开 = 自动定罪 | Emma 与 Zack 锁死庄园 |

### Lula 的口罩设定与化名

**前提（Unit10 既定）**：Lula 因慢性气管问题在 Unit10 期间长期戴医用口罩——这是她的护士职业习惯 + 健康需要。**Zack 在 Unit10 见到她时她始终戴着口罩，从未见过她的脸**。

**Mickey 临死部署**（Unit4 末新增伏笔）：Mickey 在 42 楼坠落前给 Lula 寄信/秘密见面 → 坦白自己 = Whale（间接害死 Frank） → 给她 Tidewater 集团内部情报 → 请她进庄园。Lula 选择原谅 Mickey 并接过他的使命。

**进庄园的伪装**：
- 化名 **Mary Jones**
- **不戴口罩**（反常——为了装"健康护工"不被 Tidewater 集团怀疑）
- 这是她为执行任务**主动放弃护身符**——给她角色加一层牺牲感

**Loop 5 反转的物理触发点**：混乱中 Lula 出于本能/工作需要戴回口罩 → Zack 看见戴口罩的她 → 瞬间认出。

### 议事桌横向 micro-drama

议事桌不只是公开发言场，是 NPC 之间复杂关系的同框观察台。玩家通过观察 NPC 间的眼神/座位/话头转移可以察觉以下张力：

| 关系 | micro-drama 内容 |
|---|---|
| Moore ↔ Vivian | Moore 怕 Vivian 手里的 Webb 档案；Vivian 知道但不戳穿 |
| Moore ↔ 老 Charles | Moore 想新综合体融资但怕被绑死，议事桌上恭顺背后是焦虑 |
| Vivian ↔ 老 Charles | 庇护协议存在（玩家 L4 才发现）；议事桌上 Vivian 的"紧张"被误读为复仇眼神 |
| Watts ↔ Tidewater 集团 | Watts 是 Tidewater 弃子，L3 拒签即接受被清退；玩家 L3 揭穿小 Charles 时浮现 Mickey 档案"Watts 拒绝判决列表" |
| Watts ↔ Harrison（镜像）| 见过妥协者下场，主动选择不参与——和 Harrison 形成"妥协被杀 vs 拒绝被弃"的镜像 |
| Watts ↔ Lula | L3 被袭击后 Lula 救下他；养伤期间形成同盟（L4-5 Lula 暴露时 Watts 法律侧背书）|
| Foster ↔ Emma | 孤儿院同学（L2 揭穿 Foster 培育产物时浮现）|
| Moore ↔ Lula | Moore 不认识 Lula 的脸（盲点）——L5 Lula 暴露时 Moore 的反应才是关键戏 |
| Vivian ↔ Lula | Vivian 觉得护工"奇怪"但不深究（她不关心和自己活路无关的事）|
| 老 Charles ↔ 小 Charles | 继承权剥夺（玩家 L4 后期发现）|

## 三、每个 Loop 的推进

### Loop 1：守夜人的报告

#### 【情节】
- 议事 Day1：老 Charles 主持，当面羞辱 Emma "我们养了你"，向 Watts 炫耀"南区清退已完成 11 人"
- **议事末散会前老 Charles 的三场私下戏**（既塑造他该死的质感，也为各 NPC 紧张铺垫）：
  - 与 Watts 私下谈话——拉拢/施压（要他签法律备案）
  - 与 Moore 谈账目——让 Moore 紧张（暗示要审计他银行黑账）
  - 与 Foster 暗示验尸口径——通过基金渠道下指示（为他自己潜在的"意外"做铺垫，他没想到这反而被儿子用上）
- 议事 Day1 散会后，老 Charles 死在书房楼梯下
- Emma 坐书房手上有血、记忆空白——**显性怀疑对象**
- 笑面虎首次登场：小 Charles 在议事桌为 Emma 辩护、反对拘押
- Zack 与 Emma 重逢冲突（Vivian 拉 Zack 当保镖进场，Emma 第一反应是愤怒）

#### 【案件证据】

**指证对象**：Pierce（议事会安保）

**Pierce 的 3 层谎言**：

| 层 | 公开陈述 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 全程巡查 | 物证：阳台烟蒂 | 人证：Vivian 客房目击 |
| L2 | 没看见任何人进书房 | 物证：走廊男士皮鞋纹路 | 人证：Lula 送药目击 |
| L3 | 听到响声才发现 | 时序：Lula 送药登记本 | 时序：Foster 死亡时间 |

**击穿后真相**：Pierce 被"幽会掩护"脚本欺骗，**凶手是预谋已久的 Tidewater 集团内部人**。

#### 【副怀疑】（供投票候选）

- **Emma**：显性嫌疑（手上血 / 复仇动机 / 现场）——错的方向，但玩家在 L1 几乎必投
- **Moore**：老 Charles 可能要审计他，借机杀人——动机扎实，玩家可投
- **Vivian**：议事桌上对老 Charles 眼神紧绷，玩家误读为复仇——实际是失去庇护人的焦虑

#### 【相互推动】
- **情节 → 证据**：笑面虎为 Emma 辩护 → 玩家以为 Emma 仍是嫌疑 → 调查转向"还有谁在场"
- **证据 → 情节**：击穿 Pierce 揭示"凶手有内部脚本" → 暗示凶手熟悉 Tidewater 集团 → Zack 看 Pierce 像看 Mickey（另一个把"不问"当生存技能的人）

### Loop 2：验尸的偏倚

#### 【情节】
- Foster 报告"意外坠落"——案件似将草草定性为意外
- Lula 第一次暗线行动：半夜复检尸体，发现 Foster 漏掉的痕迹但不告诉 Zack
- Emma 的动机被 1912 资料具象化：Sean O'Malley 死法（坠落+头撞）与老 Charles 完全一致——**仪式性复仇**框架
- Emma 自我怀疑萌芽
- **Vivian 的私下小动作**（玩家私聊或观察可发现）：
  - Day2 晚她私下接触小 Charles 试探续约："我希望和您父亲的安排能继续"——小 Charles 暧昧回应不表态
  - Moore 也在私下找 Vivian——"多给你钱别公开档案"——Vivian 礼貌拒绝（她对档案的承诺只对老 Charles 有效）

#### 【案件证据】

**指证对象**：Foster（议事会驻场法医）

**Foster 的 3 层谎言**：

| 层 | 公开陈述 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 单独验尸 | 物证：太平间登记缺口 | 物证：Lula 复检擦拭痕迹 |
| L2 | 伤都是坠落造成 | 物证：颈部勒痕角度 | 物证：衣领外人纤维 |
| L3 | 尸体没被外人接触 | 物证：地毯纤维位置 | 物证：拖拽方向 |

**击穿后真相**：老 Charles 被掐死后搬到楼梯下伪装。Foster 报告偏倚是被基金电话指示的——Foster 是 Miller 慈善基金 1916 年受益人。

#### 【副怀疑】（供投票候选）

- **Moore**：基金电话指示来源——他是基金董事之一，可能就是发起电话的人。玩家此 Loop 投 Moore 合理
- **Emma**：仪式性复仇动机加深（Sean 死法 = 老 Charles 死法），玩家仍可投

#### 【相互推动】
- **情节 → 证据**：Lula 暗线复检 → 提供 Foster 偏倚的物证基础（玩家此时不知 Lula 是谁）
- **证据 → 情节**：Foster 是 Miller 慈善基金培育产物 → Foster = 没反水版的 Emma → Emma 当场意识到她不是孤例 → **Emma 与 Foster 镜像戏成立**

### Loop 3：笑面虎的叙事管理

#### 【情节】
- 议事 Day3：Watts 拒签法律备案
- **笑面虎仁慈姿态登顶**：小 Charles 私下对 Zack 说"Emma 是受害者，如果是她杀的，我能理解，我会替她抗下来"
- Emma 听说后第一次哭——她以为终于有了家族盟友
- 当晚 Watts 被袭击重伤——Lula 半夜路过救下

#### 【案件证据】

**指证对象**：小 Charles（**叙事管理层面**，不是凶手层面）

**小 Charles 的 3 层谎言**：

| 层 | 公开陈述 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 只资助 Emma | 物证：Mickey 档案"线人管理册" 20+ 记者 | 物证：资助合同档案 |
| L2 | 从不干预 | 人证：5 名记者职业轨迹被调整 | 人证：Watts 一份未公开判决涉及"被调线"记者 |
| L3 | 资助是慈善 | 物证：合同条款"职业方向变更需经基金认可" | 物证：部分资助挂钩"舆论合作义务" |

**击穿后真相**：小 Charles 在管理芝加哥新闻叙事，任何不听话的都要清退。**袭击 Watts 与叙事管理同一逻辑**。

**关键埋伏**：Watts 重伤前看见袭击者背影——穿那双皮鞋。**但 Watts 昏迷，玩家此时不知道他看见了什么**（Loop4 反转伏笔）。

#### 【副怀疑】（供投票候选）

- **Vivian**：袭击 Watts 是她？（Watts 拒签拖她承包，动机直接）——错的方向，但玩家在 L3 几乎必投
- **Watts**：自导自演被袭击博同情？（他想拉拢 Zack 提供 Mickey 档案支持）——错的方向，L4 苏醒指认背影时解除
- **Moore**：仍有金融绑定动机，议事桌上对老 Charles 死越来越紧张

#### 【相互推动】
- **情节 → 证据**：笑面虎仁慈姿态登顶 → 玩家根本不会把"叙事管理"和"杀人"挂钩 → 指证只指向"管理"不指向"凶手"
- **证据 → 情节**：揭穿"管理工具"性质 → 笑面虎裂开第一道缝 → Emma 第一次怀疑赞助人诚意，但仍区分"控制叙事 vs 杀人"

### Loop 4：Emma 的反向指证

#### 【情节】
- Emma 记忆部分回归 + Lula 镇静剂残留化验帮她梳理时间线
- Emma 在议事桌**当众认罪**："是我推了父亲"
- **笑面虎仁慈姿态极致**：小 Charles 立刻表态"我们家族不追究 Emma"——玩家以为案件结束
- Zack 不接受。他要"指证" Emma 自己的认罪——**NDC 史上第一次反向指证**

#### 【案件证据】

**指证对象**：Emma（**反向指证**——击穿她自己"我杀了父亲"的认罪）

**Emma 认罪的 3 层谎言**：

| 层 | Emma 说的 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 我推了父亲致死 | 物证：镇静剂残留浓度（力气只有 1/3） | 物证：颈部勒痕角度（Emma 体位达不到） |
| L2 | 父亲死时我在场 | 人证：Lula 21:58 听见呻吟（Emma 21:55 已离开） | 时序：Foster 死亡时间 22:00-22:10 |
| L3 | 我手上的血是父亲的 | 物证：血迹检测——Emma 自己鼻血（镇静剂副作用） | 物证：录音机外壳同源痕迹 |

**击穿后真相**：Emma 推了父亲，但**没有杀死他**。父亲倒地呻吟，Emma 失忆离开。**有人在她离开后进入书房，完成致命一击**——凶手仍未锁定。

**关键钩子**：小 Charles 在反向指证打断他"替罪"时，表情**第一次失控**——他急于让 Emma 顶罪。**只有玩家通过外部视角看见**，Emma 与 Zack 沉浸在重逢里没注意到。

#### 【副怀疑】（供投票候选）

- **小 Charles**：首次失控钩子被玩家看见——玩家可能在 L4 末投他，但情感上 Emma 与 Zack 仍不接受
- **Vivian**：协议曝光（老 Charles 保险柜里的协议备忘）后，玩家重新读她的动机——她可能为续约杀人？错的，但 L4 投票仍有此候选

#### 【相互推动】
- **情节 → 证据**：笑面虎"替罪"姿态登顶 → 推动 Emma 当众认罪 → 创造"反向指证"的戏剧条件
- **证据 → 情节**：物理证据洗清 Emma → Emma 弧光完成（"你跟父亲一样，但你还没失去你自己"）→ 笑面虎失控钩子被玩家看见（不被角色看见）

### Loop 5：Mary Jones 是谁

#### 【情节】
- 议事 Day5：Emma 把 Frank 1912 资料原件砸上议事桌，议程崩盘

### Loop 6：录音的版本

#### 【情节】
- 议事 Day6：最终签约日，小 Charles 已成议事会实质主持人
- 录音被调取播放
- **笑面虎在自己声音里彻底崩塌**
- 小 Charles 最后陈词——主题落点

#### 【案件证据】

**指证对象**：小 Charles（**阻挠录音调取层面**——与 Loop3 的"叙事管理"是不同谎言）

**小 Charles 的 3 层谎言**：

| 层 | 公开陈述 | 击穿维度 A | 击穿维度 B |
|---|---|---|---|
| L1 | 录音是父亲私人遗物 | 法律：Emma 录音开头采访声明 | 物证：录音机型号属新闻设备 |
| L2 | 录音内容涉及商业机密 | 物证：录音前段无商业内容 | 法律：法庭可封闭播放 |
| L3 | 录音由家族保管 | 物证：录音机已被警方扣押 | 物证：法律所有权属采访者 |

**击穿后真相**：录音可调取播放。

**录音内容（三段戏剧效果递进）**：
1. 老 Charles 亲口讲 1912 对 Sean O'Malley 的谋杀（语气炫耀）
2. Emma 推父亲 → 父亲倒地呻吟 → Emma 离开（与 Loop4 反向指证印证）
3. **门开 → 脚步声 → 小 Charles 的声音"父亲，我来了" → 扭打声 → 闷响**

**笑面虎最后陈词**：

> 「你父亲的事，我父亲做的。我今天不过是杀了一个会毁掉集团的老人。**你以为我没爱过你？我爱过你。我以为我在做对的事。**」

**主题落点**：小 Charles 不是装的，他真心以为自己 16 年都是 Emma 的好赞助人。**笑面虎是真的笑**——系统让坏人也以为自己是好人。

#### 【副怀疑】（玩家进入 L6 时的多嫌疑格局）

L6 不再有投票（终局选择 = 三选项处置笑面虎），**但案件本身还未确定**——L5 末小 Charles 的鞋纹路反击 + 推理链没有铁证，导致玩家进入 L6 时仍有多个 viable 嫌疑人。每人都有具体动机和动线，**只有录音才能定案**。

| 候选人 | 动机 | 动线 | 录音的排除/锁定 |
|---|---|---|---|
| **小 Charles**（真凶）| 法律信托剥夺继承权 / 集团崩溃压力 / 弑父继承 | 议事末他"接待 Emma 来访"作借口可走通往书房的内廊 → 22:00 进书房 | 录音段 3 直接锁定："父亲，我来了" |
| **Moore** | 老 Charles 当晚已暗示要审计他银行黑账 / 他知道账目暴露 = 入狱 | 客房与会计核账 21:30-23:30（3 秘书证人——**但秘书可被收买**；Moore 也是基金董事，可能通过厨房-走廊路线绕道） | 录音段 3 步态/呼吸节奏 ≠ Moore（他膝盖旧伤走路微跛）|
| **Pierce** | 反水可能是演戏——他可能是真凶，假装被骗以转移视线 / 老 Charles 死前可能威胁要清退他 | 全程在走廊（有时间窗口靠近书房）/ 他的"幽会掩护"可能是反向掩护自己 | 录音段 3 嗓音 = 小 Charles 的精炼语，**不是** Pierce 的粗哑嗓音 |

**已排除**：Vivian（L4 Watts"男性背影"）/ Foster（太平间登记）/ Emma（L4 反向指证）/ Watts（自己受害）/ Lula（同盟+时间不对）

**L6 戏剧动作**：小 Charles 在阻止录音调取时，**主动把矛头指向 Moore 和 Pierce**："你们调录音听吧，但你们已经看到推理是多么不严谨——Moore 的不在场证明可疑，Pierce 反水可能是演戏。" 笑面虎在用"开放性怀疑"做最后挣扎。

录音播放 → 三个候选同时被验证 → **只锁定小 Charles**。

#### 【相互推动】
- **情节 → 证据**：议事会进入签约日，所有人在场 → 录音播放达到最大公共冲击
- **证据 → 情节**：录音锁定 1912 + 老 Charles 之死 + 南区罪证 → 笑面虎崩塌 = 侦破链闭合 = 主题落点

## 六、每日结构与投票系统

### 每日 4 段结构

每个 Loop = 每个议事日，分 4 段：

| 时段 | 内容 | 玩家可做 |
|---|---|---|
| 上午议事 | 当日议程 + 戏剧锚点 | 旁观议事桌，观察 NPC 立场 |
| 下午探索 | 1-on-1 私聊 / 搜证 / 太平间复检 | **核心证据收集时段** |
| 晚上公开指证 | 议事桌全员在场 | 提交指证（须先在下午集齐 3 层击穿维度） |
| 深夜投票 | 投"最怀疑谁" | 决定第二天 NPC 关系状态 |

### 公开指证场（议事桌）

- 议事桌是 Unit5 唯一的公开空间——议事 / 指证 / 录音播放都在这里
- 被指证者必须**在全员面前**当场退守，其他 NPC 见证全过程
- 社交压力 = 指证的额外武器；笑面虎必须维持表演，越压力越裂

**关键公开场次**：
- Day4：Emma 当众自认凶手 → Zack 当众反向指证 → 全员见证 Emma 弧光完成
- Day5：Lula 当众被识破护工身份 → 当众转盟
- Day6：录音播放 → 笑面虎崩 → 玩家做终局选择

### 自由探索的 1-on-1 私聊

- 下午时段，Zack 可去任何 NPC 房间私聊
- 私聊 = 玩家收集情报的核心时段；部分 NPC 在公开场不说的话会私下说
- 笑面虎也会主动找 Zack 私聊（如 Day3"我会替 Emma 抗下"是私聊，不是公开发言）
- 私聊解锁可被投票后果影响——前夜投票怀疑谁，第二天他不会跟你私聊

### 投票系统（5 次 + 1 次终局）

| 时机 | 投票内容 |
|---|---|
| Day1-5 夜末 | "最怀疑谁" → 第二天关闭一扇门换打开另一扇门 |
| Day6 录音播放后 | **终局选择**：如何处置笑面虎 → 3 个结局 |

Day1-5 取舍机制：怀疑 NPC X → X 第二天关闭与 Zack 合作 / 但因被怀疑而向他人或玩家释放某条信息。具体在 state 设计阶段细化。

### Day6 终局 3 选项

| 选项 | 性质 | 结局基调 |
|---|---|---|
| 当场报警 | 系统外正义 | 警方介入 / 小 Charles 入狱 / Tidewater 律师团反扑 / 较干净的"法律胜利" |
| Emma 写报道公开 | 舆论正义 | Tidewater 集团形象崩塌 / 反扑空间最大 / Emma 完成 Sean 遗志 / 较脏但有代际传承 |
| 用 Mickey 档案威胁退位 | 系统内正义 | Zack 用 Mickey 的方法 / 集团仍存 / Zack 自己沾色 |

### Day6 终局 × NPC 命运矩阵

玩家的终局选择不只决定小 Charles，**波及所有人**——这是"选择是唯一属于个人的东西"主题的具体体现。

| 选项 | 小 Charles | Vivian | Moore | Foster | Pierce | Watts | Lula |
|---|---|---|---|---|---|---|---|
| 当场报警 | 入狱 / 律师团反扑 | 销毁档案 / 失酒吧 | 银行审计 / 起诉 | 法医执照吊销 | 警局调查 / 解职 | 关键证人 / 法律支持 | 1912 资料交警方 / 公开身份 |
| Emma 公开报道 | 形象崩塌 / 律师挽救 | 沉默苟住 / 失庇护 | 黑账曝光 / 挤兑 | 公开身份曝光 | 警务系统震动 | 法律侧背书 Emma | 资料合 Mickey 档案曝光 / 高风险 |
| Mickey 档案威胁退位 | 退位 / 集团仍存 | **续约新主**（最舒服）| **被 Zack 反工具** | 继续 / 被 Zack 监视 | 续约 / 被 Zack 反掌握 | **退出舞台**（不愿走 Mickey 路）| 沉默归隐 / Frank 资料封存 |

→ 选项 3 是最复杂的——**Zack 沾色**，他用 Mickey 的方法继续在系统里运作。Vivian 和 Moore 都因此活得最舒服，**这反过来让玩家意识到"系统内正义"的代价**：Watts 这种最干净的盟友会拒绝 Zack 这条路。

3 个结局呼应主题"选择是唯一属于个人的东西"——前 5 次投票决定 NPC 关系网，第 6 次决定故事终局。
