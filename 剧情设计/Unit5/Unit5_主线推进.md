# Unit5（第五章 · 晚宴与裁决）主线推进文档

> 配套 v3 底本《主线故事全貌》§14（序幕：真相作为武器）/ §15（晚宴与嫁祸）/ §16（尾声：城市继续）。本文档只讲"这一章把主线推进到哪一层"，不复述案件流程。
> 每个信息点标三层：
> 【背景】设计者知道、玩家本章**拿不到** ｜ 【暗示】玩家能瞥见碎片、**不足以下结论** ｜ 【明示】本章玩家**必须拿到**的揭露
>
> Unit5 是终章，分三段推进：**序幕（晚宴前 Emma 找小 Charles）→ 晚宴（当众质问 + 父子密谈弑父嫁祸）→ 尾声（结案后城市继续）**。本章把前四章全部主线债务一次清偿，几乎全开——所以三层标签里【背景】最少，大量信息升到【明示】。

## 一、本章主线定位（进度 100%）

Unit5 是收束章，任务是**坐实前四章所有悬而未决的真相，并把它们一次性闭环进裁决**。Mystery 阶段结束，主线机器被完整拆开摆在台面上。三段功能分明：

- **序幕**：小 Charles 把真相**武器化**——他不隐瞒，而是精准地把 1912 真相喂给 Emma，让她带着完整、真实、被所有人看见的动机出现在晚宴，成为最佳嫌疑人。主线在这里揭示"权力如何把真相本身变成嫁祸工具"。
- **晚宴**：慈善晚宴变案发现场。Emma 当众质问 → 老 Charles 缓兵 → 宾客散去后父子密谈 → 小 Charles 弑父嫁祸 Emma。四大主线闭环（金融、Tidewater、Lawson 私生子、Eleanor 法庭证词）在调查中逐一坐实。
- **尾声**：小 Charles 被捕起诉，南区计划进入司法审查暂停，但城市开发逻辑没有死。主题落点——"事情结了，城市继续。"

## 二、各主线模块各推进到哪

### 金融 / 司法线（湖滨信托 / Lakeshore / Moore 闭环）
- 【明示】金融链彻底坐实成完整闭环：**湖滨信托银行制造债务 → Lakeshore Investment Co. 承接资金与控股 → Miller 家族是实际收益方 → 司法系统曾保护过这条链**。
- 【明示】通过 Harold Moore / Lakeshore 账目追问"分赃"：南区综合商业开发计划执行后，世博会选址地段收益如何分配——证明银行不是单独作恶，而是在等开发收益回流。
- （前四章此线是逐层显形：Unit2 银行登场、Unit3 上司法保护、Unit4 下 Miller 是真实投资方。Unit5 是把它落到证据层、说"钱最后进了谁口袋"。）

### Tidewater 执行线（投毒 + 银月会链闭合）
- 【明示】Tidewater 完整犯罪链呈现：**Tidewater 负责地产开发 / Lawson 名义负责收购与项目接收 / Mickey-Whale 负责法律与暴力接口 / 银月会执行地面威胁、灭口与投毒 / Miller 家族幕后控制总体方向**。
- 【明示】Vivian 在物证辅助下指认：**强行推进南区计划、利用与 Tidewater 合作的银月会对南区居民投毒的人，是小 Charles**。Vivian 的指认动机是自保、清算与重新站位的混合，不是"突然变成正义证人"。

### Miller 家族形象线（恩人外壳彻底崩塌 + 父子相残）
- 【明示】慈善晚宴变案发现场：Emma 当众质问老 Charles 关于 1912 父亲之死、关于 Miller 与 Tidewater 的关系。
- 【明示】老 Charles 以"明天发布会、请你演讲"缓兵——他**仍在把 Emma 当公关棋子**用（"主动公开、勇于承担"的形象工具），不是良心发现。
- 【明示】宾客散去后父子密谈：老 Charles 要踩刹车（风险控制，非忏悔），小 Charles 等不了，争吵终点是**小 Charles 弑父并嫁祸 Emma**。晚宴上的公开对峙使 Emma 成为头号嫌疑人——她有最真实、最公开、最被所有人看见的动机。

### Lawson 线（私生子坐实，补强弑父动机）
- 【明示】调查中坐实：**Tidewater 名义总裁 Lawson 是老 Charles 的私生子**。
- 【明示】这补强小 Charles 的弑父动机——不只为掩盖 1912、不只为防计划停摆，而是因为他发现**父亲真正交给 Lawson 的东西可能比给自己的更实在**，而 Lawson 偏偏并不渴望权力（只爱宝石、矿物、神秘学）。小 Charles 争了一生的东西，在 Lawson 手里像别人递来的文件。
- 【明示】终局 Lawson **阻止 Pierce 销毁罪证**，让小 Charles 被捕；随后卸任，把 Tidewater 残余开发接口留给 Vivian。这不是 Lawson 变好人，而是他不愿再替小 Charles 和父亲的机器收拾残局（他知道自己也会被调查、被吞没）。

### Eleanor Foster 线（1912 谋杀真相全揭）
- 【明示】Eleanor 在**法庭上念出 Sean O'Malley 的名字**，陈述 1912 那份假尸检报告的真相。证词完成三件事：① 证明 Sean 不是地陷遇难者，而是被谋杀后抛入事故现场；② 证明 Miller 资助体系曾压迫法医系统改写死因；③ 让 Emma 的父亲第一次从"事故数字"变回一个被点名的人。
- 这是 Eleanor 等了十六年的赎罪——不是轰烈反击，而是在终于该说话时说出被压了十六年的真话。

### 揭露动作（本章四大闭环 + Emma/Zack 情感收束）
- 【明示·四大揭露】① Miller 与 Lakeshore 的金融关系；② Miller 与 Tidewater 的控制关系；③ Tidewater 与银月会的投毒/暴力清退关系；④ 1912 Sean O'Malley 谋杀真相。
- 【明示·情感收束】Emma 在书房差点杀小 Charles，靠**自己曾用来救 Zack 的价值观**停手（"Mickey 把人算成代价，我们不能继承这套算法"）。她停下不是被 Zack 当场说服，而是救过 Zack 的价值观反过来救了她自己。
- 【明示】Zack 赶到，真正完成的不是"阻止 Emma"，而是**接住她停下之后的崩塌**——一句"你停下来了"。这意味着 Emma 没有变成 Mickey，没有把父亲的名字交给复仇，最终选择曝光 / 作证。

### 南区清退进度
- 【明示】Emma 闯入、老 Charles 之死、小 Charles 被捕、Vivian 指认、Eleanor 作证，使未完成程序中的南区居民援助计划进入**司法审查、暂时停止**。
- 【明示·非彻底胜利】部分家庭得以留下，但：已死亡的人回不来；已签的合同未必全部追回；已转移地块仍可能留在复杂法律程序中；世博会与城市开发不会因一个家族倒塌而停止。
- 【明示·尾声】Lawson 卸任；**Vivian 接手 Tidewater 残余开发接口**，成为城市继续运转的不确定承接者（她接的是一份"脏礼物"，既有生路也有污染）。最后一幕：两个穿 Tidewater 西装的人站在 Lakeshore 大楼外，平静地谈南区收购条款。

## 三、防提前破梗清单（本章纪律）

本章主线几乎全开，已无后续章节需要保密的真相。但仍须守住两条边界：

- **本文档是主线层文档，不剧透案件层指证解法 / 凶手揭破的具体节奏**——"小 Charles 弑父嫁祸 Emma、玩家如何一步步推翻嫁祸、每轮指证用哪件证据打哪个论点"的节奏由**案件层《Unit5_大纲_0601_重写版》推进**。本文只声明主线要在本章坐实哪些真相，不规定它们以什么破案顺序、什么指证手法被玩家拿到。
- **命名别混（三实体 + 城市名）**：
  - 蓝月亮 = 酒吧 ｜ 银月会 = 黑帮组织 ｜ 银月 = 赌场——三者不同，不可互串。
  - 城市名一律"艾斯弗德"。
  - Tidewater = 潮汐开发公司 ｜ 湖滨信托银行（行长 Harold Moore）≠ Lakeshore Investment Co.（Miller 隐藏金融的上层投资壳）。
  - Lawson Vanderbilt = Tidewater 名义总裁兼老 Charles 私生子 ｜ Mickey = Whale ｜ Eleanor Foster = 法医（1912 假尸检见证者）｜ Sean O'Malley = Emma 父、1912 被谋杀 ｜ Pierce = 听命于 Lawson 的人 ｜ Vivian = 被 Webb 利用→证人→接手 Tidewater 残余接口 ｜ 凶手 = 小 Charles。

## 四、交接尾声 / 系列收束

Unit5 不向"下一章"交接钩子——它是终章，向**结局**交接。本章把前四章全部主线债务结清：

- Unit1 的"有网"→ Unit5 把网的每条线（金融、地产、暴力、家族）都标出名字与归属。
- Unit2 的银行、Unit3 的司法与地图、Unit4 的 Whale 身份与三角结构，全部在本章落到证据层并进入裁决。
- 情感线收束：Emma 没有变成 Mickey；Zack 拒绝 Patrick / Mickey 式的远方大义，选择保护眼前的人；Eleanor 补上十六年的真话；Mickey 的赎罪与算法以"留下证据也留下伤口"作结。

系列落点不是"正义彻底胜利"，而是"**事情结了，城市继续**"：Miller 倒下，但把土地、债务、慈善、法律与暴力连接起来的城市逻辑没有消失。Vivian 带着伤口、资源和一份脏礼物留在城市里——她最终会把南区变成什么样，是结尾留下的不安。
