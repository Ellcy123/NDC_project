# Unit 5 大纲 v0.4 — 舞台 + 夹层暗门版

## 标题：系统没有主人 — 笑面虎与他建造的牢笼

**时间**：1928 年 12 月 8-13 日（南区清退完成第 3 天 → 第 8 天）
**结构**：暴风雪山庄密闭 / 6 天议事 / 6 Loop 调查 / 每日投票
**核心命题**：系统让坏人也以为自己是好人

**与 v0.3 的差异**：案件场景从"书房楼梯命案"改为"晚会舞台命案 + 老 Charles 私人夹层暗门"。案发夜真实顺序、Emma 洗清方式、凶手锁定路径、关键物证链全部重写。但 Tidewater 主题 / 笑面虎弧光 / 暴风雪山庄四重锁 / Emma 嫌疑曲线等主框架保留。

## 阅读说明

每个 Loop 分三栏：

- **【情节】** 戏剧推进 / 角色弧光 / 主题渗透
- **【案件证据】** 指证对象 / 谎言层级 / 击穿维度（具体物件、台词、动线放 state 文件）
- **【相互推动】** 情节如何变成证据 / 证据如何打开情节

两条线不是平行的——**击穿一个谎言 = 释放一段戏剧**，**一段戏剧 = 暴露下一个谎言的破绽**。

### Tidewater 集团状态

- 南区清退完成（11 户 11 人死，含 Mrs. O'Hara），Mickey 死 3 天
- Zack 手握 Mickey 移交的 Tidewater 集团 30 年完整档案

### 议事桌的真实意图

不是商业会议，是 **1912 港口分赃模式的 1928 翻版**。老 Charles 用同一套"清退—合作—洗白"逻辑，把血染地块洗成合法资产。

### 庄园建筑设定（**新增 — 案件结构基础**）

Miller 家族在芝加哥郊外的私人庄园。本次议事会期间所有 NPC 都住在庄园里，暴风雪封路 5+ 天。

**建筑核心**：

| 区域 | 功能 |
|---|---|
| **大议事厅** | 6 天议事在此召开 |
| **礼堂 / 小晚会厅** | Day1 晚开幕晚会，**带舞台**（含旋转式入场楼梯——演员/演讲者从二层夹层后台经楼梯亮相） |
| 老 Charles 书房 | 庄园二层，私人空间 |
| **老 Charles 私人夹层（关键）** | 老 Charles 自建的秘密夹层，**连通他的书房与礼堂舞台后台**。这是他多年来"无需公开露面就能在私下处理事务"的私人通道。**除小 Charles 外，全庄园无人知道夹层存在** |
| 各 NPC 客房 | 二层走廊东西两翼 |
| 仆人区 / 厨房 / 储藏 | 一层后部 |

**夹层暗门的物理设定**：
- 书房一侧：壁炉旁挂画后的暗门
- 礼堂舞台一侧：**舞台旋转楼梯上方的二层暗门**——平时楼梯转到正常位置时，暗门外刚好对接楼梯顶端；楼梯转开 = 暗门外是 4 米空中（**这是本案物理机关的核心**）
- **暗门外侧机关（关键）**：礼堂二层一侧的门框上嵌有一颗**Miller 家族纹章宝石**（嵌在金属卡座里），按压宝石 → 触发暗锁 → 门开启。从夹层内侧推门则不需要宝石（机关只对外侧有效）。**机关接受任何家族纹章宝石按入触发**——和庄园主门的家族钥匙制度一致——但卡座里默认嵌的那颗是老 Charles 私藏的、**无公开编号**的特殊宝石

**为什么没人知道夹层**：老 Charles 30 年前秘密建造，连小 Charles 都是 3 个月前发现继承权被剥夺后偷查父亲行踪时撞破。**小 Charles 知道这是父亲的死穴 — 既是物理弱点，也是心理证物**（夹层里还藏着老 Charles 的私人罪证）。

### 旋转楼梯的当晚状态（**新增 — 案件物理触发**）

礼堂舞台的旋转楼梯**当晚因为晚会演出准备被技师移动过**——为了让 Emma 演讲时从"舞台正面"上场而不是"二层入口"，21:00 前由庄园仆人/技师把楼梯转到了非常规位置（B 位置），并在礼堂后台留下了**晚会准备清单 / 技师交接笔记**，明确记录了这次调整。

Emma 下午做过演讲彩排（17:00 左右），所以她也亲眼看见过楼梯在 B 位置。

这意味着案发时间窗（21:30-22:00）内：
- 老 Charles 从夹层书房一侧入口进来后，想从舞台一侧暗门逃出 → **暗门外没有楼梯承接 → 4 米踩空跌落到舞台**

**L3 玩家如何发现"楼梯被人调回"**：① 礼堂后台找到晚会准备清单（环境物件）→ ② 追问 Pierce（他 21:15 巡查时楼梯在 B 位置）→ ③ Emma 记忆恢复后回忆下午彩排（L4 后期补强）→ 现场看到的楼梯却在 A 位置（常规）→ **有人在 21:15 之后把楼梯调回了 A 位置**。

### 小 Charles 的杀机与原计划

- 3 个月前他发现父亲用法律信托剥夺他继承权
- 议事会是他动手的最后窗口期
- **他的原计划设计**（不是临时起意，下面"案发夜时间线"详细展开）：
  1. 给 Emma 茶/酒里掺镇静剂，让她演讲前后失意识
  2. 提前与老 Charles 约定"晚会开始前在舞台二层见面"（理由是商议演讲对 Emma 的羞辱安排）
  3. 等老 Charles 按约来舞台 → 刺杀
  4. 把昏迷的 Emma 摆在尸体旁，伪装"Emma 推楼梯 + 补刀"
  5. 把旋转楼梯调回常规位置（强化"楼梯事故"假象）
  6. 礼堂所有门从内部反锁
  7. 通过夹层暗门离开 → 回到书房 → 制造"自己一直在书房处理文件"的不在场证明

### Miller 家族宝石钥匙制度（**新增 — 开篇演示场景**）

Day1 各方陆续抵达庄园时，**小 Charles 在庄园主门口当众用他随身的宝石钥匙开门**，向 Zack / Vivian / Moore / Foster / Watts / Emma / Lula / Pierce 等所有宾客演示这套机关：

- 每位 Miller 家族正式成员都有一颗**家族纹章宝石**作为庄园主要门户的开锁件
- 宝石按入金属卡座 → 触发暗锁 → 门开启
- 小 Charles 的台词："这是父亲三十年前的设计——他相信家族里每个人都该有'只属于自己'的钥匙。每颗宝石都有自己的编号和印记。"
- **每颗公开登记的宝石都有个人编号**——丢失、交付、转手都能溯源

**这条设定的双重作用**：

1. **公开知识层**：全员都看见过宝石的外形 + 知道这是 Miller 家族的机关件 → L4 玩家看到血迹里的椭圆空白时，能自然联想到"这是某种宝石压过的痕迹"
2. **隐藏知识层**：礼堂舞台二层的夹层暗门外侧机关嵌着**一颗没有公开登记编号的特殊宝石**——老 Charles 私藏的、专为夹层做的，带家族纹章但无个人编号。除小 Charles 外无人知道它的存在和位置

**这条设定如何为案件锁凶服务**：
- 普通人捡到一颗带 Miller 纹章的宝石，第一反应会以为是哪位家族客人遗失（无威胁感）
- **但只有知道"夹层暗门也用了这种宝石机关"的人，才会立刻意识到"这颗宝石来自暗门，必须藏走"**
- → 小 Charles 拿走宝石的动作 = 他认出宝石来源 = 他知道夹层

## 二、案发夜正向时间线（**全新章节 — 案件核心**）

### 17:00 Emma 演讲彩排

- Emma 下午在礼堂做演讲彩排
- 此时旋转楼梯已被技师调到 B 位置（非常规，便于 Emma 从舞台正面上场）
- Emma 亲眼看见过楼梯在 B 位置——这条记忆 L4 后期会回收

### 19:00-20:30 议事 Day1 开幕

- 老 Charles 主持议事会开幕仪式
- 当面羞辱 Emma "我们养了你"
- **Emma 全程随身带着记者速记本 + 钢笔**——她以"采访"名义记录所有公开发言，这是她进庄园前就向小 Charles 申请到的"复仇前的程序正义姿态"
- 宣布晚会议程：21:30 礼堂晚会，Emma 上台演讲——**这是老 Charles 设的局，要她在芝加哥半个上流社会面前公开"答谢 Miller 家族"**
- 议事散会前老 Charles 的三场私下戏（不变）：
  - 与 Watts 私下谈话——拉拢/施压（要他签法律备案）
  - 与 Moore 谈账目——让 Moore 紧张（暗示要审计他银行黑账）
  - 与 Foster 暗示验尸口径——通过基金渠道下指示（为他自己潜在的"意外"做铺垫）

### 20:30-20:50 老 Charles 单独"采访"Emma（**速记本关键时段**）

- 老 Charles 把 Emma 单独叫到书房 → 公然炫耀 1912 谋杀 Sean O'Malley 的细节 + 揭露 Miller 家族慈善基金如何"养大"Emma → 想看她崩溃
- **Emma 速记下这段对话**——这是后续证明 1912 真相的核心文字证据（替代原录音段 ①）
- 此时 Emma 还没喝下镇静酒，思路清楚，速记完整

### 20:50-21:10 各人准备晚会 + Emma 镇静剂发作

- Emma 离开书房 → 小 Charles 截住她 → 给她演讲前的"安神酒"（掺低剂量镇静剂）→ Emma 喝下后开始头晕、手抖、鼻血
- 镇静剂发作 → Emma 字迹凌乱 → 21:10 后速记本上只有破碎笔画
- Emma 把速记本塞进演讲台底（21:15 左右失意识前最后的清醒动作）→ **这本速记本 L2-L4 才会被玩家在后台搜到，且字迹"前清后乱"的对比本身就是 Emma 镇静剂中毒的物证**
- **同一时段**：老 Charles 回书房，**护工 Mary Jones（Lula）按惯例送每日药品**
- Lula 提前在老 Charles 的日常药里掺了少量安眠药——她要的不是杀，是让老 Charles 昏睡后趁机翻他书房里的私人档案
- 老 Charles 出于多年偏执习惯每次吃药前都会先看一眼药粉颜色和气味 → 发现药粉颜色偏白偏粉 → **他怀疑 Lula** → 他没吃药，假装吃了

### 21:00-21:15 老 Charles 的反应

- 老 Charles 不动声色让 Lula 离开
- 他不去礼堂正常路线，因为他怕"如果护工真要害我，外面肯定还有人接应"
- 他选择**走自己的私人夹层**——从书房壁炉旁暗门进去，准备从舞台一侧暗门出现在礼堂二层
- 这是他多年来的偏执习惯：感到危险就走秘密通道

### 21:15-21:25 Moore 的偶然介入

- Moore 在书房附近走廊徘徊（他想找老 Charles 单独谈账目，希望在晚会前争取一次私下沟通）
- 他**看见老 Charles 神色异常地走进书房，又听到壁炉方向有奇怪的开合声**
- Moore 推开书房门，**看见壁炉旁的画歪了**——他试探着推画，画后是暗门
- Moore 跟进夹层（**他撞见了夹层的存在，但他不知道夹层另一端在哪**）
- 在夹层狭窄通道里追上老 Charles → 想逼谈账目
- 老 Charles 既惊又怒："你怎么也知道这里？" → 转身想从舞台一侧暗门逃
- **Moore 没追到末端**——夹层通道有岔口，Moore 一时分不清方向，原路退回了书房

### 21:25-21:30 老 Charles 的坠落

- 老 Charles 从夹层内侧推开舞台一侧暗门（内侧不需要宝石机关）
- **暗门外是 4 米空中**——旋转楼梯被晚会技师调到非常规位置（B 位置）准备 Emma 演讲
- 老 Charles 反应不及，伸手抓门框 / 外侧机关支撑
- **力学撕扯让外侧机关卡座变形 → 一颗带 Miller 家族纹章的宝石从卡座中崩出**
- 老 Charles 跌到舞台地面——后脑、肩膀、肋骨受伤，眩晕，但**没有立即死亡**
- 那颗宝石静静躺在舞台地面，靠近他身体

### 21:25-21:35 Moore 的退场 / Watts 的走廊片段

- Moore 慌忙从书房暗门退出
- 他听见远处礼堂方向有"砰"的闷响（老 Charles 摔下声），但他**以为是晚会舞台道具事故**
- Moore 快速回到自己客房
- **走廊里他和 Watts 擦肩而过**——Watts 看见 Moore 衣袖灰尘、神色慌张
- **Watts 当时也隐约听见远处礼堂方向的闷响**，但他在找老 Charles 谈拒签，注意力在书房——他记下了这件事，没有在意（L4 苏醒后释放）

### 21:30-21:38 小 Charles 来到舞台

- 小 Charles 按原计划：先把已经失意识的 Emma 从后台抬到舞台
- 进入礼堂 → 看见父亲已经在舞台地面踉跄站着
- **小 Charles 不知道夹层这一端发生过坠落**——他以为父亲按约从二层入口下来了，只是身体不适
- 礼堂光线昏暗（晚会还没开始，舞台灯没全亮）
- 他甚至以为父亲是被自己安排的某种"私下手段"搞晕的
- **他没有改变核心计划**：继续刺杀

### 21:38-21:45 刺杀

- 小 Charles 走近老 Charles
- 老 Charles 此时已经半昏迷，认出儿子，嘴里嘟囔了一句模糊的话
- 小 Charles 拿出预先准备的匕首

## 三、小 Charles 原计划 vs 当晚实际偏差（**全新章节**）

### 原计划（小 Charles 视角，他以为发生的事）

1. Emma 喝下镇静酒 → 演讲前在后台失意识 ✓
2. 老 Charles 按约来舞台见我 ✓（**他以为父亲按约来了，实际上父亲是摔下来的**）
3. 我刺杀父亲 ✓
4. 把 Emma 摆成补刀者 ✓
5. 调楼梯 + 锁门 + 走暗门 ✓

### 实际偏差（玩家最终会还原的真相）

- 老 Charles 没有按约来——他是在 Lula 下药 + Moore 尾随后从夹层暗门**摔到舞台**的
- 老 Charles 已经先于刺杀受过伤 → **死亡时序变成"摔伤在前、刀伤在后"**（验尸可分辨）
- 老 Charles 从暗门摔落时抓住门框 / 撕扯外侧机关 → **暗门宝石脱落在舞台**
- 小 Charles 杀完离开时没注意到那颗宝石（光线 / 血迹 / 急）
- 22:05 他作为"震惊的儿子"赶到现场时才看见 → **在众人围观下偷偷捡走**——这是他全晚最高难度的计划外动作
- 拿走宝石前血迹已经喷溅 → **地面留下椭圆形空白**——成为锁定他的物证

### 关键戏剧讽刺

小 Charles 一直以为"我什么都安排好了"。
**他不知道**：父亲早他几分钟从一个他没料到的方向出现在舞台。
**他不知道**：Lula、Moore 在他不知情的角落里各自推动了案件。
**他更不知道**：自己唯一一个计划外的动作（在众人面前偷偷捡走宝石）反而是唯一能锁定他的物证——而这个动作的高难度恰恰来自他离开时的疏忽。

笑面虎以为自己掌控全局——**他错以为父亲按约来了**，就像他错以为自己 16 年都是 Emma 的好赞助人。**两种"我以为我在做对的事"的错觉，结构上完全相同**。

### 链条 E：暗门宝石 + 血迹椭圆空白（**最强物证锤**）

- L4 钩子：玩家二次勘察舞台 → 注意到血迹中有一个**椭圆形空白** → 联想到开篇小 Charles 演示过的 Miller 家族宝石 → 推测"某种宝石压住过血"

- **谁在 22:05+ 有机会接触尸体附近地面**：第一发现的 Pierce、被叫来的 Foster、闻讯赶到的小 Charles、随后陆续到场的部分宾客 — 但只有小 Charles 有动机

> 「你父亲的事，我父亲做的。我今天不过是杀了一个会毁掉集团的老人。**你以为我没爱过你？我爱过你。我以为我在做对的事。**」

**新版加成**：自白时小 Charles 自己说出"父亲，我来了"那个夜晚的细节——**他描述自己是按约赴会的语气**，不是处理意外的语气。**这反过来证明他是按原计划行凶**，不是临时见到机会下手——**所以这是一级谋杀，不是激情杀人**。这个细节由他自己口中说出比录音播放更有戏剧重量——他终于承认了。

#### 【副怀疑】（玩家进入 L6 时的多嫌疑格局）

L6 不再有投票（终局选择 = 三选项处置笑面虎）。**进入 L6 时，物证链已经指向小 Charles，但仍有 viable 候选**：

| 候选人 | 动机 | 当庭对质如何排除 |
|---|---|---|
| **小 Charles**（真凶）| 法律信托剥夺继承权 / 弑父继承 | 六方证词 + 物证锁链全部指向；最终自白确认 |
| **Moore** | 怕审计 | Moore 自白只到夹层岔口；Foster 反水排除（膝盖旧伤步态不符）|
| **Pierce** | 反水可能是演戏 | Foster 反水排除（嗓音 ≠ 案发现场气场；22:00 破门时已被多人见到位置）|

**已排除**：Vivian / Foster / Emma / Watts / Lula（夹层路线 + 时间不对）

**L6 戏剧动作**：小 Charles 在对质过程中**主动把矛头指向 Moore 和 Pierce**："你们看到推理是多么不严谨——Moore 撞见过夹层入口，Pierce 反水可能是演戏。"笑面虎在用"开放性怀疑"做最后挣扎。

当庭对质 → 三个候选同时被验证 → **只锁定小 Charles** → 他自己崩溃自白。

#### 【相互推动】
- **情节 → 证据**：议事会进入签约日，所有人在场 → 当庭对质达到最大公共冲击
- **证据 → 情节**：六方证词 + 物证锁定 1912 + 老 Charles 之死 + 南区罪证 → 笑面虎自白崩塌 = 主题落点
