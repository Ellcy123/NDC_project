# Unit2（第二章 · 母亲的影子）主线推进文档

> 配套 v3 底本《主线故事全貌》§7。本文档只讲"这一章要把主线推进到哪一层"，不复述案件流程。
> 每个信息点标三层：
> 【背景】设计者知道、玩家本章**拿不到** ｜ 【暗示】玩家能瞥见碎片、**不足以下结论** ｜ 【明示】本章玩家**必须拿到**的揭露

## 一、本章主线定位（进度 30%）
让第二次南区清退的**金融执行轨道第一次显形**。Frank 案不是重点，Frank 案的主线意义是证明：所谓"南区居民援助计划"不是援助，而是清退工具。核心不是"Frank 被某个人害死"，而是"Frank 是一套银行机制压出来的结果"。同时本章把 Unit1 那个飘在黑帮影子里的 Whale，第一次接到一个具体的地产集团上——但**只接到接口，不接到主人**。

## 二、五条主线模块各推进到哪

### 金融 / 司法线
- 【明示】**湖滨信托银行正式登场**，执行"南区居民援助计划"，由行长 Harold Moore 负责。计划对外叫援助，对内是债务陷阱：诱导老人和工人家庭签复杂贷款、用隐藏费用 / 复利 / 违约条款快速扩大债务、骗取或控制房产证明、在目标无力还款时以"合法程序"推动拍卖 / 抵押处置 / 低价转让。
- 【明示】**Harold Moore 首次浮现**——玩家第一次知道银行有个具体的负责人，但他只是"执行层的脸"，不是网主。
- 【明示】**Lakeshore Investment Co. 首次浮现**，但**只是一个可疑投资壳**：玩家知道银行背后还有钱，**不知道钱的主人**。绝不在本章把它和 Miller 连起来。
- 【背景】Lakeshore 真正的功能是 Miller 控制金融与地产的中转结构，让 Miller 不必出现在任何贷款、房产、司法文件上。这一层本章**完全拿不到**。
- 【背景】Harrison 等司法腐败节点正在保护这条链，但司法线本章**不出现**——留到 Unit3 上。

### Tidewater 执行线
- 【明示】**银月会与 Tidewater 的 Whale 关联确定**：银月会俱乐部调查让玩家得知，银月会最近的合作对象是 Tidewater 集团的 Whale。这一步把 Unit1 那个"黑帮影子 Whale"**推进为"地产集团接口"**——Whale 不只是黑帮，他长在一个地产集团里。
- 【暗示】Mickey 以律师身份帮 Zack，游说 / 迫使 Harold Moore 让步，使 Zack 拿到 Frank 与银行腐败有关的材料。表面上 Mickey 是"帮忙的人"。
- 【背景·暗层】Mickey 能让银行让步，**不是因为他"特别能说"，而是因为他在 Tidewater / Lakeshore 体系中有真正位置**；他真正在做的，是控制这次调查能看到什么、不能看到什么。这一层本章绝不点破，只让玩家**事后回看**才懂——本章他必须显得像个尽心帮忙的童年朋友兼律师。
- 【背景】Tidewater 是清退土地的最终接收方。这层本章不推进，留到 Unit3 结局"地产接收方显形"。

### Miller 家族形象线
- 【明示】**恩人形象加深**——本章是 Miller 家族最"像好人"的阶段之一。Emma 看到的是：Miller 资助 1912 年受伤工人家属；Miller 资助医院、设特殊事故基金、**间接援助 Margaret**；Miller 资助法医学，**法医 Eleanor Foster 的职业路径与 Miller 体系有关**；Emma 自己靠 Miller 奖学金完成教育。
- 【明示·本章的张力点】这些善意必须真实可信，因为正是它们要在本章撑起第一个主线张力：**同一个社会声誉系统，一边救 Margaret，一边让湖滨信托银行逼死 Frank；一边养大 Emma，一边把 1912 年的血藏起来。**玩家此刻不能下"Miller 是坏人"的结论，只能感到一种说不清的别扭。
- 【背景】Miller 是整张网的顶端、是 Lakeshore 背后的真实投资方。本章连影子都不能正面给——Miller 家族成员本章**不登场**（首次登场是 Unit3 上的小 Charles）。

### 揭露动作（本章的"显影"）
本章要让玩家牢牢掌握三件事：
- 【明示】① 湖滨信托银行不是普通银行，而是**针对南区居民设计债务陷阱**——由 Frank 案曝光。
- 【明示】② Lakeshore Investment Co. 是**银行背后更高一层的资金壳**（但只到"壳"为止，不到主人）。
- 【明示】③ Whale 与 Tidewater / 银月会的关系**比普通黑帮深**。
- 【暗示】这三件事拼在一起，让玩家第一次模糊感到"银行—资金壳—地产—黑帮"之间有一条线，但还远不能连成结构，更指不到任何一个家族。

### 南区清退进度
- 【明示】**第二次清退的金融执行轨道启动**。玩家第一次看见清退的"手段"——合同、贷款、房产证、债务。
- 【明示·玩家感知】此时机器**主要靠纸面运转**，不需要大规模杀人。清退从 Unit1 的"远处低鸣"变成"看得见的一只手"，但这只手还戴着"援助"的手套。
- 【背景】清退的范围、目标名单、区域地图本章仍不展示（地图要到 Unit3 结局才显现）；目标是 1912 年事故家庭及后代这一层也**拿不到**。

## 三、防提前破梗清单（本章绝不能让玩家知道）
- **Mickey = Whale**（留 Unit4 下）——本章他必须只是帮忙的律师 / 童年朋友
- **Miller 是幕后总指挥 / Miller 是 Lakeshore 与 Tidewater 背后的真实投资方**（留 Unit4 下）——本章 Miller 只有恩情
- **Lakeshore → Miller** 的连线（本章 Lakeshore 只到"可疑投资壳"为止，主人不明）
- **Lawson 是老 Charles 私生子**（留 Unit5）
- **银月会 / Silver Moon 投毒、O'Hara、大型物理清退**（留 Unit4→Unit5）
- **1912 年是 Miller 制造的第一次清退 / Sean O'Malley 是被谋杀后抛尸**（结构真相留 Unit4，反转真相留 Unit5）——本章 1912 仍停在"港口工伤事故 + Miller 事后资助"的大众认知层
- **Harrison / 司法保护层**（留 Unit3 上）
- **南区清退区域地图、目标名单（1912 家属及后代）**（留 Unit3 结局）
- 命名别混：**蓝月亮**=酒吧（Unit1 案发地）｜ **银月会**=黑帮组织（Whale 背后）｜ **银月**=赌场——三个不同实体，不混用；城市名一律**艾斯弗德**

## 四、交接下一章（Unit3 上）的钩子
本章把"银行—Lakeshore 壳—Whale/Tidewater 接口"摆上桌，但留了三个没回答的口子，正是 Unit3 上的开门砖：
- **银行作恶为什么没人管？**——指向 Unit3 上的司法保护层（Harrison 腐败、保险操作进入法庭记录）。
- **Lakeshore 这个壳后面的钱到底是谁的？**——本章只到"可疑"，Unit3 上把它推进为"司法腐败的资金来源"，仍不揭 Miller。
- **Whale 接在 Tidewater 上，那 Tidewater 又是谁的？**——本章只接到接口，地产接收方与 Miller 家族成员登场都留给 Unit3。
- **Miller 的恩情张力**已经埋下——Emma 对 Miller 的感激越真，后面崩塌越重；本章只负责把这份恩情写到玩家也愿意相信。
