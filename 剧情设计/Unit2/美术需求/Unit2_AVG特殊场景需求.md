## §1 · L1 开篇 Margaret 鞋坊火灾现场（2001）

**类型**：opening_cutscene
**触发**：游戏开场自动进入 L1 开篇；先于 L1 自由探索（限时搜证）
**衔接**：本段结束后 Zack 进入鞋坊内开始 5 分钟限时搜证

**叙事**：Zack 接到匿名通报赶到 Margaret 鞋坊火灾现场。Morrison 已在场夹烟站着，冷淡告知"死者应该就是 Margaret"。焦味未散、玻璃碎片散落，卧室门半掩，门内可见盖白布的身影。Morrison 给出 5 分钟搜证窗口——尸体马上就要被抬走。L1 调查的起点。

**在场角色**：Zack、Emma、Morrison、3-4 名警员（背景哑角）

**美术资产**：火灾后鞋坊店铺区背景新建（焦黑/烟雾/白布身影）；Morrison 在火灾现场的环境光重打（Unit9 旧立绘需适配）；Emma 紧跟 Zack 进门的姿势（Unit2 新建立绘项）

## §2 · L2 开篇 圣心医院 Mickey 首次登场（2006）

**类型**：opening_cutscene
**触发**：L1 Expose Morrison 完成后自动接 L1 末过场（事务所 Mickey 电报，已在《突发事件》#1），玩家进入 L2 时自动进入本场
**衔接**：本段结束后 Mickey + Zack 出发去 湖滨信托银行（→ 自由探索）

**叙事**：圣心医院 Margaret 病房——Margaret 床上昏迷、监护设备运作。Mickey Donnelly 以 Frank Kowalski 代理律师 + Margaret 救助者双重身份正式登场，自我介绍 + 透露 Frank 原本约定今日来律所检举银行的关键背景。Emma 进门碰头，三人分工：Mickey + Zack 查银行，Emma → 法医 + 医院。

**在场角色**：Zack、Mickey、Emma、Margaret（昏迷）
**中途出场角色**：Emma（碰头确认分工）

**美术资产**：Mickey 立绘新建（西装坐姿 + 起身站姿两套，沉稳笑容带 Whale 伪装的微妙余韵）；圣心医院病房背景新建；Margaret 昏迷躯体立绘（侧脸闭眼、纱布缠头）

## §3 · L3 开篇 市政厅拒信（2016）

**类型**：opening_cutscene
**触发**：L2 Expose Leonard 完成 + Zack 持 Leonard 假信封 → L3 自动进入本场
**衔接**：本段结束 Zack 转身回 湖滨信托银行门口 → 衔接《突发事件》#4 Mickey 银行门口登场

**叙事**：芝加哥市政厅大门石阶。Zack 站在台阶下递出 Leonard 给的"帮忙"信封，警卫扫一眼直接摆手"这不够，需要有效律师签名"，连门都不让进。Leonard 的伪善被立刻验证为陷阱，调查陷入僵局——Zack 转身回银行门口想其他办法（衔接 Mickey 登场动态漫画）。

**在场角色**：Zack、市政厅警卫（哑角）

**美术资产**：市政厅外景背景新建（1920s 芝加哥市政厅大门石阶 + 远景城市轨道电车/烟囱）；警卫立绘哑角新建（俯视姿态 + 制服严整）

## §4 · L3 指证后 Moore 贵宾室钥匙交付（2012）

**类型**：post_expose_cutscene
**触发**：L3 Moore Expose 三轮全部击穿后自动触发
**衔接**：本段结束 Zack 持一把"不知是哪里的"旧铁钥匙离开，L3 收束 → L4 自动进入 §5

**叙事**：Moore 在 Mickey 法律威胁下主动妥协，坦白 Frank 贷款真相（信用贷款 + Leonard 代签）+ 银行与 Danny 的私下交易（停催债 + 拆迁分钱承诺）。允许查看（不交付）Frank 房产证明 2307；Moore 翻开房产证时，文件夹层里掉出一把旧铁钥匙（2308），他解释"Danny 一并交来的——自己也说不上是哪里的，只说是从 Frank 抽屉里拿来"。一把"不知是哪里"的神秘钥匙，是 Unit2 第一重叙事反转的视觉锚点。

**在场角色**：Zack、Mickey、Moore

**美术资产**：Moore "失态"新立绘表情（傲慢褪去 + 一种"被自己也吓到了"的疲惫）；房产证明 2307 道具特写帧；旧铁钥匙特写（铁制粗齿，从房产证文件夹层里掉出的瞬间）

## §5 · L4 开篇 圣心医院战术碰头（2006）

**类型**：opening_cutscene
**触发**：L3 Expose Moore + post_expose 完成 → L4 自动进入本场
**衔接**：本段结束三人分散——Zack 用 2308 钥匙开鞋坊地下室（2010）开始 L4 自由探索

**叙事**：圣心医院 Margaret 病房——Margaret 仍昏迷，床头花瓶里换了新鲜花。Mickey 在椅子上坐着翻文件，简短点拨调查路线（地下室 → Lula → Danny）。Emma 端两杯咖啡进来，补一句医院 Miller 资助的轻描淡写（伏笔轻放，Unit11+ 才回收）。三人对视点头分散：Mickey → 法律事务；Emma → Margaret + Foster；Zack → 追 Lula 真相 + Danny 指证。

## §6 · L5 开篇 Frank 家主屋小铁盒 + 律师/侦探分支（2018）

**类型**：opening_cutscene + branch_choice
**触发**：L4 Expose Danny + post_expose 完成 → L5 自动进入本场（动态漫画段"Lula 取床后密码锁铁盒 + 玩家答密码"已在《突发事件》#6）
**衔接**：分支选"继续当侦探" → Zack 推进 L5 自由探索；选"当律师" → 触发律师坏结局支线（本版本暂不做）

**叙事**：承接《突发事件》#6——Lula 把铁盒从床缝里取出放在桌上，玩家用对话选项答出密码 1923（玫瑰戒指 "F & L 1923" 即线索；答错 Lula 失望但允许无限重试）。密码答对后 Lula 拨动密码盘开盒——里面是 Frank 攒了八年的银行举报材料 + 给 Lula 的情书。Lula 第一次捧读 Frank 八年的告白，恳求 Zack 和 Mickey 帮她查清 Frank 死亡的真相。Mickey 双手捧起举报材料表情沉重"这些材料我拿回去打官司"，公文包打开 → 材料放入 → 两个铜扣合上 → 2502 从玩家背包移除。Mickey 转向 Zack 提邀请："跟我一起回事务所，当律师能让你做更多事"——画面冻结，玩家二选一界面浮现：[当律师 / 继续当侦探]。

**在场角色**：Zack、Mickey、Lula

**美术资产**：密码锁小铁盒道具特写（铁皮 + 四位数密码盘，已在《突发事件》#6）；**密码对话选项 UI**（4 个年份按钮 + Lula 失望/欣慰反应立绘切换）；Mickey 公文包合扣特写（棕色皮制 + 两个铜扣，跨 Unit 视觉回收点）；**Lula 捧读情书 + 恳求的姿态新立绘**（L4 隔窗举证那种克制气质 → 本场转为情感主动的表达层）；**律师/侦探分支选择 UI**（系统级界面，律师坏结局支线暂不做）

**类型**：transition_cutscene
**触发**：L5 Vinnie Expose 招供后自动触发
**衔接**：本段结束直接进入 §8 圣心医院 Margaret 苏醒

**叙事**：Silver Moon 赌场外——Vinnie 戴铐被警察押上警车，缓缓驶离。Zack 和 Emma 觉得事情似乎告一段落，前往电话亭打电话给 Foster 报喜 + 顺便询问母亲状况。Foster 在电话另一端语速急切，告知双重消息：Margaret 苏醒了，让 Zack/Emma 来医院一趟——**Frank 似乎并不是因为火灾被烧死，很可能是和某人博弈后的凶杀**。"Frank 不是烧死的"那一瞬的反应特写已在《突发事件》#7。

**在场角色**：Zack、Emma、Vinnie（前段被押上车）；Foster（电话另一端，仅声）
**中途退场角色**：Vinnie（被警察押走后离场）

**美术资产**：Vinnie 戴铐上警车特写（街角夜景 + 警车 + Vinnie 涣散眼神望向桌上空酒杯感的画面延续）；电话亭外景背景新建（街角夜景 + 路灯 + 电话亭轮廓）；Emma 接电话的两段表情递进（报喜的轻松 → 接收 Foster 双消息后的震惊）
