## 事件 1 · L1 事务所电报惊悚

**Loop / 场景**：L1 / 2005 Zack 侦探事务所
**类型**：🔵 插入剧情（嵌在 L1 Morrison Expose 完成后回事务所的过场起始）
**触发条件**：L1 Morrison post_expose 完成 → Zack 回事务所准备整理思路 → 推门进事务所瞬间桌上一张电报浮现（实际是助手 Emma 在 Zack 出门时收到的，但 Zack 不知道）

**叙事作用**：
- 第一次出现"Mickey"这个名字——为 L2 末/L3 开篇 Mickey 登场埋名字伏笔（玩家第一次听到这个律师名）
- 第一次确认"Margaret 还活着 + 在医院"——把 L1 已知 Margaret 被打晕 + Mickey 救人这条隐藏线索锚定
- 通过"现成的电报放在桌上"营造调查节奏的被推动感——下一步去医院不是 Zack 主动选择，是被卷进去
- 关键设计：电报内容简洁干硬（1920s 风格 STOP 标点），不写细节，所有"为什么 Mickey 救她 / Mickey 是谁"的悬疑留到 L2

**在场角色**：Zack（独自一人，Emma 暂时不在）

**美术资产**：待美术设计
- 关键帧 1：Zack 推开事务所木门那一瞬，仰角 + 进门视角，桌上一张折叠电报独自放在烟灰缸旁，台灯刚好打亮电报的折痕
- 关键帧 2：Zack 走近、戴上眼镜（如有此动作），手伸过去翻开电报——特写电报正面的 1920s 等宽打字机字体
- 关键帧 3：电报内容大字呈现，玩家可读关键字：`MARGARET ALIVE` / `SACRED HEART HOSPITAL` / `M. DONNELLY`
- 视觉重点：电报折痕 + 打字机字体的机械感 + "M. DONNELLY" 落款位置略大
- 画面外：事务所暖色台灯 + 窗外路灯 + 远处城市夜噪

**音效**：
- 事务所门推开的木门吱呀声 + 关门后的瞬间静默
- 翻开电报的纸张折痕"啪"轻响
- 阅读时 Zack 心跳节拍（低频 0.5s × 2 拍）
- 远处城市夜噪（路面汽车声 / 远处犬吠）保持环境感

**台词参考**：本段无人物台词——电报是物件揭示，Zack 只在末尾低声自语一两字（最终以草稿为准），如：
> Zack："……Margaret 还活着？"
> Zack（看落款）："Donnelly……谁？"

## 事件 2 · L2 Tony 试探 → Vinnie 口误暴露 Russo

**Loop / 场景**：L2 / 2015 Silver Moon 赌场
**类型**：⚡ 突发事件（进入场景即触发，玩家被动旁听）
**触发条件**：玩家进入 2015 Silver Moon 那一刻自动触发——Tony 和 Vinnie 已经在吧台闲聊，Zack 推门进门时正好赶上 Tony 用"上了岸的那个谁"做引子试探 Vinnie；Vinnie 被自己的防御逻辑出卖，主动报出 Tony 从未提及的名字"Russo"。Zack 全程在画面边缘旁听，未参与对话

**叙事作用**：
- L2 R1 谎言锚的核心物证补强——Moore 已在贵宾室口头说出"Leonard 原名 Russo"（2082001）。Vinnie 在 Tony 完全没说 Russo 的情况下自己跳出来否认这个名字——这是"否认中暴露"的高级反向供词，比直接回答更难辩白（疑点 2201 condition）
- 关键设计：Tony 全程没说"Russo"两个字，只用"上了岸的那个谁"做钓饵——Vinnie 的紧张反应把名字自己说出来。Tony 在 Zack 不在场时也在做这件事——他是真的在帮 Frank 翻案，不只是配合 Zack 调查（Tony 角色弧光"老练酒保眼里有事，嘴上没事"的强化）
- 演出层：Zack 是无意闯入者——他没设置陷阱，只是恰好走对了时间。这种"信息免费送上门"的设计减轻玩家"我必须主动套话"的负担，并让 Tony 的角色形象更具独立行动力

**在场角色**：Tony、Vinnie（吧台主体对话）；Zack（画面边缘旁听，刚推门入场）

**美术资产**：待美术设计
- 关键帧 1：吧台斜下 45° 主体构图——Tony 在画面右侧端着酒擦杯（眼神看着 Vinnie），Vinnie 在吧台前坐姿肩颈紧绷（占画面左 2/3）；Zack 在画面最右后景刚推开酒馆木门，背影 + 半侧脸停在门口（玩家代入的"我刚进来"视角）
- 关键帧 2：Vinnie 那句"都说了，我不记得什么 Russo 了！"**说出口的瞬间**——上半身往前压、声音大半度，眼神有半秒的"刚才我说什么了"——这是这段动态漫画的"啊哈"帧
- 关键帧 3：Tony 笑着收尾"行行行"+ 立刻擦杯转移——Zack 在门口刚好把这句话听全，手仍停在门把上还没完全合上
- 视觉重点：Tony 和 Vinnie 的对话主体 + Zack 在画面边缘的"旁听者"位置感——三人三角形构图但 Zack 不打断对话
- 画面外：背景酒客喧闹、钢琴片段保持常态——突发但不张扬，"事故"全在表情里

**音效**：
- 酒馆门被推开的木门吱呀声 + 远处室内酒客喧闹音淡入（Zack 进场环境感）
- 杯沿擦布的细微摩擦声（Tony 边擦边问）
- Vinnie 椅腿磨地板的"嗤"短促声（被惹毛的本能动作）
- "Russo" 三个字落地的那一刻——背景钢琴音轻微淡下半拍（演出强调点，玩家潜意识会听出"刚才出了点什么"）
- Tony 收尾"行行行"立刻回到常态酒馆音量

**台词参考**（最终以对白草稿为准）：
- Tony（擦杯，随口）："之前上了岸的那个谁，你还记得吗？"
- Vinnie（低头）："……不记得。"
- Tony："你也知道，最近的活也不好干，要是还有门路的话……"
- Vinnie（突然抬头，声音大半度）："都说了，我不记得什么 Russo 了！"
- Tony（笑，立刻收）："行行行。"
- → 证词收集：205△口误："我不记得什么 Russo!"

## 事件 3 · L2 偷听 Danny ↔ Leonard 商量 Frank 房产证

**Loop / 场景**：L2 / 2011 湖滨信托银行大厅（Leonard 办公室门外）
**类型**：⚡ 突发事件（自由探索期玩家走位触发）
**触发条件**：玩家在银行大厅自由探索时走近 Leonard 办公室门口（2014 入口外），门半掩，从缝里偷听到一段不完整的对话——Danny 的声音和 Leonard 的低语；玩家停留几秒后 Danny 推门出来，两人擦肩而过

**叙事作用**：
- L3 post_expose Moore 揭示"Danny 把房产证给银行"的伏笔——L2 玩家此时听见碎片信息，不理解全貌（"房产证"+"催债"+"分钱"几个关键词漂浮），但 L3 揭穿后回想会形成"原来那时候就在做这笔交易"的恍然
- 第一次确认 Danny 跟 Leonard 有私下往来——为 L4 指证 Danny 时"R2 没有动机"的击穿点埋接触链
- 关键设计：玩家**不能听清完整对话**——音频做成半语 + 偶尔清晰几个词的处理。Danny 推门出来时 Zack 还没来得及缩回去，两人短暂对视，Danny 急忙低头快步走开

**在场角色**：Zack（门外偷听）、Danny（门内 → 推门）、Leonard（门内，只闻声不见人）

**美术资产**：待美术设计
- 关键帧 1：Leonard 办公室半掩的门 + Zack 侧身贴在门外墙上的偷听姿势，画面左侧门缝透出一线办公室暖光
- 关键帧 2：透过门缝看见 Leonard 办公桌的局部（指尖压着文件、不见全脸），Danny 的西装下摆和侧脸局部
- 关键帧 3：门突然推开 + Danny 撞见 Zack 的对视瞬间——半秒——然后 Danny 低头快步走过画面左侧
- 视觉重点：门缝里的局部视角（玩家代入感）+ Danny 撞见时的尴尬僵硬 + Leonard 始终未露脸（保留"幕后人"伪装直到 L3）
- 画面外：银行大厅其他职员的常态打字声 / 顾客脚步声——突发但环境保持稳定

**音效**：
- 门缝内的对话用"半透明"处理：低频含糊声 + 偶尔几个清晰词（"房产证"……"催债"……"分给你"……）
- Zack 屏住呼吸的环境静默
- Danny 推门那一刻的金属把手"咔哒"
- Danny 鞋底快步走过大厅地砖的"嗒嗒"渐远

**台词参考**（玩家可清楚听到的关键碎片，其余含糊）：
- Leonard（含糊低语）："……手续……不会再有人去你家……"
- Danny（含糊）："……那房子又不是我的……"
- Leonard（清晰一句）："那就这么说定了，房产证你尽快拿来。"
- Danny 推门那刻撞见 Zack——两人对视半秒——Danny 低头走过：
- Danny（避开眼神，匆忙）："……借过。"

## 事件 4 · L3 Mickey 律师银行门口登场

**Loop / 场景**：L3 / 湖滨信托银行门口
**类型**：cutscene 内动态漫画段（嵌在 L3 opening cutscene_sequence 的 Mickey 登场节点）
**触发条件**：承接 L3 opening 前序（市政厅拒信过场属于 opening cutscene，归《Unit2_AVG特殊场景需求》） → Zack 转身回 湖滨信托银行门口想其他办法时，一个陌生男人走上台阶取出传票递给门卫；转身看到 Zack 后微微一笑说出对方名字

**叙事作用**：
- 童年好友重逢的关键节点——L2 末调查陷入僵局，Mickey 的出现 = 调查转机的视觉化锚点。"绝处逢生"的戏剧反差让玩家记住这一刻 Mickey 的出场分量
- "Whale 伏笔已种"——Mickey 的笑容里有"我等你很久了"的微妙（Unit2 内玩家不会注意，Unit11+ 揭示其真实身份后回看会发现）
- 关键设计：Mickey 不是被叫来的，是自己出现的——专业 + 善意 + 主动登场的三重组合让玩家立刻接纳他作为"我方"。这是 Whale 整套伪装的视觉起点

**在场角色**：Zack、Mickey、湖滨信托银行门卫（背景）

**美术资产**：待美术设计
- 关键帧 1：湖滨信托银行门口 + 一个陌生男人从画面右侧走入台阶，公文包打开取出一叠文件——画面中 Zack 在台阶下侧站着、眼神被新出现的陌生人吸引
- 关键帧 2：男人递文件给门卫的近景——传票上有法院红章可读，门卫一改之前市政厅警卫的冷漠（"这是法院签署的"= 立刻进入接待程序）
- 关键帧 3：男人转身——童年好友重逢的微笑特写，能看清他的脸（Mickey 第一次正面出场）。笑容里要带"专业 + 故人 + 一丝难以察觉的从容"——这是 Whale 伪装的视觉起点
- 视觉重点：Mickey 转身的瞬间表情——专业律师的姿态 + 突然认出 Zack 的微笑，两个表情在一帧内完成
- 画面外：1920s 芝加哥城市街道烟火气——电车、行人、湖滨信托银行金属铭牌

**音效**：
- Mickey 鞋底踩台阶的稳健声（与之前市政厅警卫鞋声形成对照——同样的台阶，不同的人）
- Mickey 公文包打开的皮制扣扣声 + 纸张抽出的摩擦声
- 笑出声前的半秒静默（"Zack Brennan？" 前的停顿）
- 背景：城市街道常态环境感

**台词参考**（最终以对白草稿为准）：
- Mickey（对门卫，专业语调）："这是法院签署的传票，请通知你们行长。"
- （Mickey 转身看到 Zack）
- Mickey（笑）："Zack Brennan？好久不见。我叫 Mickey Donnelly——Frank Kowalski 的代理律师。"

## 事件 5 · L4 Lula 隔窗举证放弃继承

**Loop / 场景**：L4 / 2020 Frank 家厕所侧墙小窗
**类型**：turn_cutscene（嵌在 L4 Danny Expose R3 内）
**触发条件**：R3 Danny 攀咬"Lula 骗签遗嘱"后强制触发——Lula 出现在厕所侧墙小窗外，举起已签好的放弃继承协议给 Danny 看，平静宣告

**叙事作用**：
- L4 R3 击穿件之一——Lula 用一张已签好的协议 + 一句陈述句直接撕碎 Danny "她骗叔叔签遗嘱"的全部攀咬。她已经签好了，不是来谈判的，是来宣告的
- Unit2 第二重叙事反转（Frank 不是赌博老赖而是爱情英雄）的情感高峰
- 关键设计：Lula 出现的位置高于 Danny（小窗在厕所侧墙高处）+ Danny 蹲马桶仰望——空间权力关系直接倒置。Danny 一辈子用偏见和谎言堵 Lula 的嘴，这一刻 Lula 占据高位、Danny 在画面最低点（马桶上）
- "看到了吗 我签了 我不要这笔钱"是 Lula 在 Unit2 内唯一一句**主动陈述**台词（之前她的台词大多是回答 Zack 询问）——把"她从来不想要这笔钱"用一句话锁死，让玩家和 Danny 同时被堵死
- 视角倒置：Danny 的视野被门框 + 窗框双重限制——他只能仰望那张举起的纸和 Lula 的脸（Characters/danny.md 设计备注："被限制的视角本身就是他这个人物的隐喻"）

**在场角色**：Danny（厕所内马桶上仰望）、Lula（小窗外，举证宣告）；Zack/Mickey 在厕所外间（不进画面焦点，仅环境音呼吸）

**美术资产**：待美术设计
- 关键帧 1（约 1s）：仰角机位——Danny 蹲在马桶上的主观视角，看到 30cm×40cm 通风小窗框中突然出现 Lula 的脸 + 上半身。Lula 双手举起一张折开的协议书贴近窗格内侧（让 Danny 能从下方看清纸面），协议末端 `Lula Washington` 签名清晰可读
- 关键帧 2（约 2s）：Lula 视线越过协议看向 Danny——平静、不哭、不愤怒，眼神有一种"事情就到这里"的笃定。仰角让她的脸 + 举着的协议在画面上方占据主导，Danny 在画面下沿挤成一小块的窘态轮廓
- 关键帧 3（约 1s）：Lula 把协议从窗格里递进来，落在内侧窗台上——动作干净利落，不带情绪。Danny 嘴角往上扯了一下又松开，没有任何反驳台词
- 视觉重点：举起的协议 + 签名 + Lula 脸 + Danny 仰望的窘态——四件东西在一帧内构成"权力倒置"的完整图像
- 画面外：Zack/Mickey 在厕所外间的呼吸停顿、远处后巷狗叫一声（生活感保持）

**音效**：
- 窗格被轻推开的细微铁锈摩擦声（Lula 凑近窗外的预备动作）
- Lula 说话时的语调——不大、不冷、不带胜利感，像在交代一件已完成的事
- 说完后的彻底静默 1.5s（Danny 的肠胃声/Zack 的糖声都消失了）——给 Danny 的失语留空间
- 协议落在窗台上的纸张轻响"啪"

**台词参考**（最终以对白草稿为准）：
- Lula（举起协议贴近窗格，平静）："看到了吗。我签了。我不要这笔钱。"
- （Lula 把协议递进内侧窗台）
- Danny（嘴角扯了一下，没说出任何话）："……"

## 事件 6 · L5 小铁盒床缝开启 → 举报材料 + 八年情书显形

**Loop / 场景**：L5 / 2018 Frank 家主屋开间
**类型**：🔵 插入剧情（嵌在 L5 opening 起始）
**触发条件**：L5 自动进入开篇——Mickey 拿着钥匙 2408 在 Frank 家主屋等 Zack；Zack 进屋后两人挪开 Frank 单人床，从床缝里取出 15×10 cm 铁皮小盒，2408 钥匙开锁；盒子打开

**叙事作用**：
- L5 调查的核心驱动——盒内有 Frank 攒了八年的银行举报材料（L5 离场的关键叙事支点）+ 给 Lula 的八年情书（Frank 真感情的物证补强）
- 第一次让玩家见到 Frank 的"私人内心"——之前 Frank 只在他人转述里存在，这一刻通过他亲笔留下的东西第一次"出场"
- 关键设计：盒子开启的瞬间是 Frank 这个无台词角色在 Unit2 内的"现身"——所有遗物的质感（颤抖笔迹 / 八年情书的纸张老化 / 举报材料的密集程度）共同构成他的形象
- "等好了告诉你"的伏笔回收——Lula 在 L4 提到 Frank 案发前说"我在准备一个东西，等好了告诉你"，这盒就是"那个东西"

**在场角色**：Zack、Mickey

**美术资产**：待美术设计
- 关键帧 1：Mickey 弯腰指床缝 + Zack 蹲下挪床的合作动作（两人配合，不是 Zack 单打独斗——为 Mickey 的"帮手"形象加分）
- 关键帧 2：床被挪开后地板缝隙里露出小铁盒的局部（生锈、磨损边角）
- 关键帧 3：Zack 双手捧起铁盒放在桌上的特写——2408 钥匙插入锁孔的近景
- 关键帧 4：盒盖被掀开的瞬间——一叠纸张露出：上层是 Frank 颤抖笔迹的银行举报材料（数页厚），下层是用细绳捆好的八年情书（信封发黄、纸边卷曲），夹在两者之间是给 Lula 的求婚戒指小绒盒（L4 已经获取）的复制品占位 / 不出现 / 由设计补足
- 视觉重点：盒盖打开那一帧的"内容物全景"——让玩家一眼看到"Frank 这一辈子值得保留的东西全在这里"
- 画面外：Frank 单人床的整洁度（已铺好、床头小桌摆着旧眼镜 / 1923 年的家庭照片）

**音效**：
- 床脚拖动木地板的"嘎吱"短促 2-3 声
- 铁盒被捧起的金属磕地"咚"
- 钥匙插入锁孔的"咔哒"+ 旋转的金属转动声
- 盒盖打开的纸张展开"沙"
- 背景：主屋寂静的"无人居住感"——只有窗外远处城市的低频环境声

**台词参考**（最终以对白草稿为准）：

- Zack（挪开床）："Mickey，这个盒子……"
- Mickey（沉默一拍）："打开看看。Frank 准备了很久。"
- （盒盖打开，纸张露出）
- Zack（低声）："这老头……攒了八年。"

## 事件 7 · L5 Foster 电话死因反转

**Loop / 场景**：L5 末 / 2006 圣心医院外走廊过场（或事务所，由开发实际定）
**类型**：🔵 插入剧情（嵌在 L5 Vinnie Expose 结束后的过场起始）
**触发条件**：L5 Vinnie 招供后 Emma 给 Foster 打电话核实——Foster 在电话另一端语速急切地说出 Frank 的死因鉴定结果

**叙事作用**：
- Unit2 最大的剧情反转锚点——Vinnie 招供之前玩家以为"Frank 死于火灾窒息" / Vinnie 招供时声称"我杀的"——Foster 这通电话直接颠覆：**Frank 是钝器击打致死，不是火灾，也不是 Vinnie 当时的描述**
- 把 L5 看似封闭的指证撕开一道口子——Vinnie 在替谁顶？为什么招供细节不对？引擎点火，把玩家直接送进 L6 Leonard 调查
- 关键设计：电话单边对话——玩家只听见 Emma 这一头的反应，Foster 的声音通过电话听筒处理（含糊但能听清关键词）。Emma 的表情从平静 → 震惊 → 倒抽气是这段动态漫画的情绪驱动
- Foster 解释"花了一整天才有初步结论"——为后续 L6 callback 法医鉴定 2601 物证的存在性铺垫（这通电话是口头确认，2601 才是物证落地）

**在场角色**：Zack、Emma；Foster（电话另一端，仅声）

**美术资产**：待美术设计
- 关键帧 1：Emma 站在医院走廊电话亭旁，听筒贴耳，肩膀放松 + 嘴角带着调查告一段落的轻松感——这是反转前的"前置安全感"帧
- 关键帧 2：Emma 听到关键词的瞬间——眉头骤紧 + 肩膀微抬——表情转折的中间帧
- 关键帧 3：Emma 转头对 Zack 张嘴说出"Frank 不是被烧死的"——Zack 在画面对侧倚墙吃糖的姿态僵住，糖在嘴边停下半秒
- 视觉重点：Emma 表情三段递进（轻松 → 震惊 → 报告）+ Zack 那一瞬间糖停住的小动作（呼应 Unit9 Zack 的特征性吃糖）
- 画面外：医院走廊的常态环境感——护士推车经过 / 远处病房铃声——把"突发"压在常态之上更尖锐

**音效**：
- 电话听筒里 Foster 声音的"金属化"处理（1920s 电话音质）：含糊低频 + 偶尔清晰的关键词："钝器""头部""不是烧的""花了一整天"
- Emma 抽气的短促吸气声
- Zack 嘴里糖的轻微"咔"——停住的那一刻
- 背景医院环境音保持稳定（强调"突发"是局部事件，世界依然在转）

**台词参考**（最终以对白草稿为准）：
- Emma（接电话，语调轻松）："Foster？嗨，是我。Vinnie 招供了，我们在等正式手续——"
- （电话另一端 Foster 急切说话，玩家听见含糊声 + 几个关键词）
- Emma（眉头骤紧）："等一下。你说什么？"
- （Foster 重复一遍，玩家这次能听清片段："钝器……头部撞击……不是火灾"）
- Emma（捂住听筒，转头对 Zack）："Foster 说……Frank 不是被烧死的。"
- Zack（糖在嘴边停住半秒）："……什么？"
- Emma："钝器击打致死。Vinnie 的招供——细节不对。"
