**版本**：v2.1 / 2026-06-05
**用途**：美术对接 Unit2 全 6 个 Loop 的突发瞬间，统一以"方格动态漫画"形式呈现的关键事件清单。
**整合范围**：⚡ 突发事件 / 🔵 插入剧情 / turn_cutscene 内动态漫画段。**不含**自由探索期常规搜证、Loop 开篇/收束碰头戏（归《Unit2_AVG特殊场景需求》）、Talk 内证物出示（归对白配置）。

---

## 格式说明

**触发档位**两级：
- **点击弹出格子**：玩家主动点击触发，格子弹出后可阅读，再次点击关闭
- **全动画**：自动触发完整动画序列，无需玩家点击开关；全 Unit2 最多 2 个，仅用于全篇情节重点（本 Unit 为事件 4 + 事件 6）

**美术资产分两层**：
- **AVG层立绘**：标准 AVG 界面里的特殊姿势/道具——展示角色"此刻状态"但通用立绘无法表达的细节（背景场景 + 角色立绘层，格子弹出前即在场）
- **动态漫画格子**：弹出的方格内容——专门展示道具细节、空间关系、动作卡点等 AVG 镜头语言做不到的内容；每格末尾标注 `→ 绑定台词` 一句

## 事件 1 · L1 事务所电报惊悚

**Loop / 场景**：L1 / 2005 Zack 侦探事务所
**类型**：🔵 插入剧情（嵌在 L1 Morrison Expose 完成后回事务所的过场起始）
**触发档位**：点击弹出格子
**触发条件**：L1 Morrison post_expose 完成 → Zack 和 Emma 一同回事务所 → Zack 率先注意到桌上多了一张电报，Emma 顺着他的视线也看到

**叙事作用**：
- 第一次出现"Mickey"这个名字——为 L2 末/L3 开篇 Mickey 登场埋名字伏笔
- 第一次确认"Margaret 还活着 + 在医院"——把 L1 已知 Margaret 被打晕 + Mickey 救人这条隐藏线索锚定
- 通过"现成的电报放在桌上"营造调查节奏的被推动感——下一步去医院不是 Zack 主动选择，是被卷进去
- 关键设计：电报内容简洁干硬（1920s 风格 STOP 标点），不写细节，所有"为什么 Mickey 救她 / Mickey 是谁"的悬疑留到 L2

**在场角色**：Zack、Emma（两人一同返回事务所）

**美术资产**：

**AVG层立绘**（触发前即在场）：
- Zack：推开事务所木门刹那的疲惫 + 警觉混合姿势——视线已被桌上某样东西吸引，手还停在门框
- Emma：紧跟在 Zack 身后半步、顺着他视线看向桌面的侧身姿势
- 场景道具：事务所桌面特写可见——折叠电报独放在烟灰缸旁，台灯光落在电报上

**动态漫画格子**（点击触发，共 3 格）：
- 格 1：仰角 + 进门视角——桌上电报在台灯光圈里独自发亮，烟灰缸旁，折痕整齐；画面边缘是事务所门框 + Zack 手的局部
  → 绑定台词｜扎克·布伦南："等等。桌上多了什么东西。"
- 格 2：Zack 走近伸手拿起电报的手部特写——电报正面 `TIDEWATER UNION TELEGRAM` 等宽打字机字体清晰可读，纸张折痕质感
  → 绑定台词｜艾玛·奥马利："电报？你离开前它在那儿吗？"
- 格 3：电报内容大字呈现，玩家可读关键字：`MARGARET ALIVE` / `SACRED HEART HOSPITAL` / `M. DONNELLY`（落款位置略大）
  → 绑定台词｜扎克·布伦南："圣心医院……三楼。她还活着。Emma，她没死！"

**音效**：
- 事务所门推开的木门吱呀声 + 关门后的瞬间静默
- 翻开电报的纸张折痕"啪"轻响
- 阅读时 Zack 心跳节拍（低频 0.5s × 2 拍）
- 远处城市夜噪（路面汽车声 / 远处犬吠）保持环境感

**台词参考**（台本原文，格绑定台词的来源）：
- 扎克·布伦南 [直起身子，视线扫向桌面，疲惫一扫而空，眼神变得锐利]："等等。桌上多了什么东西。"
- 艾玛·奥马利 [顺着他的视线看去]："电报？你离开前它在那儿吗？"
- 扎克·布伦南 [拆开电报，视线扫过上面的字，瞳孔猛地收缩，声音因为极度的震惊和突如其来的狂喜而颤抖]："圣心医院……三楼。她还活着。Emma，她没死！"
- 艾玛·奥马利 [立刻冲过来，看了一眼电报]："Margaret？！老天，Zack！这是谁发来的？"
- 扎克·布伦南 [死死攥着那张纸，眼眶发红，嘴角却忍不住上扬]："Mickey。我父亲以前的搭档。"

## 事件 2 · L2 偷听 Danny ↔ Leonard 商量 Frank 房产证

**Loop / 场景**：L2 / 2011 湖滨信托银行大厅（Leonard 开放工位区）
**类型**：⚡ 突发事件（自由探索期玩家走位触发）
**触发档位**：点击弹出格子
**触发条件**：玩家进入 2011 银行大厅即自动触发演出帧——Zack 走向 Leonard 工位区时，大厅角落 Danny 正俯身在 Leonard 开放工位前低声交谈；Zack 走近数步旁听，隐约听到碎片信息；Danny 说完直起身离开，转身撞见 Zack，对视半秒后低头快步离场

**叙事作用**：
- L3 post_expose Moore 揭示"Danny 把房产证给银行"的伏笔——L2 玩家此时听见碎片信息，不理解全貌，但 L3 揭穿后回想会形成"原来那时候就在做这笔交易"的恍然
- 第一次确认 Danny 跟 Leonard 有私下往来——为 L4 指证 Danny 时"R2 没有动机"的击穿点埋接触链
- 关键设计：玩家不能听清完整对话；Danny 推门出来时 Zack 还没来得及缩回去，两人短暂对视

**在场角色**：Zack（走近旁听）、Danny（工位前俯身 → 直起身离场）、Leonard（工位内，只闻声不见全脸）

**美术资产**：

**AVG层立绘**（触发前即在场）：
- Zack：大厅内走向工位区的行走姿势，步伐已慢下来（察觉有人在低声交谈）
- Danny：俯身在 Leonard 开放工位前的低语姿势——手撑桌沿、身体前倾、声音压低
- Leonard：坐在工位稍稍前倾，脸始终半掩不正面出镜（保留"幕后人"伪装直到 L3）

**动态漫画格子**（点击触发，共 3 格）：
- 格 1：银行大厅斜俯视视角——Danny 俯身 + Leonard 局部（脸不入镜）；Zack 在中距背影停步旁听
  → 绑定台词｜丹尼："Leo，房产证的事到底怎么说？我已经把叔叔的东西带来了……"
- 格 2：Zack 主观视角——Danny 侧脸（手撑桌沿）+ Leonard 俯身看文件的局部；Leonard 脸始终半掩
  → 绑定台词｜伦纳德："你再仔细想想，真的一点都没提过？这事很重要——"
- 格 3：Danny 直起身转身——发现 Zack 站在身后，两人对视半秒；Danny 低头快步从画面左侧走过
  → 绑定台词｜丹尼："……借过。"

**音效**：
- Danny 和 Leonard 的低语"环境压制"处理：两人凑近的含糊底噪 + 偶尔浮出清晰词（"房产证"……"手续"……"尽快拿来"……）
- Zack 屏住呼吸的环境静默
- Danny 转身发现 Zack 时的一拍静默（所有声音骤停半秒）
- Danny 鞋底快步走过大厅地砖的"嗒嗒"渐远

**台词参考**（台本原文，格绑定台词的来源）：
- 丹尼 [手撑着桌沿，声音压低，语气带着急]："Leo，房产证的事到底怎么说？我已经把叔叔的东西带来了……"
- 伦纳德 [低头翻着文件，没有立刻抬眼]："手续还在走。银行这边现在每笔都卡得很紧，急不得。"
- 丹尼："可我这边真的急等这笔钱用……"
- 伦纳德 [身体微微前倾，声音更低，有些不甘心]："你再仔细想想，真的一点都没提过？这事很重要——他不可能什么都没……"
- 丹尼 [转身撞见 Zack，避开眼神，匆忙]："……借过。"

## 事件 3 · L4 Danny 困厕所·Zack 次卧搜证

**Loop / 场景**：L4 / 2009 Frank 家次卧
**类型**：⚡ 突发事件（次卧搜证自由探索段，玩家翻到婚戒时自动触发）
**触发档位**：点击弹出格子
**触发条件**：玩家在 2009 次卧自由探索时点击床垫下方 → 发现金婚戒（P & M 1901）→ 厕所里传来 Danny 的隔墙嘶喊

**叙事作用**：
- 建立 L4 核心喜剧对位结构——Danny 亲眼看着 Zack 翻出他最烫手的证物，却完全无力阻止
- 确认"Danny 当晚真的在家"的物理反讽锚点——他被困在厕所，出不去
- 为 L4 Expose 期间"气窗缝眼睛"演出做物理铺垫

**在场角色**：Zack（次卧，搜证）；Danny（厕所内，听见但出不来）

**美术资产**：

**AVG层立绘**（触发前即在场）：
- Zack：蹲下翻床垫的搜证姿势，手伸向床垫下凹槽
- 场景道具：厕所虚掩木门——门框上方隐约可见约 15cm 高的过梁气窗缝（1920 年代廉租房通风遗留），气窗缝里有一双眼睛死盯着婚戒方向

**动态漫画格子**（点击触发，共 2 格）：
- 格 1：Zack 从床垫凹槽取出婚戒的手部近景——金戒内圈刻字 `P & M 1901` 清晰；画面中景右侧气窗缝里隐约有一双眼睛锁定那枚戒指（"被看见的秘密 + 无力的目击者"构图）
  → 绑定台词｜丹尼·科瓦尔斯基："喂！外头是谁？！手里头拿着的东西都给我放下！那是我的房间，别乱翻！"
- 格 2：Danny 主观视角——从厕所内仰视过梁气窗缝，可见 Zack 起身将婚戒放进外套口袋的下半身动作；Danny 双手扶门板（出不去），眼睛贴近气窗缝
  → 绑定台词｜扎克·布伦南："你的房间？我还以为 Kowalski 家里进贼了呢。"

**音效**：
- Zack 掀床垫的布料翻动"嘶啦"+ 床板摩擦轻响
- 婚戒落入掌心的轻微"叮"
- Danny 从厕所门板后传出的声音：压低却上扬的"喂！！！"——闷声，字字清晰
- 厕所深处的肚子咕噜声在最紧张叫喊的正后方出现

**台词参考**（台本原文，格绑定台词的来源）：
- 丹尼·科瓦尔斯基 [隔着墙壁，声音发闷，夹杂着冲水声和痛苦的呻吟]："喂！外头是谁？！我听见动静了！手里头拿着的东西都给我放下！那是我的房间，别乱翻！"
- 扎克·布伦南 [把婚戒放进口袋，转头看向声音传来的方向]："你的房间？我还以为 Kowalski 家里进贼了呢。"
- 丹尼·科瓦尔斯基 [声音急了，想冲出来但被身体状况绊住，气急败坏]："你才是贼！……哎哟……这该死的肚子……听见没有，等我出来！"

## 事件 4 · L4 Lula 隔窗举证放弃继承【全动画】

**Loop / 场景**：L4 / 2020 Frank 家厕所侧墙小窗
**类型**：turn_cutscene（嵌在 L4 Danny Expose R3 内）
**触发档位**：全动画
**触发条件**：R3 Danny 攀咬"Lula 骗签遗嘱"后强制触发——Lula 出现在厕所侧墙小窗外，举起已签好的放弃继承协议给 Danny 看，平静宣告

**叙事作用**：
- L4 R3 击穿件之一——Lula 用一张已签好的协议 + 一句陈述句直接撕碎 Danny "她骗叔叔签遗嘱"的全部攀咬
- Unit2 第二重叙事反转（Frank 不是赌博老赖而是爱情英雄）的情感高峰
- 关键设计：Lula 出现的位置高于 Danny（小窗在侧墙高处）+ Danny 蹲马桶仰望——空间权力关系直接倒置
- "现在，Frank 的房子和钱，不归我，不归你，也不归银行"是 Lula 在 Unit2 内唯一一句主动陈述台词

**在场角色**：Danny（厕所内马桶上仰望）、Lula（小窗外，举证宣告）；Zack/Mickey（厕所外间，不进画面焦点）

**美术资产**：

**AVG层立绘**（全动画序列内）：
- Danny：蹲马桶仰望的狼狈姿势——一只手按肚子出汗、仰头角度随 Lula 出现而定格
- Lula：出现在厕所侧墙小窗外的上半身——双手举起一张展开的协议书贴近窗格内侧，眼神越过协议看向 Danny（平静、不哭、不愤怒）

**全动画格子序列**（共 3 格，完整动画节奏）：
- 格 1（约 1s）：仰角机位——Danny 蹲马桶的主观视角，30cm×40cm 通风小窗框中突然出现 Lula 脸 + 上半身；双手举起协议书贴近窗格，末端 `Lula Washington` 签名清晰可读
  → 绑定台词｜露拉："Danny。"
- 格 2（约 2s）：Lula 视线越过协议看向 Danny——平静笃定，"事情就到这里"的眼神；仰角让 Lula 脸 + 协议在画面上方主导，Danny 在画面下沿挤成窘态轮廓（权力倒置的完整构图）
  → 绑定台词｜露拉："现在，Frank 的房子和钱，不归我，不归你，也不归银行。按照遗嘱的附加条款，它们现在归于艾斯弗德工人互助会。"
- 格 3（约 1s）：Lula 将协议从窗格里递进来，落在内侧窗台——动作干净利落，不带情绪；Danny 嘴角往上扯了一下又松开，无反驳台词
  → 绑定台词｜露拉："Frank 留给我的，从来都不是什么房子和钱，而是那五年。"

**音效**：
- 窗格被轻推开的细微铁锈摩擦声
- Lula 说话时的语调——不大、不冷、不带胜利感，像在交代一件已完成的事
- 说完后的彻底静默 1.5s（给 Danny 的失语留空间）
- 协议落在窗台上的纸张轻响"啪"

**台词参考**（台本原文，格绑定台词的来源）：
- 米奇·唐纳利 [在小窗外开口，依然是一副律师的严谨口吻]："Washington 女士，请将你刚刚向我说过的话，再向 Danny Kowalski 先生转述一次。"
- 露拉 [脸出现在窗格里，看向 Danny]："Danny。"
- 露拉 [把纸放下，抬头，语气平的，像陈述天气]："我现在要签署这份放弃继承遗嘱的声明。由 Frank Kowalski 生前委托的律师证明，它具有法律效益。"
- 露拉："现在，Frank 的房子和钱，不归我，不归你，也不归银行。按照遗嘱的附加条款，它们现在归于艾斯弗德工人互助会。"
- 丹尼："不！我的房子！"
- 露拉 [转向 Zack，不再看 Danny]："Frank 留给我的，从来都不是什么房子和钱，而是那五年。"

## 事件 5 · L5 Lula 取铁盒 + 玩家答密码 → 举报材料 + 八年情书显形

**Loop / 场景**：L5 / 2018 Frank 家主屋开间
**类型**：🔵 插入剧情（嵌在 L5 opening 起始）
**触发档位**：点击弹出格子
**触发条件**：L5 自动进入开篇——Zack 和 Mickey 已在 Frank 主屋陷入僵局；门口传来脚步声，Lula 进场，主动说出 Frank 生前交给她一个铁盒；三人围桌，Lula 透露密码是一个年份；玩家用对话选项答出 1923（玫瑰戒指刻字 "F & L 1923" 即线索）；答错 Lula 失望摇头，允许无限重试；答对后 Lula 拨动密码盘，三声咔哒，盒盖弹起

**叙事作用**：
- L5 调查的核心驱动——盒内有 Frank 攒了八年的银行举报材料 + 给 Lula 的八年情书
- 第一次让玩家见到 Frank 的"私人内心"——之前 Frank 只在他人转述里存在，这一刻通过他亲笔留下的东西第一次"出场"
