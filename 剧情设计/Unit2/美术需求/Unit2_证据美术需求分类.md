## 2101 - 尸体手指照片

**重点（信息表达必不可少）**：
1. 主体：一只盖在白布下露出来的老年男性左手特写，画面构图围绕**左手无名指**展开
2. 无名指根部**无任何戒指**——金属、布、绳一概没有
3. 无名指根部皮肤**无佩戴压痕、无肤色色差、无环状凹陷**——肉眼可辨该手指从未长期戴过任何戒指（与"刚摘掉"的手区分开）
4. 手指属于老年男性（关节略粗、皮肤松弛、指甲偏厚发黄）——视觉上可与"老年女性 Margaret 的手"产生明显反差，玩家一看就能起疑
5. 照片右下角可读小字：`Ashford P.D. — Evidence Ref. 11.06.1928` 或同等案号格式（暗示这是 Zack 现场拍下的取证照）

**美术参考（不影响推理）**：
- 1920s 警用记录照片质感：黑白银盐相纸，约 13×9 cm（4×5 inch），边缘有齿孔或白边
- 闪光灯过曝感——手部高光偏白，背景白布灰阶
- 白布褶皱的轻微阴影，手腕处一截可见
- 焦味、烟尘、消防水渍未直接出现在画面里，但整体色温偏冷
- 背景可见一小角焦黑地板或铁担架边缘暗示死亡现场
- 拍摄角度：俯视 45°，让无名指处于画面构图焦点

## 2102 - Margaret 相册

**重点（信息表达必不可少）**：
1. 翻开的相册跨页特写，能同时看到 **3-5 张** Margaret 不同年龄段的照片（年轻、中年、晚年各阶段）
2. 每一张照片中 Margaret 的**左手无名指都清晰戴着一枚金色婚戒**——金属反光必须可辨识为戒指轮廓，不是肤色差异
3. 照片下方手写英文小字标注年份，能让玩家读到跨度，如 `Margaret & Patrick, Summer 1895` / `Christmas 1910` / `Ashford, 1923`
4. 至少一张特写需让婚戒占据视觉显眼位置（如握手、捧物、托腮等让左手前置的姿势）
5. 相册扉页或封面可读烫金 / 烫银英文标题：`Brennan Family — 1895` 或 `The Brennans`

**美术参考（不影响推理）**：
- 1920s 家庭相册质感：深棕色或暗红色压花皮革封面，烫金字，约 25×20 cm
- 黑色厚衬纸内页，照片用四角金属花角夹住
- 照片均为黑白 / 棕褐调（sepia），边缘略泛黄
- 翻开后的折痕、书脊布带、丝带书签等家庭相册细节
- 内页有少量手写说明（钢笔字，墨水偏暗）
- 桌面背景：可见一角木质化妆台或针线篓的旧家居氛围

## 2103 - 空的首饰盒

**重点（信息表达必不可少）**：
1. 打开的桃木首饰盒俯视图，盒内**深红色丝绒衬里**
2. 最显眼的位置有一个**圆形戒指凹槽**——尺寸明显是为单枚婚戒预留（直径约 2 cm 的圆形凹陷）
3. 这个戒指凹槽**空置**——里面没有任何戒指
4. 凹槽内绒布上方有**新鲜的轻微浅压痕 / 起绒方向被扰动**——明显是"今天才被取出"的痕迹（与长期空置的均匀绒面区分开）
5. 周围其他凹槽**完好放着首饰**：可见至少 1 枚胸针 + 1 对耳环 / 1 串珍珠链——证明这盒首饰平时被使用且爱护，唯独戒指那一格被临时动过

**美术参考（不影响推理）**：
- 桃木材质，盒身约 18×12×6 cm，盒盖完全打开呈 180° 平摊
- 盒盖内侧有小镜（可看到镜中反光，但不重要）
- 深红色（酒红 / 勃艮第）压花丝绒衬里
- 各凹槽用绒布隔条分割成网格
- 旁边可摆放一只 Margaret 的旧梳子或银镜柄做氛围（不喧宾夺主）
- 光线：暖黄台灯侧光，强调凹槽阴影
- 卧室梳妆台木纹背景

## 2107 - Patrick 遗物匣

**重点（信息表达必不可少）**：
1. 橡木小匣顶盖上刻有英文首字母缩写：`P.B.`（即 Patrick Brennan）——字母刻痕清晰可辨
2. **匣身正面嵌着一枚氧化发暗的工人运动领袖徽章**——1890s 美国工人组织风格的圆形锡/黄铜章，徽章中心或下缘可读小字明确身份：`P. BRENNAN — ORGANIZER` 或 `STRIKE COMMITTEE · A.R.U. 1894`（American Railway Union 1894 Pullman 大罢工的组织委员会成员徽章）；徽章本身的"署名 + 头衔"格式必须让玩家一眼看懂 Patrick **不是普通参与者，是带领工人运动的组织者/领袖**
3. **匣身侧面（或顶盖一侧、与 P.B. 错开位置）刻有一行较小的英文短语**（磨损但仍可读）：`FOR THE MANY WE PAY THE FEW` —— Patrick 1890s–1900s 在工人聚会与码头篝火上反复宣讲的独创信条（非公开名言、不见诸报端，只在 Patrick 最亲近的工人/学生圈子里口传），是 Zack 童年记忆里反复听过的话；**跨 Unit Chekhov's gun**——Unit4 L5 揭示 Mickey 把这句话刻进每一本工作笔记本扉页、并设为保险柜密码 FTMWPTF，是 Whale 身份的思想 DNA
4. 匣子完好无损——位置摆在卧室角落最稳当的位置，未被火烧到，说明 Margaret 一直珍藏（情感锚点）
5. 匣盖**关闭**——画面中 Zack 没有打开它

> 注：2107 为 envir 类型，不可拾取，无 doubt / expose 引用——但 `P.B.` 首字母 + 工人徽章 + `FOR THE MANY WE PAY THE FEW` 刻字三项是 Patrick 工人运动激进分子身份的视觉锚 + 跨 Unit 伏笔承载，归为重点。

> **悬疑感约束**：Unit2 L1 玩家看到这些视觉信息时，Zack 不能旁白点破"父亲教过我这句话"；不在 Unit2 任何 NPC 对白里复述或解释 `FOR THE MANY WE PAY THE FEW`——这句话的全部意义只在 Unit4 解开。

**美术参考（不影响推理）**：
- 深色橡木材质，纹理细腻，约 25×18×10 cm 长方形
- 哑光木蜡油表面，岁月把边角磨得圆润
- 黄铜小搭扣（已扣上）、两侧黄铜小铰链
- 顶盖中央 `P.B.` 刻字略带手工雕痕，不像机刻
- `FOR THE MANY WE PAY THE FEW` 刻字工艺：字号比 `P.B.` 略小，刻痕较浅但工整——像是 Patrick 亲手刻入或托人雕在自己最珍贵的私物上的庄重感（不是机刻、不是装饰花体）
- 工人运动领袖徽章约 2.5-3 cm 直径，氧化后呈深铜色或暗银色，嵌入式而非贴附（徽章本身略凹陷于匣身，像是匣子专门为它做了一个嵌位——表明这枚徽章对 Patrick 的份量）；徽章上 `ORGANIZER` 或 `STRIKE COMMITTEE` 头衔字样必须清晰可读，不能被氧化模糊到辨识不出
- 摆放位置：卧室一角的旧木椅或床头小柜上
- 旁边可放一张相框背面（隐约的家庭氛围）
- 光线：偏暗的烛黄余光，营造怀旧、未被火灾打扰的角落感

## 2108 - 地下室的门（打不开）

**重点（信息表达必不可少）**：
1. 一扇**向下通往地下室**的旧木门，位于卧室区角落地面 / 矮墙处（玩家能看出这是"下行的门"，不是普通房间门——可以是斜向下嵌入墙体的低门，或半埋在地面的活板门）
2. **新换的铸铁锁**——锁体崭新、漆面发亮，与木门陈旧的色调形成对比，肉眼一眼就能看出"锁是后来换的"（这是 L3 解锁需要的关键视觉信号）
3. 门**紧闭**，门把手或铸铁锁体上有 Zack 试推过的痕迹但未开
4. 门下方地面**无积灰、门槛处可见反复踩踏的脚印磨痕**——说明此门虽锁但**经常被人开启使用**（与崭新铸铁锁形成微妙反差：明明是经常走的门，却换了一把新锁——是 Frank+Lula 秘密约会空间的视觉伏笔，L3/L4 才解开）

**美术参考（不影响推理）**：
- 旧橡木 / 松木门板，被烟尘熏过的灰黑色调
- 门体尺寸约 80×170 cm，可见铁皮加固带
- 锁的新旧对比：门是 1900 年代老物，锁明显是 1920s 末新工业制品（造型上"重型矩形锁体 + 黄铜钥匙孔"）
- 门把手为旧式铸铁圆环或拉环
- 门下方地面焦黑度比门体本身轻——暗示火灾没烧到门下，或门下有空气流（地下室通风）；门槛踩踏痕呈"中间凹两侧高"的形态，是长期反复经过的物理结果
- 光照：手电筒或台灯侧光打在锁体上，强调"锁的崭新"

## 2204 - 威胁便条

**重点（信息表达必不可少）**：
