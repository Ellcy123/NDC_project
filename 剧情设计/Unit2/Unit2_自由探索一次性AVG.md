# Unit2 · 自由探索一次性 AVG 交互

---

## 文档说明

**适用范围**：自由探索阶段，玩家点击场景中的特定对象或 NPC 触发的**纯对话型 AVG**。

**区别于**：
- Talk 系统（Morrison / Leonard 等可反复对话的 NPC）
- 动态漫画突发事件（`Unit2_AVG突发事件动态漫画.md`）

**共同特征**：
- 纯 AVG 对话格式，无选项，无证据出示，无 @evidence/@round
- **一次性**：触发后对象状态改变或消失，不可重复触发
- 触发后记录状态标志（flag），后续 Talk / 场景不重复播放

---

## 格式规范

每条交互按以下格式记录：

```
### [编号] [交互名称]
所属场景：Scene XXXX
所属 Loop：loopX
触发对象：（场景中的具体对象或 NPC 名称）
触发条件：（解锁该交互的前置条件，如"首次进入场景"/"已获得XX证据"等）
消失/状态：（交互后对象的变化，如"消失"/"变为不可交互"/"进入新状态"）
对白：（AVG 格式对白）
```
