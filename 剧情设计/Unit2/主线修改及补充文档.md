## 问题 1：Tidewater 完全缺失

### 现状

v3 要求：Silver Moon 俱乐部调查让玩家得知"Silver Moon 最近的合作对象是 Tidewater 集团的 Whale"，把 Unit1 的 Whale 从黑帮影子推进到地产集团接口。Unit2 主线推进也标为【明示】。

大纲里 Silver Moon 只是 Tony 的赌场/酒吧，用于推动案件（连号钞票、Vinnie 线索）。Tidewater 从头到尾未出现，Whale 没有完成从"黑帮"到"地产集团接口"的认知升级。

### 修改方案：利用 L2 Silver Moon 场景，补两处

**① 环境线索**：Silver Moon 赌场吧台后方墙上新挂了一块铜牌或商业牌匾，刻着 "Tidewater Development Corp. — Entertainment Partner"（或类似的合作商标识）。玩家点击可查看，Zack 内心旁白："Tidewater……南区最大的地产商，连赌场都有他们的份。"——纯环境叙事，不挂疑点，不阻断案件流程。

**② Tony 对话补一句**：L2 调酒考验通过后，Tony 给 Vinnie 消费证词的那段对话里，Tony 随口抱怨一句——"最近生意不一样了，上面换了合作方——叫什么来着，Tidewater。Whale 是他们的人，现在连我这小赌场都得看他们脸色。"——不需要玩家追问，是 Tony 抱怨式的自然流露。

**放这里的原因**：L2 是玩家第一次进 Silver Moon，Tony 破冰成功后正是他话最多的时候，加一句抱怨最自然。一句话完成 v3 要求的认知升级（Whale = Tidewater 的人），不干扰案件推理。

### 需要改的文件

- `Unit2_大纲.md` L2 Silver Moon 赌场自由探索段 → 补环境线索描述 + Tony 对话
- 对应 state 文件（落 state 时同步）

## 问题 3："南区居民援助计划"名称未出现

### 现状

v3 说银行以"南区居民援助计划"名义向南区工人家庭推销贷款。这个名字是后续 Unit3-5 反复引用的锚点。大纲只写了"掠夺性贷款"的实质，没有使用这个具体计划名。

### 修改方案：L3 Moore 被击破后的妥协对话里，让 Moore 自己说出这个名字

现有剧情：Moore 被指证击破后，在 Mickey 威胁下恐惧审计，坦白"Frank 并非赌博欠债，是我们内部做的手脚"。

**补一段 Moore 的辩解**：Moore 坦白时加一段自我开脱——"这不是诈骗，这是银行董事会批准的'南区居民援助计划'，旨在帮助南区居民获得低门槛信贷。Frank 只是计划的正常参与者，协议框架完全合法。"

**Emma 反应**："援助？年化 112% 的利率也叫援助？"

**放这里的原因**：Moore 是银行行长，他用官方名称自我辩护最自然；"援助计划"和"年化 112%"的反差让玩家记住这个名字，后续 Unit 再出现时有锚点。

### 需要改的文件

- `Unit2_大纲.md` L3 指证后情节段 → 补 Moore 辩解对话 + Emma 反问

## 问题 4：Miller 恩人形象不够丰满，张力未构建

### 现状

大纲有 Miller 资助 Emma / Foster / 医院的🔵伏笔标注，但 v3 要求的几个层面缺失：
- "Miller 资助 1912 年受伤工人家属"——未提
- "Miller 设立特殊事故基金、间接援助 Margaret"——只说"医院是 Miller 集团资助的"，没具体到 Margaret 的医疗减免
- v3 说本章要形成第一个主线张力："同一个社会声誉系统，一边救 Margaret，一边让湖滨信托银行逼死 Frank"——大纲未在叙事层面构建

### 修改方案：L1 和 L4 各补一处

**① L1 节点（Emma 电话亭，现有台词提到 Miller 资助处）**

现有：Emma 转述 Foster 初步意见时提到"她们两个都受到 Miller 集团的资助"。

→ 补充 Emma 多说一句："Miller 基金这些年一直在资助 1912 年港口事故家属的子女——我就是靠他们的奖学金读完的书。Foster 也是，她的法医学研究经费也来自 Miller。"

效果：一句话同时建立三个信息点（1912 事故家属资助 + Emma 奖学金 + Foster 研究经费），由 Emma 用感激语气说出，玩家记住"Miller = 帮过很多人的好家族"。

**② L4 节点（圣心医院探望 Margaret，现有台词提到医院 Miller 资助处）**

现有："医院也是 Miller 集团资助的，因此 Margaret 的医疗费用大大减轻。"

→ 改为更具体的表述，由 Emma 或 Mickey 说出："Margaret 的医药费比我想的少很多——圣心医院有一个 Miller 特殊事故基金，专门给南区工伤家属减免费用。母亲的费用就是走的这个基金。"

**紧接着构建张力**：同一幕对话里，Zack 或 Emma 把话题转到 Frank 的案子——"湖滨信托银行把 Frank 逼成这样……" 玩家在同一场景里先听到"Miller 救了母亲"再听到"银行害死 Frank"，张力自然成立，不需要明示两者的关系。

### 需要改的文件

- `Unit2_大纲.md` L1 自由探索→警局外（电话亭）段 → 补 Emma 台词
- `Unit2_大纲.md` L4 开篇剧情（圣心医院）段 → 改 Emma/Mickey 台词 + 补 Zack 转场对话
