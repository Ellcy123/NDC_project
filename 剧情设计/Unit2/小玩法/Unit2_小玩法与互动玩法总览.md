# Unit2 小玩法与互动玩法总览

**文档版本**: v1.0
**范围**: `剧情设计/Unit2/小玩法/` 下 4 个小玩法 + 2 个互动玩法
**用途**: 给策划、美术、实现快速判断每个玩法的定位、触发方式、玩家动作和产出边界。

---

## 0. 分类口径

### 小玩法

有明确玩法规则或独立玩法界面，玩家需要完成一段操作流程，通常直接产出物证或派生证据。

Unit2 当前小玩法：

- L2 Silver Moon 骰子赌博
- L3 穿孔卡片机贷款演算
- L4 玫瑰花蕊找戒指
- L6 鞋坊照片找不同

### 互动玩法

不是完整小游戏，而是由场景、物品或对话触发的短互动。它通常负责取得证词、改变取证状态，或承接某个小玩法之后的后续取证。

Unit2 当前互动玩法：

- L2 Silver Moon 招牌酒考验
- L6 镊子取出眼镜碎片

### 不在本文件范围内

纯 AVG 突发演出不归入小玩法 / 互动玩法，例如 L1 电报、L2 银行大厅偷听、L4 Danny 困厕所、L5 Foster 电话、L6 Edith 推箱等。它们已由 `剧情设计/Unit2/美术需求/Unit2_AVG突发事件动态漫画.md` 统一管理。

## 2. L2 Silver Moon 两段的区别

### 互动玩法：招牌酒考验

对应文件：`互动玩法/L2_SilverMoon_BeesKnees招牌酒考验.md`

作用：让玩家通过观察环境线索取得 Tony 的信任，拿到 Vinnie 最近“赚了一笔”的证词。

玩家动作：

1. 在 Silver Moon 查看吧台、海报、蜂蜜罐等环境线索。
2. 推断招牌酒 Bee's Knees 的配方。
3. 回 Tony 对话里选择“金酒 + 柠檬 + 蜂蜜”。
4. Tony 认可 Zack 懂行，给出 2092002。

边界：

- 这是场景线索 + 对话选择。
- 不进入骰子界面。
- 不产出 2206。

### 小玩法：骰子赌博

对应文件：`L2_赌场_骰子赌博.md`

作用：让玩家通过赢 Tony 的骰子局，取得 2206 连号钞票。

玩家动作：

1. 选择押“大 / 小 / 奇 / 偶”。
2. Tony 摇两颗骰子并结算。
3. 玩家需要连续赢 3 局。
4. 首次达成后，Tony 给三张连号钞票 902B / 903B / 904B。

边界：

- 可重复玩。
- 2206 只在首次连续赢 3 局时获得一次。
- Tony 的“赚了一笔”证词不在骰子玩法内产出。

## 3. L3 穿孔卡片机贷款演算

对应文件：`L3_银行_贷款计算器.md`

分类：小玩法。

作用：让玩家把 Frank 的贷款协议送入银行档案室的穿孔卡片机，由机器算出年化 112%，取得 2701。

玩家动作：

1. 在银行档案室点击穿孔卡片机。
2. 从可分析文件中选择 2303 贷款协议合订本。
3. 在协议页里点选四个关键数字：600、12%、60 个月、50$。
4. 拉下制表机手柄。
5. 机器打印结果纸带，产出 2701。

边界：

- 玩家不是直接打开 2303 分析。
- 玩家不做复杂字段录入，只识别关键条款。
- 玩法重点是“机器算出来”，不是手工数学题。

## 4. L4 玫瑰花蕊找戒指

对应文件：`L4_地下室_玫瑰花蕊找戒指.md`

分类：小玩法。

作用：让玩家用水杯浇醒 Frank 没来得及照看的玫瑰，发现 2403 求婚戒指。

玩家动作：

1. 在地下室调查 2409 枯萎玫瑰。
2. 选择背包里的 2410 水杯，对玫瑰使用。
3. 玫瑰复活后，点击花蕊。
4. 戒指显形，刻字 F & L 1923 高亮。
5. 获得 2403。

边界：

- 玩家感知是“拿水杯给玫瑰浇水”。
- 不表现为普通背包合成。
- 2403 不作为场景直接拾取物，只由本玩法产出。

## 5. L6 鞋坊找不同与镊子取碎片的区别

### 小玩法：鞋坊照片找不同

对应文件：`L6_鞋坊_鞋楦找茬.md`

作用：让玩家用 2602 案发前照片对照鞋坊废墟，找出现场发生过打斗的物理痕迹。

玩家动作：

1. 在物品栏点击 2602 房屋抵押证明的“分析”。
2. 系统自动切到 L6 鞋坊废墟对照场景。
3. 玩家拖动照片，对齐窗框 / 工作台 / 地砖。
4. 玩家点击两处异常：鞋楦移位、货物箱凹陷。
5. 解锁 / 记录 2604 和 2606。

边界：

- 找不同目标只有鞋楦移位和货物箱凹陷。
- 不包含柜面刮擦痕。
- 不包含 2605 眼镜碎片。

### 互动玩法：镊子取出眼镜碎片

对应文件：`互动玩法/L6_鞋楦移开后_镊子取眼镜碎片.md`

作用：找不同完成后，玩家移开 2604，才看到柜面缝隙里的反光；再用 2603 镊子取出 2605。

玩家动作：

1. 完成鞋坊照片找不同。
2. 移开 / 拿走 2604 铸铁鞋楦。
3. 点击柜面缝隙里的反光。
4. 系统呼出可用物品选择页。
5. 玩家选择 2603 医用镊子。
6. Zack 用镊子夹出碎片。
7. 获得 2605 眼镜碎片。

边界：

- 这是找不同后的后续取证，不是找不同目标。
- 未持有或未选择 2603 时，只提示需要细一点的工具。
- 不在这里直接说“这是 Leonard 的镜片”，后续仍需与 2610 维修单对照。

## 6. 后续新增规则

新增内容时按以下口径归类：

- 有独立规则、连续操作或明确玩法界面的，放到 `小玩法/`。
- 场景、物品、对话触发的短互动，放到 `小玩法/互动玩法/`。
- 纯 AVG 突发演出，放到 `美术需求/Unit2_AVG突发事件动态漫画.md`。

小玩法和互动玩法文档不写数据配置块，只写设计、触发、玩家流程、产出、表现需求和指证链关系。
