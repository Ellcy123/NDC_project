# 玩法设计文档：鞋楦找茬（Unit2 适配版）

**文档版本**: v2.0
**隶属章节**: Unit 2 - 循环 6
**底层机制**: 案发前照片叠加 + 找不同
**设计目标**: 玩家把 Margaret 包里的案发前店铺照片叠加到现实废墟上，亲眼看出**工作台上的铸铁鞋楦从原位"飞"到了别处**——这不是火灾自然倒下，是有人抄起它打人。Frank 死亡的物理痕迹第一次被肉眼定格。

---

## 0. 前因后果

**前因——证据从哪来**

L6 玩家在 2006 圣心医院 Margaret 病房苏醒段，Foster 把 Margaret 案发当天准备拿去办抵押的随身包还给 Zack。包里两件关键物：

- **2602 房屋抵押证明**——Margaret 准备办抵押用，附带**两三天前刚拍的店铺内景照片**。照片角度正好覆盖案发的柜子/工作台区域，可见**铸铁鞋楦在工作台原位**。
- **2603 医用镊子**——Foster 给的，备注"取柜面缝隙细物用"。

带着这两件物件，玩家回到 2001 Margaret 鞋坊店铺区（L6 深度搜证状态）触发本玩法。

**后果——证据用在哪**

玩法不产派生证据，但**揭示三件已有物证的位置和可拾取状态**：

- **2604 铸铁鞋楦**（移位的本体——1920s 鞋坊标配工具，足跟部圆弧实心直径约 5–7 cm，与法医报告 2601 的钝器尺寸吻合）
- **2605 眼镜碎片**（鞋楦原位（柜面/工作台一角）足跟下方移开后才显形的柜面缝隙——需持 2603 医用镊子才能取出；鞋楦遮挡是关键证明：碎片必然在鞋楦被放下之前就掉进缝隙，即案发那一刻）
- **2606 货物箱凹陷**（鞋楦移位区附近 3 cm 凹陷，成人背部高度）

**指证用途**：
- **疑点 2601**（"鞋坊柜面缝隙里的眼镜碎片是谁的？"）condition：2605 + 2136001 Margaret 目击
- **疑点 2603**（"鞋坊里发生过打斗？"）condition：2606 + 2602
- **L6 Expose R2 击穿件**：2605 + 2610 眼镜维修单——击穿 Leonard"碎片不能证明是我的"
- **L6 Expose R3 击穿件**：2606 + 2602 双件——击穿 Leonard"拜访摔坏眼镜"（货物箱凹陷直接证明发生过激烈打斗）

## 1. 玩法概述

- **玩法类型**: spot_the_difference / overlay_comparison（复用 Unit2 框架）
- **核心体验**: 叠加照片 → 重叠透视 → 发现违和感 → 点击三处异常 → 揭示物证位置
- **预期时长**: 60–90 秒
- **触发条件**: 持有 2602 房屋抵押证明 + 进入 2001 鞋坊店铺区（L6 状态）；Zack 凑近案发柜子/工作台区域，主动调用 2602 照片
- **失败惩罚**: 无。点错位置会有 Zack 提示但不计错

## 2. 界面与交互

### 界面布局

```
┌─────────────────────────────────────────────────────┐
│              鞋坊店铺区废墟（焦黑底图）                │
│                                                     │
│   ┌─ 照片框（玩家拖动叠加）──┐                       │
│   │                         │                       │
│   │ [照片：案发前的店铺]     │                       │
│   │  工作台｜鞋楦（原位）     │  ⚙ 工作台（焦黑）       │
│   │  货物箱（整齐）          │  ⚙ 货物箱靠墙          │
│   │                         │  ⚙ 鞋楦偏离原位         │
│   └─────────────────────────┘                       │
└─────────────────────────────────────────────────────┘

  叠加锁定后两幅画面对比，三处异常待点击：
  ① 鞋楦移位（照片原位在工作台一角 → 现实偏离原位）
  ② 柜面划痕（照片整洁 → 现实钝器擦痕）
  ③ 货物箱凹陷（照片完好 → 现实 3cm 凹陷，成人背部高度）
```

- **底层**：当前 2001 鞋坊废墟（焦黑色调）
- **上层**：玩家手持 2602 附带的泛黄旧照片，拖动叠加
- **照片透视吻合时自动锁定**（以柜子/窗框/地砖为参照点）

### 交互（三步极简）

1. **叠加锁定** — 玩家拖动照片到位置，参照物吻合后自动"咔"一声锁定（约 3 秒）
2. **点击三处异常** — 鞋楦移位 / 柜面划痕 / 货物箱凹陷；每点一处高亮金边 + Zack 一句独白
3. **揭示 + 取证**：
   - 2604 铸铁鞋楦 → 直接拾取
   - 2606 货物箱凹陷 → 自动记录进 CASE BOARD（envir）
   - 2605 眼镜碎片在柜面缝隙——**持 2603 医用镊子点击缝隙触发取出**（约 3 秒嵌套小动画）；不持镊子 Zack 说"我需要细一点的工具"

**Zack 独白**（按点击顺序）：
> ① "这鞋楦不是火烧倒的——它从这边跳到了那边。"
> ② "柜面全是刮擦痕。这种重的铸铁件，有人抄起来挥过。"
> ③ "3cm 凹陷，正好在背部的高度。这里发生过激烈打斗——有人被狠狠撞到了箱子上。"
>
> （持镊子取碎片瞬间）"烟熏黄色——颜色对得上 Leonard 的旧镜片。这是 Leonard 的眼镜。"

## 3. 结果与产出

**产出证据**: 无派生证据；揭示三件已有物证的位置和可拾取状态：

| ID | 名称 | 位置 | 取法 |
|---|---|---|---|
| **2604** | 铸铁鞋楦 | 工作台原位附近偏离的倒位 | 直接拾取 |
| **2605** | 眼镜碎片（烟熏黄色旧镜片） | 鞋楦原位（柜面/工作台一角）足跟下方移开后露出的柜面缝隙 | 需持 2603 医用镊子（嵌套小动作） |
| **2606** | 凹陷的货物箱 | 鞋楦移位区附近打斗痕迹区 | clue，拍照入背包 |

**疑点解锁**:
- 疑点 2601（眼镜碎片是谁的？）等待 2136001 Margaret 目击配齐
- 疑点 2603（鞋坊里发生过打斗？）此时 2606 + 2602 双件齐 → **直接解锁**

**指证用途**:
- L6 Expose R2：2605 眼镜碎片 + 2610 眼镜维修单 → 击穿 Leonard"碎片不能证明是我的"
- L6 Expose R3：2606 货物箱凹陷 + 2602 房屋抵押证明 → 击穿 Leonard"拜访摔坏眼镜"——3cm 凹陷不是摔的，是被打的

**玩家自跳的关键推理**（玩法不喂答案）：
- 拿到 2601 法医报告：致命部位钝器**直径约 5–7 cm、重物 1.5–3 公斤**——不点名工具
- 在鞋坊看到 2604 铸铁鞋楦：足跟部圆弧实心直径约 5–7 cm，全长 25 cm 重约 2 公斤
- **玩家自己反应过来**："鞋楦的足跟正好这个尺寸"——啊哈时刻

## 4. 美术 / 音效需求

复用 Unit2 原版资产 + Unit2 特定改造：

美术：
- **复用**：照片叠加 UI、透明度渐变、参照物吸附锁定、双层时空对比画面、Zack 手持照片
- **Unit2 特定**：
  - 2602 照片：1928 年代灰白照片质感，边角微微泛黄，店铺内景以**工作台 + 鞋楦原位**为视觉中心
  - 2001 店铺区废墟：焦黑色调主导，鞋楦移位轨迹用淡淡的"拖痕"美术暗示（不是显式标线，是柜面/地砖灰尘被拖过的痕迹）
  - 货物箱凹陷：3cm 内凹的木制皮箱，凹陷高度约 1.2m（成人背部位置）
  - 柜面缝隙：工作台一角后沿/侧沿的窄缝，鞋楦足跟原位移开后露出；眼镜碎片用一点反光暗示
  - 铸铁鞋楦：足部形状实心铸铁件 + 一根铁杆，全长 25 cm、足跟圆弧直径 5–7 cm、重约 2 公斤；深灰黑铸铁氧化包浆 + 足跟侧面血迹与烟熏混合的暗红色污渍

音效：
- 照片锁定：木质"咔"声
- 异常点点击：清脆叮当 + Zack 低声台词
- 镊子取碎片：金属夹住玻璃的微响（嵌套动作约 3 秒）
- 整体环境：火灾后的死寂——只有远处风声、断续的电线滋滋声

## 7. 设计备注

- **这是 Unit2 最具"福尔摩斯感"的玩法**——不是分析、不是计算，是**还原历史**。玩家把"过去的照片"压在"现在的废墟"上，用肉眼读出"这里发生过什么"。这种体验在所有 4 个玩法里最接近 1940s 黑色电影侦探的破案瞬间。

- **Foster 在医院给 Margaret 包**这个剧情前置很重要——它确立了 Foster 不是单纯的法医，而是**有意识帮 Zack 推进调查**的角色（Miller 集团资助伏笔的轻量铺垫）。她给 2602 + 2603 两件物，是因为她**预判到**鞋坊废墟里需要这两样工具。这个细节在 Unit2 内不暴雷，但敏锐的玩家会注意到 Foster 太"刚好"。

- **不要让 Zack 提前点破"Leonard 来过"**——找茬过程中 Zack 的独白只描述物理痕迹（"鞋楦不是火烧倒的"、"3cm 凹陷"），不说"这是 Leonard"。直到取出 2605 眼镜碎片那一刻，颜色匹配 Leonard 旧镜片才能让 Zack 说出"这是 Leonard 的眼镜"——这个推理的最后一步必须留给玩家自己跳。

- **凶器是鞋坊标配工具的"在场逻辑"** ——铸铁鞋楦是 1920s 鞋坊必备的标准工具，它本来就在工作台上，案发那一刻 Leonard 顺手抄起来，比"在 Margaret 卧室里翻出一个烛台来打人"自然得多。这种"凶器就在工作场所内、是手边随机能拿到的东西"的设计，让"失手打死"的失控感更真实——不是预谋。

- **法医报告不点名凶器**：Foster 的报告只给"圆弧形钝器、致命部位 5–7 cm、重物 1.5–3 公斤"。玩家拿到鞋楦后**自己**反应过来"足跟正好这个尺寸"。这条"啊哈"推理留给玩家完成，符合悬疑感优先、不喂答案的原则。

- **货物箱凹陷"3cm + 成人背部高度"的设计精度**：3cm 凹陷高度对应**成人背部**被撞击留下的痕迹——Frank 被 Leonard 抄起鞋楦挥过来时背部撞到货物箱，所以才有这个凹陷。这是 Leonard 杀 Frank 的**物理时间线**复原——比任何证词都直接。
