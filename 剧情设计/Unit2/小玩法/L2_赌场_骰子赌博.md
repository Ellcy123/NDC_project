# 玩法设计文档：Silver Moon 骰子赌博

**文档版本**: v3.0（简化版）
**隶属章节**: Unit 2 - 循环 2
**玩法类型**: dice_betting_simple
**设计目标**: 让玩家通过连续赢 Tony 三局骰子，亲手拿到 2206「Tony 找零的连号钞票」。这不是完整赌博系统，而是一段短、可重复、带 1920s 地下赌场气质的取证互动。

---

## 0. 前因后果

**前因——为什么能玩**

Zack 在 Silver Moon 与 Tony 搭上话后，Tony 愿意陪他玩一轮骰子。招牌酒考验不写在本文档，单独见 `剧情设计/Unit2/小玩法/互动玩法/L2_SilverMoon_BeesKnees招牌酒考验.md`。

**后果——证据用在哪**

玩家首次连续赢下三局骰子后，Tony 用三张新钞赔付 / 找零。Zack 注意到序列号连续，获得：

- **2206** Tony 找零的连号钞票（902B / 903B / 904B）

2206 与 2202 O'Hara 手里的两张新钞（901B / 905B）并排后，能拼出 901B-905B 完整连号，用于 L2 Expose R2 击穿 Leonard“我和 Vinnie 只是认识”的退守。

---

## 1. 玩法概述

- **核心体验**: 押大小 / 奇偶 → Tony 摇骰 → 连续赢 3 次 → 首次达成时拿到连号钞票
- **预期时长**: 一轮约 60-90 秒
- **触发条件**: L2 Silver Moon，与 Tony 对话后选择“玩一局骰子”
- **失败惩罚**: 无。输了可以立刻再玩
- **重复规则**: 骰子游戏可重复玩，但 2206 只在第一次连续赢 3 次时获得一次

---

## 2. 玩家流程

1. Tony 把骰盅推到吧台上，说明“连着赢三把，钱就是你的。”
2. 每一小局，玩家选择一个押注项：**大 / 小 / 奇 / 偶**。
3. Tony 摇两颗骰子，骰盅落桌后揭开。
4. 系统结算本局输赢。
5. 赢一局，连胜计数 +1；输一局，连胜计数清零，但玩家可以立刻从第一局重来。
6. 如果这是玩家第一次连续赢 3 局，Tony 抽出三张新钞递给 Zack。
7. Zack 看到序列号 **902B / 903B / 904B** 连着，记录 2206。

输掉时，Tony 只调侃一句，玩家可以选择立刻重开这一轮。不要让玩家因为运气被卡住，重玩入口必须很顺。

---

## 3. 规则口径

两颗六面骰，按总点数结算。

- **小**: 2-6
- **大**: 8-12
- **7**: Tony 赢
- **奇 / 偶**: 按总点数奇偶判断，7 仍算 Tony 赢

这套规则不追求真实赌场完整度，只服务“玩家连续赢三次，Tony 拿连号钞票出来”的取证节奏。

---

## 4. 结果与产出

**首次连续赢 3 次产出**

- 2206「Tony 找零的连号钞票」

**后续重复游玩**

- 不再产出 2206
- 可保留 Tony 的短台词或少量金钱表现，但这些不进入证据系统

**Zack 观察**

> “902、903、904……号码连着。”

Zack 不需要替玩家解释结论，只记录事实。玩家之后与 2202 对照时自己完成啊哈。

## 5. 美术 / 音效需求

美术：

- Silver Moon 吧台或小骰桌特写
- 两颗 1920s 质感骰子
- 皮革骰盅
- 三张新钞特写，序列号 902B / 903B / 904B 必须清楚

音效：

- 骰盅晃动声
- 骰子落桌声
- Tony 轻笑 / 调侃的短反应
- 新钞放到吧台上的纸张摩擦声
