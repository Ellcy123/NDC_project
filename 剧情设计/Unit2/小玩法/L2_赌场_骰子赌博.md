# 玩法设计文档：Silver Moon 破冰（调酒考验 + 骰子赌博）

**文档版本**: v2.0（重写——原 v1.0 是纯骰子氛围玩法，本版加入"调酒身份验证"段，整合为两段式陪 Tony 喝一局体验）
**隶属章节**: Unit 10 - 循环 2（首次开放，跨章节可重玩）
**设计目标**: 让玩家通过 1920s 禁酒令时期 speakeasy 的"圈内人验证"仪式（调酒暗号）拿到 Tony 的证词，再通过陪 Tony 玩一局骰子赌博拿到 Vinnie 的钞票——两段同一场景同一 NPC 的连续体验，**重现 1920s 地下酒吧"先验身份、再做生意"的真实文化逻辑**。

## 0. 前因后果

**前因——玩家在 L2 进入 Silver Moon 寻找 Vinnie 与 Tony**

Zack 在 L2 自由探索阶段从 O'Hara 处拿到 Leonard 名片（2201），由此得知南区有个 Silver Moon 赌场——Vinnie 常驻。Zack 进店时 Vinnie 正在角落，Tony 在吧台。

但 Tony 不是熟客的陌生人不会主动交底。Zack 必须通过 1920s 禁酒令时期的**"圈内人仪式"**——Tony 测试这个生面孔是不是真的"懂行"。这就是调酒考验的来源。

## 1. 玩法概述

- **玩法类型**: trust_verification_minigame（身份验证）+ dice_betting（骰子押注）的两段式组合
- **核心体验**: 调酒答题 → Tony 视为自己人 → 给口风 → 陪喝一局骰子 → Tony 投桃报李给钞票
- **预期时长**: 调酒 ~1-2 分钟 + 骰子 ~2-3 分钟，整体 ~3-5 分钟
- **触发条件**: 进入 2015 Silver Moon，主动找 Tony 搭话——对话选择"陪 Tony 喝一杯"
- **失败惩罚**: 两段都有保底（调酒可重试，骰子有"Tony 借本钱"机制），玩家不会被卡死

---

## 2. 1920s 考据

### 调酒考验的真实背景

1920-1933 美国禁酒令期间，地下酒吧（speakeasy）的"圈内人验证"是真实存在的文化：
- **暗号**：进门时说特定密语
- **熟人推荐**：老顾客带新人
- **行话/秘方**：能说出某酒配方 = 这个圈子的人
- **小费规矩**：1920s 美国调酒师对"懂行"的客人会给更多耐心

Tony 用调酒题考验 Zack——不是想 humiliate 他，是**在判断 Zack 是不是警察/线人**。1920s 黑帮酒吧老板这是日常自保。**答对 = 你不是来抓我的，你是这个文化里的人**。

### 骰子赌博的真实背景

1920s 美国地下赌场最流行三种：**扑克、骰子（Craps）、轮盘**。骰子节奏最快、门槛最低、最适合作为陪喝时的消遣。**1920s 街头掷骰**是真实底层文化场景。

## 3. 第一段：调酒考验（对话内嵌 + 环境线索）

**本质是对话考验，不是独立玩法 UI**——Tony 在普通对话流里出一道题，玩家在 Silver Moon 自由探索时观察环境找线索，回到对话选对答案。整个过程发生在 2015 场景的常规自由探索阶段。

### 触发流程

```
玩家在 Silver Moon 自由探索 → 走到吧台找 Tony 套话
     |
     v
Zack 选对话分支："我常来这种地方"（假装熟客）
     |
     v
Tony 反诘："那你知道我们这边的招牌酒用什么吗？"
     |
     v
对话弹出 4 选 1 选项（无独立 UI，普通对话选项框）
     |
     +─→ 直接选答（凭运气/记忆）
     +─→ 选"我再想想" → 玩家回到自由探索找环境线索 → 回 Tony 重选
```

### 环境线索（散落在 Silver Moon 自由探索里）

玩家可以在 2015 场景里走动观察这些线索，不需要专门的 UI：

| 线索位置 | 内容 | 暗示答案 |
|---|---|---|
| **墙上海报一角** | 手写"Bee's Knees - $0.50"小字（褪色但可辨） | Bee's Knees 是本店招牌 |
| **吧台尽头的调酒小单** | Tony 手写"Gin + Lemon + Honey"配方 | Bee's Knees 三件 |
| **酒架上的蜂蜜罐** | 罐口贴着"Bee's"标签 | 招牌酒含蜂蜜 |

玩家凑近观察任意 1-2 处即可推出答案——**不需要凑齐三处**，1920s 玩家就是这样靠零碎细节"懂行"的。

### 对话选项

```
Tony："那你知道我们这边的招牌酒用什么吗？"

> A. 金酒 + 柠檬汁 + 蜂蜜  ← ✓ Bee's Knees
> B. 威士忌 + 苦艾酒 + 苦精
> C. 朗姆酒 + 糖浆 + 柠檬
> D. 我再想想（回自由探索找线索后可重选）
```

### 答对 vs 答错

- **答对** → Tony 笑了一下："小子，你懂行。" → 视为"自己人" → 主动谈起 Vinnie + Frank（下面信任段）
- **答错** → Tony 摇头："这都不知道。再想想吧。" → 不愤怒，但不交底 → 对话回到上一层，玩家可以选 D 重看环境再来

> ⚠ 答错无惩罚，可无限重试。Tony 不会失去耐心——他只是没"认"你。

### Tony 信任段对话（答对后）

> "Vinnie 那家伙最近常来——花新钞，说什么'今天又赚了一笔'。"（→ 2092002）
>
> "对了，你妈雇的那个 Frank——老头也常来这一带喝酒。一杯啤酒能坐一晚上。我看挺孤单的，丧偶嘛，可怜人。"（→ 2092003；Tony 不知 Frank 真实动机是监管侄子 Danny）

Zack 内心独白（拿到 2092003 时）：
> "Frank 老头……一杯啤酒坐一晚上。"

## 4. 第二段：骰子赌博（可选）

**触发**：调酒答对、Tony 给完证词后，对话里 Tony 端骰子邀请：

> "Brennan，要不要来一场紧张刺激的赌博？打发打发时间。"

玩家选 **「玩 / 不玩」**：

- **选玩** → 进入骰桌（下文）
- **选不玩** → Tony 收回骰子，不强求；可再坐下重触发；始终不玩 = 看 Margaret 面子 Tony 仍给 2206（保底），但失去跨章节骰子关系

> ⚠ 调酒（必）+ 骰子（选）= 圈内人验证 + 情谊邀请的 1920s 文化结构。

### 界面布局

```
┌────────────────────────────────────────────────────┐
│              墨绿羊毛毡骰桌特写                       │
│                                                    │
│   钱包余额 [$ 50]                                  │
│   ┌───────────────────────────┐                   │
│   │                           │                   │
│   │     [骰盅 + 两颗骰子]      │                   │
│   │                           │                   │
│   └───────────────────────────┘                   │
│                                                    │
│   赢钱进度  ████░░░░░░  30$ 解锁                    │
│                                                    │
│   [大]   [小]   [奇]   [偶]      [摇骰]            │
│   金额调整：+1$  +5$  +10$       [离桌]            │
└────────────────────────────────────────────────────┘
```

- **桌面特写**：墨绿羊毛毡 + 皮革骰盅 + 两颗象牙骰
- **顶部**：Zack 钱包余额（动态）
- **中部**：骰桌与骰子滚动空间
- **进度条**：累计赢钱（目标 30$ 解锁 2206）
- **底部**：四个押注按钮（大/小/奇/偶）+ 金额调整 + 摇骰 + 离桌

### 交互（4 步极简）

1. **押注** — 点四个按钮之一 + 选金额（+1/+5/+10）
2. **摇骰** — 点击摇骰 → 骰盅晃动 1.5 秒 → 落桌滚动 1.5 秒停稳
3. **结算** — 押中 1:1 赔率；点数 7 退还押注（"庄家说今天交个朋友"）
4. **累计赢 30$** → 解锁 Tony 结账给 2206

### Tony 结账（不知是线索）

> "行了行了，你这眼神挺准。"
>
> Tony 从围裙口袋摸出钱——抽出三张新钞放到 Zack 面前，顺手一句：
>
> "这几张是上礼拜 Vinnie 来这儿喝酒结账留下的零钱。你拿去吧。"

Tony 自己不知这是线索；Zack 看着钞票才发现 **902B / 903B / 904B** 号码连着的。

→ item 2206 进入背包；Tony 在 L2 期间始终不知"侦探价值"

### 保底机制

- **连输 5 把 + 钱包<5$** → Tony "算了，免单一杯威士忌，记账上" → 钱包恢复 10$ + 直接给 2206
- **完全跳过骰子** → 看 Margaret 面子给 2206，失去跨章节关系

Zack 内心独白（看着三张钞票）：
> "902、903、904——号码连着的。"
>
> "Tony 不知道这玩意儿能干什么。但我知道。"

## 9. 设计备注

- **调酒考验不是炫技，是验证**：1920s speakeasy 文化里这种"暗号/秘方"考验**真实存在**。Tony 出题不是 humiliate Zack，是判断他是不是警察/线人——这是禁酒令时期黑帮酒吧老板的日常自保。如果写手把这段写成"嘲弄 Zack 不懂"，文化感会丢；要写成 Tony 在严肃地"决定要不要交底"。

- **2092003 是 Frank 立体化的关键 testimony**：L2 玩家拿到时只觉得"Frank 是个孤独老头"，与 Tony 视角一致；**L4 拿 Lula 证词后玩家自己反转**——Frank 不是孤独，是父爱。这种"信息层次反转"是 Unit2 推理体验的精华之一。**写手不要让 Tony 在 L2 透露任何 Frank 真实动机的暗示**——Tony 必须保持"我以为他只是个孤单老头"的认知。

- **调酒必经 vs 骰子可选**：调酒考验是 Tony 给证词的硬门槛（不答对就不给 2092002/2092003）——但骰子赌博是对话选项触发，**玩家可以选择不玩**。选不玩仍能拿 2206（看 Margaret 面子，不卡主线），但失去 Tony 跨章节熟客关系 + 自己赢来钞票的参与感。这种"必经 + 可选"组合贴合 1920s 文化：身份验证是规矩，一起赌一局是情谊。

- **保底机制必须做扎实**：调酒可重试（无惩罚）；骰子连输 + 钱包<5$ 触发 Tony"借本钱"+ 给钞票；玩家选"不玩骰子"也能拿 2206。主线推理永远不会被卡死。

- **跨章节**：调酒考验完成一次后 Tony 视 Zack 为熟客；后续 Unit 玩家回 Silver Moon 时 Tony 进门点头微笑，骰子仍可玩。这是 Unit2 留给玩家的"长情"——一个跨越多个 Unit 的小关系。
