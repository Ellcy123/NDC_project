- 对Danny的种族主义和对Rose的刻板印象感到厌恶
  
第四阶段：悲剧的见证者（循环5-6）

Rose的作用：
- 提供Frank的保险箱钥匙
- 在旁边见证保险箱开启
- 得知Frank准备举报Moore，为正义献身
- Rose理解Frank："他做的事不是为了自己"
  
Rose的成长：
- 从"只想和Frank在一起"到"理解Frank的正义感"
- 从"躲在暗处"到"在阳光下说爱"
- 从"被动接受"到"主动守护Frank的记忆"
  
第五阶段：守护记忆（循环6结尾）

Rose的悼念：
- 站在Margaret鞋坊废墟前
- 手里拿着Frank的照片
- "现在我终于可以光明正大地说我爱他了"
  
Rose的承诺：
- 守护Frank的记忆
- 让Frank的勇气被世人知晓
- "他死得有意义"

🎯 设计意图

叙事功能

1. 第二重反转的核心（更强的反转力度）：
  - Rose的"不正经"让反转更震撼
  - 第一印象：活泼、打扮时髦、在1928年被视为"不正经"
  - Danny的诬陷因此更有说服力："那种女人肯定是骗子"
  - 玩家也会怀疑：这种性格真的能真心爱一个73岁老人吗？
  - 反转：Rose的活泼是真实性格，但对Frank的爱同样真挚
  - 更深的反转：Rose一开始为了保护Frank而拒绝，提议秘密约会
  - 让玩家反思：为什么我们会因为外表（活泼、时髦）就否定真诚？
    
2. 打破刻板印象 + 性格与行为的反差：
  - 活泼≠不真诚，外表≠内心，热情≠不顾后果
  - 反差的力量：
    - Frank：不善言辞、倔强固执，但在爱情上主动追求
    - Rose：活泼外向、不在乎世俗，但在这段感情中谨慎保护Frank
  - 两个"不正常"的人（倔老头+活泼女人）真心相爱
  - Frank欣赏Rose的活力："像太阳一样"
  - Rose欣赏Frank的勇气："倔老头，我的英雄"
  - 这是对保守社会的反抗，也是对刻板印象的挑战
    
3. 对比Leonard的镜像：
  - Rose和Frank敢于面对真实感受，不掩饰自己
  - Rose："我就是爱笑，怎么了？"
  - Frank："我们没有做错什么"
  - 虽然被迫秘密约会，但内心坦荡
  - 对比Leonard逃避真实18年
    
4. 系统性压榨的受害者（多重压迫）：
  - 种族歧视：墨西哥移民
  - 性别歧视：女性、单身
  - 外表歧视：打扮"新潮"被视为"不正经"
  - 年龄歧视：老年恋被嘲笑
  - 多重压迫让Rose和Frank的爱不被接受
    
情感共鸣点

1. 打破刻板印象的震撼：
  - 第一印象：Rose活泼、时髦，"肯定是骗子"
  - 反转：Rose的活泼是真实性格，爱Frank也是真心
  - 让玩家反思：为什么我们会因为外表就否定真诚？
  - "活泼的人就不能有真爱吗？" "时髦的女人就一定不正经吗？"
  - 引发对刻板印象的深刻反思
    
2. 性格与行为反差的感动（不合常规的美）：
  - Frank：不善言辞但主动追求，不在乎世俗
    - 出院后每天等Rose下班，追了两个星期
    - "名声能当饭吃吗？我只想和您在一起"
  - Rose：活泼外向但谨慎保护，为Frank着想
    - 一开始拒绝："人们会说您老糊涂了"
    - 提议地下室约会："我不想让您被人嘲笑"
  - 反差的美：热情的人反而更慎重，固执的人反而更勇敢
  - Frank欣赏Rose的活力："像太阳一样"
  - Rose欣赏Frank的勇气："倔老头，我的英雄"
  - 真爱无关年龄、种族、性格，关键是互相理解
    
3. 悲剧的哀伤（但Rose仍保持活力）：
  - "现在我可以在阳光下说了！可是他听不到了..."
  - Rose失去Frank，但仍守护记忆："他不会希望我哭丧着脸"
  - Rose即使悲伤，也保持活力："为了他，我会继续笑"
  - 让玩家为他们的爱情哀悼，也为Rose的坚强感动

📝 创作要点

对话创作

1. 活泼直率：
  - Rose说话快、爽朗，爱用感叹号
  - 用简单直接的话表达情感
  - 例："Frank？好人！很好的人！" "我爱他！就这么简单！"
  - 说话时常带笑容，手势多
    
2. 反差的真诚：
  - 表面活泼，但讲到深情时也会哽咽
  - 笑中带泪，不掩饰情感
  - 例："他说我'像太阳'！（大笑）就因为这句话...（哽咽）我就知道这个老头不一样"
  - 用反差展现真诚
    
3. 哀伤但不失活力：
  - Frank死后，Rose悲伤但不是"温柔哭泣"
  - 而是"大声哭，然后擦干眼泪，继续笑"
  - 例："他听不到了...（擦眼泪）但他不会希望我哭丧着脸！我会继续笑！为了他！"
    
情感刻画

1. 医院相遇的活力（打破第一印象）：
  - 要详细刻画Rose如何用活力感染Frank
  - 小细节：Rose的开朗、Frank的惊讶、"像太阳一样"
  - 让玩家看到：Rose的活泼不是装的，是真实性格
  - Frank欣赏这种活力，不是被"迷惑"
    
2. 5年秘密约会的真诚（外表≠内心）：
  - 照片墙、情书、Rose的珍藏
  - Rose虽然活泼，但对这段感情极度认真
  - Frank的情书内容：欣赏Rose的性格，不是外表
  - 让玩家感受到：活泼的人也能有真挚的爱
    
3. 失去Frank的悲伤（但不失活力）：
  - Rose的悲伤要真实、深刻
  - 但不是"温柔哭泣"，而是"大声哭，然后擦干眼泪"
  - Rose："他不会希望我哭丧着脸"
  - 让玩家感受到：Rose用自己的方式守护Frank

🎨 视觉设计建议

外观（1928年"新女性"风格）

- 年龄感：48岁，但充满活力，不显老
- 种族特征：棕色皮肤，墨西哥裔
- "新潮"穿着（对1928年来说）：
  - 裙子比一般女性短（到小腿中部）
  - 颜色明亮（红色、黄色），不是保守的深色
  - 喜欢戴首饰（耳环、项链），化淡妆
  - 常戴Frank送的围巾，但系法时髦
- 在1928年的评价："不像正经女人" "打扮得花里胡哨"
  
肢体语言（充满活力）

- 日常：动作快，步伐轻盈，说话时手势多
- 照顾病人时：开朗，用笑容感染病人
- 讲述故事时：手舞足蹈，笑中带泪
- 被诽谤时：不是"坚定挺直"，而是"愤怒反驳"，有力量
- 悼念Frank时：大声哭，然后擦干眼泪，强颜欢笑
  
情感表达（外放而真诚）

- 回忆Frank：爽朗地笑，眼中有光
- 讲述医院相遇：活泼地讲述，手势多
- 深情时刻：笑容收起，哽咽，但不掩饰
- Frank死后：泪流满面，但不是"温柔哭泣"，而是"放声大哭"
- 守护记忆：擦干眼泪，强颜欢笑："他不会希望我哭丧着脸"
  
重要道具

- 照片墙：Rose小公寓里，墙上全是她和Frank的照片
- 情书：Frank写给Rose的信，Rose珍藏
- 围巾：Frank送的礼物，Rose一直戴着
- 保险箱钥匙：Frank留给Rose，象征信任
