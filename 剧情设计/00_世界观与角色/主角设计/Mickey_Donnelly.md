# Mickey Donnelly（主角设计）

> 系列级人物档。聚焦人设、弧光、立场、在主线中的作用，**不复述各 Unit 剧情事件**（事件见各 Unit 大纲 / state）。配套 v3《主线故事全貌》§17.3。

## 一、一句话
被系统消化的理想主义者——想从内部摧毁权力机器，最终成了机器最锋利的工具。

## 二、人设
- 身份：律师（Donnelly & Associates）、Zack 童年朋友；隐藏身份 **Whale**——Tidewater 内部协调黑警、银月会与灭口的职能代号（1925 年成为）。
- 底色：工人运动与父辈死亡塑造的正义信念，被"为了更大的正义可以牺牲少数"的算法逐步侵蚀。
- 气质：体面、可信、善于让脏事在纸面上干净；对 Zack 有真实情谊，也始终在计算他能看到什么。

## 三、弧光（起承转合）
- 起：受工人运动与父亲之死影响，主动进系统，要从内部摧毁它。
- 承：成为 Tidewater 法律顾问 → Whale，掌握黑警 / 黑帮 / 灭口 / 壳公司接口。越想掌握系统，越成为系统。
- 转：知情人接连消失、Zack 逼近；他承认身份、交出档案。
- 合：坠楼。留下证据，也留下伤口——理想主义若只剩计算，会变成压迫者最锋利的工具。

## 四、立场与矛盾
不是纯反派：有理想主义残余（某些节点会破坏系统利益），也有冷酷算法（把人算成代价）。临终留档既是赎罪，也仍是"替所有人安排路线"的老毛病——连善意都要替别人计算。

## 五、在主线中的作用
- 主线引擎：玩家与 Whale / Tidewater / Miller 之间的活体接口，他的"帮忙"一直在控制调查能看到什么。
- Zack 的镜子：Zack 最痛的不是被骗，而是 Mickey 用这件事证明"他们是一样的人"——这套算法是 Zack 必须拒绝的东西。
- 物证起点：留下的 1912 / 1928 档案 + O'Hara，是后续真相坐实的起点。

## 六、关系锚点
Zack（童年友 / 律师 / 真身 Whale，信任崩塌核心）｜Tidewater 体系（法律 + 暴力接口，上是 Miller，平级 Moore 银行 / Lawson 地产）｜O'Hara（临终留给 Zack 与 Emma 的帮手）
