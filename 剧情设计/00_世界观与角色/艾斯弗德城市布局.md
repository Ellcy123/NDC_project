# 艾斯弗德城市布局

**1928 年 · 以芝加哥为原型的虚构城市**

> 本文档整理游戏中出现的所有场所，按城市片区归类。配合案情地图使用。
> 城市东面紧邻卡夏瓦湖（Lake Kashawa，原型密歇根湖）；艾斯弗德河呈 Y 字形贯穿城市，主干从湖口向西流，在中心商业区附近分为南北两条支流。高架环线铁路（L 线）围出 Loop 商业区。

---

## 北区 (North Side)

北区是艾斯弗德的富人区，老钱家族、社会名流和高级专业人士聚居于此。

### 黄金海岸 (Gold Coast)

| 场所 | 说明 |
|------|------|
| Miller 庄园 | 老 Charles 的私人庄园。维多利亚式豪宅，内设礼堂（旋转楼梯+舞台+暗门）、老 Charles 书房（壁炉旁挂画后通私人夹层）、议事厅、客房区、服务区。Unit5 慈善晚宴与案发地。庄园内还设有慈善基金会办公区和私人画廊 |

## 南区 (South Side)

游戏核心区域。工人阶级、移民家庭聚居地，1928 年第二次南区清退的目标区域。

### 南区居民区 (清退目标区)

1912 年事故工人家庭及后代的栖身地。被清退的房产最终流向 Tidewater。

| 场所 | 说明 |
|------|------|
| Margaret 鞋坊 (Brennan 鞋坊) | 皮鞋修缮坊。店铺区（展示柜台、铸铁鞋楦）、后方卧室区（含开放厨房）、地下室（有后院独立入口）。Unit2 主要场景，Frank 死亡地点。Zack 的私人伤口——南区清退从背景机器变成主角生活的现实 |
| Frank Kowalski 家 | 与 Margaret 鞋坊同一街角。典型南区工人住房：主屋开间（客厅+床+开放厨房一体）、次卧（Danny）、厕所（后巷小窗）。床与墙缝藏有密码锁铁盒 |
| O'Hara 家 | Frank 家隔壁约 5 米。工人家庭住宅，O'Hara 与女儿 Sarah 住所。Unit4→Unit5 衔接中 O'Hara 死亡、留下 Tidewater/Silver Moon 清退材料 |
| Thomas Sullivan 公寓楼 | 多层砖楼。一楼 Sullivan 夫妇、二楼 Seamus Byrne（走廊吸烟观察位）、三楼 Thomas 与 Helen。楼顶天台（案发地）有护栏。Unit3 坠楼案 |

## 码头区 (Dock Area)

艾斯弗德河汇入卡夏瓦湖的河口区域。与南区居民区地理相邻——Unit3 结局揭露"被清退地块与 1912 年 Tidewater 收购地块地理相邻"。

| 场所 | 说明 |
|------|------|
| 1912 旧码头地块 | 第一次南区清退的起点。港口工程地基坍塌事故现场，Sean O'Malley 被谋杀后抛尸于此。事故后 Tidewater 低价收购 |
| Tidewater 仓库 / 商业区 | 1912 年收购码头地块后建设的仓库、码头配套、工人宿舍和商业区 |
| 港口货运码头 | 繁忙货运活动，工人阶级聚集地 |
| 工会活动场所 | 1903 年码头罢工旧址，Liam Donnelly（Mickey 之父）殉难处 |

## 世博会开发预定区

偏西南方向。世博会选址窗口让南区土地再次升值，也让 1928 年清退突然加速。这片区域是 Tidewater "南区综合商业开发计划"的终极目标——清空工人住宅区后在此开发商业地产，套现世博会带来的地价与开发资格。

> 按主线揭露层级排列，标注各场所在叙事中的系统功能。

| 主线层 | 核心场所 | 功能 |
|--------|----------|------|
| 第一层：干净的脸 | Miller 庄园、圣心医院、Emma 事务所 | 慈善、资助、体面——Miller 的善意形象 |
| 第二层：金融与司法 | 湖滨信托银行、Lakeshore 大楼、法院 | 债务陷阱、投资壳、司法保护——系统的金融骨架 |
