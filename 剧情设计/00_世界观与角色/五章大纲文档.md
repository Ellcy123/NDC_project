🎯 Miller集团完整投机计划

第一次投机（1912年港口地块）

时间：1912年

目标：港口周边工业区地块

手段：
1. Miller集团收买工会内部人员
2. 故意制造港口"意外"事故
3. 工人死亡 → 家属急需钱 → 低价收购房产
4. Sean O'Malley第一个发现安全隐患，向工会举报
5. 工会泄露给Miller，Sean被安排到最危险区域
6. Sean死于第一次崩塌，年仅41岁
结果：
- Miller成功低价收购港口周边大片地块
- 16年后（1928年），该区域成为芝加哥最繁华的商业区
- Miller集团赚取巨额利润（保守估计$5000万以上）
受害者：
- Sean O'Malley（Emma的父亲，41岁）：第一个发现安全隐患的工人，向工会举报，被工会泄露给Miller，被安排到最危险区域，死于第一次崩塌
- Mary Fitzgerald O'Malley（Emma的母亲，37岁）：丈夫重伤后为其输血，后因感染在1913年去世，享年38岁
Emma的特殊悲剧：
- 1912年父亲Sean死于事故，母亲Mary为救丈夫输血后感染
- 1913年母亲去世，9岁的Emma成为孤儿，进入芝加哥圣玛丽孤儿院
- Miller集团用"慈善"资助孤儿上学（包括Emma），实为掩盖罪行
- Emma被资助者正是杀害她父母的凶手

第二次投机（1928年南区清退计划）

时间：1928年

目标：1912年港口地块周边的贫民区（现已升值）

逻辑链：
1. 1912年Miller低价收购港口地块 → 建设商业区
2. 16年后港口地块成为繁华商业区 → 周边贫民区地价暴涨
3. 周边贫民区居住的正是1912年事故工人的家属和后代
4. Miller要清退贫民区 → 扩张商业区 → 代际剥削完成
四条剥削线：

1. 债务陷阱（Unit 2）：
  - Moore First Bank诱导掠夺性贷款
  - 目标：夺取南区工人房产，清空"贫民窟"
2. 保险欺诈线（Unit 3）：
  - Continental Bank（与Moore合作）诱导债务人杀妻骗保
  - 债务压垮家庭 → 保险代理人"建议"杀妻骗保 → 银行从保险理赔中分利
  - 目标：既夺取房产，又从保险金中获利（双重剥削）
3. 暴力清场（Unit 1-4）：
  - Whale（Mickey） + 黑社会
  - 直接清除不配合的"障碍"
4. 法律掩护（Unit 1-4）：
  - Morrison（腐败警察）+ Harrison法官
  - 为Miller集团提供法律保护伞
系统性设计：
- Miller集团不直接出面
- 通过银行、黑社会、警察、法官的多层代理人执行
- 每一层都可以"合法"否认与Miller的关系
- 底层人只看到直接压迫者（银行、黑手党），看不到真正幕后黑手
