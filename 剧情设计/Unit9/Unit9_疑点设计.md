# Unit9 疑点系统设计

> 严格对齐 Miro mindmap 中 🔴 标红的疑点节点 + "四大疑点"总结节点。
>
> **机制（进度 = 疑点 + 疑点碎片）**：
> - 🔴 疑点：本循环内 2 样东西（🟢证据 / ⚪证言 / 🩷口述）都拿到 → 直接合成完整疑点
> - 🔸 疑点碎片：本循环只拿到一半，需要和另一循环的另一半拼合 → 合并成完整疑点
> - 完整疑点是解锁 EXPOSE 按钮的前置条件
> - 所有指证用到的证据/证言都必须被纳入至少一个疑点或疑点碎片
> - 疑点文本严格复用 Miro mindmap 原文（包含"四大疑点"节点），不自行扩展
>
> **ID 占位符**：
> - 完整疑点：`U9-D{NN}`（按全 Unit 顺序编号，与指证使用循环无关）
> - 疑点碎片：`U9-F{NN}`
> - 证据/证词 ID 仍为占位符，EPI 正式编号待挂接后统一替换

#### 🔴 U9-D01：23:30 的枪响真的是在会客室发生的吗？

```
组成：
  🟢 Vivian 小手枪（分析玩法：无硝烟反应）
+ ⚪ Rosa 目击证词"11:30 亲眼看见 Vivian 开枪"

同循环两样齐 → 直接产出疑点
```

**玩家心理**：Rosa 说 Vivian 11:30 开枪，但 Vivian 的手枪没有硝烟——要么手枪没开过，要么 11:30 的枪声不是从这把枪发出来的。Zack 又没搜到别的枪或弹壳，那么 11:30 的枪响到底是不是在这间房里发生的？

**对应指证**：L1 R1（教学失败——本作唯一一次"指证打不穿"的系统教学：单一条件的疑点不能击破多条件的谎言）

---

#### 🔴 U9-D02：Webb 本人真正怀疑的凶手是谁？

```
组成：
  🟢 Webb 和 Zack 的委托协议书（保护 Vivian + 确保继承遗产）
+ ⚪ Emma 大门口对话（Webb 不会紧急雇一个侦探，除非预见自己会死）

同循环两样齐 → 直接产出疑点
```

**玩家心理**：Webb 提前做了两件事——雇侦探 + 布置遗产继承。他预见到自己会死。那么他到底在防谁？不会是当晚"意外"在场的 Vivian 那么简单。

**对应指证**：L4 R3（Vivian 指证最后一轮——Vivian 不是 Webb 防的那个人，Whale 才是）

#### 🔴 U9-D03：Webb 的酒吧生意到底在做些什么？他还在干其他的事吗？


**对应指证**：
- L2 R1（用勒索照片反驳"正当生意"）
- L2 R2（用账目差反驳"收入都来自私酒"）
- L2 R3（Tommy 退守到"古董买卖"——用 🟢 花瓶/油画现场照反驳"东西卖给了 Coleman/Thompson"。本轮证据不单独构成疑点，属于同一疑点解锁后的谎言层级递进）

```
同循环两样齐 → 直接产出疑点
```
