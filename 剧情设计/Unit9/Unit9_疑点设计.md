# Unit9 疑点系统设计

> **机制（进度 = 疑点 + 疑点碎片，二层合并）**：
>
> - **疑点**：两件证据/证言在玩家手里**同时存在**后产出
>   - 同循环两件齐 → 直接产出疑点
>   - 跨循环：先一件，后一件——先有的那件以"碎片"形式存着，后来的一件到手 → 合并成疑点
> - **疑点碎片**：单独一件证据/证言，等待跨循环的另一件来配对
> - **合并只发生一次**：碎片 + 另一件 = 疑点（**不允许**碎片 + 碎片 = 疑点的三层合并）
> - **疑点是 EXPOSE 按钮的解锁门槛**：每循环有一组 clearDoubts，全部解锁后才能指证
> - **文本来源**：L1–L4 + 转折的疑点严格复用 Miro mindmap 🔴 节点 + "四大疑点"节点；L5/L6 按同一套机制方法推导（image 示意）
>
> **ID 占位符**：
>
> - 完整疑点：`U9-D{NN}`
> - 疑点碎片：`U9-F{NN}`
> - 证据/证词 ID 仍为占位符

## 二、疑点碎片总览（2 个）

| ID | 获得于 | 碎片文本 | 单件来源 | 等待的另一件 | 合并为 |
|:---:|:---:|:---|:---|:---|:---:|
| U9-F01 | L2 | Jimmy 很缺钱（碎片） | 🩷 口述"Jimmy 很缺钱"（厨房场景） | L5 ⚪ Anna 怀孕证言 | U9-D13 |

**说明**：

- L1 获得的玻璃碎片、Rosa 给的 .45 弹壳虽然都在 L1 就进入玩家物品栏，但它们在本 Unit 里**不先以碎片形式存在**——到它们参与的疑点组成时（L3 / L4 / L4转折），另一件也同时在手里，直接成疑点。
