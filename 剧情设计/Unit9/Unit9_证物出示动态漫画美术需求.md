# Unit9 证物出示·小方格动态漫画美术需求

> **用途**：本文档列出 Unit9（EPI09）中所有需要小方格动态漫画呈现的"证物出示"瞬间。每格小漫画必须按"单一局部视角"作画——指定机位、画面内元素、画面外暗示、视觉重点；不使用全景对比或上下分屏。
>
> **关键约束**：
> 1. **一格一物证**——每格小漫画只能呈现一件物证。即使原 Expose 设计是组合出示（如 R1 要求 [9702, 9101] 同时摆出击破口径），也必须拆成多格。
> 2. **拆分判断基于击破维度**——同维度重复证明的物证只画 1 件代表，不同维度的物证必须拆开各画 1 格。
> 3. **不许同框延续**——第 2 格画面里完全不出现前一格物证（不能虚焦背景），靠前后帧记忆完成对照。
> 4. **证词不外显**——游戏没有"对话气泡 / NPC 入画暗示证词"功能。漫画画面内只呈现物证 + 出示者/接收者的局部反应；证词类信息（如 9033001 倒垃圾时间、9075001 厨房口音、9085001 Anna 怀孕、9054002 Tommy 渠道）一律由游戏内对话脚本承载，不在漫画里画对话气泡或代表性 NPC 入画。
>
> **取舍原则**：每场指证（Expose）取最戏剧化的出示瞬间；L5 指证 James 因证据链最复杂取多格；Talk 中的所有玩家/Zack 主动出示节点全部保留。
>
> **数量统计**：Expose 17 格 + Talk 5 格 = **22 格小方格动态漫画**

### 共用画面规约

- 全部 22 格统一使用**单一局部视角**——以**手部 + 物证 + NPC 局部反应（半脸/手势/警徽反光）**为主，避免全景人物。
- **一格一物证**：画面内不能同时出现两件并列物证；如需对比，靠前后两格的观众记忆完成。
- **画面外**栏的内容是给观众"想象"的（典型如 NPC 完整表情、远景），不入画。
- **视觉重点**栏指定每格观众第一眼应捕捉到的元素，决定光照与对焦中心。
- 1920s 芝加哥时代质感：纸张泛黄、铅笔字、烟灰、雪茄烟雾、煤油灯反光、皮手套、黄铜枪身。
- 跨格重复物证（9103 出现 3 次、9401 出现 3 次）需保持同一物品的视觉一致性（同一份纸、同一枚弹壳）。
- 证词类信息（如 9033001 倒垃圾、9075001 厨房口音、9085001 Anna 怀孕、9054002 Tommy 渠道）**不出漫画**，也不在漫画画面里以对话气泡 / NPC 入画 / 时间道具暗示——全部由游戏内对话脚本承载。
