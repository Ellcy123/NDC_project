## 美术呈现约束

**每一格方格漫画只能展示一个局部画面。** 不能像电影分镜那样展现全景或多人对比构图。每格的"美术画面建议"必须明确写清四件事：

1. **构图机位**：低机位仰角 / 俯视特写 / 贴脸近景 / 侧面 3/4 角度等
2. **画面内**：具体到占画面比例的元素（人物的哪个部位、道具、环境特征）
3. **画面外**（不画但要表达）：通过视线方向、肩膀一角、灯光氛围、声音文字等间接传达
4. **视觉重点**：这一格要传达的核心信息或情绪

不要写"A 和 B 的对比"这种需要全景的描述——拆成"画面只取 A，B 通过 X 暗示"。

## 二、事件详细设计

### 事件 0 · L1 酒保拒绝 → Emma 暗语引路

**Loop / 场景**：L1 / 9004 蓝月亮酒吧大门口

**事件类型**：嵌在 L1 Opening cutscene 起始瞬间（无玩家操作）

**叙事作用**：
- 开篇第一组角色基调——确立 Zack"褪色风衣的私家侦探"与 Emma"已经混进上流圈的女记者"的反差关系
- 通过"King Oliver sent us."暗号点明蓝月亮的地下俱乐部属性（禁酒令背景）
- Emma 报暗号的从容感为后续她"主动引路"的角色定位埋伏笔

**触发条件**：游戏开场自动进入 L1 Opening；该方格漫画位于 Opening 最起始，先于 Zack/Emma 进入大堂、先于 23:30 枪响。

**角色分工与动作**：

| 角色 | 位置 | 动作 / 心境 |
|------|------|-------------|
| Zack | 酒吧门外台阶 | 一身褪色风衣，被酒保上下打量；不卑不亢但显然不属于这里 |
| 酒保 | 酒吧大门内侧 | 单手按门、冷脸堵路；典型禁酒令时期看门人——只放熟客和暗号通过 |
| Emma | Zack 身侧半步 | 主动上前一步、压低嗓音报暗号；姿态熟练，说明她不是第一次来 |

**美术画面（2 格）**：

**① 局部：门内视角的酒保特写**
- 构图机位：低机位 / 仰角，画面只取酒保的上半身
- 画面内：酒保的脸（占 1/3，单眉微挑、视线向下打量）、按门的手（占 1/3，手指紧扣门框）、门缝透出的暖黄灯光与模糊人影（占 1/3，暗示门内是热闹的酒吧大堂）
- 画面外：Zack 在画面下方台阶上——通过酒保俯视的视线方向 + 画面下沿露出的 Zack 风衣肩膀一角暗示
- 视觉重点：酒保"看门狗"姿态——禁酒令时期典型 speakeasy 守门人的拒绝感

**② 局部：Emma 报暗号瞬间的双人贴脸近景**
- 构图机位：齐胸景，画面只取 Emma 与酒保的脸侧，Zack 不入画
- 画面内：左侧 Emma 半张侧脸、嘴形微张正在低声说话（嘴边可有一缕呼出的白气增强"压低声音的私密感"）；右侧酒保的脸由"冷漠"切换到"会意"的瞬间——眉毛微挑、视线从 Emma 移向门外
- 画面外：Zack 站在 Emma 身后——通过 Emma 肩膀后方一角风衣布料 + 酒保已经在看向他的视线方向暗示
- 视觉重点：暗号生效的"机关咔哒一声"瞬间，Emma 的从容与酒保态度的反转

**台词建议**（嵌入 Opening 旁白 / 字幕）：
- 酒保："Members only, pal."（拦）
- Emma（压低）："King Oliver sent us."
- 酒保（短暂停顿后侧身）："...Have a good night."

**下游影响**：
- state：[loop1_state.yaml](state/loop1_state.yaml) opening.description（约第 67-78 行）已包含此情节，可在 description 中追加注释 `# 此段以小方格漫画形式呈现`

### 事件 1 · L2 Emma 接相机失手 → 柜底翻出勒索照片

**Loop / 场景**：L2 / 9006 歌舞厅小包厢

**事件类型**：⚡ 突发事件（自由探索期玩家动作触发）

**叙事作用**：
- 双产出节点——同时获得 9209 便携相机（拍摄工具）+ 9201 勒索照片（仙人跳产物）
- 相机打碎是引出柜底信封的"桥梁动作"，让"找到勒索照片"不显得过于直白——玩家是因为"蹲下捡碎片"才偶然摸到柜底信封
- 揭示蓝月亮 VIP 小包厢的真实用途（勒索拍摄角落）

**触发条件**：玩家主动点击歌舞厅小包厢角落矮木柜 → 拉开发现便携相机 → 递给 Emma 时失手。

**节奏（精确匹配 Loop2 草稿 §2 节点 901002050–901002059）**：
1. Zack 拉开矮木柜，从里面取出一台落灰的便携相机（机身有薄灰，卷片轴上仍留有底片）
2. Zack："这个相机还有底片。" Emma："35mm 的便携相机？怎么会出现在这？" → 玩家获得 9209 入背包
3. Emma 接相机时失手打碎——Zack 眼疾手快接住相机本体但碎片洒落，Emma 退后半步喃喃："这……我也……"
4. 收拾碎片时 Zack 蹲下，手伸进柜底——指尖碰到一个硬角，抽出来是一个厚信封
5. 信封中是两张暧昧合影（Vivian × Thompson / Coleman），背面有几处淡黄褐色污渍——这正是 9201 勒索照片原始未显影态 → 玩家获得 9201 入背包

**美术画面（2 格）**：

**① 局部：相机滑落瞬间的手部特写**
- 构图机位：齐手景，俯视略斜
- 画面内：画面正中是脱手的便携相机（落灰的机身可见、卷片轴一角）、镜头朝上反光；右侧 Emma 的两只手张开但没接到（指尖距离相机有半掌的"差一点点"距离）；左侧 Zack 一只手伸出来已经接到机身但来不及接到一片碎玻璃——一片镜片碎玻璃斜划出画面边缘
- 画面外：两人的脸不入画——通过手部紧张的张力 + 相机滑落的运动模糊轨迹，让观众脑补 Emma 那句"这……我也……"的慌乱
- 视觉重点：失手的瞬间"差一点点"的停格感——相机即将砸地但还未砸到的张力

**② 局部：Zack 蹲在柜底从厚信封抽出照片的特写**
- 构图机位：低机位 / 侧斜角，模拟蹲下时的视角
- 画面内：画面左下角是地上散落的玻璃碎片（占 1/4，作为"为什么会蹲下"的因果暗示）；画面正中是 Zack 一只手伸进矮柜底部、另一只手已经把厚信封抽出来一半——信封口微开，露出黑白照片的一角（背面有几处淡黄褐色污渍，但具体内容看不清）
- 画面外：Emma 也蹲下来——通过 Emma 的一只手伸进画面右上角（指着信封）+ 一句对白框"Brennan！柜子底下还藏着这个！"暗示她在 Zack 旁边
- 视觉重点：信封从隐蔽处现身的"翻出真相"瞬间——以"看不清照片内容"留悬念给后续 9201 分析

### 事件 2 · L2 Tommy 抽屉弹壳惊鸿一瞥

**Loop / 场景**：L2 / 9010 Tommy 办公室

**事件类型**：⚡ 突发事件 + 证据预曝光（道具不入背包，留待 L5）

**叙事作用**：
- L2 仅以"视觉惊鸿一瞥"形式预曝光 .45 警用弹壳（item 9506），玩家此时不获得任何道具
- 留待 L5 Tommy 交付信任后才从抽屉取出——是 Tommy 角色弧线"从拒绝到信任"的物理凭据
- 弹底印记 "MILLER ARMS" 与本案 9401 同批次——为 L4-L5 米勒集团线埋伏笔

**触发条件**：玩家点击 Tommy 办公桌半掩的抽屉（抽屉里有金属物件微微反光作为视觉提示）。

**节奏（精确匹配 Loop2 草稿 §3 节点 905002400–905002405）**：
1. Zack 注意到 Tommy 办公桌上半掩的抽屉，里面有金属物件反光
2. Zack 伸手拉开抽屉，看见一枚生了锈的 .45 警用弹壳躺在文件最深处
3. Zack 评论："带着底印的警用弹壳？你留着这玩意儿当护身符？"——只来得及看清弹壳底部的特殊印记
4. Tommy 猛地站起身，"啪"地把抽屉推回锁死，语气变得又冷又硬："这是我自己的事。跟今晚的案子没半点关系。"
5. Zack 举手退后："行。你的地盘。"——不硬抢

**美术画面（2 格）**：

**① 局部：半掩抽屉内的弹壳特写**
- 构图机位：俯视，模拟 Zack 拉开抽屉低头看的视角
- 画面内：画面正中是被拉开的抽屉一角（占 2/3）——抽屉里堆着发黄的文件、账本边缘、一支钢笔；最深处文件的缝隙间露出一枚生锈的 .45 警用弹壳，弹壳底部的特殊印记（"MILLER ARMS"字样）在台灯黄光下隐约可见但又不够清晰
- 画面外：Zack 不入画——只通过画面顶部边缘伸进来的 Zack 一只手（拉抽屉的动作）暗示
- 视觉重点：弹壳的"惊鸿一瞥"——印记字样模糊到玩家能感觉"这上面有东西"但不能完全识读，制造"想看清又看不清"的钩子

**② 局部：Tommy 推抽屉锁死瞬间的手部特写**
- 构图机位：齐胸景 / 侧面
- 画面内：画面正中是 Tommy 一只手"啪"地按在抽屉上的瞬间——手指张开按死、动作幅度大、伴随抽屉砸进桌体的运动模糊；右侧露出 Tommy 的下巴和半张嘴的局部（嘴角紧绷）
- 画面外：Zack 站在桌对面退后——通过画面左侧边缘 Zack 一只手举起的"我退我退"姿态暗示；弹壳已被遮住——通过抽屉关闭的视觉切断"再看一眼"的可能
- 视觉重点：Tommy 的反应力度——这是"被踩到地雷"的本能防御，而不是普通的"别翻我抽屉"

**台词建议**：
- Zack（看到弹壳时）："带着底印的警用弹壳？你留着这玩意儿当护身符？"
- Tommy（推抽屉锁死）："这是我自己的事。跟今晚的案子没半点关系。"
- Zack（退后）："行。你的地盘。"

**② 局部：Zack 蹲下从地砖捡起烧焦碎片的特写**

### 事件 4 · L5 炉锅沸溢 → James 口音暴露

**Loop / 场景**：L5 / 9009 James 厨房

**事件类型**：⚡ 突发事件（无人引起的厨房事故，James 反射性脱口）

### 事件 5 · L6 Whale 调查电话
