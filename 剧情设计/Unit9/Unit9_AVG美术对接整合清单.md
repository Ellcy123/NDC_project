## 三、突发事件

### 事件 0 · L1 酒保拒绝 → Emma 暗语引路

> Loop / 场景：L1 / 9004 蓝月亮酒吧大门口
> 类型：嵌在 L1 Opening cutscene 起始瞬间（无玩家操作）
> 触发条件：游戏开场自动进入 L1 Opening；该方格漫画位于 Opening 最起始，先于 Zack/Emma 进入大堂、先于 23:30 枪响

**叙事作用**：
- 开篇第一组角色基调——确立 Zack"褪色风衣的私家侦探"与 Emma"已经混进上流圈的女记者"的反差关系

- Emma 报暗号的从容感为后续她"主动引路"的角色定位埋伏笔

**在场角色**：酒保、Emma、Zack

**美术资产**：待美术设计

**音效**：
- 关门声脆响
- 三道插销依次拉开（"咔哒——咔哒——咔哒"，由内向外）
- Emma 入场前奏（脚步声 / 衣料摩擦声，可选）

### 事件 4 · L5 炉锅沸溢 → James 口音暴露

> Loop / 场景：L5 / 9009 James 厨房
> 类型：⚡ 突发事件（无人引起的厨房事故，James 反射性脱口）
> 触发条件：三条问询分支汇合后强制触发——对话进行到一半，炉子上的锅突然剧烈沸溢（高汤漫出、蒸汽冲起，或壁炉旁堆放的木柴/托盘因火势窜高而倾倒），无人引起的厨房事故

**叙事作用**：
- 揭示 James 是爱尔兰裔移民——"Don't be bothering me!"是 Hiberno-English 特有句法
- 与 Vivian 在 23:05 门口听到的那句话完全一致（证词 9064001）→ 锁定"23:05 隔门吼 Vivian 的人是 James，不是 Webb"
- 口音特征以证词 9075001 记录，供 L5 Expose R2（口音身份）使用——非物证形态，靠"亲耳听见"完成
- 关键设计：James 不是对 Zack 说，而是对失控的炉火/沸溢的反射性咒骂——是他忙乱中压不住的本能反应（如果是对 Zack 说，玩家会怀疑是表演；对锅说=真实下意识）

**在场角色**：Zack、James

**美术资产**：待美术设计

**音效**：
- 高汤溢出咕嘟声 + 蒸汽喷射"嘶——"声 + 金属托盘掀翻砸地"哐"声（三层叠加）

**台词**：
- James（对锅，反射性）："Don't be bothering me!"（字幕标注：Hiberno-English / 爱尔兰口音）
- James（用"做菜烦躁"遮掩）："……抱歉，这锅不能洒。"
- Zack（心理旁白）：[Vivian 在门口听到的那句话……是这个声音。]
