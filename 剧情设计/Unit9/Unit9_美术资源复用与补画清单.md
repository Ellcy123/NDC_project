### 🆕 立绘 ① `EPI09\SC9003_npc_Vivian1/2` — Webb 会客室的 Vivian

**出现 Loop / 场景任务状态**：Loop 1 / 9003 Webb 会客室（🎬 限制探索）

**角色情境**：案发现场。Vivian 站在 Webb 尸体旁——她手里握着一把小手枪（9101），Rosa 刚刚当场指控她杀了 Webb。Vivian 本人**酒后迷糊**，刚推门发现 Webb 已死，此刻分不清"是不是自己刚刚杀了他"。

**立绘关键**：
- 情绪：**恍惚 + 震惊 + 懵**（不是凶手得意也不是无辜抗议）
- 姿态：僵在原地，手里有枪（但不是举枪姿势——她刚来不久，还没来得及反应）
- 妆容/着装：演出妆 + 演出服（她演完出来准备会客）
- 脸色：因酒意微微发红，但表情凝固

**不要画成**：凶手得意状 / 无辜委屈状——她当下的核心情绪就是"不知道发生了什么"。

**其他情绪变种建议**（若资源允许，可选）：点击态可以比基础态更惊恐一点，嘴唇微抖。

### 🆕 立绘 ② `EPI09\SC9003_npc_Rosa1/2` — Webb 会客室的 Rosa

**出现 Loop / 场景任务状态**：Loop 1 / 9003 Webb 会客室（🎬 限制探索）

**角色情境**：案发现场。Rosa 是当晚的清洁工，案发时在储藏室打杂。她是"第一个到达现场的证人"——站在 Vivian 身旁，**指着 Vivian 激动喊"我亲眼看见她杀了 Webb 先生！"**。但 Rosa 心中其实极度恐惧——她当晚收到了倒写 J 的威胁纸条，被迫指控 Vivian。

**立绘关键**：
- 情绪：**表面激动指控 + 内心恐惧**（外表是控诉者，内核是被胁迫者）
- 姿态：手臂指向 Vivian（或指向已故 Webb 方向），肩膀紧绷
- 着装：酒吧清洁工制服（围裙）
- 脸色：苍白，眼神闪烁（不敢直视 Vivian 的脸）

**重点传达**：她不是冷静陈述，而是略带颤抖的激动——这样 L1 玩家就能感受到"她说的可能不全是真的"的不安感（悬念铺设）。

### 🆕 立绘 ③ `EPI09\SC9003_npc_Morrison1/2` — Webb 会客室的 Morrison

**出现 Loop / 场景任务状态**：Loop 1 / 9003 Webb 会客室（🎬 限制探索）

**角色情境**：案发现场。Morrison 是初到现场的警探，L1 他的设定是"**嫌麻烦但姑且秉公**"——不暗示被收买（被收买伏笔 L4 才揭示）。他在现场进行初步勘验，准备带 Vivian 回警局。

**立绘关键**：
- 情绪：**职业漠然 + 略显不耐烦**（不是正义警察，但也还没暴露腐败）
- 姿态：手插口袋或抱臂，身体微微前倾观察尸体
- 着装：1920s 警探制服（风衣 + 警帽），胸前警徽
- 脸色：平静，眼神打量但不过多情绪

**不要画成**：愤怒/紧张/惊讶——他是见过太多案子的老警察，对尸体无特别反应。

**注意**：Morrison 在此场景中**不能暴露**任何"他认识这把手枪/这个弹壳品牌/米勒集团"的细节（这些是 L4-L6 才揭示的）。

### 🆕 立绘 ⑤ `EPI09\SC9015_npc_Anna1/2` — James 家客厅的 Anna

**出现 Loop / 场景任务状态**：Loop 5 / 9015 James 家客厅（🔓 自由探索）

**角色情境**：L5 Anna 首次出场。Anna 是 James 的妻子，**怀孕中后期**（根据 state，她说"孩子快出生了"，推测孕晚期）。她在家里迎接 Zack 和 Emma 的拜访。她**不知道 James 是凶手**——只知道丈夫接了一件"重要的差事"，能解决经济困难。

**立绘关键**：
- 情绪：**略带警惕但本质温和**（孕期疲惫 + 对陌生访客的防备 + 爱尔兰移民的隐忍）
- 姿态：扶着肚子或倚在门框/椅子旁（孕妇自然姿态）
- 着装：1920s 爱尔兰移民家庭主妇简朴连衣裙 + 围裙；色调暖但低饱和（米白/浅棕/淡蓝）
- 脸色：疲惫但温和，眼神有一丝"不愿多事"的克制
- 发型：简单盘起或扎低马尾（劳动妇女风格）

**重点传达**：
1. 孕期体态明显（肚子微微凸起，一只手本能托在腹部）
2. 爱尔兰裔特征（若可能：浅发 / 淡雀斑 / 苍白肤色）
3. 克制的温和——不是外向，也不是冷漠

**不要画成**：年轻俏皮女主人 / 华丽贵妇——她是被经济压力压着的移民主妇。

**其他情绪变种建议**（若资源允许，可选；但对 L5 末的情感转折非常有用）：
- 点击态可以稍显紧张（她内心其实知道丈夫最近不对劲）
- **（可选）Anna_fearful**：L5 末 James 自杀消息到达时用——脸色苍白、掩嘴、眼眶泛红
- **（可选）Anna_collapsed**：听闻 James 死讯后的悲伤姿态——扶墙、双手掩脸

**资源参考**：EPI01 `D:\NDC\Assets\Resources\Art\MovieClip\epi01\AVG\jimmyhome_anna*` 系列（5 张 AVG 分镜图：base / concerned / fearful / emmatoanna / emmatoanna_pained）可作为**人物设计参考**（同一角色 Anna 的 Unit1 版本），但 Unit9 场景立绘需要重新画成"场景 NPC"规格（透明背景 / 可摆位）。
