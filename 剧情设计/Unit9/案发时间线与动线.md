## 一、空间布局（关键垂直关系）

```
┌────────────────────────────────────────────────┐
│ 2F  [Jimmy 厨房 9201]  ← Jimmy 整晚不在场 / 23:30 假枪声
│         │                 [Webb 办公室 9203][Tommy 办公室 9202]
│         │  ↕ 楼梯（Jimmy 23:00-23:05 下楼杀人、事后上楼归位）
│         │
│ 1F  [Webb 会客室 9101]  ← 案发现场（Jimmy 厨房正下方）
│         │                 [酒吧大堂 9102][酒吧歌舞厅 9103]
│         │                 [歌舞厅小包厢 9104][一楼走廊 9105]
│         │  窗户朝外
│         ▼
│     [一楼侧巷 9106]  ← 玻璃碎片掉落处 / Rosa 23:20 垃圾袋压碎片
│
│ B1  [Rosa 储藏室 9302][Vivian 化妆室 9301]
│
│ 酒吧外  [Jimmy 家][Morrison 家 9402][蓝月亮后门 9007]
└────────────────────────────────────────────────┘
```

### 诡计的空间诗学

Jimmy 厨房（2F 南侧）就在 Webb 会客室（1F 南侧）正上方。
- Jimmy 先下到 1F 杀 Webb，事后返回 2F 厨房
- 23:30 再从 2F 厨房窗口朝外空射假枪声——枪声源自 2F 厨房，但**没人能自行精确定位**
- Rosa 按指令跑进 1F Webb 会客室指认 Vivian "刚刚开枪"，**所有听众的注意力被她的证词钉在 1F 会客室方向**
- 所有宾客/员工自动补完"Webb 刚被杀"的叙事
- **Vivian 是全场唯一从 1F Webb 会客室内听出枪声"从头顶方向传来"的人**——这是 L5 揭露假枪声诡计时的叙事伏笔（L4 post_expose 时 Vivian 给出该证言）

## 二、真实案发时间线（ground truth）

### 23:00（音乐高潮启动）

- 《芝加哥狂想曲》进入鼓点最强高潮段，音量 5 级
- **Vivian 上个月特意申请把 11 月 3 日节目从《Moonlight Bay》（月光湾）改为《芝加哥狂想》——动机：让 Webb 听到，和她最后共舞一曲**
- 空间：1F 歌舞厅（驻唱舞台 + 舞池 + 小包厢）

### 23:00–23:05（真实杀人）

- Jimmy 从 2F 厨房下楼到 1F Webb 会客室
- **用米勒集团特制手枪枪杀 Webb**（.45 ACP 口径；枪身侧面铭刻 "MILLER ARMS · CHI" 铭文 + 米勒集团狮形印记）
- 音乐高潮的鼓点掩盖了真实枪声（歌舞厅/大堂的宾客听不出 1F 会客室方向有枪响）
- 子弹穿透 Webb 后从 1F 会客室窗户射出，击碎玻璃，碎片落在同层 1F 侧巷地面（带有硝烟痕迹）

### 23:05（Vivian 敲门 / Jimmy 模仿 Webb）

- Vivian（11:05 从 1F 歌舞厅过来到达 Webb 会客室门口，带了 .25 ACP Colt Vest Pocket Model 1908，原本想亲手杀 Webb）敲门
- Jimmy 锁住 Webb 会客室门、模仿 Webb 口音吼"**Don't be bothering me!**"——爱尔兰式习语，后续 L5 口音谜题的伏笔
- Vivian 退走，准备 11:30 自己送酒的时间再来

### 23:05–23:10（Jimmy 归位）

- Jimmy 上楼返回 2F 厨房
- 把自己归位到 11:30 不在场的证明里

### 23:20（Rosa 倒垃圾 — 关键物证时序锚点）

- Rosa 从 B1 储藏室上 1F，走到侧巷垃圾区（Webb 会客室窗户外），把垃圾袋放下
- **垃圾袋正好压在 23:00–23:05 之间从 1F Webb 会客室窗户飞出的玻璃碎片上方**
- L3 推理链的锚点：「垃圾袋 23:20 → 玻璃碎片压在袋下 → 玻璃破裂必早于 23:20」

### 23:30（假枪声 + 误导指认）

- Jimmy 从 2F 厨房窗口朝外空射一枪——**假枪声**
- 枪声从 2F 厨房传出，但宾客没有精确听觉定位
- Rosa 按事先收到的匿名纸条指令（+ 女儿医疗单威胁）紧随其后跑进 1F Webb 会客室指认 Vivian "刚刚开枪"
- 证词把所有人的归因锚定在 1F 会客室
- 所有宾客/员工都误以为 23:30 才是 Webb 被杀的时刻
- Vivian 此时按 11:30 送酒惯例推门进入 Webb 会客室，看见 Webb 尸体后陷入混乱（手中 .25 柯尔特从未开过火）
- Vivian 听出的"枪声从头顶方向传来"——这条空间不一致作为揭露诡计的叙事线索（L4 post_expose 时由 Vivian 自己说出）
- Rosa 趁乱从地上捡走一枚 .45 弹壳（后来 L3 交给 Zack / Morrison 私藏了另一枚 L6 在家被找到）

### 23:30+（Zack 和 Emma 登场）

- Zack 和 Emma 从 1F 酒吧大堂穿过人群冲向 1F Webb 会客室
- 到达时：Rosa 站在门边 / Vivian 站在 Webb 尸体旁手握小手枪（面容呆滞）
- Morrison 随后赶到——他平时就常来蓝月亮喝酒，和 Webb 关系还过得去
- Morrison 本来醉醺醺地一副不想干活的样子；听说死的是 Webb，一边嫌麻烦一边还是姑且秉公办事
- 扫一眼现场就宣布 Webb 死亡，把目光钉在了手握小手枪的 Vivian 身上
- **此时 Morrison 尚不知本案会牵出 Whale**（L4 看见 .45 警用弹壳才变脸）

## 四、玩家还原推理链（分 Loop 披露节奏）

### L1 教学层（两轮指证连败）

- 玩家知道：Webb 死于 23:30、Vivian 持枪、Rosa 目击
- L1 R1 击穿：Vivian 手枪无硝烟 → **有局限**（Morrison 指出硝烟可能消散）
- L1 R2：玩家无其他可用证据，tutorial_failure
- L1 结束：换得 72h 调查期、Morrison 把 Vivian 带到酒吧一角做笔录问话（L2 期间玩家无法接触；仍在酒吧内，不同于 L4 末尾正式押走）
- 疑点 9101 挂起 → L2 翻页归档为"未解"（待 L3 回填）

### L2 生意性质层（揭勒索）

- 玩家还原：Webb 做敲诈勒索生意、Tommy 是前执行人、**现任勒索人另有其人**、背后牵涉 Whale
- 疑点 9201（勒索照片 VIP 身份重合）/ 9202（两本账差 $11,000）/ 9203（假古董掩护）全部解决
- 碎片 F1（Jimmy 很缺钱）挂起待合并

### L3 时间线层 + Rosa 被胁迫层

- 玩家还原：
  - 真实枪击发生在 23:20 之前（玻璃碎片压在垃圾袋下）
  - Rosa 是被胁迫作伪证（纸条 + 女儿医疗单）
  - 胁迫者书写特征：倒写 J
  - Webb 心里仍重视 Vivian（保险箱密码 = 相遇日）
  - Webb 生前已警觉到 Whale 是威胁（Webb 勒索日记扉页亲笔写 "Whale — Danger"）——L3 Emma opening 已口述过 Tommy 暗示的名字，9308 是 Webb 亲笔警觉的首次物证，情感重量加重；叙事证据、不挂疑点
- L1 疑点 9101 在历史面板回填真相
- 疑点 9301 / 9302 / 9303 解决
- post_expose：Rosa 交出 .45 警用弹壳（L4 F2 碎片基础）

### L4 口径层 + Whale 坐实

- 玩家还原：
  - 凶器是 .45 ACP，不是 Vivian 的 .25 ACP
  - Whale 已渗透酒吧（Rita 间谍）、已行动（合同）——从 L3 一个被 Webb 警觉的名字升级为有实际动作的主使
  - Webb 死亡区间锁定 23:05–23:20
  - Morrison 看弹壳**变脸**——玩家看到但不懂为什么
- 疑点 9401 / 9402 解决；F2 碎片（.45 弹壳）挂起
- **情节转折**：Morrison 强押 Vivian——"做对了所有事，系统拒绝接受正确的答案"

### L5 真凶层 + 动机闭合层

- 玩家还原（三合一推理峰值）：
  - 真实案发时间锁定 23:00–23:05（鼓点高潮掩盖枪声）
  - Vivian 在 23:05 听到的"Webb"是 Jimmy 模仿（爱尔兰习语 Don't be bothering me! + 爱尔兰裔身份）
  - Jimmy 动机：Anna 怀孕急需钱 + Whale 的委托合同 + 他作为现勒索人的渠道
  - 倒写 J = Jimmy（英语练习书对应威胁纸条）——L3 的 9302 胁迫者身份补完
- 疑点 9501 / 9502（F1 合并）/ 9503 / 9504 解决
- **Jimmy 自杀**——为保护 Anna；现场留下米勒集团特制手枪
- Zack/Emma 并肩

### L6 保护伞层 + Unit 结尾

- 玩家还原：
  - Morrison 看 .45 警用弹壳变脸 = 他认出米勒集团线索（L6 通过手枪铭文"MILLER ARMS · CHI"揭示）
  - Morrison 欠疤面煞星 5000 美元赌债（赛马），借了 Whale 的钱
  - Morrison 被 Whale 用债务钉在棋盘上，负责给 Vivian 快速结案
  - 凶器 = 米勒集团特制手枪（.45 弹壳 + 手枪口径 + 铭文匹配）
- 疑点 9601 / 9602（F2 合并）解决
- **Unit 结尾动画 Act1-4**：
  - Act1 制度性的正义（Morrison 被处分为"失职"而非腐败）
  - Act2 Vivian 选择留下经营蓝月亮帮非法移民拿身份
  - Act3 Vivian 用 9304 字条密码打开保险箱取出 9602 南区计划录音盘碎片
  - Act4 Zack 上留声机听到 Margaret Brennan + 南区清退名单 + 火海鞋坊定格（Unit2 开门砖）

## 八、玩家情感弧锚点

| Loop | 玩家体验 | 情绪 |
|:---:|:---|:---|
| L1 | 两轮指证连败，Emma 帮换 72h | 困惑中的行动本能——被迫开始跑，还没方向感 |
| L2 | 揭穿勒索生意，Webb 本人变成问题 | 认知升级 + 愤怒——他死了但原因比死亡本身重 |
| L3 | 翻转 Rosa（从撒谎者到被胁迫母亲）+ 时间线锁死 | 推理快感 + 心疼同步——悲悯 > 胜利 |
