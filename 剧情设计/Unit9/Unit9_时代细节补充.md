### 0.1 《月光小夜曲》年代错误

**问题位置：** `Unit9_大纲.md` L5 节目修改申请
> "她申请将 11 月 3 日《月光小夜曲》改为《芝加哥狂想》"

**问题原因：** 《Moonlight Serenade》历史上是 1939 年 Glenn Miller 作品，1925 年尚未存在。

**修改建议（三选一）：**
- **方案 A（真实 1920s 前乐曲）：**《Moonlight Bay》(1912) — 舒缓四拍，与"被换下的温柔曲目"意象契合
- **方案 B（真实 1920s 前乐曲）：**《Shine On, Harvest Moon》(1908) — 更诗意，更具"月夜"氛围
- **方案 C（虚构曲目，避开历史锚点）：**《月光湾》/《银河小夜曲》— 彻底架空，不会被考据党出戏

**只改曲名一处，不影响任何推理。**《芝加哥狂想曲》（主旋律）保持不变。

## 1. Vivian 的手枪：Colt Vest Pocket Model 1908

### 当前设定
`Unit9_大纲.md` L20：
> 持有的是一把 .25 ACP 柯尔特袖珍手枪

### 建议细化
明确为 **Colt Vest Pocket Model 1908（.25 ACP）**。

### 为什么改
- **Colt Vest Pocket Model 1908** 是 1920s 美国市场上最典型的"女士随身枪"——广告将其定位为"vest pocket protection for ladies"（女士背心口袋护身）
- 枪身极小（约 114mm，重 368g），可藏于晚礼服手包或吊袜带
- 与 Vivian"被保护的受害者"意象高度契合——她的枪从一开始就是"自卫用"而非"作案用"，这种时代印记强化了"她不是凶手"的直觉

### 修改位置
| 文件 | 位置 | 修改 |
|---|---|---|
| `Unit9_大纲.md` | L20 | "一把 .25 ACP 柯尔特袖珍手枪" → "一把 **Colt Vest Pocket Model 1908（.25 ACP 口径）**——1920s 典型的女士随身护身枪" |
| `Unit9_大纲.md` | L134 隐藏线索 | "Vivian 手里那把小手枪的具体型号" 保持不变（玩家此时仍不知型号） |
| `Unit9_大纲.md` | L360 L4 R1 | ".25 ACP" 可补一句"（Colt Vest Pocket Model 1908 / 1920s 女士护身枪制式）" |

### 可选加分项
- Vivian 化妆间的"痛苦记录"环境道具可加一条：**枪盒内衬着的一张 Colt 公司广告卡**（"A Lady's Silent Protection"），暗示 Webb 当年送枪给她是"保护"而非"武装"。
- 这一小小的视觉细节能让玩家在 L4 指证 Vivian 时，情感上更能接受"Webb 真的是在保护她"的反转。

## 2. Morrison 的赌债具体化：赛马输光

### 当前设定
`Unit9_大纲.md` L42 / L549：
> 欠疤面煞星 5000 美元赌债
> 🟢 Morrison 欠疤面煞星 5000 美元的赌债欠条，附带一封威胁信

### 建议细化
**Morrison 欠疤面煞星 5000 美元赌债 —— 输在 Washington Park 赛马场的一整个赛季。**

证据物理形态调整为：
- 🟢 **Washington Park 赛马场一整个赛季的输钱马票存根（厚厚一叠）**
- 🟢 **疤面煞星签字的 5000 美元欠条**（附威胁信）

两件合为一组"赌债证据包"，仍对应原 L6 R2 指证证据。

### 为什么改
- **Washington Park 赛马场** 是 1920s 芝加哥南郊（Homewood）真实的赛马场（1884 开场，1926 关场，1925 正值鼎盛）——真实地标落地
- 1925 年芝加哥警探年薪约 **$1,500–$2,000**，$5,000 赌债 ≈ 3 倍年薪 → Morrison 根本还不起 → **必须借 Whale 钱** → 被钉死在债务上
- 赛马 vs 抽象赌债：前者有具体地点、具体赌法、具体经济数字，玩家看到"一整个赛季的马票存根"时，能直观感受到 Morrison 的堕落不是"一次失足"而是"系统性上瘾"
- 为 L6 的 Morrison 指证提供更强的情绪落点：他不是坏人，他是被自己的瘾毒害的人

### 修改位置
| 文件 | 位置 | 修改 |
|---|---|---|
| `Unit9_大纲.md` | L42 | "欠疤面煞星 5000 美元的赌债" → "欠疤面煞星 5000 美元赌债——输在 Washington Park 赛马场的一整个赛季" |
| `Unit9_大纲.md` | L549 | 分成两件物理证据："🟢 Washington Park 马票存根（一整赛季输光记录）🟢 疤面煞星的 5000 美元欠条 + 威胁信" |
| `Unit9_大纲.md` | L558 碎片 U9-F03 | "Morrison 欠了一大笔赌债" 描述可补"（赛马输光）" |
| `Unit9_大纲.md` | L576 L6 R2 | 击破证据从 "🟢 赌债欠条" → "🟢 赌债欠条 + 🟢 马票存根"（两件合为一组，但 R2 的逻辑不变） |

### 可选加分项
- 其中一张马票背面有铅笔潦草记录的"Scarface — 3pm Wed"字样，暗示 Morrison 和疤面煞星的碰头节奏 → 给 L6 R3（Whale 关系）留一条隐性暗线，但不强制使用

## 3. Speakeasy 开场：观察孔 + 暗号（Loop1 开篇）

### 当前设定
`Unit9_大纲.md` L100：
> Zack 和 Emma 站在蓝月亮酒吧门口，酒保拦住 Zack——是 Emma 把他带进去的，她对"今夜会诞生新闻"有记者的嗅觉。两人边聊边进入大厅，爵士乐和烟雾裹住他们。

### 建议补写
在"酒保拦住 Zack"之后、"两人边聊边进入"之前，插入一段 speakeasy 标志性入场仪式：

> 橡木重门上钉着一块黄铜观察孔盖。酒保滑开小窗，只露出一双审视的眼睛。
> "今晚有包厢吗，先生？"
> Emma 不慌不忙，从手包里抽出一张 Blue Moon 会员卡，贴近观察孔，低声说："**King Oliver sent us.**"
> 观察孔盖啪地合上，三道插销依次拉开，门内飘出烟雾、低音贝斯和威士忌的气息。

### 为什么加
- **观察孔 + 暗号 + 会员卡** 是 1920s speakeasy 三件套——真实且高辨识度
- **King Oliver** 是 1922–1924 在芝加哥 Lincoln Gardens 驻场的真实爵士乐手，Louis Armstrong 的导师，是当时芝加哥地下酒吧老玩家都知道的名字——玩家（或考据党）一眼就能定位"这是 1920s 芝加哥"
- 零成本拉满时代质感；同时给 Emma 一次展现她"记者嗅觉 = 广泛的人脉"的机会（她为什么有 Blue Moon 会员卡？她认不认识 King Oliver 本人？留给 Unit2+ 回收的悬念）

## 4. Jimmy 的爱尔兰裔身份再锁死

### 当前设定
`Unit9_大纲.md` L27 / L445：
> **Jimmy**（约 35 岁，爱尔兰裔二代移民）
> 🟢 Jimmy 家里的英语练习书——是 Jimmy 勒索了 Rosa（倒着写错的 J）
> 🟢 Jimmy 的爱尔兰裔身份

### 建议细化

**4.1 英语练习书具体为 Berlitz 教材**
- 物品名称：🟢 **Berlitz Method for English Learners（贝立兹英语自学教材）**
- 历史支撑：Berlitz Language Schools 成立于 1878 年，1920s 在美国极其流行的成人英语自学品牌，移民常用
- 扉页签名：**"James O'Connor, Oct. 1922"**（移民登陆后三年开始学英语，与"二代爱尔兰移民"身份一致）
- 倒写 J 的笔迹特征：出自扉页签名的"James"——玩家看到扉页 J 的倒写 → 对上威胁纸条的 J → 完成身份闭合

**4.2 姓氏锁定：O'Connor**
- Jimmy 的全名：**James "Jimmy" O'Connor**（O'Connor 是最典型的爱尔兰姓氏之一）
- 在 Jimmy 的工资单、厨房工作表、给 Whale 的回信上都使用此签名
- 让"爱尔兰裔"身份从标签变成实体（名字本身就在说话）

**4.3 爱尔兰习语的第二次出现（可选）**
- Jimmy 给 Whale 的回信结尾可加一句：**"—— Slán, J."**（Slán = 爱尔兰盖尔语"再见"）
- L5 指证 R2（口音维度）时，这句"Slán"可作为"Don't be bothering me"之外的**第二条口音证据**——不强制，但使口音维度更扎实
- 若不使用，保留 Slán 作为纯氛围细节即可

### 为什么改
- 从"爱尔兰裔"这个抽象标签 → 具体的姓名 O'Connor、具体的教材 Berlitz、具体的母语习语 Slán → **身份不再是设定卡上的属性，而是物件上的痕迹**
- L5 R2（口音维度击穿）的说服力更强：不只是一句口误，而是"姓氏 + 签名 + 教材 + 习语 + 母语"五重锁定
- Anna 证言部分也可补一句小细节：她提到 Jimmy 每晚睡前会低声哼爱尔兰童谣"Too-Ra-Loo-Ra-Loo-Ral"给肚子里的孩子听——给这个悲剧角色一点"他到底为谁做这一切"的温度

## 5. 委托协议书 / 威胁纸条的物理介质

### 当前设定
`Unit9_大纲.md`：
- L107 / L122：Webb 委托协议书
- L259 / L276：Rosa 收到的威胁纸条

当前只写"协议书"和"纸条"，没有物理介质细节。

### 建议细化

**5.1 Webb 委托协议书的物理介质**
- 物理形态：**Underwood No. 5 打字机打印 + Webb 本人钢笔签名 + 蜡封**
- 历史支撑：Underwood No. 5（1900 发售，1920s 是美国律师/商人最标配的打字机之一）—— 1920s 正规法律文件的标配组合
- 叙事用途：
  - 协议书上有清晰的打字机钢印痕迹（每台打字机都有独特磨损，时代法医技术可鉴定）
  - Webb 的签名用**红色火漆蜡封**压着一枚私章 —— 显示 Webb 对这份协议的郑重程度
  - 蜡封的完好表明：协议书从签订到 Zack 手中全程未被拆封——这是一份"真的"协议，不是栽赃伪造

**5.2 Rosa 的威胁纸条的物理介质**
- 物理形态：**一张 Blue Moon 酒吧的点餐单背面，用廉价铅笔匆匆写就，铅痕深浅不均，边缘有油渍**
- 叙事用途：
  - 点餐单背面 → 写字的人拿得到酒吧的点餐单 → **酒吧内部人员**（排除外人栽赃）
  - 廉价铅笔 + 深浅不均 → 匆忙、紧张、情绪激动下的书写
  - 边缘油渍 → **厨房环境**（给"Jimmy 在厨房抽空写"的场景一个物理证据）
  - 与 Jimmy 的 Berlitz 扉页签名"James"（钢笔书写，字迹工整）对照：**同一个人在不同书写环境下的笔迹差异**——但倒写 J 的习惯不变。这一细节让"笔迹比对"的推理更扎实

### 为什么改
- 1920s 没有 DNA、没有计算机数据库，**文书鉴定（打字机、笔迹、纸张、墨水）是那个年代最重要的法证手段之一** —— 补充物理介质 = 承认时代的技术局限，同时利用它
- Webb 协议书的"蜡封未拆"给 Zack 拿出它的那一刻多了一层仪式感（"Webb 在死前就已经完成了对 Vivian 的守护"）
- Rosa 纸条的"点餐单 + 厨房油渍"是 L3 指证后不动声色的环境暗示 —— 观察力敏锐的玩家在 L3 看到油渍时可能就开始怀疑厨房了，L5 真相揭晓时会有"啊哈，早就留了伏笔"的回响感
