## 0. 资产命名规范

延续 EPI01 既有命名风格（`SC{ID}_bg_{EnglishName}.png` / `{slug}_{role}.png` / `loop{N}_{slug}.png`），全小写 snake_case。

| 资产类型 | 命名格式 | 示例 |
|---|---|---|
| 场景背景 | `SC{场景ID}_bg_{EnglishName}.png` | `SC9004_bg_MainHall.png` |
| 同场景多状态 | `SC{场景ID}_bg_{EnglishName}_{state}.png` | `SC9024_bg_MainHall_l4.png` |
| 一次性 cutscene 背景 | `CutAway/l{N}_{slug}.png` | `CutAway/l6_maple_fire.png` |
| 动态漫画分格 | `l{N}_{event_slug}_{NN}.png` | `l1_barkeeper_passcode_01.png` |
| Talk 证物出示分格 | `l{N}_t{N}_{item_slug}_{NN}.png` | `l1_t1_webb_contract_01.png` |
| 立绘 | **不在本文档命名**——已在 HTML 配置中维护 | — |

**分格序号说明**：`_NN` 两位数字，从 `_01` 起，**格数 TBD 时按需追加（最多 `_99`），不影响前序 ID**。

存放路径建议：
```
Resources/Art/Scene/Backgrounds/EPI09/        # 场景背景
Resources/Art/Scene/Backgrounds/EPI09/CutAway/ # 一次性 cutscene 背景
Resources/Art/MovieClip/epi09/AVG/            # AVG 分格 / 动态漫画
```

⚠️ **SceneConfig 同步**：本文档场景命名定稿后，`preview_new2/data/table/SceneConfig.json` 的 `backgroundImage` 字段需相应同步——当前多数 sceneId 仍指向 EPI01 复用路径。
