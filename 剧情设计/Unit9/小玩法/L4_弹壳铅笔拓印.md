# 玩法设计文档：弹壳铅笔拓印 (Pencil Rubbing)

**文档版本**: v1.0
**隶属章节**: Unit 9 - 循环4
**玩法类型**: pencil_rubbing（铅笔拓印）
**设计目标**: 玩家用铅笔在蜡纸上来回擦过倒置弹壳，让弹底印记从模糊到清晰渐渐浮现——产出派生证据"9702 .45 警用弹壳（弹底 MILLER ARMS 印记）"，为 L4 R1 击穿 Morrison "Vivian 的手枪就是凶器"提供口径维度支撑，并在 L6 R1 与 9601 米勒集团手枪铭文交叉比对。

---

## 0. 前因后果

Rosa 在 L3 指证结束后将一枚弹壳（9401）悄悄塞给 Zack，此时玩家只知"一枚从会客室地板上捡来的弹壳，外观普通"。

L4 玩家对 9401 进行拓印分析，弹底印记分两层显现——外圈"WCC - .45 ACP"确认口径，内圈"MILLER ARMS"在 L4 暂时仅作记录，含义留待 L6 揭示。

产出证据 9702 在 L4 R1 用于击穿 Morrison "Vivian 的手枪就是凶器"（.45 vs .25 口径不符）；L6 R1 复用，与 9601 米勒集团特制手枪枪身铭文"MILLER ARMS · CHI"精准呼应，两件物证交叉封死 Morrison 的第一道防线。

---

## 1. 玩法概述

### 触发条件

玩家持有 9401（.45 警用弹壳，L3 post_expose Rosa 给出）且循环进度为 L4，在证据详情中选择"拓印分析"后触发。

### 核心体验

弹壳倒置固定在蜡纸上 → 玩家拖动铅笔在纸面来回擦几下 → 弹底印记渐渐浮现（两层：口径外圈先现，品牌内圈后现）→ 系统弹出弹药识别条 → Zack 默记内圈小字

整个过程像小时候用铅笔在硬币上擦花纹——手感直觉，不需要教程。

### 预期时长

约 15 秒

### 流程图

```
证据详情：9401 .45 警用弹壳
  玩家选择"拓印分析"
        │
        ▼
  进入拓印界面
  俯视桌面：一张蜡纸，弹壳已压在中央，铅笔在手边
  UI 淡出小字提示："侧锋推铅笔，让印记浮出来。"
        │
        ▼
  玩家横向拖动铅笔（或轻触屏幕来回擦）
        │
        ▼
  第一层浮现：外圈"WCC - .45 ACP"
  系统弹药识别条淡入：".45 ACP · 警用标准口径 · 不符 Vivian 手枪的 .25 口径"
        │
        ▼
  继续擦几下（或稍作停顿后自动推进）
        │
        ▼
  第二层浮现：内圈"MILLER ARMS"
  Zack（内心独白）："底部还有第二圈小字……MILLER ARMS？没听过这个名字。先记下。"
        │
        ▼
  派生证据生成
  获得 9702「.45 警用弹壳（弹底 MILLER ARMS 印记）」
  9401 保留（不消耗）
```

## 2. 1920s 考据

铅笔拓印（pencil rubbing）在 1920 年代是法医学界通用的硬币、印章、金属标记记录手段，无需任何专业设备，只需一张薄纸和一支软铅笔。警用弹药在这一时期已普遍推行工厂批号制度，弹底印记包含口径标注和厂商批号，是识别弹药来源的标准凭据。.45 ACP（Automatic Colt Pistol）在 1920s 美国警务系统中已是标准配备口径，与民用轻型手枪的 .25 ACP 在口径上差异明显，任何接触过警用弹药的人都能一眼分辨。文中提到的"底径 .452 英寸"是 1920s 真实行业用语，不是事后造词。

## 3. 交互设计

### 界面布局

俯视视角，桌面满屏。蜡纸铺开，弹壳底朝下压在中央（弹底面对纸面）。铅笔已备在旁边，画面边缘有轻微的暖黄聚光（纯视觉氛围，非场景设定）。

```
┌──────────────────────────────────────────────┐
│              分析界面 · 拓印                  │
│                                              │
│         ┌──────────────────────┐             │
│         │   [ 蜡 纸 ]          │             │
│         │                      │             │
│         │      ○               │             │
│         │    （弹壳底朝下）      │             │
│         │                      │             │
│         └──────────────────────┘             │
│                                              │
│    ← 铅笔  拖动方向：横向左右 →               │
│                                              │
└──────────────────────────────────────────────┘
```

### 操作

玩家按住铅笔图标，在弹壳投影区域横向来回拖动（PC 端：按住左键拖动；触屏：按住滑动）。不需要精确轨迹，只要在区域内来回擦过即可推进显影。

来回 2-3 次后第一层浮现，再擦 2-3 次后第二层浮现。节奏自然，不设强制计数。

### 印记显现逻辑

| 阶段 | 内容 | 视觉效果 |
|------|------|---------|
| 初始 | 空白蜡纸，弹壳轮廓隐约可见 | 灰白底，弹壳圆形轮廓 |
| 第一层浮现 | 外圈"WCC - .45 ACP" | 铅笔灰色笔触逐渐加深，字母从模糊到清晰 |
| 第二层浮现 | 内圈小字"MILLER ARMS" | 内圈字迹比外圈更细，渐显时有轻微颗粒质感 |
| 完成 | 完整弹底拓印图案 | 轻微发光提示，确认按钮出现 |

## 7. 指证链整合

### 跨 Loop 使用路径

```
L3 post_expose
  Rosa 交出 9401（外观普通弹壳，口径未知）
        │
        ▼
L4 证据详情 → 拓印分析
  ★ 铅笔拓印小玩法（本文档）
  输入：9401
  输出：9702（弹底拓印：.45 ACP + MILLER ARMS 内圈）
  ——玩家此时知道口径，不知道 MILLER ARMS 的含义——
        │
        ▼
L4 Expose R1（指证 Vivian）
  谎言："我真的杀了 Webb！"
  出示：9401（物理弹壳）+ 9101（Vivian 的 .25 ACP 手枪 / 派生 9711 无硝烟分析）
  击穿逻辑：弹壳口径 .45 ACP ≠ Vivian 手枪口径 .25 ACP → 凶器不是 Vivian 的枪
  9702 拓印图样辅助呈现口径字符，强化可读性
        │
        ▼ （L4-L5 中间——9702 静置于背包）
        │
        ▼
L6 Expose R1（指证 Morrison）
  谎言："Vivian 的手枪就是凶器！她就是凶手！"
  出示：9401（.45 ACP 弹壳，弹底"MILLER ARMS"）+ 9601（米勒集团特制手枪，铭文"MILLER ARMS · CHI"）
  击穿逻辑：
    ① 9401 弹底"MILLER ARMS" ← 铅笔拓印使其可读，L4 已完成
    ② 9601 枪身铭文"MILLER ARMS · CHI" ← L5 post_expose 获得
    ③ 两件物证铭文互相呼应 → 凶器是米勒集团特制手枪，不是 Vivian 的 .25 ACP
  Morrison 的第一道防线被封死
```

### 跨 Loop 复用的推理价值

9702 是这条推理链的**可读性锚点**：9401 弹底印记在物理上肉眼难以辨认，铅笔拓印让"MILLER ARMS"从隐藏信息变为可陈列的证据图样。L4 时玩家记录下这个名字却不明其意，L6 看到 9601 枪身铭文的瞬间形成"回头看才恍然大悟"的体验——这正是 L4 玩家不懂 L6 才揭示的设计意图。

### 陷阱说明（L6 R1）

仅出示 9601 米勒手枪而不出示 9401（含 9702）：Morrison 可辩"还有其他 .45 手枪，不能证明这就是现场那把"。9401 弹壳是现场实物，与 9601 铭文组合才能封死"这把枪就是从这枚弹壳里射出的"的逻辑链。两件必须同时出示。
