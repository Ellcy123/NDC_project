**玩法类型**: oblique_light_impression（侧光压痕揭示）
**核心动词**: 照
**预期时长**: ~15 秒
**输入**: 9302 威胁纸条（L3 Rosa Expose R2 post_expose 获得）
**派生产出**: 9707 带倒写 J 压痕的威胁纸条

## 0. 前因后果

**从哪来**：Rosa 在 L3 R2 被击穿后崩溃，交出威胁纸条（9302）。纸条正面是胁迫内容，Zack 和 Emma 读完后注意到字迹里有一个 J 是倒着写的——但仅凭正面墨迹，倒写 J 还只是孤证。Emma 或 Foster 提示：把这张纸拿到台灯下照照，起草时的压痕可能还在背面。

**用在哪**：

- **L3 R2 强化**：9302 已被用于击穿 Rosa 的谎言；9707 作为派生证据，在 Expose 后续对白中可进一步强调"胁迫者在草稿里也习惯性倒写 J"，加固倒写 J 这一书写特征的证据厚度。
- **L5 R3 字迹一致性链**：9707 与 9402 被撕毁合同、9503 Jimmy 回信、9504 英语练习书共同构成字迹三物闭环，是 L5 post_expose_dialogue 中 Zack 点名"威胁 Rosa 的也是你"的叙事依据之一（9707 纸条压痕中的倒写 J 与 9402/9503 字迹三物互相印证，指向 Jimmy）。

**跨 Loop 复用示意**：

```
L3 Rosa Expose R2 → 获得 9302（威胁纸条）
         │
         ↓
  ★ 小玩法：侧光压痕揭示
         │
         ↓
  派生 9707（带倒写 J 压痕的威胁纸条）
         │
         ├── L3 R2 后续对白：强化倒写 J 书写特征
         │
         └── L5 R3 字迹闭环：9707 + 9402 + 9503 + 9504 → 指向 Jimmy
```

## 1. 玩法概述

玩家将威胁纸条（9302）拖到场景中的台灯旁，调整角度让光线以低角度掠射纸面。胁迫者写纸条之前曾在同一张纸或叠在其上的纸上起草练习，铅笔压力在纸背留下了压痕。随着角度接近最佳位置，压痕从纸面浮现：几个不成句的草稿词，其中有一个 J 字母——钩朝反方向，倒着写。

整个体验是一次"调灯→压痕浮现→定格倒写 J"的连贯瞬间，设计为快节奏收口：玩家完成 L3 最繁重的指证后，这 15 秒给一个小小的"啊哈"奖励，同时为 L5 铺下伏笔。

**触发时机**：Rosa post_expose 获得 9302 后，Emma 自动开口提示（见第 3 节）。

---

## 2. 1920s 考据

侧光压痕揭示（Oblique / Raking Light）在 1920 年代已是文物修复与笔迹鉴定的常规手法——用一盏可调角度的台灯，让光线以极低角度掠射纸面，纸上的凹凸起伏便会投射出肉眼可见的阴影。对于铅笔起草留下的压痕，1920 年代的侦探和文书鉴定员完全熟悉这种技巧，不需要任何超时代的设备。Foster 或 Emma 提示"拿到台灯下照"，完全符合 1925 年芝加哥的技术背景与职业习惯。

## 3. 交互设计

### 触发对白

Rosa post_expose 结束、9302 进入背包后，自动触发短对白：

```
Emma："这张纸……你注意到那个 J 了吗？写法很奇怪。"
Zack："倒着写的。"
Emma："把它拿到台灯下照照。写字之前打草稿的人，
       压痕很少会消失干净。"
```

UI 提示：9302 图标右上角出现"可分析"闪烁标记。

### 操作流程

**① 拖拽纸条**
玩家把 9302 从背包拖到场景台灯的光圈范围内，纸条平铺展示在桌面上。

**② 角度调整**
屏幕一侧出现一个弧形调节滑块（模拟台灯俯仰角度）。
- 0°–20°：纸面均匀照亮，什么都看不到
- 20°–35°：纸背隐约有起伏，轮廓模糊
- ★35°–50°：压痕清晰浮现，字迹可辨，系统锁定

**③ 压痕显现**
达到最佳角度后，纸面渐变动画播放约 1.5 秒：草稿词语从纸背的阴影里缓缓浮出——先是几个扭曲的字母，最后定格在一个高亮的 J，钩朝左、倒着写。

**④ 高亮倒写 J**
系统在倒写 J 上画出金色描边，停顿 0.8 秒。

**⑤ Zack 收口台词**（不可跳过）
```
Zack："这个倒写的 J——胁迫者和 Rosa 的关系，或许比我们想的更近。"
```

**⑥ 派生产出**
9302 替换为 9707「带倒写 J 压痕的威胁纸条」，进入背包。

### 容错

若玩家 20 秒未找到最佳角度，Foster（若在场）或 Emma 补一句：
```
"再低一点——台灯角度越低，压痕的阴影越长。"
```
若 40 秒未操作，系统自动将滑块缓慢移至最佳角度并锁定。

## 5. 美术与音效需求

**美术**：

- 9302 纸条正面：已有现成描述（餐巾纸 + 铅笔潦草文字），沿用即可
- 压痕层（纸背）：独立 Alpha 图层，内容为 3–4 个散乱的字母练习，其中一个倒写 J；字体手绘风、歪斜；倒写 J 高亮帧单独切图
- 台灯光圈：小范围暖黄色渐变光晕，半径约纸条宽度的 1.5 倍
- 滑块 UI：弧形小组件，沿台灯灯臂方向放置，移动端可设计为单指上滑

**音效**：

| 时机 | 音效 |
|------|------|
| 压痕开始显现 | 低沉弦乐单音缓入（约 1 秒渐强） |
| 压痕锁定 | 柔和的相机快门声（模拟 Foster 拍照记录） |
| 倒写 J 高亮 | 单一清亮"叮"音，短促 |
| Zack 台词后收尾 | 弦乐以不解决和弦收口，暗示悬念留存 |

## 7. 指证链整合

### L3：倒写 J 初次确立

```
Rosa Expose R2
  → 出示 9302（威胁纸条）+ 9303（女儿医疗单）
  → Rosa 崩溃，交出 9302
         │
         ↓
  ★ 小玩法：侧光压痕揭示
         │
    倒写 J 高亮定格
    Zack："这个倒写的 J——胁迫者和 Rosa 的关系，或许比我们想的更近。"
         │
         ↓
  获得 9707（带压痕的纸条）
  → 疑点 9302「Rosa 是被胁迫作伪证的？」加固
  → 胁迫者身份线索（书写习惯）植入玩家记忆
```

**此刻不告诉玩家这是 Jimmy**——倒写 J 只是一个书写特征线索，留给 L5 完成身份追踪。

### L5：字迹三物闭环，指向 Jimmy

```
L5 自由探索
  ├── 9402 被撕毁的合同（L4 取得）——字迹有倒写 J，与 9302 高度一致
  ├── 9503 Jimmy 给 Whale 的回信（L5 Jimmy 家获取）——Jimmy 签名中 J 倒写
  └── 9504 Berlitz 英语练习书（L5 Jimmy 家 envir，不拾取）——倒写 J 习惯，爱尔兰裔书写偏差

L5 Expose R3 + post_expose_dialogue
  Zack："威胁 Rosa 的也是你——那个纸条和合同上面的 J 都是倒着写的。"
  引用：9707（L3 压痕揭示）/ 9402（L4 合同）/ 9504（L5 练习书观察）
  → 三个 Loop 的证据在此汇聚，玩家回头看 L3 小玩法："原来那个倒写的 J 早就在说 Jimmy"
```

**推理设计说明**：9707 在 L3 是伏笔，在 L5 是佐证。L3 玩家只知道"胁迫者有倒写 J 的习惯"；L5 玩家把这个习惯与 Jimmy 家的练习书对上，才完成身份追踪链的闭合。中间两个循环的信息积累（9402 合同字迹 → 9503 回信签名 → 9504 练习书）是正式指向 Jimmy 所必需的，9707 独立一件不足以定身份，符合逻辑闭环原则。
