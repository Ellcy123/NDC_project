# 玩法设计文档：女儿生日密码锁 (Daughter's Birthday Padlock)

**文档版本**: v1.0
**隶属章节**: Unit 9 - 循环 3
**设计目标**: 让玩家亲手用一个清洁工母亲最温柔的数字，打开她最残忍的秘密——把"获取证据"变成"理解一个人"。

---

## 0. 前因后果

**前因——证据从哪来**

L3 Rosa 储藏室（场景 9001）搜证。Rosa 本人被 Morrison 叫到大堂候询不在场，但她的私人物品仍留在此处。墙上挂着一张黑白女儿照片，下方手写"**我的天使 · 0915**"；角落木架上摆着一个上锁的小铁盒。

**后果——证据用在哪**

玩法解锁后，盒内同时取出：
- **9302**「威胁纸条（倒写 J）」—— L3 Expose R2 核心击穿件 + L5 字迹闭环伏笔
- **9303**「女儿医疗单」—— L3 Expose R2 辅助击穿件，证明胁迫者掌握 Rosa 女儿的具体病情

---

## 1. 玩法概述

- **玩法类型**: combination_lock（密码挂锁破解）
- **核心体验**: 观察墙上照片的日期 → 铁盒四位拨轮转出 0-9-1-5 → 开锁同时取出两件证据
- **预期时长**: ~25 秒
- **触发条件**: 玩家在 9001 储藏室搜证时点击铁盒，自动进入玩法
- **特别说明**: 这不是一个物证分析玩法，是**情感玩法**。推理价值不变（获取 9302 + 9303），但获取方式成了一次对 Rosa 人物的理解——密码是女儿生日，盒里是威胁杀女儿的纸条。玩家击穿 Rosa 之前，先懂她。

---

## 2. 1920s 考据

1920s 的密码挂锁（combination padlock）是家庭常见物品。Master Lock 公司 1921 年创立，推广三/四位拨轮式密码锁进入普通家庭；小型铁匣配密码锁广泛用于存放私人信件、现金、证件。清洁工阶层的 Rosa 拥有这样一个小铁盒完全符合时代背景，用生日作密码也是普遍习惯——简单、难忘、只有自己知道。

---

## 3. 交互设计

```
【场景入口 · 储藏室角落】
  昏暗的地下储藏室，低瓦数吊灯摇晃
  视线内有两个交互点：
    左侧墙上 · 一张黑白女儿照片（约 10 岁女孩）
              照片下方 Rosa 手写标签："我的天使 · 0915"
    右下角 · 一个小铁盒（盒面四位数字拨轮，均为 0）

【步骤 1 · 观察照片】
  玩家点击墙上照片 → 放大特写
  看清手写字样："我的天使 · 0915"
  Zack（内心独白，轻声）："0915……她的生日？"

【步骤 2 · 转动拨轮】
  玩家点击铁盒 → 切换到密码锁特写
  四个拨轮默认 0000
  玩家从左至右依次拖动转出 0-9-1-5
  每对齐一位拨轮有轻微"咔"反馈
  四位全对 → 咔嚓一声 → 盒盖自动弹开

【步骤 3 · 取出证据】
  盒内：
    · 一张折叠的餐巾纸（9302 威胁纸条）
    · 一张医院单据（9303 女儿医疗单）
  玩家点击即可同时拾取

  Zack（沉默片刻）：
  "她用女儿的生日锁着这个盒子。锁着威胁杀女儿的纸条。"

  Zack（更轻，像是自语）：
  "……这是一个母亲能找到的，最安全的地方了。"

【容错】
  玩家转错不触发失败——拨轮可随意调整
  15 秒未对齐：UI 下方淡出提示"转头看看周围——Rosa 把重要的东西放在看得见的地方。"
  30 秒未对齐：UI 下方淡出提示"照片下方的标签：'我的天使 · 0915'。"
  UI 下方始终有"MM-DD"格式小提示
```

## 4. 结果与产出

**直接获得**（不产出派生 ID——铁盒仅是容器，原件入包）：
- **9302 威胁纸条（倒写 J）**
- **9303 女儿医疗单**

**指证用途**（沿用原 state 设计）：
- **L3 Expose R2**：9302 + 9303 组合击穿 Rosa "我亲眼看见 Vivian 有枪" 的退守谎言
- **L5 Expose R3**：9302 的倒写 J 与 9402 / 9503 / 9504 构成字迹闭环伏笔

**情感价值**（玩法新增的不可替代性）：
- 玩家亲手转出 0-9-1-5 的瞬间，**在击穿 Rosa 之前先理解了她**
- L3 R2 Rosa 崩溃哭泣时，共情不再来自"我战胜了撒谎者"，而是"我看见了一个被掐住软肋的母亲"
- 让 L3 整个叙事从"破案"升华为"看见人"
