## 二、核心诡计结构

```
真实杀人 (23:00-23:05)          假枪声 (23:30)
     │                              │
     ▼                              ▼
Jimmy 在会客室枪杀 Webb      Jimmy 从二楼空射一枪
(使用 .45 ACP)
     │                              │
     ▼                              ▼
音乐高潮掩盖真实枪声         制造虚假的"案发时间"
(P1: 消音器 or 隔音待确认)
     │                              │
     ▼                              ▼
子弹穿窗→碎片落地            所有人以为 23:30 才是案发
     │                              │
     ▼                              ▼
23:05 模仿 Webb 爱尔兰口音   Rosa 按纸条指令指认 Vivian
骗走门外的 Vivian             "目击 Vivian 在 23:30 射杀 Webb"
```

**玩家需要还原的推理链：**
1. 垃圾袋下有碎片 → 碎片产生时间早于 23:20（Rosa 倒垃圾时间）
2. 碎片来自子弹穿窗（硝烟分析）→ 真实枪击发生在 23:20 之前
3. 因此 Rosa 声称的"23:30 目击"不可能是真实案发时间
4. 23:00 音乐高潮 → 枪声被掩盖 → 真实案发时间锁定
