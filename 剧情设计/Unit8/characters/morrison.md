叙事作用：

- loop4 的转折推手——看到 .45 弹壳脸色骤变，拿走 1103，夺走 1308，强押 Vivian，把 Part1 的胜利一举推翻

- 他不是幕后黑手，而是被 Whale 经济捆绑的棋子；他的腐败不来自主动作恶，而来自赌债危机与别无选择
- 代表1920年代芝加哥系统性腐败的缩影——警察不是正义的守护者，而是可以被买通的工具

核心主题：一个曾经优秀的警探如何在绝望中一步步堕落

参考原型：
- 黑色电影中的腐败警探：有能力、有经验，但被金钱和恐惧腐蚀
- 禁酒令时代芝加哥真实的警察腐败：从上到下的系统性贿赂
- "被迫选择恶"的灰色反派：不是天生邪恶，而是被推入深渊

核心困境：赌债、Whale 的钱，和一个无法追查的案子

Morrison 曾经是一个好警察。

二十年的服役，功勋奖章，处理过无数案件。他的妻子以他为荣。

但赌博毁了一切。

Morrison 的堕落轨迹：
1. 赌博成瘾：开始时只是小赌，逐渐失控
2. 欠下巨额赌债：欠疤面煞星地下赌场 $5000，被人威胁若不还钱就让他"从警察局消失"（证据：loop6 搜出赌债欠条 + 威胁信，即 1602）
3. 催债威胁升级：走投无路，警察的薪水永远还不清
4. Whale 出手：Whale 以"W 先生"名义借给 Morrison $6000（时间：Webb 案发生前约三周）——Morrison 用这笔钱还清了疤面煞星的债（证据：loop6 搜出借据，即 1603）

然后 Webb 案来了。

Whale 的无声要求：
- Morrison 欠了 Whale 的钱，案子来了，结案就是偿债
- Webb 被 Whale 雇佣的 Jimmy 杀死，Morrison 只需要快速结案、压下证据
- Whale 没有发出明确指令——Morrison 懂得这层意思
- Morrison 认出了 .45 弹壳的批次属于米勒集团：追查下去意味着触碰 Whale，他知道后果

Morrison 的选择：
- 追查 → 对抗 Whale，与自己的债务人为敌，后果难以想象
- 压案 → Vivian 入狱，Zack 的调查中止，Whale 满意，债务抹平
- 他选择了压案，用二十年的专业技能为 Whale 服务了一次

但这个选择的代价：
- 他把一个无罪的人关进了监狱
- 他夺走了关键物证（1308 弹壳），藏在自己家里
- "他不是笨蛋，他知道销毁物证是什么罪——所以他只是藏着"（Emma 的判断，loop6 opening）

性格特征

核心性格

1. 暴躁而有压迫感：
  - Morrison 的第一反应永远是威胁和权威
  - loop1 接案时咆哮、推人、用职权压制
  - 用恐惧来控制局面，因为他自己就被恐惧控制着

2. 老练的掩饰能力（散漫背后是计算）：
  - 二十年的办案经验让他擅长掌控场面
  - loop1 的"醉醺醺"是状态，不妨碍他评估现场
  - loop4 阶段 A 在大厅"随口闲聊"，实际是在评估 Zack 调查到了哪一步（参见 morrison_s405 ）
  - 他知道销毁物证风险更大，所以只是"藏起来"——这是一种老警察的自我保护本能

3. 内心有良知，但来得太晚：
  - Morrison 不是一个彻底的恶人
  - loop6 第二轮退守时语气里有"疲惫，而不是愤怒"——他没有真正想陷害 Vivian，只是想让一切安静下去
  - 被逼到极限时，他握着笔停顿了很久，没有争辩，把名字签了上去
  - "他是一个典型的'被环境腐蚀的普通人'，而不是主动作恶者"（loop6 morrison_expose ）

4. 被绝望吞噬的挣扎者：
  - Morrison 不是因为贪婪而腐败，是因为走投无路
  - 他欠的不只是钱，还有对家人安全的责任
  - 在他的世界观里，这座城市不给好人出路

语言风格

说话特点

1. 威胁性的命令口吻：
  - 习惯用命令句、反问句
  - "闭嘴！你有权保持沉默！""别逼我再铐你。"
  - 用音量和态势压制对方

2. 散漫掩盖计算：
  - loop4 大厅等待时随口追问"你查到什么了"——语气漫不经心，但眼神里有一丝审视
  - 这是评估，不是闲聊

3. 被揭穿时的失控与重新掌控：
  - loop4 看到 .45 弹壳时脸色骤变——那不是普通警官面对证据的反应，而是认出了不该被发现的东西
  - loop6 第一轮退守时"用手摸了摸桌面的相框，才重新抬起头"——短暂失控后重新压制
  - 暴怒已经用不上了——这次他没有外援，也没有围观者

4. 最终认罪时的疲惫：
  - 第三轮谎言崩塌后，Morrison 低下头，不再开口辩解
  - 握着笔停顿很久，没有任何争辩，把名字签了上去
  - 这是一个被压垮的人——不是被正义压垮，而是被自己的选择压垮

关键关系网

与 Whale 的关系：经济捆绑的棋子

0417 版 Morrison 和 Whale 的关系，不是旧版那种"接到纸条被直接收买"的主动合谋，而是一种更隐秘的经济捆绑：
- Whale 在 Webb 案发前三周，以"W 先生"名义借给 Morrison $6000（1603）
- Morrison 用这笔钱还清了疤面煞星的赌债，救了家人的安全
- 案子来了，Morrison 认出了 .45 弹壳的米勒集团批次——他知道追查下去意味着什么
- 他选择了快速结案：夺走 1308 弹壳，拿走 1103 协议书，强押 Vivian
- Whale 没有发出命令，Morrison 只是履行了一个无声的债务

Morrison 是 Whale 的棋子，但他从未与 Whale 正面接触——loop6 是 Whale 第一次以声音直接出现，而那次是打给 Zack 的，不是打给 Morrison 的。

与 Webb 的关系：辖区的灰色交易

- Morrison 是蓝月亮所在街区的治安负责人，平时常来蓝月亮喝酒，和 Webb 关系还行
- loop1 开场他是醉醺醺赶到现场的——他今晚在吧台喝酒，听到枪响才过来查
- 这个"醉酒但秉公"的初始状态是重要的性格基线：loop1 的 Morrison 还没有完全变成 Whale 的工具，他只是被动偷懒，倾向于最简单的解释快速了事
- loop5-6 他选择压案、强押 Vivian，与 loop1 那个"嫌麻烦但还是姑且秉公"的 Morrison 形成鲜明对比——Whale 介入（通过借款）改变了他的选择（loop1 morrison_001 ）

与 Rosa 的关系：（0417 版调整）

0417 版中 Morrison 与 Rosa 之间没有直接的威胁关系。Rosa 是被 Jimmy 的威胁纸条操控的，Morrison 并不参与那个层面的安排。Morrison 的"腐败"体现在对证据的压制（夺走弹壳）和对案件的快速结案，而不是直接威胁证人。

与 Mrs. Morrison 的关系：被蒙在鼓里的妻子

Mrs. Morrison 不知道真相：
- 她以为丈夫是一个尽职的好警察
- 她自然地配合了 Zack 和 Emma 的访谈，无意中可能暴露了 Morrison 的行动时间线
- Mrs. Morrison 是 Morrison 内心最深处的软肋：赌债危机最初压垮他，也是因为疤面煞星威胁了他家人的安全
- 真相暴露后，她的信任会崩溃——Morrison 每次的选择，都有一部分在为她，又有一部分在背叛她

与 Zack 的关系：对手，最终是证据的受审者

两人关系的演变：
- loop1：Morrison 是有权力的接案警官，Zack 是刚进现场的调查委托人——Morrison 勉强给了 72 小时，但始终有压迫感
- loop3：Morrison 汇报勘验结果，言简意赅地说"子弹缺失"，刻意不深入——Zack 注意到他说话方式异常简短
- loop4 阶段 A：Morrison 在大厅随口问"查到什么了"，实际是在评估 Zack 调查到了哪一步
- loop4 转折：Morrison 看到 .45 弹壳后当机立断，拿走证据，强押 Vivian——权力关系彻底逆转为 Morrison 的压制
- loop6：Zack 推开 Morrison 办公室的门，没有外援，没有围观，只有三个人和一张桌子——Morrison 第一次无法用权威表演

- 这句话的表层伏笔：Rosa 后来交出的弹壳（1308）正是那颗"缺失"的子弹
- 深层伏笔：Morrison 自己也藏了一颗——他说"子弹缺失"时，他自己就是藏弹壳的人之一

场景3：loop4 大厅等待与转折（s405，morrison_s405 + morrison_003_turn）

转折的核心：
- Morrison 看到 .45 弹壳时，那不是"惊讶"，而是"认出了某种他不希望被发现的东西"
- 他的反应不是普通警官，而是认出了米勒集团批次的人——这暗示他知道 Whale 的运作层级
- 他当机立断拿走所有关键证据，这是一种老警察的危机处理本能：压制，而不是面对

情感重点：
- 这是 Part1 胜利被逆转的时刻——玩家刚刚救了 Vivian，转眼一切被推翻
- Morrison 的变脸建立了 loop5-6 的核心动机：他和 Whale 到底是什么关系？

场景4：loop6 Morrison 家搜查（s601）

Morrison 不在场，但他的家说出了他不肯说的话：
- 1601（藏在书桌抽屉深处的弹壳）：Emma 的推断成立——他没有销毁，只是藏起来
- 1602（皱巴巴的欠条和威胁信）：疤面煞星的债是起点，他不是天生的棋子，他是被债务推进去的
- 1603（"W 先生"的借据）：Whale 早在三周前就已经把 Morrison 买下了，用的是一笔钱，不是一张纸条

电话突发事件——Whale 打来电话（此时 Morrison 不在场，是 Zack 接的）：
- Whale 知道 Zack 在查案，知道 Jimmy 已经死了
- 他不是来威胁，而是说："证明给我看——这个城市里，是否真正还存在正义。"
- 这句话与他雇佣 Jimmy 杀人的行为之间，有一道很深的裂缝（s601 phone_call_event ）

情感重点：
- Morrison 的家是他内心的剖面：廉价的家具，皱巴巴的欠条，藏在抽屉深处的弹壳
- 他不是一个住在豪宅里的腐败警官，他是一个债台高筑、被逼到做出糟糕选择的普通人

场景5：loop6 警局单独对质（s602 Expose）

对质的设计核心：没有外援，没有围观，只有三个人和一张桌子。

Morrison 第一次失去了"表演对象"：
- 平时他可以用权威、用咆哮、用警服上的徽章来控制场面
- 这次，他只能独自面对摆在桌上的证据
- 第一轮退守时用手摸了摸桌面的相框——那是他妻子的方向，也是他最后的支撑
- 第三轮崩塌时低头，不再开口——这不是投降，是一种比投降更深的疲惫

认罪签字时，Morrison 握着笔停顿了很久，没有任何争辩。这是整个 Unit8 对这个角色最重要的一笔：他不是为了被感化而认罪，他是因为没有退路才认罪——但那个疲惫里，有一点点解脱的味道。

创作要点

对话创作

1. 威胁时的语言特征：
  - 命令句为主，短促有力
  - 大量使用职务权威，而不是粗口（和旧版有微妙差异——0417 版 Morrison 更多是用"职权"来压人）
  - "别逼我再铐你。""你查到什么了？"
  - 用音量压制对方，不给对方说话的空间

2. 散漫但计算的闲聊：
  - loop4 阶段 A 的"随口"是表演
  - 问话方式：简短、似乎不在乎——但他记住每一个答案
  - 不说多余的话，用沉默制造对方的不安

3. 崩溃时的特征：
  - 不再暴怒（暴怒是 loop4 之前有外援时的武器，单独对质时没有用武之地）
  - 反而是沉默、摸桌面的相框、握住笔的手
  - 每一次退守都比上一次更疲惫，而不是更愤怒

4. 认罪时的真情：
  - 没有台词
  - 握笔的停顿是这个角色最后的内心戏
  - 不要让他说"我错了"——他签字，那已经够了
