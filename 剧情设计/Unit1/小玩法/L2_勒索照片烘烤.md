# 玩法设计文档：勒索照片烘烤 (Invisible Ink Reveal)

**文档版本**: v1.0
**隶属章节**: Unit 9 - 循环 2
**设计目标**: 让玩家亲手将一张普通的暧昧合影变成勒索账本，用"烤"这个动作把 Webb 的精明和残忍一并点燃。

---

## 0. 前因后果

**前因——证据从哪来**

L2 小包厢（场景 9006）搜证，拾取 9201「勒索用的照片」。
正面：Vivian 与一名中产男士的暧昧合影，看起来不过是花边新闻。
背面：几处淡黄褐色污渍，肉眼看不出意义（此为显影前状态，需 state 改造，见第 6 节）。

**后果——证据用在哪**

完成显影，产出 9706「带金额显影的勒索照片」，用于 L2 Expose R1：
击穿 Tommy 的谎言证词 9052001——"酒吧里都是正当生意，除了私酒之外没有别的交易"。
照片背面的金额字迹证明这张合影不是隐私，是一张定价明确的勒索账单。

---

## 1. 玩法概述

- **玩法类型**: invisible_ink_reveal（隐形墨水显影）
- **核心体验**: 拖照片靠近煤油灯 → 背面字迹从污渍里浮现
- **预期时长**: ~20 秒
- **触发条件**: 持有 9201，查看证据详情时发现背面污渍，点击"仔细查看背面"进入玩法

---

## 2. 1920s 考据

柠檬汁在 1920s 已是民间熟知的隐形墨水——挤出液体书写，干后无色透明，
加热后有机酸氧化变褐，字迹重新浮现。不需要任何设备，勒索者随手可得。
Webb 用它把真实金额藏在照片背面，表面上这就是一张跑码头的艺人和恩客的合影；
只有他和被勒索者才知道背面写了什么。这是精明，也是习惯性的冷酷。

---

## 3. 交互设计

玩法只有一个动作。

```
【界面布局】
  左侧：9201 勒索照片（正面朝下，背面对着玩家）
  右侧：桌面角落的煤油灯，火焰微微晃动

【操作】
  玩家拖动照片 → 移向煤油灯火焰旁
  （照片与火焰的距离决定加热速度，不需要精确操作，
   拖到"热区"内即可，无失败惩罚）

  【UI 提示 · 玩家首次靠近火焰时】
  画面下方淡出一行小字："正面始终朝下，别烤到相片本身。"

【动画过程——约 6 秒】
  阶段 1：照片背面边缘微微泛黄，隐约有热气线条（正面始终朝下保护）
  阶段 2：背面污渍开始加深，字迹轮廓从模糊到清晰
  阶段 3：两行字完整显现——
           花瓶 — $4,000
           油画 — $7,000
  照片背面边缘出现轻微焦褐色，正面明胶乳剂保持完整，提示"火候刚刚好"

【完成判定】
  字迹显现后，照片自动"落回"桌面
  显影完成，触发 Zack 台词

【过热意外（彩蛋分支）——仅触发一次】
  如果玩家把照片拖进火焰核心（fire_zone 内）并停留 > 2 秒：
    → 照片一角突然冒烟 → 小火苗"腾"地窜起
    → 画面快速晃动（Zack 手抖）
    → Zack（急促）："靠——！"
    → Zack 一把把照片从火上拽开，顺手按灭火苗
    → 0.5 秒后火被拍灭，照片一角留下一块焦黑
       （仅视觉痕迹，不影响信息完整性，字迹照样能显）

  扑灭后系统提示（画面下方淡出小字）：
    "再来一次。这次离远点。"

  照片自动回到热区外起始位置，玩家继续玩法，
  但本次玩法不再二次触发此分支（避免重复利用）
```

Zack（看着照片背面，低声）：
"柠檬汁。他把真实金额藏在照片背面……这不是花边新闻，这是一本账本。"

## 5. 美术 / 音效需求

美术：
- 9201 照片正面：暧昧合影构图，1920s 人像摄影质感，黑白或棕调
- 9201 照片背面（显影前）：泛黄纸面，有几处不规则淡黄褐色水渍，无可读文字
- 照片背面（显影中）：字迹从褐渍里逐字浮现，笔迹潦草偏行书
- 照片背面（显影后）：两行字稳定清晰，纸张边缘轻微焦褐
- 煤油灯：桌面油灯，玻璃罩内橙黄火焰，有轻微摇曳动画
- 热区提示：照片靠近火焰时出现淡橙色晕圈（非 UI，环境光效）
- **过热彩蛋**：照片一角起火的 2D 火苗粒子动画（0.5-1 秒）+ 扑灭后照片一角的焦黑烧痕贴图

音效：
- 照片靠近火焰：细微纸张受热的"咝"声，低频热气环境音
- 字迹浮现：像钢笔墨水渗纸的轻微沙沙声，逐字叠加
- 显影完成：一声低沉的"啪"（照片落回桌面），随即安静
- **过热起火**：一声短促的"腾"火苗窜起声 + Zack 急促的"靠！" + Zack 拍灭火苗的"啪"一声

## 6. 数据配置

```yaml
mini_game:
  id: MG_9201_invisible_ink
  type: invisible_ink_reveal
  input_item: 9201
  output_item: 9706
  heat_zone_radius: 120          # px，煤油灯周围热区半径（正常加热区）
  fire_zone_radius: 40           # px，火焰核心区（触发过热彩蛋）
  fire_dwell_threshold: 2.0      # 秒，在 fire_zone 内停留超过此值触发起火
  max_fire_events: 1             # 每次玩法至多触发 1 次过热分支（避免重复）
  reveal_duration: 6.0           # 字迹从模糊到完整显现的秒数
  trigger_text: "照片背面有几处淡黄褐色污渍——用火烤一烤也许能看出什么。"
  fire_event:
    outcome: "non_destructive"   # 不销毁证据，不打断玩法
    visual_mark: "scorched_corner"  # 照片一角留焦黑痕迹（仅视觉）
    reset_position: "outside_heat_zone"  # 扑灭后照片回到热区外
```
