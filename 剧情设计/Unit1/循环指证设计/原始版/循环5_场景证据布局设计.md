**🎯 指证系统设计**

**指证流程：4轮递进式情感对质**

**设计理念**：每次指证击破Vivian一层情感伪装，她被迫升级谎言但暴露更大破绽

**第一轮指证：爱情假象的破绽**

**Vivian初始表演**：
*\"Zack先生，Webb的死对我打击很大\...我们相爱了这么多年，突然失去他，我真的不知道该怎么办。我每天都在想念他，经常忍不住哭泣\...\"*

**需要证据**：给Webb的分手信

**指证过程**：

  -------------------------------------------------------------------------
  Plain Text\
  Zack：\"Vivian，如果你们真的相爱，你能解释一下这个吗？\"（出示分手信）\
  Vivian：（脸色瞬变）\"这\...这个\...\"\
  \
  Zack：\"\'我已经受够了，Webb，我们之间结束了。\'这是你的笔迹吧？\"\
  Emma：\"看她的表情，Zack，你击中要害了！\"

  -------------------------------------------------------------------------

**Vivian第一次谎言升级**：
*（慌张）\"那只是\...那只是我们吵架时写的！情侣之间总会有这种冲动的时候！我写了分手信，但我们很快就和好了！事实上，我们还计划一起去法国旅行，船票都已经买好了！这难道不能证明我们的感情很好吗？\"*

**第二轮指证：旅行计划的谎言**

**Vivian新谎言**：
*\"我知道分手信让你误解了，但真相是我们已经和好了。我们甚至计划下个月去法国度蜜月，船票早就订好了。一对准备分手的人会做这种计划吗？\"*

**需要证据**：撕碎的船票碎片

**指证过程**：

  ---------------------------------------------------------------------------------
  Plain Text\
  Zack：\"既然船票都订好了，那你应该还保留着，对吧？\"（出示塑料袋中的船票碎片）\
  Vivian：（震惊，后退一步）\"你\...你怎么会有这个？\"\
  \
  Emma：\"在你房间的垃圾桶里找到的。如果真的准备去旅行，人们不会撕掉船票。\"

  ---------------------------------------------------------------------------------

**Vivian第二次谎言升级**：
*（深呼吸，努力冷静）\"好吧\...我承认我们最近确实有些问题。我撕掉船票是因为我们吵架了，但这不意味着我们要分手！这只是情侣间的正常摩擦。我们已经决定要和平解决分歧，好聚好散。没有仇恨，没有报复，只是平静地结束这段关系。\"*

**第三轮指证：和平分手的伪装**

**Vivian新谎言**：
*\"我知道这些证据看起来不太好，但真相是我们确实在分手过程中。不过这是一次成熟的、和平的分手。我们都是成年人，没有必要带着仇恨和怨念。我对Webb没有任何恶意。\"*

**需要证据**：Vivian个人日记片段 + Tommy关于Webb与Rita关系的证词

**指证过程**：

  -------------------------------------------------------------------------------------------------------------------
  Plain Text\
  Zack：\"真的没有恶意吗？那你能解释一下你曾经写下的这些话吗？\"（出示日记）\
  Vivian：（脸色苍白）\"那只是\...情绪化的话\...\"\
  \
  Zack：\"\'如果有人背叛了我，我会让他遭受世间最痛苦的事情。\'这是你的笔迹吧？\"\
  Zack：（立即追击，出示Tommy证词）\"Tommy告诉我，Webb最近和一个叫Rita的女老板走得很近。这就是你说的\'背叛\'吧？\"\
  \
  Vivian：（开始颤抖）\"不\...不是你想的那样\...\"

  -------------------------------------------------------------------------------------------------------------------

**Vivian第三次谎言升级**：
*（声音开始哽咽）\"好吧\...我承认我确实很愤怒\...我确实写过那些话\...但那只是因为我太爱他了！发现他和Rita的关系时，我确实想过要报复\...我承认我有杀他的冲动\...但是我没有！我真的没有杀他！我甚至很久都没有见过他了\"*

**第四轮指证：动机确认但否认行凶**

**Vivian承认动机但否认行凶**：
*（完全破防，痛哭）\"是的！我承认我恨他！我为他付出了这么多年，和那些恶心的老男人上床，收集他们的秘密，都是为了总有一天我们能有足够的钱离开这里！但我一次又一次地等待，他一次又一次地承诺\...结果呢？他居然和那个Rita搞在一起！但是即使这样，我也没有杀他！我确实想过，我确实计划过，但最终我做不到！我不是杀手！\"*

**关键时间线获得**：

  -----------------------------------------------------------------------------------------------------------------------------
  Plain Text\
  Zack：（冷静下来）\"我相信你没有杀Webb，Vivian。但你的证词很重要。你能告诉我那天晚上你在哪里吗？有没有看到什么可疑的人？\"\
  \
  Vivian：（擦拭眼泪）\"那晚我曾经去过Webb的办公室，但他让我今晚不要再来，于是我就走了，当时是11点10分左右.\"

  -----------------------------------------------------------------------------------------------------------------------------
