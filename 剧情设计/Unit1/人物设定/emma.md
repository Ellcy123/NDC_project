Unit 1：Webb谋杀案 - 用笔和镜头追查真相的记者搭档

---

📋 基本信息

姓名：Emma O'Malley
年龄：约25-28岁（推断，被Morrison称为"小姑娘"）
职业：《芝加哥先驱报》记者
种族：爱尔兰裔（白人）
居住地：芝加哥（具体不详）
NPC ID：102

重要背景：
- 案发前已独立调查Webb的非法活动数月（"几个月来，我一直怀疑他在进行某种非法活动"）
- 案发当晚潜入蓝月亮歌舞厅寻找证据，藏身储物间时目睹后门有人离开
- 拍到黑衣人的模糊背影照片——不足以当法庭证据，但足以威慑Morrison
- 凭借记者证、相机和对时间线的精确记录，成功阻止Morrison当场逮捕Zack
- 在整个调查过程中既是Zack的搭档，也是每轮循环的对话引导者

外貌特征：
- 身材：年轻女性，行动敏捷
- 面容：年轻、聪慧，眼神坚定而好奇
- 穿着：1920年代女记者装束，随身携带记者证、笔记本和相机
- 特征：经常拿出笔记本记录，手持相机是标志性配置
- 气质：勇敢、机敏、坦诚直率，带着记者特有的好奇心和执着

🎭 核心定位

角色功能：protagonist（主角搭档），每轮循环的开场叙事推动者和调查策划者
叙事作用：
- 每轮循环开篇通过AVG对话总结上轮发现、提出新调查方向，引导玩家进入下一阶段
- 与Zack形成互补搭档：Zack负责推理和指证，Emma负责情报搜集、社交渗透和证据整理
- 案件的感性视角——在Zack冷硬理性的对立面提供人性关怀（如关注Rosa的困境、Jimmy的绝望）
- 调查过程中的积极行动者：伪装进入Morrison家、说服Mrs. Morrison打开书房、整理证据链
- 连接玩家与故事的桥梁——为玩家梳理线索、提供调查思路

核心主题：真相的追求者——在危险与道德困境中用记者的方式寻找正义

💔 核心困境

1. 真相与安全的博弈：
   - Morrison直接威胁她："妨碍司法的罪名够你受的"
   - Whale的存在意味着继续调查有生命危险
   - 但她从未退缩——"我们都在赌"
   - 驱动力是记者对真相的执着和对Webb非法活动的长期关注

2. 功利动机与真诚关怀的交织：
   - 初始动机并不纯粹："真相。还有一个好故事。"——坦白自己想要独家新闻
   - 但随着调查深入，对Rosa的困境、Jimmy的绝望产生真实的同情
   - 从"一个好故事"逐渐转变为对正义的追求
   - 这种转变与Zack从孤立到连接的弧线互相呼应

3. 年轻记者在危险世界中的处境：
   - 独自调查Webb数月，说明她有胆识和能力
   - 但面对Morrison这样的腐败警察和Whale这样的幕后势力，她也有脆弱的一面
   - 在Morrison家搜证时的犹豫："听起来风险很大"——很快被Zack一激就恢复斗志

4. 与Zack的信任建立：
   - 合作基础不是信任而是互相需要
   - Zack的冷硬和不信任是她需要持续突破的障碍
   - 她用坦诚而非讨好来赢得Zack的尊重

🔗 关键关系网

- Zack Brennan：从陌生人到搭档的核心关系。Emma主动提出合作，用"你需要不进监狱"的现实逻辑说服Zack。二人的分工明确：Emma问，Zack观察；Emma社交渗透，Zack搜证推理。Emma叫Zack "Mr. Brennan"（正式中带尊重），Zack偶尔叫她"O'Malley"。她是唯一一个敢调侃Zack吃糖习惯的人——"你去采访的时候能不能别叼着糖？看起来很不专业。"
- Morrison：最初的对手。Emma在开场用时间线质疑和媒体威胁阻止Morrison逮捕Zack，展现了她在高压下的勇气和机智。"我只是记录事实。您要是现在抓人，明天《先驱报》头版就是：警方草率定案。"
- Mrs. Morrison：L2中Emma伪装成记者采访警察家庭，成功获得Mrs. Morrison的信任并进入书房。展现了她作为记者的社交渗透能力——"女人和女人之间比较容易建立信任"。
- Rosa：Emma对Rosa表现出比Zack更多的同情——"她女儿的照片，还有那些医院账单...她看起来很绝望""也许她是被逼的？"这种感性视角平衡了Zack的冷硬理性。
- Webb（死者）：Emma案发前已独立调查Webb数月，掌握Webb非法活动的部分线索。Webb的死打乱了她的调查计划，但也提供了更大的新闻价值。
- Tommy / Vivian / Jimmy：通过对话获取关键情报的对象，Emma通常负责情报搜集和证据整理，为Zack的指证提供支撑。
- Whale：间接威胁对象——Whale在电话中暗示"继续调查会危及Zack和Emma的安全"。

📍 循环出现

- L1（目击者的沉默）：在VIP包厢的储物间冲出阻止Morrison逮捕Zack，展示模糊照片和时间线质疑。酒吧外与Zack结成被迫联盟。搜证阶段与Zack讨论证据（工作记录卡、氯仿毛巾），为指证Rosa做准备。开场对话引导者（emma_001: 酒吧门口对话；emma_002: 搜证后总结对话）。
- L2（腐败的警徽）：总结L1发现，提出去Morrison家搜证的计划。伪装成《芝加哥先驱报》记者，说服Mrs. Morrison允许进入书房。在客厅掩护Zack搜证，负责拖住Mrs. Morrison。结尾对话中总结Morrison只是受雇者，提出追查Webb生意的方向。核心台词："Morrison太太，像您丈夫这样尽职的警察，书房里一定都是正常的工作资料吧？"
- L3（古董的密码）：总结L2的发现，确定调查Webb勒索网络的方向。Whale的电话间接威胁到Emma的安全。协助Zack整理Webb生意的财务证据，为指证Tommy做准备。
- L4（厨师的秘密）：追查Whale线索，协助Zack在Jimmy家和厨房搜集证据。在指证Jimmy的过程中补充关键信息（"这是你接受'Whale'贿赂的记录"）。
- L5（歌姬的仇恨）：协助调查Vivian与Webb的感情线索，分析分手信、碎船票和日记。
- L6（真相大白）：在Jimmy厨房参与最终对峙。Jimmy崩溃时试图劝说："想想你的孩子需要父亲！"Jimmy死后与Zack一起打开Webb保险柜，发现遗书和录音带，将故事引向下一章。核心台词："他为了让家人过上好日子选择了犯罪，最后又为了保护家人选择了死亡。贫穷和绝望把一个普通人变成了杀手。"

💭 内心活动

Emma的心理弧线是从职业驱动的功利主义到被案件真相所震撼后的道德觉醒。

开场时的Emma是一个有野心的年轻记者。她独自调查Webb数月，说明她有超出年龄的胆识和执行力。在VIP包厢冲出来救Zack时，她的动机是复合的：既出于正义感（"一个真正的杀手不会让自己陷入这么明显的陷阱"），也出于新闻直觉（这可能是一个重大新闻）。她坦白承认自己想要"真相和一个好故事"——这种诚实反而赢得了Zack的接受。

Emma与Zack的合作模式是互补而非对等的。Zack是推理引擎，Emma是信息网络。她的记者身份赋予了合法的调查掩护——能以采访为名进入Morrison家、能用媒体威慑作为保护伞。她的社交能力弥补了Zack的孤狼本性。但她也清楚自己的定位，从不与Zack争夺调查的主导权（"当然。你是专家。"）。

在调查过程中，Emma逐渐从旁观的记者变成了情感上的参与者。她注意到Rosa的女儿照片和医院账单——Zack看到的是"绝望的人做绝望的事"，Emma看到的是"也许她是被逼的？"这种视角差异不是软弱，而是人性关怀。它让Zack的冷硬推理不至于变成无情的机器。

L2中伪装进入Morrison家是Emma的高光时刻。她展现了记者特有的社交操控能力——用"英勇警探的家庭生活"报道的幌子打开Mrs. Morrison的心防，用赞美和共情获取信任，用"对Morrison警探的形象也是个好宣传"说服进入书房。这些手法老练而不恶意，展现了她在专业领域的成熟。

Jimmy的死对Emma的冲击是巨大的。她最后的总结——"贫穷和绝望把一个普通人变成了杀手"——不再是新闻角度的分析，而是发自内心的悲悯。案件从"一个好故事"变成了真实的人间悲剧，每一个嫌疑人背后都是被系统压迫的普通人。

🎬 关键场景

1. VIP包厢的冲入（L1开场）：Emma从储物间冲出，面对持枪的Morrison毫不退缩，用记者证、相机和时间线数据逼退Morrison。"枪响11点47分，您破门11点54分。警局到这里要十二分钟。"——在极端危险中展现出超凡的冷静和机敏。

2. 酒吧外的合作谈判（L1）：与Zack从陌生到结盟的完整过程。Emma的策略是坦诚而非讨好——承认照片模糊、承认自己在赌、承认想要好故事。"你需要不进监狱"比任何花言巧语都有效。

3. 搜证后的总结对话（L1 emma_002）：Emma整理证据并与Zack讨论Rosa的谎言。在这个场景中，她同时展现了分析能力（"工作记录显示她在一楼走廊"）和同理心（"她女儿的照片，还有那些医院账单..."）。

4. Morrison家的渗透行动（L2）：Emma伪装成记者采访Mrs. Morrison，一步步获取信任并进入书房。Zack的评价："你确实比普通记者要狡猾。"Emma的回应："我更愿意说是'专业'。"这段展现了她作为记者的核心竞争力。

5. Morrison太太面前的表演（L2）：Emma用"警探家属的不易""让读者更了解警察的工作状态"等话术打开Mrs. Morrison的心防。展现了她在社交渗透方面的专业素养。

6. Jimmy自杀后的沉默（L6结局）：在Jimmy的厨房，Emma目睹了一个为家人做出最极端选择的人。"他为了让家人过上好日子选择了犯罪，最后又为了保护家人选择了死亡。"——这句话标志着Emma从记者角度到人性角度的彻底转变。

7. 保险柜的秘密（L6尾声）：Emma与Zack一起打开Webb的保险柜，读出遗书内容，发现录音带。她在案件收束阶段继续发挥情报解读的作用，将线索引向下一章。

📝 创作要点

对话创作

1. 称呼习惯：在正式场合和其他人面前叫"Mr. Brennan"，后期偶尔直呼"Brennan"或"Zack"（关系拉近的标志）。被Zack叫"O'Malley"时带有一种搭档间的默契。

2. 坦诚直率：Emma不绕弯子。承认照片模糊、承认在赌、承认想要好故事——这种诚实是她最大的武器。
   - 例："一个模糊的背影。不够当证据，但够吓唬Morrison。"

3. 积极推动：在对话中经常是Emma提出下一步方案。她是行动的发起者。
   - 例："那我们去他家搜查。""我是《芝加哥先驱报》的记者。我可以说是来采访'英勇警探的家庭生活'。"

4. 感性平衡：在Zack冷硬的判断之后提供人性关怀的视角。
   - 例："但是...她女儿的照片，还有那些医院账单...她看起来很绝望。"
   - 例："如果她崩溃了怎么办？"

5. 偶尔调侃Zack：是唯一敢说Zack吃糖不专业的人。这种轻松互动缓解调查的紧张氛围。
   - 例："你吃了多少糖了？""你去采访的时候能不能别叼着糖？看起来很不专业。"

6. 对话长度和节奏：Emma的台词比Zack长，信息量更大，常带有分析和推理。与Zack的短句形成节奏对比。
