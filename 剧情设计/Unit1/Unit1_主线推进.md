# Unit1（第一章 · 黑哨之夜）主线推进文档

> 配套 v3 底本《主线故事全貌》§6。本文档只讲"这一章要把主线推进到哪一层"，不复述案件流程。
> 每个信息点标三层：
> 【背景】设计者知道、玩家本章**拿不到** ｜ 【暗示】玩家能瞥见碎片、**不足以下结论** ｜ 【明示】本章玩家**必须拿到**的揭露

## 一、本章主线定位（进度 10%）
让玩家第一次"感到系统存在"，但**绝不解释系统**。玩家看到的是酒吧、勒索、录音碎片、黑警、几个看似松散的名字。核心不是"Webb 死于谁手"，而是"Webb 为什么会死——他摸到了比酒吧更大的东西"。

## 二、五条主线模块各推进到哪

### 金融 / 司法线
- 【背景】湖滨信托银行、Harold Moore、Lakeshore 投资壳都在运转，但本章**完全不出现**。玩家对金融线零认知。

### Tidewater 执行线
- 【背景】Tidewater 是南区最大地产商，靠 1912 事故后低价收的码头地块发家，是后面两次清退的土地接收方。
- 【背景】Webb 之死的真正动因 = 两份筹码：① 通过 Vivian 掌握了足以威胁 Tidewater 总裁 Lawson 的材料；② 勒索银月会背后的 Whale。**这两条玩家本章拿不到**，只在后续回看才懂。
- 【暗示】Vivian、Whale、Morrison、银月会之间有暗线——名字反复出现、有人紧张、有人灭口——但**看不见结构**，更不知谁是网主。
- 【明示】Tidewater 作为背景地产商存在；银月会是有黑帮底色的组织；Webb 死得不简单。

### Miller 家族形象线
- 【背景】Miller 是整张网的顶端，但本章连影子都不该有。
- 【明示】**只有恩情**：Emma 靠 Miller 奖学金读完书，Miller 长期资助南区事故家属。这份恩情必须写得真实可信——玩家此刻不能觉得 Miller 像反派，否则后面"感激崩塌"没有力量。

### 揭露动作（本章的"显影"）
- 【明示】Webb 被杀；录音盘碎片出现"1912 年""南区地块""银月会"。
- 【暗示】三个词各自的钩子：1912→把案件连到旧事故；南区地块→连到土地；银月会→连到黑帮。单独看都不构成结论。
- 【暗示·私人钩子】结尾录音提到 Margaret Brennan / 南区鞋坊（清退名单上有 Zack 母亲）——主线第一次刺到 Zack 私人生活：这不是普通案子，侦探的家人也在系统射程内。

### 南区清退进度
- 【背景】清退机器已运转，第一批名单、贷款计划、区域地图都已存在。
- 【玩家感知】只像远处机器的低鸣——让人不安，但**看不见规模**。本章不展示任何名单 / 地图。

## 三、防提前破梗清单（本章绝不能让玩家知道）
- Mickey = Whale（留第四章） ｜ Miller 是幕后总指挥（留第四章）
- Lawson 是私生子、银月会投毒、O'Hara、Lakeshore 三角（均后续）
- "Webb 握有威胁 Lawson 的材料"——只是背景动因，不讲给玩家
- 命名别混：银月会(黑帮) / 银月(赌场) / 蓝月亮(本章案发酒吧) 是三个不同的东西

## 四、交接下一章（Unit2）的钩子
湖滨信托银行 / 1912 家属名单 / 南区清退计划 / 时间窗口 / 第一批目标——这些是 Unit2 谜题的开门砖。本章只埋"存在"和"重量"，不揭内容。
