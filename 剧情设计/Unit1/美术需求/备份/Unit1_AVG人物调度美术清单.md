# Unit1 AVG 人物调度美术清单

> 版本：v1.0 / 2026-04-29
> 用途：给美术对接人物在 AVG 场景中的进场、退场、站位、姿态切换。覆盖 Unit1 全部 6 个 Loop。
> 来源：从 `AVG/对话配置工作及草稿/Unit1/Loop{1-6}_生成草稿.md` 提取所有影响人物布局的关键调度动作。

---

## 一、核心原则（先看这里）

### 1. 场景不是合影

每个 AVG 场景都不是"全员合照"。同一个 sceneId 内会有：

- 开场已在场的人
- 中途推门进来的人
- 中段退场又返回的人
- 全场背对/侧身/镜中倒影的人

如果按"全员合照"画一张静态图，就会破坏戏剧节奏。每场戏都需要明确：开场布局 → 中途变化点 → 结束布局。

### 2. 门口/门外是独立空间

Zack 推门停在门口、Emma 跟在身后、Morrison 从走廊推门——这些都是"门"作为戏剧分界的硬动作。门口这个位置是独立站位，不是房间内部。

### 3. 背影演出是有意的

L2/L3 厨房 James 全程背对着门、L3 Vivian 用镜子反照——这是设计上的"不让你看清脸"。立绘必须支持背影 / 侧身 / 镜中倒影状态，不能只有正脸版本。

### 4. 退场要分段

- Morrison 退场是"边走边嘟囔，声音渐远"
- James 是"背对着走向储藏室"
- Mrs. Morrison 是"拖鞋摩擦声向厨房远去"

都是音轨 + 动作配合，不能一秒切走。

### 5. 关键体态信号

部分角色全场维持某个固定姿态作为信息密码：

- Anna：手放腹部（暗示怀孕）
- Rosa：腰背不靠墙（紧绷）
- Mrs. Morrison：整晚才失手一瞬（演技与破绽）

立绘需要为这些核心体态准备差分表情/姿态。

## 二、Loop 1

### Loop1 §1 蓝月亮门口（sceneId 9021） → 切到 9004 大堂

| 节拍 | 调度 |
|---|---|
| 节拍一 | Zack 单独被酒保拦在暗门外，Emma 不在场 |
| 节拍二 | Emma 才出现，推门入内、Zack 跟进 |

注意：节拍一和节拍二是不同的人物布局，一个只有 Zack + 酒保，一个加上 Emma。

---

### Loop1 §1 9003 Webb 会客室（最容易踩坑）

开场实际只有 4 人 / 1 尸：

- Webb 尸体（地上）
- Vivian（房间正中，持枪呆立，眼神涣散）
- Rosa（门边靠墙）
- Zack + Emma（推门进来停在门口）

Morrison 是中途进场（901001032 之前的注释明确）：

> 走廊里传来脚步声，迟了半拍……Morrison 推开门，帽檐压着，身上隐约有酒气。

→ 美术如果把 Morrison 画在开场全景里，就破坏了"Vivian 持枪呆立 + Rosa 当场指证 + 警察迟到"的悬念节奏。Morrison 必须晚进来，配合走廊脚步声，玩家先看到的是无人主持的混乱现场。

§2 / §3（Vivian Talk / Rosa Talk）期间，Morrison 持续蹲在尸体旁勘验——这是背景人物状态，不是消失。§4 节拍开场说"Zack 转回来走向仍蹲在尸体旁的 Morrison"。

### Loop3 §X Morrison 退场（两段式）

非常重要的退场节奏，美术分两个动作：

| 步骤 | 描述 |
|---|---|
| 第一段 | Morrison 走了两步又回头（肩膀先动头再跟）→ 抛一句关键信息 |
| 第二段 | 帽檐往下压了压、转身往外走、边走边嘟囔，声音越来越远 |

不能一次走完。

## 八、给程序的对接需求摘要

为支持上述调度，程序需要在 AVG 系统中支持以下机制：

1. 同一 sceneId 内增量加 / 减角色立绘（不切场景）
2. 在指定对话 ID 触发"角色进场动画 + 音效"（参见《需求 6：场景中出现新人物》）
3. 在指定对话 ID 触发"角色退场动画 + 音效"（参见后续需求）
4. 立绘锚点系统（stage_left / stage_center / stage_right / behind_zack 等）
5. 立绘姿态/表情差分切换（同一角色多套 sprite）
6. 镜中倒影 / 背影 / 侧身等特殊状态立绘加载
