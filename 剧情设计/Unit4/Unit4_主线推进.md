# Unit4（第四章）主线推进文档

> 配套 v3 底本《主线故事全貌》§10（Unit4 上）+ §11（Unit4 中）+ §12（Unit4 下）+ §13（Unit4→Unit5）。本文档只讲"这一章要把主线推进到哪一层"，不复述案件流程。
> 每个信息点标三层：
> 【背景】设计者知道、玩家本章**拿不到** ｜ 【暗示】玩家能瞥见碎片、**不足以下结论** ｜ 【明示】本章玩家**必须拿到**的揭露

## 一、本章主线定位（进度 80% → 100% 前夕）

Unit4 是全剧**揭露最重的一章**。前三章一层层铺起来的"网"，在这一章被掀开顶盖：玩家不再只是怀疑系统存在，而是看见系统的真身、看见它的指挥层、看见它怎么杀人。本章按**四段递进**推进，节奏一段比一段重：

- **（上）两轨合流 + 知道真相会死人。** 金融轨道与地产轨道在这里合流，"南区综合商业开发计划"首次完整命名，玩家终于知道此前所有债务 / 司法 / 房产流转都服务于一个有商业目标、有时间窗口、有空间范围的开发项目。Harrison 改判后被灭口——系统从"腐败（拿钱办事）"升级为"有人在主动控制风险、清理失控者"。
- **（中）知情人接连消失。** Morrison（Unit1 起就在场的腐败黑警）试图向 Zack 说出 Whale 真身时被灭口——Mickey（作为 Whale）亲手处理、伪造现场并压下真相。本段把 Zack 推到 Whale 真身的边缘，进入最痛苦的认知状态：**他不是不知道该怀疑谁，而是不敢怀疑。**
- **（下）Mickey 的圆。** Zack 质问 Mickey，Mickey 亲口承认自己 = Whale，并指认 Miller→Lakeshore→Tidewater 三角结构。Mickey 坠楼死亡，遗留两类内部档案。Emma 在档案里看见父亲 Sean O'Malley 的名字——父亲之死第一次从"事故"变成"谋杀疑云"。
- **（Unit4→Unit5 断点）井与崩塌。** Mickey 档案已经给到水源 / 井区异常安排，足够让 Zack 推出"南区清退可能会从水源下手"；但 Zack 没有第一时间把这份信息完整摊开，也没有和 Emma / Watts 一起推到底。小 Charles 未经老 Charles 授权，把原本还在准备 / 后续阶段的清退计划提前执行，让银月会对南区公用井投毒——计划从"体系内劝退"升级为"过激物理清退"。Emma 情绪崩溃、离开 Zack 前往 Miller 庄园找小 Charles，情感裂痕真正爆发。

本章把系统的几乎全部结构都端上桌，但**严守 v3 §18.1"每个 Unit 只揭一层"的累积版**：能在本章坐实的就坐实（计划命名、Whale 真身、三角结构、灭口模式），而几个临近终局的关键真相——Sean 被谋杀的彻底坐实、Lawson 私生子、小 Charles 弑父嫁祸、投毒主使的完整指认、金融分赃链的证据层——**虽然近在眼前，仍必须留给 Unit5**。本章给的是"几乎成形、却差最后一块证词"的状态，这正是 Unit5 小 Charles 能利用 Emma 的前提。

## 二、五条主线模块各推进到哪

### 金融 / 司法线

- 【明示·上】**Harrison 改判后被灭口**。他原本是 Miller 金融网络的司法工具，但一旦做出损害体系利益的判决（改判让银行保险操作进入法庭记录、让他这个司法工具第一次背叛体系），就变成需要清理的风险。**点清死因性质**：Harrison 不是死于"单纯腐败"，而是死于"损害 Miller 金融网络利益、背叛了体系"——这与"拿钱办事"的腐败有本质区别，说明背后有人在主动控制风险。
- 【明示·下】**Miller 是真实投资方**。Mickey 亲口承认：Tidewater 和湖滨信托银行背后的投资结构真正指向 Miller 家族。Lakeshore Investment Co. 不再只是 Unit2/Unit3 的"可疑壳 / 司法腐败资金来源"，而是 **Miller 控制金融与地产的中转结构**。必须点清分工：Moore 是银行执行人、Lawson 是地产执行人、Mickey 是法律 + 暴力接口，**Miller 才是总指挥层**——不是 Moore 在控制一切，也不是 Lawson 在控制一切。
- 【背景·下】金融分赃链（世博会选址收益如何在 Miller / Lakeshore / Moore 之间分配）的**证据层坐实留 Unit5**。本章 Mickey 给的是"口头指认 + 档案指向"，不是可在法庭立案的分赃账目。
- 【背景·Unit4→Unit5】Mickey 档案显示，1928 年清退**原计划主要依靠湖滨信托银行的债务陷阱和房产接收**；但档案中已经出现水源维护、私人井区标注、"可处理点"这类危险安排。它们不等于一张写明"投毒"的命令，却足够让 Zack 推出：如果清退加速，水源会是最可能被下手的接口。投毒是小 Charles 越权提前执行的加速，不是老 Charles 原本节奏里的既定公开步骤。

### Tidewater 执行线

- 【明示·上】**"南区综合商业开发计划"首次完整命名**。Harrison 遗留文件揭露 Tidewater 需要这些房产的真正目的：清空这片土地、套现世博会选址带来的地价与开发资格。此前的债务、司法、房产流转，在本段被收束成"一个有商业目标、有时间窗口、有空间范围的开发项目"。
- 【暗示·上】Mickey 否认 Tidewater 内部正在执行这个计划。他**否认得越熟练，Zack 对他的怀疑越重**，但本段还不能把 Mickey 与 Whale 直接画等号。
- 【明示·中】**Morrison 死于即将说出 Whale**。Morrison 是 Unit1 就在场的腐败黑警，是黑警、Whale 和案件真相之间的活证人；当他试图向 Zack 揭露 Whale 真身时被 Mickey 灭口。Mickey（作为 Whale）亲手将现场伪造成自杀，但 Whale 的痕迹仍留在现场（枪、左撇子破绽等），有人持续遮掩。
- 【明示·中】**Mickey 压下 Morrison 死亡真相**。Mickey 亲手把现场做成自杀、在警局按"自杀"口径翻供压下真相，留下越来越多与自己相关的疑点（如枪放错手）。注意呈现方式：不是玩家"聪明地猜到 Mickey 是 Whale"，而是 **Zack 亲眼看见 Mickey 在关键节点遮掩、说不通**——怀疑由眼见的行为堆出来，不是推理跳跃。
- 【明示·下】**Mickey = Whale**。Zack 质问后，Mickey 承认自己既是律师也是 Whale：负责把 Tidewater 无法写进合同的部分交给黑警和银月会，负责灭口、威胁、暴力、压制不配合者。**关键定性**：他不是外部黑帮，也不是被黑帮胁迫的律师，而是 **Tidewater 内部执行层与黑帮 / 警察之间的接口**。
- 【明示·Unit4→Unit5】**投毒把 Tidewater 与银月会的关系从影子变成事实**。小 Charles 为强行推进计划，让银月会对南区公用井投毒。O'Hara 死后留下的材料证明 Tidewater 清退的升级逻辑：先用 Whale 做个人清退；一旦 Tidewater 与银月会联系成熟，便可考虑大型清退；**这种大型清退与 1912 年事故制造方式同源**——把人命伪装成事故，把土地问题伪装成城市运转中的自然灾害。

### Miller 家族形象线

- 【明示·上】**阴影加重但未正面暴露**。Harrison 的死发生在"损害 Miller 金融网络利益"之后，这让玩家意识到：能调动银行、司法、地产和灭口的人，不会只是 Harold Moore 或 Lawson 这种执行层。Miller 的阴影压到台前，但本段**不正面给 Miller 任何确证**。
- 【背景·中】本段 Miller 家族**不必增加正面信息**，重点全部放在 Zack 对 Mickey 的信任崩塌上。Miller 是远处的系统，Mickey 是眼前的人——本段是"眼前的人"的崩塌时刻。
- 【明示·下】**恩人外壳崩塌**。Mickey 坠楼前亲口指认 Miller 家族是 Tidewater 与 Lakeshore 背后的真正投资方。至此，在 Zack 和玩家的主线认知里，Miller 从 Emma 的恩人变成清退背后的**真正指挥者**。Miller 十六年的恩情开始摇晃。
- 【明示·Unit4→Unit5】**小 Charles 失控**。他未经老 Charles 授权越权推进计划、让银月会投毒。点清两代人的差异：老 Charles 用二十多年搭结构、等窗口、控制节奏；小 Charles 只看见世博会时间窗口、继承压力和眼前风险——**他等不了**。他不是简单的"更坏"，而是更急、更短视、更愿意把系统的隐性暴力变成显性暴力。（这为 Unit5 父子冲突埋逻辑，但**老 Charles 踩刹车 / 父子冲突 / 弑父本身留 Unit5**。）

### 揭露动作（本章的"显影"）

按四段顺序列出本章必须给到玩家的揭露：

- 【明示·上】"南区综合商业开发计划"首次完整命名；金融轨道与地产轨道合流——主线从"有人在清退南区"升级为"有人为了世博会套现、系统性清空土地"。
- 【明示·上】Harrison 改判后被灭口（因损害 Miller 金融网络利益、背叛体系）。
- 【明示·中】Morrison 被灭口，**形成模式**：Webb 死于知道太多 / Harrison 死于背叛体系 / Morrison 死于即将说出 Whale。玩家得到的不是答案，而是**规律**——知情者正在接连消失。
- 【明示·下】Mickey 的 Whale 身份彻底揭露。
- 【明示·下】Miller→Lakeshore→Tidewater 三角结构曝光（Miller 总指挥 / Moore 银行执行 / Lawson 地产执行 / Mickey 法律 + 暴力接口）。
- 【明示·下·遗留档案】Mickey 留下两类内部档案：
  - `1912 年档案`：Tidewater 地基造假材料单（证明港口事故并非单纯意外）+ 针对 Sean O'Malley 的"特殊处置令"（**老 Charles 亲笔签名**）。
  - `1928 年档案`：南区综合商业开发计划清理名单（含 O'Hara）+ 水源维护 / 私人井区异常安排。档案不直接写"投毒令"，但已经足够推出水源风险；此阶段仍不能完整指认小 Charles 或银月会。
- 【明示·下】Mickey 在爆炸后预感自己可能死亡，给 Zack 和 Emma 留下帮手 **O'Hara**——既是赎罪，也仍带着他"安排别人该怎样走"的老毛病。
- 【明示·Unit4→Unit5】小 Charles 越权让银月会对南区公用井投毒：计划从"体系内劝退"升级为"过激物理清退"。

### 南区清退进度

- 【明示·上】**清退进入公开计划层**。此前玩家只看见债务和个案，本段知道这些动作都服务于一个开发计划——清退不再是模糊压迫，而是**有范围、有名字、有截止时间的项目**。
- 【明示·中】清退继续逼近暴力阶段。任何会泄露 Tidewater / 银月会 / Whale 关系的人，都可能被处理（保护机制收尾，知情人口被堵住）。
- 【明示·下】金融劝退已接近完成，名单上的家庭被逼到边缘。**按原计划仍主要通过银行和合同完成清退，暴力是备用手段**——这一定性很重要，它让 Unit4→Unit5 的投毒显得是"越权升级"而非"既定方案"。
- 【明示·Unit4→Unit5】投毒造成第一批核心名单家庭大量死亡。**注意状态拿捏**：不是"整个南区居民房产清退彻底完成"，而是——第一批最难处理、最容易暴露计划的核心钉子户被一次性拔掉；部分房产清退已事实完成；整体计划接近落地，但仍有未完成程序，后续才会因 Unit5 曝光进入司法审查。

## 二·补 — 本章情感线（明示层 + 暗层，本章必须立住的情感支点）

### Mickey 坠楼后的第一次救赎（明示）

- 【明示·下】Mickey 坠楼后，Zack 站在打开的窗边，反复回想自己没能抓住他，也反复回想 Unit3 里被 Mickey 推着越过的那条线。他最痛的不是"被骗"，而是 **Mickey 用这件事证明他们是一样的人**。
- 【明示·下】Emma 没有用安慰把他拉回来。她先**关上那扇窗**，让 Zack 不再站在 Mickey 最后选择的位置上；然后告诉 Zack：**"被利用不等于选择成为他。"** Mickey 把人算成代价，Zack 不是。这句话回收前三个 Unit 里 Emma 对 Zack 的影响（她一直把 Zack 从冷硬的现实主义里拉回来，让他看见案件背后的人）。
- 【明示·下】Zack 因此第一次从"我是不是也变成 Mickey"里退出来，重新选择把证据交给世界，而不是继承 Mickey 的算法。

### Emma 的认知状态：从"事故"到"谋杀疑云"（明示 + 防破梗边界）

- 【明示·下】Emma 在档案里看到 Sean O'Malley 的名字。对她而言，此阶段父亲之死**第一次从"事故"变成"谋杀疑云"**。
- 【背景·关键边界】但本章**还没坐实"Miller 杀父"**。真正击中 Emma 的不是"已经坐实"，而是"开始动摇"——Miller 十六年的恩情开始摇晃，最后一块证实还要等她去见小 Charles（Unit5）。这条边界必须守死：写成"已坐实"会提前烧掉 Unit5 小 Charles"真相武器化"的全部张力。

### Zack 隐瞒档案的逻辑（暗层 — 设计者意图，玩家本章读不透）

- 【背景】Zack 暂时隐瞒档案内容，**不是因为不信任 Emma**，而是因为 Webb、Harrison、Morrison、Mickey 都死在"知道太多"之后，街对面有人盯 Emma，说明她很可能已被纳入清理名单。他的判断是：先找到 O'Hara、确认名单和可用证据，再决定怎么告诉 Emma。
- 【背景·讽刺核心】这个判断有合理性，但**不是正确的**：Mickey 档案里不仅有 Sean 的名字，也有水源 / 井区异常安排。Zack 手里已经有可能救人的信息，却先选择控制信息、保护 Emma、判断她能承受什么；他没有第一时间把材料摊开，也没有和 Emma / Watts 一起把"水源风险"推到底。Zack 刚被 Emma 从 Mickey 的算法里拉出来，却又在恐惧里**重复了同一种算法**——先判断 Emma 能知道多少，再决定她何时可以知道父亲怎么死、何时可以参与可能救人的推理。但这正是 Emma 后来愤怒的点：**"你不是在保护我，你是在替我决定我什么时候可以知道真相；而这一次，真相可能本来能救人。"**
- 这一层是本章情感线的暗雷，**在 Unit4→Unit5 断点引爆**，不在本段点破其讽刺性。

### Emma 离开 Zack、前往 Miller 庄园（Unit4→Unit5 断点 · 明示行为 + 认知逻辑）

- 【明示·Unit4→Unit5】Emma 看见 Sean 相关材料情绪崩溃，离开 Zack，前往 Miller 庄园找小 Charles。
- 【明示·认知逻辑】这个选择不是纯粹冲动，而是有当时的认知逻辑：Mickey 已死；Zack 明显隐瞒了她；她带着"几乎成形、却差最后证词"的怀疑；老 Charles 太远（像真正系统的顶端），而**小 Charles 是她唯一认识、曾经提醒她"别深追、会危险"的 Miller 成员**——她以为小 Charles 也许知道部分真相、能解释 Miller 与 Tidewater 的关系、甚至能阻止失控局面。
- 【背景】这正是 Unit5 小 Charles 能利用她的原因：她去找他时**不是已掌握完整结论，而是带着一个几乎成形、却还差最后证词的怀疑**——小 Charles 给她的不是安慰，而是把这个怀疑彻底坐实的"完整真相"（真相武器化）。这层留 Unit5。

## 三、防提前破梗清单（这些虽然临近终局，但仍必须留给 Unit5）

- **Sean 被谋杀的"彻底坐实"**——本章只到"谋杀疑云"（看到名字 + 老 Charles 签名的特殊处置令 + 地基造假材料），**彻底坐实靠 Eleanor Foster 的法庭证词，留 Unit5**。
- **Lawson 是老 Charles 私生子**——本章 Lawson 仍只是 Tidewater 名义总裁 / 地产执行端的签字人，**私生子身份的坐实留 Unit5**。
- **小 Charles 弑父 + 嫁祸 Emma**——本章只到小 Charles"越权失控"，**弑父与嫁祸留 Unit5**（父子冲突、老 Charles 踩刹车也留 Unit5）。
- **投毒主使的完整指认**——本章断点可以让玩家知道：Mickey 档案已能推出水源风险，小 Charles 把计划提前并越权推进，投毒很可能由银月会执行；但**完整指认（Vivian 物证指认小 Charles 借银月会投毒）留 Unit5**。本章给的是"足够后悔的指向"，不是"法庭级坐实"。
- **金融分赃链的证据层坐实**——Miller / Lakeshore / Moore 如何分世博会收益，**证据层留 Unit5**。本章 Mickey 给的是口头指认 + 档案指向。
- **Zack 隐瞒逻辑的"讽刺性"在 Unit4 内才点破**——这层在 Unit4→Unit5 断点 Emma 爆发时点破即可，不要在（下）段就替玩家说穿 Zack 在重复 Mickey 的算法。
- 命名别混：**蓝月亮** = 酒吧（Unit1 案发地）｜ **银月会** = 黑帮组织（Whale / 暴力 / 投毒执行接口）｜ **银月** = 赌场——三个不同实体，不混用；城市名一律**艾斯弗德**。
- 实体别混：**Tidewater** = 潮汐开发公司｜**湖滨信托银行**（行长 Harold Moore）｜**Lakeshore Investment Co.** = Miller 隐藏金融的上层投资壳——三者分工不能写混（Moore 银行执行 / Lawson 地产执行 / Mickey 法律 + 暴力接口 / Miller 总指挥）。

## 四、交接下一章（Unit5）的钩子

本章把系统的真身几乎全部端上桌，但每一条都留了"差最后一块"的口子，正是 Unit5 收束前四章全部主线债务的开门砖：

- **Sean 的"谋杀疑云"已立 + 老 Charles 签名的特殊处置令 + 1912 地基造假材料** → Unit5 由 Eleanor Foster 法庭念出 Sean O'Malley 的名字，把谋杀**彻底坐实**，让父亲第一次从"事故数字"变回一个被点名的人。
- **Emma 带着"几乎成形、却差最后证词"的怀疑去 Miller 庄园找小 Charles** → Unit5 序幕小 Charles 把 1912 真相精准交给她（真相武器化），给她当众质问老 Charles 的完整动机，也为嫁祸埋线。
- **小 Charles 越权投毒、失控** → Unit4 末尾已经给出水源风险可被推出、但 Zack 没及时共享 / 推到底的痛点；Unit5 老 Charles 要踩刹车、小 Charles 等不了，父子冲突终点是弑父 + 嫁祸 Emma；Vivian 在物证辅助下指认小 Charles 借银月会投毒，闭合 Tidewater–银月会暴力链。
- **Miller→Lakeshore→Tidewater 三角已口头指认** → Unit5 把金融分赃关系落到证据层（Moore / Lakeshore 账目、世博会收益分配），形成"银行造债务 / Lakeshore 承接资金 / Miller 实际收益 / 司法保护过这条链"的完整闭环。
- **Lawson 作为地产执行端 + Mickey 遗留档案指向 Tidewater** → Unit5 坐实 Lawson 私生子身份与继承动机，并由 Lawson 在终局阻止销毁罪证、让小 Charles 被捕。
- **Zack 隐瞒 / Emma 裂痕已爆发** → Unit5 收束这条情感线：Emma 在复仇边缘停下（用她当初救 Zack 的价值观反过来救自己），没有变成 Mickey；Zack 接住她停下之后的崩塌。
- **Mickey 档案 + O'Hara 已牺牲后留下的材料** → 前者让水源风险在 U4 末已经可推，后者把 Tidewater 与银月会的大型清退方案落到现场结果，二者共同成为 Unit5 揭穿投毒链的物证起点。
- 本章只把这些钩子推到"差最后一块证词 / 差最后一次坐实"的位置，**最后的坐实、嫁祸、法庭与情感收束全部交给 Unit5**。
