## 一、全章核心

U4 不是一章"揭晓 Mickey 是坏人"的故事。

它讲的是 Zack 亲手把几类原本互不相干的东西放到一起：Harrison 的改判、圣心医院的儿童死亡、Morrison 的灭口、O'Hara 的拒售、Patrick 的工运遗产，以及 Mickey 一次次真实有效的帮助。玩家最后发现，这些事情背后有同一个人，也有同一种算法。

这套算法不是简单的恶意，而是：

> 我比你更了解局势。
> 所以我可以替你决定什么时候知道真相，也可以替别人决定谁值得成为代价。

本章对人物保持开放，对这种权力保持明确的警惕：

- Harrison 和 Morrison 有罪，但他们仍有回头与开口的权利。
- Patrick 确实改善了八百个工人的生活，但他没有权替 Liam 说"值得"。
- 1919 年的 Patrick 仍先冒险推动最后一次证据转运，却在危机真正落到一个具体的人身上时放弃成果、自己折返；这证明他能够作出与 1903 年不同的选择，却不能取消他再次把风险带进家庭的责任。
- Mickey 把"被救"误读成"被选中"，继承了 Patrick 已经放弃的旧版本，并用后来每一笔牺牲证明自己值得被救。
- Mickey 确实救过 Rosa 和 Mary，但真实成果不能赋予他分配人命的资格。
- Zack 的动机比 Mickey 温和得多，但他在恐惧里也会靠近同一条边界。

## 二、图例与执行规则

- 🟢 证据 / 道具：玩家可以获得、拍照、分析或用于指证。
- ⚪ 证词：NPC 说出的关键有效信息，可进入证词池或用于明确推理。
- 🟡 谎言：当前指证中会被击破的具体说法。
- 🟣 环境叙事：用于讲故事、铺情绪或后续回收，通常不用于指证。
- 🔵 铺垫 / 回收：不能单独出现，必须附着在其他图标上。
- 🔒 动机信息：指向角色为什么作出选择。
- 🎮 玩法：玩家需要主动操作、比对、连线或输入。
- 🧪 派生证据：由分析、组合或记忆复审产生。
- 💡 尤里卡：希望由玩家自己想通的关键时刻。

执行硬规则：

1. L1-L4 每轮谎言最多由两件材料击破。
2. 指证材料必须在该轮指证开始前可获得。
3. 价值判断不能伪装成事实谎言；情感坦白放在事实防线被击穿之后。
4. 每个 Loop 只完成一个主要认知翻转，同时交出进入后续段落的行动理由；L1-L4 进入下一 Loop，L5 进入非 Loop 终幕。
5. Zack 只指出矛盾，不替玩家把所有结论说完。

## 三、全章事实时间线

### 1903—1919：算法的来源与第一次修正

- 1903 年，Patrick 作为工会法律顾问坚持罢工谈判。清场中 Liam Donnelly 和另外两名工人死亡，少年 Mickey 右脸受伤、右下牙缺损。
- 三人的死亡推动舆论与谈判，约八百名工人的工时从十四小时缩短到十小时。
- Patrick 说："如果 Liam 知道结果，他也会觉得值得。"他可能是在安慰别人，也可能是在安慰自己，但他替 Liam 作出了决定。
- Mickey 最初恨 Patrick，后来为了不让父亲白死，选择相信 Patrick 的解释。
- 1919 年，Mickey 一边为 Miller 体系提供法律服务，一边仍向 Patrick 声称自己在"从内部取证"。交接点已经有暴露的迹象，Patrick 仍同意完成最后一次原始账簿转运，也接受了 Mickey"内部通行权限还能多撑十分钟"的判断——两人都没有把升级后的风险告诉家里。
- 清场提前开始后，有一张登记表如果被人发现，会同时暴露工人联系人和 Mickey 的内部访问痕迹。Mickey 为了撕掉它，主动留在内侧档案室，错过了撤离窗口。清场人员开始纵火销毁没能带出的档案。
- Patrick 已经带着装有唯一原始账簿的封存箱到达内侧防火闸门。最后一名转运员负伤被拖向外侧，现场没有人能接走封存箱。闸门一旦关闭就无法返回。Patrick 听见"西侧还有一个人"，来不及确认被困者的身份，放下封存箱，转身折返。
- 封存箱随后被清场人员焚毁，只剩可对应交接编号的金属扣。Patrick 把被困的 Mickey 送出侧门后遭警棍重击，送医后始终没有恢复可交流意识，于次日凌晨因内伤去世。
- Zack 从 1919 年起就知道父亲在清场中把 Mickey 带了出来；Margaret 当时给他的版本是两人都没来得及撤出，Patrick 只是在共同逃生时受伤。她隐去的是 Patrick 已经到达内侧出口、原本准备回家、却放下账簿主动折返的完整选择。
- Patrick 葬礼后，Mickey 对 Zack 说"他没做完的，我会接着做"，并坚持 Patrick 最后选择了真正理解他的人；同年 Donnelly & Associates 开始承接银行、医院与地产法律业务。Margaret 亲眼看见 Zack 被"父亲最后选了 Mickey"刺中，也看见 Mickey 把获救解释成使命交接，于是封存了返程票、补充记录和账簿扣，不愿让 Zack 沿这条解释继续走下去。

### U3 后段：Zack 的第一次隐瞒

- Emma 发出帮助 Mary 的报道后，Zack 才最终确认 Mary 与 Helen 提前策划了 Thomas 的死亡。
- Zack 没有告诉 Emma。他认为报道已经救了 Mary，继续揭露只会毁掉已经救下的人。
- 这不是 U3 报道造假，而是 Zack 在真相形成之后，替 Emma 决定她不需要知道完整版本。

### U4 当下

- Harrison 在 Mary / Helen 案上两次被要求收回改判意见，仍第三次提交合法杀人 / 正当防卫方向的正式意见。
- 同期，他连续两个月调阅圣心医院旧案，整理自己收受资金、儿童死亡与南区开发计划之间的关系。
- Harrison 从"可控的腐败法官"变成准备违抗系统、可能公开材料的风险源，因此被杀。
- L1 后，系统允许 Zack 沿圣心医院执行层调查，因为牺牲 Whitfield 可以保护 1919 法律壳和地产结构。
- L2 后，Morrison 看见 Whitfield 被迅速交出去，又收到要求次日交出旧案文件的通知。他意识到 Harrison 的清理方式正在轮到自己。
- L3 当夜，Morrison 尝试联系 Zack。Mickey 因当晚警察与银月会可用执行者都被其他任务锁死，只能亲自到场。
- Mickey 杀死 Morrison、匆忙把枪摆进错误的右手；Miller 上层经 Pierce 在 Morrison 被标记为风险后另行安排了定时清场。Mickey 只知道现场会被处理，不知道装置同样允许牺牲返回现场的自己。
- L4 中，O'Hara 的当下拒售让 Patrick、Liam、Margaret 与 Mickey 的过去重新进入现实。
- L5 中，Mickey 主动邀请 Zack 来四十二层进行一场受控对质，但没有准备交出私人保险柜里的身份材料。Zack 与 Emma 用 L4 重新进入推理链的 Patrick 遗物匣刻句破解保险柜，主动越过 Mickey 预设的会面边界。
- Mickey 坠落后，系统没有停止。小 Charles 已经提前推进针对南区水源的过激清退。

# Loop 1：守门人的钥匙（起）

## 本轮功能

核心问题：Harrison 是因为腐败而死，还是因为准备背叛腐败系统而死？
指证对象：Watts。
指证轮次：3。
本轮边界：只能确认 Harrison 正在回头、死亡与改判及旧案复查高度重合；不能在 L1 直接证明是谁下令杀他。Mickey 只留下"知道得太准"的轻微异样。

## 开篇剧情

**法院东翼 / Harrison 办公室外**

Zack 和 Emma 原本来补办 Mary / Helen 案文件，却遇到法院东翼封锁。Harrison 前一夜死在办公室里，现场只能先确认他死于枪伤。

Pierce 负责现场。他的态度始终体面：桌面与公共档案可以查看，Harrison 私人内柜、保险柜和封存箱必须等法院授权。Harrison 已确认死于枪伤，具体供弹来源尚未完成比对。

Watts 抱着 Harrison 的钥匙链和旧卷宗箱，不肯交给 Pierce，也不肯交给 Zack。他不是相信 Harrison 无罪，而是不肯让任何人只用"腐败法官畏罪死亡"六个字写完 Harrison。

Mickey 也在。他说自己来法院处理另一个案件，听档案室的人说 Harrison 最近一直在翻圣心医院旧案，因此建议 Zack 先查调阅记录。

行动目标成立：玩家先调查 Harrison 最近在做什么，再击穿 Watts 的守门理由，获得保险柜和私人内柜。

## 自由探索

### Harrison 外间办公室

- 🟢 Harrison 本周公开日程：只有日常庭审和例行会议，没有需要连续深夜工作的常规事项。
- 🟢 法院夜间出入登记：Harrison 连续十七晚独自进入档案室或办公室，时间集中在闭庭后。
- 🟢 `1919-A` 入账存根：从普通费用抽屉中找到。付款方栏只有匿名代号 `1919-A`，备注栏却连续引用圣心医院赔偿案号；款项最终进入 Harrison 私人账户。Harrison 长年把这些存根夹在普通收入凭据中，没有刻意销毁。
- 🟢 未完成的辞职信草稿：在打字机废纸篮中，日期为 11 月 27 日，写有"我不能一面要求他人承认罪行，一面继续否认自己的部分"。
- 🔵🟣 南区开发计划复写纸：只能辨认出标题"南区综合商业开发计划"和"Tidewater / Lakeshore"两个词，正文仍在保险柜里。
- 🟢 Harrison 案弹头 / 弹壳与警局配发弹药样本：现场弹壳和尸内弹头由法院证物员提供比对，警局合同批次使用独有的深蓝底火漆与双环压口。玩家完成弹药比对后，只能确认执行者能接触警用供弹渠道，不能单独证明 Pierce 或 Morrison 动手。

### 法院档案室 / 档案管理员

这次聊天只提供程序事实，不评价 Harrison 的人格。

- 🟢 Harrison 两个月调阅索引：七次调阅圣心医院儿童死亡旧案，三次调阅南区地产判决，两次调阅自己签过的赔偿裁定。
- ⚪ 档案管理员：Harrison 每次都自己抄写，不让书记员代办；他最近没有让 Mickey 来取过这些卷宗。

Mickey 先前明确说"档案室的人告诉我他在查圣心医院"。档案管理员的说法不能证明 Mickey 撒谎，因为他可能通过电话或别的书记员得知；它只留下一个后续会变味的问题：Mickey 为什么知道得这么具体？

### Zack 公文包 / U3 已持有材料

- 🟢 Mary / Helen 案改判往来信：Harrison 第一次提出改判后被以舆论风险驳回；第二次补充事实依据后再次被退回；第三次仍坚持提交正式意见。
- 🔵🟣 第三次提交回执：时间就在 Harrison 死前三天。L1 只能形成时间关联，L5 才从 Mickey 的行动记录中坐实它是清理触发点。

### Watts / 可对话

- 🟡 Watts：Harrison 只是工作多，最近没有偷偷重查任何旧案。
- ⚪ Watts：Harrison 不让书记员碰最后几晚的材料。
- 🟣 Watts 怀中的钥匙链：他每次听见 Pierce说"畏罪"都会攥紧钥匙，但仍不肯解释原因。

## 指证 Watts

| 轮次 | 谎言 | 击破材料 | 玩家可得结论 |
|:---:|:---|:---|:---|
| R1 | 🟡 Harrison 只是处理日常工作，没有秘密重查旧案 | 🟢 本周公开日程 + 🟢 两个月调阅索引 | 深夜工作内容与公开日程不符，他在系统性复查医院、地产和自己的判决 |
| R2 | 🟡 即使他查旧案，也是在调查别人，不是在查自己 | 🟢 `1919-A` 入账存根 + 🟢 调阅索引中的本人赔偿裁定 | Harrison 把自己收过的钱和自己签过的裁定放进了同一条调查线 |
| R3 | 🟡 他不会真的违抗法院，更不会把事情公开 | 🟢 Mary / Helen 三次改判往来信 + 🟢 未完成的辞职信草稿 | Harrison 已经连续违抗上层意见，并开始准备承担公开后果 |

## 指证后：保险柜只打开这一次

Watts 交出钥匙。他承认 Harrison 死前告诉他：如果第二天没有来，就不要先把钥匙交给 Pierce。

保险柜里不是腐败藏宝箱，而是一只尚未完成的自首包：

- 🟢 圣心医院儿童死亡对照表：五年七例，家庭背景、慈善项目、药剂批次和责任归属高度相似。
- 🟢 Harrison 亲笔资金流向图：圣心医院控股集团专项拨款账户 → 匿名中转账户 `1919-A` → Harrison 私人账户。Harrison 通过案号、日期和金额复原了资金方向，但没有查到开户实体、签字人或实际控制人；L4 的旧信封只形成 Donnelly 关联怀疑，L5 才由保险柜内的银行授权附页坐实。
- 🟢 南区综合商业开发计划摘要：首次完整命名项目；写明以世博会候选地窗口为期限，通过银行债务、司法裁定和地产接收清空南区地块。文件出现 Tidewater 与 Lakeshore，但没有 Miller 名字。
- 🟢 1912 事故家庭索引：附在开发计划末页，使用旧事故赔偿编号标记住户；同一批编号后来又出现在"南区居民援助计划"的贷款、违约与房产接收记录中。
- 🟣 Harrison 未完成的自首证物箱：它是保险柜内的场景容器，不额外生成一张总证据卡。Mary 改判、医院名单、资金图和开发计划按实际打包顺序分层放置；箱盖内的交接标签写着"如果他们再次要求我撤回，就交给外部律师"。这套物理顺序说明 Harrison 在第三次提交改判后开始整理自首材料，箱内各份正式文书仍分别承担自己的证明责任。
- 🟣 给 Watts 的便签："你替我守过太多次门。这一次，不要替我挡住后果。"

🎮 地图叠加：

玩家把 U3 结尾取得的"南区援助计划内部地图"与本次开发计划边界、1912 事故家庭索引叠在一起。

- 银行制造债务最密集的街区，与 Tidewater 计划优先接收的地块重合。
- 被标为"优先接触"的家庭，大多沿用 1912 事故赔偿编号；当前住户是当年事故家庭及其后代。
- U1 录音里的"1912 / 南区地块"、U2 的银行债务、U3 的房产流向，在这一刻第一次成为同一条地理路线。

- 🧪 两次清退叠加结论：1912 年事故把这些家庭赶离码头，1928 年计划正在沿着他们最后的居住区再次推进；世博会窗口让第二次清退进入倒计时。

这里让玩家自己完成第一层判断：Harrison 有罪，但他的死亡发生在他第一次准备让自己的罪成为证据之后。

## L1 结尾与 L2 钥匙

Rosa 的名字出现在儿童死亡对照表最后一页，Isabel 是最新一例。法院通知显示，圣心医院将在次日由慈善项目负责人 Whitfield 出庭，把死亡责任正式归给 Rosa。

Mickey 看完材料后只说：

> "先救还活着的人。账户不会在明天下午被判有罪，Rosa 会。"

这句话合理，也确实正确。它把玩家带入 L2，同时第一次建立 Mickey 的行动方式：他从不需要提供假方向，只需要替 Zack 排定真相的优先级。

## 玩家体验

💡 保险柜不是赃物库，是一个来不及交出去的自首包。

玩家离开 L1 时应该同时相信三件事：

- Harrison 长期腐败。
- Harrison 死前真的准备回头。
- 杀他的人很可能不是在惩罚一个坏人，而是在处理一个失去控制的工具。

# Loop 2：被挑过的胜利（承）

## 本轮功能

核心问题：Isabel 的死是 Rosa 擅自加量，还是圣心医院慈善项目制度性使用问题药剂？
指证对象：Whitfield。
指证轮次：3。
本轮边界：Mickey 提供的材料必须真实有效。玩家可以感觉他总能决定先查什么，但此时不应认为他伪造证据或就是 Whale。

## 开篇剧情

**Zack 侦探事务所 / 次日清晨**

Rosa 抱着 Isabel 的照片和慈善项目药盒来找 Zack。Whitfield 当天下午将在法院作证，医院提出的条件是：Rosa 承认自己在家擅自增加剂量，医院可以不追究她"误用慈善药品"的责任。

Rosa 最害怕的不是坐牢，而是 Isabel 的死亡记录从此写成"被母亲害死"。

Foster 来电说，Harrison 留下的一支同批次样本已经完成初检。Mickey 随后带来一份医院采购记录，明确说材料来自医院理事会中的旧客户关系。

Emma 追问 `1919-A` 匿名中间账户。Mickey 回答：

> "账户可以继续藏一天。Whitfield 下午说完那句话，Isabel 就会被写死一辈子。"

玩家应当认可这个优先级，因为 Rosa 确实等不了。

## 自由探索

### Rosa / Zack 事务所

- 🟢 医院配发红线注射器与十三日封签药盒：玻璃注射器带医院配发标记，红线处装有不可移动的金属止动环，单次抽取不能越过规定剂量。药盒外沿保留与盒体一体的十三日指纹确认条，不作为独立纸面证据进入背包；十三格均按日开启并留有 Rosa 的指纹确认。
- 🟢 Isabel 使用后的封签药瓶：原始物件可进入分析区。玩家用医院标定的初始装量测量当前剩余液量，得到 🧪 `经容量分析的回收药瓶`：十三次红线剂量与剩余量完全吻合，没有额外一次注射所需的药量缺口；瓶塞穿刺痕只作辅助，不承担精确计数。
- ⚪ Rosa：她不识单位，只会把药液抽到护士画出的红线，从未拿到第二只药瓶。

这组材料不是证明 Rosa"不懂所以不会犯错"，而是用实际消耗量证明她没有在规定剂量之外额外抽取药液。

### 圣心医院 / Miller 事故基金病房

这一区域只承担主线情感铺垫，不进入 Whitfield 指证。

- 🟣 Miller 事故基金病房：墙上挂着 Miller 家族事故基金铭牌，印章与 Emma 的奖学金证明、Foster 的早期研究资助和 Margaret 医疗减免文件相同。
- 🟣 正常治疗记录：病房不是空壳，许多工伤家庭与儿童确实在这里获得过免费治疗；护士能说出几个康复出院的孩子。
- 此处只让"普通基金病房确实救过人"成立，不提前展示问题批次如何进入项目、又发给了哪些家庭；这层答案留给后续化验与采购记录连接。

Emma 看着铭牌，想起自己大学第二年差点因欠费退学。她已经把课本装进箱子，Miller 奖学金的缴费回执却在最后一天送到学校。

> Emma："不是一个写在履历里的名字。"
> "那笔钱真的让我留了下来。"

Foster 看了一眼铭牌，只说：

> "我的第一套法医仪器，也是这个基金付的。"

她没有评价 Miller，把目光移回病房里的儿童姓名牌。玩家此时仍不能证明 Miller 与医院犯罪有关，只会感到一份真实恩情正在靠近一件真实伤害。

### 法院实验室 / Foster

- 🟢 经化验的问题批次药剂：Foster 把 Harrison 留下的原始样本直接交还为分析后的物件。药剂有效成分浓度波动异常，并含不应出现在成品中的降解杂质；连续使用可能造成严重低血糖反应与器官损伤，不再额外生成一张世界内化验报告纸。
- 🟢 同批次药瓶封签组：Foster 已依法从医院留存柜调取另外两名死亡儿童的空药瓶。玩家把两只空瓶的瓶颈封签与 Harrison 样本、Rosa 药瓶封签组合，确认四者属于同一采购批次。
- ⚪ Foster：这不是"保存不好"能造成的单支异常，同批封签在进入家庭前就有问题。

Foster 把四只药瓶并排放好，划掉证物标签模板上的"无法排除家庭保存不当"，改写为：

> "异常在封签批次进入家庭前已经存在。"

她没有替医院判断主观故意，只把自己能够确认的责任边界写得不再含糊。这是她在 U5 最终说出 Sean 死因之前的一次小幅前进。

### 圣心医院 / 预审走廊 / Whitfield

Whitfield 是慈善医疗项目的行政与医疗负责人。他不是幕后最高层，但负责签署项目说明、对外作证和把责任落到家庭身上。

在正式指证前，他只主动给出第一层防线：

- 🟡 Whitfield：Rosa 没按说明使用，在每日规定剂量之外又擅自增加了药量。
- 🟣 Whitfield 随身携带统一格式的"家属误用说明"，上面已经预留 Rosa 的签字位置。

自由探索阶段只取得 R1 谎言。R2、R3 必须在前一轮防线被击穿后，由 Whitfield 当场退守形成，不能提前进入证词池。

### Mickey / 法院会客室

- 🟢 圣心医院采购与发放记录：同一批号列在慈善项目正式采购单、医院入库号与发放清单上，发放对象集中于底层移民儿童项目；它不是护士私领或家庭之间转手取得。
- 🔵🟣 缺失的第十九页：卷内目录把第 19 页列为"理事会特别采购批准"，玩家拿到的副本从第 18 页直接跳到第 20 页。Mickey 明确说执行层副本不含签字原件，原件被单独封存。

这份材料足以证明医院制度性采购、入库并发放了问题批次，却不能证明第十九页由谁签字，更不能证明签字人在批准时已经知道药物有害。上层批准、资金来源与 `1919-A` 的实际作用继续留在缺页之后。

## 指证 Whitfield

| 轮次 | 谎言 | 击破材料 | 玩家可得结论 |
|:---:|:---|:---|:---|
| R1 | 🟡 Rosa 在每日规定剂量之外又擅自增加药量 | 🟢 医院配发红线注射器与十三日封签药盒 + 🧪 经容量分析的回收药瓶 | 药瓶实际消耗量与十三次规定剂量完全对应，没有额外注射所需的药量缺口 |
| R2 | 🟡 即使 Rosa 没有加量，也只是 Isabel 一例药物事故 | 🟢 五年七例儿童死亡对照表 + 🟢 同批次药瓶封签组 | 相同项目、相近症状和重复批次构成长期模式，不是孤例 |
| R3 | 🟡 这批药从未进入医院正式项目，只是个别护士私下领取、个别家庭自行误用 | 🟢 经化验的问题批次药剂 + 🟢 采购与发放记录 | 该批次经医院项目正式采购、入库与发放，责任不能推给个别护士或家庭；本轮不证明审批者事前知情 |

### 指证中的退守逻辑

R1 被击穿后，Whitfield 无法再坚持 Rosa 擅自增加剂量。他先承认最小范围的事实，再把问题缩成单个家庭：

> Whitfield："好，就算 Rosa 没有加量，也只能证明这一个家庭没有误用。"
> "Isabel 仍然只是一次个别药物事故。"

- 🟡 R2 新谎言：Isabel 只是个别药物事故。

R2 被击穿后，Whitfield 不再否认类似死亡，却试图把问题缩成脱离医院制度的零散违规：

> Whitfield："相似病例也不能证明这是医院项目。"
> "可能是护士从别处拿来的，也可能是这些家庭彼此交换过药。"

- 🟡 R3 新谎言：问题批次从未进入医院正式采购与发放程序。

## 指证后

Whitfield 被迫承认问题批次属于医院正式慈善项目，并进入了医院的采购、入库与发放链。他仍坚持自己只是按名单执行；是谁批准这批采购、批准时掌握了什么，继续被缺失的第十九页扣住。

Rosa 当场拒绝在"家属误用说明"上签字。Isabel 的死亡原因进入重审，圣心医院执行层遭到调查。

这场胜利必须是真的：Rosa 获救，Isabel 的名字不再只属于一张统计表，医院也确实失去继续甩锅的空间。

散庭后，书记员把"暂停追究 Rosa、重审 Isabel 死因"的回执交给她。站在人群后面的 Mickey 这才短暂闭上眼，松开一直扣在公文包锁扣上的手。

Rosa 抱着回执和 Isabel 的照片向他道谢，照片从手边滑下。Mickey 先一步托住相框，等她重新抱稳才松手。他没有去看被带走的 Whitfield，也没有把话题转回 `1919-A`。

> Mickey："把回执收好。"

这口气松下来是真的，他希望 Rosa 获救也是真的。后来改变的是玩家对"这场胜利为什么被允许停在 Whitfield 这一层"的理解，不是这份情感本身。

与此同时，玩家会留下三个问题：

- 为什么问题批次只进入底层儿童项目？
- 第十九页上的上层批准人是谁？
- Mickey 为什么总能拿到恰好足够赢、却不能继续往上的材料？

## L2 到 L3：一次足够的铺垫

庭审结束后，法院走廊。

Morrison 靠在墙边抽烟。他看见 Whitfield 被警员带去补充笔录，又看见 Pierce 把医院原始采购箱直接送往封存车。

他叫住 Zack：

> Morrison："Brennan。"
> Zack："怎么，又要说内部牵扯太多？"

Morrison 看了一眼 Mickey，又看了一眼 Pierce。

> Morrison："……算了。"
> Morrison："别太相信法院里的胜利。"
> Zack："你到底想说什么？"
> Morrison："等我想好了再说。"

他离开时，法院书记员追上来，递给他一只密封通知袋。玩家看不见内容。

真相层：通知要求 Morrison 次日上午交出旧南区警务档案、配枪登记和全部私人办案记录，由 Pierce 接管。Morrison 刚看见系统把 Whitfield作为止损对象交出去，立刻认出自己也进入了同一程序。

这一层不在 L2 解释，L3 从 Morrison 桌上的拆封通知中回收。

## 玩家体验

💡 Mickey 没有骗玩家。他只是替玩家决定了今天应该赢哪一场。

L2 的满足感必须成立，阴影才会成立。玩家不是后悔救 Rosa，而是在后来意识到：一场善意且真实的胜利，也可以被更大的系统当作损失控制。

# Loop 3：最后一页（承转）

## 本轮功能

核心问题：Morrison 是畏罪自杀，还是在准备开口时被人改写成自杀？
指证对象：Mrs. Morrison / Doris。
指证轮次：3。
本轮边界：玩家可以确认 Morrison 被杀、Mickey 在正式枪位证词上撒谎；不能确认 Mickey 就是 Whale，也不能仅凭本轮材料证明他亲手杀人。

本轮的认知翻转不是"Morrison 原来是好人"。他仍是黑警，也确实做过许多无法洗掉的事。翻转在于：

> 一个坏人准备说出真相时，系统仍然没有权把他的最后一页写成疯子、自杀者和炸弹犯。

### 为什么 Mickey 必须亲自动手

本轮信息只给到一半，L5 完整回收：

- 🟢 法院证物转运调度单：Pierce 和可用的黑警当晚都被锁在 Harrison / 圣心医院材料转运上。
- 🔵🟣 晚间报纸号外：银月会常用的西区执行人员当晚正被用于冲击一个工会互助站，数人被捕，街面力量无法临时回调。
- L5 的 Mickey 操作记录会写明："法院封存夜；西区人手失联；无中间人。"

这三层一起说明：Morrison 突然决定当夜开口，留给 Whale 的窗口不足一小时。Mickey 不是为了戏剧效果亲自出场，而是他已经没有一个既能灭口、又知道如何伪造警务现场的中间人可用。

他也不能只等 23:50 的清场装置。Morrison 已经穿戴好准备搭街车离开，备份也已经放进外部寄存柜；固定清场只能毁掉仍留在宅邸的人和材料，拦不住一个即将带着线索移动的目标。22:43 的来电让 Mickey 必须先把 Morrison 留在屋里，再让既有清场程序接手。

但"亲自灭口"和"安装清场装置"属于两条命令链。前者是 Mickey 在突发窗口中的选择；后者是 Miller 上层经 Pierce 在 17:40 启动的独立风险控制。两条命令在同一所房子里重叠，才让系统第一次把自己的执行接口也放进可接受损耗。

Doris 的声音断断续续：

## 自由探索：爆炸前

### Morrison 书房

- 🟢 伪造遗书：把 Thomas 的死亡写成"畏罪自杀"，并预告他准备炸毁住所报复社会。字迹近似，但用词刻意粗暴，像在替后续爆炸预先解释。
- 🟢 磨号手枪与枪内未击发余弹：现场枪不是 Morrison 登记配枪，枪号已经磨损。玩家直接分析枪内未击发余弹，其底火、压口与 James 案米勒体系特制手枪的余弹属于同一供枪渠道，但不能据此认定是同一把枪。
- 🟢 古巴雪茄烟蒂：压在沙发后地毯缝里，右侧有不规则咬痕。本轮只记录"不是 Thomas 常抽的牌子"，牙痕到 L4 才激活。
- 🟣 左手生活痕迹：墨水瓶、警徽盒、酒杯垫都在书桌左侧。它只做环境铺垫，Thomas 的惯用手由 Doris 生活证词确认。
- 🟢 Pierce 档案移交通知：要求 Thomas 次日上午交出旧案文件和配枪登记，签发时间在 Whitfield 被带走补充笔录之前。

### 门厅与电话桌

- 🟢 外出准备：大衣、帽子和手杖已经放在门边，口袋里只有街车零钱；车钥匙仍在书房抽屉。Thomas 准备低调离开，不使用容易被 Whale 眼线认出的私人汽车。

### 煤气检修痕迹

- 🟣 松动煤气阀、被布条压住的窗缝：Emma 先闻到煤气味，再看见时钟秒针经过异常铜线接点。

## 突发事件：爆炸前的最后一眼

取得本场必需材料后自动触发。Pierce 赶到并要求停止搜查，Zack 指向尸体右手："那就先记这一处。枪在右手，手指根本没有握住。"

当下玩家感受到的是：有人要把现场和调查者一起清掉。L5 才意识到更冷的一层：清场程序也没有把 Mickey 排除在"可接受损耗"之外。

## 指证 Doris

Doris 在爆炸后的临时问询室里迅速退回官方口径。她不是想替 Thomas 洗白，也不是参与灭口。她害怕两种人同时进入她的家：

- 警局会把 Thomas 写成疯子，用爆炸掩掉谋杀。
- Zack 会把 Thomas 写成最后一刻回头的英雄，让所有人假装一次开口足以洗掉二十年的黑警生活。

问询室的工作人员送来三杯咖啡。Doris 下意识把第四只空杯拉到 Thomas 原本会坐的位置，放进两块方糖，停了几秒又把糖取出来。她没有哭，也没有说自己原谅了他。

她想保护的是这个家的叙述权，也是自己在这段婚姻里留下的生活。这个动机可以理解，但她为了保住这份叙述权，主动把客厅第二只酒杯说成自己的，也差点烧掉 Thomas 留下的交接。

## 指证后：Doris 的选择

Doris 没有立刻允许 Zack 带走钥匙。她先从手袋里取出烧到一半的信，放到空咖啡杯旁；随后才承认缝纫盒里的钥匙由 Thomas 当晚交给她，并正式允许 Zack 取走这把已经登记过的原物钥匙。

她承认 Thomas 晚饭后把钥匙和信封交给她，说如果自己没回来，就烧掉信、留下钥匙，等她判断该相信谁。她烧信不是为了替警局灭证，而是不愿让 Thomas 用一封遗言把自己写成最后一刻的英雄；她留下半封，是因为也不愿让 Pierce替他写成疯子。

她问 Zack：

> "你会把他写成什么？"

Zack 不替 Thomas赎罪，只回答：

> "写他做过什么。好的坏的，都写。"

这句话也提前照见 Zack 对 Emma 的问题：他说别人有权得到完整事实，但他自己还没有做到。

Doris 随后下意识纠正一名警员对现场的描述：

> "Thomas 用左手。写字、倒酒、擦警徽，一直都是左手。"

- ⚪ Doris 关于 Thomas 左撇子的生活证词。

## 警局翻供

爆炸后，Pierce 试图以"原始现场已毁、无法排除自杀"暂定死因。

Zack 没有只靠左撇子下结论。他把几层事实依次放到桌上：

- Thomas 已经穿戴好，准备去找 Zack。
- 他先后联系 Brennan 与 Donnelly，并接待了访客。
- 遗书提前替尚未发生的爆炸准备了口径。
- 枪在惯用左手相反的一侧，握姿僵硬。

Zack：

> "任何一件都可以解释。"
> "但这些解释全都只朝一个方向方便：让一个准备出门的人，看起来像准备自杀。"

Pierce 没有正面回答，而是让记录员询问 Mickey：

> "Donnelly 先生，你在爆炸前也进入过书房。你第一次看见尸体时，枪在哪只手？"

Mickey 看见 Zack 摊开的调查笔记。他知道 Zack 和玩家都看见了右手错误，也知道爆炸让官方第一现场无法复核。

他停顿的那一秒不是被问倒，而是在决定亲手把 Zack 推开。

> Mickey："左手。Doris 也确认 Thomas 是左撇子。"

Zack 抬头。

Zack 把调查笔记推过去：

> "你进门时看见的是右手。"

Mickey：

> "那是你写在私人笔记里的说法。"
> "我只能陈述我看见的。"

Pierce 顺势把 Zack 的私人记录、爆炸后的原始现场损毁和 Mickey 的第三方正式笔录放到一起：两份相互冲突的个人陈述，不足以立刻推翻初步死因。这个说法不能证明 Morrison 自杀，却足以制造程序争议、拖延改判。

走廊里，Zack追上 Mickey：

> Zack："你看见的是右手。"
> Mickey："Thomas 留下的东西已经在你手里。"
> Zack："你刚才帮他们把他写成自杀。"
> Mickey："我帮你保住了还能继续查的东西。你现在逼 Pierce 承认谋杀，他会把钥匙、笔记和所有旧案一起收回去。"

这句话最难受的地方是，它并不全是假话。Mickey 总能用一个真实风险包住自己真正想压下的事实。

Mickey 离开后，Emma 低声问：

> Emma："他看见了右手。"
> Zack："我知道。"
> Emma："那他为什么说左手？"

Zack 看了一眼手里的寄存柜钥匙：

> "也许是在替我们拖时间。也许是在替别的人。"

Emma：

> "寄存柜，要叫他吗？"

Zack：

> "不。Morrison 留下的东西，我们先看。"

## L3 结尾：只打开门，不读取答案

# Loop 4：她藏起来的不是一封信（转）

## 本轮功能

核心问题：Margaret 为什么明知 O'Hara 拒绝出售、下一次复访会有警察到场，仍劝她接受报价、搬走，并要求她不要找 Zack？
指证对象：Margaret。
指证轮次：3。
本轮边界：玩家可以高度怀疑 Mickey 就是 Morrison 的访客、Donnelly & Associates 与 `1919-A` 账户有关；不能仅凭童年记忆永久锁定 Whale 身份。

本轮要让过去通过一件正在发生的事重新进入现实：O'Hara 不是为了替 Patrick 和 Mickey 提供背景而出现，她此刻正因拒绝出售房屋而受到威胁。Margaret 的旧创伤因此变成一个当下选择。

## 开篇：读取寄存柜

**车站寄存区 / 清晨**

L4 直接承接 L3 最后一个画面。Zack 和 Emma 把两只封套放到候车长椅上，不再重复开柜过程。

### 封套一：Morrison 写给 Zack 的未寄出口供

Thomas 没有请求原谅，也没有把自己写成英雄。他只写清三件事：

- 他替 Pierce 和 Whale 压过旧案，知道自己没有资格要求 Zack相信。
- Whitfield 被交出去后，他收到同样的"交出文件、接受接管"通知，才知道自己已经进入下一批清理。
- 他不想让警局把自己的最后选择写成自杀、疯狂或报复社会。

最后一句：

> "别替我洗干净。也别让他们替我写完。"

### 封套二：Lakeshore 清退协警通知

- 🟢 Lakeshore 清退协警通知：O'Hara 家被列为第一批核心拒售户；Lakeshore 将在次日上午再次上门，要求辖区警员到场"维持民事秩序"。Thomas 之所以拿到它，是因为这本来就是发给警局的协查副本。
- 文件备注："住户已两次明确拒售，第三次接触按重点户处理。"

Thomas 不是因为认识 O'Hara 才选中这张纸。他把 Pierce 要求移交的当前协警文件一并备份，而这张排在最前面，时间就在第二天早晨。

行动目标成立：先赶在辖区警员到场、Lakeshore 的复访升级为强制清退之前找到 O'Hara，确认她为什么拒绝出售，也确认是谁一直劝她不要找 Zack。

## O'Hara 家：当下危机

O'Hara 家不是已经搬空的空屋。她仍住在这里，也明确拒绝出售，是第一批清退名单里最难处理的一户。

O'Hara 是 Brennan 家多年的旧邻居，Zack 小时候习惯叫她 Aunt O'Hara，但两人没有血缘关系。

门口没有打包好的行李，Lakeshore 送来的纸箱仍折叠着靠在墙边。O'Hara 不是在等待更高报价，她根本没有答应出售。

她看见 Zack 时先叫了一声"Zackie"，随后因为门外还有 Lakeshore 的复访人员而改口叫"Brennan 先生"。这不是卖亲情，而是让玩家知道：她本来就认识并信任 Zack，却被 Margaret 劝住了。

### 自由探索 / O'Hara

- 🟢 Lakeshore 最终收购协议：报价明显偏低，限四十八小时签署；O'Hara 在签名页写下"不出售"，整份协议仍压在桌上。
- ⚪ O'Hara：她明确告诉过 Margaret 自己不会卖，也把 Lakeshore 威胁下次带警察上门的事说了。Margaret 仍劝她接受报价、先搬走。
- 🟢 Margaret 写给 O'Hara 的短便条："把报价签了，先搬来我这里。别找 Zack。他一碰这件事，就不会只替你搬一次家。"便条有 Margaret 签名。
- 🟣 折叠未用的搬家纸箱：Lakeshore 提前送来，但 O'Hara 没有打开。她不是为了抬价拖延，而是拒绝出售。
- 🟣 后巷井检口：厨房手压泵连接私人井，但因早年地块分割，带锁的铁制检修盖留在后巷、院墙之外。此时只作为住宅结构信息，不产生投毒结论。

O'Hara 的立场要清楚：

> "我不是在等他们多加几个钱。这房子不卖。他们今天带警察来，答案也一样。"

Zack 没有在确认威胁后直接离开。他借 O'Hara 写下的"不出售"、协警通知与限时协议，通过 Watts 向值班法官申请二十四小时临时停止执行；书记处负责登记并送达：Lakeshore 可以继续提出报价，但复核完成前，警员不得协助进入房屋、搬动财物或强制带离住户。

Emma 提议先把八岁的 Sarah 送到 Rosa Martinez 家住一晚。Rosa 与 O'Hara 同属南区家长互助会，Sarah 以前在她家留宿过；这个去处不是临时冒出的陌生安全屋。O'Hara 同意女儿暂时离开，却拒绝自己跟着走。

> O'Hara："你们可以替我把门守住。"
> "不能替我决定这还是不是我的家。"

- 🟢 临时停止执行回执：有效至次日傍晚，只能拦住警务协助和强制搬离，不能阻止具有市政外观的公共卫生检查。
- 🟣 Sarah 的小旅行箱：只装了一套衣服和课本。她临走前仍从窗边看见两名"水质维护人员"在街口逐户核对门牌，为终幕保留生活视角证词。

因此 Zack 并非把一个已知处于危险中的人留在原地不管。临时停止执行真实挡住了警务协助、强制入屋和搬离；后来的下手者只能伪装成公共卫生人员，从后巷的院外井检口动手。他们没有进入房屋或院子，法律保护真实生效，却被另一条执行路径绕过。

## Margaret 家：主动整理过去

Margaret 的家里也在打包，但她没有被 Lakeshore 清退。她原本就考虑搬去北边教会附近的小公寓；听说 O'Hara 仍准备找 Zack 后，她突然提前打包 Patrick 与 Zack 的旧物。

因此搬家不是巧合，也不是行政迫迁。它是 Margaret 在风险逼近时作出的防御动作：把东西封箱、把 O'Hara 送走、在 Zack 到来前重新把旧故事收回去。

门厅里有 O'Hara 刚用过的茶杯。Margaret 看见 Zack 手里的便条，第一句话不是问 O'Hara 是否安全，而是：

> "她还是去找你了？"

### 自由探索 / Margaret 家

Zack 是来问 Margaret 为什么逼 O'Hara 接受报价、又为什么阻止她求助。谈话发生在半开的搬家箱之间，Patrick 的旧物已经被 Margaret 主动翻出、准备封存，因此所有用于指证的材料都能在指证前出现。

- 🟣 搬家标签：最初日期是下个月，后来被改成三天后；改期时间正是 O'Hara 再次拒绝 Lakeshore 的第二天。
- 🟢 1903 南区码头旧照片：Patrick、Liam、少年 Mickey 与工人们站在货栈前。
- 🟢 Patrick 葬礼册与 1919 旧信封：信封抬头为 Donnelly & Associates，日期在 Patrick 去世后不久。
- 🟢 1919 年清场补充记录：这份未并入公开死亡摘要的家属副本中，受伤转运员记录 Patrick 已带着封存箱抵达内侧防火闸门；有人喊"西侧还有一个人"后，Patrick 在身份尚未确认时放下封存箱折返。Mickey 的陈述只补全 Patrick 最终把自己送出侧门。诊疗附页确认 Patrick 此后始终没有恢复可交流意识。
- 🟢 Patrick 遗物匣：回收 U2 已出现的橡木小匣。匣盖刻有 `P.B.`，正面嵌着 Patrick 的工人运动领袖徽章，侧面刻有完整信条 `FOR THE MANY WE PAY THE FEW`。这行刻字在 U2 只作为不可收集的环境信息出现，没有触发 Zack 解释；L4 中 Margaret 主动把遗物匣交给 Zack，刻句才正式进入当前推理链。
  - 观察一 / 刻句：回收 Patrick 的信条并提供 L5 七位密码线索，但不能证明 Mickey = Whale。
  - 观察二 / 未使用返程票：支持 Patrick 当晚原本有回家安排。
  - 观察三 / 焦黑编号金属扣：消防人员从闸门内侧残骸中找回，编号对应装有账簿的封存箱；它与清场补充记录共同证明封存箱到达出口内侧后没有越过闸门。
  - 原消防证物标签留在匣内，只说明返程票与金属扣的保管来源，不作为独立证据卡。
- 🔒 Margaret 写给 Mickey 但未寄出的短笺：折在补充记录最后一页，指证前只能看见抬头。R3 击穿后展开，正文是："葬礼那天你说你会接着做。我已经拦不住一个。那晚的完整经过，不要再对 Zack 提。"

注意：玩家在指证前可以查看 1919 年记录，以及遗物匣内的返程票和编号金属扣，因此能够证明 Margaret 隐瞒了"Patrick 已到出口后主动折返"的关键事实。遗物匣的刻句只提供思想与密码线索，不承担折返证明；Margaret 短笺和折起的 1903 工会备忘正文均在 R3 击穿后展开，分别补全她的隐瞒动机，以及 3 人死亡与 800 名工人的完整因果。

## 记忆锚点玩法

本轮记忆必须明确视角来源，不能让一张照片凭空提供 Zack 不可能知道的事实。

### 锚点一：1903 旧照片 → Zack 的童年碎片

Zack 当年年纪很小。他不记得码头清场本身，只记得清场后的 Brennan 家：少年 Mickey坐在门口，嘴角全是血，右下牙缺了一角，一直问 Liam 什么时候回来。

- 🧪 记忆观察：Mickey 右下牙有长期缺损。
- 玩家把观察点与 🟢 Morrison 现场古巴雪茄烟蒂连线。
- 🧪 派生结论：烟蒂右侧不规则咬痕与 Mickey 旧牙伤位置高度吻合。

它只能说明 Morrison 死前访客极可能是 Mickey，不能作为单独定罪材料。

### 锚点二：1919 葬礼册 → Zack 的明确记忆

Patrick 葬礼后，Mickey 对 Zack 说：

> "他没做完的，我会接着做。"

同一时期的 Donnelly & Associates 信封把这句话从单纯情感承诺变成一条可能的组织与法律线索。

Zack 一直知道 Patrick 最后把 Mickey 带了出来，却不知道父亲在此之前已经抵达出口。Mickey 此后也没有讲清自己为何错过撤离窗口，更没有告诉 Zack 那份账簿依靠自己的 Miller 内部权限才得以接触；说清这些会让 Donnelly & Associates 在 1919 年的起点受到追问，也会破坏"Patrick 最后选择了我"的私人解释。Margaret 压住家属材料，Mickey 压住自己在行动中的角色，两种沉默共同维持了 Zack 所知的简化版本。

- 🧪 记忆观察：1919 年同时出现两项可观察事实——Mickey 在葬礼后说"我会接着做"，Donnelly & Associates 抬头开始出现。
- 玩家把它与 🟢 Harrison 资金流向图中的匿名 `1919-A` 中间账户连线。
- 🧪 派生结论：`1919-A` 可能不是普通年份编号，而与 Patrick 去世后开始使用 Donnelly & Associates 抬头的法律网络有关；此时仍只是高概率方向，实际控制人留到 L5 的银行授权附页坐实。

### 锚点三：1919 清场记录 → 对葬礼承诺的第一次重读

清场记录、返程票与烧变形的账簿扣只提供可验证事实，不直接替玩家解释人物内心：

1. 返程票支持 Patrick 当晚原本有回家安排；转运员的时间与位置记录则证明他已经带着封存箱到达内侧出口。
2. Patrick 听见"还有一个人"后立即折返，当时尚未确认被困者就是 Mickey；封存箱因此未能越过闸门，随后被焚毁。
3. Mickey 最终被 Patrick 带出，Patrick 此后没有恢复可交流意识。现有材料中不存在正式或口头使命交接。

这些事实与 Mickey 葬礼上"他选择了我来接着做"的说法之间出现明显距离，但 L4 不替玩家裁定 Patrick 的完整思想，也不直接宣布 Mickey 的解释错误。它只把两种解释同时放进玩家心里，更不以此证明 Mickey 就是 Whale。

## U3 公文包复审：Emma 与 Zack 的小吵架

这场戏不再突然插入。触发原因是 Emma 再次看见遗物匣侧面的 `FOR THE MANY WE PAY THE FEW`，发现这句曾经只是家庭旧物上的刻字，如今与 Mickey 在 U3 劝 Zack 越过程序线时使用的逻辑高度相似。

她和 Zack 回到门厅，从 Zack 公文包里取出 Mary / Helen 旧案，想确认 Mickey 当时到底怎样一步步引导 Zack。

复审时，Emma 看见 U3 结尾才形成的最终时间线：Mary 与 Helen 在 19 号已提前布置天台，Mary 负责不在场与说辞，Helen 负责涂油、破坏护栏与灌酒配合。

她先发现事实：

> "Mary 是蓄意的？"

然后才看到文件日期，确认 Zack 是在报道发出后、U3 结束前得知。这场争吵因此不是"你骗我写了假报道"，而是：Zack 后来知道了完整事实，却让 Emma 继续停在比较容易承受、也比较方便救人的版本里。

> Zack："Thomas 确实准备杀她。"
> Emma："我现在问的不是 Thomas。"
> Emma："所以不是我发稿前你就知道。是我发稿以后，你知道了，然后决定我不需要知道。"
> Zack："那篇报道已经救了她。"
> Emma："你救了她以后，替我决定我救下的到底是谁。"

Emma 不要求 Zack 此刻推翻 Mary 案，也不否认报道救了人。

> Emma："我理解你为什么这么做。"
> Emma："别把理解当成同意。"

两人没有彻底吵崩。他们带着这道尚未解决的裂缝回到 Margaret 面前，使接下来的指证不只是查 Patrick，也是 Zack 被迫看见自己正在重复家族习惯。

## 指证 Margaret

### Margaret 的防御逻辑

Margaret 的核心不是反对正义，也不是希望 O'Hara失去权益。她见过 Patrick 把"不能不管"变成一条永远不会结束的路。葬礼后，Mickey 对 Zack 说"他选择了我，我会接着做"，Zack 的第一反应不是拒绝，而是问"从哪里开始查"。Margaret 当场打断，告诉两人证据已经烧毁，这件事到此为止。她后来私下拿补充记录纠正 Mickey：Patrick 折返时只听见"还有一个人"，并不知道里面是谁；Mickey 却坚持 Patrick 知道只有自己会留在内室，也知道自己能继续。

这是 Margaret 确实尝试过、也确实失败过的一次阻止。她不知道 Mickey 后来每一步是怎样走成的，却亲眼看见救命事实在葬礼后被解释成了使命交接。

她真正害怕的不是 Zack 知道父亲救过 Mickey——Zack 从 1919 年起就知道——而是 Zack 看见父亲已经有回家的安排、已经抵达出口，却仍主动放弃账簿和唯一撤离窗口。她判断这会让 Zack 把"没回来"理解成英雄最后的自我证明，并再次追问父亲未完成的调查；死去的 Patrick 仍会把 Zack 从她身边带走。

她判断 O'Hara 接受报价、尽快离开至少能活下来，也判断 Zack 一旦接手就会一路追到 Lakeshore 和 Whale。她的错误不是想保护两个人，而是相信自己有权分别替 O'Hara 和 Zack 决定他们应当放弃什么、知道多少。

| 轮次 | 谎言 | 击破材料 | 玩家可得结论 |
|:---:|:---|:---|:---|
| R1 | 🟡 O'Hara 已经准备接受报价，我只是在帮她安排搬家 | 🟢 写有"不出售"的最终收购协议 + ⚪ O'Hara"明确告诉 Margaret 不会卖"的证词 | Margaret 知道 O'Hara 从未同意出售，仍在替她决定离开 |
| R2 | 🟡 我没有让她瞒着你，是她自己不想麻烦你 | 🟢 Margaret 写给 O'Hara 的短便条 + ⚪ O'Hara 关于 Margaret 劝阻求助的证词 | Margaret 明确要求 O'Hara 不找 Zack，并主动控制求助路径 |

收束：

> Zack："你不是怕我恨 Mickey。"
> Margaret："我怕你跟着他。"
> "葬礼那天，他说他会接着做。我已经拦不住一个了。"
> "我不能让你也把你父亲没回来，当成他把路留给了你。"

R3 击穿后，Margaret 才展开写给 Mickey 却没有寄出的短笺。它不是新的击破材料，只确认她当年确实准备要求 Mickey 继续沉默，也确认她把"阻止继承"置于 Zack 的知情权之前。

## 指证后：核心记忆才开放

Margaret 不再替 Zack 管理过去。她把 1903 年工会备忘与 1919 年清场记录并排展开，让两次选择第一次同时出现。

### 第一段：1903——Patrick 替别人说"值得"

- 🟢 Patrick 旧工会备忘：3 dead / 800 workers / 14h → 10h。
- ⚪ Margaret 1903 当晚证词：Patrick 说三个人的死亡换来了八百个工人的工时改善，并说 Liam 如果知道结果，也会觉得值得。
- 🔒 Margaret 的记忆：她记得的不是"三个人"，而是 Mickey 坐在 Brennan 家门口，等一个永远不会回来的父亲。

🎮 1903 核心叙述记忆：

这不是 Zack 自己的回忆，而是由 Margaret 叙述、玩家在固定片段中观察的过去。玩家看见三件事同时成立：

1. Patrick 没有故意害死 Liam，且确实为八百个工人争取到十小时工时。
2. Patrick 在胜利后替 Liam 说了"值得"，把一个具体死者放进一张结果正确的账里。
3. 少年 Mickey 坐在 Brennan 家门口，嘴角带血，手里攥着 Liam 的旧帽子，一遍遍问父亲什么时候回来。

### 第二段：1919——已经到达出口的人再次折返

- 🟢 清场补充记录：Patrick 已经带着封存箱到达内侧防火闸门；听见"西侧还有一个人"后，在身份尚未确认时返回。Mickey 的陈述只补全 Patrick 最终把自己送出侧门。
- 🟢 Patrick 遗物匣内物件：未使用的返程电车票支持 Patrick 当晚原本有回家安排；带交接编号的焦黑金属扣证明装有账簿的封存箱未越过闸门，并在火后才从内侧残骸中被找回。
- ⚪ 诊疗附页：Patrick 送医后始终没有恢复可交流意识，于次日凌晨因内伤去世；不存在未被记录的临终交接窗口。

🎮 1919 事实重建：

这不是 Margaret 亲眼看见的记忆，而是由清场记录、遗物与诊疗附页共同锁定的固定片段。玩家看见 Patrick 已经抵达内侧闸门，最后一名转运员正被拖向外侧。有人从走廊喊"西侧还有一个人"，没有报名字。Patrick 看了一眼封存箱，把它留在闸门内侧，转身重新进入混乱。

Zack 看见返程票时，第一次亲自说出这次重读对他的意义：

> "我一直以为，他那晚根本没打算回来。"

玩家只能确认三件事，不在 L4 得到统一价值答案：

1. Patrick 明知交接点可能暴露，仍同意最后一次转运，也接受了 Mickey 对内部权限的过度自信；这场危险并非与他无关。
2. 到达内侧出口后，他听见尚未确认身份的人被困，选择放下封存箱并亲自折返；没有证据显示他在折返前知道里面是 Mickey。
3. 这次选择不能取消 Liam 的死亡，也不能取消 Patrick 又一次没有回到 Margaret 与 Zack 身边的代价。

这些事实既没有证明 Patrick 完成了彻底的思想转变，也没有证明 Mickey 的"父亲选择了我"必然成立。L4 到此停下，不提前概括 Mickey 此后九年的全部选择，也不以 Patrick 的死证明 Mickey 就是 Whale。

💡 留给 L5 的未完成问题：

> Patrick 折返时甚至没有听见名字，Mickey 为什么仍如此确信"他最后选的是我"？

Margaret 只说明自己的感受，不替 Zack 给 Patrick 定案：

> "我只知道，那次他没有叫别人回去。"
> "他自己回去了。"
> "可替他等的人，还是我们。"

Margaret 的成长动作不能只是交材料。她当着 Zack 的面给 O'Hara 打电话：

> "别签。"
> "Zack 已经知道了。接下来让他自己决定要不要管，也让你自己决定要不要留下。"

O'Hara 告诉她停止执行回执已经送到，Sarah 也在去 Rosa 家的路上，但自己会留下。Margaret 沉默片刻，没有再劝她搬走。

她第一次把决定还给两个人，也第一次接受"把决定还回去"并不保证结果安全。

## L4 结尾：四十二层邀请

Margaret 家电话响起。Mickey 直接找 Zack。

他没有承认身份，只说：

> "我听说你翻出了 Patrick 的箱子。"
> "如果你准备把我的名字写到 Whale 旁边，别靠一支雪茄和一段童年记忆。"
> "午夜前来四十二层。把你认为能证明这份怀疑的东西都带上。"

Zack 问他为什么。

> Mickey："因为怀疑一个朋友很容易。"
> "知道以后还愿不愿意承担，比较难。"

他停了一下，又补充：

> "带 Emma 来。"

这句话既像挑衅，也像 Mickey 已经知道 Zack 会替 Emma 决定什么该听。

L4 至此形成进入 L5 的明确理由：Zack 不是无缘无故闯入整层，而是受邀进入会面区域；但这份邀请仍不等于搜查私人保险柜的授权。

## 玩家体验

💡 L4 的核心不是"世界突然变了"，而是旧信息获得了新含义。

- Morrison 的烟蒂不再只是陌生访客痕迹。
- `1919-A` 不再只是一个匿名账户代号。
- Patrick 的死亡不再只是"清场中没来得及撤出"：他曾经到达出口，又在尚未听见名字时折返。
- Mickey 葬礼上的"我会接着做"不再只是悼念，但玩家此时仍不能替 Patrick 判定它究竟是不是交棒。
- Zack 在 U3 的保护也不再只是一次善意隐瞒。

玩家离开 L4 时应当处于：

> 我几乎知道 Whale 是谁。
> 但我仍然要亲手证明，而不是让回忆替我定罪。

# Loop 5：四十二层（合）

## 本轮功能

核心问题：Zack 能否在 Mickey 承认之前，先用玩家亲手取得的材料证明 Mickey = Whale？
指证对象：Mickey。
结构：Whale 身份事实指证（Expose）+ 4 个控辩阶段。
本轮边界：可以坐实 Mickey 的身份以及 Miller → Lakeshore → Tidewater 的控制结构；保险柜中的 Sean 与水源内封套在坠落前保持封闭，Sean 的谋杀疑云与水源风险在非 Loop 终幕才进入 Zack 的认知。不能解开 Sean 的完整执行链、小 Charles 的越权投毒、Lawson 身份与 U5 嫁祸。

## Mickey 为什么主动邀请 Zack

L3 爆炸后，Mickey 从一张内部清场副本中发现：清场命令写的是"现场人员不作区分"。Miller 系统明知他可能被 Pierce 叫回，仍然允许爆炸按时发生。

这没有让 Mickey 立刻悔悟，却摧毁了他的一层自我想象：他一直认为自己是负责计算代价的人，第一次明确看见系统也把他列在"可接受损耗"里。

同时，Morrison 现场的右手错误已经暴露，Zack 又在调查 Patrick 旧物。Mickey 判断自己的 Whale 身份很快会被 Miller 先行清理。

因此四十二层不是一次粗心失窃，也不是 Mickey 主动把全部罪证摆给 Zack。它是 Mickey 有意安排的一次最后会面：

- 他主动邀请 Zack 带着既有证据来质问，想控制最后一次对话发生的地点、时间与见证人。
- 他仍把真正的身份材料与 Miller 保险锁在私人保险柜里，没有准备主动交出；那既是自保底牌，也是他尚未放弃的控制权。
- 他没有想到 Zack 会把 U2 见过、一直只被当作家庭遗物的刻句与自己的私人保险柜连接起来，也没有想到 Margaret 会在 L4 把遗物匣及其内部材料交给 Zack。
- 他还在测试 Zack：看见这些成果和代价后，Zack 会审判他、理解他，还是接替他。

这个动机依然带着 Mickey 的老毛病。他愿意接受质问，却仍想决定真相最多走到哪一层。保险柜真正被打开，是 Zack 第一次从 Mickey 设计的会面里夺回主动权。

## 开篇剧情

**四十二层 / Donnelly & Associates / 午夜前**

Zack 和 Emma 按邀请来到四十二层。外间没有职员，会客区与 Mickey 准备用来谈话的私人办公室都没有锁；桌面和开放书架只有正常律师业务材料，私人保险柜仍然上锁，房间里也没有任何罪证被特意摊开。桌上只放着一张写有一句话的卡片：

> "怀疑不是证明。你父亲教过我们的。"

这是 Mickey 转述 Patrick 曾对孩子们说过的话，也是在要求 Zack 按侦探的方式完成最后一步。

办公室电话可以直通市政厅，但此刻没有正在进行的通话；听筒旁留着一张催 Mickey 回电的便签。电梯指示灯停在楼下，说明 Mickey 不在房间里，但随时可能返回。

Emma 明确指出：

> "他让我们进来，不等于他会让我们带着东西出去。"

L4 的邀请只解释 Zack 与 Emma 为什么能进入会客区和谈话办公室，不等于 Mickey 授权他们搜查私人材料。Mickey 预期两人带着外部证据来对质；打开保险柜，是 Zack 主动越过他预设的会面边界、从他手里夺回证据主动权。

行动目标成立：先用 L1-L4 已有材料核查 Mickey 的法律壳与 Morrison 访客身份；如果他仍把关键答案锁在私人区域，就在他回来前把高度怀疑转化成四条互相补强的证明链。

### 字母锁保险柜

取每个词首字母，得到 `FTMWPTF`。

玩家必须亲手输入 Mickey 用来解释自己一生的句子，才能看见那句话实际保存了什么；这不是接受 Mickey 安排的一场考试，而是用被母亲保存下来的家庭历史撬开他的控制。

## 保险柜内容

保险柜打开后，Zack 与 Emma 始终共同检视外层材料：Zack 负责把操作记录与前三章事实对上，Emma 复核法人、基金印章与项目文件。除仍保持密封的 Miller 内封套外，不存在 Zack 在 L5 单独看过、Emma 不知情的身份材料。

### 1. 🟢 `1919-A` 银行授权附页与 1925 年内部接口接管记录

这份文件把 Mickey 的两段人生明确分开：

- 1919 年的银行授权附页把 `1919-A` 记为 Donnelly & Associates 受托结算子账户，唯一授权签字人为 Michael F. Donnelly；账户用途栏引用的正是 Harrison 存根上的圣心医院赔偿案号。
- 自 1919 年起持续经营的 Donnelly & Associates 是法律壳，负责合同、产权、银行重组和诉讼拖延。
- 1925 年，Mickey 才正式接管内部代号 `W`，职责栏写着"警务协调 / 银月会联络 / 非合同障碍处理"。
- 接管签批来自 Tidewater 上层项目办公室，抄送 Lakeshore 风险委员会；Miller 名字仍不直接出现在任命正文里。

玩家可以由上下文确认 `W` 就是 Whale，但这份记录的更重要作用是守住人物 chronology：Mickey 不是从少年时期起就以 Whale 身份杀人。他先用六年时间经营法律壳、告诉自己在接近系统核心，直到 1925 年才主动接过暴力接口，并逐渐成为系统的一部分。

再把操作簿与法人卷宗上的 Mickey 已知亲笔批注、Mickey 定制钢笔连接：

- 🧪 书写者一致性：操作簿与已正式归属于 Mickey 的批注笔迹一致，关键转折处也有同一缺口笔尖造成的稳定断墨；操作簿由 Mickey 长期书写。这个结论只确认作者，不替玩家证明每条记录都真实或每道命令都已执行。

### 3. 🟢 Mickey 功业簿

这不是给法庭看的账，而是 Mickey 私下用来证明自己没有白做的记录。它保留真实成果，也保留被他压成代号的人：

- O'Brien：清理；罢工网络保留。
- Webb：同时持有 Lawson 私人材料与 Whale 勒索记录；清理；银月会接口稳定。
- Moore：允许银行执行层让步；Frank 线可暴露；Lakeshore 不进入法庭记录。
- Harrison：清理；圣心医院执行层可控暴露。
- Morrison：亲自处理；身份暂保；右手错误。
- Rosa：免于定罪；医院项目终止。
- Mary：脱离死刑风险；Brennan 完成越线测试。

Liam 的旧工牌夹在封面内侧。Mickey 不是忘记了父亲，而是每次翻开功业簿都让父亲的遗物替后面的账背书。

玩家应该在这里产生复杂感受：Rosa 和 Mary 的结果没有造假，Mickey 确实帮过人；同一本簿子也把 Morrison、Harrison 和 O'Brien 写成了清理成本。

玩家也终于能回看前三章：

- U1 的 Webb 不是因为勒索几个酒吧客人而死，而是同时碰到了 Lawson 与 Whale 两条接口。
- U2 Mickey 能逼 Moore 让步，不只是律师口才，而是他有权决定牺牲银行执行层、保护 Lakeshore。
- Webb、Harrison、Morrison 不是三起互不相关的死亡：一个掌握了跨接口材料，一个拒绝继续充当司法工具，一个准备带走旧警务文件。他们都在从"可利用"变成"不可控制"时被清理。

她没有把这份恩情说成骗局。电梯指示灯已经开始上升，两人先把未拆内封套与外卷一起收入档案袋。真正内容要等脱离四十二层监视后才读取。

## 身份摊牌

电梯到达四十二层。Mickey 走进办公室，看见保险柜已经打开，脚步第一次停在门口。他先看见桌上的 Patrick 遗物匣，随即明白 Zack 已经把那句家庭旧话与自己的私人密码连在一起。他显然预料到 Zack 会质问，却没有预料到两人能打开这只私人保险柜；惊讶只持续了一瞬，他没有再问他们是怎么做到的。

Zack 依次摆出法人卷宗、复原后的操作记录和牙痕比对。

> Zack："`1919-A` 的实际控制人是你。"
> Zack："Harrison 第三次拒绝撤回后，你给我留下医院线。"
> Zack："Morrison 电话没接、现场摆错右手、警局改成左手，这些都在你的记录里。"

最后，他把两支雪茄放到一起。

Mickey 不再否认：

> "是。我是 Whale。"

这句承认不是反转，而是玩家完成证明后的情感落锤。

Mickey 随后确认职责页的含义，但不提供 U5 才能落地的分赃数字：

> "Moore 让人欠债，Lakeshore 给债务和估价披一层投资外衣。"
> "Lawson 的团队接走房子。文件能解决的交给律师，文件解决不了的交给 Pierce 和银月会。"
> "我负责让这两种方法之间没有缝。"
> "Miller 不需要在每一页签字。他们只决定期限、方向，以及可以损失多少人。"

至此玩家才能准确理解：Mickey 不是受黑帮胁迫的外部律师，也不是银月会老大；Whale 是 Tidewater 体系内部连接法律、警察与地面暴力的接口。

## Mickey 控辩：4 个阶段

玩法标注：本轮存在正式 Expose。玩家先提交四条证据链，完成"Mickey = Whale"的事实指证；Mickey 承认身份后，才进入以下四个控辩阶段。后半讨论的是权力与责任，不把价值判断伪装成可被物证击破的事实谎言。

### 阶段一：身份不是全部

Mickey 的论点：

> "证明我是 Whale，只证明我做过这些事。"
> "它不证明这些事没有救人。"

他主动列举 Rosa、Mary、圣心医院项目终止和工人法律援助。Zack 不能说这些成果是假的，因为那会削弱整个主题。

> Mickey："病房是真的。你的大学也是真的。"
> Emma："所以你们才总觉得，给过一些东西，就有权拿走另一些。"

玩家查看功业簿，第一次看到"救下的人"和"被清理的人"处于同一种表格格式。

Zack 只问：

> "Rosa 知道她获救的代价是谁吗？"
> "Harrison 和 Morrison 有机会说自己愿不愿意付吗？"

本阶段不争论结果大小，只把问题从"结果有没有发生"转成"谁获得了替别人签字的权力"。

### 阶段二：你也做过同样的事

Mickey 的论点：

> "Mary 等不起法院。"
> "你签了不该签的文件，知道真相后也没有告诉 Emma。"
> "你不是反对我的方法。你只是反对这次由我来算。"

这一击必须成立。Zack 不以"我是好人"退守，也不能靠 Emma 替他辩护。

Zack：

> "我做过。"
> "而且我差点把救到人，当成我有权继续隐瞒的证明。"

Emma 没有原谅式地打断，只重复 L4 的边界：

> "理解不是同意。救到人，也不是下一次替别人决定的许可证。"

Zack 与 Mickey 的区别不被写成天生善恶，而是 Zack 仍能承认行为需要被别人质问。

### 阶段三：系统需要我

Mickey 的论点：

> "Miller 还在。Lakeshore 和 Tidewater 还在。"
> "我停下以后，谁留在里面把档案带出来？谁替下一个 Rosa 留一条路？"

Zack 出示自动清场副本。

> Zack："他们知道你可能回到 Morrison 家。"
> Zack："爆炸没有为你停。"

Mickey 第一次失去现成回答。

> Zack："你一直以为自己是写名单的人。"
> "在 Miller 那里，你也只是名单上的一格。"

这是 L3 爆炸的真正回收。Mickey 的悲剧不是善意全假，而是他为系统训练出的算法最终也毫不犹豫地计算了他。

### 阶段四：Liam 与 Patrick

Mickey 把 Liam 工牌从功业簿中取出来：

> "三个人，八百个人。"
> "一个病房，一整个街区。"
> "Patrick 当年也算过。只是他赢了，所以你们叫他 Patrick Brennan。"

Zack 不否认工时改善：

> "我父亲错的地方，不是他想救八百个人。"
> "是他替 Liam 说了'值得'。"

Mickey 随即把争论从理念推向两个人最私密的旧伤：

> "可他最后选的是我，不是你。"
> "1919 年，他已经出去了。他还是回头找我。"
> "他知道谁能把没做完的事接下去。"

这句话必须真正打中 Zack。Zack 不能立刻用完整道理反击；他先说出压了多年的愤怒：

> "他为了救你，又一次没有回家。"
> "可你活下来，就活成了现在这样。"

Mickey 把这句话理解为嫉妒，继续坚持 Patrick 的折返证明自己才是被认可的继承者：

> "你有 Margaret。你可以回家。"
> "他留给我的，只有那件没做完的事。"

Zack 这时才把 1919 年的物理事实重新摆回两人之间：Patrick 在听见被困者姓名以前就已经折返；他没有把账簿交给 Mickey，受伤后也没有恢复可交流意识，更不可能安排 Donnelly & Associates。

> "你一直把被救，当成被选中。"
> "他转身的时候，根本没人告诉他里面是你。"
> "你把一次救命，活成了一场加冕。"
> "他救的是 Mickey，不是 Whale。"

Mickey仍试图守住最后一种解释：Patrick了解他的习惯，知道只有他会为清除登记页留在内室，因此"不知道名字"不等于"不知道是谁"。这个反驳不能被旁白直接判成谎言，但它也不再拥有事实上的使命交接作为支撑。

Mickey 第一次无法用成果数字回答。他追问：

> "如果不是让我接着做，他为什么救我？"
> "如果我什么都不完成，他的死算什么？"

Zack：

> "让你活。"
> "一条命救下另一条命，到这里就够了。"
> "后面那些人，是你自己决定让他们死的。"

Zack 把功业簿翻到 Morrison、Harrison 与 O'Brien：

> "你也不是在延续 Liam 的遗愿。"
> "你是在逼后来每一个死去的人，都替 Liam 的死证明 Patrick 当年算对了。"
> "你替他们签字，再拿结果告诉自己，他们会同意。"

💡 终极尤里卡：

> Mickey 不是不知道自己杀了谁。
> 他无法接受 Liam 只是死了，也无法接受 Patrick 的折返可能没有授予自己任何使命，因此让后来的死亡不断替"我为什么被救"制造答案。

## Miller 的最后条件

市政厅线路再次响起。Mickey 接起后，对面没有问他是否受伤，也没有问 Zack 查到了多少，只通知他：楼下车辆已经准备好。

对方给出的不是工作安排，而是一条可以活着离开四十二层的路：Mickey 取回保险外卷与尚未拆开的内封套，交给楼下司机，再独自上车；Miller 会把今晚解释成一次内部文件交接，并暂时保留他的律师身份。Zack 与 Emma 可以离开，但不能带走任何档案。

这不是口头价值观辩论，而是 Mickey 当下仍能选择的一条生路。

他看向 Zack 手里的档案袋。只要开口要求归还、拖到楼下人员上来，保险仍有可能回到系统手里。

Mickey 看着档案袋，只对电话那头说：

> "档案不归你们了。"

他拔掉电话线，把通往市政厅的听筒放到桌面，没有再向 Zack 索回档案。

这是他第一次先让一件自己无法继续控制的证据留在别人手里。它不能抵消此前的死亡，但使随后的坠落不再只是"输掉辩论后的自毁"。

## 坠落

Mickey 走到窗边。窗外不是抽象黑夜，楼下已经停着 Miller 系统的黑色轿车。

那辆车不是证明 Mickey 无路可走；它正是他刚刚拒绝的生路。交回档案上车、留下配合 Zack、走下楼面对审判，都仍是选择。坠落不能被写成系统逼出的唯一出口。

他没有彻底忏悔，也没有说自己所有选择都错。他只是第一次无法确定自己一直代替死人给出的答案。

> Mickey："Liam 呢？"
> "Patrick 呢？"
> "他们会觉得我至少做对过一点吗？"

Zack：

> "我不知道。"
> "你也不知道。"

这正是 Mickey 一生不肯接受的答案：没有人可以替死者补签同意。

楼下轿车第二次鸣笛。走廊电梯开始上行。

Mickey 退到窗边，像是要避开办公室门口，也像是已经作出了另一个决定。Zack伸手抓住他的袖口，布料在窗框边缘撕裂。

画面只保留 Zack 抓空的手与向下坠落的身影，不替玩家确认最后一瞬是 Mickey 主动放弃、脚下失衡，还是在楼下与门外压力逼近时作出的混合选择。

这一镜只确认 Zack 伸出去的是抓人的手，不替玩家判断 Mickey 最后一瞬的意图。

## 坠落后的安静余波

Zack 仍保持着伸手的姿势，掌心只剩一小片被撕下的袖料。Emma 走到窗边，把窗关上，没有说"你们不一样"，也没有替他解释刚才发生了什么。

被 Mickey 拔掉的电话没有声音。楼下轿车仍停着，电梯一层层上升。

Emma 收起两人共同看过的身份材料。她接过 Zack 手里的档案袋，打开袋口确认 Miller 内封套的防拆线仍然完整，随后重新封好袋口。

Zack 捡起 Liam 的旧工牌，把它压回功业簿封面。两人没有对白。直到电梯铃在下一层响起，他们才离开办公室。

Loop 5 在这里结束。四十二层的胜利先落地，下一场危机不在 Mickey 坠落的同一口气里开始。

## 玩家体验

💡 抓住了 Whale，却没有抓住制造 Whale 的机器。

玩家应该同时得到满足和失重：

- Mickey 的身份是自己证明的。
- Mickey 的善意和成果没有被抹掉。
- 这些成果仍然不能替他的受害者签字。
- Mickey 死后，Miller、Lakeshore、Tidewater 与已经启动的清退计划仍然存在。

# 非 Loop 终幕：第二次没有说完（Unit4 → Unit5）

## 终幕功能

Mickey 坠落不是 U4 的唯一结尾。真正的断点是：Zack 刚刚拒绝 Mickey 替别人管理真相的算法，却在离开四十二层后，因为害怕 Emma 成为下一个知情死者，再次选择先控制她能看见什么。

这次错误必须写得公平：

- 街对面仍有人监视时，收起档案是正确的战术判断。
- 进入安全地点后，继续压住 Sean 与水源，才是 Zack 主动重复旧错误。
- O'Hara 在 Zack 打开内封套前已经喝下被污染的井水，她的死亡不是街上战术延迟造成的后果。
- Zack 的持续沉默会让公共水点预警变迟、更多居民暴露，也会剥夺 Emma 共同判断的权利；但不直接制造数条本可避免的人命。

## 楼下：只能延迟，不能隐瞒

Zack 和 Emma 带着 Miller 保险外卷与尚未拆开的内封套离开大楼。

街对面黑色轿车的车窗半降，楼门附近还有一名假装读报的人盯着档案袋。监视者没有立刻抢夺文件：四十二层仍是律师楼，门口有夜班职员与路人；他们更想确认档案会被送给谁、还有多少副本。

Zack 没有让 Emma 在街上打开内封套，也没有把这一刻包装成一次价值观选择。

> Zack："别在这里拆。"
> "他们在等我们告诉他们，这只袋子里哪一页最重要。"

Emma 看见监视车辆，接受这次战术延迟。两人先带着整只档案袋离开；这里没有"告诉 Emma / 隐瞒 Emma"的二选一，收档是双方共同认可的反跟踪动作。

## 摆脱监视

Zack 联系 Watts，不是让他突然充当行动队成员。Harrison 曾把法院钥匙和自首包托付给他；Watts 能调用一辆不显眼的法院档案车，也能从市政档案通道进入旧法院附属楼。

三人先用普通出租车制造一条可见路线，再在档案通道换乘法院车辆。黑色轿车继续跟着第一辆车，读报人则留在律师楼确认 Mickey 的死讯。

Watts 把 Zack 和 Emma 带进一间只有法院内线电话、没有临街窗户的旧档案室。此时监视已经暂时摆脱，继续隐瞒不再是现场安全的必然要求。

## 安全档案室：内封套

时间接近凌晨四点二十分。为了确认监视车辆没有跟到档案原件，三人已经在法院档案车、市政档案通道与两处封闭库房之间连续换路数小时。

Emma 先拨 O'Hara 家的电话，铃声持续很久却无人接听；她随即联系 Rosa Martinez，确认 Sarah 已安全到达，并提醒她不要从原路送孩子回家。Watts 去核对 O'Hara 临时停止执行回执是否已经送达辖区。

Zack 留在主档案桌前整理保险外卷。他本可以等 Emma 回来再拆内封套，却先剪断防拆线，独自看见两组材料。

### 1912 内页

- 🟢 Tidewater 地基材料替换单：承重材料在事故前被主动降级。
- 🟢 Sean O'Malley 特殊处置页：要求将 Sean 与工人队伍分开，并在事故记录完成前"处理其异议"；有老 Charles 签名，但不直接写杀人命令。
- 特殊处置页使用 Miller 事故基金压印，与 Emma 的奖学金、Foster 的研究资助和 L2 病房铭牌属于同一枚印章。
- 边界：Sean 的死亡从普通工伤变成高度明确的谋杀疑云；谁亲手执行、如何弃尸和伪造事故，仍留 U5。

### 1928 后插入附件

- 🟢 后插入的水源维护页：纸张批次、日期格式与原计划不同；在尚未报告污染前，提前安排 O'Hara 私人井与两个公共水点"封闭、清洗、重新开放"。
- 附件没有投毒物名称、银月会执行人、小 Charles 签名或老 Charles 的正常审批。
- 边界：足以看出有人越过原本以债务和房产接收为主的节奏，追加了水源层面的危险行动；不足以在 U4 完整指认主使与执行链。

Emma 回到主档案室时，先看见内封套已经拆开。

> Emma："里面是什么？"

Zack 想起 Webb、Harrison、Morrison 和 Mickey 都死在"知道太多"之后。他没有邀请 Emma 一起判断风险，只从外卷中抽出两人已经见过的 O'Hara 名单。

> Zack："O'Hara 还在第一批优先处理名单里。"
> "停止执行只能挡住警察。车一调出来，我们就去确认她是否安全。"

Emma 看了一眼被剪断的防拆线：

> "我问的是你刚拆开的那只封套。"
> "里面只有这一页需要现在处理？"

> Zack："现在能确认的，只有这个地址。其他是还没对完的旧项目材料。"

他拿出的 O'Hara 名单是真的，后面的回答却把 Sean 与水源降格成了"尚未确认的旧材料"。这已经不只是没说完，而是 Zack 主动替 Emma 判断什么才算需要知道。

> Emma："上车以后，把剩下的给我看。"

Watts 恰在这时回来说档案车已经调出。他看见防拆线断开，先记录开封时间与内页数量，再把外卷和内页一起锁进法院证物转运箱。O'Hara 的电话始终无人接听，Emma 选择先出发，不等于接受 Zack 的内容判断；证物箱在抵达现场前不再开启，她此时没有理由知道箱里还有父亲页和公共水点页。

第一次在街上收起档案是正确的安全判断。现在继续不说，是 Zack 主动重复旧错误：他先替 Emma 决定哪些风险可以知道、哪些真相应该推迟。

从档案室到南区约五十分钟。等待值班司机把档案车从内院调出、清点材料并赶往南区的过程中，Zack 仍没有提水源附件和 Sean。这段持续沉默明确越过战术延迟，进入替 Emma 管理真相；更关键的是，Watts 因而没有在出发时就通知公共卫生值班员。

## O'Hara 与水源行动的真实时间边界

- 22:20：Mickey 身份即将暴露已经不可避免，小 Charles 一侧先启动 O'Hara 私井的测试行动。伪装成公共卫生人员的执行者从后巷院外井检口投入污染物。
- 22:40 前后：O'Hara 使用井水煮茶。此时 Zack 尚未打开四十二层的保险柜，也没有取得任何水源附件。
- Mickey 坠落后：上层确认 Whale 接口失效，两个公共水点的"维护"被继续推进，安排在清晨开放。
- 凌晨四点二十分：Zack 在安全档案室看到水源页。此时已经无法挽回 O'Hara 喝下的水，但若立即共享，Emma 与 Watts 可以在出发前联络公共卫生机构。
- 清晨五点：第一处公共水点按"维护后重新开放"的安排短暂启用；第二处尚未启用。Zack、Emma 与 Watts 正在前往南区，尚不知道必须先打给卫生机构。

这条时间线保证两件事同时成立：O'Hara 的死亡不是街上战术延迟造成的后果；Zack 在安全地点继续沉默仍然造成了真实但不过量的行动代价。

## 次日清晨：O'Hara 家

Zack、Emma 和 Watts 乘法院档案车抵达南区。Rosa 与 Sarah 仍留在 Rosa 家；Emma 此前已通过法院内线确认孩子安全。三人到 O'Hara 门前时，敲门始终没有回应。Emma 从临街厨房窗看见 O'Hara 倒在地上，Watts 随即叫来消防队破门；在应急人员进入前，前门内闩、院门和窗锁都被完整记录。

### 现场

- 🟣 应急人员进入前，院门与前门仍从内锁住，临时停止执行回执贴在门内；此前没有强行入屋或入院痕迹。
- 🟢 O'Hara 已经倒在厨房死亡；水杯、烧水壶与后巷井检口内有相同异常沉淀。
- 🟢 后巷井检口：锁盖有新扳手痕和伪造公共卫生铅封，证明来人从院墙外接触水源。
- 🔵🟢 井口旁遗落的工具箱仓储标签：编号格式与 U2 银月赌场酒水、清洁用品供货票属于同一仓储系统。它能证明执行者使用银月会地面物流，不能单独指认具体经手人或下令者。
- ⚪ Sarah（由 Rosa 在电话里转述）：她在 L4 离开前看见两名"水质维护人员"逐户核对门牌；她当夜一直住在 Rosa 家，没有饮用私人井水。
- ⚪ 邻居：那批人没有进入院子，只在后巷检修口停留，并说清晨还要检查两处公共水点。
- 🟢 O'Hara 最后留下的拒售页："房子不卖。签名只能是我的。"

临时停止执行真实拦住了警察协助、强制入屋与搬离。执行者只能换成公共卫生外观，从院外水路绕过；保护不是没有发生，而是系统换了一条接口。

Emma 看见水壶沉淀后，先让 Watts 再次通知 Rosa：不要带 Sarah 回来，也不要让孩子接触附近水点。她随后立刻问档案里是否出现过水源。

Zack 的停顿已经给出答案。

Watts 打开法院证物转运箱。Emma 抽出被重新压回去的水源附件，又看见下面的 Sean 页面。她第一次知道 Zack 在安全档案室究竟删掉了什么。

Watts 没有凭法院书记身份指挥警察。他把伪造卫生铅封与法院停止执行回执交给已经到场的消防队，请消防队通过调度台通知辖区公共卫生值班员封闭水点。

第一处公共水点附近已有十余人出现恶心、眩晕和腹痛，暂时没有死亡；第二处水点在开放前被封闭。Zack 的延误扩大了暴露与救治压力，但 O'Hara 是唯一已经死亡的人，她在 Zack 取得情报前就已喝下井水。

## Emma 的爆发

Emma 的愤怒分两层，顺序不能颠倒。

第一层是眼前的共同判断：

> "你知道不只是 O'Hara。"
> "你在有电话、有 Watts、没有人盯着我们的地方看见了水源页。"
> "可你只给了我一个地址。"

第二层才是 Sean：

Emma 把水源附件掀开，看见 Sean 页面上的事故基金压印。那正是她在奖学金、Foster 仪器和医院病房上见过的印章。

> "他们用这枚印章替我交学费。"
> "也用它批准'处理'我父亲。"

> "你还看见了我父亲。"
> "你不是在保护我。"
> "你是在决定我什么时候有资格知道，他的死可能根本不是事故。"

Zack 可以说知道太多的人正在一个个死，也可以说自己想先确认 O'Hara。这些理由都是真的，但不能自动使决定正确。

Emma：

> "那你可以告诉我有人在盯。"
> "你可以让我和你一起决定怎么活下来。"
> "你不能先删掉一半，再把剩下的一半叫保护。"

这句话把 Margaret、Mickey 与 Zack 的行为连在一起：三个人都出于不同程度的爱、恐惧或责任感，先替别人管理了真相。

## Emma 离开 Zack、前往 Miller 庄园

当日上午，公共水点的伤者完成初步转送，原件也被送回法院保管。Emma 没有在争吵后无目的地跑开，也没有忽略知情者接连死亡的现实。

她手里仍缺一块能把 Sean 特殊处置页变成完整真相的证词，也需要知道是谁把水源附件插入原计划。小 Charles 是她唯一亲自接触过、也能通过慈善办公室直接找到的 Miller 家族成员。

Emma 先完成三项安全动作：

1. 她让 Watts 保管 Sean 页面与水源附件原件，并在法院制作两套带时间戳的完整照相副本，另单独复制一张 Sean 页面供自己携带。
2. 两套完整副本分别装入寄往州检察官与报社的密封信封。Emma 写明去向、邀请人和晚上十点的回报时限；如果她没有按时从庄园打回电话，Watts 立刻把信封送出。这项安排也明确告知 Miller 慈善办公室。
3. 她要求小 Charles 在当天晚上将举行、已有宾客、记者和工作人员登记的 Miller 慈善晚宴见面，不接受私人书房里的单独会面；法院司机负责送她到庄园并留在门外。

午后，Emma 从报纸社会版确认慈善晚宴照常举行，又在附近药房借电话联系 Miller 慈善办公室。小 Charles 得知她看见 Sean 的名字后没有否认，只说：

> "今晚来庄园。"
> "晚宴照常举行。我会让所有人看见你进门。"
> "我会告诉你 1912 年到底发生了什么。"

这句话在玩家当下听来像他愿意提供公开安全与完整答案；到 U5 才会明白，宾客的目击正是小 Charles 需要的东西——Emma 越公开、越真实地愤怒，越适合成为老 Charles 死后的嫌疑人。

Zack 想跟她去。Emma 拒绝的只是让 Zack 再次主导这次询问，不是拒绝全部安全措施：

> "你已经替我决定过一次。"
> "十点以前我会打电话。十点以后，按我们说好的做。"

她没有把原件带走，也没有让自己的行程成为秘密。她选择承担风险，不等于拒绝安全判断。

Zack 没有获得立刻修复关系的机会。公共水点仍有伤者需要后续观察，Watts 也需要保护原件与 Sarah 的临时住处。他只能看着 Emma 带着 Sean 页面副本登上法院车辆；司机在庄园门外等她，十点回报时限已经开始。

U4 的最后行动不是"Emma 将来会去庄园"，而是她已经踏上这条路；U5 从小 Charles 如何利用这次公开接触开始。

## 进入 U5 的状态

U4 结束时，玩家必须明确知道：

1. Mickey = Whale 已经坐实，但 Whale 不是系统最高点。
2. Miller → Lakeshore → Tidewater 结构已出现，金融分赃证据仍未完全落地。
3. Sean 的死已成为谋杀疑云，但谁亲手执行、如何伪装仍未坐实。
4. O'Hara 已经因私人井污染死亡；第一处公共水点造成多人症状、第二处被及时封闭。银月会地面物流关联成立，幕后越权者和具体执行链留给 U5。
5. Emma 没有把 O'Hara 的死归责给 Zack；两人的裂缝来自 Zack 在安全后仍替她管理真相与风险，并让公共水点失去一次更早预警的机会。
6. Emma 已留下原件、回报时限和公开行程，带着"几乎成形、但差最后证词"的怀疑前往 Miller 庄园。

💡 U4 最后的尤里卡：

> 我们抓住了 Whale。
> 但海水没有停。
> 更糟的是，Zack 刚刚拒绝的那套算法，也可以用"保护"这种更温柔的名字重新出现。

# 本版修正后的关键因果闭环

## Harrison 为什么死

第三次拒绝撤回 Mary 改判意见 → 开始打包医院、账户与地产计划 → Mickey 操作簿记录"第三次拒绝撤回"并启动清理。
结论：不是因为他"忽然变好"，而是因为一个长期可控的腐败工具开始产生公开风险。

## Morrison 为什么当夜开口

看见 Whitfield 被止损交出 → 收到次日交出全部旧案材料的通知 → 认出自己进入 Harrison 相同清理程序 → 当夜先找 Zack、后找 Mickey 谈条件。
结论：他的转向来自自保和拒绝被改写，不是突然良心发现。

## Mickey 为什么亲自动手

Morrison 临时决定当夜离屋 → 外部寄存柜已经存有备份 → 23:50 的固定清场只能炸掉仍在宅邸的人与文件 → Pierce 与黑警被法院封存锁住 → 银月会常用人手在西区行动中失联 → 没有既能截住移动目标、又懂警务伪装的中间人。
结论：Mickey 不能只等爆炸，亲自动手是时间、目标移动与资源共同造成的选择，也因此留下右手错误。

## Margaret 为什么隐瞒

Zack 从 1919 年起就知道 Patrick 把 Mickey 带出了清场 → 葬礼上 Mickey 把这件事解释成"最后选择了我"，Zack随即追问该从哪里继续调查 → Margaret当场阻断，并隐去 Patrick原本准备回家、已经到达内侧出口、在尚不知道被困者身份时放下账簿折返的完整经过 → 1928 年 O'Hara 的现实求助再次让 Zack说出"不能不管"，Margaret因而劝签、藏物、阻断求助。
结论：她没有伪造 Patrick救过 Mickey 的事实，也不是维护他的名誉；她想阻止死去的 Patrick再把 Zack带走，却通过管理儿子如何理解父亲，再次剥夺了 Zack的知情与选择权。

## 保险档案为什么最终留下

爆炸命令证明 Miller 同样准备牺牲他 → Morrison 失误使身份即将暴露 → Mickey 主动安排最后对质，却仍相信私人密码能守住身份材料与保险 → Zack 与 Emma 回收 U2 已见的 Patrick 遗物匣刻句并破解密码 → Miller 以交回档案换取生路 → Mickey 最终拒绝索回档案。
结论：身份证明属于玩家从 Mickey 手里夺回的主动权；档案得以保留，才属于 Mickey 最后一次真实选择。

## Zack 的错误为什么成立

街上收档是双方共同认可的安全动作 → 离开监视后获得完整共享机会 → 他仍只说 O'Hara、不说水源和 Sean → 现实风险扩大，Emma失去共同判断资格。
结论：O'Hara 在 Zack 取得材料前已经喝下井水，她的死亡不是街上战术延迟或 Zack 后续隐瞒造成的；剧情批评的是安全后仍继续替别人决定，并让公共水点预警与救治因此变迟。

## O'Hara 为什么在已有保护下仍然死亡

值班法官签发停止执行 → 警察不能协助入屋、搬离或强制签约 → Sarah 被提前送到 Rosa 家 → O'Hara 自主选择留守 → 执行者伪造公共卫生身份，从院墙外的后巷井检口污染水源。
结论：主角的保护行动真实发生，也救下了 Sarah；系统不是无视保护，而是从法律清退切换到另一条尚未被防住的水源接口。

# 写作边界

- 不把 Harrison、Morrison 洗白。他们的回头不取消既有责任。
- 不把 Patrick 的 1919 选择写成彻底洗白或完整思想证明。他先同意一次已经过险的证据转运，参与制造了危机；最后的折返只证明他能够作出与 1903 年不同的行动，不能取消 Liam 的死亡、长期缺席家庭的事实，也不能替他宣布已经解决所有价值矛盾。
- 不写成 Patrick 用"许多人的生命"交换 Mickey。他放弃的是可能帮助许多人的唯一原始账簿和调查成果，冒险时只替自己作决定。
- 不给 Patrick 安排"替我继续""完成使命"之类遗言。没有任何事实支持使命交接；那是 Mickey 为承受幸存者负担而作出的解释。
- 不让 Patrick 在折返前确认被困者就是 Mickey。事实只证明他听见"还有一个人"后返回；Mickey可以主张 Patrick凭习惯猜到是自己，但不能把这种主张写成已证实事实。
- 不把 Margaret 写成反正义、怕麻烦或维护 Patrick 名誉。她害怕的是 Zack 把父亲最后一次没回家理解成值得继承的道路；她的错误在于控制真相，而不在于这份恐惧本身。
- 不把 Mickey 写成 Patrick 有意选中的继承人。Mickey 的悲剧恰恰是把"被当作一个人救下"误读为"被当作继承人选中"。
- 不把 Emma 写成只会要求公开一切。她接受战术上的延迟，但拒绝永久被排除在决定之外。
- 不让 Mickey 的真实帮助变成骗局。Rosa、Mary 与工人援助都必须真实有效。
- 不让 Zack 靠"杀人就是错"轻易赢过 Mickey。他必须承认自己的相似处，再拒绝替别人签字。
- 不用牙痕单独定罪。牙痕只与法律壳、操作记录和时间线共同构成身份闭环。
- 不让水源附件直接写"投毒命令"。U4 只给足够后悔与行动的风险，不提前完成 U5 指认。
- 不在 U4 坐实 Lawson 私生子、小 Charles 弑父嫁祸、Sean 完整执行链和金融分赃终局证据。

# 一句话版本

U4 不是让玩家发现 Mickey 一直在假装做好人。

它让玩家发现：Mickey 做过的好事有很多是真的，他承受过的痛也是真的；但他为了让第一场牺牲显得值得，逐渐相信自己有权替所有后来的人安排真相、风险与死亡。Zack 最终拒绝了这套算法，却又在保护 Emma 时短暂重复它。Mickey 坠落以后，真正留在城市里的不只是 Miller 的机器，还有一个更难回答的问题：当控制以善意的名字出现时，我们还能不能认出它？
