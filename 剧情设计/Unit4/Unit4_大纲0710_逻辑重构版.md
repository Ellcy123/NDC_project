## 一、全章核心

U4 不是一章“揭晓 Mickey 是坏人”的故事。

它讲的是 Zack 亲手把几类原本互不相干的东西放到一起：Harrison 的改判、圣心医院的儿童死亡、Morrison 的灭口、O'Hara 的拒售、Patrick 的工运遗产，以及 Mickey 一次次真实有效的帮助。玩家最后发现，这些事情背后有同一个人，也有同一种算法。

这套算法不是简单的恶意，而是：

> 我比你更了解局势。  
> 所以我可以替你决定什么时候知道真相，也可以替别人决定谁值得成为代价。

本章对人物保持开放，对这种权力保持明确的警惕：

- Harrison 和 Morrison 有罪，但他们仍有回头与开口的权利。
- Patrick 确实改善了八百个工人的生活，但他没有权替 Liam 说“值得”。
- Mickey 确实救过 Rosa 和 Mary，但真实成果不能赋予他分配人命的资格。
- Zack 的动机比 Mickey 温和得多，但他在恐惧里也会靠近同一条边界。

## 二、图例与执行规则

- 🟢 证据 / 道具：玩家可以获得、拍照、分析或用于指证。
- ⚪ 证词：NPC 说出的关键有效信息，可进入证词池或用于明确推理。
- 🟡 谎言：当前指证中会被击破的具体说法。
- 🟣 环境叙事：用于讲故事、铺情绪或后续回收，通常不用于指证。
- 🔵 铺垫 / 回收：不能单独出现，必须附着在其他图标上。
- 🔒 动机信息：指向角色为什么作出选择。
- 🎮 玩法：玩家需要主动操作、比对、连线或输入。
- 🧪 派生证据：由分析、组合或记忆复审产生。
- 💡 尤里卡：希望由玩家自己想通的关键时刻。

执行硬规则：

1. L1-L4 每轮谎言最多由两件材料击破。
2. 指证材料必须在该轮指证开始前可获得。
3. 价值判断不能伪装成事实谎言；情感坦白放在事实防线被击穿之后。
4. 每个 Loop 只完成一个主要认知翻转，同时交出下一 Loop 的行动理由。
5. Zack 只指出矛盾，不替玩家把所有结论说完。

## 三、全章事实时间线

### 1903—1919：算法的来源

- 1903 年，Patrick 作为工会法律顾问坚持罢工谈判。清场中 Liam Donnelly 和另外两名工人死亡，少年 Mickey 右脸受伤、右下牙缺损。
- 三人的死亡推动舆论与谈判，约八百名工人的工时从十四小时缩短到十小时。
- Patrick 说：“如果 Liam 知道结果，他也会觉得值得。”他可能是在安慰别人，也可能是在安慰自己，但他替 Liam 作出了决定。
- Mickey 最初恨 Patrick，后来为了不让父亲白死，选择相信 Patrick 的解释。
- 1919 年 Patrick 去世后，Mickey 对 Zack 说“我会接着做”；同年 Donnelly & Associates 开始承接银行、医院与地产法律业务。

### U3 后段：Zack 的第一次隐瞒

- Emma 发出帮助 Mary 的报道后，Zack 才最终确认 Mary 与 Helen 提前策划了 Thomas 的死亡。
- Zack 没有告诉 Emma。他认为报道已经救了 Mary，继续揭露只会毁掉已经救下的人。
- 这不是 U3 报道造假，而是 Zack 在真相形成之后，替 Emma 决定她不需要知道完整版本。

### U4 当下

- Harrison 在 Mary / Helen 案上两次被要求收回改判意见，仍第三次提交合法杀人 / 正当防卫方向的正式意见。
- 同期，他连续两个月调阅圣心医院旧案，整理自己收受资金、儿童死亡与南区开发计划之间的关系。
- Harrison 从“可控的腐败法官”变成准备违抗系统、可能公开材料的风险源，因此被杀。
- L1 后，系统允许 Zack 沿圣心医院执行层调查，因为牺牲 Whitfield 可以保护 1919 法律壳和地产结构。
- L2 后，Morrison 看见 Whitfield 被迅速交出去，又收到要求次日交出旧案文件的通知。他意识到 Harrison 的清理方式正在轮到自己。
- L3 当夜，Morrison 尝试联系 Zack。Mickey 因当晚警察与银月会可用执行者都被其他任务锁死，只能亲自到场。
- Mickey 杀死 Morrison、摆错持枪手、留下延时爆炸装置；Zack 因 Doris 提前回家而意外先于 Pierce 到场。
- L4 中，O'Hara 的当下拒售让 Patrick、Liam、Margaret 与 Mickey 的过去重新进入现实。
- L5 中，Mickey 主动让 Zack 来四十二层。他既想让 Zack 看见自己做过什么，也在测试 Zack 会不会成为下一个“懂得计算的人”。
- Mickey 坠落后，系统没有停止。小 Charles 已经提前推进针对南区水源的过激清退。

# Loop 1：守门人的钥匙（起）

## 本轮功能

核心问题：Harrison 是因为腐败而死，还是因为准备背叛腐败系统而死？  
指证对象：Watts。  
指证轮次：3。  
本轮边界：只能确认 Harrison 正在回头、死亡与改判及旧案复查高度重合；不能在 L1 直接证明是谁下令杀他。Mickey 只留下“知道得太准”的轻微异样。

## 开篇剧情

**法院东翼 / Harrison 办公室外**

Zack 和 Emma 原本来补办 Mary / Helen 案文件，却遇到法院东翼封锁。Harrison 前一夜死在办公室里，死因为警局制式弹药造成的枪伤。

Pierce 负责现场。他的态度始终体面：桌面与公共档案可以查看，Harrison 私人内柜、保险柜和封存箱必须等法院授权。

Watts 抱着 Harrison 的钥匙链和旧卷宗箱，不肯交给 Pierce，也不肯交给 Zack。他不是相信 Harrison 无罪，而是不肯让任何人只用“腐败法官畏罪死亡”六个字写完 Harrison。

Mickey 也在。他说自己来法院处理另一个案件，听档案室的人说 Harrison 最近一直在翻圣心医院旧案，因此建议 Zack 先查调阅记录。

行动目标成立：玩家先调查 Harrison 最近在做什么，再击穿 Watts 的守门理由，获得保险柜和私人内柜。

## 自由探索

### Harrison 外间办公室

- 🟢 Harrison 本周公开日程：只有日常庭审和例行会议，没有需要连续深夜工作的常规事项。
- 🟢 法院夜间出入登记：Harrison 连续十七晚独自进入档案室或办公室，时间集中在闭庭后。
- 🟢 圣心医院汇款存根：从普通费用抽屉中找到。Harrison 长年把这类汇款当作正常收入，并没有刻意销毁。
- 🟢 未完成的辞职信草稿：在打字机废纸篮中，日期为 11 月 27 日，写有“我不能一面要求他人承认罪行，一面继续否认自己的部分”。
- 🔵🟣 南区开发计划复写纸：只能辨认出标题“南区综合商业开发计划”和“Tidewater / Lakeshore”两个词，正文仍在保险柜里。
- 🟢 弹道初步：Harrison 死于警局制式弹药。它说明执行者能接触警用渠道，但不能单独证明 Pierce 或 Morrison 动手。

### 法院档案室 / 档案管理员

这次聊天只提供程序事实，不评价 Harrison 的人格。

- 🟢 Harrison 两个月调阅索引：七次调阅圣心医院儿童死亡旧案，三次调阅南区地产判决，两次调阅自己签过的赔偿裁定。
- ⚪ 档案管理员：Harrison 每次都自己抄写，不让书记员代办；他最近没有让 Mickey 来取过这些卷宗。

Mickey 先前明确说“档案室的人告诉我他在查圣心医院”。档案管理员的说法不能证明 Mickey 撒谎，因为他可能通过电话或别的书记员得知；它只留下一个后续会变味的问题：Mickey 为什么知道得这么具体？

### Zack 公文包 / U3 已持有材料

- 🟢 Mary / Helen 案改判往来信：Harrison 第一次提出改判后被以舆论风险驳回；第二次补充事实依据后再次被退回；第三次仍坚持提交正式意见。
- 🔵🟣 第三次提交回执：时间就在 Harrison 死前三天。L1 只能形成时间关联，L5 才从 Mickey 的行动记录中坐实它是清理触发点。

### Watts / 可对话

- 🟡 Watts：Harrison 只是工作多，最近没有偷偷重查任何旧案。
- ⚪ Watts：Harrison 不让书记员碰最后几晚的材料。
- 🟣 Watts 怀中的钥匙链：他每次听见 Pierce说“畏罪”都会攥紧钥匙，但仍不肯解释原因。

## 指证 Watts

| 轮次 | 谎言 | 击破材料 | 玩家可得结论 |
|:---:|:---|:---|:---|
| R1 | 🟡 Harrison 只是处理日常工作，没有秘密重查旧案 | 🟢 本周公开日程 + 🟢 两个月调阅索引 | 深夜工作内容与公开日程不符，他在系统性复查医院、地产和自己的判决 |
| R2 | 🟡 即使他查旧案，也是在调查别人，不是在查自己 | 🟢 圣心医院汇款存根 + 🟢 调阅索引中的本人赔偿裁定 | Harrison 把自己收过的钱和自己签过的裁定放进了同一条调查线 |
| R3 | 🟡 他不会真的违抗法院，更不会把事情公开 | 🟢 Mary / Helen 三次改判往来信 + 🟢 未完成的辞职信草稿 | Harrison 已经连续违抗上层意见，并开始准备承担公开后果 |

## 指证后：保险柜只打开这一次

Watts 交出钥匙。他承认 Harrison 死前告诉他：如果第二天没有来，就不要先把钥匙交给 Pierce。

保险柜里不是腐败藏宝箱，而是一只尚未完成的自首包：

- 🟢 圣心医院儿童死亡对照表：五年七例，家庭背景、慈善项目、药剂批次和责任归属高度相似。
- 🟢 Harrison 亲笔资金流向图：圣心医院控股集团 → Harrison 私人账户 → 1919 中间账户。中间账户名称暂时只显示 Donnelly & Associates 的缩写。
- 🟢 南区综合商业开发计划摘要：首次完整命名项目；写明以世博会候选地窗口为期限，通过银行债务、司法裁定和地产接收清空南区地块。文件出现 Tidewater 与 Lakeshore，但没有 Miller 名字。
- 🟢 1912 事故家庭索引：附在开发计划末页，使用旧事故赔偿编号标记住户；同一批编号后来又出现在“南区居民援助计划”的贷款、违约与房产接收记录中。
- 🟢 Harrison 最后工作顺序表：第三次提交 Mary 改判意见后，他开始打包医院名单、资金图和开发计划；旁边写着“如果他们再次要求我撤回，就交给外部律师”。
- 🟣 给 Watts 的便签：“你替我守过太多次门。这一次，不要替我挡住后果。”

🎮 地图叠加：

玩家把 U3 结尾取得的“南区援助计划内部地图”与本次开发计划边界、1912 事故家庭索引叠在一起。

- 银行制造债务最密集的街区，与 Tidewater 计划优先接收的地块重合。
- 被标为“优先接触”的家庭，大多沿用 1912 事故赔偿编号；当前住户是当年事故家庭及其后代。
- U1 录音里的“1912 / 南区地块”、U2 的银行债务、U3 的房产流向，在这一刻第一次成为同一条地理路线。

- 🧪 两次清退叠加结论：1912 年事故把这些家庭赶离码头，1928 年计划正在沿着他们最后的居住区再次推进；世博会窗口让第二次清退进入倒计时。

这里让玩家自己完成第一层判断：Harrison 有罪，但他的死亡发生在他第一次准备让自己的罪成为证据之后。

## L1 结尾与 L2 钥匙

Rosa 的名字出现在儿童死亡对照表最后一页，Isabel 是最新一例。法院通知显示，圣心医院将在次日由慈善项目负责人 Whitfield 出庭，把死亡责任正式归给 Rosa。

Mickey 看完材料后只说：

> “先救还活着的人。账户不会在明天下午被判有罪，Rosa 会。”

这句话合理，也确实正确。它把玩家带入 L2，同时第一次建立 Mickey 的行动方式：他从不需要提供假方向，只需要替 Zack 排定真相的优先级。

## 玩家体验

💡 保险柜不是赃物库，是一个来不及交出去的自首包。

玩家离开 L1 时应该同时相信三件事：

- Harrison 长期腐败。
- Harrison 死前真的准备回头。
- 杀他的人很可能不是在惩罚一个坏人，而是在处理一个失去控制的工具。

# Loop 2：被挑过的胜利（承）

## 本轮功能

核心问题：Isabel 的死是 Rosa 擅自加量，还是圣心医院慈善项目制度性使用问题药剂？  
指证对象：Whitfield。  
指证轮次：3。  
本轮边界：Mickey 提供的材料必须真实有效。玩家可以感觉他总能决定先查什么，但此时不应认为他伪造证据或就是 Whale。

## 开篇剧情

**Zack 侦探事务所 / 次日清晨**

Rosa 抱着 Isabel 的照片和慈善项目药盒来找 Zack。Whitfield 当天下午将在法院作证，医院提出的条件是：Rosa 承认自己在家擅自增加剂量，医院可以不追究她“误用慈善药品”的责任。

Rosa 最害怕的不是坐牢，而是 Isabel 的死亡记录从此写成“被母亲害死”。

Foster 来电说，Harrison 留下的一支同批次样本已经完成初检。Mickey 随后带来一份医院采购记录，明确说材料来自医院理事会中的旧客户关系。

Emma追问 1919 中间账户。Mickey 回答：

> “账户可以继续藏一天。Whitfield 下午说完那句话，Isabel 就会被写死一辈子。”

玩家应当认可这个优先级，因为 Rosa 确实等不了。

## 自由探索

### Rosa / Zack 事务所

- 🟢 慈善项目每日剂量卡：医院配发固定浓度的 5cc 封签玻璃药瓶与专用刻度注射器，要求 Rosa 每天只抽到护士画出的红线，不允许自行调整；Isabel 的卡片上每天都有 Rosa 的按指纹确认。
- 🟢 回收药瓶剩余量鉴定：药瓶初始装量、十三次规定剂量与当前剩余液量完全相符，没有额外一次注射所需的液量缺口；瓶塞也只有正常反复穿刺痕迹。
- ⚪ Rosa：她不识单位，只会把药液抽到护士画出的红线，从未拿到第二只药瓶。

这组材料不是证明 Rosa “不懂所以不会犯错”，而是用实际消耗量证明她没有在规定剂量之外额外抽取药液。

### 圣心医院 / Miller 事故基金病房

这一区域只承担主线情感铺垫，不进入 Whitfield 指证。

- 🟣 Miller 事故基金病房：墙上挂着 Miller 家族事故基金铭牌，印章与 Emma 的奖学金证明、Foster 的早期研究资助和 Margaret 医疗减免文件相同。
- 🟣 正常治疗记录：病房不是空壳，许多工伤家庭与儿童确实在这里获得过免费治疗；护士能说出几个康复出院的孩子。
- 同一所医院里，普通基金病房真实救人，慈善试用项目却把问题批次集中发给最难申诉的移民家庭。善意与伤害不是一真一假，而是在同一套声誉系统里同时发生。

Emma 看着铭牌说：

> “我读大学时，奖学金文件上也是这个印章。”

Foster 没有评价 Miller，只说自己的第一套法医仪器也来自同一基金。玩家此时仍不能证明 Miller 与医院犯罪有关，只会感到一份真实恩情正在靠近一件真实伤害。

### 法院实验室 / Foster

- 🟢 同批次药剂化验报告：有效成分浓度波动异常，并含不应出现在成品中的降解杂质；连续使用可能造成严重低血糖反应与器官损伤。
- 🟢 批次封签对照：Harrison 留下的样本、Rosa 药盒封签和另外两名死亡儿童病历上的批号属于同一采购批次。
- ⚪ Foster：这不是“保存不好”能造成的单支异常，同批封签在进入家庭前就有问题。

### 圣心医院 / 预审走廊 / Whitfield

Whitfield 是慈善医疗项目的行政与医疗负责人。他不是幕后最高层，但负责签署项目说明、对外作证和把责任落到家庭身上。

在正式指证前，他只主动给出第一层防线：

- 🟡 Whitfield：Rosa 没按说明使用，在每日规定剂量之外又擅自增加了药量。
- 🟣 Whitfield 随身携带统一格式的“家属误用说明”，上面已经预留 Rosa 的签字位置。

自由探索阶段只取得 R1 谎言。R2、R3 必须在前一轮防线被击穿后，由 Whitfield 当场退守形成，不能提前进入证词池。

### Mickey / 法院会客室

- 🟢 圣心医院采购与发放记录：显示该问题批次以“低成本慈善试用”名义采购，并集中发往底层移民儿童项目。
- 🔵🟣 缺失的上层授权页：页码从 18 跳到 20。Mickey 明确说他只能拿到理事会执行层副本，原件授权页被单独封存。

这份材料足以打掉 Whitfield，却暂时无法继续追到谁批准、谁获利、1919 账户为何参与。

## 指证 Whitfield

| 轮次 | 谎言 | 击破材料 | 玩家可得结论 |
|:---:|:---|:---|:---|
| R1 | 🟡 Rosa 在每日规定剂量之外又擅自增加药量 | 🟢 每日剂量卡 + 🟢 回收药瓶剩余量鉴定 | 药瓶实际消耗量与十三次规定剂量完全对应，没有额外注射所需的药量缺口 |
| R2 | 🟡 即使 Rosa 没有加量，也只是 Isabel 一例药物事故 | 🟢 五年七例儿童死亡对照表 + 🟢 批次封签对照 | 相同项目、相近症状和重复批次构成长期模式，不是孤例 |
| R3 | 🟡 医院从未批准或制度化使用问题批次 | 🟢 同批次药剂化验报告 + 🟢 采购与发放记录 | 医院明知批次属于低成本试用，仍把它集中发往无法反抗的家庭 |

### 指证中的退守逻辑

R1 被击穿后，Whitfield 无法再坚持 Rosa 擅自增加剂量。他先承认最小范围的事实，再把问题缩成单个家庭：

> Whitfield：“好，就算 Rosa 没有加量，也只能证明这一个家庭没有误用。”  
> “Isabel 仍然只是一次个别药物事故。”

- 🟡 R2 新谎言：Isabel 只是个别药物事故。

R2 被击穿后，Whitfield 不再否认类似死亡，却把责任推给无法在场说话的药品制造与采购程序：

> Whitfield：“相似病例不等于医院明知有问题。”  
> “医院只负责临床发放，采购部门从未批准过不合格批次。”

- 🟡 R3 新谎言：医院没有制度化批准或使用问题批次。

## 指证后

Whitfield 被迫承认慈善项目使用了未经正常标准验收的批次。他仍坚持自己只是按采购名单执行，并把上层授权归给已经封存的理事会程序。

Rosa 当场拒绝在“家属误用说明”上签字。Isabel 的死亡原因进入重审，圣心医院执行层遭到调查。

这场胜利必须是真的：Rosa 获救，Isabel 的名字不再只属于一张统计表，医院也确实失去继续甩锅的空间。

与此同时，玩家会留下三个问题：

- 为什么问题批次只进入底层儿童项目？
- 谁拿走了采购记录的第十九页？
- Mickey 为什么总能拿到恰好足够赢、却不能继续往上的材料？

## L2 到 L3：一次足够的铺垫

庭审结束后，法院走廊。

Morrison 靠在墙边抽烟。他看见 Whitfield 被警员带去补充笔录，又看见 Pierce 把医院原始采购箱直接送往封存车。

他叫住 Zack：

> Morrison：“Brennan。”  
> Zack：“怎么，又要说内部牵扯太多？”

Morrison 看了一眼 Mickey，又看了一眼 Pierce。

> Morrison：“……算了。”  
> Morrison：“别太相信法院里的胜利。”  
> Zack：“你到底想说什么？”  
> Morrison：“等我想好了再说。”

他离开时，法院书记员追上来，递给他一只密封通知袋。玩家看不见内容。

真相层：通知要求 Morrison 次日上午交出旧南区警务档案、配枪登记和全部私人办案记录，由 Pierce 接管。Morrison 刚看见系统把 Whitfield作为止损对象交出去，立刻认出自己也进入了同一程序。

这一层不在 L2 解释，L3 从 Morrison 桌上的拆封通知中回收。

## 玩家体验

💡 Mickey 没有骗玩家。他只是替玩家决定了今天应该赢哪一场。

L2 的满足感必须成立，阴影才会成立。玩家不是后悔救 Rosa，而是在后来意识到：一场善意且真实的胜利，也可以被更大的系统当作损失控制。

# Loop 3：最后一页（承转）

## 本轮功能

核心问题：Morrison 是畏罪自杀，还是在准备开口时被人改写成自杀？  
指证对象：Mrs. Morrison / Doris。  
指证轮次：3。  
本轮边界：玩家可以确认 Morrison 被杀、Mickey 在正式枪位证词上撒谎；不能确认 Mickey 就是 Whale，也不能仅凭本轮材料证明他亲手杀人。

本轮的认知翻转不是“Morrison 原来是好人”。他仍是黑警，也确实做过许多无法洗掉的事。翻转在于：

> 一个坏人准备说出真相时，系统仍然没有权把他的最后一页写成疯子、自杀者和炸弹犯。

### 为什么 Mickey 必须亲自动手

本轮信息只给到一半，L5 完整回收：

- 🟢 法院证物转运调度单：Pierce 和可用的黑警当晚都被锁在 Harrison / 圣心医院材料转运上。
- 🔵🟣 晚间报纸号外：银月会常用的西区执行人员当晚正被用于冲击一个工会互助站，数人被捕，街面力量无法临时回调。
- L5 的 Mickey 操作记录会写明：“法院封存夜；西区人手失联；无中间人。”

这三层一起说明：Morrison 突然决定当夜开口，留给 Whale 的窗口不足一小时。Mickey 不是为了戏剧效果亲自出场，而是他已经没有一个既能灭口、又知道如何伪造警务现场的中间人可用。

## 开篇剧情

**Zack 侦探事务所 / 夜**

Emma 正在整理 Whitfield 庭审材料，电话突然响起。人工接线员说 Morrison 宅邸有人坚持要接 Brennan。

Doris 的声音断断续续：

> “Thomas 倒在书房。”  
> “警局说车都在法院。”  
> “壁炉里有你的名字。”

电话随后断掉。

Zack 想起 Morrison 白天那句“别太相信法院里的胜利”，立刻带 Emma 赶往宅邸。

## 自由探索：爆炸前

### Morrison 书房

- 🟢 伪造遗书：把 Thomas 的死亡写成“畏罪自杀”，并预告他准备炸毁住所报复社会。字迹近似，但用词刻意粗暴，像在替后续爆炸预先解释。
- 🟢 配枪与弹药初检：现场枪不是 Morrison 登记配枪；弹药与 James 案米勒体系特制手枪属于同一供枪渠道，但枪号已磨损，不能据此认定是同一把枪。
- 🟢 古巴雪茄烟蒂：压在沙发后地毯缝里，右侧有不规则咬痕。本轮只记录“不是 Thomas 常抽的牌子”，牙痕到 L4 才激活。
- 🟣 左手生活痕迹：墨水瓶、警徽盒、酒杯垫都在书桌左侧。它只做环境铺垫，Thomas 的惯用手由 Doris 生活证词确认。
- 🟢 Pierce 档案移交通知：要求 Thomas 次日上午交出旧案文件和配枪登记，签发时间在 Whitfield 被带走补充笔录之前。

### 门厅与电话桌

- 🟢 外出准备：大衣、帽子和手杖已经放在门边，口袋里只有街车零钱；车钥匙仍在书房抽屉。Thomas 准备低调离开，不使用容易被 Whale 眼线认出的私人汽车。
- 🟢 两只酒杯与古巴雪茄灰：一只杯子有 Thomas 的唇印，另一只杯底只剩未喝完的酒；雪茄灰落在访客座位和电话扶手旁。
- 🟢 车站寄存柜领取凭条：夹在 Thomas 大衣内袋，编号和寄存时间都早于死亡。
- 🟢 壁炉里烧剩的 Brennan 信封角：Doris 在慌乱中只烧掉一部分，说明 Thomas 留下过明确指向 Zack 的书面内容。

### 煤气检修痕迹

- 🟢 伪造煤气检修卡：日期是当天下午，签名栏使用不存在的市政工号；Doris确认家里没有申报检修。
- 🟣 壁炉时钟后的新铜线、松动煤气阀、被布条压住的窗缝：Emma先闻到煤气味，再看见时钟秒针经过铜线接点。

## 突发事件：爆炸前的最后一眼

取得本场必需材料后自动触发。Pierce 赶到并要求停止搜查，Zack 指向尸体右手：“那就先记这一处。枪在右手，手指根本没有握住。”

Mickey 随后进门，也清楚看见枪位。Emma 闻到煤气，发现时钟后的铜线即将触发，立刻拉着两人撤离。爆炸摧毁书房，右手枪位只留下 Zack 的私人调查记录，没有形成正式现场照片。

Emma 不是凭直觉救人。玩家已经看过检修卡和异常布置，她只是在最后几秒把这些信息连了起来。

当下玩家感受到的是：有人要把现场和调查者一起清掉。L5 才意识到更冷的一层：清场程序也没有把 Mickey 排除在“可接受损耗”之外。

## 自由探索：爆炸后

### 夜班电话交换台 / 接线员

接线员只提供自己确实经手的线路，不再承担远距离目击功能。

- ⚪ 夜班接线员：22:36，Thomas 要接 Brennan 事务所；无人接听后，他说“那我自己过去”。
- ⚪ 夜班接线员：22:43，Thomas 改接 Donnelly & Associates，并要求转告 Mickey：“我手里有东西。你要是还想让这件事有个体面的收尾，就现在来。”
- ⚪ 夜班接线员：Donnelly 办公室接起后确认会转告；这通电话足以解释为什么 Mickey 当晚知道 Morrison 有材料，但不能证明他最终到场杀人。

### 法院外圈 / 调度材料

- 🟢 Harrison 案证物转运调度单：21:30 至 23:20，Pierce 负责法院东门与市政档案库交接，相关警员不得离岗。
- 🔵🟣 晚间报纸号外：西区工会互助站遭人冲击，参与者中有数名银月会常用打手；它当下只是城市新闻，L5 才与“无中间人”操作记录连上。

调度单解释了警局为什么迟到，也解释 Pierce 为什么不是最先发现现场的人。他在 23:20 后离开法院转运点，接到死亡报告才赶来。

## 指证 Doris

Doris 在爆炸后的临时问询室里迅速退回官方口径。她不是想替 Thomas 洗白，也不是参与灭口。她害怕两种人同时进入她的家：

- 警局会把 Thomas 写成疯子，用爆炸掩掉谋杀。
- Zack 会把 Thomas 写成最后一刻回头的英雄，让所有人假装一次开口足以洗掉二十年的黑警生活。

她想保护的是这个家的叙述权，也是她自己在这段婚姻里留下的生活。这个动机可以理解，但她的做法仍然让 Thomas 留下的真相差点被烧掉。

| 轮次 | 谎言 | 击破材料 | 玩家可得结论 |
|:---:|:---|:---|:---|
| R1 | 🟡 Thomas 当晚没有打算离开，只是回家休息 | 🟢 外出准备 + ⚪ 接 Brennan 未通的证词 | 他已经准备搭街车去找 Zack，不是在家等待自杀 |
| R2 | 🟡 Thomas 只打过电话，没有任何访客真正进屋 | 🟢 两只酒杯与雪茄灰 + ⚪ 接 Donnelly 的证词 | 死前确实有人进屋谈话；到访时间与 Thomas 召来 Donnelly 高度重合，但本轮仍不能只凭这两件材料锁定 Mickey |
| R3 | 🟡 Thomas 没有留下材料，也没有让我替他保管东西 | 🟢 车站寄存柜领取凭条 + 🟢 烧剩的 Brennan 信封角 | Thomas 已把备份存到外部地点，并明确留下给 Zack 的交接说明 |

## 指证后：Doris 的选择

Doris 交出藏在缝纫盒夹层里的寄存柜钥匙。她承认 Thomas 晚饭后把钥匙和信封交给她，说如果自己没回来，就烧掉信、留下钥匙，等她判断该相信谁。

她问 Zack：

> “你会把他写成什么？”

Zack 不替 Thomas赎罪，只回答：

> “写他做过什么。好的坏的，都写。”

这句话也提前照见 Zack 对 Emma 的问题：他说别人有权得到完整事实，但他自己还没有做到。

Doris 随后下意识纠正一名警员对现场的描述：

> “Thomas 用左手。写字、倒酒、擦警徽，一直都是左手。”

- ⚪ Doris 关于 Thomas 左撇子的生活证词。

## 警局翻供

爆炸后，Pierce 试图以“原始现场已毁、无法排除自杀”暂定死因。

Zack 没有只靠左撇子下结论。他把几层事实依次放到桌上：

- Thomas 已经穿戴好，准备去找 Zack。
- 他先后联系 Brennan 与 Donnelly，并接待了访客。
- 遗书提前替尚未发生的爆炸准备了口径。
- 枪在惯用左手相反的一侧，握姿僵硬。

Zack：

> “任何一件都可以解释。”  
> “但这些解释全都只朝一个方向方便：让一个准备出门的人，看起来像准备自杀。”

Pierce 没有正面回答，而是让记录员询问 Mickey：

> “Donnelly 先生，你在爆炸前也进入过书房。你第一次看见尸体时，枪在哪只手？”

Mickey 看见 Zack 摊开的调查笔记。他知道 Zack 和玩家都看见了右手错误，也知道爆炸让官方第一现场无法复核。

他停顿的那一秒不是被问倒，而是在决定亲手把 Zack 推开。

> Mickey：“左手。Doris 也确认 Thomas 是左撇子。”

Zack 抬头。

Zack 把调查笔记推过去：

> “你进门时看见的是右手。”

Mickey：

> “那是你写在私人笔记里的说法。”  
> “我只能陈述我看见的。”

Pierce 顺势把 Zack 的私人记录、爆炸后的原始现场损毁和 Mickey 的第三方正式笔录放到一起：两份相互冲突的个人陈述，不足以立刻推翻初步死因。这个说法不能证明 Morrison 自杀，却足以制造程序争议、拖延改判。

走廊里，Zack追上 Mickey：

> Zack：“你看见的是右手。”  
> Mickey：“Thomas 留下的东西已经在你手里。”  
> Zack：“你刚才帮他们把他写成自杀。”  
> Mickey：“我帮你保住了还能继续查的东西。你现在逼 Pierce 承认谋杀，他会把钥匙、笔记和所有旧案一起收回去。”

这句话最难受的地方是，它并不全是假话。Mickey 总能用一个真实风险包住自己真正想压下的事实。

Mickey 离开后，Emma 低声问：

> Emma：“他看见了右手。”  
> Zack：“我知道。”  
> Emma：“那他为什么说左手？”

Zack 看了一眼手里的寄存柜钥匙：

> “也许是在替我们拖时间。也许是在替别的人。”

Emma：

> “寄存柜，要叫他吗？”

Zack：

> “不。Morrison 留下的东西，我们先看。”

## L3 结尾：只打开门，不读取答案

Zack 和 Emma 来到车站寄存区。钥匙能打开其中一只柜门。

柜里能看见两只封套的标题：

- `给 Brennan：别让他们替我写完。`
- `Lakeshore / O'Hara。`

远处车站钟声响起，Loop 在 Zack 把两只封套取出来时结束。

L3 不在这里继续前往 O'Hara 家，也不重复读取寄存柜。下一轮从打开封套开始。

## 玩家体验

💡 Mickey 不是判断错了，也不是记忆模糊。他在公开程序里选择了一个能让真相失效的位置。

玩家离开 L3 时应当确认：

- Morrison 不是在等待自杀，他原本要去找 Zack。
- 有一名与 Donnelly 联系有关的访客在死前到过宅邸。
- 爆炸是提前布置的清场，不是 Morrison 临死报复。
- Mickey 在最关键的枪位事实点上主动撒谎。
- Morrison 留下的东西表明 O'Hara 将是下一次警务协助清退的目标。

玩家仍不能确认：

- Mickey 就是死前访客。

# Loop 4：她藏起来的不是一封信（转）

## 本轮功能

核心问题：Margaret 为什么明知 O'Hara 拒绝出售、下一次复访会有警察到场，仍劝她接受报价、搬走，并要求她不要找 Zack？  
指证对象：Margaret。  
指证轮次：3。  
本轮边界：玩家可以高度怀疑 Mickey 就是 Morrison 的访客、Donnelly & Associates 与 1919 账户有关；不能仅凭童年记忆永久锁定 Whale 身份。

本轮要让过去通过一件正在发生的事重新进入现实：O'Hara 不是为了替 Patrick 和 Mickey 提供背景而出现，她此刻正因拒绝出售房屋而受到威胁。Margaret 的旧创伤因此变成一个当下选择。

## 开篇：读取寄存柜

**车站寄存区 / 清晨**

L4 直接承接 L3 最后一个画面。Zack 和 Emma 把两只封套放到候车长椅上，不再重复开柜过程。

### 封套一：Morrison 写给 Zack 的未寄出口供

Thomas 没有请求原谅，也没有把自己写成英雄。他只写清三件事：

- 他替 Pierce 和 Whale 压过旧案，知道自己没有资格要求 Zack相信。
- Whitfield 被交出去后，他收到同样的“交出文件、接受接管”通知，才知道自己已经进入下一批清理。
- 他不想让警局把自己的最后选择写成自杀、疯狂或报复社会。

最后一句：

> “别替我洗干净。也别让他们替我写完。”

### 封套二：Lakeshore 清退协警通知

- 🟢 Lakeshore 清退协警通知：O'Hara 家被列为第一批核心拒售户；Lakeshore 将在次日上午再次上门，要求辖区警员到场“维持民事秩序”。Thomas 之所以拿到它，是因为这本来就是发给警局的协查副本。
- 文件备注：“住户已两次明确拒售，第三次接触按重点户处理。”

Thomas 不是因为认识 O'Hara 才选中这张纸。他把 Pierce 要求移交的当前协警文件一并备份，而这张排在最前面，时间就在第二天早晨。

行动目标成立：先赶在警察与 Lakeshore 之前找到 O'Hara，确认她为什么拒绝出售，也确认是谁一直劝她不要找 Zack。

## O'Hara 家：当下危机

O'Hara 家不是已经搬空的空屋。她仍住在这里，也明确拒绝出售，是第一批清退名单里最难处理的一户。

O'Hara 是 Brennan 家多年的旧邻居，Zack 小时候习惯叫她 Aunt O'Hara，但两人没有血缘关系。

门口没有打包好的行李，Lakeshore 送来的纸箱仍折叠着靠在墙边。O'Hara 不是在等待更高报价，她根本没有答应出售。

她看见 Zack 时先叫了一声“Zackie”，随后因为门外还有 Lakeshore 的复访人员而改口叫“Brennan 先生”。这不是卖亲情，而是让玩家知道：她本来就认识并信任 Zack，却被 Margaret 劝住了。

### 自由探索 / O'Hara

- 🟢 Lakeshore 最终收购协议：报价明显偏低，限四十八小时签署；O'Hara 在签名页写下“不出售”，整份协议仍压在桌上。
- ⚪ O'Hara：她明确告诉过 Margaret 自己不会卖，也把 Lakeshore 威胁下次带警察上门的事说了。Margaret 仍劝她接受报价、先搬走。
- 🟢 Margaret 写给 O'Hara 的短便条：“把报价签了，先搬来我这里。别找 Zack。他一碰这件事，就不会只替你搬一次家。”便条有 Margaret 签名。
- 🟣 折叠未用的搬家纸箱：Lakeshore 提前送来，但 O'Hara 没有打开。她不是为了抬价拖延，而是拒绝出售。

O'Hara 的立场要清楚：

> “我不是在等他们多加几个钱。这房子不卖。他们明天带警察来，答案也一样。”

## Margaret 家：主动整理过去

Margaret 的家里也在打包，但她没有被 Lakeshore 清退。她原本就考虑搬去北边教会附近的小公寓；听说 O'Hara 仍准备找 Zack 后，她突然提前打包 Patrick 与 Zack 的旧物。

因此搬家不是巧合，也不是行政迫迁。它是 Margaret 在风险逼近时作出的防御动作：把东西封箱、把 O'Hara 送走、在 Zack 到来前重新把旧故事收回去。

门厅里有 O'Hara 刚用过的茶杯。Margaret 看见 Zack 手里的便条，第一句话不是问 O'Hara 是否安全，而是：

> “她还是去找你了？”

### 自由探索 / Margaret 家

Zack 是来问 Margaret 为什么逼 O'Hara 接受报价、又为什么阻止她求助。谈话发生在半开的搬家箱之间，Patrick 的旧物已经被 Margaret 主动翻出、准备封存，因此所有用于指证的材料都能在指证前出现。

- 🟣 搬家标签：最初日期是下个月，后来被改成三天后；改期时间正是 O'Hara 再次拒绝 Lakeshore 的第二天。
- 🟢 1903 南区码头旧照片：Patrick、Liam、少年 Mickey 与工人们站在货栈前。
- 🟣 1908 年警局合影：Patrick、年轻 Morrison、少年 Mickey 同框。它只负责说明这些人长期处于同一张关系网，不单独产出推理结论。
- 🟢 Patrick 葬礼册与 1919 旧信封：信封抬头为 Donnelly & Associates，日期在 Patrick 去世后不久。
- 🟢 Zack 童年练习本：其中一页被整齐撕去，只剩“For the many...”开头和半截撕痕。
- 🟢 与练习本吻合的残页：夹在一份折起、尚未展开的 Patrick 工会备忘中，撕口、墨迹和纸张纤维完全吻合；残页补全“For the many, we pay the few”。
- 🟢 Margaret 未寄出的短笺：与残页放在一起，写着“我不能再把一个孩子交给一场事业。我宁可他恨 Patrick，也不能让他学会替死人说值得。”

注意：玩家在指证前可以看到残页和短笺，但折起的工会备忘正文仍未展开。玩家知道 Margaret 有意藏住这句话，却还不知道 3 人死亡与 800 名工人的完整事件。

## 记忆锚点玩法

本轮记忆必须明确视角来源，不能让一张照片凭空提供 Zack 不可能知道的事实。

### 锚点一：1903 旧照片 → Zack 的童年碎片

Zack 当年年纪很小。他不记得码头清场本身，只记得清场后的 Brennan 家：少年 Mickey坐在门口，嘴角全是血，右下牙缺了一角，一直问 Liam 什么时候回来。

- 🧪 记忆观察：Mickey 右下牙有长期缺损。
- 玩家把观察点与 🟢 Morrison 现场古巴雪茄烟蒂连线。
- 🧪 派生结论：烟蒂右侧不规则咬痕与 Mickey 旧牙伤位置高度吻合。

它只能说明 Morrison 死前访客极可能是 Mickey，不能作为单独定罪材料。

### 锚点二：1919 葬礼册 → Zack 的明确记忆

Patrick 葬礼后，Mickey 对 Zack 说：

> “他没做完的，我会接着做。”

同一时期的 Donnelly & Associates 信封把这句话从单纯情感承诺变成一条可能的组织与法律线索。

- 🧪 记忆观察：1919 是 Mickey 接手 Patrick 关系与事业的起点。
- 玩家把它与 🟢 Harrison 资金流向图中的 1919 D&A 中间账户连线。
- 🧪 派生结论：D&A 很可能不是普通年份账户，而是 Donnelly & Associates 的长期法律接口。

### 1908 合影的处理

1908 合影不再生成“少年 Mickey 学会观察关系网”这种无法验证的心理结论。玩家只从画面中看见：Mickey 很早就认识 Patrick、Morrison 与警局人物。它是故事信息，不进入指证。

## U3 公文包复审：Emma 与 Zack 的小吵架

这场戏不再突然插入。触发原因是 Emma 看见残页上的“For the many, we pay the few”，发现它与 Mickey 在 U3 劝 Zack 越过程序线时使用的措辞高度相似。

她和 Zack 回到门厅，从 Zack 公文包里取出 Mary / Helen 旧案，想确认 Mickey 当时到底怎样一步步引导 Zack。

复审时，Emma 看见 U3 结尾才形成的最终时间线：Mary 与 Helen 在 19 号已提前布置天台，Mary 负责不在场与说辞，Helen 负责涂油、破坏护栏与灌酒配合。

她先发现事实：

> “Mary 是蓄意的？”

然后才看到文件日期，确认 Zack 是在报道发出后、U3 结束前得知。这场争吵因此不是“你骗我写了假报道”，而是：Zack 后来知道了完整事实，却让 Emma 继续停在比较容易承受、也比较方便救人的版本里。

> Zack：“Thomas 确实准备杀她。”  
> Emma：“我现在问的不是 Thomas。”  
> Emma：“所以不是我发稿前你就知道。是我发稿以后，你知道了，然后决定我不需要知道。”  
> Zack：“那篇报道已经救了她。”  
> Emma：“你救了她以后，替我决定我救下的到底是谁。”

Emma 不要求 Zack 此刻推翻 Mary 案，也不否认报道救了人。

> Emma：“我理解你为什么这么做。”  
> Emma：“别把理解当成同意。”

两人没有彻底吵崩。他们带着这道尚未解决的裂缝回到 Margaret 面前，使接下来的指证不只是查 Patrick，也是 Zack 被迫看见自己正在重复家族习惯。

## 指证 Margaret

### Margaret 的防御逻辑

Margaret 的核心不是反对正义，也不是希望 O'Hara失去权益。她见过 Patrick 把“不能不管”变成一条永远不会结束的路，也见过 Mickey 为了让 Liam 的死有意义而接受同一种语言。

她判断 O'Hara 接受报价、尽快离开至少能活下来，也判断 Zack 一旦接手就会一路追到 Lakeshore 和 Whale。她的错误不是想保护两个人，而是相信自己有权分别替 O'Hara 和 Zack 决定他们应当放弃什么、知道多少。

| 轮次 | 谎言 | 击破材料 | 玩家可得结论 |
|:---:|:---|:---|:---|
| R1 | 🟡 O'Hara 已经准备接受报价，我只是在帮她安排搬家 | 🟢 写有“不出售”的最终收购协议 + ⚪ O'Hara“明确告诉 Margaret 不会卖”的证词 | Margaret 知道 O'Hara 从未同意出售，仍在替她决定离开 |
| R2 | 🟡 我没有让她瞒着你，是她自己不想麻烦你 | 🟢 Margaret 写给 O'Hara 的短便条 + ⚪ O'Hara 关于 Margaret 劝阻求助的证词 | Margaret 明确要求 O'Hara 不找 Zack，并主动控制求助路径 |
| R3 | 🟡 我撕掉练习本，只是不想让一个孩子恨自己的父亲 | 🟢 练习本与吻合残页 + 🟢 Margaret 未寄出的短笺 | 她真正害怕的不是 Zack 恨 Patrick，而是 Zack 理解并继承 Patrick 的“值得”逻辑 |

收束：

> Zack：“你不是怕我恨他。”  
> Margaret：“我怕你听完以后，觉得他是对的。”

## 指证后：核心记忆才开放

Margaret 不再替 Zack 管理过去。她亲手展开夹着残页的工会备忘，并补全自己亲历的部分。

- 🟢 Patrick 旧工会备忘：3 dead / 800 workers / 14h → 10h。
- ⚪ Margaret 1903 当晚证词：Patrick 说三个人的死亡换来了八百个工人的工时改善，并说 Liam 如果知道结果，也会觉得值得。
- 🔒 Margaret 的记忆：她记得的不是“三个人”，而是 Mickey 坐在 Brennan 家门口，等一个永远不会回来的父亲。

🎮 核心叙述记忆：

这不是 Zack 自己的回忆，而是由 Margaret 叙述、玩家在固定片段中观察的过去。玩家看见三件事同时成立：

1. Patrick 没有故意害死 Liam，且确实为八百个工人争取到十小时工时。
2. Patrick 在胜利后替 Liam 说了“值得”，把一个具体死者放进一张结果正确的账里。
3. Mickey 为了不让父亲白死，先逼自己接受这张账，后来又不断制造新的账来证明第一笔牺牲有意义。

💡 情感尤里卡：

> Mickey 不是不痛。  
> 他是太怕 Liam 白死，所以不断要求后来每一个死者替 Liam 的死证明“值得”。

Margaret 的成长动作不能只是交材料。她当着 Zack 的面给 O'Hara 打电话：

> “别签。”  
> “Zack 已经知道了。接下来让他自己决定要不要管，也让你自己决定要不要留下。”

她第一次把决定还给两个人。

## L4 结尾：四十二层邀请

Margaret 家电话响起。Mickey 直接找 Zack。

他没有承认身份，只说：

> “我听说你翻出了 Patrick 的箱子。”  
> “如果你准备把我的名字写到 Whale 旁边，别靠一支雪茄和一段童年记忆。”  
> “午夜前来四十二层。把你认为能证明我的东西都带上。”

Zack 问他为什么。

> Mickey：“因为怀疑一个朋友很容易。”  
> “知道以后还愿不愿意承担，比较难。”

他停了一下，又补充：

> “带 Emma 来。”

这句话既像挑衅，也像 Mickey 已经知道 Zack 会替 Emma 决定什么该听。

L4 至此形成进入 L5 的明确理由：不是 Zack 无授权闯入朋友办公室，而是 Mickey 主动发出一场带条件的对质邀请。

## 玩家体验

💡 L4 的核心不是“世界突然变了”，而是旧信息获得了新含义。

- Morrison 的烟蒂不再只是陌生访客痕迹。
- 1919 不再只是账户年份。
- Patrick 的工运胜利不再只是家庭荣誉。
- Zack 在 U3 的保护也不再只是一次善意隐瞒。

玩家离开 L4 时应当处于：

> 我几乎知道 Whale 是谁。  
> 但我仍然要亲手证明，而不是让回忆替我定罪。

# Loop 5：四十二层（合）

## 本轮功能

核心问题：Zack 能否在 Mickey 承认之前，先用玩家亲手取得的材料证明 Mickey = Whale？  
指证对象：Mickey。  
结构：身份摊牌 + 4 个控辩阶段。  
本轮边界：可以坐实 Mickey 的身份、Miller → Lakeshore → Tidewater 结构和 Sean 的谋杀疑云；不能解开 Sean 的完整执行链、小 Charles 的越权投毒、Lawson 身份与 U5 嫁祸。

## Mickey 为什么主动邀请 Zack

L3 爆炸后，Mickey 从一张内部清场副本中发现：清场命令写的是“现场人员不作区分”。Miller 系统明知他可能被 Pierce 叫回，仍然允许爆炸按时发生。

这没有让 Mickey 立刻悔悟，却摧毁了他的一层自我想象：他一直认为自己是负责计算代价的人，第一次明确看见系统也把他列在“可接受损耗”里。

同时，Morrison 现场的右手错误已经暴露，Zack 又在调查 Patrick 旧物。Mickey 判断自己的 Whale 身份很快会被 Miller 先行清理。

因此四十二层不是一次粗心失窃，而是 Mickey 有意安排的最后局面：

- 他把可以指向自己的材料留在办公室，让 Zack 有机会证明，而不是只听他承认。
- 他把 Miller 档案作为自保保险，也作为自己可能死亡后的反击。
- 他还在测试 Zack：看见这些成果和代价后，Zack 会审判他、理解他，还是接替他。

这个动机依然带着 Mickey 的老毛病。他即使准备交出真相，也要设计别人取得真相的方式。

## 开篇剧情

**四十二层 / Donnelly & Associates / 午夜前**

Zack 和 Emma 按邀请来到四十二层。外间没有职员，门锁已经被人为留在开启状态。桌上放着一张只有一句话的卡片：

> “怀疑不是证明。你教过我的。”

这是 Mickey 转述 Patrick 曾对孩子们说过的话，也是在要求 Zack 按侦探的方式完成最后一步。

电话线路仍接着市政厅，听筒旁有一张催 Mickey 回电的便签。电梯指示灯停在楼下，说明 Mickey 不在房间里，但随时可能返回。

Emma 明确指出：

> “他让我们进来，不等于他会让我们带着东西出去。”

行动目标成立：在 Mickey 回来之前，把高度怀疑转化成三条彼此独立的证明链。

## 自由探索

### 外间办公室

- 🟣 未挂断的市政厅电话：对方只反复要求 Donnelly 先生尽快回电，不透露内容。
- 🟣 当日预约簿：午夜前没有正式客户，说明 Mickey 特意清空了办公室。
- 🟣 电梯层数指示器：稳定制造搜索窗口压力，不承担证据功能。

### Mickey 办公室 / 法律壳

- 🟢 Donnelly & Associates 法人卷宗：法人代表 Michael F. Donnelly，1919 年开始持续经营；业务修订范围逐步加入银行重组、医院理事会、地产接收和市政争议。
- 🟢 Mickey 日常工作笔记：与保险柜内操作记录使用相同缩写、日期格式、墨水和修改习惯，但正文只记录正常律师事务。
- 🟢 Mickey 的半支古巴雪茄：放在他自己的烟灰缸里，右侧有稳定的不规则咬痕。
- 🟣 工人互助会旧照片：Mickey 多年来确实资助过工人法律援助，不把他所有公共行为写成伪装。

🎮 法律壳连线：

玩家把法人卷宗与 L1 Harrison 资金图中的 1919 D&A 账户连接，产出：

- 🧪 派生结论：Harrison 资金链的 1919 中间账户由 Mickey 长期控制的法律实体承接。

这证明 Mickey 控制资金接口，但单独还不能证明他就是 Whale。

### 字母锁保险柜

保险柜使用七位字母锁。办公室里没有把完整密码直接写在桌面。

玩家已经从 L4 取得被撕下的完整句子：

> For The Many We Pay The Few

取每个词首字母，得到 `FTMWPTF`。

这个门锁不是普通谜语，而是主题动作：玩家必须亲手输入 Mickey 用来解释自己一生的句子，才能看见那句话实际保存了什么。

## 保险柜内容

### 1. 🟢 1925 年内部接口接管记录

这份文件把 Mickey 的两段人生明确分开：

- 1919 年成立的 Donnelly & Associates 是法律壳，负责合同、产权、银行重组和诉讼拖延。
- 1925 年，Mickey 才正式接管内部代号 `W`，职责栏写着“警务协调 / 银月会联络 / 非合同障碍处理”。
- 接管签批来自 Tidewater 上层项目办公室，抄送 Lakeshore 风险委员会；Miller 名字仍不直接出现在任命正文里。

玩家可以由上下文确认 `W` 就是 Whale，但这份记录的更重要作用是守住人物 chronology：Mickey 不是从少年时期起就以 Whale 身份杀人。他先用六年时间经营法律壳、告诉自己在接近系统核心，直到 1925 年才主动接过暴力接口，并逐渐成为系统的一部分。

### 2. 🟢 编码操作簿

它不叫“Whale 指令笔记本”，也不会像自白书一样写满犯罪名称。封面只是普通案件编号，内页使用首字母、时间、条件和后续处理栏。

玩家能读取两组关键页：

**H 页：**

- “第三次拒绝撤回。”
- “P：法院东门。”
- “保留医院线，B 可见；1919 不可见。”
- “完成后转司法封存。”

**T.M. 页：**

- “Brennan line failed。”
- “法院封存夜；西区人手失联；无中间人。”
- “右手错误。”
- “正式笔录修正为左手。”
- “清场自动执行；现场人员不作区分。”

这些句子不会单独告诉玩家全部答案。玩家必须用 L1 的 Harrison 第三次改判、L2 的医院线、L3 的调度单、玩家亲眼见到的右手枪位与爆炸装置逐项对应。

🎮 操作记录复原：

- 🧪 Harrison 页结论：Harrison 的第三次拒绝撤回触发清理；执行者有意给 Zack 留下医院材料，把调查控制在可牺牲的 Whitfield 层。
- 🧪 Morrison 页结论：写记录的人知道 Brennan 电话未接、没有中间人、右手摆错、警局改口和自动爆炸。这不是事后听说，而是行动控制者的视角。

再把操作簿书写习惯与 Mickey 日常工作笔记连接：

- 🧪 书写者一致性：缩写、改写符号、日期顺序和墨水批次一致，操作簿由 Mickey 长期书写。

### 3. 🟢 Mickey 功业簿

这不是给法庭看的账，而是 Mickey 私下用来证明自己没有白做的记录。它保留真实成果，也保留被他压成代号的人：

- O'Brien：清理；罢工网络保留。
- Webb：同时持有 Lawson 私人材料与 Whale 勒索记录；清理；银月会接口稳定。
- Moore：允许银行执行层让步；Frank 线可暴露；Lakeshore 不进入法庭记录。
- Harrison：清理；圣心医院执行层可控暴露。
- Morrison：亲自处理；身份暂保；右手错误。
- Rosa：免于定罪；医院项目终止。
- Mary：脱离死刑风险；Brennan 完成越线测试。

Liam 的旧工牌夹在封面内侧。Mickey 不是忘记了父亲，而是每次翻开功业簿都让父亲的遗物替后面的账背书。

玩家应该在这里产生复杂感受：Rosa 和 Mary 的结果没有造假，Mickey 确实帮过人；同一本簿子也把 Morrison、Harrison 和 O'Brien 写成了清理成本。

玩家也终于能回看前三章：

- U1 的 Webb 不是因为勒索几个酒吧客人而死，而是同时碰到了 Lawson 与 Whale 两条接口。
- U2 Mickey 能逼 Moore 让步，不只是律师口才，而是他有权决定牺牲银行执行层、保护 Lakeshore。
- Webb、Harrison、Morrison 不是三起互不相关的死亡：一个掌握了跨接口材料，一个拒绝继续充当司法工具，一个准备带走旧警务文件。他们都在从“可利用”变成“不可控制”时被清理。

### 4. 🟢 Miller 保险档案

这是 Mickey 为防止 Miller 清理自己而留下的保险，不是他原本准备交给警方的自首材料。

**1912 档案：**

- Tidewater 地基材料替换单：承重材料在事故前被主动降级。
- Sean O'Malley 特殊处置页：要求将 Sean 与工人队伍分开、“在事故记录完成前处理其异议”，有老 Charles 签名，但不直接写杀人命令。
- 结果：Sean 的死亡从普通工伤变成高度明确的谋杀疑云；完整执行人与弃尸过程仍留 U5。

特殊处置页右上角有 Miller 事故基金的压印，与 L2 病房铭牌、Emma 奖学金和 Foster 研究资助使用同一枚印章。Emma 第一眼认出的不是 Charles 签名，而是这枚陪伴自己长大的标记。

> Emma：“他们用这枚印章替我交学费。”  
> “也用它批准处理我父亲。”

这时不能让角色说“慈善全是假的”。病房、奖学金和仪器都真实存在；更痛的事实是，同一套权力一边真实救人，一边把另一些人写成可以处理的风险。

**1928 档案：**

- 南区综合商业开发计划执行名单：O'Hara 是第一批核心拒售户，标注“优先处理”。
- 南区执行职责页：湖滨信托银行 / Moore 负责制造债务与违约；Lakeshore 负责资金、估价和项目审批；Tidewater / Lawson 团队负责接收房产与推进工程；Mickey / Whale 负责法律障碍并协调 Pierce、银月会；Miller 项目委员会控制投资方向、期限和“可接受损耗”。
- 项目收益页只写预期地价、开发资格与 Miller 项目委员会的最终审批权，不出现实际回款或分赃账户；钱最终如何分配仍留给 U5 的 Moore / Lakeshore 账目坐实。
- 后插入的水源维护附页：纸张批次、日期格式和签批顺序都与原计划不同；在尚未报告污染前，提前安排 O'Hara 私人井与两个公共水点“封闭、清洗、重新开放”。
- 边界：附页足以证明有人越过原本以债务和房产接收为主的节奏，追加了水源层面的过激方案；没有投毒物名称、银月会执行人、小 Charles 签名或正常的老 Charles 审批。

### 5. 🟢 自动清场副本

这张纸是 Mickey 发出四十二层邀请的直接诱因。

- 清场条件写明：“Morrison 现场在 23:35 自动处理。”
- 人员栏只写：“现场暴露可接受。”
- 备注显示 Pierce 知道 Mickey 可能被要求返回，却没有暂停清场。

它证明 Miller 系统并没有把 Mickey 当成不可替代的核心。必要时，Whale 也只是 The Few。

## Mickey 返回前的证明状态

| 证明链 | 玩家完成的连接 | 能证明什么 |
|:---|:---|:---|
| 法律壳与身份演变 | 1919 D&A 账户 + 法人卷宗 + 1925 接管记录 | Mickey 先控制法律资金接口，后主动接管 Whale 的警务 / 银月会 / 非合同障碍职责 |
| 命令源 | 编码操作簿 + 各 Loop 已知事实 + 书写者一致性 | Mickey 掌握 Harrison 与 Morrison 行动的内部视角，并亲自书写后续修正 |
| Morrison 访客 | L3 烟蒂 + L4 牙伤记忆 + Mickey 半支雪茄 | Morrison 死前抽古巴雪茄的访客高度锁定为 Mickey |
| 系统层级 | Miller 保险档案 + 自动清场副本 | Mickey 是 Whale 和执行接口，但不是最高指挥者，也随时可以被系统牺牲 |

Mickey 进场前，玩家已经能自己得出身份结论。接下来的承认只确认人物选择，不替玩家补证据。

## 身份摊牌

电梯到达四十二层。Mickey 走进办公室，看见保险柜已经打开，没有惊讶。

Zack 依次摆出法人卷宗、复原后的操作记录和牙痕比对。

> Zack：“1919 的账户是你的。”  
> Zack：“Harrison 第三次拒绝撤回后，你给我留下医院线。”  
> Zack：“Morrison 电话没接、现场摆错右手、警局改成左手，这些都在你的记录里。”

最后，他把两支雪茄放到一起。

Mickey 不再否认：

> “是。我是 Whale。”

这句承认不是反转，而是玩家完成证明后的情感落锤。

Mickey 随后确认职责页的含义，但不提供 U5 才能落地的分赃数字：

> “Moore 让人欠债，Lakeshore 给债务和估价披一层投资外衣。”  
> “Lawson 的团队接走房子。文件能解决的交给律师，文件解决不了的交给 Pierce 和银月会。”  
> “我负责让这两种方法之间没有缝。”  
> “Miller 不需要在每一页签字。他们只决定期限、方向，以及可以损失多少人。”

至此玩家才能准确理解：Mickey 不是受黑帮胁迫的外部律师，也不是银月会老大；Whale 是 Tidewater 体系内部连接法律、警察与地面暴力的接口。

## Mickey 控辩：4 个阶段

### 阶段一：身份不是全部

Mickey 的论点：

> “证明我是 Whale，只证明我做过这些事。”  
> “它不证明这些事没有救人。”

他主动列举 Rosa、Mary、圣心医院项目终止和工人法律援助。Zack 不能说这些成果是假的，因为那会削弱整个主题。

> Mickey：“病房是真的。你的大学也是真的。”  
> Emma：“所以你们才总觉得，给过一些东西，就有权拿走另一些。”

玩家查看功业簿，第一次看到“救下的人”和“被清理的人”处于同一种表格格式。

Zack 只问：

> “Rosa 知道她获救的代价是谁吗？”  
> “Harrison 和 Morrison 有机会说自己愿不愿意付吗？”

本阶段不争论结果大小，只把问题从“结果有没有发生”转成“谁获得了替别人签字的权力”。

### 阶段二：你也做过同样的事

Mickey 的论点：

> “Mary 等不起法院。”  
> “你签了不该签的文件，知道真相后也没有告诉 Emma。”  
> “你不是反对我的方法。你只是反对这次由我来算。”

这一击必须成立。Zack 不以“我是好人”退守，也不能靠 Emma 替他辩护。

Zack：

> “我做过。”  
> “而且我差点把救到人，当成我有权继续隐瞒的证明。”

Emma 没有原谅式地打断，只重复 L4 的边界：

> “理解不是同意。救到人，也不是下一次替别人决定的许可证。”

Zack 与 Mickey 的区别不被写成天生善恶，而是 Zack 仍能承认行为需要被别人质问。

### 阶段三：系统需要我

Mickey 的论点：

> “Miller 还在。Lakeshore 和 Tidewater 还在。”  
> “我停下以后，谁留在里面把档案带出来？谁替下一个 Rosa 留一条路？”

Zack 出示自动清场副本。

> Zack：“他们知道你可能回到 Morrison 家。”  
> Zack：“爆炸没有为你停。”

Mickey 第一次失去现成回答。

> Zack：“你一直以为自己是写名单的人。”  
> “在 Miller 那里，你也只是名单上的一格。”

这是 L3 爆炸的真正回收。Mickey 的悲剧不是善意全假，而是他为系统训练出的算法最终也毫不犹豫地计算了他。

### 阶段四：Liam 与 Patrick

Mickey 把 Liam 工牌从功业簿中取出来：

> “三个人，八百个人。”  
> “一个病房，一整个街区。”  
> “Patrick 当年也算过。只是他赢了，所以你们叫他 Patrick Brennan。”

Zack 不否认工时改善：

> “我父亲错的地方，不是他想救八百个人。”  
> “是他替 Liam 说了‘值得’。”

Zack 把功业簿翻到 Morrison、Harrison 与 O'Brien：

> “你也不是在延续 Liam 的遗愿。”  
> “你是在逼后来每一个死去的人，都替 Liam 的死证明 Patrick 当年算对了。”  
> “你替他们签字，再拿结果告诉自己，他们会同意。”

💡 终极尤里卡：

> Mickey 不是不知道自己杀了谁。  
> 他是把每一个“谁”改成代号，好让第一笔牺牲永远显得有意义。

## 坠落

Mickey 走到窗边。窗外不是抽象黑夜，楼下已经停着 Miller 系统的黑色轿车。他知道即使没有 Zack，自己也很难带着档案离开四十二层。

他没有彻底忏悔，也没有说自己所有选择都错。他只是第一次无法确定自己一直代替死人给出的答案。

> Mickey：“Liam 呢？”  
> “Patrick 呢？”  
> “他们会觉得我至少做对过一点吗？”

Zack：

> “我不知道。”  
> “你也不知道。”

这正是 Mickey 一生不肯接受的答案：没有人可以替死者补签同意。

Mickey 退上窗沿。Zack伸手。Mickey 看着那只手，没有抓住，松开了窗框。

开场倒叙在这里闭环：Zack伸出去的不是推人的手，而是一只来不及被接受的手。

## 坠落后的第一次救赎

Zack站在打开的窗边，盯着自己刚才没能抓住 Mickey 的手。他最痛的不是被骗，而是 Mickey 证明了他们确实有相似之处。

Emma 没有立刻说“你们不一样”。她先把窗关上，让 Zack 不再站在 Mickey 最后选择的位置。

然后她说：

> “被他利用，不等于选择成为他。”  
> “但你接下来怎么做，才会决定你是不是。”

这不是替 Zack 宣判无罪，而是把选择重新交给他。

## 玩家体验

💡 抓住了 Whale，却没有抓住制造 Whale 的机器。

玩家应该同时得到满足和失重：

- Mickey 的身份是自己证明的。
- Mickey 的善意和成果没有被抹掉。
- 这些成果仍然不能替他的受害者签字。
- Mickey 死后，Miller、Lakeshore、Tidewater 与已经启动的清退计划仍然存在。

# Unit4 → Unit5：第二次没有说完

## 断点功能

Mickey 坠落不是 U4 的真正结尾。真正的结尾是 Zack 刚刚拒绝 Mickey 的算法，却因为害怕 Emma 成为下一个知情死者，再次选择先控制信息。

这次错误必须写得公平：玩家不会因为“告诉 Emma 真相”本身受到惩罚。危险来自地点与时机；Zack 的道德错误发生在两人已经离开监视范围之后。

## 楼下：安全选择与 Bad Ending

Zack 和 Emma 带着 Miller 保险档案离开大楼。街对面黑色轿车的车窗半降，里面的人持续观察 Emma。楼门附近还有一名假装读报的人盯着档案袋。

此处沿用 `Unit4_Emma档案BadEnding优化方案.md` 已确定的选择文字：

- **给 Emma 看档案** → Bad End“过早的警告”：Emma 会立刻采取正确行动，但监视者也会确认她已掌握行动情报，并沿她的警告路线找到 O'Hara、Sarah 与材料。
- **先把档案收起来** → 主线。

这个选择表达的是“不要在敌人面前公开情报”，不表达“Emma 不应该知道”。

## 离开监视后：Zack 的第二次选择

Zack 先带 Emma 摆脱第一辆跟踪车。他联系 Watts，不是因为 Watts 突然变成行动队成员，而是因为 Harrison 曾把法院钥匙和自首包托付给他；Watts 能调用一辆不显眼的法院档案车，也知道怎样从市政档案通道离开监视区域。

三人在一处法院旧档案室短暂停留。此时外部危险已经暂时解除，Zack 有机会把档案完整交给 Emma。

他翻到三页：

- Sean O'Malley 特殊处置页。
- O'Hara 第一批核心拒售户页面。
- 尚未报告污染就提前安排的私人井与公共水点维护页。

Emma问：

> “你看到了什么？”

Zack 先看见 Sean 的名字，又想起 Webb、Harrison、Morrison 和 Mickey 都死在“知道太多”之后。他把 Sean 页与水源附件压回档案，只抽出 O'Hara 页面。

> Zack：“O'Hara 被列为第一批优先处理。”  
> “天亮后，我们借 Watts 的车去找她。”

这句话是真的，但不完整。

第一次在街上收起档案是正确的安全判断。现在继续不说，是 Zack 主动重复旧错误：他没有邀请 Emma 一起判断风险，而是先替她决定哪一页可以知道。

次日清晨，三人乘法院档案车前往南区。路程约五十分钟，跟踪者已经被甩开。Zack 在这五十分钟里仍然没有提水源附件和 Sean。这段持续沉默让他的行为明确越过“战术延迟”，进入替 Emma 管理真相。

## O'Hara 与水源事件

### 真实时间边界

小 Charles 的越权行动已经在 Mickey 坠落前一晚开始执行。O'Hara 私人井是最早被处理的测试点；两个公共水点安排在次日清晨继续“维护”。Zack 等人抵达前，第一处公共水点已经短暂开放，第二处仍来得及封闭。

因此，即使 Zack 离开监视后立刻说出全部内容，也未必来得及救下已经饮水的 O'Hara。但完整共享能让 Emma、Watts 一起更早看出这不是一户人的危险，并在五十分钟路程中先安排警员封闭公共水点。

### 到达 O'Hara 家

Zack、Emma 和 Watts 乘法院档案车抵达。Watts 同行有两个明确作用：提供未被监视的交通路线；用 Harrison 留给他的法院身份联络辖区警员封闭后续水点。他不是参与情感对峙的人。

- 🟣 院门没有锁，未封的搬家箱仍放在门边。
- 🟢 O'Hara 已经倒在厨房死亡；水杯和烧水壶里有相同异常沉淀。
- ⚪ Sarah 幸存：她昨夜嫌水有金属味，没有喝 O'Hara 煮的茶。她记得维修人员反复确认 O'Hara 是否仍拒绝出售。
- 🟢 假维修挂牌：来人自称 Lakeshore 外包水质维护，在 O'Hara 拒售后的当晚进入院子。
- 🔵🟢 假维修工具箱仓储编号：编号格式与 U2 银月赌场酒水、清洁用品的供货票属于同一仓储系统。它足以证明这批“维修人员”使用了银月会的地面物流，但不能证明具体经手人或下令者。
- ⚪ 邻居：同一批人说明早还要检查两处公共水点。
- 🟢 O'Hara 最后留下的拒售页：“房子不卖。别替我签。”

U5 回收边界：Vivian 能根据仓储编号认出负责这批货物的银月会渠道，再与其他物证一起指认具体执行关系；U4 不让 Zack 仅凭编号提前说出小 Charles。

Emma 看见异常沉淀，立刻问档案里是否提过水。Zack的停顿已经给出答案。

她从档案袋里抽出被压回去的水源附件，又看见下面的 Sean 页面。

Watts 立刻联络辖区警员封闭第二处公共水点，Emma 带 Sarah 离开厨房。第一处公共水点附近已有数名核心拒签家庭成员死亡，十余人出现症状；第二处水点被及时封闭，更大范围的饮水才没有继续发生。

这一结果同时守住两个尺度：投毒已经从个人灭口升级成公共清退，第一批核心名单家庭付出了真实代价；但整个南区没有被一次性写成全部死亡，U5 仍有居民、证人和未完成程序可以争取。Sarah 也能成为 U5 的情感与证词承接者，而不是为了惩罚角色被牺牲。

## Emma 的爆发

Emma 的愤怒分两层，顺序不能颠倒。

第一层是眼前：

> “你知道不只是 O'Hara。”  
> “你看见了水源页，却只告诉我一个地址。”

第二层才是 Sean：

> “你还看见了我父亲。”  
> “你不是在保护我。”  
> “你是在决定我什么时候有资格知道他是怎么死的。”

Zack 可以说街对面有人盯她，也可以说知道太多的人正在一个个死。这些理由都是真的，但不能自动使决定正确。

Emma：

> “那你可以告诉我有人在盯。”  
> “你可以让我和你一起决定怎么活下来。”  
> “你不能先删掉一半，再把剩下的一半叫保护。”

这句话把 Margaret、Mickey 与 Zack 的行为连在一起：三个人都出于不同程度的爱、恐惧或责任感，先替别人管理了真相。

## Emma 离开 Zack、前往 Miller 庄园

Emma 没有在争吵后无目的地跑开。她手里仍然缺一块能把 Sean 的特殊处置页变成完整真相的证词，也需要知道追加水源方案的人为什么突然改变原计划。

她在 U3 曾被小 Charles 单独带去谈 Miller 慈善事务。即使那次谈话没有提供主线答案，小 Charles 仍是她唯一亲自接触过、也能通过慈善办公室直接找到的 Miller 家族成员。

Emma 先让 Watts 把 Sarah 送去安全地点，又在附近药房借电话联系 Miller 慈善办公室。小 Charles 得知她看见 Sean 的名字后，没有否认，只说：

> “来庄园。”  
> “我会告诉你 1912 年到底发生了什么。”

这句话在玩家当下听来像唯一愿意回答她的人；到 U5 才会明白，小 Charles 正在主动把完整真相递给 Emma，让她带着真实、公开、所有人都能理解的愤怒进入晚宴。

Zack 想跟她去。Emma 拒绝：

> “你已经替我决定过一次。”  
> “这一次，我自己去问。”

Zack 没有获得立刻修复关系的机会。公共水点仍有伤者需要处理，Sarah 也需要安置。他只能看着 Emma 带着 Sean 页面副本离开，车灯驶向 Miller 庄园。

U4 的最后行动不是“Emma 将来会去庄园”，而是她已经踏上这条路；U5 从小 Charles 如何利用这次主动接触开始。

## 进入 U5 的状态

U4 结束时，玩家必须明确知道：

1. Mickey = Whale 已经坐实，但 Whale 不是系统最高点。
2. Miller → Lakeshore → Tidewater 结构已出现，金融分赃证据仍未完全落地。
3. Sean 的死已成为谋杀疑云，但谁亲手执行、如何伪装仍未坐实。
4. 南区水源清退已经造成第一批核心拒签家庭伤亡；银月会地面物流关联已经成立，幕后越权者和具体执行链留给 U5。
5. Emma 与 Zack 的裂缝不只是 O'Hara之死，而是 Zack再次替她管理了真相与风险。
6. Emma 已带着“几乎成形、但差最后证词”的怀疑前往 Miller 庄园，而不是带着已经完全解决的答案。

💡 U4 最后的尤里卡：

> 我们抓住了 Whale。  
> 但海水没有停。  
> 更糟的是，Zack 刚刚拒绝的那套算法，也可以用“保护”这种更温柔的名字重新出现。

# 本版修正后的关键因果闭环

## Harrison 为什么死

第三次拒绝撤回 Mary 改判意见 → 开始打包医院、账户与地产计划 → Mickey 操作簿记录“第三次拒绝撤回”并启动清理。  
结论：不是因为他“忽然变好”，而是因为一个长期可控的腐败工具开始产生公开风险。

## Morrison 为什么当夜开口

看见 Whitfield 被止损交出 → 收到次日交出全部旧案材料的通知 → 认出自己进入 Harrison 相同清理程序 → 当夜先找 Zack、后找 Mickey 谈条件。  
结论：他的转向来自自保和拒绝被改写，不是突然良心发现。

## Mickey 为什么亲自动手

Morrison 临时决定当夜行动 → Pierce 与黑警被法院封存锁住 → 银月会常用人手在西区行动中失联 → 没有既懂警务伪装又能临时信任的中间人。  
结论：亲自动手是时间与资源共同造成的迫不得已，也因此留下右手错误。

## Margaret 为什么隐瞒

O'Hara 的文件使 1903 旧事重新浮现 → Margaret 害怕 O'Hara被伤害，也害怕 Zack沿 Patrick 的“不能不管”一路走下去 → 她劝签、藏物、阻断求助。  
结论：她想保护小家，却通过替别人决定而再次伤害关系。

## Mickey 为什么留下办公室证据

爆炸命令证明 Miller 同样准备牺牲他 → Morrison 失误使身份即将暴露 → 他需要启动 Miller 保险，也需要确认 Zack会怎样理解 Patrick 的遗产。  
结论：四十二层是有意安排的最后对质，不是三十年操盘者突然忘记锁门。

## Zack 的错误为什么成立

街上收档案是安全选择 → 离开监视后获得完整共享机会 → 他仍只说 O'Hara、不说水源和 Sean → 现实风险扩大，Emma失去共同判断资格。  
结论：游戏不惩罚说真话；剧情批评的是安全后仍继续替别人决定。

# 写作边界

- 不把 Harrison、Morrison 洗白。他们的回头不取消既有责任。
- 不把 Margaret 写成反正义或怕麻烦。她害怕的是具体家庭再次被伟大目标吞掉。
- 不把 Emma 写成只会要求公开一切。她接受战术上的延迟，但拒绝永久被排除在决定之外。
- 不让 Mickey 的真实帮助变成骗局。Rosa、Mary 与工人援助都必须真实有效。
- 不让 Zack 靠“杀人就是错”轻易赢过 Mickey。他必须承认自己的相似处，再拒绝替别人签字。
- 不用牙痕单独定罪。牙痕只与法律壳、操作记录和时间线共同构成身份闭环。
- 不让水源附件直接写“投毒命令”。U4 只给足够后悔与行动的风险，不提前完成 U5 指认。
- 不在 U4 坐实 Lawson 私生子、小 Charles 弑父嫁祸、Sean 完整执行链和金融分赃终局证据。

# 一句话版本

U4 不是让玩家发现 Mickey 一直在假装做好人。

它让玩家发现：Mickey 做过的好事有很多是真的，他承受过的痛也是真的；但他为了让第一场牺牲显得值得，逐渐相信自己有权替所有后来的人安排真相、风险与死亡。Zack 最终拒绝了这套算法，却又在保护 Emma 时短暂重复它。Mickey 坠落以后，真正留在城市里的不只是 Miller 的机器，还有一个更难回答的问题：当控制以善意的名字出现时，我们还能不能认出它？
