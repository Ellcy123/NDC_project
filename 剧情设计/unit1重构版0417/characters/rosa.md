核心困境：保护女儿还是说出真相

案发当晚，Rosa 按照工作时间，在酒吧内进行清洁工作。晚上11点20分，她出去丢了一次垃圾。

她收到了一张匿名威胁纸条，要求她在11点半前往Webb的会客室，对外声称自己目击了Vivian杀害Webb——否则威胁者会杀死她的女儿。纸条附带有女儿的医院诊断记录，证明威胁者掌握了她最脆弱的秘密。

Rosa的选择：
- 说出真相 → 威胁者报复，女儿Miguel的安全无法保证
- 配合谎言 → 一个无辜的人（Vivian）被陷害入狱

Rosa选择了配合谎言，因为Miguel是她的一切。

她面临的不是善恶的选择，而是生存的选择：
- 她没有积蓄，没有其他收入来源
- Miguel的药物费用每月都在增长
- 在1920年代的芝加哥，一个拉丁裔单亲母亲没有任何社会安全网
- 威胁纸条的内容极其具体，证明威胁者知道她女儿的真实病情

（注：0417版与旧版的核心区别在于：威胁者是Jimmy——他写了纸条，利用"Vivian每晚23:30给Webb送酒"这个酒吧内部习惯嫁祸Vivian。这一真相在loop5才由字迹三物（1306+1401+1502）闭环揭示。Rosa本人始终不知道威胁者的身份。）

性格特征

核心性格

1. 恐惧驱动的谨慎：
  - Rosa的每一句话、每一个动作都被恐惧支配
  - 她说话时语速快、声音颤抖、不敢直视对方
  - 她的谎言并不高明，因为她不是一个习惯撒谎的人
  - 例：被问"刚才发生了什么"，她说"什么都没看到"——Zack还没问她看到什么

2. 深沉的母爱：
  - Miguel是她活着的全部理由
  - 储藏室里最显眼的不是清洁工具，而是女儿的照片
  - 工具箱的密码是女儿的生日：0915
  - 她愿意为Miguel做任何事，包括成为配合谎言的工具

3. 本性善良，内心有良知：
  - 她知道Vivian是无辜的，这让她备受煎熬
  - 即使被迫作证，她仍然在细节上留有认知盲区——她主动提及"23:20倒垃圾"，完全没有意识到这个时间节点会成为击穿自己谎言的关键证据
  - "我不能说...他们会杀了我的。"——她不是不想说，而是不敢说

4. 感恩与信任的建立：
  - loop3 第二轮指证彻底击穿后，她崩溃哭泣，说出一切
  - 她主动给出弹壳，是因为她用这枚弹壳来保护自己：如果威胁者反悔，她可以拿它去找警察
  - 在loop3 post_expose 阶段，她请求Zack不要声张，仍然担心威胁者伤害她的女儿
  - 一旦相信Zack能保护她，她愿意全力配合

语言风格

说话特点

1. 紧张时语无伦次：
  - 说话时频繁中断、重复、自我纠正
  - "没...没有！什么都没看到！我一直在地下室走廊打扫...从十点开始...不，是九点半..."

  - 被逼到极限时会放弃所有伪装

  - 她的供述来自于相信Zack能保护她，而不是因为被证据彻底压垮——逻辑上的击穿是第一步，信任的建立才是真正让她开口的原因

关键关系网

与Jimmy的关系：被施害者与施害者（Rosa不知道）

0417版的核心改变：Rosa的威胁者不是Morrison，而是Jimmy。

- Jimmy写了威胁纸条（1306），附带女儿医疗单，要求Rosa在23:30出面指认Vivian
- Jimmy了解酒吧内部规律（Vivian每晚送酒的习惯），才能写出如此精准的嫁祸指令
- Rosa始终不知道威胁者是谁——她只知道"有人威胁了她"
- loop5 post_expose_dialogue 中，Zack以字迹三物（1306+1401+1502）当面揭示Jimmy就是威胁者，但此时Rosa不在场；这条线的闭环是在Jimmy崩溃后由叙事呈现，不是通过Rosa

与Vivian的关系：被迫陷害者与受害者

- Rosa知道Vivian是无辜的——这是让她内心备受煎熬的根源
- 她被迫指认Vivian，是因为威胁纸条利用了"Vivian每晚送酒"这个习惯
- 在loop3 post_expose 后，Rosa的分析"嫁祸者很熟悉酒吧"直接推进了d1005疑点
- Rosa对Vivian没有个人恩怨；她是工具，Vivian是目标

与Morrison的关系：无直接胁迫关系（0417版改动）

- 旧版设定中Morrison是胁迫Rosa的人；0417版已改为Jimmy
- Morrison在loop1-3出现时，Rosa与他没有直接的施害关系
- Morrison的腐败线索（与Whale的关联）在loop5-6才揭示，与Rosa的谎言故事线是独立的两条线

与Webb的关系：雇主与底层员工

- Webb是蓝月亮的老板，Rosa只是最底层的清洁工
- Webb要求月底彻底清洁，Rosa只是执行命令
- Webb的死对Rosa来说不是悲伤，而是恐惧——她担心失去工作

与Miguel的关系：一切的原因

- Miguel是Rosa所有行为的动机
- 为了Miguel，她可以忍受低贱的工作、威胁者的要求、良心的折磨
- Miguel生病需要昂贵药物，Rosa的经济压力是威胁者得以胁迫她的根本原因
- 储藏室里的女儿照片、医院就诊清单、用生日做密码的工具箱——一切都指向这个8岁的女孩

与Zack的关系：从恐惧到信任

- loop1 初次见面：Rosa把Zack当作又一个威胁——不知道他是好人还是坏人
- loop1 指证后：谎言1被击穿，Rosa的可信度受损，但谎言2在loop1无法反驳——Zack换取72小时调查期，没有把Rosa当罪犯处理
- loop3 指证后：第一轮击穿时间线，Rosa退守；第二轮拿出威胁纸条，Rosa彻底崩溃
  - Rosa需要看到Zack拿出了"那张纸条"——证明Zack真的理解她是受害者——才会全盘托出
  - 崩溃后她主动给出弹壳（1308），请求保密，并分析嫁祸者"很熟悉酒吧"
- loop3 之后：Rosa从"被审讯者"变为"潜在协助者"

tutorial_failure的设计意图：

loop1结束后Rosa的叙事意义：
她的谎言1被推翻，可信度受损；谎言2悬而未决。玩家带着"Rosa的时间线还有问题"的悬念进入loop2。

创作要点

对话创作

1. 恐惧时的语言特征：
  - 频繁使用省略号表示中断和犹豫
  - 重复否定："没...没有！""什么都没看到！"
  - 自我纠正暴露漏洞："从十点开始...不，是九点半..."
  - 用恳求代替反驳："求您了...我还有个女儿..."

2. 提及倒垃圾时要自然随意：
  - Rosa提到"23:20倒垃圾"时，措辞要自然随意——她完全不知道这个细节的推理价值
  - 不能让她显得在刻意给出信息，也不能有任何"想要帮助Zack"的暗示
  - 这是她的认知盲区，不是她的选择

3. 崩溃时的真情：
  - 哭泣时声音变小而非变大
  - 最后的供述几乎是耳语——因为她害怕隔墙有耳
  - 供述时仍然在请求保密："求您不要让他知道我说了..."
  - 崩溃的触发点是威胁纸条被Zack拿出来——是因为她相信Zack，不只是因为被逻辑击败

4. loop1和loop3的情绪基调差异：
  - loop1：更加惊惶，说谎缺乏技巧，恐惧感明显可见
  - loop3：疲惫、如履薄冰，恐惧已经内化为日常状态，但崩溃更深——她守了这么久的谎终于撑不住了

5. Rosa永远不点名威胁者：
  - 她不知道是谁写了那张纸条
  - 她的所有供述中只说"那个写纸条的人"或"他"，没有名字
  - 不要让Rosa出现任何暗示她认识威胁者的表述

设计意图

叙事功能

1. 两轮假胜利与真击穿的节奏设计：
  - loop1 round_1击穿谎言1（假胜利第一层），round_2打不穿时间线谎言（挫败教程）
  - loop3 round_1击穿时间线（partial_victory，但Rosa退守），round_2拿出纸条才真正全穿（full_breakthrough）
  - 这种节奏训练了玩家"证据要对应维度"的推理直觉

2. 道德灰度的引入：
  - Rosa不是坏人，她是被迫的
  - 玩家在指证她时会感到矛盾——她是谎言的传播者，但也是威胁的受害者
  - 这种矛盾感是本作叙事的核心体验之一

3. 跨循环证据枢纽：
  - 威胁纸条（1306）：loop3获得，loop5 post_expose_dialogue 字迹闭环，指向Jimmy
  - 弹壳（1308）：loop3获得，loop4型号鉴定（.45 ACP），loop4末被Morrison夺走，loop6以1601重新发现
  - Rosa的23:20倒垃圾证言（3021002）：loop3解锁d1003（时间线谜题正式解决）
  - Rosa是连接真凶（Jimmy）和腐败警察（Morrison）两条主线的叙事纽带

4. loop1假胜利的设计意义：
  - 打穿谎言1但打不穿谎言2，是游戏对"证据不够就别轻易下结论"原则的第一次教学
  - 玩家在loop1体验到"不是有证据就能赢"，才会在loop3珍惜找到玻璃碎片+倒垃圾证言的完整推理链

情感共鸣点

1. 母爱的力量与无奈：
  - 每个发现Rosa储物间证据的玩家都会理解她为什么配合谎言
  - 女儿照片、医疗清单、用生日做密码——这些细节构建深度同理心

2. 认知盲区的讽刺：
  - Rosa无意中提供了击穿自己谎言的证据（23:20倒垃圾）
  - 这种设计让玩家感到"这个世界对她不公平"——她被人利用，却还无意中帮了侦探

3. 信任的珍贵：
  - Rosa在loop3崩溃后愿意给Zack弹壳，是因为Zack证明了自己真的在寻找真相、不是要害她
  - 在一个充满欺骗的世界里，信任是最珍贵的东西
