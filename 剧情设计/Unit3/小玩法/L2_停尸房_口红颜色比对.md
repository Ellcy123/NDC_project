# 玩法设计文档：口红颜色比对 (Lipstick Color Comparison)

**文档版本**: v0.1（框架）
**隶属章节**: Unit 3 - 循环2
**设计目标**: 玩家亲自比对衬衫领口口红印（珊瑚红）与Mary的口红（玫瑰红），发现颜色明显不同

---

## 1. 玩法概述

- **玩法类型**: compare_analyze（颜色比对）
- **核心体验**: 并排查看两个口红色样 → 比对颜色差异（珊瑚红偏橙红 vs 玫瑰红偏粉红） → 确认不一致 → 生成分析报告
- **预期时长**: 20-40秒
- **对应剧情**: Zack比对Thomas衬衫领口的口红印与Mary日常使用的口红，发现颜色不同，说明衬衫口红非Mary所有

## 2. 触发条件

玩家同时持有以下证据：
1. **32-I-03**「Thomas的衬衫」— 领口有珊瑚红口红印
2. **32-I-04**「Mary的玫瑰红口红」— Rose Blush色号

## 3. 输入与输出

- **输入证据**: 32-I-03（衬衫/珊瑚红口红印）+ 32-I-04（Mary的玫瑰红口红）
- **输出证据**: 32-E-03（口红成分分析报告）— 颜色不一致，衬衫口红非Mary所有

## 4. 玩法要点（待细化）

- 左右分屏：左=衬衫领口口红印特写（Coral珊瑚红），右=Mary口红涂抹色样（Rose玫瑰红）
- 核心视觉差异：珊瑚红更鲜艳偏橙调，玫瑰红更柔和偏粉调
- 玩家标记"不一致"后生成报告
- 交互简单但结论重要：衬衫口红属于第三方（指向Helen）
