# L5 压痕比对

**Loop**: L5 · **场景**: SC3005 法医办公室

## 玩法简述

用侧光灯照射 Bernard 办公桌上的皮质垫板，让肉眼不可见的书写压痕显现，再与 Thomas 家找到的标注报纸叠合比对，确认报纸批注就是在这张垫板上写的。比对过程中额外发现垫板边缘的花形压痕（悬念，指向 L6）。

## 比对方式

### Phase 1 — 侧光揭示

1. 玩家调整侧光灯角度（滑块 0°→90°）
2. 角度越接近 45°-60°，垫板表面隐藏的压痕越清晰——从完全不可见到完整显现
3. 压痕分层出现：先是圈画轨迹（压力最大），再是粗笔画文字，最后是细笔画
4. 找到最佳角度后锁定，Foster 拍照记录

### Phase 2 — 叠合比对

1. 标注报纸以半透明覆盖在垫板上方
2. 玩家微调位置对齐
3. 系统逐项高亮匹配点：圈画轨迹吻合 → 字迹吻合 → 位置分布一致
4. 力度分析：压痕力度不均匀（起笔重收笔轻），是"边说边写"的节奏 → 证明面对面现场教学，而非事先准备材料

### Phase 3 — 额外发现

1. 叠合完成后，垫板右侧边缘出现微光提示
2. 玩家点击 → 放大 → 发现五瓣花形压痕（直径约 1.2cm，硬物长时间按压所致）
3. 来源留为悬念，Zack 只说"是某种饰品的印记"

## 输入 → 产出

- 输入：Bernard 皮质垫板 + 标注报纸（L2 伏笔证据）
- 主产出：垫板压痕侧光分析报告（垫板消耗，报纸保留）
- 额外产出：🟢 垫板花形压痕（Clue，持久保留）

## 指证用途

- L5 R1 — 击穿"标准客户服务"：压痕与报纸批注完全吻合 + 力度节奏证明面对面教学
- L5 R2 — 花形压痕追击：Bernard 承认有戴花形手链的女人来访
- L6 R5 — 花形压痕与 Mary 手链比对，证明 Mary 到过银行
