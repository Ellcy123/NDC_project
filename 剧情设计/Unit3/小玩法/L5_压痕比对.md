# L5 压痕比对

**Loop**: L5 · **场景**: SC3014 Bernard 办公室

## 玩法简述

用侧光观察 Bernard 办公桌上的皮质垫板，让肉眼不可见的书写压痕显现，再与 Thomas 家找到的标注报纸叠合比对，确认报纸批注就是在这张垫板上写的。比对过程中额外发现垫板上的手工花形压痕（一片花瓣有明显缺口）——指向"另一位来咨询的女人"，但 L5 不指认身份（L6 R5 才完成指认）。

> **场景注意**：突发事件——Charles 登门带 Emma 去聊报社头版报道。Bernard 办公室内的小玩法完全由 Zack 单独完成，Bernard 本人对面端坐。

## 比对方式

### Phase 1 — 侧光揭示

1. 玩家调整侧光（窗户旁的角度可控台灯或卷帘缝隙投光）
2. 角度越接近 45°-60°，垫板表面隐藏的压痕越清晰——从完全不可见到完整显现
3. 压痕分层出现：先是圈画轨迹（压力最大），再是粗笔画文字，最后是细笔画
4. 找到最佳角度后锁定，Zack 自己拍照记录

### Phase 2 — 叠合比对

1. 标注报纸（L2 已拾取的 3206）以半透明覆盖在垫板上方
2. 玩家微调位置对齐
3. 系统逐项高亮匹配点：圈画轨迹吻合 → 字迹吻合 → 位置分布一致
4. 力度分析：压痕力度不均匀（起笔重收笔轻），是"边说边写"的节奏 → 证明面对面现场教学，而非事先准备材料

### Phase 3 — 额外发现

1. 叠合完成后，垫板边缘出现微光提示
2. 玩家点击 → 放大 → 发现手工花形压痕（直径约 1.2 cm，硬物长时间按压所致）
3. 关键特征：**其中一片花瓣有明显缺口**——是手工锻造的独有签名，工厂统一压制的花形饰品没有这种特征
4. Zack 留为悬念："是某只手工花形手链的印记，戴这种手链的人来过"——L5 不指认是谁

## 输入 → 产出

- 操作核心：玩家对 Bernard 皮质垫板（3501）执行"分析"——下面两个证物都是分析垫板得出的产物
- 标注报纸压痕比对报告（派生 3570）：垫板压痕与标注报纸（3206，L2 已拾取）叠合比对完全吻合后，对垫板分析得出
- 垫板上的手链压痕（Clue · 3502，持久保留）：对垫板分析时发现的手工花形压痕（L6 R5 才与 Mary 母亲手链 3202 比对指认）

## 指证用途

- **L5 R1** — 击穿"按标准流程办保单 · 没做多余的事"：出示标注报纸压痕比对报告（3570）——压痕与报纸批注完全吻合 + 力度节奏证明面对面现场教学
- **L5 R2** — 花形压痕追击：Bernard 被迫承认有戴这种手链的女人来咨询过（咨询内容是"欠贷款 + 买保险的人意外死后债务处理"）
- **L6 R5** — 花瓣缺口压痕与 Mary 母亲遗物花形手链（3202）比对，**L6 才完成"那个女人就是 Mary"的指认**
