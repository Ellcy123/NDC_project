# 玩法设计文档：笔迹比对 (Handwriting Comparison)

**文档版本**: v0.1（框架）
**隶属章节**: Unit 3 - 循环5
**设计目标**: 玩家亲自比对报纸批注字迹与Thomas日记/遗书字迹，发现笔迹完全不同——批注者有专业保险背景，而Thomas只有小学文化水平

---

## 1. 玩法概述

- **玩法类型**: spot_the_difference（笔迹找差异）
- **核心体验**: 左右对比两份笔迹 → 找出差异点（字形/用词/专业度） → 确认非同一人 → 生成笔迹比对报告
- **预期时长**: 30-60秒
- **对应剧情**: Zack在Foster实验室比对标注报纸上的批注字迹与Thomas的笔迹样本（日记、遗书），发现完全不同

## 2. 触发条件

玩家同时持有以下证据：
1. **35-D-01**「标注报纸」— 工整字迹、专业保险术语、银行缩写
2. Thomas笔迹样本（日记/遗书）— 拼写错误多、用词简单、笔迹稚拙

## 3. 输入与输出

- **输入证据**: 35-D-01（标注报纸）+ Thomas笔迹样本
- **输出证据**: 35-E-01（笔迹比对报告）+ 35-C-01（Thomas笔迹样本对比照片）

## 4. 玩法要点（待细化）

- 左右分屏：左=Thomas日记字迹（拼写错误、用词口语化、小学文化水平），右=报纸批注字迹（工整专业、保险术语准确）
- 差异维度：
  - 字形：Thomas笔迹稚拙不稳 vs 批注笔迹工整流畅
  - 用词：Thomas用简单口语 vs 批注用专业保险术语（如Continental银行内部缩写）
  - 拼写：Thomas有大量拼写错误 vs 批注零拼写错误
- Thomas的低教育水平是关键反差——小学文化程度不可能写出专业保险术语
- 玩家标记差异点后，确认"非同一人书写" → 生成报告
