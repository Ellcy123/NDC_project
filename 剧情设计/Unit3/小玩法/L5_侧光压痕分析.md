## 1. 玩法概述

### 核心体验

调整侧光灯角度 → 垫板表面压痕从无到有渐变显现（"发现隐形墨水"般的惊艳时刻） → 将报纸批注与垫板压痕叠合比对 → 确认轨迹/字迹/位置完全吻合 → 在垫板边缘发现意外的花形压痕（留为悬念） → 生成分析报告

### 历史背景：侧光摄影/掠射光技术 (Oblique / Raking Light)

侧光摄影（也称"掠射光分析", Raking Light Analysis）是文物修复和法医痕迹学的经典技术。原理极其简单：当光线以极低角度照射物体表面时，即使微小的凹凸起伏也会投射出明显的阴影，使肉眼不可见的纹理、划痕、压痕清晰显现。

1920年代，这一技术已在博物馆文物修复和犯罪侦查中广泛使用。1928年FBI前身机构(BOI)的文件伪造调查中就曾使用侧光摄影揭示已被擦除的铅笔痕迹。对于皮质书写垫板这类柔软材质，钢笔书写产生的压痕在侧光下尤为清晰 — 因为皮革记忆变形的能力远强于纸张。

**玩家操作的合理性**: Foster作为法医拥有摄影设备和可调角度灯具，玩家"亲手调整灯光角度"是完全符合1920年代技术水平的操作，不需要任何超时代的设备。

### 预期时长

40-60秒（Phase 1光线调整约15-20秒 + Phase 2叠合比对约15-20秒 + Phase 3花形发现约10-15秒）

### 触发条件

玩家同时持有以下证据：
1. **3504**「Bernard办公桌皮质垫板」(35-I-01) — 表面有多层压痕（正常光下不明显）
2. **3501**「标注报纸」(35-D-01) — 批注内容作为压痕轨迹的对比参照物

### 触发方式

**强制触发Foster提示** — 当玩家同时持有3504和3501时，自动触发：

```
Foster："Brennan先生，你找到垫板了？"
Zack："对，我在Bernard办公室搜到了。"
Foster："把垫板和那份报纸都带来实验室。我可以用侧光比对压痕和批注是否吻合。"
Zack："明白，我马上过去。"
```

**UI强提示**: 3504和3501右上角出现闪烁的"可比对"图标。

### 输入与输出

```
输入: 3504(垫板) + 3501(报纸)
  │
  ├─ 主产出: 3505 垫板压痕侧光分析报告 (35-E-02)
  │           + 3506 对比照片 (35-C-02, 作为3505附件)
  │
  └─ 额外发现: 3507 垫板花形压痕照片 (35-C-03, Clue类型, 持久保留, 悬念)
```

## 3. Phase 1: 光线角度调整（核心交互）

### 设计理念

这是整个小玩法的**"wow moment"** — 玩家通过自己的操作，亲眼看着一张看似平整的皮质垫板上，隐藏的文字和圈画像隐形墨水一样一点一点浮现出来。这种"发现"的感觉是本小玩法区别于所有其他比对类小玩法的核心体验。

### 光线角度与压痕可见度映射

| 角度区间 | 压痕可见度 | 视觉表现 | 音效 | Foster反应 |
|----------|-----------|---------|------|-----------|
| **0°~15°** | 0% — 完全不可见 | 垫板表面均匀照亮，看起来就是一张普通的旧皮垫。表面纹理清晰但无压痕痕迹。酒渍隐约可见 | 安静的环境音 | 沉默 |
| **15°~30°** | 0~10% — 几乎不可见 | 极其微弱的明暗起伏开始出现，但无法辨认出任何文字或图形。像是"可能有什么，也可能只是皮革纹理" | 环境音 | "再低一点……侧光角度越低，阴影越长" |
| **30°~45°** | 10~60% — 逐渐显现 | **关键过渡区**：模糊的线条开始出现，先是圈画的弧线（因为压力最大），然后是较粗的笔画。文字仍不清晰，但能看出"这里有字" | 轻微的紧张弦乐 | "有了……你看到了吗？继续调" |
| **★45°~60°** | 60~100% — 最佳区间 | **完美显现**：所有压痕清晰可见。圈画轨迹、批注文字、甚至笔画的粗细变化都能看清。深压痕（圈画处）呈现为深色阴影，浅压痕（文字尾笔）呈现为淡色阴影。整个垫板仿佛被"解码"了 | 弦乐渐强→确认音效 | "就是这个角度！看——压痕完全显现了" |
| **60°~75°** | 100~40% — 开始过曝 | 光线角度过高，阴影开始缩短。深压痕仍可见但对比度下降，浅压痕的文字开始模糊。整体变得"太亮" | 音乐减弱 | "过了……阴影在缩短，往回调一点" |
| **75°~90°** | 40~0% — 严重过曝 | 光线近乎垂直照射，压痕几乎完全消失。垫板表面被均匀照亮，回到"普通旧皮垫"的状态 | 回到环境音 | "太高了，这样什么都看不见" |

### 视觉渐变详细设计

**压痕显现顺序**（模拟真实侧光效果，深压痕先出现）：

1. **首先出现** (30°~35°): 圈画轨迹 — 因为用力最重（Bernard在报纸上圈出重点段落时钢笔压力最大），凹痕最深，最先形成可见阴影
2. **随后出现** (35°~45°): 粗笔画文字 — "ACC.D""注意90天""受益人"等笔画较重的字，模糊但可辨认
3. **最后出现** (45°~55°): 细笔画和收尾笔 — 完整的批注文字全部显现，包括尾笔的力度变化（证明"边说边写"的节奏）

**力度不均匀的视觉表现**（关键证据点）：

压痕并非所有部分同等清晰。某些字的起笔重、收笔轻（说话到一半停顿再继续写），某些圈画的起点和终点力度不同（一边指着报纸一边画圈）。这种"不均匀感"是证明"面对面现场教学"的核心视觉证据——如果是事先准备的标注，力度应该均匀稳定。

### 交互细节

**滑块物理反馈**:
- 滑块有轻微阻尼感（不是瞬间跳到目标位置，而是平滑滑动）
- 在45°~60°区间，滑块有轻微的"吸附感"（磁性手感），暗示玩家"你找到对的位置了"
- 滑块旁实时显示角度数值（精确到个位数）

**锁定机制**:
- 当玩家将滑块调整到45°~60°区间并停留1.5秒后，系统自动锁定
- 锁定时屏幕轻微闪烁（类似相机快门效果）
- Foster语音确认："完美。我来拍摄记录"
- 滑块锁定后不可再拖动，过渡到Phase 2

**容错设计**:
- 如果玩家在30秒内未找到最佳角度，Foster会给出更明确的提示："试试45度左右……侧光分析的最佳角度通常在45到60度之间"
- 如果玩家反复在错误区间停留，角度数值字体会变色提示（红色=太低/太高，绿色=接近最佳）

## 4. Phase 2: 叠合比对

### 设计理念

Phase 1让玩家"发现"了压痕的存在，Phase 2让玩家亲手验证"压痕就是那份报纸留下的"。报纸批注与垫板压痕的完美重合是第二个"啊哈时刻"。

### 交互流程

**步骤 1 — 报纸滑入**:
- Phase 1锁定后，上方的报纸参照区开始向下滑动
- 报纸变为半透明（opacity约60%），叠在垫板表面上方
- 中间分隔线消失
- Foster语音："现在我们把报纸放上去，看看轨迹是否吻合"

**步骤 2 — 玩家微调对齐**:
- 报纸初始位置有轻微偏移（约5-10像素），需要玩家拖动微调
- 拖动范围有限（不能拖到完全偏离的位置），确保玩家体验到"对齐"的操作感而不会卡关
- 对齐判定：报纸上至少2个标志性圈画/文字与垫板压痕的重合度达到90%以上

**步骤 3 — 匹配高亮**:
- 对齐成功后，系统逐项高亮匹配点（每项间隔0.8秒的动画）：

| 匹配顺序 | 匹配内容 | 高亮颜色 | Zack/Foster台词 |
|----------|---------|----------|----------------|
| 1 | 第一个圈画轨迹重合 | 金色描边 | Zack: "圈画轨迹……吻合" |
| 2 | "ACC.D"字迹重合 | 金色描边 | Foster: "字迹轨迹完全一致" |
| 3 | "注意90天免赔期"位置重合 | 金色描边 | Zack: "位置分布也完全一致" |
| 4 | 整体叠合完成 | 全部金色 + 确认闪烁 | Foster: "结论确凿——这份报纸就是在这张垫板上写的" |

**步骤 4 — 力度分析结论**:
- 叠合完成后，系统自动在垫板压痕上标注力度热力图（深色=重压，浅色=轻压）
- Foster解说关键发现：

```
Foster："注意看压痕的力度分布——"
Foster："圈画处压力最大，这很正常，Bernard在强调重点。"
```

## 5. Phase 3: 额外发现 — 花形压痕（悬念）

### 设计理念

Phase 2结束后，玩家以为小玩法已经完成，但垫板右侧边缘有一个"意外发现"等着他们。这是一个**惊喜时刻** — 玩家在分析已知证据时无意中发现了全新的线索。花形压痕的来源在此刻不揭晓，留为悬念，在Expose R2阶段B由Zack主动追击时使用。

### 触发方式

**视觉提示**:
- Phase 2叠合完成后（所有匹配高亮结束），垫板右侧边缘出现一个微弱的闪烁光点
- 光点位于垫板表面区域内、报纸覆盖区域之外（即报纸没有盖住的部分）
- 闪烁频率：每2秒闪一次，亮度从0%→30%→0%，非常微妙
- 光点位置对应花形压痕所在处

**文字提示**（如果玩家5秒内未点击）:
```
Zack内心独白: "等等……垫板边缘好像还有什么……"
```

**二次提示**（如果玩家10秒内未点击）:
```
Foster: "Brennan先生，垫板右侧边缘……我这边也看到了一个不寻常的痕迹。"
```

### 交互流程

**步骤 1 — 点击边缘光点**:
- 玩家点击垫板右侧边缘的闪烁区域
- 画面平滑缩放至该区域（zoom in动画约0.8秒）

**步骤 2 — 花形压痕特写**:
- 缩放后显示一个清晰的五瓣花形凹痕，直径约1.2cm
- 压痕轮廓规整，五个花瓣大小均匀，中心有一个小圆点
- 边缘锐利，不是磨损造成的——是硬物长时间按压在皮革表面留下的
- Foster自动测量："直径大约1.2厘米，五瓣，非常规整。不是钢笔或办公用品的痕迹"

**步骤 3 — 突发闪回（Sudden Flashback）**:

## 6. 结果与结论

### 主产出证据

**3505 — 垫板压痕侧光分析报告 (35-E-02)**

报告核心内容：
1. 技术说明：侧光照射垫板表面，不同深度的压痕产生不同光影效果
2. 发现：垫板表面显现出一组清晰的压痕
   - 压痕轨迹与报纸批注的圈画轨迹**完全吻合**
   - 压痕字迹与报纸批注的字迹**完全吻合**
   - 压痕位置分布与报纸批注的位置分布**完全一致**
3. 压痕特征分析：
   - 圈画处压痕最深（用力重）
   - 字迹处压痕力度不均匀（**边说边写的节奏**）
   - 非预先准备材料，而是**现场讲解时书写**
4. 结论：报纸批注是在Bernard办公桌的这张垫板上书写的，且是面对面讲解时现场书写的

**3506 — 垫板压痕vs报纸批注对比照片 (35-C-02, 作为3505附件)**

左：垫板侧光照片（压痕清晰可见）
右：报纸批注照片
标注：圈画轨迹吻合、字迹吻合、位置分布吻合

## 8. 视听设计

### 8.1 光影变化 — 核心视觉体验

光影变化是本小玩法最重要的视觉设计。垫板表面从"什么都看不到"到"压痕完全显现"的渐变过程，是整个Unit3小玩法系列中最具视觉冲击力的时刻。

**垫板表面光影渲染要点**：

| 元素 | 正面光(0°) | 侧光(45°~60°) | 过曝(90°) |
|------|-----------|---------------|----------|
| 皮革底色 | 深棕色，均匀 | 一侧偏亮，一侧偏暗 | 整体偏白 |
| 皮革纹理 | 清晰可见 | 被压痕阴影部分遮盖 | 被冲淡 |
| 压痕（深） | 不可见 | **深色阴影，轮廓锐利** | 阴影缩短，模糊 |
| 压痕（浅） | 不可见 | **浅色阴影，轮廓柔和** | 不可见 |
| 酒渍 | 隐约可见 | 可见但不突出 | 不可见 |
| 光源方向感 | 无 | **强烈的方向性**（左→右或右→左） | 无（垂直照射） |

**关键视觉效果**：
- 压痕显现应像"墨水在纸上缓慢渗透"一样自然，不是突然出现
- 阴影方向必须与灯具角度一致（灯具从右侧照射，阴影向左延伸）
- 深压痕和浅压痕的显现有先后顺序（深先浅后），给玩家"逐渐发现越来越多细节"的感觉

**叠合阶段的视觉**：
- 报纸半透明叠在垫板上，两者的笔迹/圈画重合时产生金色描边光晕
- 匹配成功的区域发出微弱的暖光脉动，像老式胶片上的标记
- 力度热力图采用暖色调渐变（深红=重压 → 橙色=中压 → 浅黄=轻压）

### 8.2 音效设计

| 阶段 | 音效 | 描述 |
|------|------|------|
| 进入小玩法 | 实验室环境音 | Foster实验室的日常背景：远处的钟声、窗外的雨声、桌上仪器的嗡鸣 |
| 滑块拖动 | 机械滑轨声 | 金属滑轨的低沉摩擦声（灯具角度调节的物理声音），与滑块移动同步 |
| 压痕开始显现(30°) | 低频弦乐渐入 | 大提琴的低音长弓，缓慢渐强，暗示"有什么正在浮现" |
| 压痕清晰(45°~60°) | 弦乐组和弦 | 弦乐从低音推至中音，形成"发现"的和弦解决感。不是突然的惊叹，而是缓慢的"原来如此" |
| 锁定最佳角度 | 相机快门声 | 清脆的"咔嚓"一声，代表Foster拍摄记录。同时弦乐定格在和弦上 |
| 报纸滑入叠合 | 纸张摩擦声 | 纸在皮革表面滑动的柔软声音 |
| 每项匹配高亮 | 确认提示音 | 低沉的"嗡"一声（像老式验钞机的确认音），每次匹配间隔0.8秒 |
| 全部匹配完成 | 完整确认音 | 三个音符的上升和弦（Do-Mi-Sol），明确的"任务完成"感 |
| 花形压痕闪烁 | 高频单音 | 极轻的"叮"一声，每次闪烁伴随一声，像远处的玻璃风铃 |
| 点击花形压痕 | 放大镜对焦声 | 透镜旋转对焦的机械声 |
| Zack独白（悬念） | 悬疑音效 | 低沉的弦乐单音+轻微不和谐和弦，暗示"这个发现很重要但答案未知" |

### 8.3 音乐设计

整体音乐基调：**安静紧张 → 发现惊喜 → 确认满足 → 意外发现 → 悬念留白**

- Phase 1 (0°~30°): 极简钢琴单音 + 环境音，营造"安静的实验室"氛围
- Phase 1 (30°~60°): 弦乐缓慢加入，随压痕显现渐强。到45°~60°时弦乐达到高点
- Phase 2: 弦乐维持中等强度，每次匹配成功时有一个小的音乐"解决"
- Phase 2结束: 短暂的满足感和弦（玩家以为结束了）
- Phase 3: 音乐突然变安静（只剩钢琴单音），花形发现时加入一段带悬疑感的新旋律线（暗示"这个发现很重要但答案未知"），以不解决的和弦结尾——留下悬念感

### 8.4 镜头与动画

| 时刻 | 镜头行为 |
|------|---------|
| 进入小玩法 | 从Foster实验室全景缓慢推进到桌面特写 |
| Phase 1调整中 | 固定机位，灯具角度图示随滑块同步旋转 |
| 锁定最佳角度 | 微微推进垫板表面（强调压痕细节） |
| 报纸滑入 | 从上方45度角切入的动画（像翻开一张透明胶片） |
| 匹配高亮 | 每次匹配时轻微震动（screen shake, 1-2像素, 0.1秒） |
| 全部匹配完成 | 短暂的慢镜头效果（0.5秒），然后恢复正常 |
| 花形发现zoom | 平滑缩放至花形区域（0.8秒），附带景深效果（周围虚化） |

## 11. 开发备注

### 11.1 技术优先级

| 优先级 | 实现内容 | 说明 |
|--------|---------|------|
| **P0** | 滑块 → 压痕可见度实时映射 | 核心交互，渐变效果必须流畅。建议使用Shader控制压痕图层的alpha值，而非逐帧切图 |
| **P0** | 压痕分层显现（深先浅后） | 至少3层（圈画/粗笔画/细笔画），每层有独立的alpha曲线 |
| **P0** | 叠合比对的对齐检测 | 简单的位置偏移量检测即可，不需要复杂的图像匹配算法 |
| **P1** | 光源方向感（阴影方向随角度变化） | 使用法线贴图(Normal Map)可以高效实现皮革表面的光影变化 |
| **P1** | 力度热力图 | Phase 2结束后的静态叠加层，预渲染即可 |
| **P1** | 花形压痕闪烁 + 缩放动画 | 标准UI动画，不需要特殊技术 |
| **P2** | 滑块吸附感（最佳区间的磁性反馈） | 手柄震动 + 滑块阻尼变化，增强手感但非必需 |
| **P2** | 花形压痕悬念演出 | Zack独白+Foster分析的对话演出，标准对话系统即可实现 |

### 11.3 Shader设计建议

侧光压痕效果的核心是一个自定义Shader，输入参数为"光线角度"(0~90)，输出为垫板表面的最终渲染：

```
// 伪代码
float angle = slider_value; // 0~90
float shadow_length = cos(angle * PI / 180); // 角度越低阴影越长
float shadow_intensity = bell_curve(angle, center=52, spread=15); // 最佳角度52°附近最强

for each pressure_layer in [circles, heavy, fine]:
    float layer_threshold = layer.min_angle;
    float layer_alpha = smoothstep(layer_threshold, layer_threshold+15, angle)
                       * shadow_intensity;
    blend(base_texture, layer.texture, layer_alpha);

    // 法线贴图驱动的阴影方向
    float3 light_dir = normalize(float3(cos(angle), sin(angle), 0));
    float shadow = dot(normal_map.sample(), light_dir);
    apply_shadow(shadow * shadow_length);
```

### 11.4 与其他小玩法的差异

| 维度 | 其他小玩法(比对类) | 侧光压痕分析 |
|------|-------------------|-------------|
| 核心交互 | 找差异 / 确认一致 | **光线角度调整 → 渐变揭示** |
| 视觉变化 | 静态对比两张图 | **动态光影实时变化** |
| 惊喜感 | 结果可预期 | **从"看不见"到"完全显现"的震撼** |
| 叙事功能 | 确认单一结论 | **主结论 + 额外发现(花形压痕)** |
| 情绪曲线 | 平缓(确认→完成) | **递进(好奇→发现→确认→意外→悬念)** |
| 技术难度 | 低(静态图+UI) | **中高(实时光影渲染+分层显现)** |

### 11.6 设计备忘

1. **"看不见"到"看得见"是核心** — 所有视觉设计围绕这个转变。如果压痕从一开始就隐约可见，整个体验就毁了。0°~30°区间必须做到完全不可见。
2. **花形发现必须是"意外"且"存疑"** — Phase 3的设计目标是让玩家觉得"我本来是在分析压痕，突然发现了别的东西"。不能提前暗示花形压痕的存在，也不能在此刻揭示来源——只留下悬念。
3. **力度分析是叙事核心** — Phase 2的力度热力图和Foster的解说是本小玩法最重要的叙事输出（"边说边写" = 面对面教学），技术实现上可以用预渲染静态图，但叙事呈现必须到位。
4. **与笔迹比对的递进关系** — 笔迹比对证明"Bernard写的"，侧光分析证明"在Bernard桌上面对面写的"。两个小玩法形成递进，不是重复。指证时R1用笔迹报告，R2用压痕报告，各有不同维度的证明力。
5. **花形压痕→Expose R2阶段B的衔接** — 3507在小玩法中作为悬念产出，Zack在Expose R2击穿"标准服务"后主动拿出3507追击Bernard。花形首饰与Mary手链的关联在L5收尾事件中由Emma首次提示，不在小玩法内揭示。
