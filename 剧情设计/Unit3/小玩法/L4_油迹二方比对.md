# 玩法设计文档：油迹二方比对 (Two-Source Oil Cross-Reference)

**文档版本**: v1.0
**隶属章节**: Unit 3 - 循环4
**设计目标**: 玩家将楼顶门把手外侧油迹"盖章比对"到L3已确认的化学图谱上，快速验证同源——核心"顿悟时刻"不在化学匹配（预期之中），而在**方向性推理**（外侧有油/内侧干净 = 有人从楼梯间侧进入楼顶）

---

## 0. 前因后果 (Context & Impact)

### 前因——证据从哪来

| 证据 | 获取场景 | 获取方式 |
|------|---------|---------|
| 3401「楼顶门外侧把手油迹照片」 | L4 SC3002 楼顶三次搜证 | Foster关门时手蹭到门把手外侧发现油腻感，侧光下可见薄油膜；**内侧把手干净**（3402作为对照） |
| 3312「油样本化学比对报告」(33-E-02) | L3 小玩法派生（跨循环复用） | L3已通过完整化学比对确认：楼顶地面油迹 = Helen家轮椅维护油（Standard Oil Lubricant，添加剂比例14.2%，石油基润滑剂） |

**触发动机**: L3已证明Helen家的轮椅维护油出现在楼顶地面（涂油是预谋行为）。L4开篇Foster发现门把手外侧也有油——如果门把手油与地面油同源，那就不只是"楼顶有油"的问题，而是"有人沾着油从楼梯间侧推门进入楼顶"。这是对L3证据的二次应用（跨循环证据复用），用同一份油报告击穿一条全新的推理维度。

**设计意图——为什么不合并到L3?**
- L3的化学比对回答的问题是"地面油是谁家的？"→ 指向Helen家
- L4的二方比对回答的问题是"谁从楼梯间进入了楼顶？"→ 指向Helen当晚在场
- 两次比对服务于不同的推理链条，刻意分循环揭示，符合"每轮循环只揭示一层真相"原则

### 后果——证据用在哪

- **直接用途**: **L4 Expose R1** — Helen坚守"那晚在家"最后防线（L3三轮指证后仍未承认当晚到场）
  - 出示3405（油迹二方比对报告）→ 门把手外侧油 = 地面油 = Helen家轮椅油 → **有人沾着轮椅维护油从楼梯间侧进入楼顶**
  - 外侧有油 + 内侧干净 = 单向进入（从楼梯间到楼顶），不是从楼顶出去
  - 配合3401(门把手照片)作为视觉辅助，3402(内侧干净照片)由Zack口头引用
  - **Helen被迫承认当晚上过楼顶**——"Thomas叫我上去检查油干没干"
  - 连锁揭露：
    - Helen手伤真相：L3所谓"修轮椅"割伤 → 实为涂油时被护栏锋利处(3404)割伤 → 解释为什么下午Thomas帮Helen开门
    - 两双相似鞋子(L3获得3403) + 脚印深浅不一的真相：L1脚印是两个体重不同女人的混合脚印
- **剧情意义**: 彻底击穿Helen"那晚在家"的防线。从此Helen无法否认案发当晚在场。这是L3→L4证据链的关键桥梁——L3证明Helen参与了准备工作，L4证明Helen当晚亲自到场。

## 1. 玩法概述

- **玩法类型**: stamp_match（盖章比对——快速验证型）
- **核心体验**: 将门把手油迹采样"盖印"到L3已确认的化学图谱上 → 快速匹配确认 → 触发方向性推理（外有内无 = 单向进入）
- **预期时长**: **20-30秒**（刻意比L3的30-60秒更短——玩家已做过完整化学比对，这次是"再看一眼就确认"的节奏）
- **对应剧情**: Zack在Foster实验室，将楼顶门把手外侧油迹采样与L3已有的化学比对报告进行快速交叉验证——不需要重做一遍完整分析，只需确认新样本是否落在已知图谱内
- **设计基调**: L3是"发现"（第一次确认油的来源），L4是"确认"（用已知结论验证新线索）。玩家的情绪不是"这是什么油？"而是"果然又是同一种油——等等，这说明什么？"

### 与L3化学比对的差异

| 维度 | L3 油样本化学比对 | L4 油迹二方比对 |
|------|-----------------|----------------|
| 目的 | 确认地面油 = Helen家轮椅油 | 确认门把手油 = 地面油 = 同源 |
| 玩家认知 | 第一次接触化学比对流程 | 已熟悉流程，期待快速验证 |
| 交互深度 | 逐项比对4个维度（基础成分/粘度/添加剂/化学指纹） | 一键盖章比对（整体图谱叠合） |
| 时长 | 30-60秒 | 20-30秒 |
| "aha moment" | 化学成分一致 → Helen家的油在楼顶 | 方向性推理 → 有人从楼梯间侧进入 |
| 后续触发 | 证明Helen参与准备工作 | 证明Helen当晚到场 |

**Step 1: 展示（3秒）**

**Step 2: 盖章比对（5-8秒）**
- 玩家将左侧新样本图谱**拖拽**到右侧已知图谱的参照基线区域

  - 波峰/波谷位置逐一对齐
  - 每对齐一处，闪烁绿色确认标记

## 5. 结果与结论

### 5.1 比对报告内容（3405 道具描述）

**油迹二方比对报告 (34-E-01)**

> **比对对象**:
> - 样本A: 楼顶安全门外侧把手油膜（L4新采样）
> - 样本B: 楼顶地面油迹化学图谱（L3已确认 = Standard Oil Lubricant轮椅维护油）
>
> **比对结论**: 两份样本化学成分完全一致
> - 基础成分: 石油基润滑剂 ✓
> - 品牌标识: Standard Oil Lubricant ✓
> - 添加剂比例: 14.2% ✓
> - 化学指纹: 完全匹配 ✓
>
> **关键发现**:
> - 门把手**外侧**（楼梯间侧）检出油膜，**内侧**（天台侧）干净
> - 油膜分布特征符合"手掌推门"的接触模式
> - 结合地面油迹（L3确认来源于Helen家轮椅维护油）——同一种油出现在两个位置
>
> **推理意义**: 有人手上沾有轮椅维护油，从楼梯间侧推开安全门进入天台。

### 5.2 方向性推理——核心价值

本小玩法的价值不在化学匹配（玩家在L3已经做过，结果在预期之内），而在**方向性逻辑推理**：

| 事实 | 推理 |
|------|------|
| 外侧把手有油 | 推门的人手上有油 |
| 内侧把手干净 | 推门的人没有从天台侧出去（否则内侧也会留油） |
| 外有+内无 = 单向 | 这个人是从楼梯间侧**进入**天台，不是从天台**出去** |
| 油 = Helen家轮椅维护油 | 这个人的手上沾有Helen家的轮椅维护油 |
| Helen是唯一接触过轮椅油的人 | Helen独自从楼梯间上了楼顶 |

**连锁推理**（Expose R1中Zack口头引用）：
- Helen手伤（L3声称"修轮椅割伤"）→ 实为涂油时被护栏锋利处(3404)割伤 → 解释下午Thomas帮Helen开门（手受伤开门不便）
- 两双相似鞋子(3403) → L1脚印深浅不一 = 两个体重不同的女人（Mary + Helen）穿着相似鞋子留下的混合脚印

## 6. 特殊情况处理

### 6.1 玩家行为边界

| 情况 | 处理 |
|------|------|
| 玩家尝试反向拖拽（把L3报告拖到门把手上） | 系统自动纠正方向，提示"将新样本对准参照基线" |
| 玩家长时间不操作（>15秒） | Foster提示："把新图谱叠上去就行了——和上次一样的方法" |
| 玩家点击门把手照片放大后不关闭 | 点击空白区域关闭，或5秒后自动缩回 |
| 玩家在叠合动画期间点击 | 忽略输入，等待动画完成 |
| 玩家在方向性展示阶段快速跳过 | 允许快速跳过动画，但Zack独白文字必须完整显示后才能点击[生成报告] |

### 6.2 证据缺失保护

| 情况 | 处理 |
|------|------|
| 只有3401没有3312 | 不可能发生——3312是L3派生，进入L4时必定持有 |
| 只有3312没有3401 | 分析界面中3312旁不显示"可比对"标记，直到获取3401 |
| 3405已生成后再次尝试 | 提示"已完成比对——报告在背包中" |

### 6.3 叙事一致性保护

- 本小玩法的Zack独白中**不能**出现"Helen"以外的嫌疑人——此时门把手油迹只能指向Helen（唯一接触过轮椅维护油的人）
- 不能在比对结论中直接说"Helen上了楼顶"（那是Expose R1的指证结论），只能说"有人沾着轮椅维护油从楼梯间侧进入"，暗示极强但留给Expose正式击穿

## 7. 视听设计

### 7.1 音效

| 时机 | 音效 | 说明 |
|------|------|------|
| 进入小玩法 | 实验室环境音（低频） | 延续L3实验室氛围，但更安静（暗示"快速操作"） |
| 拖拽图谱 | 纸张滑动声（轻） | 轻量反馈，不像L3那样有沉重的实验器材声 |
| 图谱吸附对齐 | 磁性吸附声（轻脆） | 快速"咔嗒"——比L3的逐项确认音效更简洁 |
| 叠合匹配成功 | 金属印章按压声 + 短促确认音 | "盖章"感——区别于L3的化学分析完成音效 |
| 切换到方向性展示 | 低音弦乐渐入 | 氛围转换：从实验室的理性 → 推理的紧迫感 |
| 箭头动画（进入方向） | 门轴转动声 + 脚步声（单人） | 模拟Helen推门进入的场景音，强化方向性认知 |
| Zack独白 | BGM降至背景 + 配音 | "Helen……你那晚上了楼顶"——配音重音在"上了楼顶" |
| 生成报告 | 纸张翻页 + 打字机盖印 | 与L3报告生成音效一致，保持系列统一 |

### 7.2 视觉

| 元素 | 设计 |
|------|------|
| 整体色调 | 实验室冷光（延续L3），但方向性展示切换为暖黄色（楼道煤气灯色调） |
| 门把手照片 | 侧光效果下油膜呈现彩虹光泽（薄膜干涉），突出"这里有东西" |
| 图谱叠合动画 | 新旧曲线分别为蓝色/绿色，匹配后合并为金色 |
| 方向性示意图 | 俯视门板线稿，楼梯间侧标注油迹符号(☢)，天台侧标注干净符号(○) |
| 进入方向箭头 | 粗金色箭头，从左向右穿过门板，带有脉冲动画 |
| Zack独白字幕 | "Helen"二字高亮（琥珀色），其余白色 |

### 7.3 镜头/过渡

| 阶段 | 镜头 |
|------|------|
| Step 1 展示 | 平稳双栏展示，无运镜 |
| Step 2 盖章 | 轻微推进（缩放到叠合区域） |

## 10. 开发备注

### 10.1 优先级

- **P0**: 拖拽叠合交互 + 匹配动画 + 方向性展示 + Zack独白
- **P1**: 门把手侧光照片油膜高亮 + 俯视门板示意图 + 箭头动画
- **P2**: Foster画外音 + 环境音效 + 镜头过渡
- **P3**: 图谱曲线渲染细节（波峰对齐动画的精度）

### 10.2 技术要点

1. **跨循环证据引用**: 3312的数据从L3小玩法输出中读取，确保四项指标数值（品牌、成分、添加剂14.2%、化学指纹）与L3完全一致。避免硬编码重复数据。
2. **盖章交互复用**: stamp_match类型可从L3的compare_analyze简化而来——去掉逐项手动确认，保留图谱叠合核心模块。
3. **方向性展示**: 门板俯视示意图为独立UI模块，可复用于Expose R1中Zack的口头推理演示。建议提取为共享组件。
4. **Zack独白分句控制**: 独白分5句，每句间隔0.8秒，最后一句"Helen……你那晚上了楼顶"加重停顿（1.2秒间隔）+ "Helen"高亮。
5. **不要出现"100%匹配"字样**: 化学比对说"完全一致"，不要量化为百分比——避免玩家质疑"99.9%算不算一致"。

### 10.3 QA检查清单

- [ ] 3312的四项指标数值是否与L3小玩法输出完全一致
- [ ] 方向性展示中"外侧/内侧"标注是否正确（外侧 = 楼梯间侧，内侧 = 天台侧）
- [ ] Zack独白是否在Expose R1之前就提到了"Helen"——确认设计意图是在小玩法阶段已明确指向Helen（因为只有她接触过轮椅油）
- [ ] 证据流转：3401和3312是否在小玩法后仍保留在背包中
- [ ] 3405生成后，Expose R1中出示3405的交互是否正常触发
- [ ] 玩家跳过独白后，3405道具描述中是否包含方向性推理的完整结论（不依赖动画传递信息）
- [ ] 叠合动画在低帧率设备上是否流畅（建议最低30fps）

### 10.4 设计决策记录

| 决策 | 理由 |
|------|------|
| 比L3短（20-30秒 vs 30-60秒） | 玩家已熟悉化学比对流程，重复完整流程会感到冗余 |
| 用"盖章"而非"逐项确认" | 强调"验证"而非"发现"——答案在预期之内 |
| 方向性展示占总时长过半 | 真正的新信息是方向性逻辑，化学匹配只是前提 |
| 独白中直接点名Helen | 此时已无其他嫌疑人接触过轮椅油，不点名反而显得刻意回避 |
| 不消耗3312 | 作为L3核心派生证据，可能在后续循环中被查看或引用 |
