# 玩法设计文档：织物比对 (Fabric Comparison)

**文档版本**: v0.1（框架）
**隶属章节**: Unit 3 - 循环2
**设计目标**: 玩家亲自比对Thomas袖扣织物纤维与Mary祈祷披肩，发现材质/颜色/编织纹理完全一致

---

## 1. 玩法概述

- **玩法类型**: compare_analyze（视觉比对）
- **核心体验**: 显微镜下观察两份织物样本 → 逐项比对（材质/颜色/编织纹理） → 确认一致 → 生成比对报告
- **预期时长**: 30-60秒
- **对应剧情**: Zack在Foster实验室，用显微镜比对Thomas袖扣处的异源织物纤维与Mary祈祷披肩破损处的织物

## 2. 触发条件

玩家同时持有以下证据：
1. **32-E-01**「Foster织物分析报告」— 确认袖扣处有异源织物纤维
2. **32-I-05**「祈祷披肩」— Mary的黑色蕾丝祈祷披肩（边缘破损）

## 3. 输入与输出

- **输入证据**: 32-E-01（袖扣织物纤维）+ 32-I-05（祈祷披肩）
- **输出证据**: 32-E-02（织物比对报告）— 材质/颜色/编织纹理完全一致

## 4. 玩法要点（待细化）

- 左右分屏显微镜视角：左=袖扣织物纤维，右=披肩破损处织物
- 比对维度：材质（高档蕾丝）、颜色（黑色）、编织纹理（玫瑰花纹）
- 三项全部确认一致后，生成比对报告
- 参考Unit2 L6烛台找茬的找茬/比对交互模式
