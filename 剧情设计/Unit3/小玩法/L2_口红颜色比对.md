# L2 口红颜色比对

**Loop**: L2 · **场景**: SC3012 Smith 家卧室

## 玩法简述

将 Thomas 衬衫上的深红色（Deep Crimson）口红印与 Mary 藏在梳妆台深处的五支浅色口红逐一比对，确认没有任何一支颜色一致。Mary 平时几乎不化妆，但私下保留着这些浅色口红，形成角色层面的隐藏信息。

## 比对方式

1. **采样**：用棉签从衬衫衣领/胸口蘸取口红色样，涂在白瓷比色板左侧
2. **取样**：从 Mary 藏起的五支浅色口红中分别取样，依次涂在比色板右侧
3. **比对**：六个色样并排，系统提取色相 / 饱和度差异并可视化呈现
4. **结论**：五支浅色口红均与衬衫深红口红印不一致，生成比对报告

| 维度 | 衬衫口红 | Mary 口红盒 |
|------|---------|---------|
| 颜色 | → Deep Crimson 深红色，偏暗红 | → 五支均为浅色系：浅玫瑰、淡桃、雾粉、浅珊瑚、裸米色 |
| 痕迹推断 | 约一周前留下，已有自然褪色 | 多数很少使用，符合 Mary 平时不化妆的形象 |
| 口红风格 | 饱和、显眼、偏艳丽 | 克制、低饱和、偏隐藏 |

## 输入 → 产出

- 输入：Thomas 衣柜底层的沾口红印衬衫（3207）+ Mary 浅色口红盒（3208）
- 产出：口红颜色比对报告（派生证据 3270）

## 指证用途

L2 R3 — 击穿"我没有理由害 Thomas（我没动机）"：衬衫上的 Deep Crimson 深红口红印与 Mary 藏起的五支浅色口红全部不符 → Mary 早就知道 Thomas 另有女人，并非毫无动机。
