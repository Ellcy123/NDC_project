# 玩法设计文档：口红颜色比对 (Lipstick Color Comparison)

**文档版本**: v1.0
**隶属章节**: Unit 3 - 循环2
**设计目标**: 通过模拟1920年代化妆品色彩鉴别技术，让玩家亲手比对两种口红的颜色差异，将"衬衫口红不是Mary的"这一抽象推论转化为直观的视觉证据。

---

## 0. 前因后果 (Context & Impact)

### 前因——证据从哪来

| 证据 | 获取场景 | 获取方式 |
|------|---------|---------|
| 3203「Thomas的衬衫」(32-I-03) | L2 SC3003 Thomas公寓搜证 | 衣柜深处发现藏了至少三个月的衬衫，领口有**珊瑚红**口红印（Maybelline Coral Flame） |
| 3204「Mary的玫瑰红口红」(32-I-04) | L2 SC3003 同场景 | Mary梳妆台上唯一的口红，**玫瑰红** Elizabeth Arden Rose Blush |

**触发动机**: 两种口红颜色肉眼可辨不同，但需要正式比对生成报告才能作为指证证据。

### 后果——证据用在哪

- **直接用途**: **L2 Expose R3** — Mary谎称"最近才听说Thomas和Helen的关系"
  - 出示3213（口红比对报告）→ 衬衫口红(珊瑚红) ≠ Mary口红(玫瑰红)，颜色/质地/化学成分完全不同
  - 衬衫藏在衣柜深处至少三个月 → **Mary早在三个月前就知道奸情，不是"最近才听说"**
  - Mary被迫承认三个月前发现便条和衬衫口红印，情绪从被动辩解转为压抑愤怒
- **剧情意义**: 衬衫口红指向第三方女性（Helen），为L3 Helen线索链埋下伏笔

## 1. 玩法概述 (Overview)

- **玩法类型**: forensic_analysis / color_comparison
- **核心体验**: **取样采集** → **色卡比对** → **差异标注** → **生成报告**
- **预期时长**: 30-50秒
- **对应剧情**: Zack对Thomas衬衫领口的口红印与Mary日常使用的口红进行正式色彩鉴别，确认两者颜色完全不同

### 1.1 历史背景 (Historical Context)

```
┌─────────────────────────────────────────────────────────────┐
│                 【1920年代口红鉴别】                          │
│                                                             │
│  1920年代口红主要成分：蜂蜡基底 + 矿物色素                  │
│  • Maybelline（1915年创立）：铅丹(Pb₃O₄)+胭脂虫红 → 珊瑚红 │
│  • Elizabeth Arden（1910年创立）：氧化铁(Fe₂O₃) → 玫瑰红   │
│  两种配方的矿物色素完全不同，颜色差异肉眼可辨              │
│                                                             │
│  设计意义：                                                  │
│  • 品牌差异体现社会阶层（Maybelline平价 vs Arden高端）      │
│  • 颜色差异即身份差异——口红是1920年代女性的"指纹"          │
│  • 玩家通过色彩比对亲手确认"这不是Mary的口红"              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 触发条件 (Entry Trigger)

玩家需同时持有以下证据：
1. **3203「Thomas的衬衫」** — 领口有珊瑚红口红印
2. **3204「Mary的玫瑰红口红」** — Rose Blush色号

**触发表现**:
- 当玩家在证据界面尝试对这两件证据进行"分析"操作时
- Zack独白："衬衫上的口红颜色……跟Mary的口红看起来不太一样。我需要仔细比对一下。"
- Emma提示："Foster那里有比色用的白瓷板，可以把两种口红都涂上去对比。"
- 进入小玩法界面

## 4. 特殊情况处理 (Edge Cases)

### 4.1 操作顺序颠倒
- **场景**: 玩家先涂Mary口红，再采衬衫样本
- **处理**: 允许。比色板左右不强制顺序，系统会根据实际涂抹位置自动标注来源

### 4.2 拖拽目标错误
- **场景**: 玩家把棉签拖到Mary口红上（而非衬衫）
- **处理**: 棉签弹回，Emma："棉签是用来从衬衫上取样的，口红直接涂就行。"

### 4.3 玩家中途退出
- **处理**: 弹窗确认"比对尚未完成，确定要退出吗？"。退出不保存进度，可重新开始。

### 4.4 重复操作
- **场景**: 证据生成后玩家尝试再次触发
- **处理**: 入口消失（一次性玩法），可在证据详情中查看"分析记录"

## 8. 开发备注 (Dev Notes)

- **美术资源需求**:
  - 白瓷比色板2D素材（含分隔线）
  - 棉签图标（未染色/珊瑚红染色两态）
  - 口红膏体图标（Rose Blush色号包装）
  - 珊瑚红/玫瑰红色块纹理
  - 色条可视化UI组件

- **技术要点**:
  - 拖拽交互需碰撞检测（棉签→衬衫→比色板区域判定）
  - 色样涂抹需要遮罩动画（从无到有的填充效果）
  - 分析数据的色条动画用Tween实现

- **本地化**:
  - 对白文本支持多语言
  - 品牌名保留英文原名
  - Key示例：`GAME_MINI_LIPSTICK_TITLE`, `GAME_MINI_LIPSTICK_HINT_SWAB`
