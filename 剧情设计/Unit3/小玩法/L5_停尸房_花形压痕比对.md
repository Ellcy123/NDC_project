# 玩法设计文档：花形压痕比对 (Daisy Imprint Comparison)

**文档版本**: v0.1（框架）
**隶属章节**: Unit 3 - 循环5
**设计目标**: 玩家亲自比对垫板上的五瓣花形压痕与Mary手链雏菊吊坠，发现形状/尺寸完全一致——证明Mary曾坐在Bernard桌前

---

## 1. 玩法概述

- **玩法类型**: compare_analyze（形状叠合比对）
- **核心体验**: 查看花形压痕特写 → 与Mary手链雏菊吊坠叠合 → 确认形状/尺寸完全一致 → 生成比对报告
- **预期时长**: 20-40秒
- **对应剧情**: Zack在Foster实验室用显微镜比对垫板花形压痕与Mary手链上的雏菊吊坠，发现完全吻合——配合RelationNetwork（循环2 Mary手链）建立Mary↔Bernard跨循环连接

## 2. 触发条件

玩家同时持有以下证据：
1. **35-C-03**「垫板花形压痕照片」— 五瓣花形，直径1.2cm（侧光分析中发现）
2. **35-I-02**「Mary手链照片」— 雏菊吊坠，直径1.2cm

## 3. 输入与输出

- **输入证据**: 35-C-03（花形压痕照片）+ 35-I-02（Mary手链照片）
- **输出证据**: 35-E-03（花形压痕比对报告）— 完全一致，Mary曾长时间坐在Bernard桌前

## 4. 玩法要点（待细化）

- 左右分屏或叠合模式：
  - 左=垫板花形压痕（侧光下的凹陷阴影，五瓣轮廓规整）
  - 右=Mary手链雏菊吊坠（立体浮雕，五瓣花瓣清晰）
- 比对维度：花瓣数量（五瓣）、整体形状、直径（均1.2cm）、花瓣弧度
- 叠合确认后生成报告
- 报告关键补充信息："压痕深度显示不是短暂触碰，而是较长时间接触（至少10-15分钟）。压痕位置非常集中，说明手腕几乎没动——也许她非常紧张，整个人僵住了"
