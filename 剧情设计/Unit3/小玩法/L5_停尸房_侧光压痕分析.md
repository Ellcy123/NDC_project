# 玩法设计文档：侧光压痕分析 (Side-Light Impression Analysis)

**文档版本**: v0.1（框架）
**隶属章节**: Unit 3 - 循环5
**设计目标**: 玩家亲自调整侧光灯角度，让垫板上的隐藏压痕逐渐显现，并与报纸批注对比发现完全吻合

---

## 1. 玩法概述

- **玩法类型**: interactive_reveal（交互式揭示）
- **核心体验**: 调整侧光灯角度 → 压痕逐渐显现 → 与报纸批注对比 → 确认吻合 → 生成压痕分析报告
- **预期时长**: 40-60秒
- **对应剧情**: Zack在Foster实验室用侧光灯照射Bernard办公桌垫板，发现隐藏的书写压痕与标注报纸的批注完全吻合——证明报纸是在Bernard桌上批注的

## 2. 触发条件

玩家同时持有以下证据：
1. **35-I-01**「Bernard办公桌皮质垫板」— 表面有多层压痕（正常光下不明显）
2. **35-D-01**「标注报纸」— 批注内容作为对比参照

## 3. 输入与输出

- **输入证据**: 35-I-01（垫板）+ 35-D-01（标注报纸）
- **输出证据**: 35-E-02（垫板压痕侧光分析报告）+ 35-C-02（垫板压痕vs报纸批注对比照片）

## 4. 玩法要点（待细化）

- 核心交互：拖拽滑块调整侧光角度（0°-90°）
  - 0°-30°：压痕不明显
  - **45°-60°：压痕逐渐清晰**（最佳区间）
  - 70°-90°：过曝，压痕模糊
- 上半屏：报纸批注照片（固定参照）
- 下半屏：垫板表面（随侧光角度变化，压痕逐渐显现）
- 达到最佳角度后，压痕与报纸批注自动叠合对比 → 玩家确认"完全吻合"
- 额外发现：垫板右侧边缘有花形压痕（为R3伏笔）→ 生成 35-C-03（垫板花形压痕照片）
