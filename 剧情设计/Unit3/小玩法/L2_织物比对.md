# 玩法设计文档：搜身观察 (Body Search Observation)

**文档版本**: v2.0
**隶属章节**: Unit 3 - 循环2
**设计目标**: 玩家在L2见到Mary时，因手持袖扣纤维样本（3105），注意到Mary身上的黑色蕾丝披肩，触发搜身观察交互；通过点击热区观察两处细节，获得Clue证据3205（祈祷披肩）和3207（Mary手链照片）

---

## 0. 前因后果 (Context & Impact)

### 前因——证据从哪来

| 证据 | 获取场景 | 获取方式 |
|------|---------|---------|
| 3105「袖扣纤维样本」| L1 Foster实验室，首次尸检移交 | Foster从Thomas遗体右袖扣提取异源织物纤维——黑色蕾丝，高档手工品，推测来源为披肩或围巾 |

**触发动机**: 玩家持有3105进入Mary所在场景，Zack目光落在Mary身上的黑色蕾丝披肩上，纤维样本与眼前实物形成关联，触发内心独白，解锁搜身观察选项。

### 后果——证据用在哪

| 证据产出 | 后续用途 |
|---------|---------|
| 3205「祈祷披肩」(Clue) | **L2 Expose R4（最终轮）** — 出示3205，以披肩纤维与Thomas袖扣纤维特征一致、披肩右下角有勾丝破损，证明Thomas曾抓过Mary的披肩，双方有近距离肢体接触，击穿Mary"没见到Thomas"的谎言 |
| 3207「Mary手链照片」(Clue) | **L5 伏笔** — 手链上的雏菊吊坠与L5后期发现的花形压痕关联；Mary下意识抚摸手链的习惯动作本身也是其心理状态的隐性线索 |

> **设计说明：** 3212（织物比对报告）已从游戏中完全删除。Expose R4直接使用3205（搜身观察获得的Clue）而非派生的分析报告。3205的证明力来自：① Zack目视观察到黑色蕾丝特征与3105（袖扣纤维样本）描述一致，② 披肩右下角的勾丝破损与Thomas袖扣处的附着纤维形成物理对应——这条推理链不需要实验室二次确认。

## 1. 玩法概述 (Overview)

### 1.1 基本信息

- **玩法类型**: body_search_observation（搜身观察）
- **核心体验**: 进入人物立绘观察界面 → 点击热区 → Zack描述细节 → 获得Clue证据
- **预期时长**: 30-45秒
- **对应剧情**: Zack持有纤维报告见到Mary，注意到她披肩与报告描述高度吻合，触发主动观察；观察过程中还意外注意到Mary下意识抚摸手链的习惯动作

### 1.2 核心设计理念

```
┌─────────────────────────────────────────────────────────────┐
│               【搜身观察交互模型】                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  传统搜证：场景点击 → 获取物品 → 进背包                    │
│  搜身观察：对人物点击 → Zack主动描述 → 记录为Clue          │
│                                                             │
│  设计意义：                                                  │
│  - 观察对象是"人"而非场景，传递"侦探读人"的体验感          │
│  - Zack的描述语言带有推理色彩，不是中立记录                 │
│  - 两个热区的发现性质不同：                                 │
│    热区1（披肩）：主动目标，Zack带着问题来观察              │
│    热区2（手链）：意外发现，因Mary的紧张行为引起注意        │
│  - 短小精悍，不打断叙事节奏                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 触发条件 (Entry Trigger)

**必要条件（系统检测）：**
1. 玩家**持有3105**「袖扣纤维样本」
2. 玩家当前场景中**Mary在场**（SC3013公寓楼门口 或 SC3003公寓客厅）

**触发表现（满足条件后自动触发）：**
- 对话进行中/对话刚结束时，Zack内心独白浮现
- 独白消散后，对话选项栏出现新选项

具体触发时机见 Section 2.1。

> **推理严密性备注：** 3205的证明力不依赖实验室比对。推理成立的条件是：（1）特征描述足够具体（黑色蕾丝、高档手工品），排除了大量常见织物；（2）破损与附着现象形成物理关联，不是单纯的"外观相似"。玩家能通过两步逻辑独立推导出结论，不需要Foster的权威背书。

## 6. 美术与音效需求 (Art & Audio Requirements)

### 6.1 美术需求

| 需求项 | 说明 | 优先级 |
|--------|------|--------|
| Mary立绘——搜身观察专用姿态 | 右手抚摸左腕，略显局促，全身或3/4身位 | 高 |
| 热区A视觉：披肩细节 | 黑色蕾丝纹理清晰可辨，右下角勾丝破损可见 | 高 |
| 热区B视觉：手腕/手链细节 | 银质细链 + 雏菊小吊坠，可辨认为花形 | 高 |
| 特写镜头帧：披肩近景 | 热区A点击后的局部放大图，突出勾丝破损 | 中 |
| 特写镜头帧：手链近景 | 热区B点击后的局部放大图，突出雏菊吊坠 | 中 |
| 热区发光效果 | 热区A：标准发光轮廓；热区B：弱化发光（意外发现） | 低 |

### 6.2 音效需求

| 音效 | 触发时机 | 说明 |
|------|---------|------|
| 拍照快门声 | 热区B点击，Zack说"记下来"后 | 模拟Zack随手拍照记录，配合"照片"这一证据类型 |
