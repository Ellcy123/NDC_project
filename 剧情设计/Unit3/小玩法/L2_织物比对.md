**文档版本**: v1.0
**隶属章节**: Unit 3 - 循环2

## 1. 玩法概述 (Overview)

### 1.1 基本信息

- **玩法类型**: microscope_compare（显微镜调焦比对）
- **核心体验**: **调焦旋钮交互** → 三级放大逐层揭示比对维度 → 每级确认一致/不一致 → 生成比对报告
- **预期时长**: 45-75秒
- **对应剧情**: Zack在Foster实验室，用光学显微镜逐级放大比对Thomas袖扣处的异源织物纤维与Mary祈祷披肩破损处的织物

### 1.2 核心设计理念

与传统左右静态对比不同，本玩法采用**显微镜调焦**作为核心交互模型：

```
┌─────────────────────────────────────────────────────────────┐
│                 【调焦交互模型】                              │
│                                                             │
│  传统比对：左右分屏 → 肉眼找差异 → 确认                    │
│  本玩法：  旋转调焦旋钮 → 逐级放大 → 每级揭示新维度        │
│                                                             │
│  设计意义：                                                  │
│  - 调焦操作模拟真实显微镜体验，增强沉浸感                   │
│  - 三级放大=三层信息递进，每层给玩家一次"发现感"            │
│  - 玩家主动控制观察深度，而非被动接受结果                   │
│  - 符合1920年代刑侦实验室场景设定                           │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 触发条件 (Entry Trigger)

玩家需满足以下条件：

1. **持有32-E-01**「Foster织物分析报告」— 确认袖扣处有异源织物纤维
2. **持有32-I-05**「祈祷披肩」— Mary的黑色蕾丝祈祷披肩（边缘破损）

**触发表现**:
- 玩家在证据界面将两项证据关联后，Zack独白："袖扣上的纤维和这条披肩的材质看起来很像... Foster实验室有显微镜，可以做精确比对。"
- 出现选项 **[前往Foster实验室比对]**
- 点击后进入显微镜比对界面

## 4. 特殊情况处理 (Edge Case Handling)

### 4.1 玩家选择"不一致"

- **场景**: 玩家在任意倍率下点击 [确认不一致]
- **处理逻辑**:
  - 10x/40x阶段点击"不一致":
    - Foster: "再仔细看看？两边的[材质/颜色]特征非常接近。"
    - 不阻止进度，但面板标记为"待定"
    - 允许玩家重新观察后改判
  - 100x阶段点击"不一致":
    - Foster: "你确定？花纹间距和交织角度的数据是精确测量的......要不再看一次？"
    - 视野自动微调，高亮花纹关键比对点
    - 允许第二次判断
  - **连续两次判断"不一致"**: 仍生成报告，但结论措辞调整为"高度相似但玩家存疑"
  - **设计原则**: 不因玩家误判阻断主线，报告一定会生成，只是Zack的独白语气更谨慎

### 4.2 玩家反复调焦

- **场景**: 玩家在已确认的倍率之间来回切换
- **处理逻辑**:
  - 已确认的倍率画面正常显示，进度标记保持
  - 信息面板显示该倍率已确认结果（灰色）
  - 不触发重复对话
  - Foster仅在首次回退时说: "想再确认一下？没问题。"

### 4.3 玩家中途退出

- **场景**: 玩家在比对过程中点击 [退出比对]
- **处理逻辑**:
  - 弹窗: "显微镜比对尚未完成，确定要退出吗？" [继续比对] / [退出]
  - 退出后进度保存——下次进入从上次确认的最后一级继续
  - 样本不消耗，可随时重新进入

### 4.4 调焦旋钮超出范围

- **场景**: 玩家在10x时尝试继续降低倍率，或在100x时尝试继续放大
- **处理逻辑**:
  - 旋钮到达端点后无法继续拖拽，有物理阻挡感
  - 10x下限: 旋钮轻微弹回 + 轻击音效
  - 100x上限: 旋钮轻微弹回 + Foster: "已经是最高倍率了。"

### 4.5 重复进入

- **场景**: 证据生成后玩家尝试再次触发比对
- **处理逻辑**:
  - 入口消失（一次性玩法）
  - 可在证据详情界面查看"比对记录"回顾完整结果

### 证据链中的逻辑严密性

| 逻辑环节 | 说明 |
|:---|:---|
| **为什么需要三级比对？** | 材质和颜色一致可能是巧合（市面上黑色蕾丝很多），但手工蕾丝的编织花纹是唯一的"指纹"——花纹+间距+角度三项精确吻合才能排除巧合 |
| **为什么不直接肉眼比较？** | 肉眼只能判断"都是黑色蕾丝"，无法确认是同一件织物。显微镜下的编织角度和间距是肉眼不可见的决定性证据 |
| **为什么Mary不能辩解？** | 披肩是Mary的私人物品（教堂专用），Thomas袖扣上不可能"碰巧"沾上她披肩的纤维——除非两人有过近距离肢体接触 |
| **信息递进节奏** | 10x"都是蕾丝"(弱关联) → 40x"同一种染料"(中关联) → 100x"同一件织物"(强关联)，给玩家三次递进的发现感 |

### 8.2 技术要点

- **调焦旋钮**: 需要自定义drag handler，支持垂直拖拽映射到三挡位，挡位之间有吸附和弹性
- **目镜视野过渡**: 使用render texture + blur shader实现倍率切换时的模糊→清晰过渡
- **贴图切换**: 三级倍率对应三组贴图，过渡期间做crossfade避免硬切
- **确认延迟**: 进入新倍率后设置`CONFIRM_BUTTON_DELAY`(1.5s)，防止玩家跳过观察直接点确认
- **进度持久化**: 中途退出需保存已确认的倍率进度（存入PlayerPrefs或存档系统）

### 8.3 与其他小玩法的共性

| 共性模块 | 可复用组件 |
|:---|:---|
| Foster实验室场景 | 背景图、台灯、环境音效 → 与油样本化学比对(L3)共用 |
| 报告生成UI | 仿旧纸张模板、打字机动画、盖章效果 → 全局复用 |

| 比对确认面板 | 左右对比 + 确认按钮 → 与口红颜色比对(L2)类似 |

### 8.4 本地化

- 对白文本需支持多语言
- 显微镜标注（倍率、测量值）保持英文+数值，无需翻译
- Key示例: `MG_FABRIC_FOSTER_INTRO`, `MG_FABRIC_10X_ZACK`, `MG_FABRIC_CONFIRM_MATCH`, etc.

### 8.5 与口红比对(L2)的差异

| 维度 | 口红比对 | 织物比对 |
|:---|:---|:---|
| 交互模型 | 左右静态分屏比对 | 显微镜调焦旋钮，三级放大 |
| 比对结论 | 不一致（珊瑚红 ≠ 玫瑰红） | 一致（同一件织物） |
| 比对层数 | 单层（颜色差异明显） | 三层递进（材质→颜色→纹理） |
| 发现感 | 一步到位的差异确认 | 逐层递进的匹配确认 |
| 预期时长 | 20-40秒 | 45-75秒 |
| 叙事功能 | 证明"不是Mary的口红" | 证明"Thomas接触过Mary的披肩" |
