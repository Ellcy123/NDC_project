# Unit3 AVG 人物立绘与情绪状态需求

**版本**：v1.0 / 2026-06-27
**用途**：根据 `剧情设计/Unit3/state/loop{1-6}_state.yaml`，整理 Unit3 AVG 中角色需要表达的特殊状态、情绪层次和进出场姿态。
**边界**：本文件不写具体资产命名，不写具体尺寸，不重复普通中性立绘；只记录剧情明确需要的角色状态。

> **说明**：Unit3 处于 state 设计阶段，对白尚未落地，下文"剧情依据"以 state 节点（opening / scenes / expose / post_expose / interrupts）为准；待对白草稿完成后补充对白 ID。

---

## 规则

- 标准站姿、普通说话表情不在本文档列出。
- 只记录"剧情信息必须依赖姿态/情绪才能成立"的状态。
- 电话声、远端声音不需要实体立绘，除非剧情中有明确画面出场。
- 同一角色如果在不同 Loop 情绪差别很大，应拆成不同状态，而不是一套立绘全程复用。
- **Unit3 的特殊挑战**：Mary 与 Helen 全程是"表象 vs 真相"双层人物——前 4 Loop 要让玩家信服她们是受害者 / 被胁迫者，L6 才揭示是冷静合谋。她们的立绘要能"演得过去受害者表象"，又在回看时经得起"其实一直在演"的推敲。这是本 Unit 立绘的核心难点。

## Loop 1

### Mickey

**剧情依据**：opening（事务所委托）；L1 NPC L1_opening_mickey

**状态需求**：
- 登门委托：克制、审慎，像旧识也像已经掌握局面的人；讲"贷款 + 保险 + 湖滨信托银行"与 Frank 案同模式时转严肃。
- 不在 Zack 面前过早渲染"银行就是凶手"，留一点"过度及时"的神秘感。

**叙事含义**：玩家要相信他是可靠的委托人，但他真身 Whale 要到 Unit4 才揭——L1 的从容里要藏一点说不清的笃定。

### Foster

**剧情依据**：SC3005 法医办公室；L1_scene3005_foster

**状态需求**：
- 职业严谨而克制：用语经过斟酌，"血液里含有酒精"式的保守措辞代替精确数值；只陈述事实，不主动推断动机。
- 察觉到酒精浓度异常和手部油痕两个反常细节，但因检测报告未出而把精确数据留到后续——这种"知道却不说满"的克制要能从神态看出来。

**叙事含义**：Foster 是"用精确制造模糊"的法医，她的克制本身是 L6 反转的伏笔。

### Morrison

**剧情依据**：SC3004 警局指证；L1 expose；post_expose

**状态需求**：
- 敷衍、懒散、不耐烦（"又是你俩"式）。
- 被外墙抓痕、女性脚印逐轮逼问时从搪塞到恼怒甩手。
- 最后摆烂放任：抛出"脚印可能是 Mary"，以"内部牵扯太多"为由不想管，却默许 Zack 继续查。

**叙事含义**：Morrison 体现警方腐败与调查阻力；他的放任是剧情得以推进的"懒惰的善意"。

### Zack / Emma

**剧情依据**：opening；各场景调查

**状态需求**：
- Zack：接案后的冷静侦探基调；现场推理（抓痕、脚印）时的专注。
- Emma：记者式介入，仰看天台护栏时对"普通人被谋杀、警方却掩盖"的不平。

## Loop 2

### Mary

**剧情依据**：SC3013 公寓楼门口（拦截 + 指证对象）；L2 expose 3 轮；post_expose

**状态需求**：
- 初见：神色沉静、眉宇间压着挥不去的哀色，但不慌不乱、不躲——主动开始解释昨晚不在场。这种"主动给不在场证明"本身要显得自然（其实是预设好的说辞）。
- 三轮被拆穿时：克制下的紧绷，用"恳求"与"示弱"退守，不咆哮——压抑情绪是她的习惯。
- post_expose 退守自述：承认上过天台、给出"Thomas 想推我、我推回去他失力坠落"的版本，反复说"我会被当成凶手吗"。

**叙事含义**：L2 要让玩家把她读成"在场但被动的受害者"。她的紧绷与示弱是真的有压力，但压力来源（怕暴露合谋）要藏住——回看时这些"恳求"才显出表演成分。

### 神父

**剧情依据**：SC3007 教堂；L2 神父证词

**状态需求**：
- 温和、审慎；对"为亡女守夜"和家暴处境有同情底色，但不替 Mary 遮掩，只如实说"21:00 之后我离开了，之后她是否还在我没看见"。

**叙事含义**：神父的克制中立是 L2 推理（教堂时间窗）成立的支点。

## Loop 3

### Helen

**剧情依据**：SC3008 Helen 家（指证对象）；interrupts `helen_lover_reveal`；L3 expose 3 轮；post_expose

**状态需求**：
- 基调：外冷内紧，眼神闪烁、回答简短；被逼问时迅速给出"我只是旁听 / 没去过天台"式的自我合理化。
- 被 Emma 点破口红 = 出轨对象的瞬间：被看穿的一刹那（微表情），但不当场崩溃，随即转入"情人但无瓜葛"的退守。
- 被缺件复位（螺丝帽）逼到承认上过天台、做过陷阱时：从冷硬到被迫松口，但仍坚称"被 Thomas 命令独自做、Thomas 是意外坠楼"。
- 环境：女性权益剪报墙是她内心世界的投射（童年创伤），人物状态要和这面墙的氛围呼应。

**叙事含义**：Helen 是本 Unit 情感复杂度最高的 NPC——她的"被胁迫者"表象要可信，又要在 L6 经得起"主动合谋"的回看。冷，是她的保护色。

### Bernard

**剧情依据**：SC3009 银行公共区；L3 Bernard 证词（非指证对象）

**状态需求**：
- 职业性礼貌 + 距离感，"按制度办事"式回应；只在公共区简短回答事实，不让人进办公室、不主动延伸。

**叙事含义**：平庸之恶的第一次露面，为 L5 指证埋伏笔。

### Seamus

**剧情依据**：SC3006 二楼走廊；L3 证词（红鲱鱼）

**状态需求**：
- 话痨、刻薄、自鸣得意；对女人带刻板偏见、对男人有共情；用"你们不知道里面的事"维持自我重要感。

**叙事含义**：L3 的红鲱鱼，看似可疑实则无关；他的"什么都想说"是 L6 关键证词的铺垫。

## Loop 4

### Helen

**剧情依据**：SC3008 补充指证；L4 expose 3 轮；post_expose

**状态需求**：
- 紧张但配合，眼神依旧闪烁；被追问油迹时端出"之前撒了油、当晚蹭到的"合理化说辞。
- R3 被 Margaret 目击击破后：语塞、不知如何辩解——这个"卡住"的瞬间很重要，是 Mary 登场救场的前置。

**叙事含义**：Helen 第一次被逼到说不出话，把叙事主动权交给了 Mary。

### Margaret

**剧情依据**：SC3010 Margaret 鞋坊；L4 目击证词（Zack 母亲）

**状态需求**：
- 出于母性想保护弱者（Mary）、对家暴丈夫（Thomas）有道德愤怒；讲目击时是笃定的、真诚相信自己看到的。
- 她的证词带"美化色彩"——不是说谎，而是认知偏差自动填补；状态上要显得越真诚越好（这样 L6 揭穿时才更刺痛）。

**叙事含义**：Margaret 是 Zack 的母亲，她的目击直接支撑"正当防卫"假结局；她的真诚是全案最温柔也最危险的一环。

### Mary

**剧情依据**：post_expose（登场求情，见《突发事件》文档）；SC3493 报社采访

**状态需求**：
- 登场救场：沉稳、克制、有掌控感——但这层掌控要藏在"为朋友顶罪的悲情"之下，玩家此刻不能看穿。
- 报社采访：从压抑到陈述家暴遭遇的舒缓，对 Emma 克制而坦诚，塑造受害者的公众形象。

**叙事含义**：L4 是 Mary"悲情掌控"的第一次集中表演；她揽罪、安排说法的从容，是 L6 反转的最大伏笔。

### Mickey

**剧情依据**：SC3492 市政厅衔接

**状态需求**：
- 克制专业的辩护律师姿态，为 Mary 争取合法杀人、为 Helen 争取无罪；不流露多余情绪。

## Loop 5

### Bernard

**剧情依据**：SC3014 Bernard 办公室（指证对象）；L5 expose 2 轮；post_expose

**状态需求**：
- 礼貌的疏离 + 紧绷；被压痕比对、手链压痕逐轮逼到时，退到"按制度办事"。
- 被压住后用"我也是打工人"示弱开脱；承认诱导后请 Zack 别再打探。

**叙事含义**：平庸之恶的完整呈现——他不恨任何人，提成照收，被追问时的防线是冷漠的推卸而非愤怒。

### Helen

**剧情依据**：SC3008（判决后补充证词）

**状态需求**：
- 比 L4 轻松一些，眼神更直接；愿意补充"Bernard 主动展开'用妻子做受益人'的话题"——想让公众认清银行是源头，和她的女性权益立场一致。

### Charles Miller

**剧情依据**：interrupts `charles_visit`（见《突发事件》文档）

**状态需求**：
- Charles Miller（40 岁出头，Emma 的慈善投资人）：成熟稳重，礼貌但带着要紧事的急切，登门找 Emma 说慈善事务上有要紧事要谈。轻量出场（Miller 家族线，Unit4+ 引子）。

## Loop 6

### Mickey

**剧情依据**：SC3011 控辩对决（5 轮）；opening；post_expose

**状态需求**：
- 既是友方辩护人、又是 Zack 的推理对手——控辩对决要有"棋逢对手"的张力，不是敌对审讯。
- 逐轮被推进时冷静防守（油迹时间链、涂油人选等反诉）；最终投降认输时干脆，让 Zack 拿材料一起去找法官。
- 全程是控盘者气质，即使认输也不慌。

**叙事含义**：Mickey 是 L6 的节奏掌控者；他的"亦友亦敌"是这场对决的情感底色（真身 Whale 不在本 Unit 揭）。

### Foster

**剧情依据**：SC3005 法医办公室（精密报告）；SC3695 关键剧情 AVG

**状态需求**：
- 推出盖红章的精密检测报告时：克制但明确、锋利——把"血液酒精 0.25%、无有效攻击能力"这个颠覆性结论直接摊上桌。
- 不渲染情绪，用事实把"正当防卫"基础打碎。

**叙事含义**：L1 埋的"用精确制造模糊"在这里收口——她终于给出精确数值。

### Seamus

**剧情依据**：SC3006 二楼走廊（见《突发事件》文档）

**状态需求**：
- 话痨、自鸣得意、毫无戒心地用 Helen 送的油上油；说"19 号晚送的最后一瓶、以后也不需要了"时越随意越好。

**叙事含义**：无意识证人——他不知道自己说出的是关键，这种"浑然不觉"是事件的力度来源。

### Mary

**剧情依据**：post_expose 真相揭示；SC3693 结局道谢

**状态需求**：
- 真相层（玩家通过推理得知）：冷静的策划者——识破 Thomas 杀妻骗保后与 Helen 联手反杀。
- 表演层（道谢场面）：比报社更克制，压低声音，塑造"走出阴影"的形象；隐瞒策划者身份。
- 道谢时的平静要经得起双重解读：玩家若已知真相，会从这份平静里看出后怕；若按假结局理解，则是受害者的释然。

**叙事含义**：Mary 是第三重反转的道德中心。她全程不崩、不露，立绘最难的是"同一张脸，两种读法"。

### Emma

**剧情依据**：SC3692 / SC3693 结局衔接

**状态需求**：
- 从报社回事务所时敏锐察觉 Zack 异样，但克制不强逼。
- 疑惑"银行在执行南区清退、想要当地房产，怎么这么容易让 Mary 卖给 TideWater"。

**叙事含义**：Emma 与 Zack 之间形成知识差（Zack 隐瞒了 Mary 蓄意谋杀）——她的疑惑和被蒙在鼓里是 Unit4+ 的伏笔。

### Leonard

**剧情依据**：SC3694 结局衔接

**状态需求**：
- 审慎交出银行内部消息（南区清退房产流入 TideWater），点到为止；不知 TideWater 为何想要这些房子。轻量出场，Unit4+ 引子。

### Zack（L6 综合）

**剧情依据**：控辩对决；母亲证词被推翻；结局二选项

**状态需求**：
- 与好友 Mickey 控辩对决时的克制锋利——逼问对手却带着"对朋友下手"的复杂。
- 当推理触及母亲 Margaret 证词的关键细节不可能成立时——这是全案最私人的冲击，要有"在真相和母亲之间"的痛。
- 结局二选项（销毁证据放过 Mary / 拿起证据追诉）：最终的挣扎与抉择，无论选哪个都不是胜利者的轻松。

**叙事含义**：L6 把 Zack 从"查案的侦探"推到"要为真相付出私人代价的人"——他的疲惫与挣扎是结局的情感落点。
