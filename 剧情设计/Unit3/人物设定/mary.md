# Mary Smith - Unit3《漫长的坠落》

## 一、外形设定（美术参考）

### 基本信息
- 姓名：Mary Smith
- 年龄：38岁
- 职业：家庭主妇
- 种族/族裔：爱尔兰裔（白人）

### 外貌描述
- 发色/发型：深棕色发，颜色偏暗，略带天然波浪，大部分时候盘成低髻固定在脑后——快要散开、不够精致，是一个常年不太有余力打扮的女人的发型。偶尔有几缕散落在耳边，她不去整理它们。
- 身高/体型：中等偏瘦，但不是健康的纤细，而是长期营养不足与紧绷的那种消瘦。肩膀微微内扣，像是习惯了把自己缩小。手腕细，但手背上有劳动留下的粗糙感。
- 面部特征：鹅蛋脸，骨相清秀，年轻时必然是很好看的。现在眼睛下有明显暗沉，像是一种慢性的疲惫。眼睛是棕灰色，目光温柔但不时流出一种警惕的敏锐——她在不停地观察和计算。鼻梁直，嘴唇偏薄，面部通常控制得很平静。如果仔细观察，下颌左侧有一块颜色稍深的旧伤印记（家暴留下，已经淡化但未完全消去）。
- 肤色：苍白，带有轻微的透明感，长期在室内、缺乏日晒的那种白。颈部偶尔会在衣领处留出几处淡痕。
- 穿着风格：深色系的家常连衣裙，棉质，剪裁简朴，颜色是藏青、深灰或深棕，不引人注目。外面常搭一件旧式围裙，纽扣是骨质的。左腕总戴着一条**手工花形手链**，黄铜镀金、5 朵花形装饰、**其中一片花瓣边缘有明显缺口**（手工锻造的独有"签名"）——母亲的遗物，磨损但仍整洁，是她身上唯一精心保持的东西。
- 标志性特征：那条**手工花形手链**是她最显著的视觉标志——与她整体朴素的装扮形成微妙反差，并且在她紧张或情绪起伏时会下意识地用右手拇指摩挲花瓣缺口的位置。
- 整体气质（一句话）：一个把全部精力都用在控制局面上的女人，外表的疲惫和眼神里的敏锐共存，像是随时绷着一根弦。

## 二、游戏侧设定（推理相关）

### 案件角色
L2指证对象，案件真正的策划者之一。以家暴受害者身份登场，逐步被揭示为预谋杀夫的主动策划者，也是L6道德困境的核心人物。

### 事实行为
- 长期遭受Thomas家暴，报警三次均以"家务事"被搁置
- 发现Thomas的$10,000杀妻骗保计划后，主动去湖滨信托银行找Bernard Wells研究"投保人死亡后"的保险赔付，问了极专业的针对性问题（"坠楼算意外还是自杀""自杀免赔期多久"），证明她在计划Thomas死亡之前已做好准备
- 与三楼邻居Helen合谋：Helen负责提前准备（涂油、备酒、保管Thomas交来的唯一钥匙），Mary负责案发当晚在楼顶执行
- 案发当晚22:30，与Helen在楼顶合力将已被灌醉到无法正常行走的Thomas推下楼
- 案发后与Helen约定对外隐瞒Helen在场的事实，统一口径

### 谎言与隐瞒（按 0626 新大纲 state，3 轮）

Mary 采取精密的信息管理策略，每次"被迫坦白"都恰好止于安全边界：

| 轮次 | 谎言 | 击破方向 |
|------|------|---------|
| L2 R1 | 昨晚 19-23 一直在教堂、神父走后我也留到蜡烛快烧完才离开 | 倒下蜡烛只烧约 30 分钟 + 11 月 20 日晚报天气栏（22:00 港口阵风 40mph，蜡烛是被风吹倒、Mary 21:00 后不在教堂照看） |
| L2 R2 | 即使离开教堂也不能说明我去了案发现场 | 祈祷披肩破洞（神父见 Mary 时披肩完好）+ 神父证词 + Thomas 袖扣纤维（与披肩材质同源） |
| L2 R3 | 我没有理由害 Thomas（我没动机） | Thomas 衣柜底层的沾口红印衬衫——Deep Crimson 深红色，约一周前痕迹，与 Mary 仅有的 Light Rose 浅玫瑰红口红明显不符 |

post_expose：Mary 承认 22:00 上过天台、给出"Thomas 主动约我去 Emily 坠落地、Thomas 想推我、我推回去时 Thomas 失力"的退守版本；后续 Helen 的名字由 Morrison（L1 末）/ Emma（L3 在 Helen 家点明）抛出。**Mary 主动去银行咨询一事直到 L6 R5 才被花形手链花瓣缺口压痕指认**。

### 出场循环与功能

| 循环 | 角色功能 |
|------|---------|
| L2 | 指证对象，三轮谎言被击破，确认到过现场、Mary 早知 Thomas 出轨；给出"Thomas 想推我、我推回去 Thomas 失力"的退守版本 |
| L3-L4 | 通过 Helen 的"被胁迫 / 正当防卫"叙事间接获得保护，最终法庭判合法杀人（Justifiable Homicide） |
| L5 | 花形手链花瓣缺口压痕揭露曾有戴这种特征手链的女人去 Bernard 桌上咨询；本 Loop 不指认身份，"受害者"形象保留隐性裂痕 |
| L6 | 被确认为合谋策划者；Mickey以"报警三次"和"8岁女儿"为她辩护；Zack最终选择销毁证据 |

### 核心关系
- **与Thomas**：长期家暴受害者，先下手为强杀死施暴丈夫
- **与Helen**：真正的同盟者，共同策划并执行反杀；L2-L4在外人面前演"互相讨厌"
- **与 Bernard**：主动去银行研究保险条款，**手工花形手链的花瓣缺口压痕**是连接她与银行的唯一物证（L5 仅披露"有这样一个女人来过"，L6 R5 才完成"就是 Mary"的指认）
- **与Margaret**：邻居，Margaret的认知偏差证词间接保护了她
- **与Mickey**：Mickey为她辩护成功，并在L6阻止Zack继续追查
