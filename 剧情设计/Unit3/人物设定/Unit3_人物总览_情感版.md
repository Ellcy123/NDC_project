# Unit3《漫长的坠落》人物总览（情感版）

九个角色，一桩坠楼案。每个人都在用自己的方式回应同一个问题：**当法律无法保护弱者时，你愿意走多远？**

---

## Thomas Sullivan — 废墟中的危险

**42岁 / 失业工厂工人 / 爱尔兰裔 / 男**

脏金发掺杂灰白，从未认真梳理。6英尺，年轻时壮实的工厂体型已松弛走形，有明显啤酒肚，但手上还留着劳动老茧和打架硬茧。浅灰绿色眼睛因酗酒常年浮肿发红，鼻梁有一道旧伤弯折，面颊通红，肤色暗红夹蜡黄。脱线的粗呢工装外套口袋里常塞着瓷质小酒壶，鞋跟磨平，鞋带断了重新打的结。走路脚步偏重，总发出沉闷的皮鞋声。

**一句话气质：** 曾经可以是镇上最壮的汉子，现在是一个被酒和债务泡烂的、仍然危险的废墟。

**情感内核：** Thomas是这个故事的原罪。他不是一个单纯的恶人，而是一个被时代和自身缺陷碾碎后、把碎片砸向身边人的男人。失业、$12,000的债务滚雪球、走投无路——他选择了最卑劣的出路：为妻子投保$10,000，计划把Mary从楼顶推下去伪装成意外。他甚至以毁掉名声威胁邻居Helen帮忙。但讽刺的是，案发当晚他已被灌醉到血液酒精浓度0.25%——连走直线都做不到，更不可能攻击任何人。他从施暴者变成了被反杀者，从楼顶坠落时只留下本能的一抓。Thomas死了，但他的暴力留下的伤痕，驱动着之后每一个角色的选择。

---

## Mary Sullivan — 绷紧的弦

**38岁 / 家庭主妇 / 爱尔兰裔 / 女**

深棕色发盘成快要散开的低髻——一个常年没有余力打扮的女人。中等偏瘦，肩膀微微内扣，像习惯了把自己缩小。骨相清秀，年轻时必然很好看，现在眼下有明显暗沉。棕灰色眼睛温柔却流出警惕的敏锐——她在不停地观察和计算。下颌左侧有一块颜色稍深的旧伤印记（家暴留下，已淡化未完全消去）。深色家常连衣裙朴素到不引人注目。右腕总戴一条银质手链，吊坠是小小的雏菊造型——母亲遗物，是她身上唯一精心保持的东西。紧张时下意识用拇指摩挲吊坠。

**一句话气质：** 一个把全部精力都用在控制局面上的女人，外表的疲惫和眼神里的敏锐共存，像是随时绷着一根弦。

**情感内核：** Mary是本案最复杂的情感中心。她报警三次，三次都被当作"家务事"搁置。法律抛弃了她，于是她自己动手。她去银行研究保险条款时问的问题精准到令人心寒——"坠楼算意外还是自杀""自杀免赔期多久"——这不是一个绝望的受害者，这是一个冷静的策划者。她与Helen合力将醉到不省人事的丈夫推下楼顶。案发后，她精密地管理着每一次"被迫坦白"，每次恰好止于安全边界。但支撑她走到这一步的，不只是恨，还有那个8岁的女儿——她不能让女儿在Thomas的拳头下长大。玩家在L6面对的最终选择，本质上是在问：**一个被社会系统性遗弃的家暴受害者，她的预谋反杀，还能被称为正义吗？**

## Helen — 关着的门

**40岁 / 无业独居 / 白人 / 女**

9个角色里唯一真正的黑发，盘成严谨的高髻，不留任何碎发——与其说是发型，不如说是一种盔甲。最高的女性角色，高挑纤细，脊背挺直，动作缓慢精确，很少有多余动作。高颧骨，面部轮廓偏硬，暗棕近黑的眼睛极少眨，与人对视时让对方不适——不是敌意，是一个没有任何东西可以失去的人的眼神。深色长裙保守地遮住手腕和脚踝，纽扣系到顶，没有任何首饰、任何装饰。平底皮鞋走路几乎无声。墙上贴满女性权益报纸剪报，是房间里唯一有情感投入的东西。

**一句话气质：** 一个把悲伤和愤怒压缩成了冷漠的女人，外表像一扇关着的门，里面储满了足以炸开一切的东西。

**情感内核：** Helen的一切行动都指向8岁那年的记忆——目睹父亲将母亲家暴至精神失常。她母亲最后在轮椅上度过余生。帮Mary，是完成"当年8岁的自己想做却做不到的事"。她用母亲轮椅的维护油在楼顶布下陷阱，用母亲药柜里的物品作为工具——母亲的遗物成了复仇的武器。她在L3-L4中表演"被胁迫的帮凶"和"最后关头心软的女人"，每一层都恰到好处。但真相是：**油涂在Thomas一侧**，陷阱从一开始就是针对他的。Helen不是心软的旁观者，她是从头到尾的主动合谋者。案件结束后她离开芝加哥，像一扇终于完成了使命的门，安静地关上。那种把悲伤和愤怒压缩成冷漠的能力，是她用一生练就的。

## Morrison — 认命的人

**约50-55岁 / 警探 / 白人 / 男**

盐椒灰发用发油压得服服帖帖——不是为了好看，是一个职业警探"应有的样子"。中等偏矮，体型偏胖，皮带总被顶出一圈。暗灰色眼睛疲惫而世故，总像在思考比当前场景更无聊的事，嘴角自然下垂，不说话也显得不耐烦。通常叼一支烟，说话时眼神飘走。深棕色粗呢外套肘部已磨出光泽，领带松掉半圈，警探徽章位置不够端正——唯一认真打理的是擦得够亮的黑皮鞋，某种职业习惯的残留。

**一句话气质：** 一个见过太多、期待太少、把"认命"当成人生哲学的中年警探，腐败不是因为野心，而是因为懒得坚持原则。

**情感内核：** Morrison想把Thomas案草草定性为自杀结案，动机是双重的：Continental银行打过招呼（判自杀可没收房产抵债），同时他对家暴受害者Mary有真实的同情。这两个动机恰好指向同一个结论，让他可以心安理得地不去较真。被Zack两轮指证击破后，他疲惫认输，主动供出线索。他留下的那句警告——"真相不一定是你想要的"——不是威胁，是一个老警察对年轻同行真诚的忠告。他比任何人都早预见到，这个案子的真相会把Zack撕裂。

## Bernard Wells — 人形机器

**40岁 / Continental银行保险代理人 / 白人 / 男**

栗棕色发用发蜡梳得一丝不苟，中分线像用尺子量过——任何细节都必须在控制之中。中等偏胖，久坐型体型，坐姿永远端正。面部特征均匀，放在人群里不显眼——银行职员应有的"不留印象"的外表。细框椭圆金色眼镜后浅褐色眼睛平静而空洞，说话时唇部动作极小，像连表情都在节省。三件套西装没有一丝褶皱，领带夹每天位置相同。桌上永远放着同一款笔、同一摞文件。没有任何私人物品。

**一句话气质：** 一台穿着三件套西装的人形机器，20年的银行工作把他磨成了一个精准但彻底空洞的职员——最可怕的地方在于他根本不知道自己有多可怕。

**情感内核：** Bernard是"平庸之恶"的完美载体。他手把手教Thomas如何让"意外看起来合理"，又在不知情的情况下接待了来研究"反向保险条款"的Mary。他不恨任何人，也不爱任何人。客户问什么他就答什么，提成照收，良心不痛。当Zack追问时，他的防线不是愤怒或恐惧，而是冷漠的推卸："这是我的工作。"每次说这句话语气都一模一样，让人越来越觉得不对劲的那种一致。他是系统性犯罪机制里的一颗螺丝钉，拧紧了就转，不问机器在碾碎什么。

## Margaret Brennan — 爱得太深的母亲

**约55-60岁 / Zack的母亲 / 爱尔兰裔 / 女**

银白夹灰的发丝里鬓角偶尔还留着一缕暗红的光——年轻时爱尔兰红发的痕迹。9个角色里最矮的，但肩膀宽阔，体型扎实——从未停止劳动的女性特有的厚实感，像多少年来她自己就是家里最牢固的那根柱子。笑纹和愁纹叠在一起的宽脸盘，深蓝灰色眼睛温暖而坚定——见过太多苦难但还是选择相信别人。朴素连衣裙领口一圈细碎花边——在实用和体面之间的平衡。手上戴着婚戒，即便丈夫已不在。说话时双手习惯交叠放在腹前——一个习惯了等待和承受的人的姿势。

**一句话气质：** 一个用整个身体撑过了几十年风浪的爱尔兰母亲，矮小但不弱小，她的善良和她的认知偏差来自同一个地方——她爱得太深、看得太久。

**情感内核：** Margaret是本案最心碎的角色。她的证词直接推动了"正当防卫"判决——她说看到了Thomas攻击Mary的剪影。但物理上，从地面仰角65度、有护栏遮挡、深夜无照明的条件下，她不可能看清谁推了谁。那部分"记忆"是十年目睹Thomas家暴Mary的经验自动填补的——**她的善良无意识地改写了她的眼睛所见。** 她真诚地相信自己看到的就是真相，没有主观撒谎。L6，当Zack（她的亲生儿子）发现母亲的证词关键细节不可能时——这是全案最私人的情感冲击。他必须在"真相"和"母亲"之间做出选择。而Margaret永远不会知道自己的爱造成了什么。

## Mickey Donnelly — 温暖的铠甲

**30岁 / 辩护律师 / 爱尔兰裔 / 男**

深红棕色Auburn发——9个角色里最暖的发色，光线下透出红铜光泽，比Bernard的油光梳理更有生气。中等偏高，精干，肩宽腰紧，西装穿在身上有型。方下颌，笑起来有深深法令纹。绿灰色眼睛敏锐而温暖——能用眼神让人觉得被认真听见了，说话时习惯用眼神锁定对方，不是压迫，是说服。三件套西装剪裁比Morrison精良，保养比Bernard更有温度——Bernard的西装是制服，Mickey的是铠甲。出入法庭和酒吧穿同一套，不区分场合。

**一句话气质：** 一个30岁就已经掌握了说服力和魅力的律师，法庭上的精准和朋友间的温度在他身上并不矛盾——但这种完美本身，在Unit 4会成为最大的警示。

**情感内核：** Mickey是Zack的童年好友，12年后在Unit2重逢。他选择为Mary辩护，不是因为不知道真相——他知道，或至少高度怀疑Mary预谋。但他选择以"人性正义"超越法律条文。L6的五轮控辩对决，他从证据辩护逐步转向情感呼吁，最后转向请求：**"她报了三次警，没有人救她。她有一个8岁的女儿。"** 他揭示Foster、Margaret都在保护Mary——整个社区都在用自己的方式，做法律没有做到的事。L0与Zack举杯时的笑是他最放松的时刻，L5"别再查了"时的眼神是他唯一的裂缝。他在Unit3中建立的道德权威，将在Unit4被揭露为"Whale"时形成最令人震惊的反差——**一个你信任的人，一个你认为代表正义的人，他的另一面是什么？**

## Seamus Byrne — 窗边的哨兵

**58岁 / 退休码头工人 / 爱尔兰裔 / 男**

白发夹银，发根残留着暗红色——年轻时浓郁的爱尔兰红发的幽灵。骨架宽阔，手掌巨大，但右膝旧伤（1912年港口事故）让他驼背偏重，比原本矮了一截。脸上纹路是大条的沟壑，码头日晒留下的红铜色是9个角色里最深的肤色。浅蓝色眼睛大部分时候有点空洞，一旦找到听众立刻亮起来，带着"终于有人来了"的兴奋。穿着亡妻手织的对襟毛线背心，磨损明显但仍穿着——未必是留念，更像是惯性。

**一句话气质：** 一个把一整栋楼当作自己全部社交生活的孤独老人，窗边的位置是他的哨位，周围每个人的行迹是他仅剩的消遣，但他本人对此毫无自觉。

**情感内核：** Seamus与案件无关，他只是一个孤独的老人。他看到Helen两周前借钥匙时手在抖，看到Thomas频繁上三楼——他记录了这一切，但不知道自己记录的是犯罪准备。和Thomas偶尔喝酒打牌，Thomas死后他有点失落。见到Zack时极热情，话多到需要引导聚焦——**他太久没有人可以说话了。** 初看他对Helen的关注程度异常，像个跟踪狂（红鲱鱼），但实际上只是一个从不出门的女人突然频繁外出，这对一个以观察邻居为唯一消遣的人来说是最大的新闻。Seamus的悲伤不是剧烈的，而是慢性的——一个用窗户框住了整个世界的人，他本人对此毫无自觉。

## Dr. Foster — 理性的裂缝

**约35-40岁 / 法医 / 白人 / 女 / INTP**

浅金发淡得接近铂银色——9个角色里最浅、最冷的发色。随手绾的低松髻，几缕碎发垂在颈侧，她从不整理——"明明很好看但完全不打理"是她整个人的视觉核心。身形清瘦，白大褂穿在身上有利落的笔挺感，手指修长纤细，指甲剪得极短。五官精致但组合起来是"冷"而非"甜"的面孔，圆形金丝眼镜后灰蓝色眼睛有"正在分析你"的清冷光——她看人像在读检验报告。素颜，唇形天生好看，不笑时嘴角微微下压——默认的"思考中"状态。说话前偶尔有短暂停顿，在选择精确度最高的措辞。没有首饰，走路几乎无声。

**一句话气质：** 实验室里最安静也最好看的那个人，冷到让人不敢搭话、精准到让人不敢质疑——但正是这种完美的专业面具，在L6被Zack撬开时，露出的是她对Mary处境的深层共情，一个理性主义者罕见的感性裂缝。

**情感内核：** Foster是专业面具最完美的角色，也是面具裂开时最令人动容的角色。她的L1尸检报告写"相当高的浓度"而非"无法攻击"，写"不像剧烈挣扎，更像本能抓了一下"却不给结论——每一个措辞都在技术上准确，但精心回避了对Mary不利的推论。这不是疏忽，是一个INTP用最擅长的方式做出的选择：**用精确度本身制造模糊。** 她向Mickey私下承认："我不想让一个受害者因为技术细节被起诉。"一个以系统和逻辑为信仰的人，为了共情打破了自己最珍视的原则。L6，当Zack追问时，她说话前的停顿比平时更长——那不再是在选择最精确的措辞，而是一个理性主义者在承认自己做了一件感性的事。1920年代能做到法医的女性，不需要证明什么，她的存在本身就是答案。但正是这种不需要证明的强大，让她在选择保护Mary时的脆弱，格外真实。

## 情感主题总结

### 保护网络

这个案子最独特的地方在于：**几乎所有人都在保护Mary。** Morrison因为同情而草草结案，Helen因为童年创伤而合谋反杀，Foster因为共情而模糊措辞，Margaret因为爱而改写记忆，Mickey因为信念而辩护，Seamus虽不知情但提供的证词客观上帮助了调查。每个人的保护方式不同——有的出于职业伦理的妥协，有的出于个人创伤的投射，有的出于无意识的偏见——但他们共同构成了一张网，兜住了一个被法律系统遗弃的女人。

### 核心道德困境

**当一个报了三次警都没有人救她的家暴受害者，带着8岁的女儿，预谋杀死了施暴丈夫——你是坚持法律的字面正义，还是承认这个世界有些裂缝只能由人来填补？**

Zack最终选择销毁证据。不是因为Mary无罪，而是因为他发现：真相的对面不是谎言，是另一种真相。
