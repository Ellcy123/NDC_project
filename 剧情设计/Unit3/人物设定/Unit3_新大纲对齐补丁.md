## 1. Mickey Donnelly

### Unit3 事件层
- **L1 开篇（SC3011）**：登门 Zack 侦探事务所主动委托。理由——在起诉湖滨信托银行的过程中调查到 Thomas 案：死者欠贷款 + 巨额保险，与 Frank 案同模式，保单来自湖滨信托银行。Mickey 本人不参与调查，请 Zack/Emma 调查后到他办公室碰头汇报
- **L4（市政厅）**：宣布为 Mary 出任辩护律师；为 Mary 争取合法杀人判决 + 为 Helen 争取无罪判决
- **L5 开篇（SC3011）**：告知 "Mary 已无罪、Helen 还在争取"；要求 Zack/Emma 继续调查 Bernard
- **L6（SC3011 控辩对决）**：与 Zack 进行 5 轮控辩对决。最终投降认输，让 Zack 拿材料一起去找法官。Mickey 问 Zack 要不要告诉 Emma；Zack 决定不告诉

### state 角色定位
- 全程 is_liar: false；L6 形式上是 Expose 对象但他是友方辩护人，不是真说谎者
- 跨 Loop 隐藏：Mickey = Whale（Unit4 才揭）

### 关键证词 ID
- 3011001 / 3011002（L1 开篇委托）
- 3014001（L4 市政厅辩护立场）
