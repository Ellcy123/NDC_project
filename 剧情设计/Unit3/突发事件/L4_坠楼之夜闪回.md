# 突发事件：坠楼之夜闪回 (Fall Night Flashback)

**文档版本**: v1.0
**隶属章节**: Unit 3 - 循环4
**触发位置**: SC3010 Margaret鞋坊，Margaret描述目击经过时
**事件类型**: sudden_flashback（不可控记忆闪回——Margaret视角）

---

## 0. 前因后果

### 前因

- 玩家在L4自由探索阶段首次拜访Margaret鞋坊（SC3010）
- Margaret犹豫但最终开始讲述案发当晚的目击经过
- Margaret说到"然后有人掉了下来"——这句话触发闪回

### 关联记忆

- **Margaret的认知偏差**：Margaret在地面仰视三楼（65度仰角），深夜22:30仅有远处煤气灯微光，1.1米护栏遮挡——物理条件决定她**不可能看清楼顶细节**
- 但Margaret看了Thomas打Mary十年，大脑自动将模糊人影解读为"Thomas在攻击Mary"
- 这一闪回**故意让玩家体验Margaret的视角**——黑暗、模糊、看不清——但玩家在L4的关注点全在"坠楼"本身，大概率不会注意到"其实什么都看不清"

### 后果

- 闪回本身**不产生任何新证据**——Margaret的目击证词(3406001)通过正常对话获取
- 闪回建立Margaret对坠楼夜的创伤记忆——这是她"一直压在心里"的来源
- **L6 R5关键伏笔**：Mickey揭示Margaret证词的认知偏差时，出示3610（楼顶目击条件勘查记录），证明65度仰角+护栏+黑暗=不可能看清。此时玩家会突然想起L4的闪回画面——"对啊，那个画面里确实什么都看不清！"→ 恍然大悟

| 条件 | 说明 |

## 3. 闪回后反应

### Margaret的创伤反应

```
[闪回结束，闪黑淡出，回到Margaret鞋坊]

[Margaret坐在柜台后面，双手紧紧攥着围裙]
[手在发抖]

[完全静音——约2.5秒]
[只有Margaret的呼吸声，略微急促]

Margaret: "……我跑了。"
          （停顿1秒）
          "我什么都没做。"
          （声音更小了）
          "跑回家锁了门。"

[Margaret低头，不再看Zack]

[短暂沉默后，对话恢复正常——玩家可继续询问或结束]
```

### 设计要点

- Margaret的反应是**创伤后的重现(re-experiencing)**——讲述那一晚让她重新经历了那个画面
- "我跑了。我什么都没做。"——这句话的潜台词是**内疚**。Margaret觉得自己应该做点什么（报警？呼救？），但她吓坏了只顾着跑
- Margaret说"什么都没做"而非"我看到Thomas推Mary"或任何具体描述——她此刻是在表达情感，不是在分析画面
- Zack不评论、不安慰、不追问闪回的内容——他等Margaret自己说完
- 玩家体验两条理解路径：
  - **被坠落震撼的玩家**：全部注意力在"有人死了"上 → 情感冲击巨大 → 对Margaret的证词深信不疑
  - **冷静观察的玩家**：注意到闪回画面极暗极模糊 → "Margaret到底看到了什么？" → 为L6 R5做好心理准备

## 5. 视听设计

### 音效时间线

| 时间 | 音效 | 描述 |
|------|------|------|
| T+0.0s Margaret台词 | — | Margaret的声音开始变得遥远（混响增加、音量降低），像从水底听到的 |
| T+0.3s 闪黑#1 | `sfx_blackout_in` | 低频"嗡"声（不是L5的高频耳鸣——是更沉闷的恐怖感），0.15秒 |
| T+0.45~1.0s 人影 | `sfx_night_wind` | 强风声（40mph的11月夜风），持续贯穿整个闪回 |
| T+0.5~1.2s 挣扎 | `sfx_distant_voices` | 极模糊的人声——被风撕成碎片，听不清任何词句，只有音调变化（急促、尖锐、断续） |
| T+1.2~1.7s 坠落 | `sfx_falling_body` | 坠落的风切声（由远及近，0.5秒），终结于沉闷撞击 |
| T+1.8s 撞击 | `sfx_impact` | 低频撞击声（不做渲染，克制处理——一声闷响，无回音） |
| T+2.25s 闪白 | `sfx_impact_flash` | 撞击延伸——极低频震动（体感向，像心脏被一拳打中） |
| T+2.4s 闪黑#2 | `sfx_blackout_out` | 与#1相同的低频"嗡"声，略降调 |
| T+2.55~5.0s 静音 | `sfx_silence` | **刻意静音2.5秒**——只有Margaret轻微的呼吸声 |

### 音乐处理

- 闪回前：Margaret对话场景的温暖BGM（鞋坊是家庭场景）
- Margaret说"有人掉了下来"：BGM **骤停**（不是淡出，是立刻切断）
- 闪回期间：**无BGM**——只有风声、模糊人声、坠落声
- 闪回结束后：**不恢复BGM**——静音段持续到Margaret说完"跑回家锁了门"
- Margaret说完后：缓慢渐入一段极低音量的弦乐颤音（不安、沉重——与L3/L4的调查BGM不同，这是专属于创伤记忆的音乐）

### 视觉效果

| 效果 | 参数 |
|------|------|
| **闪黑** | 纯黑覆盖层，Alpha: 0→1→0，持续0.15秒，缓动EaseIn（比L5闪白更突然） |
| **夜视滤镜** | 极低亮度（Brightness -70）, 高噪点(Film Grain 40%), 冷色偏移(Color Temp -20), 低对比度(Contrast -30) |
| **仰角畸变** | 画面微透视变形（barrel distortion 5%），模拟极端仰视角度 |
| **画面晃动** | 随机微位移（X轴±3px, Y轴±5px），模拟Margaret仰头看时身体晃动 |
| **风向漂移** | 画面整体以0.5px/frame向右漂移，模拟强风环境 |
| **坠落跟随** | 第3段坠落时，镜头急速下移（1秒内从仰视→平视），伴随运动模糊(Motion Blur 50%) |
| **撞击抖动** | 画面在±15px范围内高频震动，0.15秒内衰减至0 |
| **Margaret回到** | 闪黑淡出后，Margaret的画面略微过曝0.5秒再恢复正常（从黑暗回到明亮鞋坊的视觉过渡） |

## 7. L6 R5 联动说明

此闪回与L6辩论R5（Mickey揭示Margaret认知偏差）形成**精确联动**：

| L4 闪回（玩家体验） | L6 R5（Mickey分析） |
|---------------------|---------------------|
| 画面极暗看不清 | "深夜22:30，最近煤气灯15米外" |
| 仰角65度，人影在护栏上方 | "65度仰角+1.1米护栏=只能看到肩部以上" |
| 看不清是推/拉/搀扶 | "推搡vs拉扯vs搀扶——从地面完全无法区分" |
| 人影数量不确定（2-3个？） | "楼顶有几个人？（只看到模糊轮廓）" |
| 坠落是唯一清晰的事实 | "坠落是真的——但原因从地面无法判断" |

**设计意图**：L4闪回让玩家在情感冲击中"以为自己看到了真相"；L6 R5让玩家意识到"你看到的和Margaret看到的一样——什么都没看清"。这个认知翻转只有在玩家亲身体验过那个黑暗模糊的画面后才有最大杀伤力。

## 8. 开发备注

- **美术资源**：
  - 楼顶夜景仰视画面（专门制作——不能复用L1/L3/L4的楼顶场景，因为视角完全不同）
  - 2-3个人形剪影轮廓（纯黑色，无细节，边缘需刻意模糊）
  - 坠落体轮廓（纯深色，运动模糊效果）
  - Margaret鞋坊场景中Margaret的创伤反应动画（颤抖、攥围裙、低头）

- **技术要点**：
  - 闪回画面推荐预渲染视频（1.8秒+撞击0.15秒，MP4/WebM）
  - 预渲染理由：极端噪点+仰角畸变+风向漂移+坠落跟随的组合效果实时渲染性能开销大
  - 闪黑用CanvasGroup + Alpha动画（与L5闪白共用组件，仅颜色不同）
  - 撞击抖动用Camera Shake脚本（高频阻尼衰减）
  - **关键**：闪回与Margaret对话台词的衔接必须精确——"有人掉了下来"的语音播放完毕后0.3秒触发，不能有感知延迟
  - 闪回前Margaret声音的"遥远化"处理：增加Reverb Wet Mix到80%，同时Volume降至30%，模拟"声音被拉远"的效果

- **QA测试**：
  - [ ] 闪回时序精确（0.3s延迟 → 0.15s闪黑 → 1.8s画面 → 0.15s闪白 → 0.15s闪黑）
  - [ ] 闪回画面噪点在不同分辨率下不会太轻（低分辨率）或太重（高分辨率）
  - [ ] 闪回期间玩家输入完全屏蔽
  - [ ] 撞击声的低频在不同音频设备上都能感知（手机扬声器可能需要额外处理）
  - [ ] 闪回后静音段（2.5秒）不被任何系统音效打断
  - [ ] 闪回只触发一次（重进Margaret鞋坊不再触发）
  - [ ] Margaret呼吸音效在静音段中的音量合适（可闻但不突兀）

- **设计备忘**：
  1. 闪回中的楼顶人影**绝对不能有任何可辨识的特征**——不能看出性别、不能看出衣着颜色、不能判断谁在推谁。如果做得太清楚（比如能看出一个人穿裙子），就泄露了Helen在场的信息（L4 Expose才揭示）
  2. 坠落是唯一确定的事件——这与Margaret的证词一致（"有人掉下来"是确定的，但"谁推了谁"是Margaret的**解读**）。闪回要忠实呈现"物理真实"：坠落是真的，但原因看不清
  3. 撞击声要**克制**——不做渲染、不加残忍细节。一声闷响即可。过度血腥会把情感从"震撼"变成"恶心"，削弱叙事效果
  4. Margaret说"我跑了。我什么都没做"是这个事件的灵魂——一个看了Mary被打十年的邻居，在看到有人坠楼时的第一反应是跑。这不是懦弱，是**普通人面对创伤的真实反应**。L6辩论中Mickey会说"每个人都选择了保护Mary"——Margaret的"保护"不是主动选择，而是认知偏差：她自动把模糊画面解读成了"Thomas在打Mary"，因为这是她十年来看过无数次的场景
  5. **与L5花形闪回的体验差异必须明显**——L5是暖色调、安静、私密的Zack记忆；L4是冷色调、恐怖、创伤性的Margaret记忆。两个闪回的情感质地完全不同：L5让你好奇，L4让你心跳加速
