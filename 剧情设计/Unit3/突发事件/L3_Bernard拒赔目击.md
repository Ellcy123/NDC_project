# 突发事件：Bernard拒赔目击 (Bernard's Claim Rejection Witness)

**文档版本**: v1.0
**隶属章节**: Unit 3 - 循环3
**触发位置**: SC3009 Continental银行公共区域，Zack首次进入
**事件类型**: sudden_witness（不可控目击事件）

---

## 0. 前因后果

### 前因

- 玩家正在调查Helen与Thomas的关系
- L3自由探索阶段，玩家首次前往Continental银行（SC3009）准备找Bernard Wells询问保险事宜
- 此前玩家仅通过Helen搜证了解到保险单(3304)和Bernard的名字——从未见过Bernard本人

### 关联设计

- **L5 Bernard指证**：Bernard被追问时反复以"这是标准服务(standard service)"为自己辩护
- 拒赔场景中Bernard的"合同就是合同"与L5"标准服务"形成前后呼应
- 被拒赔女人丈夫的"血液酒精浓度0.15%"暗扣Thomas的0.25%和保险排除条款

### 后果

- 事件**不产生任何证据**——纯粹的角色建立与氛围铺垫
- 玩家对Bernard的第一印象从"银行工作人员"变为"系统机器的执行者"
- L5 Expose时，玩家会回想起"他对每个人都这样"——Bernard的冷酷不是针对Thomas，是职业本能
- 银行的"系统性恶"主题在此播种，L5收割

## 3. 事件后衔接

### Bernard的转场

```
[女人离开后，Bernard坐回办公桌]
[整理刚才的文件——动作流畅、机械，像整理完一份普通业务]
[抬头看到Zack站在大厅]

Bernard: "下一位。请问您预约了吗？"

[正常对话流程开始——玩家恢复控制]
```

### 设计要点

- Bernard的"下一位"是事件的灵魂——他刚拒绝了一个寡妇的赔付请求，五秒后就准备好接待下一个客户，面无表情
- Zack没有任何评论。不说"这人真冷"、不皱眉、不叹气。**零主观引导**
- 玩家需要自己感受到不适——"这个人把拒绝寡妇当日常工作"
- L5 Expose R1 Bernard说"这是标准服务"时，有心的玩家会想起"对，他就是这样对那个女人的"
- "血液酒精浓度0.15%"在L3只是一个陌生的数字；L6辩论中Thomas的0.25%出现时，敏感的玩家可能会联想到"银行连0.15%都不放过，何况0.25%"

## 5. 视听设计

### 音效时间线

| 时间 | 音效 | 描述 |
|------|------|------|
| T+0.0s 推门 | `sfx_bank_door` | 厚重玻璃门推开声，伴随门铃轻响 |
| T+0.5~7.0s 目击 | `sfx_bank_ambient` | 银行大厅环境声——远处柜台交谈声、打字机"哒哒"声、墙上钟摆"嘀嗒"声 |
| T+1.5~7.0s 对话 | — | 女人和Bernard的对话声（混在环境声中，略微"远"，Zack在入口处听） |
| T+7.0s 女人离开 | `sfx_heels_marble` | 女人高跟鞋在大理石地面的脚步声，由远及近经过Zack，再由近及远消失 |
| T+7.5s Bernard | `sfx_papers_shuffle` | 整理文件的纸张声——干脆、机械 |

### 音乐处理

- 进入银行前：L3自由探索BGM正常播放
- **推门瞬间**：BGM淡出（1秒），只留环境音
- 整个目击过程：**无BGM**——只有银行环境声和对话声。冷，安静，日常
- "下一位"之后：BGM不立即恢复。等玩家走近Bernard开始正式对话后，才缓慢渐入一段低调的弦乐（与Bernard对话的BGM）

### 视觉效果

| 效果 | 参数 |
|------|------|
| **镜头推移** | Zack入门后，镜头缓慢推向Bernard办公桌方向，1.5秒，模拟Zack扫视银行大厅 |
| **色调** | 银行内偏冷色调——青灰色大理石、黄铜金属光泽、白炽灯的硬光。与Helen家的暖色调和Thomas家的破旧感形成对比 |
| **景深** | 目击过程中，Bernard和女人在焦点上，银行其他人在虚焦——引导玩家视线 |
| **女人离场** | 焦点跟随女人走向门口，经过Zack时短暂对焦到她的脸（泪痕），然后切回Bernard |

## 7. 开发备注

- **美术资源**：
  - Continental银行大厅场景（高天花板、大理石柜台、黄铜栏杆——1920年代银行典型内饰）
  - 被拒赔寡妇（一次性NPC）：黑色丧服、黑帽、30-40岁、眼眶红肿
  - Bernard首次出场外观：深色西装马甲、金属框眼镜、头发梳得一丝不苟、面无表情

- **技术要点**：
  - 整个目击序列为预渲染过场动画（推荐）或脚本驱动的实时演出
  - 预渲染方案：8秒视频片段，含推门+目击+衔接，预加载在场景transition期间
  - 实时方案：Zack自动移动+摄像机脚本+NPC动画序列，需精确时间控制
  - 对话字幕需配合远距离视角——字幕位置和大小可能需要调整
  - **关键**：目击过程中玩家输入完全屏蔽，不能走动、不能打开菜单

- **QA测试**：
  - [ ] 仅首次进入银行时触发（重进不触发）
  - [ ] 目击期间玩家输入完全屏蔽
  - [ ] 音乐淡出和环境音切换是否平滑
  - [ ] 女人经过Zack身边时的焦点切换是否自然
  - [ ] "下一位"后玩家控制恢复时机是否正确
  - [ ] 字幕在远距离镜头下是否清晰可读

- **设计备忘**：
  1. 被拒赔寡妇的丈夫是**工厂工人**——1920年代芝加哥蓝领阶层的真实困境。工伤死亡被"酒精过失"条款否决赔付，是这个时代保险公司的常见操作
  2. "血液酒精浓度0.15%"不是随便写的数字——它低于Thomas的0.25%，暗示银行的排除标准非常严格。哪怕只是聚餐喝了几杯（0.15%≈3-4杯啤酒），都足以被拒赔
  3. Bernard在这个场景中**不是反派**——他只是在执行合同条款。这种"执行规则的冷漠"比故意的恶更令人不安。L5 Expose揭露的不是Bernard个人的坏，而是整个银行体系的冷酷效率
  4. Zack的沉默是**刻意设计**——如果Zack说"这家伙真无情"，就变成了系统替玩家表态。让玩家自己判断，才是悬疑感
  5. 女人经过Zack时**不看Zack**——她不知道Zack是谁，也不在乎。她只是另一个被银行碾过的普通人。这种"无交集的擦肩而过"比对视更有力
