## 二、真实案发时间线（ground truth）

### 案发当日下午（同日 11-23 白天）

- **Helen 独自上楼顶涂油**：用母亲轮椅维护油涂护栏旁地面 1.5m，精确覆盖大铁箱死角的落脚位
- 涂油时被护栏锋利处割伤右手 → 对外解释"修轮椅划的"
- Thomas 看见 Helen 手伤，下午帮 Helen 开过门（Helen 无法用力握把手）→ 沾油把手内侧 = Thomas 拇指印（L4 油迹二方比对线索）

### 19:00（Mary 教堂建场）

- Mary 19:00 进教堂祈祷（爱尔兰社区 St. Patrick），故意被神父注意到
- 翻祈祷册到第 23 章背诵关键段落（为之后口供准备）

### 20:00（Mary 提前撤离 — 伪造不在场证明）

- 挪书签到第 23 章、点燃蜡烛掐灭 → 伪造"一直祈祷到很晚"
- 神父看到她离开，但 Mary 对外口供锚定 21:00 才走
- 蜡烛实际只烧 15 分钟（L2 R1 击穿点）

### 20:00–20:45（45 分钟空白）

- 教堂 → 公寓步行 10 分钟
- **多出 35 分钟用于和 Helen 对口径**（L5-L6 才能反推出"这不是恐惧犹豫，是合谋协调"）

### 21:00（Thomas 上 Helen 家）

- Thomas 主动找 Helen（以为是执行"杀妻骗保"计划的最后碰头）
- Seamus 在二楼走廊抽烟时目击（L3 证词来源）

### 21:30–22:00（Helen 灌酒）

- Helen 用 Jameson 威士忌（40% ABV，$3/瓶，Thomas 喝不起）持续给 Thomas 倒
- 对 Thomas 的话术：庆祝、壮胆
- 实际目的：把 BAC 拉到 0.25%——Thomas 失去基本运动能力

### 22:30（坠楼）

- 三人到楼顶。Mary 已在场（20:45 之后某时点上的，Helen 用借口稳住 Thomas 拖到此刻）
- 公开口径（L4 法庭版）：Thomas 醉酒扑向 Mary → Helen 从背后拉 → Mary 推开 → Thomas 翻过护栏
- 真实（L6 揭穿）：Thomas 0.25% BAC 站都站不稳，"扑向 Mary"是 Helen/Mary 编造的；地面油在 Thomas 站位 + 大铁箱死角 + 两个女人合力——Thomas 是被推下去的
- 22:30 Margaret 鞋坊收摊回家路过 → 听到争吵挣扎 + 仰视看到三人剪影晃动 + 男人坠落
- Margaret 长年看着邻居 Mary 挨打、报过三次警没人管 → 认知偏差把模糊剪影自动补完为"Thomas 在攻击 Mary"

## 四、玩家还原推理链（分 Loop 披露节奏）

### L1 自杀 → 他杀

- 击穿 Morrison "自杀"：38 码女性脚印 + 着地点距墙 <1m（自杀跳通常 1.5m+）
- 玩家认知：有第二人在场（女性），Thomas 不是主动跳下

### L2 Mary 撒谎

- L2 R1 击穿教堂不在场（蜡烛 15min + 祈祷册页码）
- L2 R2 大风+蜡烛不可能自然倒（因果机制闭合）
- L2 R3 衣柜底层衬衫珊瑚红口红印（Helen 色号）→ Mary 早就知道奸情
- L2 R4 Thomas 袖扣黑色蕾丝织物 = Mary 祈祷披肩 → 在护栏边肢体接触
- 玩家认知：Mary 在场、知道奸情、与 Thomas 有接触；但她主动供出 Helen 名字、不提 Helen 在楼顶
- **悬念**：第二组女性脚印（深浅不一）

### L3 杀妻骗保

- L3 R1 Thomas 钱包钥匙 ≠ 开 Helen 门 = 开楼顶安全门；Seamus 目击 Helen 配钥匙
- L3 R2 楼顶地面油 = Helen 家轮椅维护油
- L3 R3 $10,000 保单 + Bernard 证词"Thomas 问过怎么让意外看起来合理"+ Helen 在场旁听
- Helen 防线：被胁迫叙事，始终坚持"那晚不在场"
- 玩家认知：Thomas 是恶魔（家暴+杀妻骗保+胁迫邻居）

### L4 正当防卫

- L4 R1 楼顶门外侧把手油 + Helen 鞋底油迹 → Helen 当晚在场
- L4 R2 Jameson（Thomas 喝不起）+ Helen 家同品牌 + BAC 0.25%（需持续倒酒）
- L4 R3 Margaret 目击证词（三人剪影 + 肢体冲突）
- Helen 改口：扑向 Mary → 我拉住 → Mary 推开 → 翻过护栏；"我们商量好隐瞒我在场"
- 玩家认知：**正当防卫**（虚假结局），释然
- **隐藏伏笔**（玩家此时不该深究）：油在 Thomas 一侧 / Helen "检查油"≠ 擦油 / 0.25% 还能"扑"吗 / "商量好"太冷静

### L5 银行诱导 + Mary 裂痕

- L5 R1 报纸专业批注 + 银行内部缩写 ≠ Thomas 笔迹（小学文化）
- L5 R2 Bernard 垫板压痕 = 报纸批注（面对面教学）
- L5 R3 垫板边缘雏菊压痕 = Mary 手链吊坠 → Mary 来过 Bernard 办公室
- Bernard 被迫承认有女人问过"投保人死了怎么赔""自杀免赔期"——**反向角度**
- 玩家认知：银行系统性诱导 + Mary 提前研究 Thomas 死后的保险——"受害者"形象出现裂痕

### L6 控辩辩论

- Zack 用同样证据为控方：Mickey 主动辩护逐轮被击破
  - R1 0.25% BAC → Thomas 无攻击能力（破"扑向 Mary"）
  - R2 油在 Thomas 一侧 + Helen 检查油 → Helen 从头合谋（破"心软反悔"）
  - R3 45 分钟空白 + 时间线精配 → 合谋协调（Mickey："她报警三次"）
  - R4 雏菊压痕 + Bernard 供述 → 经济动机（Mickey："她还有 8 岁女儿"）
  - R5 Margaret 物理条件不可能看清 + 长年同情驱动认知偏差 → 证词不可靠
- Zack 辩论获胜但选择自己烧证据：母亲、Foster、Helen 都在保护 Mary，他选择加入

## 八、玩家情感弧锚点

| Loop | 玩家体验 | 情绪 |
|:---:|:---|:---|
| L1 | 自杀变他杀，第二人是女的 | 好奇 + 怀疑 Mary |
| L2 | Mary 撒谎+早知奸情+在场 | 同情中起疑（家暴受害者，但有动机） |
| L3 | Thomas 是杀妻骗保犯，Helen 被胁迫 | 厌恶 Thomas，同情两位女性 |
| L4 | Margaret 目击 → 正当防卫 | **释然**（虚假结局，案子似乎结了） |
| L5 | Bernard 系统性诱导 + Mary 去过银行问反向问题 | 不安（Mary 形象出现裂痕） |
| L6 | 辩论确认合谋 + Margaret 认知偏差 + Zack 自烧证据 | 道德撕裂（赢了辩论，放弃胜利） |

**两段式弧线：**
- L1-L4 让玩家彻底相信"Mary 是受害者、Helen 是善良的人"
- 信任越深 → L5-L6 反转冲击越强 → 最终问玩家："你会揭发她们吗？"
