**诡计的空间诗学：** 油只涂 Thomas 一侧（不是原"推 Mary"叙事说的 Mary 站位）+ 大铁箱视角死角强迫 Thomas 站到指定位置——L3 玩家以为是 Thomas 杀妻陷阱反噬，L6 才看出陷阱从一开始就是给 Thomas 准备的。

**钥匙的诡计：** 全楼只有一把天台安全门钥匙，原属管理天台的 Thomas。Thomas 在他以为的"杀妻计划"准备阶段把这把唯一钥匙交给 Helen 去"布置陷阱"——钥匙自准备期起就在 Helen 手中。案发当晚 Helen 用它带 Thomas 上楼，事后将钥匙藏进猫窝。L3 玩家只当它是"Helen 参与准备"的物证，L4 才用"唯一钥匙在 Helen 手里"反证她当晚必到过楼顶，L5-L6 再用"钥匙在 Helen 处 / Thomas 当晚怎么上的楼"逼出 Helen 主导。

## 二、真实案发时间线（ground truth）

### 案发前数日（准备期）

- **Thomas 交钥匙**：Thomas 是天台管理者、全楼唯一一把天台安全门钥匙持有人。他以为 Helen 在帮他执行"把 Mary 推下楼骗保"的计划，把这把唯一钥匙交给 Helen 去"布置陷阱"。钥匙自此落在 Helen 手中（L4 反证她当晚在场的关键链节点）。

### 案发前一天

- **Helen 送出最后一瓶油**：Helen 把家里最后一瓶轮椅润滑油送给二楼 Seamus，说"以后也不需要了"。母亲上月去世、复仇将了，油对她已无用——这句话 L6 才被反推出含义（Seamus 持油为 L6 证物）。

### 案发当日下午（同日白天）

- **Helen 独自上楼顶涂油**：用钥匙进入楼顶，用母亲轮椅维护油涂护栏旁地面 1.5m，精确覆盖大铁箱死角的落脚位
- 涂油时被护栏锋利处割伤右手 → 对外解释"修轮椅划的"（L3 R3 击穿点）
- 两手沾油。Thomas 看见 Helen 手伤，下午帮手伤的 Helen 递油壶 / 开门（Helen 无法用力握把手）→ Thomas 手上沾到油（L1"Thomas 手上有油"线索来源；玩家在 L1 不知其来源，L5-L6 才闭合）

### 19:00（Mary 教堂建场）

- Mary 19:00 进教堂祈祷（爱尔兰社区 St. Patrick），故意被神父注意到
- 翻祈祷册到第 23 章背诵关键段落（为之后口供准备）

### 20:00（Mary 提前撤离 — 伪造不在场证明）

- 挪书签到第 23 章、点燃蜡烛掐灭 → 伪造"一直祈祷到很晚"
- 神父看到她离开，但 Mary 对外口供锚定 21:00 才走
- 蜡烛实际只烧 15 分钟（L2 R1 击穿点）

### 20:00–20:45（45 分钟空白）

- 教堂 → 公寓步行 10 分钟
- **多出 35 分钟用于和 Helen 对口径**（L5-L6 才能反推出"这不是恐惧犹豫，是合谋协调"）

### 21:00（Thomas 上 Helen 家）

- Thomas 主动找 Helen（以为是执行"杀妻骗保"计划的最后碰头）
- Seamus 在二楼走廊抽烟时目击（L3 证词来源）

### 21:30–22:00（Helen 灌酒）

- Helen 用 Jameson 威士忌（40% ABV，$3/瓶，Thomas 喝不起）持续给 Thomas 倒
- 对 Thomas 的话术：庆祝、壮胆
- 实际目的：把 BAC 拉到 0.25%——Thomas 失去基本运动能力

### 22:15–22:30（Helen 用钥匙带 Thomas 上楼）

- Helen 用她持有的唯一钥匙，经三楼走廊把醉到 0.25% 的 Thomas 带上楼顶
- Mary 已先到楼顶等（20:45 之后某时点上去的）

### 22:30（坠楼）

- 三人到楼顶。公开口径（L4 法庭版）：Thomas 醉酒扑向 Mary → Helen 从背后拉 → Mary 推开 → Thomas 翻过护栏
- 真实（L6 揭穿）：Thomas 0.25% BAC 站都站不稳，"扑向 Mary"是 Helen/Mary 编造的；地面油在 Thomas 站位 + 大铁箱死角 + 两个女人合力——Thomas 是被推下去的
- 22:30 Margaret 鞋坊收摊回家路过 → 听到争吵挣扎 + 仰视看到三人剪影晃动 + 男人坠落
- Margaret 长年看着邻居 Mary 挨打、报过三次警没人管 → 认知偏差把模糊剪影自动补完为"Thomas 在攻击 Mary"

### 22:30 之后（Helen 离场 + 钥匙归位）

- Helen 经三楼走廊回家，门把手**外侧（楼梯间侧）**留下沾油痕迹（L4 R1 击穿）
- Helen 把唯一钥匙带回家，藏进猫窝（猫蹭到 → 猫尾巴 / 钥匙沾油，L3 线索）

## 四、玩家还原推理链（分 Loop 披露节奏）

### L1 自杀 → 他杀（Morrison · 2 轮）

- 楼顶护栏核心区被警戒线封锁（栏杆细节 L3 才开放）；L1 搜证重心在脚印、外墙、法医
- R1 击穿"现场只有死者一人"：约 37 码女性脚印，与 Thomas 靴印面对面，深夜 22:30 在场
- R2 击穿"他自己跳下去的"：外墙抓挠痕迹（坠落瞬间本能抓握）+ Foster 坠楼分析（指甲本能反应、姿态不支持主动跳）；着地点距墙 0.8m 作辅助
- 新线索（不在指证中，作伏笔）：Thomas 手上有油（来源未知）/ Jameson 空瓶来源存疑
- 玩家认知：有第二人在场（女性），Thomas 不是主动跳下

### L2 Mary 撒谎（Mary · 4 轮）

- L2 R1 击穿教堂不在场（蜡烛 15min + 祈祷册页码）
- L2 R2 大风+蜡烛不可能自然倒（因果机制闭合）
- L2 R3 衣柜底层衬衫珊瑚红口红印（Helen 色号）→ Mary 早就知道奸情
- L2 R4 Thomas 袖扣黑色蕾丝织物 = Mary 祈祷披肩 → 在护栏边肢体接触
- 玩家认知：Mary 在场、知道奸情、与 Thomas 有接触；但她主动供出 Helen 名字、不提 Helen 在楼顶
- **悬念**：第二组女性脚印（深浅不一）

### L3 杀妻骗保（Helen · 3 轮）

> 楼顶护栏核心区在 L3 解封，第二组脚印 / 护栏破坏 / 护栏血迹 / 地面油此时才可搜
> 钥匙不再用于指证击穿——它在 L3 只作"Helen 参与准备"的物证（猫窝里的唯一钥匙、沾油），真正发力在 L4

- L3 R1 击穿"只是聊天诉苦，没有别的联系"：Bernard 证词——Helen 在场旁听意外赔付咨询 → 关系不是普通邻里
- L3 R2 击穿"Thomas 良心发现给 Mary 买保险"：$10,000 保单受益人是 Thomas → 杀妻骗保
- L3 R3 击穿"我承认 Thomas 想骗保，但我从没去过楼顶"：Helen 右手包扎伤 + 护栏锋利处陈旧血迹 → Helen 上过楼顶、参与布置陷阱
- Helen 防线：承认布置陷阱（毁护栏+涂油），表演被胁迫，**始终坚持"案发当晚不在场"**
- 玩家认知：Thomas 是恶魔（家暴+杀妻骗保+胁迫邻居）；猫窝里那把唯一钥匙先记着

### L4 正当防卫（Helen 补充 · 4 轮，假结局）

> 门把手方向定义：**外侧 = 楼梯间侧，内侧 = 天台侧**

- L4 R1 击穿"当晚没去过楼顶"：楼顶门外侧（楼梯间侧）把手油迹 + Helen 鞋底油迹 → Helen 当晚沾油从楼梯间侧上过楼顶
- L4 R2 击穿"油是下午做陷阱时蹭到的"：门内侧（天台侧）干净——若是下午布陷阱后从天台侧离开，内侧把手应有油；内侧干净 = "下午蹭到"说法不成立，是当晚另一次进入
- L4 R3 击穿"陷阱是前一天做的，当晚我没钥匙上不去"：楼顶唯一钥匙（原属 Thomas，案发后在 Helen 猫窝）→ 全楼仅此一把，钥匙在 Helen 手里 = 她当晚必能进入并带走钥匙
- L4 R4 击穿"Thomas 还没掉下去我就走了"：Margaret 目击证词（三人剪影 + 肢体冲突）→ Helen 留到坠楼时刻
- Helen 改口（假结局）：扑向 Mary → 我拉住 → Mary 推开 → 翻过护栏；"我们商量好隐瞒我在场"
- 玩家认知：**正当防卫**（虚假结局），释然
- **隐藏伏笔**（玩家此时不该深究）：油在 Thomas 一侧 / "检查油"≠ 擦油 / 0.25% 还能"扑"吗 / "商量好"太冷静 / Thomas 手上的油哪来的 / 钥匙在 Helen 处 Thomas 当晚怎么上的楼
- **L4 AVG**：报社（Mary+Helen+Emma 采访）/ 市政厅门口（Harrison 法官+Mickey 争取公正，立 Mickey 正义形象）

### L5 银行诱导 + Mary 裂痕（Bernard · 2 轮）

- L5 R1 报纸专业批注 + 银行内部缩写 ≠ Thomas 笔迹（小学文化）+ 垫板压痕 → Bernard 面对面教学
- L5 R2 垫板边缘雏菊花形压痕 = 有女人来访（指向 Mary）→ Bernard 承认有女人问过"投保人死了怎么赔""自杀免赔期"等**反向角度**问题
- **结尾 Emma 抛出油+钥匙双疑点**（为 L6 翻案蓄力）：
  - 若 Helen 当晚因手上有油导致门外把手沾油，那 **Thomas 手上的油**又是哪来的？
  - 钥匙在 Helen 处，**Thomas 当晚是怎么上的楼顶**？
  - Emma 开始怀疑报社/法庭过早定调
- 玩家认知：银行系统性诱导 + Mary 提前研究 Thomas 死后的保险——"受害者"形象出现裂痕

### L6 控辩辩论（Mickey · 5 轮）

- Zack 用同样证据为控方；Mickey 主动辩护逐轮被击破，防线：证据辩护(R1-R3)→情感呼吁(R4)→请求停止(R5)
  - R1 破"Thomas 仍可能攻击"：Foster 补充分析 0.25% BAC 无攻击能力 → 退守"至少 Mary 不是策划者"
  - R2 破"Helen 最后心软没执行"：Jameson 空瓶 + Helen 家同品牌 + Thomas 廉价酒 → Helen 灌酒，从头合谋
  - R3 破"Helen 听到吵架才去、不然门把手油怎么解释"：Seamus 持有的最后一瓶油 + 门内侧干净 + 时间线 → **进入玩家 A/B/C 推理选项**
  - R4 破"即使有预谋也可能临时起意"：垫板花形压痕 + 比对报告 → Mary 到访银行、经济动机
  - R5 破"那你认为目击者看错了？"：楼顶目击条件勘查（仰角 65°+护栏遮挡+深夜无光）+ Margaret 视力暗示 → 认知偏差，证词不可靠
- **R3 玩家推理选项**——"当天涂油 / 留下油迹的人只能是"：
  - A：Helen 自己　B：Mary 和 Thomas　**C：Helen 和 Mary（正确）**
  - 推理演出，不开长期分支；选错 Mickey/Emma 短反馈后汇合到正确推理
- Zack 辩论获胜但选择自己烧证据：母亲、Foster、Helen、Mickey 都在保护 Mary，他选择加入

## 八、玩家情感弧锚点

| Loop | 玩家体验 | 情绪 |
|:---:|:---|:---|
| L1 | 自杀变他杀，第二人是女的 | 好奇 + 怀疑 Mary |
| L2 | Mary 撒谎+早知奸情+在场 | 同情中起疑（家暴受害者，但有动机） |
| L3 | Thomas 是杀妻骗保犯，Helen 被胁迫 | 厌恶 Thomas，同情两位女性 |
| L4 | Margaret 目击 → 正当防卫（假结局） | **释然**（虚假结局，案子似乎结了） |
| L5 | Bernard 系统性诱导 + Mary 去过银行问反向问题 + Emma 油+钥匙疑点 | 不安（Mary 形象出现裂痕） |
| L6 | 辩论确认合谋 + Margaret 认知偏差 + Zack 自烧证据 | 道德撕裂（赢了辩论，放弃胜利） |

**两段式弧线：**
- L1-L4 让玩家彻底相信"Mary 是受害者、Helen 是善良的人"
- 信任越深 → L5-L6 反转冲击越强 → 最终问玩家："你会揭发她们吗？"
