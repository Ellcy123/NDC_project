# SC3019 公寓六楼走廊

## 场景信息
- **位置**：Thomas 公寓六楼，通往天台的门外侧
- **类型**：过渡场景 / 搜证场景 / 天台门外侧
- **场景编号**：SC3019
- **内部英文名**：`u3_exp_6f_hall`
- **首次解锁**：L3
- **时间版本**：白天 / 夜晚两版；L3 使用白天过渡版，L4 使用夜晚油痕检查版。
- **多 Loop 解锁层级**：
  - L3：从三楼走廊上楼后到达这里，建立“天台门外侧”的空间位置
  - L4：重点检查天台门外侧把手油痕（油痕检测，扫描层显示）
  - L5/L6：默认不开放，除非后续流程重新回到天台线
- **联通关系**：从 SC3015 公寓三楼走廊点击上楼到达；点击天台门进入 SC3002 楼顶；没有其他可进入房间。

## 场景描述

公寓六楼走廊。这里不是完整住户层，而是窄小的天台门前平台。
- 画面尽头是一扇通往天台的老式铁门，门外侧黄铜把手位于玩家这一侧
- 墙面老旧、灯泡昏暗，空间比三楼走廊更短、更压迫
- 不出现明确可进入的住户房间；如需要楼层细节，只能作为不可交互的封闭墙面 / 管线 / 杂物
- 点击天台门进入 SC3002 楼顶，返回则下楼回到 SC3015 三楼走廊

整体感觉：这是案发现场的最后一道门。玩家应清楚意识到，Helen 从三楼上来后，先经过这里，再进入天台。

## 设计说明

1. SC3019 承接“天台门外侧”，避免把门外侧油痕直接画进楼顶内部。
2. 3402 属于玩家站在楼内走廊侧看到的把手；3403 属于进入天台后看到的门内侧把手。
3. 本场景只服务三楼上行楼梯 → 六楼走廊 → 天台的动线，不扩展其他住户探索。

## 美术参考
- 1928 年廉价公寓顶层走廊 / 楼梯平台：窄、旧、灯光低，墙面有水渍和剥落墙纸
- 天台门：老式铁门，外侧黄铜把手醒目；门缝可透出天台冷光
- 时间版本：白天版从楼梯间小窗 / 门缝进冷光；夜晚版主要靠昏黄廊灯和手电，油痕扫描层更清楚
- L4 油痕表现：默认底图不直接夸张油迹；使用检测仪后，门外侧把手握持处出现浅黄褐油痕 / 扫描高亮
