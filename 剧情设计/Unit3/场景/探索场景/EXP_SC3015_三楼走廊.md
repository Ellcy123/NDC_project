# SC3015 公寓三楼走廊

## 场景信息
- **位置**：Thomas 公寓三楼，Helen 家门口与上行楼梯之间（三楼唯一住户）
- **类型**：过渡场景 / 环境叙事 / 天台入口
- **场景编号**：SC3015
- **内部英文名**：`u3_exp_3f_hall`
- **首次解锁**：L3
- **时间版本**：白天 / 夜晚两版；L3 使用白天版，L4 门把手油迹空间锚点使用夜晚版。
- **多 Loop 解锁层级**：
  - L3：第一次来到 Helen 家前，建立 Helen 家与天台入口的物理距离
  - L4：门把手油迹证据的空间锚点，具象化"楼梯间侧"
  - L6：回访 Helen 家时保留过渡，氛围更安静
- **联通关系**：从 SC3006 二楼走廊继续上楼到达；一侧进入 SC3008 Helen 家，另一侧上行楼梯直接通往 SC3002 楼顶。

## 场景描述

公寓三楼走廊。三楼只有 Helen 一户，走廊狭窄、昏暗，墙纸剥落，楼梯间小窗透进一束灰白的光。
- 一侧是 Helen 的公寓门：旧木门、门牌、门口小鞋架
- 另一侧是继续向上的楼梯，通往六楼天台入口
- 玩家点击楼梯后直接抵达六楼天台入口 / SC3002 楼顶，不进入 4–6 楼中间住户层
- 走廊地面保持普通老旧状态，不放置独立油迹证据

整体感觉：近、窄、压迫。Helen 家和楼顶之间的距离应该被玩家一眼看懂：这不是遥远的两个地点，而是一段很短的犯罪动线。

## 设计说明

此场景不承载独立证据或 NPC 对话，功能是：
1. 让玩家直观看到 Helen 家和天台入口的距离很近
2. L3 自然过渡到 Helen 家，强化"她上天台很方便"的空间直觉
3. L4 把"楼梯间侧门把手有油"从抽象描述变成玩家走过的实际位置；油迹本身属于 SC3002 天台门把手扫描 / 证据层，不在本走廊底图显示
4. 明确本公寓每层只有一户，三楼只对应 Helen 家
5. 避免 4–6 楼中间层干扰：楼梯点击后直接跳到天台入口 / SC3002

## 美术参考
- 1928 年廉价公寓三楼走廊——窄、旧、墙壁剥落
- 时间版本：白天 / 夜晚两版。白天版有楼梯间小窗冷光，但整体仍不明亮；夜晚版几乎只靠廊灯和门缝光，强化 Helen 家到天台入口这段很短的犯罪动线
- Helen 家门比其他元素更醒目，但不要出现其他住户门
- 楼梯向上延伸，可用构图暗示纵深；交互上直接跳转到六楼天台入口
