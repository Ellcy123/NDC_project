# SC3003 Sullivan家 — 客厅/厨房

## 场景信息
- **位置**：Thomas公寓一楼，Sullivan夫妇住所的客厅与厨房区域
- **类型**：搜证场景 / 指证场景 / 过渡场景
- **场景编号**：SC3003（L2搜证/指证 / L4回访搜证 / L5搜证）

## 场景描述

一楼Sullivan家，破旧但整洁。客厅墙上有拳头凹陷和断裂椅子腿，厨房角落堆着30-40个$0.10-0.15的廉价烈酒空瓶。家暴痕迹随处可见，但角落里Emily的蜡笔画和小书包暗示这个家庭还有个努力正常生活的孩子。

L2是Mary的主场——开篇对话、搜证（客厅+卧室+Emily房间）、指证Mary四轮全在此完成。搜证覆盖衬衫口红印、外出鞋、家暴痕迹、Emily房间细节等。注：祈祷披肩（3205）和Mary手链（3207）为Zack搜身观察Mary获得，非搜证。

L4回访（SC3003）：重新搜查厨房，获取廉价酒照片和Thomas欠债记录，为"Thomas买不起Jameson"建立证据链。

L5搜证（SC3003）：发现标注报纸（1928年10月保险版面，有非Thomas笔迹的专业批注），指向银行内部人参与。
