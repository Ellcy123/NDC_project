# SC3008 Helen家

## 场景信息
- **位置**：Thomas公寓楼三楼，Helen独居住所
- **类型**：搜证场景 / 指证场景 / 对话场景
- **场景编号**：SC3008（L3搜证/指证 / L4补充搜证/指证 / L6对话）

## 场景描述

三楼Helen的独居住所。母亲上个月刚去世，空房间床头还摆着成排药瓶和18年的医疗账单。客厅整面墙贴满女性权益运动的报纸剪报。窗边是母亲的轮椅和工具箱（内有Standard Oil润滑剂油壶）。猫窝深处藏着一把铜制楼顶安全门钥匙，钥匙齿缝和猫窝边缘都沾有轮椅维护油。

L3搜证+指证：发现轮椅维护油（与楼顶油迹比对）、珊瑚红口红（与Thomas衬衫口红印一致）、猫窝里的楼顶唯一钥匙、猫尾巴/猫窝油渍，以及母亲照片、日记本、女性权益剪报墙、药瓶账单等伏笔证据。随后三轮指证击破Helen参与杀妻骗保计划和布置天台陷阱的谎言。钥匙不再与Thomas钱包钥匙做匹配，因为Thomas钱包钥匙线已删除。

L4补充搜证+指证：通过楼顶门把手油迹、门内侧干净、楼顶唯一钥匙和Margaret目击证词，四轮击破Helen"那晚在家"的谎言。Jameson威士忌保留为Helen正当防卫版本中的灌酒叙事伏笔，完整发力移到L6。

L6对话：判决后Helen的态度变化。
