# SC3008 Helen家

## 场景信息
- **位置**：Thomas公寓楼三楼，Helen独居住所
- **类型**：搜证场景 / 指证场景 / 对话场景
- **场景编号**：SC3008（L3搜证/指证 / L4补充搜证/指证 / L6对话）

## 场景描述

三楼Helen的独居住所。母亲上个月刚去世，空房间床头还摆着成排药瓶和18年的医疗账单。客厅整面墙贴满女性权益运动的报纸剪报。窗边是母亲的轮椅和工具箱（内有Standard Oil润滑剂油壶）。玄关小桌上放着一把铜制安全门钥匙。

L3搜证+指证：发现轮椅维护油（与楼顶油迹比对）、珊瑚红口红（与Thomas衬衫口红印一致）、楼顶钥匙（与Thomas钱包钥匙齿形一致），以及母亲照片、日记本、女性权益剪报墙、药瓶账单等伏笔证据。随后三轮指证击破Helen的杀妻骗保计划。

L4补充搜证+指证：发现Helen门口外出鞋（鞋底有油迹残留）和Jameson威士忌（与楼顶空瓶同品牌），三轮指证击破Helen"那晚在家"的谎言。

L6对话：判决后Helen的态度变化。
