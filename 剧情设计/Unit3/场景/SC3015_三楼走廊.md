# SC3015 三楼走廊

## 场景信息
- **位置**：Thomas公寓楼三楼，连接Helen家门与楼顶安全门的走廊/楼梯间
- **类型**：过渡场景 / 环境叙事
- **场景编号**：SC3015（L3开篇过渡 / L4证据空间呈现）

## 场景描述

三楼走廊，连接Helen家和楼顶的关键通道。一侧是Helen的公寓门（木门、门牌），往上走几步楼梯就是楼顶安全门（铁门、门闩）。走廊昏暗狭窄，墙壁剥落，唯一光源是楼梯间小窗透进的光。

这段不到十米的距离是全案物理空间的核心联通点：Helen从家出门→走过走廊→上楼梯→推开安全门→到达楼顶。L4 门把手证据（外侧把手有油/内侧干净="有人沾着油从楼梯间侧进入楼顶"）中的"楼梯间侧"就是这里。

### L3 — 开篇过渡

Zack和Emma从二楼走廊（SC3006，刚和Seamus聊完）上来，经过三楼走廊到达Helen家门口。走廊里能看到两个关键元素：Helen的公寓门和通往楼顶的楼梯。玩家第一次意识到Helen家和楼顶的物理距离如此之近——几步路而已。

### L4 — 证据空间呈现

Foster 在楼顶门把手上发现油迹后，这段走廊成为证据链的空间锚点：Helen 家（轮椅维护油）→ 三楼走廊 → 楼顶安全门外侧把手（油膜）。走廊本身没有独立搜证物件，但作为连接空间，它让"从楼梯间侧进入楼顶"这个表述变得具象可见。

### L6 — 环境叙事

判决后回访Helen，走廊的氛围有所不同——安静、日常，之前笼罩的紧张感已消散。

## 关键空间元素

| 元素 | 描述 |
|------|------|
| Helen家门 | 木门，门牌号，门口有鞋架（L3可见Helen外出鞋） |
| 楼顶安全门方向 | 走廊尽头往上的楼梯，铁门隐约可见 |
| 走廊地面 | L3/L4无明显痕迹（Helen走过但未留油——鞋底油量不足以留下地面脚印） |
| 楼梯间小窗 | 唯一自然光源，白天微光 |
| 墙壁 | 剥落的壁纸，老旧公寓质感 |

## 设计说明

此场景不承载独立证据或NPC对话，功能是：
1. **物理联通**：让玩家直观感受Helen家↔楼顶的距离（几步路），理解证据链的空间逻辑
2. **开篇过渡**：L3从SC3006上来，经过走廊到Helen家门口，自然过渡
3. **证据具象化**：L4"从楼梯间侧进入楼顶"不再是抽象描述，而是玩家走过的实际空间
4. **建筑真实感**：补全公寓楼的纵向空间——一楼（Sullivan家/门口）→ 二楼走廊（Seamus）→ 三楼走廊 → 楼顶
