## 证据详细卡片

### 3201 祈祷披肩 / clue_prayer_shawl

- **类型**：Clue（关键，Expose R2 主力；搜身观察，不可拾取）
- **位置**：SC3001 Thomas 公寓楼下入口（Mary 身披）
- **美术需求**：
  - Mary 身披深紫黑祈祷披肩，整体庄重、被认真爱惜过。
  - 右肩靠袖口有约 3 cm 新破洞，边缘拉丝，像被金属勾裂。
  - 下摆有轻微街尘，不直接暗示她上过天台。
- **美术参考**：1920s 爱尔兰天主教女性简朴披肩。

### 3202 Mary 的花形手链 / clue_mary_flower_bracelet · L5/L6

- **类型**：Clue（跨 Loop；搜身观察，不可拾取）
- **位置**：SC3001 Thomas 公寓楼下入口（Mary 左手腕）
- **美术需求**：
  - Mary 左手腕戴黄铜镀金花形手链，旧但被长期珍惜。
  - 五朵手工花里有一片花瓣缺口，缺口要能放大比对。
  - 它先像母亲遗物，不像“凶手标记”。
- **美术参考**：民间手工锻造，边缘不完全对称。
- **信息节奏**：本 Loop 客观记录缺口物理特征，不点破其含义。

### 3203 家暴痕迹（墙纸撕裂）/ envir_domestic_wall_damage

- **类型**：Envir（环境叙事，不收集、不进背包）
- **位置**：SC3003 Smith 家客厅（女儿房门旁）
- **美术需求**：
  - 女儿房门旁墙纸被大片撕裂，下面露出旧拳头印。
  - 撕裂边缘已旧，不是昨晚新痕，暗示长期家暴。
  - 画面要压抑，但不要直接画暴力场面。
- **美术参考**：廉价花纹墙纸、潮湿墙皮、昏暗一楼光线。

### 3204 Emily 的蜡笔画 / envir_emily_crayon_drawing

- **类型**：Envir（环境叙事，不收集、不进背包）
- **位置**：SC3003 女儿房间门口
- **美术需求**：

  - “妈妈”画得很大，手腕花形手链醒目；边角有深色模糊人影。
  - 纸张泛黄卷边，是 Smith 家痛苦里残留的温柔痕迹。
- **美术参考**：8 岁儿童笔触，透明胶老化翘起。

### 3205 Thomas 家的廉价酒照片 / clue_thomas_cheap_liquor · L6
