## 证据详细卡片

### 3102 外墙照片 / clue_wall_scratches

- **类型**：Clue（关键，Expose R1 主力）
- **位置**：SC3001 Thomas 公寓楼下入口
- **美术需求**：
  - 红砖外墙靠近坠落侧，有几道新鲜横向抓痕。
  - 抓痕露出灰白墙皮，像指甲本能扒住墙面后滑落。
  - 只表达“求生痕迹”，不画成夸张血痕。
- **美术参考**：旧公寓涂料剥落、清晨潮气、局部放大可读。

### 3103 11 月 20 日晚报 / item_nov20_evening_paper · L2

- **类型**：Item（文件，L2 R1 主力）
- **位置**：SC3001 Thomas 公寓楼下入口（警戒线外）
- **美术需求**：
  - 1928.11.20 晚报被风吹到警戒线外，边角潮湿卷起。
  - 天气栏是不起眼的小版块，但可读 22:00 后港口强风 40 mph。
  - 财经头版冷硬，和楼下命案形成“城市照常运转”的反差。
- **美术参考**：Chicago Tribune 旧报排版，铅字密、纸张泛灰。

### 3104 女性脚印照片 / clue_female_footprints

- **类型**：Clue（关键，Expose R2 主力）
- **位置**：SC3002 Thomas 公寓楼顶
- **美术需求**：
  - 楼顶薄露水泥地上有 6 码女鞋印，方向凌乱。
  - 鞋印边缘新鲜，能读出昨夜留下；不要明确画成“两个人”。
  - 与男靴印区分明显，但不要比现场其他痕迹更戏剧化。
- **美术参考**：窄前掌、低跟、细密鞋底纹。

### 3105 男性脚印照片 / clue_male_bootprints

- **类型**：Clue（陷阱，Expose R2 错误选项）
- **位置**：SC3002 Thomas 公寓楼顶
- **美术需求**：
  - 42 码男靴印靠近栏杆封锁线，大小与 Thomas 相符。
  - 靴印深、规整，看起来相关但不异常。
  - 它应像误导项：可信、普通、没有额外戏剧暗示。
- **美术参考**：1920s 工人靴鞋底，和女鞋印同一地面材质。

### 3106 Jameson 空酒瓶 / item_jameson_empty_bottle · L6

- **类型**：Item（跨 Loop，L6 R2 用）
- **位置**：SC3002 天台东南角避风处
- **美术需求**：
  - Jameson 酒瓶躺在天台避风角，品牌标签清楚。
  - 瓶口残酒很少，瓶身比 Thomas 家环境显得“过于体面”。
  - 本 Loop 只当零散物，不暗示 Helen 来源。
- **美术参考**：棕褐玻璃、完整标签、轻微指印但不做身份判断。

### 3107 袖扣纤维样本 / item_cufflink_fiber_sample · L2

- **类型**：Item（跨 Loop，L2 R2 用）
- **位置**：SC3005 法医办公室（解剖台旁证物袋）
- **美术需求**：
  - 透明证物袋内只有一小缕黑色织物纤维。
  - 标签写右袖扣、异源黑色织物，保持法医的冷静口径。
  - 纤维质地细而暗，后续能和 Mary 披肩形成视觉同源。
- **美术参考**：蜡纸窗证物袋，手写标签克制清晰。

### 3108 尸体手部油迹 / clue_hand_oil_traces · L6

- **类型**：Clue（跨 Loop，L6 R4 主力）
- **位置**：SC3005 法医办公室解剖台
- **美术需求**：
  - Thomas 指尖与掌心有黄褐油痕，手背保持干净。
  - 油痕嵌在掌纹里，像他自己碰过某个沾油表面。
  - 只记录现象，不表现油的来源或成分。
- **美术参考**：苍白手部与油痕形成清楚对比。

### 3110 尸体手部伤痕以及墙皮灰 / clue_hand_wounds_wall_dust

- **类型**：Clue（关键，Expose R1 主力）
- **位置**：SC3005 法医办公室解剖台
- **美术需求**：
  - Thomas 手指关节和掌侧有抓挠伤，指甲缝残留灰白墙皮。
  - 伤痕应是求生时本能抓握的乱痕，不像搏斗刀伤。
  - 和外墙抓痕能形成“身体与现场互相对上”的感觉。
- **美术参考**：苍白皮肤、红痕、粉末残留，局部放大即可。

### 3109 Foster 烟斗与医学院毕业照 / envir_foster_pipe_medical_photo

- **类型**：Envir（环境叙事，不收集、不进背包）
- **位置**：SC3005 法医办公室（书桌一角）
- **美术需求**：
  - Foster 书桌角落放石楠木烟斗，旁边是 1898 医学院毕业照。
  - 照片中年轻 Foster 站在第二排，气质严谨拘束。
  - 这不是证据，只让玩家感到他是老派、克制的专业人士。
- **美术参考**：黑白合影、旧木框、烟斗有长期摩挲光泽。

- **3104 女性脚印**：仅说“6 码、凌乱、暂不能判断人数”，不暗示“两人”。
- **3106 Jameson 空瓶**：只作角落零散物，不暗示来源。
- **3107 袖扣纤维**：只标“异源黑色织物”，不暗示祈祷披肩。
- **3108 尸体手部油迹**：仅记现象，不解读来源。
- **3102 + 3110**：让玩家组合外墙抓痕与手部伤痕，自己推出“不是自杀”。
