## 证据详细卡片

### 3401 油痕检测仪器 / Oil Trace Detector

- **类型**：Clue · Analysis（🎮 能力解锁）
- **位置**：SC3005 法医办公室（Foster 处）
- **物品描述**：
  - 便携式色谱比对仪，铜壳外观
  - 约 30×20×15 cm
  - 由 Miller 家族资金赞助开发（铭牌：「Sponsored by Miller Foundation, 1928」）
  - 操作面板：油样投入口 + 比对结果窗口
  - 可对油样进行品类匹配（轮椅维护润滑油 / 缝纫机油 / 食用油 等）
- **美术参考**：1920s 早期色谱仪外观——黄铜壳 + 玻璃管 + 旋钮
- **机制**：玩家拾取后，可对 L1/L3 已收集的油痕做品类匹配（解锁 3108 手部油 / 3304 钥匙油 / 3311 地面油 的成分分析）

### 3402 门外侧把手油迹 / Door Exterior Handle Oil

- **类型**：Clue · Analysis（关键，Expose R1 主力）
- **位置**：SC3002 楼顶天台门外侧把手
- **物品描述**：
  - 玻璃取样棒 + 取样照片
  - 取样位置：天台门外侧把手中段
  - 颜色：浅黄褐色油渍
  - 油痕检测仪比对结果：**轮椅维护润滑油**

  - 标注："门外侧 / 轮椅润滑油 / 同源"
- **美术参考**：玻璃取样棒 + 比对仪结果窗口截图

### 3404 Helen 家地面油迹 / Helen Home Floor Oil

- **类型**：Clue · Analysis（🟣 环境取样）

- **美术参考**：木地板 + 黄褐色油渍
