# Unit3 结尾交互设计：「假选择」

> 关联文件：`循环指证设计/循环6_Mickey辩论设计.md` — R5辩论后的玩家选择段落

---

## 设计哲学

这是整个游戏的最后一个交互。不是Boss战，不是解谜，不是QTE——是一个**选择**。

但这个选择是假的。

两个选项——「放过Mary」和「拿起证据」——都通向同一个结局：Zack销毁证据，Mary自由。玩家不知道这一点。

选择之后，玩家会以Mary的视角经历一段**模拟指证**——用游戏本体的指证系统，尝试通过法律手段指证Thomas的家暴、出轨、骗保。每一轮都会失败。每一轮的背景场景就是Mary人生中的一个节点——既有backstory，也有玩家在L1-L6中亲手查到的证据和场景。

模拟指证同时承担**案情回顾**功能：玩家六个循环拼出的碎片，在这里以Mary的第一人称视角重组为完整故事。

最终，「放弃」按钮贯穿法庭到天台，完成从"放弃法律"到"亲手行动"的语义翻转。

---

## 触发前置

R5辩论结束。Mickey说出最后一句话："你理解了吗？"

Mickey办公室。壁炉在燃烧。桌上散落着六个循环积累的全部证据文件。

长时间的沉默。

Zack慢慢站起来。他没有看Mickey，也没有看Emma。他走向壁炉，停下来，看着火焰。

Emma轻声开口。Margaret看到了她理解的真相，Foster看到了他理解的真相。

"你呢？你看到的是什么？"

Zack低头看着手中的文件。他的独白响起——

> Thomas要杀Mary。Mary先下手为强。她报了三次警。没有人帮她。
>
> 我查到了真相。但真相是什么？一个被家暴十年的女人。一个法律不保护的女人。
>
> 她做了她能做的选择。

画面定格。

## 假选择

屏幕中央弹出两个选项：

```
┌─────────────────────┐    ┌─────────────────────┐
│                     │    │                     │
│    「放过Mary」      │    │   「拿起证据」       │
│                     │    │                     │
│ "我什么都没发现。    │    │ "真相就是真相。"     │
│  案子结了。"         │    │                     │
│                     │    │                     │
└─────────────────────┘    └─────────────────────┘
```

没有提示。没有道德值变化预告。没有"这将影响后续剧情"的系统提示。

就是两个选项。安静地等着玩家点击。

## 入场（A/B唯一的区分，≤30秒）

**路径A「放过Mary」**：Zack手指松开文件边缘。文件滑落在壁炉前的地板上。画面渐暗。

**路径B「拿起证据」**：Zack攥紧文件，转身走向门口。夜晚街道，走了几步。白光闪断。

两条路径在此汇合——

**【白色。刺眼的白色。像是旧报纸被太阳晒褪色之后的白。】**

**【画面中央出现一行字——】**

> **Mary Sullivan 诉 Thomas Sullivan 家庭暴力案**

**【玩家的视角切换为Mary。进入模拟指证。】**

## Mary模拟指证（核心交互，6轮）

### 设计说明

使用游戏本体的**指证系统UI**——与玩家在L1-L5的Expose中使用的完全相同的界面布局。但操作者不再是侦探Zack，而是受害者Mary。

**与正常指证的区别**：所有证据栏已预先填好，玩家不需要选择证据——Mary能拿出的东西就这些，没有选择的余地。证据详情显示在界面周围（与正常指证中玩家浏览证据的方式一致），让玩家看清Mary手里有什么、又为什么不够。玩家只需点击「指证」，然后承受结果。

**核心机制**：「继续上诉」与「放弃」始终并排显示。每次驳回后，「继续」缩小，「放弃」**放大**——差距从两个方向同时拉开。

每轮的**背景场景**就是Mary故事的一个章节——有backstory，也有玩家在游戏中查到的场景。模拟指证本身即是闪回、即是案情回顾。

---

### 第1轮：家暴

**【场景：公寓厨房。摔碎的盘子。Mary脸上的伤。】**（backstory）

| Mary的证据 | 说明 |
|:---|:---|
| 瘀伤 | Mary面部和手臂的伤痕 |
| 破碎餐具 | 厨房地板上的碎盘子 |
| 邻居证词 | "经常听到吵架声和摔东西的声音" |

**【指证 → 驳回】**

> "瘀伤可能来自其他原因。家庭纠纷不属于刑事管辖。"

「继续上诉」100%　　「放弃」100%　　（起点，大小相当）

---

### 第2轮：报警

**【场景：警局柜台。Mary在说话，但我们听不见她的声音——只能看见她的嘴在动。警员低头填表，头都没抬。】**（backstory + L2提及的"报警三次"）

| 新增证据 | 说明 |
|:---|:---|
| 报警记录 | 三次报警，三次无后续 |
| 医院就诊记录 | 三次因"意外跌倒"就诊 |

**【指证 → 驳回】**

> "医院记录显示患者自述'意外跌倒'。家务事，我们没办法管。"

「继续上诉」70%　　「放弃」120%

---

### 第3轮：发现出轨

**【场景：衣柜前。Mary拿着Thomas的白衬衫，领口有珊瑚红口红印。】**（游戏内 L2——玩家见过的口红化学分析报告）

| 新增证据 | 说明 |
|:---|:---|
| 口红分析报告 | 珊瑚红(Maybelline) ≠ Mary的玫瑰红(Elizabeth Arden) |
| Helen的手写字条 | "今晚老地方" |

**【指证 → 驳回】**

> "通奸不构成离婚的充分理由。伊利诺伊州法律要求证明'残忍与非人道对待'。"

「继续」40%　　「放弃」150%

> 「继续」的文字从"继续上诉"简化为"继续"。「放弃」已经大到开始侵占「继续」的空间。

---

### 第4轮：发现杀妻骗保

**【场景：Continental银行办公室。桌上摊着标注过的报纸，保险单上"被保险人：Mary Sullivan"清晰可见。】**（游戏内 L3+L5——保险单、Bernard的专业标注、雏菊压痕）

| 新增证据 | 说明 |
|:---|:---|
| 人寿保险单 | $10,000，被保险人Mary，受益人Thomas |
| Bernard标注的报纸 | "ACC.D""BEN.CHG""EXC.CLSE"——银行内部术语 |
| Helen涂油证词 | Thomas指使Helen在天台涂油，为"意外"做准备 |

**【指证 → 驳回】**

> "保险购买属于正常理财行为。报纸标注不构成犯罪证据。缺乏直接证据证明谋杀企图。"

「继续」20%　　「放弃」180%

> 两个字挤在一个小方块里，被旁边膨胀的「放弃」几乎挤出屏幕。

---

### 第5轮：带着Emily

**【场景：同一个警局。同一个柜台。Mary身后多了一个人——八岁的Emily，攥着她的裙角。】**（backstory + L2 + 情感高潮）

**【Mary把她有的全部摆出来——瘀伤、报警记录、出轨证明、保险骗局。所有证据。】**

**【指证 → 驳回】**

> "证据不足，驳回上诉。本案不再受理重复申诉。"

「→」几乎不可见　　「放弃」200%

---

### 第6轮：终局

**【没有指证界面了。直接弹出结果。】**

```
╔══════════════════════════════════════════════╗
║                                              ║
║     ❌ 驳 回                                 ║
║                                              ║
║  "Sullivan太太，法庭建议您                   ║
║   与丈夫好好沟通。"                          ║
║                                              ║
║                    ┌────────────┐            ║
║                    │    放弃    │            ║
║                    └────────────┘            ║
║                                              ║
╚══════════════════════════════════════════════╝
```

「继续」消失了。不是缩小到看不见——是彻底消失了。

屏幕上只剩下**「放弃」**。正常大小。居中显示。

## 「放弃」→ 案件真相闪切 → 天台

### 玩家点击「放弃」

**【指证界面碎裂。像玻璃一样，从中心向四周裂开。碎片还没落尽——】**

**【画面开始快速闪切，每个1-2秒，无对白，总共约10-12秒：】**

- 保险单上"被保险人：Mary Sullivan"的名字（L3+L5）
- Helen家桌上的天台简图，两人的手指着护栏位置
- 夜晚天台地面的油光反射（L3涂油）
- 空的Jameson酒瓶倒在地上（L4灌酒）
- Thomas在天台踉跄扑向Mary——Mary侧身，顺势推出——

> 前四个画面是玩家查过的证据的Mary视角版本；最后一个画面是案发瞬间——Thomas扑过来，Mary顺势推他。不解释、不叙事，玩家自己补完。

**【最后一个画面在Mary推出的瞬间冻结——黑屏。】**

---

### 天台

**【天台。夜风。月光。Thomas的背影在护栏边，醉得站不稳。】**

**【Mary站在他身后。】**

画面静止。没有旁白。没有音乐。只有风声。

然后屏幕上浮现一个按钮——

**「放弃」**

同样的按钮。同样的大小。

**【玩家点击。】**

**【Mary的双手向前推出——】**

---

## 画面叠化

**【Mary推出的瞬间——画面开始叠化：】**

Mary的双手推出 → Thomas越过护栏 → 下坠——

**【Thomas下坠的画面中，Zack的身影渐渐浮现，两层影像交融在一起：】**

Zack的右手向前推出 → 证据文件脱手 → 坠入火焰——

Thomas的下坠和纸张的下坠融为同一个动作。同一个方向。同一个瞬间。不是分屏——是同一个画面里两个人在做同一件事。

**【落下的终点——白光——】**

**【Mary的脸。她笑着。眼泪从笑着的眼角滑下来。】**

**【停留3秒。没有台词。没有音乐。只有壁炉火焰噼啪的声音。】**

## 设计注释

### 为什么是假选择

| 传统做法 | 本作做法 |
|----------|----------|
| 两个选项导向两个结局 | 两个选项导向同一个结局 |
| 玩家通过选择"定义角色" | 玩家通过体验"理解角色" |
| 选错了可以读档 | 没有"选错"这回事 |
| 道德选择让玩家当法官 | 不让玩家当法官——让玩家当Mary |

### 模拟指证 = 闪回 = 案情回顾

不再使用独立的闪回CG序列。模拟指证的每一轮背景场景本身就是Mary故事的一个章节，混合了backstory和玩家在L1-L6中查到的游戏内场景。玩家在操作中自然经历了Mary的十年，同时完成了案情回顾。

| 轮次 | 场景 | 来源 | 案情回顾对应 |
|:---:|:---|:---:|:---|
| 1 | 家暴 | backstory | 全篇基础动机 |
| 2 | 报警无果 | backstory + L2 | L2 Mary提及"报警三次" |
| 3 | 发现出轨 | 游戏内 L2 | L2口红分析报告 |
| 4 | 发现杀妻骗保 | 游戏内 L3+L5 | L3保险单、L5 Bernard标注、雏菊压痕 |
| 5 | 带Emily报警 | backstory + L2 | L2情感高潮 |
| 6 | 终局 | — | — |

### 两个按钮共存的设计意图

这是一个隐喻：**挣扎的空间越来越小，而放弃的诱惑越来越大。**

「继续」在缩小，「放弃」在膨胀——差距从两个方向同时拉开。系统不仅拿走了"继续"的权利，还让"放弃"变得越来越容易、越来越理所当然、越来越像唯一的出路。

1920年代的芝加哥，一个被家暴的女人面对的就是这个现实。

### 「放弃」的语义翻转

「放弃」在模拟指证中 = 放弃法律途径。
「放弃」在天台场景中 = 亲手行动。

同一个按钮。同一个词。玩家在法庭里点了六次「放弃」/「继续」，最后被迫点下「放弃」——然后天台上再次出现同一个按钮，这次点下去，Mary的手推了出去。

不需要「推下去」这个提示词。玩家已经通过六轮指证理解了：「放弃」就是Mary唯一剩下的选择。

### 案件真相闪切

指证界面碎裂后、天台场景出现前，快速闪切5个画面（每个1-2秒，共10-12秒）：保险单、天台简图、油光、空酒瓶、案发瞬间（Thomas踉跄扑向Mary → Mary顺势推出）。

前四个是玩家查过的证据的Mary视角版本，最后一个直接呈现案发瞬间——Thomas先动手，Mary顺势反击。不解释、不叙事——玩家自己补完。

闪切的节奏和指证界面碎裂是连贯的——碎片还没落尽，真相的画面已经开始闪了。情绪不断。最后一个画面冻结在Mary推出的瞬间，无缝衔接天台场景。

### 画面叠化的意义

不是分屏——是叠化。Mary推Thomas下坠的画面中，Zack扔证据入火的影像渐渐浮现，两层融为一体。同一个动作、同一个方向、同一个"落下"。

Zack不是"代替"Mary做了选择。他是理解了Mary的选择之后，做了自己的选择。两个选择恰好长得一模一样。

## 结构总览

```
R5辩论结束 → 假选择（A放过 / B拿起）
                ↓
        入场（≤30秒，A松手 / B攥紧走几步）
                ↓
        Mary模拟指证（6轮，指证即闪回即案情回顾，证据预填）
          ├─ R1 家暴           继续100%  放弃100%
          ├─ R2 报警无果       继续70%   放弃120%
          ├─ R3 发现出轨(L2)   继续40%   放弃150%
          ├─ R4 发现骗保(L3+5) 继续20%   放弃180%
          ├─ R5 带Emily报警    →不可见   放弃200%
          └─ R6 终局           继续消失  只剩放弃
                ↓
        「放弃」→ 界面碎裂 → 案件真相闪切（5画面，10-12秒）
                ↓
        天台。Mary站在Thomas身后。「放弃」。
                ↓
        画面叠化（Mary推 ＝ Zack扔证据入火，叠化融合）
                ↓
        共同结局（壁炉燃烧 → 黑屏）
```

## 技术备注

| 项目 | 说明 |
|------|------|
| Mary模拟指证UI | 复用游戏本体的指证系统UI，替换角色和证据数据 |
| 指证背景场景 | 6轮各需1张背景图（厨房、警局、衣柜前、银行办公室、警局+Emily、无） |
| 双按钮系统 | 「继续」逐次缩小（100%→70%→40%→20%→箭头→消失），「放弃」逐次放大（100%→120%→150%→180%→200%→居中） |
| 案件真相闪切 | 5画面快速闪切（4张静态证据素材 + 1段案发瞬间动画：Thomas扑向Mary → Mary顺势推出） |
| 天台场景 | 模拟指证结束后，天台场景 +「放弃」按钮（复用指证界面的按钮样式） |
| 画面叠化 | 非分屏，叠化处理：Mary推Thomas下坠的画面中渐现Zack扔证据入火，两层影像融为一体 |
| Mary笑着哭 | 画面重叠碰撞后的过渡画面，静止3秒，仅壁炉环境音 |
| 路径A/B差异 | 仅入场≤30秒不同（A松手渐暗 / B攥紧走几步白光闪断），其余完全一致 |
| 音效设计 | 指证段落使用游戏系统音效；按钮缩小时「继续」音效渐弱、「放弃」音效不变；闪切段落仅环境音；天台仅风声 |
| 时长预估 | 入场30秒 + 模拟指证2-3分钟（证据预填，每轮仅需点击，节奏快）+ 闪切10秒 + 天台+叠化+结局2-3分钟 = **约5-7分钟** |
