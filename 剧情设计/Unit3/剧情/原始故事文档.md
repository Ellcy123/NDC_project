# Unit 3：杀夫反击 - 原始故事文档

---

## 📋 基本案情

| 项目 | 内容 |
|:---|:---|
| **案件名称** | Thomas Sullivan坠楼案 |
| **案发时间** | 1928年11月23日晚 22:00-23:00 |
| **案发地点** | 南区公寓楼（三层建筑） |
| **死者** | Thomas Sullivan（42岁，失业酗酒工人） |
| **死因** | 从三楼楼顶坠落，头部着地，当场死亡 |

### 核心悬念

| 层次 | 表象/真相 |
|:---|:---|
| 表面1 | Thomas因债务压力自杀 |
| 表面2 | Mary因嫉妒推Thomas坠楼（情杀） |
| 表面3 | Thomas和Helen计划杀Mary，Thomas失足 |
| 表面4（法庭判决） | Mary正当防卫 |
| **真相** | Mary和Helen联手预谋推Thomas坠楼（只有Zack和Mickey知道） |

## 🏢 场景设定

### 1. Thomas家（二楼）
- **客厅**：酒瓶、家暴痕迹（墙上的洞、破碎物品）
- **卧室**：Mary和女儿睡的房间，有简易门锁（防Thomas闯入）
- **厨房**：Continental银行催债信、保险单

### 2. Helen家（三楼）
- **客厅**：母亲遗像（1928年10月去世）
- **卧室**：整洁冷清，墙上贴满"女性权益"剪报
- **书桌**：日记本（部分内容被撕掉）

### 3. 楼顶（案发现场）
- **护栏**：Thomas坠落位置
- **地面**：脚印、酒瓶（Helen灌醉Thomas的证据）

### 4. 楼下坠落点
- Thomas尸体位置
- 头部着地，血迹
- "遗书"（在Thomas口袋里）

### 5. 教堂
- Mary的"不在场证明"地点
- 但Mary提前离开（20:00-20:45空白时间）

### 6. Continental银行
- 保险代理人办公室
- Thomas的贷款记录和"建议"记录

## 🎭 核心角色

### 主要角色

#### Thomas Sullivan（42岁，失业工人）
- 失业后向Continental银行借$5,000高利贷
- 债务陷阱，还不起，银行威胁收房
- 保险代理人"建议"：给妻子买保险，出点"意外"
- 给Mary买了$10,000保险，受益人是自己（杀妻骗保计划）
- 长期酗酒家暴

#### Mary Sullivan（38岁，Thomas的妻子）
- 长期遭受家暴，身上多处伤疤
- 多次报警，警察只是"劝和"（"家务事"）
- 有8岁女儿，无法独自逃离
- 发现Thomas的保险单后，决定先下手为强

#### Helen（40岁，三楼独居邻居）
- 在邻里间被视为"冷漠刻薄的怪胎"
- 母亲（痴呆症）1928年10月刚去世，Helen失去生活唯一意义
- 童年创伤：8岁时目睹父亲家暴母亲，母亲被打成精神失常
- 18岁时父亲"失踪"（暗示被Helen杀害）
- 看到Mary的8岁女儿，想起当年8岁的自己
- 决定帮助Mary杀Thomas（"嫌疑人X的献身"）

#### Margaret Brennan（Zack的母亲）
- 关键证人，案发当晚路过楼下
- 看到楼顶有三个人
- 证词成为Mary"正当防卫"的关键

### 调查组

#### Zack Brennan（35岁，私家侦探）
- 发现所有破绽，知道真相
- 第一次面临"违法实现正义"的道德选择
- 最终销毁证据，保护Mary和Helen

#### Emma O'Malley（25岁，记者）
- 写报道揭露Continental银行诱导杀妻骗保
- 引发全城轰动，威胁Miller集团利益

#### Mickey（38岁，律师）
- 为Mary辩护，成功让法庭判"正当防卫"
- 说服Zack："有时候规则是错的，等到'正在进行的侵害'，Mary已经死了"
- 道德权威达到顶峰：不仅追求法律正义，更追求人性正义
- 为Unit 4的背叛埋下伏笔

### 循环1：自杀还是他杀？

**核心目标**：Thomas是自杀还是他杀？

#### 开篇
- Zack、Emma、Mickey三人来到南区公寓楼
- Morrison在现场："又是你，Brennan。这只是个自杀案件，别多管闲事。"
- Zack："我们只是看看。"

#### 搜证

**场景搜证：**

| 地点 | 发现 |
|:---|:---|
| 楼下坠落点 | Thomas尸体，头部着地；血迹分析符合高空坠落；遗书："我还不起债了，对不起Mary，对不起女儿" |
| Thomas家（二楼） | Continental银行催债信（欠款$5,000→$12,000）；保险单（Mary被保险人，$10,000，受益人Thomas）；酒瓶到处都是；家暴痕迹 |
| 楼顶 | 护栏处有脚印；一个空酒瓶 |

**人物对话：**
- **Mary**（哭泣）："他...他还不起债，压力太大了...我以为他只是出去散步..."
- **Helen**（冷漠）："我不了解楼下的事情。我母亲刚去世，我自己的事都忙不过来。"
- **邻居Mrs. Chen**："Thomas欠了很多债，经常喝醉打老婆。昨晚11点左右听到一声巨响..."
- **Continental银行保险代理人**："Thomas主动来买保险的，说想给妻子一份保障。"

#### 指证

| 项目 | 内容 |
|:---|:---|
| **被指证者** | Morrison |
| **谎言** | "这就是自杀，遗书都有，别浪费时间了。" |
| **证据1** | 遗书笔迹颤抖（Foster笔迹专家分析） |
| **证据2** | 遗书内容不符合Thomas的教育水平（用词太书面） |
| **证据3** | 楼顶有两组不同的脚印（男性和女性） |

**指证后信息：**
- Morrison："好吧，可能不是自杀。但这不关我事，我没收钱。"
- Morrison不再阻挠调查

#### 结尾
> **Zack**："这不是自杀，是他杀。但凶手是谁？"
>
> **Emma**："楼顶有女性脚印..."

#### 玩家体验心路历程
- 初步印象：Thomas是个"赌博欠债打老婆"的渣男，活该
- 遗书疑点引发好奇：不是自杀？
- 女性脚印引发猜测：是Mary杀的？

| 项目 | 内容 |

**场景搜证：**

| 地点 | 发现 |
|:---|:---|
| Helen家 | Thomas的物品：钱包、手表（藏在抽屉里） |
| | Helen的日记："11月23日，今晚行动。母亲，我会为你报仇。" |

> **Zack**："所以Thomas和Helen计划杀Mary，但Thomas失足了？"
>
> **Emma**："那Mary是受害者..."
>
> **Mickey**："我们需要找到那晚的目击证人。"

- **第二重反转**：从"Mary情杀"到"Thomas+Helen计划杀Mary失败"

### 循环4：Margaret的证词

- "我抬头看，看到三个人的剪影。"
- "一个男人要推一个女人（矮小身形，Mary），另一个女人（高大身形，Helen）在阻止。"

**人物对话：**

*Margaret（虚弱但清晰）：*

*Mary（哭泣）：*

*Helen：*
- "我是想保护Mary，Thomas太强壮了，我一个人对付不了他。"

### 循环5：Zack的怀疑

**核心目标**：真的是正当防卫吗？

#### 开篇
- 法庭审理结束，Mickey成功让法官判Mary"正当防卫"
- Emma的报道刊登，全城轰动
- Zack独自重新审查证据

#### 搜证

**场景搜证：**

*楼顶重建（深度分析）：*
- 酒瓶里有残留：Thomas血液酒精浓度极高（Foster尸检补充报告）
- Thomas当晚喝的酒是Helen的（酒瓶标签）
- 脚印位置：Helen和Mary的脚印都在Thomas背后（围攻位置）

*时间线重建：*

| 时间 | 事件 |
|:---|:---|
| 20:00 | Mary从教堂"提前离开" |
| 20:00-20:45 | Mary的"空白时间"（45分钟，太长） |
| 21:00 | Thomas上Helen家（邻居证词） |
| 21:30-22:00 | Helen灌醉Thomas |
| 22:30 | Thomas坠楼 |

*Thomas手上的皮肤组织（Foster深度分析）：*
- "这是新鲜的，当晚搏斗留下的，不是之前家暴的。"
- "但...组织量很少，不像是剧烈挣扎。"
- "更像是...他被推的时候本能抓了一下Mary的手臂。"

**人物对话：**

*Foster（私下对Zack）：*
- "Zack，我知道法庭判了正当防卫...但..."
- "Thomas血液酒精浓度0.25%，这是严重醉酒。"
- "一个严重醉酒的人，怎么能'要推Mary下楼'？"
- "他当时可能连站都站不稳。"

*教堂附近居民：*
- "我20:15看到Mary往南区方向走，很着急的样子。"
- "从教堂到南区公寓，走路10分钟。那她20:00-20:45这45分钟在干什么？"

#### 指证

**内心独白（无需指证NPC）：**

*Zack内心推理：*
1. Helen提前灌醉Thomas（21:00-22:00）
2. Mary提前从教堂溜出来（20:00，不是20:45）
3. Mary和Helen在20:00-22:00之间见面，商量计划
4. 22:30，两人合力将半醉的Thomas推下楼
5. **这不是正当防卫，是预谋杀人**

#### 结尾
- Zack找到Mickey私下对话

#### 玩家体验心路历程
- 胜利的喜悦：Mary被判无罪
- 随着Zack重新审查证据，不安感上升
- Foster的话引发怀疑：真的是正当防卫吗？
- 时间线推理后的震撼：这是预谋！
- 道德困境：真相大白了，但该怎么办？

### 循环6：Mickey的选择

**核心目标**：Zack和Mickey如何面对真相？

#### 开篇
- Zack找到Mickey，私下谈话

#### 对话

> **Zack**："Mickey，我发现了一些...不对劲的地方。"
>
> **Mickey**（平静）："我知道。"
>
> **Zack**（震惊）："你知道？"
>
> **Mickey**："Foster把补充报告也给我看了。Thomas血液酒精浓度0.25%，时间线对不上，Mary的'空白时间'...我都知道。"
>
> **Zack**："那你为什么还要帮她们辩护？这是预谋杀人！"
>
> **Mickey**："Thomas确实要杀Mary。她们只是早了两周。规则说要等到'正在进行的侵害'，但等到那一刻，Mary已经死了。她报警三次，警察不管。她想离婚，法律不允许。她想逃跑，Thomas会追杀她和女儿。她还能怎么办？"
>
> **Zack**："但法律是法律..."
>
> **Mickey**："法律是人定的，Zack。当法律背叛了需要保护的人，我们就必须做出选择。Thomas给Mary买了保险，计划杀她骗保，这是事实。Helen的母亲被家暴打成精神失常，她看到Mary，想起了童年的自己。她们只是在自救。"
>
> **Zack**（沉默）："..."
>

## 🎭 核心主题表达

### 主题：法律的失效

**核心问题**：当法律背叛了需要保护的人，我们该怎么办？

#### 三个层次

| 层次 | 内容 |
|:---|:---|
| 1 | 法律不保护家暴受害者：Mary报警三次，警察不管 |
| 2 | 法律要求"正在进行的侵害"：但等到那一刻，受害者已经死了 |
| 3 | 法律被资本操控：Continental银行诱导杀人，但安然无恙 |

#### 角色选择
- **Mary和Helen**：先下手为强，自救
- **Mickey**：追求人性正义，超越法律
- **Zack**：第一次违法（销毁证据），但实现了正义

#### 为Unit 4埋伏笔
- Zack违法包庇预谋杀人犯
- 当他发现Mickey也违法时（为Miller集团杀人），心理会极度矛盾
- 他会问："Mickey违法和我违法，有什么区别？"
- Mickey的回答会是："区别在于，你为弱者违法，我为强者违法。"

## 📊 结构化设计说明

### 每个循环的标准结构

1. **开篇**：情境设定，引入本循环核心问题
2. **搜证**：
   - 场景搜证：玩家探索场景，收集物证
   - 人物对话：与NPC对话，收集证词和信息
3. **指证**：3-4轮递进，戳穿谎言
4. **结尾**：本循环总结，引出下一循环

### 突发事件使用原则
- 每循环最多1个突发事件
- 用于剧情转折或关键信息揭示
- 本大纲中使用：
  - 循环4：Margaret的证词（关键转折）
  - 循环6：Mickey和Zack的对话（道德选择）

### 指证设计规范

| 循环 | 指证设计 |
|:---|:---|
| 循环1 | 1次指证（Morrison） |
| 循环2 | 1次指证（Mary） |
| 循环3 | 4轮递进（Helen） |
| 循环4 | 1次指证（Helen第4轮） |
| 循环5 | 内心独白（无需指证NPC） |
| 循环6 | 对话选择（Zack的道德选择） |

### 情感递进曲线

| 循环 | 情感 |
|:---|:---|
| 循环1 | 好奇（自杀还是他杀？） |
| 循环2 | 误导（Mary情杀？） |
| 循环3 | 同情（原来Thomas要杀Mary！） |
| 循环4 | 释然（正当防卫，正义实现） |
| 循环5 | 不安（真的是正当防卫吗？） |
| 循环6 | 震撼（这是预谋！但Zack选择包庇） |

## 💡 玩家体验设计

### 情感旅程

1. **刻板印象建立**（循环1）：Thomas是"赌博欠债打老婆"的渣男

5. **第四重真相**（循环5-6）：这是预谋...但Zack选择包庇

### 道德困境
- 玩家会问：Zack的选择对吗？
- 玩家会想：我会怎么选？
- 玩家会感受到：正义和法律的撕裂

### Helen的"嫌疑人X献身"
- Helen不是石神那样的天才
- 但她的动机更有共鸣：童年创伤、母亲的死、对Mary女儿的怜悯
- 她不是"单纯为了帮助Mary"，而是"为了完成母亲未完成的复仇"
- 她的离开是解脱，不是逃避
