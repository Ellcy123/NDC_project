## 🎯 指证逻辑说明

### 指证背景

循环4结束后，Helen和Mary的正当防卫说法似乎得到了证实：Helen心软了给Thomas灌酒想阻止他，Thomas扑向Mary，Helen从身后拉住Thomas，Mary推开Thomas，Thomas翻过护栏。Margaret的目击证词支持这一说法。案子被判正当防卫。

但Mickey在循环4结尾提醒Zack："Thomas的保险单……$10,000，受益人是Thomas……这个计划是谁想出来的？一个欠债$12,000的失业酒鬼……能想出这么精密的杀人骗保计划？"

Zack决定去Continental银行找Bernard Wells问清楚Thomas的保险单是怎么来的。

在新一轮搜证过程中，Zack发现了决定性证据：
1. **标注报纸**：Thomas家厨房旧报纸堆中发现有大量专业批注的《芝加哥论坛报》保险版面，使用了Continental银行内部缩写术语
2. **皮质垫板压痕**：Bernard办公桌垫板上的压痕与报纸批注完全吻合——是在Bernard桌上面对面写的
3. **雏菊花形压痕**：垫板上的微小花形压痕匹配Mary手链上的雏菊吊坠——证明Mary来过Bernard办公室

通过三轮指证，Zack揭穿Bernard的谎言，确认Bernard手把手教Thomas制造"意外"，并发现有女人来问过"投保人死后"的专业问题。但Bernard始终不完全认罪，只是冷漠地推给"银行政策"和"标准服务"。

### 3轮指证结构

**第一轮：报纸上的"课程"**
- Bernard谎言："Thomas来买了标准保单，我没做任何多余的事"
- 指证结果：用标注报纸+笔迹比对+银行内部术语三重证据击破，Bernard承认给Thomas做过批注，但坚持"这是标准的客户讲解"

**第二轮：垫板上的压痕**
- Bernard谎言："我只是给客户讲解条款，这是标准服务"
- 指证结果：用皮质垫板压痕+侧光对比小玩法击破，Bernard被迫承认是面对面手把手教学，透露银行的系统性诱导机制

**第三轮：垫板上的小花**
- Bernard谎言："之后再没有其他人来找过我"
- 指证结果：用垫板花形压痕+Mary手链比对击破，Bernard承认有女人来问过"投保人死后"的专业问题，但拒绝确认身份

### 🎮 小玩法：压痕比对

**触发条件**：
- 玩家获得证据35-I-01（Bernard办公桌垫板）
- 玩家获得证据35-D-01（标注报纸）
- 玩家在证据界面手动选择两者进行"比对"

**引导机制**（降低卡关风险）：
1. **Foster提示**：获得垫板后，Foster说："这张垫板表面有很多压痕。如果你有书写样本，我可以用侧光比对压痕和笔迹是否吻合。"
2. **Emma提示**：Bernard办公室搜证时，Emma说："这张垫板看起来用了很多年……表面有很多划痕和压痕。"
3. **UI标识**：证据界面两条证据显示"可比对"图标
4. **失败提示**：如果玩家多次尝试失败，Emma提示："Zack，Foster说可以比对垫板压痕和报纸批注……试试看？"

**玩法流程**：

   - 玩家联想到报纸批注可能是在这张垫板上写的

2. **第二步：手动比对**
   - 玩家在证据界面点击"比对"按钮
   - 选择证据35-I-01和证据35-D-01
   - 进入比对界面

3. **第三步：侧光比对小游戏**
   - 屏幕上方：报纸批注照片（固定）
   - 屏幕下方：垫板表面照片（初始状态压痕不明显）

     - 拖动滑块调整侧光角度（0°-90°）
     - 当角度约45°-60°时，压痕逐渐清晰显现
     - 玩家将报纸放在垫板上方，调整位置对齐
   - 当报纸批注与垫板压痕完美重合时，触发"啊哈时刻"

4. **第四步：确认结果**

   - Zack独白："报纸是在Bernard的办公桌上批注的。他把Thomas叫到桌前，手把手地教他如何制造'意外'。"

### 为什么来访女人的问题角度如此重要？

- **目的**：杀Mary骗保

- "投保人意外身亡后，原保单效力如何"
- **角度**：投保人（Thomas）死亡 → 保单如何处理？
- **目的**：研究Thomas死后的保单处理——这正是Mary的真实需求

**"自杀免赔期"和"坠楼算意外还是自杀"的深意**：

- 循环4 Margaret证词说Thomas"失足"，但如果Mary事先研究了"坠楼算意外还是自杀"，这说明她在计划如何让Thomas的死看起来像"意外"

**Mary循环2说"不知道保险"与循环5发现"Mary来过银行"的矛盾**：
- 如果Mary去过银行问保险条款，说明她**早就知道**保单存在
- 如果Mary研究过保单，她可能比任何人都更了解整个计划

### 给循环6的线索

**循环5的"隐藏线索"（为循环6埋伏笔）**：

1. **问题角度是反过来的**：
   - Thomas问"怎么让妻子出意外"
   - 来访女人问"投保人死了怎么赔"
   - 这正是Mary的真实需求：如果Thomas死了，保单怎么处理？

2. **"自杀免赔期"和"坠楼算意外还是自杀"**：
   - 暗示来访者在确认——Thomas的死不能被认定为自杀，必须是"意外"
   - 这是预谋者才会问的问题
   - 循环4 Margaret证词说Thomas"失足"，但如果Mary事先研究了这个问题……

3. **Mary来访的时间**：
   - Bernard说"11月下旬，Thomas出事前不久"
   - 她来之前已经知道Thomas的保险计划（否则不可能问出那些针对性极强的"反向"问题）
   - 说明她去银行是有预谋的调查

4. **Bernard的"不记得"太刻意**：
   - 一个保险代理人接待了同一张保单的两方当事人，却说"不记得"？
   - 他可能在保护自己——如果承认两边都接待了，等于承认他为一场谋杀案的双方提供了"咨询"

5. **雏菊压痕的精确吻合**：
   - 不是"可能是Mary"，而是**几乎可以确认是Mary**
   - 垫板压痕长时间按压（30分钟以上）——Mary在Bernard办公室坐了很久
   - 她在认真学习，不是随便问问

### 悬念设置

- **来访女人是不是Mary？**：垫板雏菊压痕高度吻合，但Bernard拒绝确认身份
- **Mary什么时候知道的保险计划？**：如果她去银行问过，说明她比任何人都更了解计划
- **Mary去银行的真实目的是什么？**：研究"Thomas死后"的保单处理——她在计划反击？
- **Mary和Helen的真正关系？**：两人能这么完美地配合——她们真的是"事后才商量"吗？

#### 1. 垫板压痕的物理合理性需加强（P0）

**方案A（推荐）：Foster技术解释**

```
Zack："所以你能区分出来？"
Foster："可以试试。我会用不同角度的侧光扫描，找出压力最重、轨迹最复杂的那一层。"
```

**方案B（备选）：垫板使用习惯**

#### 2. 雏菊压痕的"30分钟以上长时间按压"逻辑存疑（P0）

**当前设计说法**：
- 证据35-C-03："微小的花形压痕，五瓣花形，轮廓规整，直径约1.2厘米"
- 证据35-E-03："压痕深度显示是长时间按压（至少30分钟以上）"

**建议修改方案**（二选一）：

**方案A（推荐）：降低时间判断精度**
修改Foster报告（证据35-E-03）：
```
原："压痕深度显示是长时间按压（至少30分钟以上）"

```

```
Zack："你能判断多久吗？"
Foster："很难精确。但这个深度……至少不是几分钟的事。她可能坐在那里听Bernard讲解了一段时间，手腕基本保持一个姿势。"
```
→ 降低时间判断的绝对性，用"至少10-15分钟"代替"至少30分钟"

**方案B（备选）：增加Mary紧张细节**
修改Foster报告，增加推理依据：

```
Foster："压痕位置非常集中，说明她手腕几乎没动。"
Zack："为什么会这样？"
```

**PM裁定建议**：方案A最佳（降低时间精度更科学），方案B作为辅助（增加情感细节）。

```
"如果他承认两边都接待了，等于承认自己为一场谋杀案的双方提供了'咨询'……他在保护自己。"
```

→ 用玩家视角推理出Bernard的自保动机

**PM裁定建议**：方案A最佳（让Bernard用"客户隐私"作为借口更专业），方案B作为辅助（增加玩家推理环节）。
