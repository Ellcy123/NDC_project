## 1. V2 核心决策

### 1.1 与旧版的关键差异

| 模块 | 旧版方向 | V2 方向 |
|:---|:---|:---|
| L1 楼顶开放 | 楼顶可完整搜证，护栏灰尘/着地点作为核心 | 楼顶护栏区域先被警戒线拦住，栏杆细节 L3 才开放；L1 更强调外墙抓挠、Thomas 手上有油 |

| L4 结构 | 3 轮指证，结尾进入正当防卫假结局 | 4 轮指证，新增“唯一钥匙”逻辑，并保留报社、市政厅两段 AVG |
| L5 作用 | 银行诱导 + Mary 裂痕 | 同上，但在结尾由 Emma 明确提出“油 + 钥匙”矛盾，为 L6 翻案 |
| L6 形式 | 5 轮控辩辩论，最终烧证据 | 保留 5 轮，加入 Seamus 油瓶与玩家 A/B/C 推理选择 |

### 1.2 本候选版固化的待确认点

以下是已固化的 V2 解释。下游文档和配置落地均按本版执行。

1. L1 “着地点照片 0.8m”保留为搜证疑点和辅助证据，不再作为 Morrison R2 的唯一核心；R2 用“外墙抓挠 + Foster 坠楼分析”击破自杀说法。
2. L4 的门把手方向定义为：**外侧 = 楼梯间侧，内侧 = 天台侧**。若 Helen 是下午布置陷阱时沾油离开，天台侧把手应有油；但内侧干净，故“下午布陷阱时蹭到”不成立。
3. L4 用“唯一钥匙在 Helen 手里”证明 Helen 当晚必定到过楼顶；L6 用“最后一瓶油 + 门把手/内侧干净”推进“当晚油迹说法不成立”。两处承担不同功能。
4. L6 R3 的玩家选项是推理演出，正确项为 **C：Helen 和 Mary**，选错可短反馈后汇合，不开长期分支。
5. L5 垫板花形压痕与报纸批注压痕是两条线：报纸压痕指向 Bernard 面对面教学；花形压痕独立指向有女人到访，L6 再通过 Mary 手链锁定。
6. 案发真相仍为 **Mary + Helen 合力推下 Thomas**；L4 “Helen 为保护 Mary 不小心推下”是正当防卫假结局版本。

## 2. 全章概览

### 2.1 核心叙事

Thomas Sullivan 坠楼案从“自杀”开始，逐步揭开家暴、杀妻骗保、银行诱导与保护网络。玩家在 L1-L4 被引导相信 Mary 是受害者、Helen 是最后关头的保护者；L5-L6 再反转出 Mary 与 Helen 的预谋合谋。最终 Zack 赢得推理，却选择烧掉证据。

| L6 | Mickey | 5 | 正当防卫还是预谋杀人？ | Zack 证明 Mary + Helen 预谋合谋，但选择销毁证据 |

## 4. Loop 1：不是自杀

### 4.1 本轮定位

- 指证对象：Morrison
- 指证轮次：2
- 核心目标：Thomas 真的是自杀吗？
- 情绪功能：建立 Morrison 的敷衍与案件第一层疑点。

### 4.2 自由探索

**SC3001 楼下现场**

- 🟢 着地点照片（Clue）：尸体距墙约 0.8m，作为坠落姿态疑点保留。
- 🟢 外墙照片（Clue）：外墙有抓挠痕迹，说明坠落瞬间有本能抓握。
- 🟢 报纸天气版（Item·文件）：大风 40mph，留给 L2 蜡烛因果闭合。

**SC3002 楼顶**

- 🟢 女性脚印照片（Clue）：37 码，凌乱，不能直接判断为几人；L1 用来证明有女性在场。
- ⚠🟢 男性脚印照片（Clue）：Thomas 自己，作为陷阱证据。
- 🟢 Jameson 空瓶（Item）：好酒来源存疑，L6 与 Helen 灌酒线合流。
- 栏杆/护栏核心区域被警戒线拦住，L3 才开放。

**SC3005 法医办公室**

- ⚪ Foster 坠楼分析（Testimony）：BAC 0.25%，指甲/手部痕迹更像本能反应。
- 🟢 袖扣纤维（Clue）：L2 搜身观察的前置线索。
- 🟢 Thomas 手上有油（Clue）：L5-L6 串起“油 + 钥匙”矛盾。
- 🟣 Foster 烟斗毕业照（Envir）：专业、保守、冷静的角色氛围。

**SC3004 警局**

- 🟡 Morrison 谎言锚点：“烂酒鬼自己跳的。”

### 4.3 指证流程

| 轮 | Morrison 谎言 | 击破证据 | 结果 |
|:---:|:---|:---|:---|
| R1 | 现场只有死者一人 | 女性脚印照片 | 有女性在楼顶出现，Morrison 的“现场干净”不成立 |
| R2 | 他自己跳下去的 | 外墙照片 + Foster 坠楼分析 | 坠落瞬间有本能抓握，主动跳下说法不成立 |

### 4.4 指证后

- Morrison 透露脚印主人“大概率是 Mary”。
- Morrison 暗示案件内部牵扯太多，不想深查。
- 玩家带着三个疑问进入 L2：Mary 是否在场？Jameson 哪来的？Thomas 手上为什么有油？

| R1 | 当晚没去过楼顶 | 门外侧把手油迹 | Helen 承认当晚上过楼顶 |

| R3 | 陷阱是前一天做的，当晚没钥匙上不去 | 楼顶钥匙（唯一钥匙，原属 Thomas） | 钥匙在 Helen 手里；她当晚必然能进入并带走钥匙 |
| R4 | Thomas 还没掉下去时我就走了 | Margaret 目击证词 | 现场有三人剪影与挣扎，Helen 留到坠楼时刻 |

### 7.4 指证后假结局

Helen 给出正当防卫版本：

- Mary 的喊叫让 Helen 心软，Helen 上楼。
- Helen 发现 Thomas 正要把 Mary 推下楼。
- Helen 为保护 Mary 介入，Thomas 在挣扎中坠楼。
- Helen 与 Mary 事后害怕“两个女人弄死一个男人没人信是防卫”，所以统一口径隐瞒 Helen 在场。

本轮玩家结论：**正当防卫**。

## 10. 核心证据链

### 10.1 油

Helen 案发前布置陷阱涂油 → 猫尾巴 / 猫窝钥匙沾油 → 门把手两侧油迹矛盾 → Seamus 持有最后一瓶油 → L6 推翻“临时蹭油 / 心软”叙事。

### 10.2 唯一钥匙

Thomas 是天台管理者，全楼仅一把钥匙 → 案发后钥匙在 Helen 猫窝 → L4 证明 Helen 当晚上过楼顶并带走钥匙 → L5 Emma 追问 Thomas 如何上楼。

### 10.3 Thomas 手上的油

L1 发现 Thomas 手上有油 → L5 形成疑点：若油都来自 Helen 当晚上楼，Thomas 手上的油从何而来？→ L6 指向更早、更完整的预谋。

### 10.4 花形手链 / 垫板花形压痕

L2 Mary 花型手链 → L5 垫板花形压痕证明有女人来访 → L6 与 Mary 手链对上，锁定 Mary 曾主动研究保险条款。

### 10.5 标注报纸

L2 在 Sullivan 家拾取标注报纸 → L5 与 Bernard 皮质垫板压痕吻合 → 证明 Bernard 面对面教 Thomas 理解“意外死亡”赔付方案。
