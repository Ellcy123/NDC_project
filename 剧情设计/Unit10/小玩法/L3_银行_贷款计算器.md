# 玩法设计文档：贷款计算器（Unit10 适配版）

**文档版本**: v1.0
**隶属章节**: Unit 10 - 循环 3
**底层机制**: 复用 `Unit2/小玩法/L3_银行_利率计算器.md`（拖拽填空 + 动态演算 + 派生证据产出）
**设计目标**: 让玩家把 Frank 贷款协议的三条条款一条一条填进算式，亲眼看着累计债务跳出"年化 112%"——证明这不是"完全合法"，是系统性违法。

---

## 0. 前因后果

**前因——证据从哪来**

L3 玩家在 10013 Moore 银行档案室搜证（Mickey 用律师传票解锁后进入），拾取三件贷款协议条款单：

- **10303** 贷款协议第 5 条（年单利 12%）
- **10304** 贷款协议第 7 条（利滚利条款）
- **10305** 贷款协议第 12 条（月管理费 50$）

三份合在一起就是 Frank 那笔 600$ 5 年贷款的完整成本结构。单看任一条都"合法"，三条叠加才是高利贷。

**后果——证据用在哪**

玩法产出**派生证据 10701 贷款计算玩法分析报告**：

- 推论：本金 600$ + 单利 12%×5 年 + 月管理费 50$×60 月 + 利滚利复利 = 5 年合计约 3,960$
- **折合年化约 112%**，远超伊利诺伊州法 36% 上限
- 结论：协议构成系统性违法

**指证用途**：
- **L3 Expose R2**：10701 派生证据击穿 Moore 谎言 2"协议条款完全合法"
- 疑点 10302（"这真的是合法贷款吗？"）的 unlock_condition = 10701（合成完成即解锁）

## 1. 玩法概述

- **玩法类型**: data_analysis / puzzle_drag_drop（复用 Unit2 框架）
- **核心体验**: 拖拽数字填入算式槽位 → 算式自动演算 → 累计债务的"恐怖增长"动画 → 派生分析报告
- **预期时长**: 30–60 秒
- **触发条件**: 同时持有 10303 + 10304 + 10305，在 CASE BOARD 选择三件"合成"
- **失败惩罚**: 无。错位的数字会弹回原处 + Zack 简短提示

## 2. 与 Unit2 原版的差异

| 维度 | Unit2 原版 | Unit10 适配版 |
|---|---|---|
| 触发证据 | 201 催款信 + 305/306 贷款条款 | 10303 + 10304 + 10305 三件分条款单 |
| 本金 | 600$ | **600$**（沿用，1920s 工人小额贷款典型数字） |
| 利率 | 12% / 月 | **12% / 年单利**（Frank 案是单利不是月复利，但叠加管理费+利滚利后仍达 112% 年化） |

| 算式槽位数 | 4 槽 | **4 槽**（本金 / 年单利 / 期限月数 / 月管理费） |

| 击穿对象 | Unit2 银行案 | **L3 R2 击穿 Moore"协议条款完全合法"** |

## 3. 交互设计（简版，详细机制见 Unit2 原文档）

```
【界面布局】
  屏幕左侧：三张贷款条款单（10303/10304/10305）平铺在桌面，关键数字用虚线框高亮
  屏幕右侧：Zack 侦探笔记本翻开一页，预印算式模板：
           最终债务 = 本金 + (本金 × 年单利 × 期限年数) + (月管理费 × 期限月数) + 利滚利复利项

【拖拽填空】
  玩家依次把高亮数字拖入对应槽位：
    600（本金）/ 12%（年单利）/ 60（期限月数）/ 50（月管理费）
  正确：吸附 + 落笔声 + 字体变手写体
  错误：弹回 + 纸张摩擦声 + Zack："这一项不该填这里"

【开始计算】
  4 槽全满 → 笔记本下方出现红色印泥风格 [开始计算] 按钮
  点击 → 进入演算动画

【演算动画】（约 3-4 秒）
  - 算式各项依次点亮：600 → +360（单利 5 年）→ +3000（管理费 60 月）→ +复利项
  - 累计债务数字从 600$ 开始快速跳动上涨
  - 背景音越来越压抑（钟摆声 + 钢琴低音）
  - 最终定格在约 3,960$（年化 112%）
  - 屏幕底部红字浮现："Illinois State Banking Law: 36% APR 上限"——3,960$ 折合年化 112%，**违法**

【完成】
  10701 派生证据进入背包
  疑点 10302 自动解锁
  Zack 内心独白触发
```

Zack（合上笔记本）：
> "本金 600 块，五年下来要还近 4000——这不是贷款，这是绞索。Moore 给 Frank 套上的是一根法律够不到的绳子。"

## 4. 结果与产出

**产出证据**: **10701 贷款计算玩法分析报告（派生）**

描述：
> 将 Frank Kowalski 贷款协议三条款（第 5/7/12 条）综合计算的分析报告。本金 600$，年单利 12%×5 年 = 360$，月管理费 50$×60 月 = 3,000$，加利滚利复利项，5 年合计约 3,960$。折合年化（含管理费）约 112%，远超伊利诺伊州法 36% 上限（1916 年伊利诺伊州银行法）。这不是合法贷款，是系统性违法。

**指证用途**:
- L3 Expose R2：以 10701 击穿 Moore 谎言 2"协议条款完全合法"——年化 112% vs 州法 36%，无论怎么辩"投资公司马甲"都掩盖不了数学结果。
- 疑点 10302 解锁件（玩家完成合成即解锁）

**陷阱评估**:
- 10311 伊利诺伊州法律公告（单独出示）= L3 R2 陷阱选项——只能呈现"36% 上限"条文，但 Moore 会用"Lakeshore Investment Co. 不是银行"规避；需要 10701 的具体计算结果才能彻底击穿。
