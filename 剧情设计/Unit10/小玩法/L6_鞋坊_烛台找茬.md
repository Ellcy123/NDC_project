# 玩法设计文档：烛台找茬（Unit10 适配版）

**文档版本**: v1.0
**隶属章节**: Unit 10 - 循环 6
**底层机制**: 复用 `Unit2/小玩法/L6_鞋坊_烛台找茬.md`（案发前照片叠加 + 找不同）
**设计目标**: 玩家把 Margaret 包里的案发前店铺照片叠加到现实废墟上，亲眼看出烛台**从柜子左侧"飞"到了右下角**——这不是火灾自然倒下，是有人抄起它打人。Frank 死亡的物理痕迹第一次被肉眼定格。

---

## 0. 前因后果

**前因——证据从哪来**

L6 玩家在 10006 圣心医院 Margaret 病房苏醒段，Foster 把 Margaret 案发当天准备拿去办抵押的随身包还给 Zack。包里两件关键物：

- **10602 房屋抵押证明**——Margaret 准备办抵押用，附带**两三天前刚拍的店铺内景照片**。照片角度正好覆盖案发的柜子区域。
- **10603 医用镊子**——Foster 给的，备注"取地板缝隙细物用"。

带着这两件物件，玩家回到 10001 Margaret 鞋坊店铺区（L6 深度搜证状态）触发本玩法。

**后果——证据用在哪**

玩法不产派生证据，但**揭示三件已有物证的位置和可拾取状态**：

- **10604 烛台**（移位的本体）
- **10605 眼镜碎片**（烛台原位下方地板缝隙——需持 10603 医用镊子才能取出）
- **10606 货物箱凹陷**（烛台周围 3cm 凹陷，西装面料纤维残留）

**指证用途**：
- **疑点 10601**（"鞋坊地板里的眼镜碎片是谁的？"）condition：10605 + 6110001 Margaret 目击
- **疑点 10603**（"鞋坊里发生过打斗？"）condition：10606 + 10602
- **L6 Expose R2 击穿件**：10605 + 10610 眼镜维修单——击穿 Leonard"碎片不能证明是我的"
- **L6 Expose R3 击穿件**：10606 + 10602 双件——击穿 Leonard"拜访摔坏眼镜"（货物箱凹陷+西装纤维直接证明发生过打斗）

## 1. 玩法概述

- **玩法类型**: spot_the_difference / overlay_comparison（复用 Unit2 框架）
- **核心体验**: 叠加照片 → 重叠透视 → 发现违和感 → 点击三处异常 → 揭示物证位置
- **预期时长**: 60–90 秒
- **触发条件**: 持有 10602 房屋抵押证明 + 进入 10001 鞋坊店铺区（L6 状态）；Zack 凑近案发柜子区域，主动调用 10602 照片
- **失败惩罚**: 无。点错位置会有 Zack 提示但不计错

---

## 2. 与 Unit2 原版的差异（场景结构调整）

| 维度 | Unit2 原版 | Unit10 适配版 |
|---|---|---|
| **场景** | "鞋坊 1F 废墟 + 卧室区"（合并描述） | **10001 店铺区**单独——Unit10 把鞋坊分成店铺区 10001 / 卧室区 10002 / 地下室 10010 三场景，L6 阶段只 10001 开放 |
| **触发物 1** | 605 房屋抵押证明（Margaret 包里） | **10602 房屋抵押证明**——Foster 在医院 10006 交给 Zack（Margaret 苏醒后委托） |
| **触发物 2** | 无（烛台底部直接带碎片） | **10603 医用镊子**——Foster 同时给，用于取地板缝隙的眼镜碎片 |
| **派生产出** | 601 带眼镜碎片的烛台（合一派生） | **不产派生**——10604/10605/10606 是三件独立物证，玩法只揭示位置 + 转为可拾取 |
| **眼镜碎片取法** | 烛台底部一次性带出 | **地板缝隙独立取**——10605 需镊子（10603）单独操作 |
| **疑点服务** | Unit2 自身 R3 | **疑点 10601 + 10603 双向解锁**；R2 + R3 击穿件取证 |

## 3. 交互设计（三阶段）

### 阶段 1：叠加透视

```
【界面布局】
  屏幕底层：当前 10001 店铺区废墟状态（焦黑桌面、倒置的柜子、烧坏的展示架）
  鼠标/手指控制：手持泛黄旧照片（10602 附带的店铺内景）

【操作】
  玩家拖动照片在屏幕上寻找特征吻合区域
  - 照片接近正确透视角度时透明度自动调整为 50%
  - 玩家以柜子边角、窗框、地砖纹理为参照对齐
  - 完全对齐时锁定，进入"找茬模式"

【动画过程——约 5 秒】
  阶段 1：照片半透明状态，玩家手动平移（约 3 秒）
  阶段 2：参照物匹配后照片"咔"一声锁定，画面停拍 0.5 秒
  阶段 3：照片框外废墟保持焦黑现实，框内"过去"清晰显示——形成"双层时空"画面
```

### 阶段 2：找不同（三处异常）

```
【画面对比】

  [照片里 · 案发前两三天]            [废墟里 · 当下]
  ┌──────────────────────┐          ┌──────────────────────┐
  │   柜子                 │          │   柜子（焦黑）          │
  │   ╔════════╗          │          │   ╔════════╗          │
  │   ║   🕯️   ║（左立）  │   →     │   ║   ⭕    ║（空）    │
  │   ║         ║          │          │   ║   📐    ║（划痕）  │
  │   ╚════════╝          │          │   ╚════════╝          │
  │                       │          │              ↘         │
  │   [整齐货物箱]         │          │      ⚡       🕯️       │
  │   ▓▓▓▓▓▓▓▓            │          │   ▓▓▓░░▓▓▓ (倒下)     │
  │                       │          │   (3cm 凹陷)            │
  │                       │          │                         │
  └──────────────────────┘          └──────────────────────┘

  三处异常（玩家逐一点击）：
  ① 烛台位置：照片左立 → 现实右下倒
  ② 柜面划痕：照片整洁 → 现实多道钝器擦痕（烛台原位周围）
  ③ 货物箱：照片完好 → 现实中部 3cm 凹陷 + 一缕灰蓝色西装纤维
```

【操作】
- 玩家依次点击三处异常位置
- 每点击一处：照片该位置高亮金边 + Zack 简短独白
- 三处全部点完后进入阶段 3

**Zack 独白**（按点击顺序）：
> ①（指向移位的烛台）："这玩意儿不是被火烧倒的——它从这边跳到了那边。"
>
> ②（指向柜面划痕）："柜面上全是刮擦痕。烛台底座的形状，有人抄起来挥过。"
>
> ③（指向货物箱凹陷）："3cm 凹陷，纤维。这里发生过激烈打斗——有人被狠狠撞到了箱子上。"

### 阶段 3：揭示物证 + 嵌套交互

```
【触发动画】
  照片消失，画面切回完全的废墟视角
  三处异常的位置出现可交互高亮标记（金色光圈）

【可拾取/可记录列表】
  10604 烛台         → 直接拾取
  10606 货物箱凹陷   → 记录/拍照（envir，不进背包但写入 CASE BOARD）
  10605 眼镜碎片     → 卡在烛台原位下方的地板缝隙里
                       ⚠ 需要持有 10603 医用镊子才能取
                       不持镊子：Zack 看到但伸手够不到，"我需要细一点的工具"
                       持镊子：触发"镊子取碎片"嵌套动作（约 3 秒小动画）
```

Zack（取出眼镜碎片那一刻）：
> "烟熏黄色——颜色对得上 Leonard 的旧镜片。"
>
> "这是 Leonard 的眼镜。"

## 4. 结果与产出

**产出证据**: 无派生证据；揭示三件已有物证的位置和可拾取状态：

| ID | 名称 | 位置 | 取法 |
|---|---|---|---|
| **10604** | 烛台 | 柜子右下倒位 | 直接拾取 |
| **10605** | 眼镜碎片（烟熏黄色旧镜片） | 烛台原位下方地板缝隙 | 需持 10603 医用镊子（嵌套小动作） |
| **10606** | 凹陷的货物箱（含西装纤维） | 烛台周围打斗痕迹区 | envir，记录/拍照 |

**疑点解锁**:
- 疑点 10601（眼镜碎片是谁的？）等待 6110001 Margaret 目击配齐
- 疑点 10603（鞋坊里发生过打斗？）此时 10606 + 10602 双件齐 → **直接解锁**

**指证用途**:
- L6 Expose R2：10605 眼镜碎片 + 10610 眼镜维修单 → 击穿 Leonard"碎片不能证明是我的"
- L6 Expose R3：10606 货物箱凹陷 + 10602 房屋抵押证明 → 击穿 Leonard"拜访摔坏眼镜"——3cm 凹陷不是摔的，是被打的

## 5. 美术 / 音效需求

复用 Unit2 原版资产 + Unit10 特定改造：

美术：
- **复用**：照片叠加 UI、透明度渐变、参照物吸附锁定、双层时空对比画面、Zack 手持照片
- **Unit10 特定**：
  - 10602 照片：1925 年代灰白照片质感，边角微微泛黄，店铺内景以柜子为中心
  - 10001 店铺区废墟：焦黑色调主导，烛台移位轨迹用淡淡的"拖痕"美术暗示（不是显式标线，是地砖灰尘被拖过的痕迹）
  - 货物箱凹陷：3cm 内凹的金属皮箱，边缘有一缕灰蓝色西装纤维卡在凸起处
  - 地板缝隙：木质地板的接缝处，眼镜碎片用一点反光暗示

音效：
- 照片锁定：木质"咔"声
- 异常点点击：清脆叮当 + Zack 低声台词
- 镊子取碎片：金属夹住玻璃的微响（嵌套动作约 3 秒）
- 整体环境：火灾后的死寂——只有远处风声、断续的电线滋滋声

## 8. 设计备注

- **这是 Unit10 最具"福尔摩斯感"的玩法**——不是分析、不是计算，是**还原历史**。玩家把"过去的照片"压在"现在的废墟"上，用肉眼读出"这里发生过什么"。这种体验在所有 4 个玩法里最接近 1940s 黑色电影侦探的破案瞬间。

- **Foster 在医院给 Margaret 包**这个剧情前置很重要——它确立了 Foster 不是单纯的法医，而是**有意识帮 Zack 推进调查**的角色（Miller 集团资助伏笔的轻量铺垫）。她给 10602 + 10603 两件物，是因为她**预判到**鞋坊废墟里需要这两样工具。这个细节在 Unit10 内不暴雷，但敏锐的玩家会注意到 Foster 太"刚好"。

- **不要让 Zack 提前点破"Leonard 来过"**——找茬过程中 Zack 的独白只描述物理痕迹（"烛台不是火烧倒的"、"3cm 凹陷"），不说"这是 Leonard"。直到取出 10605 眼镜碎片那一刻，颜色匹配 Leonard 旧镜片才能让 Zack 说出"这是 Leonard 的眼镜"——这个推理的最后一步必须留给玩家自己跳。

- **货物箱凹陷"3cm + 西装纤维"的设计精度**：3cm 凹陷高度对应**成人背部**被撞击留下的痕迹——Frank 被 Leonard 抄起烛台挥过来时背部撞到货物箱，所以才有这个凹陷+纤维残留。这是 Leonard 杀 Frank 的**物理时间线**复原——比任何证词都直接。
