# 玩法设计文档：Silver Moon 骰子赌博 (Silver Moon Craps)

**文档版本**: v1.0
**隶属章节**: Unit 10 - 循环 2（首次开放，跨章节可重玩）
**设计目标**: 让玩家在 Silver Moon 赌场感受到 1920s 禁酒令地下赌博的氛围——和侦探推理无关，纯氛围+消遣+少量彩蛋。

---

## 0. 前因后果

**前因——为什么有这个玩法**

L2 玩家在 10015 Silver Moon 赌场调查 Vinnie 时，自然踏入 1920s 芝加哥地下赌博场所的氛围核心。**侦探推理之外，本作希望玩家能"在 1920s 玩一会儿"**——禁酒令时期的非法赌场是最具时代感的场景之一。

**后果——玩法不影响主线，但提供：**

- **氛围沉浸**：让 Silver Moon 不只是"Tony 给证词的功能场景"，而是有"人在赌"的真实地下赌场
- **少量彩蛋**：连赢 5 把会触发 Tony 的特殊对话（认可 Zack 有"赌徒命"），但 Tony 主线证词（2070002）**不依赖玩家是否玩**——证词在 Tony 自由探索对话里直接给
- **跨章节可玩**：Unit10 之后玩家若回到 Silver Moon，仍可继续玩；Zack 输赢累积在存档中，作为一个微小的"陪你走完旅程"的元素

**与主线推理完全解耦**——这是 Unit10 唯一一个"不为推理服务"的小玩法。

---

## 1. 玩法概述

- **玩法类型**: dice_betting（骰子押注）
- **核心体验**: 摇骰盅，押大小奇偶——简化版 1920s 街头骰子赌博，三秒一局
- **预期时长**: 单局 ~10 秒；玩家想玩几局玩几局，离桌即结束
- **触发条件**: 进入 10015 Silver Moon 赌场，主屏幕中央有骰桌；点击"加入"即可开局；离开赌场或选择"离桌"即可退出
- **失败惩罚**: 输的钱从 Zack 钱包扣（Zack 钱包初始 50$，输光也不影响主线——Zack 会苦笑说"先记账上"）

---

## 2. 1920s 考据

1920 年代芝加哥地下赌场最流行三种赌博：**扑克、骰子（Craps）、轮盘**。
本玩法选用**骰子**有三个理由：
1. **节奏快**：一局 10 秒完成，适合作为不打断主线的氛围玩法
2. **门槛低**：玩家不需要懂规则——大/小/奇/偶，全是直觉押注
3. **时代感强**：街头掷骰、酒吧角落押注是 1920s 美国底层文化最普遍的赌博场景，比扑克更"接地气"

**真实 Craps 简化规则**（本玩法采用）：
- 庄家摇两颗六面骰，**点数和 2–12**
- 玩家可押四种结果（赔率全部 1:1，无庄家优势 = 让玩家不会输得很惨）：
  - **大**（点数和 ≥ 8）
  - **小**（点数和 ≤ 6）
  - **奇**（3/5/7/9/11）
  - **偶**（4/6/8/10/12）
- **7 点平局**（pass，押注退回）——历史上骰子赌博"7 点"被视为转运点，本作沿用作为给玩家的"温柔保护"

## 3. 交互设计

```
【界面布局】
  屏幕中央：木质骰桌特写，正中央一个皮革骰盅，左右各放一颗六面骰
  屏幕底部：四个押注按钮（大 / 小 / 奇 / 偶），每个按钮上方显示当前押注金额
  屏幕右上：Zack 钱包余额（动态变化）
  屏幕左下：当前局数 + 累计输赢
  屏幕右下："离桌"按钮

【操作】
  阶段 1（押注）：玩家点击四个按钮之一，再点击"+1$ / +5$ / +10$"调整金额；最多押 Zack 钱包 1/2（保底）
  阶段 2（开骰）：玩家点击"摇骰"
                  骰盅腾空——晃 1.5 秒——落桌
                  两颗骰子从盅下滚出，约 1.5 秒停稳
  阶段 3（结算）：两颗骰点数高亮显示
                  押中：按钮闪绿光，钱包余额 +押注金额（赔率 1:1）
                  未中：按钮闪红光，钱包余额 -押注金额
                  7 点：按钮闪金光，押注退回（"庄家说今天交个朋友"）
                  Tony 偶尔在画面角落探头看一眼，无台词（连赢 5 把后才说话）

【完成判定】
  无强制结束。玩家点"离桌"或退出场景即结束。
  Zack 钱包余额 < 1$ 时强制结束（Tony 说"留点钱明天再来吧"）

【彩蛋触发】
  连赢 5 把：Tony 笑着递一杯水，说"你这眼睛——干这行真是浪费了"
            （彩蛋台词，与主线无关；不解锁任何额外证据）
  连输 5 把：Tony 摇头叹"算了，今天给你免单一杯威士忌——记账上"
            （Zack 钱包余额恢复到 10$，避免玩家被完全卡死）
```

Zack 内心独白（首次开桌）：
> "1920 年这条街上的人，大概都在这种桌子上输过一次又一次。"

## 5. 美术 / 音效需求

美术：
- **骰桌**：深棕色木质方桌，桌面铺墨绿色羊毛毡（1920s 经典赌桌配色），桌沿磨损发亮
- **骰盅**：黑色皮革鼓型，顶部刻有 Silver Moon 月牙标志
- **骰子**：象牙白色，点数用黑色凹刻；颗粒小（约 1cm），符合 1920s 实际尺寸
- **押注按钮**：黄铜质感，按下凹陷；金额数字用 1920s 装饰字体
- **Tony 探头**：画面右侧偶尔（每 3–4 局一次）出现 Tony 半身像，倚着吧台看一眼，无对白
- **环境**：暖黄油灯光晕、桌面上一杯威士忌冰块（半透明素材，氛围用，不可交互）

音效：
- 押注：硬币落桌"叮"一声
- 摇骰盅：皮革+骰子的摩擦声 + 桌面颤动（持续 1.5 秒）
- 落桌：木桌闷响 + 骰子滚动声
- 押中：黄铜钱币哗啦一响（约 0.5 秒）
- 未中：低音"嗯"一声
- 7 点：清脆叮当 + Tony 的低笑（彩蛋音）
- 背景：远处爵士钢琴轻声断续，酒杯碰撞声，模糊人声笑闹（持续循环）

## 6. 数据配置

```yaml
mini_game:
  id: MG_silvermoon_craps
  type: dice_betting
  scene: 10015                    # Silver Moon 赌场
  cross_chapter: true             # 跨章节可重玩
  input_items: []                 # 不消耗物品
  output_item: null               # 不产出证据
  wallet_initial: 50              # Zack 钱包初始金额（首次进入）
  bet_options: ["大", "小", "奇", "偶"]
  payout_ratio: 1.0               # 赔率 1:1
  seven_rule: refund              # 7 点退还押注
  max_bet_ratio: 0.5              # 单次押注 ≤ 钱包余额 1/2
  min_wallet: 1                   # 余额 < 1$ 强制离桌
  bankruptcy_recovery: 10         # 连输 5 把恢复 10$（避免卡死）
  achievements:
    - id: silvermoon_lucky_streak
      trigger: consecutive_wins >= 5
      effect: dialogue_unlock     # 触发 Tony 彩蛋台词
    - id: silvermoon_unlucky
      trigger: consecutive_losses >= 5
      effect: wallet_recovery     # 触发免单 + 钱包补救
  trigger_text: "试试手气？1920 年的芝加哥，每条街都有这么一张桌子。"

# ── state 说明 ──
# loop2_state.yaml → scenes → 10015 → 增加 minigame_available: ["MG_silvermoon_craps"]
# 钱包余额持久化在玩家存档（playerState.wallet），不写 state YAML
```

## 7. 玩法定位整合

```
1920s 芝加哥地下赌场氛围
        |
        v
  Silver Moon 赌场（10015）
  - L2 主线：Tony 给 testimony 2070002（独立对话路径，与玩法解耦）
  - L4/L5 主线：Tony 辨认打火机、Danny 在场（独立路径）
        |
        +─── 主线之外 ───+
                        |
                        v
                 [氛围玩法：骰子赌博]
                 - 1920s 大小奇偶押注
                 - 连赢 5 把 / 连输 5 把彩蛋
                 - 跨章节可重玩
                        |
                        v
                 Zack 钱包数字变化
                 （存档持久化，不影响推理）
```

**核心定位**：这个玩法是 Unit10 唯一"不为推理服务"的玩法，存在的意义是让 1920s 芝加哥的世界**自己呼吸**——玩家在调查之余可以坐下来掷两把骰，听爵士钢琴，看 Tony 在角落抽烟。这种"非推理时间"是侦探类型片体验的重要组成部分。
