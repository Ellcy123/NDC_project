# 玩法设计文档：Silver Moon 破冰（调酒考验 + 骰子赌博）

**文档版本**: v2.0（重写——原 v1.0 是纯骰子氛围玩法，本版加入"调酒身份验证"段，整合为两段式陪 Tony 喝一局体验）
**隶属章节**: Unit 10 - 循环 2（首次开放，跨章节可重玩）
**设计目标**: 让玩家通过 1920s 禁酒令时期 speakeasy 的"圈内人验证"仪式（调酒暗号）拿到 Tony 的证词，再通过陪 Tony 玩一局骰子赌博拿到 Vinnie 的钞票——两段同一场景同一 NPC 的连续体验，**重现 1920s 地下酒吧"先验身份、再做生意"的真实文化逻辑**。

**前因——玩家在 L2 进入 Silver Moon 寻找 Vinnie 与 Tony**

Zack 在 L2 自由探索阶段从 O'Hara 处拿到 Leonard 名片（10201），由此得知南区有个 Silver Moon 赌场——Vinnie 常驻。Zack 进店时 Vinnie 正在角落，Tony 在吧台。

## 1. 玩法概述

- **玩法类型**: trust_verification_minigame（身份验证）+ dice_betting（骰子押注）的两段式组合
- **核心体验**: 调酒答题 → Tony 视为自己人 → 给口风 → 陪喝一局骰子 → Tony 投桃报李给钞票
- **预期时长**: 调酒 ~1-2 分钟 + 骰子 ~2-3 分钟，整体 ~3-5 分钟
- **触发条件**: 进入 10015 Silver Moon，主动找 Tony 搭话——对话选择"陪 Tony 喝一杯"
- **失败惩罚**: 两段都有保底（调酒可重试，骰子有"Tony 借本钱"机制），玩家不会被卡死

## 2. 1920s 考据

### 调酒考验的真实背景

1920-1933 美国禁酒令期间，地下酒吧（speakeasy）的"圈内人验证"是真实存在的文化：
- **暗号**：进门时说特定密语
- **熟人推荐**：老顾客带新人
- **行话/秘方**：能说出某酒配方 = 这个圈子的人
- **小费规矩**：1920s 美国调酒师对"懂行"的客人会给更多耐心

Tony 用调酒题考验 Zack——不是想 humiliate 他，是**在判断 Zack 是不是警察/线人**。1920s 黑帮酒吧老板这是日常自保。**答对 = 你不是来抓我的，你是这个文化里的人**。

### 骰子赌博的真实背景

1920s 美国地下赌场最流行三种：**扑克、骰子（Craps）、轮盘**。骰子节奏最快、门槛最低、最适合作为陪喝时的消遣。**1920s 街头掷骰**是真实底层文化场景。

## 3. 第一段交互：调酒考验（信任验证）

```
【触发】
  玩家进 10015 Silver Moon → 走到吧台 → 选择"陪 Tony 喝一杯"
  Tony 笑了一下："小子，先告诉我个事。"

【界面布局】
  屏幕中央：吧台特写——Tony 擦杯子的双手 + 一字排开 6 瓶酒（金酒/威士忌/苦艾酒/蜂蜜酒/柠檬汁/糖浆）
  屏幕右上：Zack 笔记本——Tony 出的题目展示
  屏幕底部：玩家选项选择区

【可观察的环境线索】（不是必需，但答题前可以看）
  - 墙上残菜单：手写"Bee's Knees - $0.50"等价格（褪色但可辨）
  - 角落客人喝的酒：金黄色 + 浅一层柠檬皮（暗示 Bee's Knees）
  - 酒架上一瓶蜂蜜罐子（暗示蜂蜜酒）
  - 远处客人喝威士忌时晃酒杯的方式（1920s "懂行"的手法）

【题目示例（3 题随机抽 2 题）】
  Q1: "配一杯 Bee's Knees 给我看——三件，分别是什么？"
      A. 金酒 + 柠檬汁 + 蜂蜜（✓ 正确）
      B. 威士忌 + 蜂蜜 + 苏打水（✗ 错误）
      C. 朗姆酒 + 柠檬汁 + 糖浆（✗ 错误）
      D. 金酒 + 苦艾酒 + 柠檬皮（✗ 错误）

  Q2: "Manhattan 用什么酒打底？"
      A. 黑麦威士忌 + 甜苦艾酒 + 安格仕拉苦精（✓ 正确）
      B. 金酒 + 干苦艾酒 + 橄榄（✗ 错误——这是 Martini）
      C. 朗姆酒 + 柠檬汁 + 糖浆（✗ 错误）

  Q3: "Whiskey Sour 三件——加什么让它'起泡'？"
      A. 苏打水（✗ 错误）
      B. 蛋清（✓ 正确——经典 1920s 配方）
      C. 蜂蜜（✗ 错误）

【操作】
  玩家用 1-3 秒看一眼环境（可选）→ 点击答案
  答对：Tony 抬眉，"还行。"——清脆冰块倒入杯子的声音
  答错：Tony 笑骂，"年轻人，连这都不会"——给提示后重试（无惩罚）

【完成判定】
  连续答对 2 题（共 3 题，中间可错一次再答对）→ Tony 视为"自己人"
  进入信任段对话——Tony 主动谈起 Vinnie + Frank

【信任段对话——Tony 的话】
  "Vinnie 那家伙最近常来——花新钞，说什么'今天又赚了一笔'。我留意过，号码是连着的（905-906-908，他都甩出来过）。"
  → 触发 testimony 2070002 进入证词池

  "对了，你妈雇的那个 Frank——老头也常来这一带喝酒。一杯啤酒能坐一晚上。我看挺孤单的，丧偶嘛，可怜人。"
  → 触发 testimony 2070003 进入证词池
  → ⚠ Tony 不知 Frank 真实动机（在监管侄子 Danny），玩家在 L2 也不知道
```

Zack 内心独白（拿到 2070003 时）：
> "Frank 老头……一杯啤酒坐一晚上。"
>
> （此刻玩家以为这是孤独老头的孤单，L4 才会知道这是父爱）

## 4. 第二段交互：骰子赌博（赢钞票）

```
【触发】
  调酒考验答对后，Tony："再陪我玩一局——你赢了，我给你看个东西。"
  桌面切换到骰桌特写

【界面布局】
  屏幕中央：木质骰桌 + 皮革骰盅 + 两颗象牙骰
  屏幕底部：四个押注按钮（大 / 小 / 奇 / 偶）+ 金额调整（+1/+5/+10）
  屏幕右上：Zack 钱包余额（动态变化）
  屏幕左下：累计赢钱进度条（目标 30$ 解锁钞票）
  屏幕右下：连输保底计数器（连输 5 把触发"借本钱"）

【操作】
  玩家点击按钮押注 → 点击"摇骰" → 1.5 秒晃动 → 1.5 秒滚动停稳
  押中 1:1 赔率；7 点退还押注（"庄家说今天交个朋友"）

【赢钱解锁】
  累计赢 30$ 后 Tony 笑着摇头：
  "行了行了，你眼神挺准。这玩意你拿着——"
  Tony 从围裙口袋摸出三张新钞（905B / 906B / 908B），放到 Zack 面前。

  Zack 还没开口，Tony 已经接着说了：
  "这几张新票是 Vinnie 最近给的——他来这儿喝酒结账留下的零钱，我没敢花。
  你拿去看看，号码连得有意思——他甩出来的时候我就注意到了。"

  → item 10206 进入背包
  → 钞票来源叙事链闭合：Vinnie 花 → Tony 收 → Tony 转给 Zack（不是凭空给）

【连输保底】
  连输 5 把 + 钱包 < 5$ 时 Tony 主动出手：
  "算了，今天给你免单一杯威士忌——记账上。"
  Zack 钱包恢复到 10$
  + Tony 看在 Margaret 面子上，把 10206 直接给（不再要求继续赢）
  → 玩家不会被永远卡死

【完全跳过】
  玩家如果选择"不玩了"直接离开骰桌：
  Tony 摇头："Margaret 的儿子……算了。"
  把 10206 给（看在母亲面上），但 2070003 Frank 饮酒细节如果调酒段也跳过的话就不给（轻量惩罚——失去 L4 callback 钩子）
```

Zack 内心独白（拿到 10206 时）：
> "Tony 这老头——精得很，但也讲究规矩。"

美术：
- **吧台特写**：深红木质吧台、铜把手镀光、墙上挂着褪色的私酿酒广告

- **环境线索**：墙上残菜单（手写体）、酒架上瓶子排列、客人喝法的微小动画

音效：
- 调酒：冰块入杯叮当、Tony 擦杯子的轻微摩擦、客人远处低语

## 9. 设计备注

- **调酒考验不是炫技，是验证**：1920s speakeasy 文化里这种"暗号/秘方"考验**真实存在**。Tony 出题不是 humiliate Zack，是判断他是不是警察/线人——这是禁酒令时期黑帮酒吧老板的日常自保。如果写手把这段写成"嘲弄 Zack 不懂"，文化感会丢；要写成 Tony 在严肃地"决定要不要交底"。

- **2070003 是 Frank 立体化的关键 testimony**：L2 玩家拿到时只觉得"Frank 是个孤独老头"，与 Tony 视角一致；**L4 拿 Lula 证词后玩家自己反转**——Frank 不是孤独，是父爱。这种"信息层次反转"是 Unit10 推理体验的精华之一。**写手不要让 Tony 在 L2 透露任何 Frank 真实动机的暗示**——Tony 必须保持"我以为他只是个孤单老头"的认知。

- **保底机制必须做扎实**：玩家可能不愿意玩调酒/骰子，必须有"完全跳过"路径让主线推理不被卡死。但**调酒+骰子都跳的玩家失去 2070003**——这是轻量惩罚（不影响主线，影响 L4 情感深度）。

- **跨章节**：调酒考验完成一次后 Tony 视 Zack 为熟客；后续 Unit 玩家回 Silver Moon 时 Tony 进门点头微笑，骰子仍可玩。这是 Unit10 留给玩家的"长情"——一个跨越多个 Unit 的小关系。

- **Frank 饮酒细节强度**：Tony 说的是"我看他挺孤单的"——这个**误读**是设计的核心。Tony 把 Frank 解释为"丧偶老头来打发夜晚"，玩家在 L2 也会接受这个解读。直到 L4 Lula 证词才让玩家自己拼出"Frank 是为侄子来的"——这个反转的情感力道靠 Tony 此刻的"误读"撑起来。Tony 不能在 L2 说任何接近真相的话。
