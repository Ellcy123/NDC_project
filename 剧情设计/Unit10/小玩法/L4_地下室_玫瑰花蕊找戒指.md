# 玩法设计文档：玫瑰花蕊找戒指（Unit10 适配版）

**文档版本**: v1.0
**隶属章节**: Unit 10 - 循环 4
**底层机制**: 复用 `Unit2/小玩法/L4_地下室_玫瑰花蕊找戒指.md`（合并浇水 + 检查花蕊两步玩法）
**设计目标**: 让玩家亲手给一束枯萎的玫瑰浇水，看着它在 Frank 的求婚布置里重新打开——花蕊中心露出 Frank 为 Lula 定制的求婚戒指。这是 Frank 这个角色最后一次"出现"在玩家面前，用一束花和一枚戒指。

---

## 0. 前因后果

**前因——证据从哪来**

L4 玩家用 10308 地下室钥匙（L3 post_expose 从 Moore 获得）解锁 10010 Margaret 鞋坊地下室——Frank 和 Lula 五年秘密约会的空间。

地下室里有三件可拾取物：
- **10409 枯萎的玫瑰花束**（桌上空花瓶里）
- **10410 水杯**（桌边半杯清水）
- **10406 维修收据**（600$ 全用于地下室装修）

10409 和 10410 是本玩法的输入，合成产出 **10403 求婚戒指**——藏在玫瑰花蕊里 Frank 没来得及送出的求婚信物。

**后果——证据用在哪**

10403 求婚戒指（F & L 1921）+ 4040002 O'Hara 证词（Lula 四五年前就来这条街）：

- **L4 Expose R3**：双件击穿 Danny 谎言 3"是 Lula！她是个感情骗子，骗我叔叔签了遗嘱"——戒指 1921 年定制，Lula 五年前就来这条街，Frank 真感情有物理时间线为证
- **疑点 10403 解锁**："Frank 和 Lula 是真感情吗？"unlock_condition = 10403 + 4040002

**情感重量**：这个玩法是 Unit10 全本里情绪最重的瞬间之一。玩家不是在"调查"，是在"完成 Frank 死前没完成的事"——给玫瑰浇水。这个动作本身就是悼念。

---

## 1. 玩法概述

- **玩法类型**: combine_analyze / two_step_discovery（复用 Unit2 框架）
- **核心体验**: **浇水复活枯萎玫瑰** → **检查花蕊** → **取出戒指**
- **预期时长**: 45–60 秒
- **触发条件**: 同时持有 10409 枯萎玫瑰 + 10410 水杯，在 CASE BOARD 选择"合并"
- **失败惩罚**: 无。两步流程线性引导，不会失败

---

## 2. 界面与交互

**触发**：玩家拾取 10409 枯萎玫瑰花束后，**在物品栏点击它的"分析"按钮**触发玩法 UI——不在地下室场景内进行。物品栏触发后切到玩法独立界面。

### 第一步：物品栏分析 → 选水杯浇水

```
┌──────────────────────────────────────────────┐
│         分析窗口 · 物品栏弹出 UI                │
│                                              │
│      ┌───────────────┐                       │
│      │ 10409 枯萎玫瑰 │   Zack 笔记：           │
│      │   ◌◌◌         │  "花干了。需要点水。"   │
│      │ (花瓣蜷曲发褐) │                        │
│      └───────────────┘                       │
│                                              │
│   选择搭配物品：                                │
│   [ 10410 水杯 ✓ ]  ← 可选；其他不匹配          │
└──────────────────────────────────────────────┘
```

- **窗口背景**：物品栏弹出的分析卡，深木色桌面框
- **左侧**：枯萎玫瑰花束特写 + Zack 简短观察笔记
- **右侧**：背包内可搭配物品列表——只有 10410 水杯匹配

**交互**（1 步）：
1. 玩家点击 [10410 水杯] → 自动浇水动画约 4 秒
   - 茎部由褐转绿
   - 花瓣由褐转红展开
   - 花蕊中央出现金属反光
2. 复活完成 → 自动切入第二步"拨花瓣"

### 第二步：连续点击花瓣

```
┌──────────────────────────────────────────────┐
│              花蕊特写镜头                       │
│                                              │
│              ╭─────────╮                     │
│              │  🌹     │  ← 连续点击花瓣        │
│              │  🌹🌹    │     拨开 → 露出戒指   │
│              │  🌹🌹🌹   │                     │
│              ╰─────────╯                     │
└──────────────────────────────────────────────┘
```

**交互**（1 步）：
1. 玩家连续点击花瓣（约 3-5 次点击，每次拨开一层）
2. 最后一层拨开后戒指完全显形 → 自动特写
3. 戒指内圈刻字 **"F & L 1921"** 高亮 2 秒
4. 10403 进入背包；10409 + 10410 自动消耗移除

Zack（看着戒指）：
> "Frank……他没来得及送出去。"
>
> "1921——他认识 Lula 那年就开始计划了。"

## 3. 结果与产出

**产出证据**: **10403 求婚戒指（F & L 1921）**

描述：
> 一枚嵌着小碎钻的金戒指，藏在玫瑰复活后的花蕊之间。戒圈内侧刻有 'F & L 1921'——Frank Kowalski 与 Lula Washington，1921 年。戒指尺寸为女性手型，明显不是 Margaret 的尺码——是 Frank 给 Lula 定制的求婚戒指，迄今未送出。

**指证用途**:
- L4 Expose R3：以 10403 + 4040002（O'Hara：Lula 四五年前就来这条街）击穿 Danny 谎言 3——五年前 Frank 房子还没被银行盯上、遗嘱还没有写，Lula 不可能"图遗产"
- 疑点 10403 解锁件之一

**情感效果**:
- 玫瑰复活动画是 Unit10 最具悼念感的视觉瞬间
- Frank 这个全程不出场的角色，在这一刻通过"他来不及完成的浇水"被玩家代为完成
- Lula 在 post_expose 接到这枚戒指——她看着戒指刻字的镜头是 Unit10 Lula 弧光的视觉锚点

## 4. 美术 / 音效需求

复用 Unit2 原版资产 + Unit10 特定改造：

美术：
- **复用**：浇水动画、玫瑰复活渐变（褐→红）、花瓣点击拨开效果、戒指反光特写
- **Unit10 特定**：
  - 物品栏分析窗口：深木色弹出卡，左侧物品特写 + Zack 笔记字段 + 右侧搭配物品列表
  - 玫瑰品种：红玫瑰，1920s 美国本地切花常见品种（特写镜头，非场景陈设）
  - 戒指特写：金质磨砂戒圈 + 小碎钻群镶；内圈刻字"F & L 1921"用 1920s 衬线字体
  - 花蕊镜头：花瓣层次清晰可点击，最中央金属反光由微弱→强烈

音效：
- **复用**：倒水声、花朵复活的轻微 chime、花瓣摩擦声、戒指掉落金属轻响
- **Unit10 特定**：戒指刻字显形时一段极短的口风琴（accordion）音——1920s 美国意大利移民社区常用乐器，呼应 Frank 的钢厂工人移民背景

## 7. 设计备注

- **这是 Unit10 唯一带"悼念感"的玩法**——其他玩法（骰子、贷款计算器、烛台找茬等）都是侦探任务式的。本玩法在节奏上是 L4 的情绪低谷，要让玩家在浇水那一刻感觉到时间停了一下。
- **戒指出现的瞬间不需要 Zack 大段台词**——一句"Frank……他没来得及送出去"就够了。玩家自己懂。
- **Lula 在 post_expose 拿到戒指**的演出可以作为 Unit10 全本的"角色弧光视觉锚点"之一——Lula 5 年隐忍最后爆发（厕所小窗签字）+ 接戒指落幕，两个视觉锚共同完成 Lula 的弧。
- **不要把这个玩法做成"解谜机制"**——它不是谜题，是仪式。失败惩罚必须为 0，引导必须线性。玩家做不出来比做不出贷款计算器要伤害故事十倍。
