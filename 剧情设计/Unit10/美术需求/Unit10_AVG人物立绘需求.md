## Loop 4

### 场景 10001（Margaret 鞋坊店铺区，废墟状态）

- **Lula**：站在废墟边缘的静止姿态、眼神向下看（悲伤但不哭、平静的内敛表现——与《特殊场景需求》§6 小铁盒场景的"捧情书 + 恳求"立绘形成情感连续）

### 场景 10007（O'Hara 家门口）

- **O'Hara**：倚靠门框的闲适姿态 → 追问 Frank/Danny 时转身靠在门后的不安（对比 L2 的开门警惕；同一 NPC 两种心理状态）

### 场景 10020（Frank 家厕所，自由探索对话 + Danny 指证场）

**说明**：本场景为 `expose_scene` 兼对话场——Zack 推门进入后先进入自由探索式询问（Danny 隔门板对话），再触发 Expose 三轮指证。Danny 立绘**贯穿两个阶段**。

- **Danny**：蹲马桶上的狼狈姿势——一只手按肚子（出汗 + 面如土色）、一只手抓裤腰；面对 Zack 询问时局促回答；逐轮指证中的身体萎缩感（从挺身辩解 → 略低头 → 攀咬 Lula → 仰头看小窗失语）。**自由探索段 + 指证流程内通用立绘**，配合不同表情/姿势微调即可，无需画多套基础立绘
- 注：Lula 隔窗举证 R3 高潮立绘在《突发事件》#5（举证遗嘱 + 半身裁切窗格）

## Loop 5

### 场景 10001（Margaret 鞋坊废墟深度搜证）

- **Emma**（开篇过场）：站在废墟边缘观察、蹲下检查细节的专业搜证姿态

### 场景 10015（Silver Moon 赌场，自由探索 + Vinnie 指证）

**说明**：自由探索段（Zack 跟 Tony 闲聊 + 走近 Vinnie 试探）的立绘需求保留；指证轮次本身**复用 NPC 大头**，且 Vinnie 招供后直接被警察带走（衔接《特殊场景需求》§7 转场 + 《突发事件》#7），**指证场内不产生新立绘需求**。

- **Tony**：自由探索段的吧台后擦杯子持续背景动作 + 玩家靠近时抬眼观察的眼神扫描
- **Vinnie**：自由探索段的角落坐姿（稳健，警觉感弱化）；指证段无新立绘——大头表情 + 突发事件 #2 已建的"上身前压"足够覆盖

## Loop 6

### 场景 10017（Leonard 住所，Leonard 4 轮指证）

- **Leonard 防守姿态四段递进**（指证流程内立绘，每轮需新立绘表情）：
  - R1：挺胸正坐，专业冷静
  - R2：略垂肩，眼神回避
  - R3：双手按桌沉默，专业外壳第一次褪去
  - R4（诱导招供后）：望向楼梯的背叛转身 → 头低下的招供姿态（西装领带松开、双手撑额）
- **Leo 妻子**：常规对话立绘——冷漠感的中年妇女形象（无爱情婚姻的疲态，会因 Leo 没钱或惹事随时离开的待离感）。**对话 NPC**（非哑角）。详细对话设计参《剧情设计/Unit10/Characters/edith.md》（待建）；裙角在楼梯转角消失的演出帧保留（R4 诱导招供前的视觉暗示）

## 备注

- 本文档**只列特殊立绘需求**——标准 NPC 立绘（默认站姿/坐姿/中性表情）由美术规范另行管理
- 自由探索阶段 + 指证流程内（标准 NPC 大头不足以表达）的特殊立绘需求归本文档
- Loop 开篇 / post_expose / 转场 / ENDING 的立绘需求归《特殊场景需求》
- 瞬间方格动态漫画的立绘 cue 归《突发事件动态漫画》
- 立绘描述用"具体身体部位 + 具体动作 + 空间位置"，避免抽象情绪形容词
