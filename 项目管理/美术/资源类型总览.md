## A. 角色管线 (Character Pipeline)

从人物设定出发，产出角色相关的全部视觉资产。

```
人物设定
 ├──→ A1. 角色头像 (big/small)
 ├──→ A2. 基础表情底图 ──→ [策划AI] A3.表情变体 ──→ [策划AI] A4.对话视频
 └──→ A5. 场景NPC立绘 (每角色×场景 2帧)
```

### A1. 角色头像 (NPC Icon)

| 项目 | 说明 |
|------|------|
| 路径 | `Art/NPC/Icon/` |
| 规格 | 每角色2张：`{name}_big.png` + `{name}_small.png` |
| EPI01实际 | 16张（8角色 × 2） |
| 包含角色 | Rosa, Tommy, Vivian, Jimmy, Emma, Anna, Morrison, MrsMorrison, Webb |
| 负责人 | 美术 |
| 前置依赖 | 人物设定完成 |

### A2. 基础表情底图 (MovieClip Base)

| 项目 | 说明 |
|------|------|
| 路径 | `Art/MovieClip/{epi}/AVG/` |
| 规格 | 每角色×场景出1张基础表情图，策划用AI生成全部表情变体 |
| 命名 | `{scene}_{character}_{emotion}.png` |
| EPI01实际 | ~35张基础底图 |
| 负责人 | 美术出底图 |
| 前置依赖 | 人物设定 |
| 下游产出 | → A3.表情变体 → A4.对话视频 |

### A3. 表情变体 (MovieClip Variants) — 策划产出

| 项目 | 说明 |
|------|------|
| 路径 | `Art/MovieClip/{epi}/AVG/`（与底图同目录） |
| 规格 | 从底图AI生成的多情绪变体 |
| EPI01实际 | 261张（从~35张底图生成） |
| 负责人 | 策划（AI生成） |
| 前置依赖 | A2.基础表情底图完成 |

### A4. 对话视频 (avg_clip) — 策划产出

| 项目 | 说明 |
|------|------|
| 路径 | `Art/avg_clip/{EPI}/loop{N}/{character}_{sequence}/` |
| 规格 | MP4视频，角色口型动画，按Loop×场景×对话ID组织 |
| EPI01实际 | 1224个MP4（928MB），按Loop分布： |
| | - loop1: 258个 / loop2: 296个 / loop3: 182个 |
| | - loop4: 208个 / loop5: 168个 / loop6: 102个 |
| 负责人 | 策划（Kling AI生成口型视频） |
| 前置依赖 | A3.表情变体完成 + 对话草稿定稿 |

### A5. 场景NPC立绘 (Scene NPC)

| 项目 | 说明 |
|------|------|
| 路径 | `Art/Scene/NPC/{EPI}/` |
| 规格 | 每个角色×场景组合 2帧：`SC{ID}_npc_{name}1.png` + `SC{ID}_npc_{name}2.png` |
| 用途 | 场景中NPC的可视化呈现（非对话头像，是场景内看到的角色立绘） |
| EPI01实际 | 47张（23组 × 2帧），角色分布： |
| | - Rosa: 10组(20张) / Tommy: 10组(20张) |
| | - Vivian: 8组(16张) / Jimmy: 6组(12张) |
| | - Emma: 1组(2张) / Morrison: 1组(2张) |
| 配套文件 | `XYposition.txt` — 记录每张立绘在场景中的坐标位置 |
| 负责人 | 美术 |
| 前置依赖 | 人物设定 + B1.场景背景（需确定角色在场景中的位置） |
| 下游产出 | → D1.指证场景（需要角色轮廓） |

## 管线间依赖关系

| 上游 | 下游 | 依赖点 | 类型 |
|------|------|--------|------|
| A.角色管线 → | D.指证管线 | A5.场景NPC立绘 → D1.指证场景（需要角色轮廓） | 硬依赖 |
| B.场景管线 → | A.角色管线 | B1.场景背景 → A5.场景NPC立绘（确定角色坐标） | 硬依赖 |
| B.场景管线 → | C.证据管线 | B1.场景背景 → C1.证据图片（确定摆放位置） | 软依赖 |
| A2.表情底图 → | A3.表情变体 | 美术→策划交接点 | 硬依赖 |
| A3.表情变体 → | A4.对话视频 | 策划内部流转 | 硬依赖 |

> **硬依赖** = 上游未完成则下游无法开工
> **软依赖** = 可用占位符先行，但最终需要上游产出

## 推荐生产顺序

基于管线间依赖关系，每章美术的最优生产顺序：

```
第1阶段（可并行启动）
 ├── A1. 角色头像 ←── 人物设定
 ├── A2. 基础表情底图 ←── 人物设定
 └── B1. 场景背景 ←── 场景设计文档

第2阶段（依赖第1阶段）
 ├── A5. 场景NPC立绘 ←── 人物设定 + B1.场景背景
 ├── B2. 门转场图 ←── B1.场景背景
 ├── B3. 空镜转场图 ←── B1.场景背景
 └── C1-C3. 证据+道具+谜题 ←── 证据设计 + B1.场景背景

第3阶段（依赖第2阶段）
 ├── D1. 指证场景 ←── A5.场景NPC立绘
 └── D2. 指证动画 ←── D1.指证场景

策划并行线（依赖A2完成即可启动，与第2-3阶段并行）
 ├── A3. 表情变体 ←── A2.表情底图
 └── A4. 对话视频 ←── A3.表情变体 + 对话定稿
```
