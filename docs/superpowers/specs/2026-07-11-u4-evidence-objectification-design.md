# Unit4 证据物件化优化设计

## 目标

在不改变 `剧情设计/Unit4/Unit4_大纲0710_逻辑重构版_v2.md` 的五个 Loop、谎言维度、证据时序、人物责任和 U4→U5 信息边界的前提下，把约三分之一的“纸条 / 报告 / 零散文档”改成可观察、可分析或可组合的实物证据。

## 总体原则

1. 法律身份、账户归属、正式审批、机构命令必须保留文书。
2. 枪弹、药物、剂量、装置、寄存柜等物理事实优先由原物和分析结果证明。
3. 分析产物表现为“被分析后的物件”，不再默认生成一张世界内报告纸。
4. 同一主题的零散材料可以收进一个有辨识度的主题容器，但后续指证所需的具体结论仍须可单独引用。
5. 不增加新的上层知情事实，不提前坐实 Mickey、Miller、小 Charles 或银月会尚未到揭晓时点的责任。
6. 本轮只修改主准大纲，不分配 Item ID，不写 state / table / AVG 配置。

## 逐 Loop 修改设计

### Loop 1

1. `弹道初步`改为`Harrison 案弹头 / 弹壳与警局配发弹药样本`。
   - 开篇只公布 Harrison 死于枪伤，不提前公布警局制式弹药结论。玩家把现场弹壳和尸内弹头与法院证物室的警局配发样本做分析；警局合同批次具有独有的底火漆与双环压口记号。
   - 派生结论仍然只到“执行者能接触警用供弹渠道”，不能据此指认 Pierce 或 Morrison。
2. `Harrison 最后工作顺序表`改为`Harrison 未完成的自首证物箱`。
   - 箱内按实际打包顺序放置 Mary 改判、医院名单、资金图和开发计划；箱盖内的交接标签写明若再次被要求撤回，就交给外部律师。
   - 证物箱只作为保险柜内的场景容器和组织方式，不额外生成一张“总证据卡”；死亡对照表、资金图、开发计划和事故索引仍作为正式文书保留。

### Loop 2

3. `慈善项目每日剂量卡`改为`医院配发红线注射器与十三日封签药盒`。
   - 注射器玻璃管的红线处装有不可移动的金属止动环，单次抽取不能越过规定剂量；药盒外沿保留与盒体一体的十三日指纹确认条，不作为独立纸面证据进入背包。Rosa 证词确认她只会抽到红线。
4. `回收药瓶剩余量鉴定`改为`经容量分析的回收药瓶`。
   - 玩家用医院标定的初始装量分析药瓶剩余量，得到“十三次红线剂量完全吻合”的派生结论；瓶塞穿刺痕只作辅助，不承担精确计数。
5. `同批次药剂化验报告`改为`经化验的问题批次药剂`。
   - 分析后的药剂样本直接显示浓度波动与降解杂质，不生成独立报告纸。
6. `批次封签对照`改为`同批次药瓶封签组`。
   - Foster 在指证前依法从医院留存柜调取两只死亡病例空瓶。Harrison 样本、Rosa 药瓶和两只空瓶保留实体瓶颈封签；玩家组合后确认属于同一采购批次。
7. 医院采购与发放记录、第十九页缺失和儿童死亡对照表保持文书形态。

### Loop 3

8. `配枪与弹药初检`改为`磨号手枪与枪内未击发余弹`。
   - 玩家直接分析被磨去枪号的手枪和未击发余弹；余弹的压口、底火与 James 案样本指向同一供枪渠道。
   - 结论仍不能认定是同一把枪。
9. `车站寄存柜领取凭条`改为`缝纫盒夹层里的 214 号黄铜寄存柜钥匙`。
   - 指证前，爆炸震开缝纫盒夹层，玩家拍下钥匙但没有取得原物；Doris 否认钥匙与 Thomas 的材料有关。
   - Zack 通过电话向车站值班行李员核实：Thomas 当晚 19:55 以本人姓名租下 214 柜，把一只扎绳文件包放进柜内并取走钥匙；行李员没有看见文件内容。钥匙观察证据与行李员证词共同反驳“Thomas 没有留下材料，也没让我保管东西”。
   - 指证后 Doris 承认钥匙由 Thomas 当晚交给她，并正式允许 Zack 带走同一把钥匙；烧剩的 Brennan 信封继续证明预定收件人是 Zack，但不再作为 R3 的第二件击破材料。
10. `伪造煤气检修卡`改为`伪造市政煤气铅封与异常时钟接线`。
    - 爆炸前玩家只拍下铅封完整序号、新铜线和时钟接点，不会提前得到“可拆除的定时引燃器”；Emma 据此发现危险并组织撤离。
    - 爆炸后从残骸回收耐火时钟接点，并向市政核验铅封序号，派生“装置在 18:12 来访时预先安装”的结论。两名假检修员身份仍由 Doris 证词提供，L3 不能确认命令源。

### Loop 4

11. `Patrick 遗物封袋 + Zack 童年练习本 + 与练习本吻合的残页`归并为`Patrick 遗物匣`。
    - 回收 EPI02 已存在的物件 `2107 Patrick 遗物匣`：匣盖有 `P.B.`，正面嵌工运徽章，侧面刻有 `FOR THE MANY WE PAY THE FEW`。该刻句在 U2 已可作为环境信息看见，但当时不进入背包、不触发解释；L4 由 Margaret 主动交出遗物匣，才使刻句进入当前推理链。
    - U4 首次打开匣子，里面保存未使用返程票、带封存箱编号的焦黑金属扣和原证物标签。
    - 清场补充记录继续负责证明 Patrick 已抵达出口且折返时不知道被困者身份；遗物匣只证明他有返程安排、封存箱未越过闸门，并回收信条。
12. L5 保险柜密码统一为 `FTMWPTF`，不再使用当前 v2 的 `FTMTFMP`。
    - Mickey 没想到 Zack 会把 U2 见过、一直只被当作家庭遗物的刻句与自己的私人保险柜连接起来，也没想到 Margaret 会在 L4 把遗物匣及其内部材料交给 Zack。
    - L4 的 U3 公文包复审由 Emma 看到匣身刻句触发，不再由纸张残页触发。

### Loop 5

13. `Mickey 日常工作笔记`改为`Mickey 定制钢笔 + 法人卷宗上的已知亲笔批注`。
    - 钢笔带姓名缩写和有缺口的笔尖；法人卷宗保留 Mickey 的已知亲笔批注。
    - 法人卷宗中的批注有正式签署来源，能够确认出自 Mickey。与编码操作簿组合后，只用同笔迹与同一缺口笔尖造成的断墨特征得出“操作簿由 Mickey 书写”；不由钢笔链证明记录内容必然真实或命令已经执行。
14. 法人卷宗、银行授权附页、编码操作簿、功业簿、Miller 保险外卷 / 内封套、自动清场副本全部保留文书形态。
    - 这些文件分别承担正式身份、行为记录、价值主题、上层结构和命令边界，强行物件化会削弱唯一指向。

## 指证材料变更

### L2 Whitfield

- R1：`医院配发红线注射器与十三日封签药盒 + 经容量分析的回收药瓶`。
- R2：`五年七例儿童死亡对照表 + 同批次药瓶封签组`。
- R3：`经化验的问题批次药剂 + 医院采购与发放记录`。

### L3 Doris

- R3：`214 号黄铜寄存柜钥匙观察 + 车站值班行李员证词`。

R1、R2 不变；R3 证明 Thomas 当晚租下外部寄存柜、钥匙随后藏在 Doris 缝纫盒中。烧剩的 Brennan 信封和指证后半封信继续证明交接对象明确指向 Zack。

## 验收标准

1. 原有五个 Loop、指证对象和每轮谎言不变。
2. 所有替换物在首次用于疑点 / 指证前已经可获得或可分析。
3. 物件不承担超出其物理属性的结论；Patrick 遗物匣拆成“刻句、返程票、编号金属扣”三个观察结果，自首证物箱不作为额外总证据卡。
4. L1-L4 不提前坐实 Mickey = Whale；L4 不用 Patrick 遗物直接定罪。
5. U4 不提前坐实 Sean 完整执行链、水源行动主使或小 Charles 的责任。
6. `FOR THE MANY WE PAY THE FEW` 与现有 `2107 Patrick 遗物匣` 完全一致。
7. 修改后不再出现旧密码 `FTMTFMP`、练习本残页或领取凭条的残留引用。
