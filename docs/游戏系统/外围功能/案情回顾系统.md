# 案情回顾系统

## 一、核心概念

案情回顾系统（Case Review）是玩家管理游戏进度的中枢界面。通过该系统，玩家可以浏览所有章节的完成状态、查看各章节内的关键进度节点、回放已解锁的剧情视频，以及回溯到已完成章节的起点重新游玩。

每个"章节"对应策划内部的一个循环（Loop），但玩家界面中不出现"循环"字样，只显示章节主题名称（如"残酷的身份确认"）。

## 二、入口与触发

| 入口 | 说明 |
|------|------|
| OVERVIEW 按钮 | 主界面右上角，点击后直接进入**当前章节**的案情进展界面 |
| CHAPTER SELECT | 从案情进展界面左上角点击，进入章节选择界面查看所有章节 |
| 章节结束 | 指证成功后自动弹出，显示章节总结 |
| 设置菜单 | F1 设置面板中的"案情回顾"选项 |
| 开始界面 | 主菜单 CHAPTER SELECT 选项，先进入章节选择界面 |

## 三、界面结构

### 3.1 章节选择界面

水平排列所有章节卡片，每张卡片包含主题插图、章节编号、章节名称和完成状态。

| 卡片状态 | 样式 | 标题 | 交互 |
|----------|------|------|------|
| 已完成 | 报纸风格，泛黄纸张 + [SOLVED] 红色印章 | 完整章节名 | 可点击查看详情/回溯 |
| 进行中 | 拍立得照片风格 | 显示"???"（避免剧透指证对象） | 可点击返回游戏 |
| 未解锁 | 模糊拍立得 + "???" | "???" | 不可点击 |

卡片底部信息条用颜色区分状态：已完成为深红底条 + "COMPLETED"，进行中为红色底条 + "IN PROGRESS"，未解锁为灰色底条 + "COMING SOON"。

### 3.2 章节详情界面（案情进展）

点击某个章节卡片后进入详情，右侧面板纵向排列该章节的进度节点：

| 节点图标 | 类型 | 说明 | 可回放 |
|----------|------|------|--------|
| ○ | 普通节点 | 调查、对话等常规阶段 | 否 |
| ⬡ | 关键节点 | 重要剧情转折点 | 否 |
| ● | 视频节点 | 章节开场或指证真相回顾 | 是 |

每个章节包含四个标准节点：故事开端（视频）、开展调查（普通）、证人访谈（关键）、指证/真相回顾（视频）。视频节点旁有 [播放] 按钮，指证成功的视频额外标注 [真相回顾]。

详情界面顶部显示章节名称，右上角有 **[回溯至此]** 按钮。

## 四、回溯功能

回溯允许玩家从已完成章节的起点重新开始游玩。

### 回溯规则

- 回溯只影响目标章节所在的单元，其他单元的进度不受影响
- 目标章节之后的章节进度被清除，之前的章节进度保留
- 成就记录永久保留，不受回溯影响

| 数据类型 | 回溯后状态 |
|----------|-----------|
| 目标章节之前的证据/对话 | 保留 |
| 目标章节之后获得的证据/对话 | 按章节过滤，仅在最新进度可见全部 |
| NPC 状态、场景解锁 | 重置为目标章节起点 |
| 成就/统计 | 永久保留 |

### 回溯操作流程

```
切换到目标章节 → 点击 [回溯至此] → 确认弹窗（警告后续进度将清除） → 确认 → 加载该章节起点
```

确认弹窗会明确告知哪些数据将重置、哪些将保留，防止玩家误操作。

## 五、视频回放

已完成章节的视频节点支持回放，包括章节开场演出和指证成功的真相回顾视频。回放时支持播放/暂停（点击画面）、跳过（SKIP 按钮）、退出（返回按钮）。

## 六、与其他系统的关联

| 关联系统 | 关系 |
|----------|------|
| 存档系统 | 回溯功能依赖章节快照存档，每次指证成功自动创建快照 |
| 指证系统 | 指证成功触发章节完成，更新案情回顾中的章节状态 |
| 开始界面 | 主菜单 CHAPTER SELECT 跳转到章节选择界面 |
| 设置系统 | F1 设置面板提供案情回顾入口 |
