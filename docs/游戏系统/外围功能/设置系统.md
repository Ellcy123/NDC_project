# 设置系统

## 一、核心概念

设置系统（Settings & Help）是游戏的全局配置界面，玩家可在任意阶段通过 F1 键、Esc 暂停菜单或主菜单 SETTINGS 按钮唤出。界面采用左右分栏布局：左侧为复古纸张风格的设置选项区，右侧为模糊处理的游戏场景背景。

## 二、主菜单选项

设置面板顶部提供三个功能入口：

| 选项 | 功能 |
|------|------|
| 继续游戏 | 关闭设置界面，返回当前游戏。若有设置变更，按钮文本变为"保存并继续游戏"并显示金黄色脉冲动画 |
| 案情回顾 | 跳转到案情回顾系统 |
| 返回主菜单 | 退出当前游戏回到开始界面，弹出确认提示警告未保存进度将丢失 |

## 三、难度设置

提供两档难度，游戏任意阶段均可切换，确认后立即生效：

| 难度 | 核心差异 |
|------|----------|
| 普通模式 | 场景热区高亮提示、证据已收集标记、关键词下划线、组合物品数量提示、矛盾点自动高亮、指证无重试限制 |
| 专家模式 | 无任何辅助提示、标准热区大小、需手动对比发现矛盾、每轮指证最多 3 次错误 |

从专家切换到普通时弹出警告：当前章节将无法获得"专家通关"成就。难度切换不影响已收集的证据和已使用的提示次数。

### 难度差异细项

| 功能维度 | 普通模式 | 专家模式 |
|----------|----------|----------|
| 场景热区 | 可高亮显示，范围 +20% | 无提示，标准大小 |
| 已收集证据位置 | 显示勾选标记 | 不显示 |
| 对话关键词 | 可提取内容显示下划线 | 无高亮 |
| 物品组合 | 显示所需数量，可组合物品发光 | 无标记 |
| 时间线/矛盾点 | 自动高亮 | 手动对比 |
| 指证失败提示 | "证据不足" | 仅"指证失败" |

## 四、语言设置

| 语言 | 说明 |
|------|------|
| 英语（默认） | UI 文本、对话字幕、系统提示均为英语 |
| 简体中文 | 同上切换为中文 |

切换时弹出双语确认提示，确认后立即生效，无需重启。语音始终为英语配音，不受语言设置影响。

## 五、字幕设置

| 选项 | 说明 |
|------|------|
| 字幕开关 | 开启/关闭语音对应的字幕显示 |
| 字幕大小 | 小（18px）/ 中（24px）/ 大（32px） |

## 六、音频设置

| 选项 | 控制方式 | 默认值 | 影响范围 |
|------|----------|--------|----------|
| 音乐音效 | 滑块 0-100 | 70 | 背景音乐（BGM）+ 环境音效（SFX） |
| 语音 | 滑块 0-100 | 80 | 角色配音 |

调整时实时生效，语音滑块调整时播放示例语音供预览。

## 七、画面设置

| 选项 | 可选值 | 默认 |
|------|--------|------|
| 亮度 | 滑块 0-100 | 50 |
| 屏幕比例 | 16:9 / 21:9 / 16:10 / 4:3 | 16:9 |
| 显示模式 | 全屏 / 窗口化 / 无边框窗口 | 全屏 |

所有画面设置切换后立即生效。

## 八、社交链接

设置面板底部提供官方社交平台图标：Steam、YouTube、Discord、X(Twitter)、Instagram。点击后调用系统默认浏览器打开对应链接，Steam 链接优先调用客户端。无网络时提示"请检查网络连接"。

## 九、与其他系统的关联

| 关联系统 | 关系 |
|----------|------|
| 案情回顾系统 | 设置面板提供案情回顾入口 |
| 开始界面 | "返回主菜单"跳转回开始界面 |
| 难度系统 | 难度设置影响场景搜证、对话、案情分析、指证四个维度的辅助提示 |
| 多语言系统 | 语言切换调用多语言系统刷新全部 UI 文本 |
