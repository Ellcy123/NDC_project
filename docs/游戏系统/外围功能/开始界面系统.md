# 开始界面系统

## 一、核心概念

开始界面（Title Screen）是玩家启动游戏后看到的第一个界面，负责品牌展示、游戏进度入口和氛围营造。背景采用 1920 年代芝加哥街景，配合复古红黑配色和 Art Deco 字体，建立侦探游戏的视觉基调。

## 二、界面构成

### 2.1 视觉布局

- **背景**：黄昏时分的芝加哥城市街道，远处工厂烟囱、近处码头木栈道、路灯散发昏黄光芒
- **Logo 区域**：左上方，显示游戏名称和副标题
- **菜单区域**：左侧垂直排列，半透明黑色遮罩背景
- **版权信息**：底部居中

### 2.2 按钮样式

| 状态 | 视觉表现 |
|------|----------|
| 默认 | 深灰底色 + 白色文字 |
| 选中/悬停 | 红色底色 + 白色文字 + 左侧红色横条延伸至屏幕边缘 |
| 禁用 | 深灰底色 + 灰色文字（50%透明度） |
| 点击 | 亮红色底色 + 按压缩放动画 |

## 三、菜单选项

菜单根据存档状态动态变化：

| 选项 | 功能 | 显示条件 |
|------|------|----------|
| CONTINUE | 加载最近存档，继续游戏 | 有进行中存档时出现，默认选中，按钮下方显示当前章节名 |
| NEW GAME | 开始全新游戏 | 始终显示 |
| CHAPTER SELECT | 进入章节选择界面 | 始终显示，至少完成一个章节后可用 |
| ACHIEVEMENT | 查看成就列表和解锁状态 | 始终显示 |
| SETTINGS | 进入设置系统 | 始终显示 |
| DOWNLOAD | DLC/额外内容下载 | 始终显示，当前为预留功能 |
| CREDITS | 滚动字幕展示制作团队 | 始终显示 |
| QUIT | 退出游戏 | 始终显示 |

### 3.1 新游戏流程

已有存档时点击 NEW GAME 会弹出确认提示，显示当前进度和游玩时长，确认后进入难度选择：

| 难度 | 说明 |
|------|------|
| 普通模式（NORMAL） | 提供适量提示，推荐首次游玩 |
| 专家模式（EXPERT） | 无任何提示，纯粹推理挑战 |

选择难度后播放开篇视频，进入游戏。难度可在游戏过程中随时通过设置调整。

### 3.2 继续游戏流程

点击 CONTINUE 后依次执行：检测最近存档类型 → 显示加载进度 → 恢复场景、证据、对话进度 → 进入游戏。

### 3.3 章节选择

跳转到案情回顾系统的章节选择界面，与游戏中 OVERVIEW 入口不同的是：开始界面先显示所有章节卡片，点击具体章节后才进入该章节详情。

## 四、存档机制

| 存档类型 | 触发时机 | 保留策略 |
|----------|----------|----------|
| 章节存档 | 章节结束（指证成功） | 永久保留 |
| 自动存档 | 场景切换、关键对话完成、证据获取等 | 保留最近 3 个 |
| 临时存档 | 退出游戏时 | 下次启动后清除 |
| 快照存档 | 每 10 分钟自动创建 | 保留最近 1 个 |

不支持章节进行中的手动存档，确保每次游玩是完整的章节体验。退出游戏时自动触发临时存档，下次启动通过 CONTINUE 恢复。

## 五、与其他系统的关联

| 关联系统 | 关系 |
|----------|------|
| 案情回顾系统 | CHAPTER SELECT 跳转到案情回顾的章节选择界面 |
| 设置系统 | SETTINGS 跳转到设置面板 |
| 存档系统 | CONTINUE 依赖自动/临时存档恢复进度 |
| 难度系统 | NEW GAME 流程包含难度选择，影响后续游戏内提示策略 |
