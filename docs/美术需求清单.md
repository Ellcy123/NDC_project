## 六、COMBINE 物品组合

### 6.1 交互形式

1. 当鼠标拖拽某个 ITEM 时，图标被拉起并跟随鼠标。出现盒子，松开鼠标盒子消失
2. 当道具被放入其中一个格子后，出现黑色背景和对应标题，以及组合按钮
3. 当组合不成功时，两个框内的道具飞回到原始位置，并显示不能组合的提醒文字
4. 成功后播放视频合成动画（AI 动画，需 shader 绿屏抠像）
5. 选中后黑色背景色 `#000000`，90% 透明度
