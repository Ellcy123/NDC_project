# 突发事件：花形压痕闪回 (Daisy Imprint Flashback)

**文档版本**: v1.0
**隶属章节**: Unit 3 - 循环5
**触发位置**: 侧光压痕分析小玩法 Phase 3
**事件类型**: sudden_flashback（不可控记忆闪回）

---

## 0. 前因后果

### 前因

- 玩家正在进行**侧光压痕分析**小玩法
- Phase 2（报纸叠合比对）完成后，垫板右侧边缘出现闪烁光点
- 玩家点击光点 → zoom到花形压痕特写 → 看到五瓣花形凹痕（1.2cm）
- **此时触发突发闪回**

### 关联记忆

- **L2 Expose追问Mary**：Mary被Zack追问时，反复下意识地用手指摸手腕上的手链，手链上挂着一个花形吊坠
- 这个行为细节当时不起眼，但Zack的潜意识记住了

### 后果

- 闪回本身**不产生任何确认性结论**——Zack不说破，不生成新证据
- 只获得3507「垫板花形压痕照片」（Clue类型）
- 花形饰品的主人身份留为悬念
- **Expose R2阶段B**：Zack主动出示3507追击Bernard → Bernard承认有女人来访
- **过渡事件**（Expose后银行门外）：Emma首次推测性地将花形压痕与Mary手链关联

## 3. 闪回后反应

### Zack的克制反应

```
[闪回结束，回到花形压痕特写]
[Zack愣了一下——眼神微微变化，嘴唇动了一下但没说话]

Foster: "Brennan先生？"

Zack:   "……没什么。"
        （停顿0.5秒）
        "这不是文字。是某种饰品的印记。"

Foster: "压痕深度来看，至少十到十五分钟持续按压。
         而且位置非常集中——这个人几乎一动不动。"

Zack:   "……"
        （沉默2秒，Zack盯着花形压痕，表情从专业分析变为若有所思）
        "先拍下来。"
```

### 设计要点

- Zack的反应刻意**克制**——他可能想起了什么，但选择不说
- "……没什么"和最后的沉默形成**反常信号**：前两个Phase Zack都很健谈，突然变安静了
- 不说"Mary"、不说"手链"、不说"我记得"——零确认
- Foster的"Brennan先生？"暗示她注意到Zack走神了
- 玩家体验两条路径：
  - **注意到闪回的玩家**："那个画面……Mary的手链？所以花形压痕是Mary留下的？"→ 产生强烈悬疑
  - **没注意到的玩家**："Zack怎么突然安静了？"→ 感受到氛围异常但不确定原因 → 过渡事件中Emma提出时恍然大悟

## 5. 视听设计

### 音效时间线

| 时间 | 音效 | 描述 |
|------|------|------|
| T+0.3s 闪白#1 | `sfx_flash_in` | 短促高频音（类似耳鸣/相机快门），0.1秒 |
| T+0.4~1.2s 闪回 | `sfx_memory_ambient` | 极低音量的L2场景环境声（模糊的室内对话声、椅子轻响），像从水下听到的声音 |
| T+0.4~1.2s 闪回 | `sfx_chain_jingle` | 手链晃动的极轻金属声，若有若无 |
| T+1.2s 闪白#2 | `sfx_flash_out` | 与#1相同的高频音，但略低一个音调（"回来了"的感觉） |
| T+1.5s+ Zack沉默 | `sfx_silence` | 刻意的静音2秒——前面Phase 1/2一直有背景音乐，此刻突然安静 |

### 音乐处理

- 闪回触发前：Phase 2结束的确认旋律余韵
- 闪回期间：**所有背景音乐停止**，只有上述音效
- 闪回结束后：**不恢复音乐**，保持安静直到Zack说"先拍下来"
- "先拍下来"后：缓慢渐入一段新的低音弦乐（暗示"事情变了"）

### 视觉效果

| 效果 | 参数 |
|------|------|
| **闪白** | 纯白覆盖层，Alpha: 0→1→0，持续0.1秒，缓动曲线EaseInOut |
| **记忆滤镜** | ColorGrading: 暖色偏移+15, 对比度-10, 饱和度-20; 径向模糊半径30px; Film Grain 15% |
| **呼吸缩放** | 画面Scale: 1.0→1.005→1.0，周期1.6秒（模拟记忆的不稳定感） |
| **Zack表情** | 如果有角色立绘/3D模型：眼神从聚焦→失焦→重新聚焦；嘴唇微动后抿住 |

## 7. 开发备注

- **美术资源**：
  - L2 Mary场景的手链特写画面（可从L2场景素材中裁切/重绘）
  - 记忆滤镜后处理Shader（暖色偏移+径向模糊+颗粒感）
  - 闪白过渡效果（全屏白色叠加层）

- **技术要点**：
  - 闪回画面可以是预渲染的视频片段（推荐）或实时场景切换
  - 预渲染方案：0.8秒MP4/WebM视频，含记忆滤镜效果，加载时预缓存避免卡顿
  - 实时方案：切换到L2场景Camera，应用PostProcessing Volume，0.8秒后切回
  - 闪白用CanvasGroup + Alpha动画实现
  - **关键**：闪回前后不能有任何加载停顿，必须无缝衔接

- **QA测试**：
  - [ ] 闪回时序是否精确（0.3s延迟 → 0.1s闪白 → 0.8s画面 → 0.1s闪白）
  - [ ] 闪回期间玩家输入是否完全屏蔽
  - [ ] 音乐停止和静音段是否正确触发
  - [ ] 闪回只触发一次（重进小玩法不再触发）
  - [ ] 记忆滤镜效果在不同分辨率下表现一致

- **设计备忘**：
  1. 闪回画面中Mary的手链吊坠**不能太清晰**——应该是自然的记忆画面，吊坠只是画面的一部分，不是焦点。如果做得太明显（比如吊坠居中放大），就变成了"系统在告诉你答案"而不是"Zack的记忆一闪而过"
  2. Zack的"……没什么"是整个事件的灵魂——这句话要让玩家感觉到"他在说谎/隐瞒"，暗示Zack自己也不确定要不要面对这个联想
  3. 这个闪回是**玩家和Zack共享的直觉时刻**——玩家的疑问和Zack的疑问同步产生
