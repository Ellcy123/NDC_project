# SC3002 Thomas公寓楼顶

## 场景信息
- **位置**：Thomas公寓三楼楼顶天台，护栏边缘为坠落点
- **类型**：案发现场 / 搜证场景
- **场景编号**：SC3002（L1搜证 / L3二次搜证 / L4门把手+护栏检查）

## 场景描述

三楼天台，案发核心地点。护栏边缘就是Thomas的坠落点。L1开放范围只覆盖男女脚印、Jameson威士忌空瓶、外墙抓挠痕等外围线索；护栏核心区被警戒线封锁，栏杆细节、地面油迹和血迹到L3才解封。

L3带着Foster重返楼顶（SC3002），进行更细致的现场勘查。Foster绘制地面涂油示意图，发现护栏附近地面有人为涂抹的润滑油，范围精确对应坠落点。同时对L1的37码女性平底鞋印做特写分析，为后续Mary/Helen同码同款鞋的混淆埋伏笔。

L4第三次搜查（SC3002）聚焦门把手——安全门外侧（楼梯间侧）把手有极薄油膜，内侧（天台侧）干净无油迹，方向性证明有人从楼梯间带着油进入天台。此外护栏锈蚀处发现干涸血迹残留，与Helen右手伤口吻合。
